Composing Non-Conjugate Factor Graphs with Closed-Form Variational Inference

Lukashchuk, Yemets, Kouw, Bagaev, Senoz, Beck & de Vries — Eindhoven University of Technology & collaborators
Five factor-graph primitives compose freely under mean-field factorization while preserving closed-form VMP — stacking them yields routing architectures with universal approximation and calibrated uncertainty, without gradient-based learning.
Key Finding
A five-letter factor-graph alphabet (softdot, exp-link, Gamma prior, Gaussian likelihood, equality node) composes into arbitrarily deep models while every VMP message stays in closed form — Gaussian on weight edges, Gamma on precision edges.
The only non-conjugate interface (exponential link) is tractable via Gaussian MGF forward and Gamma sufficient statistics reverse. Theorem 1 guarantees this for any "proper factor graph."
Key Metric
94%
parameter reduction on Exchange Rate
Noisy Diagonal: 15,512 → 952 parameters while achieving the best NLL across all horizons. Softmax MoE baselines produced non-finite NLL — this model stays calibrated.
Why This Matters
Narrows the gap between deep learning's compositional modularity and factor graph inference — calibrated uncertainty comes out of the box, not as a post-hoc add-on.
Gating functions are learned through inference, not gradient descent. The "alphabet + grammar + runtime" metaphor makes compositional Bayesian modeling as structured as a programming language.
Critical Edge
Restricted to Gaussian/Gamma families only. No discrete latents, no multi-modal posteriors. Mean-field assumption limits posterior quality — and the approximation gap is not quantified.
The universal approximation result requires sharp routing (τ→∞), which collapses uncertainty to deterministic selection. Scalability (message passing cost vs. graph depth and data size) is unanalyzed.
Alphabet
fz softdot fexp exp link fG Gamma prior fN Gaussian lik. f= equality

Each primitive is a factor node with typed edges: solid (Gaussian weights/observations), dashed (Gamma precisions), dash-dotted (latent z). The "type system" ensures VMP messages never leave their family.

Compositional Grammar
DEPTH 0 Static ensemble. One Gamma prior per expert. Recovers Cochran 1937 inverse-variance weighting.
DEPTH 1 Input-dependent gating via "precision word." Precision becomes a function of features φ(x). This is the Precision-Gated Expert.
DEPTH 2 Split-branch routing. Two competing branches with router weights encode XOR and arbitrary decision trees. Corollary 1: universal function approximation.
Results Across Five Benchmarks

Five forecasting benchmarks (ETTh1, ETTh2, Exchange Rate, Electricity, Traffic) with horizons H ∈ {96,192,336,720}. Seven forecasters combined (five neural + two constant quantiles). VAE features with 64-dim latent space.

Strengths & Limitations
Strengths
  • Elegant formalization: proper factor graphs, typed message system
  • Clean alphabet/grammar/runtime analogy
  • Calibrated uncertainty where neural MoE fails (finite NLL)
  • 94% parameter reduction via diagonal variants
  • Open-source: GraphPPL.jl + RxInfer.jl
Limitations
  • Gaussian/Gamma only — no discrete latents, no multi-modality
  • Mean-field assumption; approximation gap not quantified
  • Universal approx. requires τ→∞, collapsing uncertainty
  • Neural MoE baselines are simple (single/two-layer)
  • Computational scalability unanalyzed
Inference Runtime

Bethe free energy minimization via message passing. Most factors use VMP; the equality node and exponential link use belief propagation. The exponential link's BP forward message produces a log-normal on γ; reverse produces a log-gamma on z. Tractability follows from the restricted alphabet: incoming messages always have known analytic forms.

Implementation: GraphPPL.jl for model specification, RxInfer.jl for execution. Alphabet maps directly — NormalMeanPrecision, GammaShapeRate, softdot, Log.

Open Questions
