WHY THIS MATTERS

Standard PEFT was built for autoregressive models. Diffusion LLMs face noise-varying input distributions across denoising steps — static LoRA fundamentally cannot adapt. NaRA closes this gap at near-zero extra cost (0.008% more parameters), making proper fine-tuning of dLLMs finally tractable.

CRITICAL EDGE
  • Only evaluated on LLaDA-8B — untested on larger dLLMs (Mercury, Dream)
  • No inference latency numbers reported despite per-step hypernetwork calls
  • Math gains modest (+0.82%), suggesting diminishing returns on some tasks

◆ The Problem

  • Diffusion LLMs denoise tokens in parallel across multiple steps; noise level λ = m / Lr (masked tokens / response length) varies continuously
  • At high λ, most tokens are masked and generation difficulty peaks; at low λ, the task is easier
  • Standard LoRA applies a static ΔW = B·A regardless of λ — empirically negligible improvement at high noise
  • dLLMs face C(L, m) possible masked configurations per step vs. a unique prefix for AR models — exponentially more diverse

◈ NaRA Solution

  • Inserts a dynamic core matrix C(λ) between LoRA's static A and B projections
  • C(λ) is generated by a lightweight globally-shared hypernetwork conditioned on noise level λ
  • Identity initialization C(λ) = Ir + η · Fφ(eλ) ensures training stability
  • Only 0.425% trainable params vs LoRA's 0.417% — a 0.008% overhead
  • Theorem 4.1 proves N arbitrary update matrices can be represented by B·Ci·A when rank ≥ max dimension

⚙ Hypernetwork Architecture

noise level λ ↓ Gaussian Fourier Embedding eλ = cos(2πkλ) ⊕ sin(2πkλ) , k ~ N(0, σ²) ↓ MLP Fφ hidden [256, 512] ↓ Core Matrix C(λ) = Ir + η · reshape(Fφ(eλ)) // η = 0.1 ↓ ΔW(λ) = B · C(λ) · A // globally shared across all layers

Key Ablation: NaRA-C (without noise conditioning)

Removing noise conditioning (learnable but unconditioned C matrix) causes catastrophic collapse: 68.13% on commonsense vs NaRA's 84.09%. This confirms the improvements come from noise awareness, not merely from adding parameters.

+ Strengths

  • Identifies a genuine, previously unaddressed structural mismatch between static PEFT and dLLMs
  • Mathematically principled with theoretical backing (Theorem 4.1, combinatorial diversity argument)
  • Extremely parameter-efficient: 0.008% extra over standard LoRA
  • Comprehensive ablation suite validates every design choice (embedding type, sharing strategy, scaling factor, rank)
  • The NaRA-C ablation is strong causal evidence that noise conditioning drives improvements
  • Cross-domain generalization to image diffusion (SDXL DreamBooth) demonstrates broader applicability
  • Fourier embedding analysis provides practical guidance: lowest variance, highest robustness
  • Public code at github.com/generaldi/NaRA with detailed hyperparameters

! Limitations

  • Only evaluated on LLaDA-8B (Base + Instruct); scalability to larger dLLMs is unknown
  • No inference latency measurements. Hypernetwork runs at every denoising step for every adapted module
  • Math improvements are modest (+0.82%), suggesting diminishing returns on tasks where noise dynamics matter less
  • Improvement over HiRA is smaller than over LoRA on some benchmarks — part of the gain may be from extra expressiveness, not purely noise awareness
  • Does not compare against DoRA, AdaLoRA, or other recent adaptive PEFT methods
  • Depends on continued growth of the dLLM paradigm, which remains nascent
  • WinoGrande zero-shot results suspiciously low (0.08), suggesting possible evaluation issues
