> **一句话总结：** 本文提出 NaRA（Noise-aware Low-Rank Adaptation），通过一个以噪声水平为条件的轻量级全局共享超网络（MLP + Gaussian Fourier Embedding）动态生成 LoRA 的低秩核心矩阵 C(lambda)，使适配参数沿扩散去噪轨迹连续变化，在 LLaDA-8B 上的常识推理（84.09% vs LoRA 80.18%）、数学推理（80.89% vs 80.07%）和代码生成基准上持续优于标准 LoRA 和 HiRA 等基线，额外参数开销仅 0.008%。

> **TL;DR:** Standard LoRA uses static adaptation matrices that ignore the noise-level dynamics of diffusion LLMs (dLLMs), causing it to fail at high noise levels where generation difficulty is highest. NaRA inserts a noise-conditioned dynamic core matrix C(lambda) = I_r + eta * F_phi(e_lambda) between LoRA's static projection matrices A and B, where F_phi is a lightweight globally-shared hypernetwork (MLP with Fourier embeddings) that takes the current noise level as input. With only 0.008% additional parameters (0.425% vs LoRA's 0.417%), NaRA achieves SOTA on LLaDA-8B across commonsense reasoning (84.09%, +3.91% over LoRA), math reasoning (80.89%, +0.82%), and code generation benchmarks. Ablation confirms noise conditioning is essential -- removing it (NaRA-C) causes catastrophic performance collapse (68.13% on commonsense).

## 核心贡献

- 首次识别了将标准 PEFT 方法（为自回归模型设计）直接应用于扩散 LLM 的根本局限：静态适配参数无法应对扩散去噪过程中噪声水平变化导致的输入分布和生成难度的显著变化。
- 提出 NaRA 框架，在静态投影矩阵 A 和 B 之间插入噪声条件化的动态核心矩阵 C(lambda)，通过全局共享的超网络根据噪声水平生成，实现沿去噪轨迹的连续自适应。
- 提供了理论证明（Theorem 4.1）：多个不同的更新矩阵可等价表示为 NaRA 的分解框架 B * C_i * A，证明了框架的表达能力。
- 在 LLaDA-8B Instruct 和 Base 两个变体上，于常识推理（8个基准）、数学推理（4个基准）和代码生成任务上全面超越 LoRA、HiRA、P-Tuning 等基线，并展示了在图像扩散模型（SDXL DreamBooth）上的跨域泛化能力。

## Key Contributions

- **Problem identification:** First to identify that standard PEFT methods (designed for autoregressive models) are structurally mismatched to dLLMs because they impose noise-agnostic low-rank updates. Empirical evidence (Figure 1) shows standard LoRA provides negligible improvement at high noise levels compared to baseline.
- **NaRA framework:** Proposes inserting a dynamic core matrix C(lambda) between static projection matrices A and B: h = W_0 * x + B * C(lambda) * A * x. The core matrix is generated by a lightweight, globally-shared hypernetwork conditioned on the noise level lambda.
- **Theoretical justification:** Theorem 4.1 proves that N arbitrary update matrices can be equivalently represented by the decomposed framework B * C_i * A when rank r >= max dimension of the union of column/row spaces. Also provides theoretical intuition for why dLLMs face exponentially more diverse input distributions than AR models at each noise level (C(L,m) possible masked configurations vs. unique prefix).
- **Comprehensive evaluation:** SOTA results on LLaDA-8B across commonsense reasoning (8 benchmarks), mathematical reasoning (4 benchmarks), code generation, plus generalization to image diffusion (SDXL DreamBooth).
- **Ablation studies:** Thorough ablations on scaling factor eta, Fourier embedding vs. MLP vs. Scalar, hypernetwork sharing strategies (global vs. module-specific), rank sensitivity, and NaRA-C (unconditioned core matrix).

## Methodology

**Core Idea:** In dLLMs, the denoising process progressively unmasks tokens. At high noise levels, many tokens are masked and generation difficulty is high; at low noise levels, fewer tokens remain and the task is easier. Standard LoRA uses a fixed update matrix DeltaW = B*A throughout, ignoring this dynamic.

**NaRA Formulation:** NaRA decomposes the weight update as DeltaW(lambda) = B * C(lambda) * A, where:
- A (r x k) and B (d x r) are static projection matrices (same as standard LoRA)
- C(lambda) (r x r) is a noise-conditioned dynamic core matrix
- lambda = m / L_r is the noise level (ratio of masked tokens to response length)

**Hypernetwork Design:**
- Noise level lambda is first embedded using Gaussian Fourier features: e_lambda = cos(2*pi*k*lambda) concat sin(2*pi*k*lambda), where k ~ N(0, sigma^2). This mitigates spectral bias in low-dimensional inputs.
- The embedding is fed into a lightweight MLP hypernetwork F_phi with hidden sizes [256, 512].
- The core matrix is formulated as an additive deviation from identity: C(lambda) = I_r + eta * F_phi(e_lambda), where eta is a scaling factor (default 0.1). This identity initialization ensures training stability.
- The hypernetwork is globally shared across all layers and modules.

**Training:** NaRA adds negligible overhead -- only 0.425% trainable parameters vs. LoRA's 0.417% (a 0.008% increase). The hypernetwork parameters are the sole additional cost. Training uses standard diffusion cross-entropy loss with the same optimization setup as LoRA.

**Key Design Insight:** A Multi-LoRA baseline (4 separate adapters for 4 noise intervals) with ~4x more parameters (1.68%) significantly underperforms NaRA (74.83% vs 80.89% on math), confirming that continuous shared subspace is superior to discrete adapter allocation.

## Results & Highlights

**Commonsense Reasoning (LLaDA-8B-Instruct, r=32):**
- NaRA: 84.09% avg -- LoRA: 80.18% -- HiRA: 82.73% -- P-Tuning: 82.45%
- Largest gains on ARC-C (85.92 vs LoRA 75.74, +10.18), OBQA (86.87 vs 77.87, +9.00), SIQA (79.82 vs 75.38, +4.44)
- On LLaDA-8B-Base: NaRA 83.83% vs LoRA 76.10% (+7.73)

**Mathematical Reasoning (LLaDA-8B-Instruct, r=32):**
- NaRA: 80.89% avg -- LoRA: 80.07% -- HiRA: 78.07% -- P-Tuning: 79.64%
- Key gain on AQuA (57.27 vs LoRA 54.45, +2.82)
- At rank 16, NaRA (80.47%) still surpasses LoRA at rank 32 (80.07%), demonstrating superior parameter efficiency

**Code Generation:** Evaluated on HumanEval and MBPP using pass@1 metric via lm-evaluation-harness.

**Image Diffusion Generalization (SDXL DreamBooth):**
- DINO: 0.0610 vs LoRA 0.0458 (+33%) | CLIP-I: 0.7365 vs 0.6901 (+6.7%) | FID: 16.71 vs 22.13 (-24.5%)

**Ablation Findings:**
- Scaling factor eta=0.1 is optimal; eta=0.01 degenerates to static LoRA; eta=1.0 causes unstable training with high variance in update norms
- Fourier embedding outperforms MLP and Scalar baselines on both accuracy (80.89% vs 80.39%/80.26%) and stability (std dev 0.34 vs 1.26/0.68)
- Global sharing strategy outperforms all module-specific alternatives (80.89% vs 79.83-80.35%), despite having fewer parameters -- noise dependency is a global property
- NaRA-C (unconditioned learnable C matrix) catastrophically fails: 68.13% on commonsense (vs NaRA 84.09%), confirming noise conditioning is essential

## Critical Assessment

- **Novelty:** Moderate-to-high. The core idea (noise-conditioned dynamic LoRA) is a natural extension of Time-Varying LoRA (Zhuang et al., 2024) from image diffusion to language diffusion. The specific application to dLLMs and the theoretical analysis are novel. The hypernetwork architecture itself is straightforward.
- **Rigor:** Strong. Theorem 4.1 provides formal expressiveness guarantees. Empirical evaluation is thorough with 3 independent seeds reported. Ablations are comprehensive (scaling factor, embedding type, sharing strategy, rank sensitivity, Multi-LoRA comparison, NaRA-C). The NaRA-C ablation is particularly convincing in isolating the contribution of noise conditioning.
- **Reproducibility:** Good. Public code repository, detailed hyperparameter table provided (rank 32, Fourier embedding dim 64, MLP hidden [256, 512], eta=0.1, AdamW lr=1e-4, batch size 32 with gradient accumulation). Standard benchmarks used.
- **Significance:** Significant for the emerging dLLM field. The gains on commonsense reasoning are substantial (+3.91% over LoRA), though math improvements are more modest (+0.82%). The cross-domain generalization to image diffusion (SDXL) is a nice bonus.
- **Red flags:** (1) Only evaluated on LLaDA-8B -- not tested on larger dLLMs or other architectures (Dream, Mercury). (2) The commonsense reasoning gains are impressive but some individual benchmarks show inconsistent patterns (e.g., BoolQ and PIQA where LoRA sometimes matches or beats NaRA). (3) The hypernetwork adds inference cost at each denoising step, though claimed negligible -- no latency numbers are reported. (4) The WinoGrande results for LLaDA-Instruct zero-shot are suspiciously low (0.08), suggesting possible evaluation issues with that specific benchmark.

## Strengths

- Identifies and rigorously demonstrates a genuine, previously unaddressed limitation of applying static PEFT to dLLMs.
- Clean, mathematically principled formulation with theoretical backing (Theorem 4.1, combinatorial argument for dLLM vs AR diversity).
- Extremely parameter-efficient: only 0.008% additional parameters over standard LoRA.
- Comprehensive ablation suite that convincingly validates each design choice.
- The NaRA-C ablation is particularly strong evidence that noise conditioning (not just an extra learnable matrix) drives the improvements.
- Cross-domain generalization demonstrated on image diffusion (SDXL), showing broader applicability.
- Fourier embedding analysis provides useful practical guidance (lowest variance, highest robustness).

## Limitations

- Only evaluated on LLaDA-8B (both Base and Instruct); scalability to larger dLLMs (e.g., LLaDA-15B, Mercury, Dream) is untested.
- No inference latency measurements reported. While the hypernetwork is "lightweight," it must be invoked at every denoising step for every adapted module, which could accumulate.
- Math reasoning improvements are modest (+0.82% avg on Instruct), suggesting the method may have diminishing returns on tasks where noise-level dynamics matter less.
- The improvement over HiRA is smaller than over LoRA on some benchmarks, suggesting part of the gain may come from the additional expressiveness of the core matrix rather than purely noise awareness.
- Dependent on the continued growth of the dLLM paradigm, which remains nascent compared to autoregressive models.
- The paper does not compare against DoRA, AdaLoRA, or other recent adaptive PEFT methods.

## Relevance

本文针对扩散语言模型提出了噪声感知的参数高效微调方法，对关注非自回归生成、高效微调和扩散模型训练动态的研究者具有直接参考价值。其跨域泛化到图像扩散模型的实验也拓宽了方法的适用范围。

## Questions for Further Exploration

- How does NaRA scale to larger dLLM models (e.g., LLaDA with more parameters, or architectures like Mercury/Dream)?
- What is the actual wall-clock inference overhead of the hypernetwork evaluation at each denoising step?
- How does NaRA interact with recent dLLM-specific reinforcement learning methods (d1, d-TreePO, SPG)?
- Can the noise-aware core matrix concept be combined with other PEFT innovations (e.g., DoRA's weight decomposition)?
- Why does NaRA underperform P-Tuning on some individual commonsense benchmarks despite higher average -- is there a task-dependent optimal noise sensitivity?
- How does the method perform on longer generation tasks where the denoising trajectory is more extended?
