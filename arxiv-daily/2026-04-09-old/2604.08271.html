An Illusion of Unlearning? Assessing Machine Unlearning Through Internal Representations

arXiv: 2604.08271v1

Date: 2026-04-09

Tags: [machine-unlearning, neural-collapse, representation-learning, privacy, classification]

一句话总结

本文揭示了现有机器遗忘(MU)方法大多只是制造了"遗忘的幻觉"——仅在输出层面压制遗忘类别的预测，而特征表征中仍保留高度可区分的信息，并提出基于类均值特征(CMF)分类器的表征级遗忘框架来真正消除遗忘数据的影响。

核心贡献

  1. 揭露"浅层遗忘"现象：通过线性探针和最近类中心(NCC)准确率等特征级指标，证明NegGrad+、SalUn、SCRUB、UNSIR、Random Label等主流MU方法遗忘类别的特征表征仍然高度线性可分。
  2. 提出特征-分类器错位(Feature-Classifier Misalignment)机制：基于Neural Collapse理论，证明现有MU方法主要通过破坏分类器权重与遗忘类均值之间的对齐来实现输出级遗忘，而非真正消除特征空间中的信息。
  3. 提出CMF遗忘框架：设计类均值特征分类器，强制特征与分类器对齐，使遗忘过程必须在表征层面进行，而非仅在输出层面。
  4. 实验证明分类器-only微调即可实现零遗忘准确率：仅在最后一层分类器上进行遗忘操作即可达到与全模型遗忘相当的输出级效果，进一步证明现有方法未触及特征表征。

Methodology

The authors conduct a systematic investigation of class-unlearning in image classification through the lens of Neural Collapse (NC). Their methodology proceeds in three stages:

Diagnostic Framework. The paper introduces three feature-level evaluation metrics complementing standard output-level metrics: (1) linear probing on frozen features, (2) nearest class center (NCC) classification accuracy, and (3) NC3 feature-classifier alignment measure. These metrics are applied to six representative MU methods (NegGrad+, Random Label, SalUn, SCRUB, UNSIR, SVD) across CIFAR-10, CIFAR-100, and Tiny-ImageNet using both ResNet and ViT architectures.

Theoretical Analysis. Under the assumption of neural collapse in the original model, the authors prove (Proposition 1) that NegGrad-based unlearning flips the forget-class classifier weight to be maximally misaligned with the forget class mean while minimally perturbing retain-class weights. This yields zero output-level forget accuracy without modifying the feature extractor at all.

CMF Classifier. The proposed solution reconstructs the classifier head at each epoch by computing centered and normalized class-mean features, then freezes it while the encoder is updated by the underlying MU objective. This ensures that any change in output predictions must be accompanied by corresponding changes in the feature space. The approach adds minimal computational overhead (one forward pass per epoch for mean computation).

Key Results

  • Linear probing recovers near-original accuracy. For example, on CIFAR-10 forgetting 1 class, Random Label achieves 0% output forget accuracy but 92.49% linear-probe forget accuracy. Similar patterns hold across all methods and datasets.
  • Classifier-only fine-tuning matches full-model unlearning. Updating only the last linear layer achieves comparable or better output-level forgetting and retaining accuracy across all MU methods, confirming that features are essentially untouched.
  • CMF-based unlearning substantially reduces feature-level forget accuracy. UNSIR with CMF achieves 12.91% output forget accuracy and 31.16% linear-probe forget accuracy on CIFAR-10 (1 class), compared to 0.48%/85.29% without CMF. The retain accuracy drop is modest.
  • CMF methods outperform the "gold standard" Retain-only Retrain baseline in feature-level forget accuracy, which is notable since that baseline already has transferable features of forgotten classes.

Critical Assessment

Strengths. The paper makes a clean and compelling argument that current output-level evaluation of MU is fundamentally inadequate. The connection to Neural Collapse theory is elegant and provides formal grounding (Proposition 1 and Corollary 1) for the observed feature-classifier misalignment phenomenon. The CMF classifier is a simple, modular intervention that can be plugged into any existing MU method. The experimental scope is thorough, covering multiple methods, datasets, architectures, and forgetting scenarios.

Weaknesses. (1) The scope is limited to class-level forgetting in image classification with known labels. How well these findings transfer to instance-level unlearning, generative model unlearning, or LLM unlearning remains an open question that the authors acknowledge. (2) The CMF approach requires computing class means from the retain set at each epoch, which assumes access to retain data in a labeled form -- a constraint that may not always hold in practice. (3) Even CMF-based methods do not reduce linear-probe forget accuracy to near-zero (e.g., ~31% for UNSIR+CMF on CIFAR-10), suggesting that complete representation-level erasure remains challenging. The inherent transferability of deep features poses a fundamental limit that no classifier-level intervention can fully overcome. (4) The paper does not compare against adversarial privacy attacks (e.g., membership inference) as complementary evaluation, though the authors argue this is not relevant for class forgetting.

Significance. This paper serves as an important reality check for the MU community. By demonstrating that output-level metrics can be trivially gamed through classifier manipulation alone, it raises the bar for evaluating future MU methods. The CMF framework is a practical contribution, though the deeper lesson is methodological: MU evaluation must include representation-level probes to be trustworthy.

相关性与展望

本文对机器遗忘领域的评估范式提出了根本性挑战，指出仅依赖输出级指标（遗忘准确率/保留准确率）无法真实反映遗忘效果。这对隐私合规、版权保护等实际应用具有重要启示——如果被遗忘的数据信息仍潜伏在模型表征中，简单的微调就可能恢复这些信息，使所谓的"遗忘"形同虚设。CMF框架提供了一种相对简洁的解决方案，但深度特征的迁移性决定了表征级完全遗忘仍是一个开放难题。未来在扩散模型和大语言模型中验证类似现象将具有重大价值。
