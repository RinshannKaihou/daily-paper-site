> **一句话总结：** 本文提出 Neural Procedural Memory (NPM)，一个免训练框架，将智能体的程序性记忆从"显式文本指令 (RAG)"转变为激活空间中的隐式 steering vectors，通过双粒度（轨迹间/轨迹内）对比经验蒸馏出引导向量并在推理时注入残差流，在 ALFWorld/WebShop/ScienceWorld/BabyAI 四个基准上达到与显式文本记忆相当的性能，且与显式 workflows 结合时产生互补协同、取得最佳平均分。

> **TL;DR:** The paper proposes Neural Procedural Memory (NPM), a training-free framework that stores agent procedural memory as implicit steering vectors (distilled from dual-granularity inter-/intra-trajectory contrastive experiences) rather than explicit textual instructions, injecting them into the residual stream at inference. On four agent benchmarks, NPM matches explicit-text baselines, outperforms static steering methods (CAA, Mass-Mean), and—combined with explicit workflows—yields the best average scores across MiniCPM3-4B, Qwen3-4B/8B.

## 核心贡献
- **提出 Neural Procedural Memory (NPM)**：将智能体程序性记忆从显式符号指令（RAG 注入文本）转变为隐式的激活空间引导向量，开辟"隐式记忆"这一正交于现有显式记忆的新范式。
- **双粒度对比经验蒸馏**：创新性地同时使用**轨迹间对比**（成功 vs. 失败轨迹）和**轨迹内对比**（同一失败轨迹中的有效步骤 vs. 退化步骤），后者缓解了复杂任务中正样本稀疏、缺少成功示范的冷启动问题。
- **免训练、不占上下文窗口**：仅在残差流注入方向向量（`h̃ = h + α·v`），不更新参数、不扩张上下文，部署开销低。
- **互补协同**：实验证明隐式激活引导与显式 workflows 结合可产生协同效应，在三个骨干模型上均取得最高平均分。
- **可解释性证据**：通过 PCA 投影与行为原语分析发现，引导向量编码了一致的任务逻辑，并在激活空间中形成有组织的结构，不同粒度的向量作用于不同的机制通路。

## Key Contributions
- **Neural Procedural Memory (NPM)**: reframes agent procedural memory from explicit symbolic instructions (RAG) to implicit activation-steering vectors—an orthogonal axis to current explicit-memory designs.
- **Dual-granularity contrastive distillation**: jointly uses **inter-trajectory contrast** (success vs. failure trajectories) and **intra-trajectory contrast** (effective vs. degenerate steps within a single failed trajectory); the latter mitigates the cold-start caused by sparse positive samples in complex tasks.
- **Training-free, context-window-free**: intervention is purely a residual-stream addition (`h̃ = h + α·v`)—no parameter updates, no context expansion, low deployment cost.
- **Complementary synergy**: hybrid (NPM + explicit workflows) achieves the best average scores across all three backbone models, evidencing that implicit and explicit memory operate as a complementary system.
- **Interpretability**: PCA projections and "behavioral primitive" analyses show steering vectors encode consistent task logic and form organized clusters; different granularities act via distinct mechanistic pathways.

## Methodology
NPM is a **training-free** framework operating directly in the model's activation space, consisting of three phases:

**1. Contrastive Experience Construction (§3.1).** Given a task, the agent generates trajectories τ = {s₁,a₁,…,s_K,a_K}. NPM builds a contrastive dataset D of paired reasoning segments (x⁺, x⁻) at two granularities:
- *Inter-trajectory contrast*: pairs a successful trajectory τ⁺∈T⁺ with a failed one τ⁻∈T⁻ to capture a global behavioral shift.
- *Intra-trajectory contrast*: when successful trajectories are sparse, splits a single failed trajectory into an **Effective step set** S_eff (steps advancing state) vs. a **Degenerate step set** S_deg, where degenerate steps are detected by *redundancy* (consecutive repeated commands / loops) or *invalidity* (environment error feedback).

**2. Steering Vector Extraction (§3.2).** Maps textual contrasts into a geometric transformation in representation space. A representation function ϕ_l(·) extracts hidden states at layer l: for inter-trajectory it takes the last-token hidden state of the full trajectory; for intra-trajectory it mean-pools tokens within each step. For each historical task Q_j these are stored as a task-specific contrastive pair set C_l^(j).

**3. Inference-Time Intervention (§3.3).** A Retrieval–Synthesis–Intervention pipeline:
- *Contextual Retrieval*: a dense retriever finds the top-K most similar historical tasks R_q.
- *Dynamic Vector Synthesis*: aggregates the stored pairs into a memory pool M_l(q) and derives a task-specific consensus direction v_l(q) = ψ(M_l(q)), dynamically selecting inter- vs. intra-trajectory contrast per task context.
- *Activation Steering*: during generation, injects the vector into the residual stream at layer l: **h̃_{l,t} = h_{l,t} + α·v_l(q)**, where α controls intervention strength. This nudges the agent away from known failure modes without expanding the context or updating weights.

The key conceptual difference from prior activation steering (CAA, Mass-Mean, which compute fixed global mean differences) is that **NPM dynamically synthesizes a task-specific vector from retrieved, contextually relevant contrastive experiences**.

## Results & Highlights
Evaluated on **ALFWorld, WebShop, ScienceWorld, BabyAI** using **MiniCPM3-4B, Qwen3-4B, Qwen3-8B** (success rate for ALFWorld, average reward for the rest).

- **Implicit steering is effective**: NPM lifts average score, e.g. MiniCPM3-4B 22.60 → 28.87; Qwen3-8B 30.63 → 36.32.
- **Beats static steering baselines**: NPM outperforms CAA and Mass-Mean (which often *degrade* performance, e.g. CAA drops MiniCPM3-4B to 18.58) because they rely on dataset-wide fixed mean shifts that cannot capture multi-step procedural skill.
- **Competitive with explicit memory**: NPM beats the *Insights* baseline on average and is competitive with the more structured *Workflows* baseline; on WebShop/Qwen3-4B it even exceeds Workflows (48.00 vs 45.73), demonstrating the "text-action disconnect" cost of explicit instructions.
- **Hybrid is best**: NPM + Workflows gives top average scores on all three backbones (34.47 / 37.60 / 41.89). On Qwen3-8B it reaches **66.42% on ALFWorld** and **31.89 on ScienceWorld**. This is the paper's headline synergistic finding: explicit text supplies symbolic planning, implicit steering enforces adherence during long horizons.
- **Dual-granularity matters**: intra-trajectory steering helps in step-intensive tasks (ALFWorld, local action errors); inter-trajectory steering helps where macro-level planning is needed (WebShop). Dynamically choosing granularity reduces variance.
- **Representational findings (§5)**: PCA projections show positive/negative representations form distinct, interpretable clusters per task type. Behavioral-primitive analyses confirm steering vectors act through distinct mechanistic pathways (inter = global planning shifts; intra = local error correction). A retrieval-size study shows task-specific primitives peak at a moderate retrieval scale and decline with over-aggregation—justifying dynamic synthesis over static global steering.

## Critical Assessment
- **Novelty — High.** Applying activation steering as a *persistent, retrieval-augmented procedural memory* substrate for agents (rather than for coarse single-turn attributes like honesty/safety) is a genuinely fresh framing that departs from mainstream RAG-based agent memory. The dual-granularity contrastive design and the "dynamic task-specific synthesis vs. static global steering" distinction are non-trivial contributions.
- **Rigor — Moderate-to-Good.** Four benchmarks × three models × multiple baselines (None / Insights / Workflows / CAA / Mass-Mean / NPM / Hybrid) is a reasonable matrix. Results are consistent in direction. However: (a) gains are modest in absolute terms on several cells (e.g., ScienceWorld/BabyAI columns are volatile, and CAA/Mass-Mean sometimes help and sometimes hurt, indicating high run variance that is not discussed via error bars or significance tests); (b) the strongest claim (Hybrid best) is somewhat expected since NPM is *added on top of* the best explicit baseline rather than compared in isolation.
- **Reproducibility — Moderate.** Training-free methods are inherently reproducible; intervention formula and contrastive construction are specified precisely. Latency/storage analyses are promised in appendices. Unclear whether reference code/seeded checkpoints will be released; dense-retriever details and α tuning protocol are not fully pinned in the main text.
- **Significance — Moderate-to-High.** If the complementarity and interpretability claims hold broadly, NPM offers a new, cheap, orthogonal axis (implicit memory) to compose with existing explicit-memory agent stacks—a practical and conceptually valuable result.
- **Red flags:**
  - NPM frequently does **not clearly beat** the strong *Workflows* baseline in isolation (e.g., MiniCPM3-4B: NPM 28.87 vs Workflows 32.68); its headline advantage is largely the *additive* Hybrid setting. The "comparable to explicit memory" framing is honest but means standalone superiority is limited.
  - **Architecture lock-in**: requires open, fully-accessible residual-stream access, excluding closed models (GPT/Claude).
  - **Cold-start**: building the contrastive repository needs *some* successful trajectories, which is hard in very complex environments.
  - Heuristic-based degenerate-step detection misses implicit logical fallacies that produce no explicit environment error.
  - The synthesized vector is a **single, constant** modulation across all generation steps—static, not stage-adaptive.

## Strengths
- Conceptually novel: implicit procedural memory via activation steering, distinct from RAG and from parametric fine-tuning.
- Training-free, parameter-free, context-window-free—efficient and easy to deploy on open models.
- Dual-granularity contrast cleverly recovers signal from failure-only data (intra-trajectory), addressing positive-sample scarcity.
- Strong interpretability story: PCA clustering and behavioral-primitive decomposition provide a transparent account of what the steering vectors encode.
- Composability: hybrid (implicit + explicit) consistently best, showing it slots into existing agent stacks rather than replacing them.

## Limitations
- Requires white-box access to the residual stream; not applicable to closed proprietary LLMs.
- Cold-start dependency on occasionally obtaining successful trajectories to seed the repository.
- Degenerate-step detection relies on surface heuristics (redundancy, invalidity) and misses subtle logical errors without explicit environmental feedback.
- Static modulation: a single steering vector is applied constantly across the whole generation; it cannot shift behavioral primitives across different execution stages.
- Modest standalone gains over the strongest explicit baseline (Workflows); headline advantage is mainly additive in the Hybrid setting.
- Reported numbers appear to be single-run; variance/error bars and significance are not presented despite noisy cross-environment results.

## Relevance
本文与智能体记忆、表征工程/激活引导、以及免训练的可控性方向高度相关，为 LLM agent 的持久化程序性记忆提供了"隐式表征"这一正交于 RAG 的新机制，对研究 agent 长程执行的鲁棒性、记忆压缩与白盒引导均有直接参考价值。

## Questions for Further Exploration
- How do NPM steering vectors **transfer across model families and scales**? Are distilled vectors portable, or architecture-locked?
- Does **Hybrid complementarity hold on more complex, real-world agentic tasks** (tool use, code agents, multi-modal embodiment) beyond text simulators?
- Can the contrastive repository be built **online/incrementally** as the agent accumulates experience, mitigating cold-start?
- Could **stage-adaptive / time-varying** steering (different vectors at planning vs. execution steps) beat the single static vector?
- How does NPM interact with **instruction-hierarchy / safety constraints**—could injected vectors override user-level instructions or be exploited as an adversarial channel?
- What is the sensitivity to layer choice *l* and intervention strength *α*, and is there an automatic calibration procedure?
