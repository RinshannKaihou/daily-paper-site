01 Methodology — distilled

NPM is a training-free framework that turns agent experience into geometric structure in the model's activation space, then re-injects that structure at inference. Memory stops being text and becomes a direction in the residual stream (h̃ = h + α·v) — no parameter updates, no context growth.

  1. 1Contrastive experience Construction. Build paired reasoning segments at two granularities: inter-trajectory (success vs. failure) captures global behavioral shifts; intra-trajectory (effective vs. degenerate steps inside a single failed run) rescues signal when success trajectories are scarce — a cold-start cure.
  2. 2Steering-Vector Extraction. Map textual contrasts into hidden states via a representation function at layer l, storing per-task contrastive pair sets.
  3. 3Inference-Time Intervention. Retrieve top-K similar tasks → dynamically synthesize a task-specific consensus vector → inject into the residual stream. The contrast with CAA / Mass-Mean: NPM is context-retrieved and task-specific, not a fixed global mean shift.

02 Strengths & Limitations

▲ Strengths

  • Conceptually novel — implicit procedural memory via steering, distinct from RAG and fine-tuning.
  • Training-, parameter-, and context-window-free; easy to deploy on open models.
  • Dual-granularity contrast recovers signal from failure-only data.
  • Strong interpretability — PCA clusters & behavioral-primitive decomposition.
  • Composable — hybrid (implicit + explicit) consistently best.

▽ Limitations

  • Needs white-box residual-stream access — closed LLMs (GPT/Claude) excluded.
  • Cold-start: still requires some successful trajectories to seed the repository.
  • Degenerate-step detection uses surface heuristics; misses silent logical errors.
  • Static modulation — one vector applied across the whole generation.
  • Single-run numbers; no variance / error bars despite noisy cells.

04 Open questions

  • Do distilled vectors transfer across model families / scales, or are they architecture-locked?
  • Does hybrid complementarity hold on real-world agentic tasks (tools, code, multimodal) beyond text simulators?
  • Can the repository be built online / incrementally to kill cold-start?
  • Would stage-adaptive steering (plan vs. execute vectors) beat the single static vector?
  • Sensitivity to layer l and strength α — is there an automatic calibration procedure?
