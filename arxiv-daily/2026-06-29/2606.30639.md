> **一句话总结：** 本文提出 WorldEvolver，一个免训练的自进化世界模型框架——在冻结下游智能体与所有模型参数的前提下，通过 Episodic Memory（检索真实动作转移）、Semantic Memory（从预测-观测不一致中抽取启发式规则）和 Selective Foresight（按置信度过滤低质量预测）三模块在部署时动态修正世界模型的上下文证据；在 ALFWorld 与 ScienceWorld 上，于三个骨干模型取得最高预测准确率，并在大多数设置下的下游智能体成功率上领先其他世界模型基线。

> **TL;DR:** WorldEvolver is a training-free, self-evolving world-model framework that revises a frozen world model's deployment-time *context* (not weights) via Episodic Memory (retrieval over realized transitions), Semantic Memory (heuristic rules mined from prediction–observation mismatches), and Selective Foresight (confidence-gated abstention). On ALFWorld and ScienceWorld it achieves the best next-observation prediction accuracy across three backbones and is the strongest world-model method for downstream agent success rate, with a controlled oracle study showing noisy foresight actively hurts agents while reliable foresight helps.

## 核心贡献
- **首次提出针对世界模型（而非智能体策略）的自进化范式**：在环境交互过程中冻结智能体 πθ 与世界模型 Wθ 的所有参数，仅通过非参数化外部记忆 Mₜ=(ME, MS) 在推理时持续修正世界模型的上下文证据，规避了在线梯度更新在大模型规模下计算昂贵、易过编辑/灾难性遗忘的问题。
- **三类互补的记忆/过滤机制**：① Episodic Memory（情景记忆，利用真实转移 (o,a,o′) 做基于动作-token Jaccard 检索的"剥削"式模拟）；② Semantic Memory（语义记忆，把预测-观测不一致经因子化元组比较后用 LLM critic 转写为带证据得分的持久启发式规则，作为"探索"分支）；③ Selective Foresight（选择性预见，用预测序列的几何平均 token 概率作置信度，低于阈值 τ 时不向智能体注入预见，构成弃权机制）。
- **配套的预测-动作对齐闭环**：当受预见影响的执行动作与草案动作不同时，世界模型用执行动作重新查询一次，保证语义记忆基于"真正执行的动作"对应的预测不一致来更新，学习信号与实际行为对齐。
- **全面的实证验证**：在 Word2World（预测准确率）与 AgentBoard（规划成功率）上，跨三个骨干（Gemma-4-26B-A4B、Qwen3.5-9B、Gemma-4-31B；规划还含 GPT-5.4-mini）、两种智能体范式（ReAct、ReflAct）对比 Zero-Shot、RAWM-φ、ITP-I，并提供一个揭示"噪声预见有害、完美预见有益"的受控 oracle 先导实验。

## Key Contributions
- **First self-evolution of the *world model* (not the agent policy):** WorldEvolver keeps the agent πθ and the world model Wθ fully frozen and instead evolves a non-parametric memory store Mₜ=(ME, MS) that supplies the frozen world model with deployment-time evidence — an auditable, parameter-free alternative to online gradient updates that are costly at LLM scale and prone to over-editing/catastrophic forgetting.
- **Three complementary mechanisms:** (i) **Episodic Memory** — exploits realized transitions (o,a,o′) via action-token Jaccard retrieval to ground next-state prediction; (ii) **Semantic Memory** — explores by turning prediction–observation mismatches into persistent textual heuristic rules with evidence scores, after factoring observations into object/relation/action tuples compared by an LLM critic; (iii) **Selective Foresight** — abstention via the geometric-mean token probability of the prediction, injecting foresight into the agent only when confidence qₜ ≥ τ.
- **Prediction–action alignment loop:** if the executed action differs from the draft action that conditioned the initial prediction, the world model is re-queried with the executed action so Semantic Memory is updated from the mismatch that actually matters for the realized behavior.
- **Comprehensive evaluation:** prediction accuracy on Word2World and planning success on AgentBoard across three backbones (Gemma-4-26B-A4B, Qwen3.5-9B, Gemma-4-31B; plus GPT-5.4-mini for planning), two agent styles (ReAct, ReflAct), vs. Zero-Shot / RAWM-φ / ITP-I, plus a controlled **oracle study** showing noisy foresight *degrades* action accuracy (e.g., 0.493→0.385 on ALFWorld) while perfect foresight improves it — motivating confidence-gated abstention.

## Methodology
**Problem formulation.** Each task is a partially observed interaction (S, A, O, T). The agent sees a textual state sₜ=(o₁,a₁,…,oₜ) and samples aₜ∼πθ(·|sₜ); a frozen world model Wθ predicts a K-step future (mainly one-step ahead, since ôₜ₊₁ directly conditions the action and oₜ₊₁ gives immediate supervision). The goal is to improve ôₜ₊₁ at deployment time while keeping πθ and Wθ frozen.

**Episodic Memory (exploitation).** Stores realized transitions ME={(oᵢ,aᵢ,oᵢ₊₁)}. Given a candidate action aₜ and retrieval size kME, it retrieves the kME most similar transitions by Jaccard over the open-vocabulary action-token set and renders them as raw text (previous obs, action, next obs). Records are appended only after execution, so retrieval never leaks future information.

**Semantic Memory (exploration).** Rather than treating a mismatch as a weight failure, it is read as feedback on the contextual memory. A mapping function g factorizes observation text into tuples, e.g. "The fridge 1 is open. In the fridge 1, you see an apple 1." → ('fridge 1','is','open') and ('apple 1','in','fridge 1'). Comparing ẑₜ₊₁=g(ôₜ₊₁) vs zₜ₊₁=g(oₜ₊₁) decides whether a real mismatch occurred; an LLM critic then converts failures into candidate rules rᵢ with evidence scores eᵢ (init 1, updated ±1/|MS| per supporting/contradicting subsequent observation). Only rules with eᵢ>0 enter the context. A batch variant accumulates kMS mismatches before revising the rule set (batch "semantic-gradient" updates).

**Selective Foresight (abstention).** From the predicted token sequence y₁:ₙ, computes mean token log-probability ℓₜ and converts it to qₜ=exp(ℓₜ)∈(0,1] (geometric-mean token probability). The agent-visible foresight is Fₜ=ôₜ₊₁ if qₜ≥τ, else ∅ — so untrustworthy simulations never reach the agent.

**Closed loop (Algorithm 1).** (1) sample draft action aₜ⁽⁰⁾; (2) retrieve episodic transitions; (3) query Wθ for (ôₜ₊₁, qₜ); (4) gate foresight by τ; (5–6) sample executed action aₜ⁽¹⁾∼πθ(·|sₜ,Fₜ); (7) **re-query Wθ with aₜ⁽¹⁾ if it differs from aₜ⁽⁰⁾** so the learning signal aligns with the executed action; (8–9) execute, append realized transition to ME; (10) accumulate mismatch-derived rules into MS; (11) return.

## Results & Highlights
- **Oracle study (Fig. 2, Gemma-4-26B-A4B):** noisy foresight *lowers* action accuracy vs. no foresight (ALFWorld 0.493→0.385; ScienceWorld 0.385→0.344), while perfect foresight raises it (→0.271/0.281 are baseline figures showing the spread) — justifying confidence-gated abstention.
- **World-model prediction (Table 1, Word2World):** WorldEvolver is best on every Exact-Match cell, e.g. Gemma-4-26B-A4B ALFWorld **52.88** EM vs RAWM-φ 20.06 / Zero-Shot 3.60; Gemma-4-31B ScienceWorld **62.03** EM vs RAWM-φ 32.84. Ablations show Episodic Memory is the dominant driver (w/o MS still ~47–57 EM), while Semantic Memory adds the largest gains on ScienceWorld for the Gemma models.
- **Agent planning (Table 2, success rate):** WorldEvolver (w/ Fₜ) is the **strongest world-model method across all 8 settings** and beats the no-world-model baseline in all 4 Gemma-4-26B-A4B cells (avg +2.24 SR on ALFWorld, +3.33 on ScienceWorld over the no-foresight variant). Selective foresight improves or ties the no-foresight variant everywhere. Notably, **RAWM-φ and ITP-I frequently underperform the no-world-model baseline**, confirming that misaligned foresight hurts planning — and that WorldEvolver's filtering is what rescues the benefit.
- **GPT-5.4-mini:** gains are more mixed — w/ Fₜ beats the no-world-model baseline in only 2 of 4 cells (+1.50 ReAct/ALFWorld, +3.33 ReflAct/ScienceWorld; −2.99 ReflAct/ALFWorld, −2.23 ReAct/ScienceWorld), because the stronger planner leaves less room for foresight.
- **Ablations (Fig. 5):** episodic retrieval size is the dominant factor — raising kME from 1→5 improves Exact Match by 16.8/23.5 pts (Gemma-4-26B-A4B on ALFWorld/ScienceWorld). Semantic batch size kMS is largely insensitive (≤2 pts), so kMS=1 is adopted.
- **Online continual learning (Fig. 4):** the cumulative best-of-L curve widens over trials, especially on ScienceWorld with Gemma-4-26B-A4B, showing memory accumulation helps most when the agent can iteratively replan; gains shrink for the already-strong GPT-5.4-mini.
- Baselines ITP-I *consistently underperforms* Zero-Shot on prediction (over-generation of imagined detail), and RAWM-φ retrieves from full state-action text where long repetitive descriptions dilute the action signal — whereas Episodic Memory matches transitions more directly.

## Critical Assessment
- **Novelty:** Strong and clearly positioned. The distinguishing claim is self-evolution of the *world model's context* (not the agent, not the weights), which sits between frozen world models, offline-tuned world models (RAWM-φ), and weight-updating self-evolving world models (cited R-Zero/DynaWeb/WebEvolver line). The three modules are individually informed by classical episodic/semantic memory (Tulving 1972) and model-based-RL trust-region ideas (Janner 2019), but their integration into a training-free, frozen-agent world-model revisor is a fresh, well-motivated combination.
- **Rigor:** Good. Multi-backbone, multi-agent, multi-environment evaluation; ablations isolate each module; the oracle study and the "baselines can be worse than no-world-model" finding lend internal credibility. Best-of-L aggregation (L=5) is standard but worth noting it favors methods that improve *some* trial.
- **Reproducibility:** Reasonable — public benchmarks, named models, stated hyperparameters (kME=5, kMS=1), and an explicit algorithm (Algorithm 1). Code release is not clearly promised in the read sections; the Semantic-Memory LLM critic and prompt templates (Appendices B/D, not fully read) are likely the main reproducibility risk.
- **Significance:** Practically meaningful: a parameter-free, drop-in upgrade that makes world-model foresight reliably helpful for closed-loop agents — valuable where retraining is impractical. The "noisy foresight hurts, so gate it" message is a useful corrective for the field.
- **Red flags:** None egregious. Minor concerns: (a) some headlining gains concentrate on the weaker Gemma backbone and a single environment (ScienceWorld); on the strongest planner (GPT-5.4-mini) the w/ Fₜ variant actually *loses* to no-world-model in 2/4 cells, tempering universal claims. (b) Selective Foresight's confidence = token log-probability assumes probability correlates with accuracy; the authors flag this themselves. (c) Single-step foresight only; multi-step rollouts untested. (d) Two text environments limit breadth.

## Strengths
- Genuinely training-free and frozen on both agent and world-model weights — lightweight and broadly applicable.
- Clean, interpretable separation of instance-level (episodic) vs. rule-level (semantic) knowledge, plus an explicit abstention gate; all memory is auditable prompt-level evidence.
- Honest, informative baselines: showing that strong world-model methods (RAWM-φ, ITP-I) can *harm* planning makes the case for filtering, not just for better prediction.
- Consistent prediction-accuracy wins across three backbones and strong planning gains on the weaker backbone.
- Insightful analysis (oracle study, hyperparameter sensitivity, cumulative-replanning curves) that explains *why* it works.

## Limitations
- **Scope:** Only two controlled text environments (ALFWorld, ScienceWorld); no web navigation, code, robotics, or multimodal agents (authors acknowledge this).
- **Single-step foresight** only; the framework's behavior under multi-step rollouts is untested.
- **Confidence signal depends on token probabilities**, unavailable from some closed APIs; correlation between qₜ and accuracy can vary across models/environments.
- **Benefit shrinks with stronger planners** (GPT-5.4-mini gains are mixed, with two net-negative cells for w/ Fₜ), so the value proposition is largest for weaker/cost-constrained backbones.
- **Semantic Memory sensitivity** is low but its net contribution over Episodic Memory alone is sometimes marginal (e.g., Qwen3.5-9B ScienceWorld EM 28.92→29.82).
- Best-of-L evaluation inflates headline success rates vs. single-trial deployment.

## Relevance
对研究 LLM 智能体长程规划、世界模型与测试时自适应的工作高度相关：WorldEvolver 提供了一种免训练、仅改上下文记忆的自进化范式，可直接作为记忆增强智能体与世界模型基线的对比点，其"置信度过滤预见"与"预测-动作对齐"机制也启发 agent 自我改进与 test-time scaling 的后续研究。

## Questions for Further Exploration
- How does each module contribute *to planning* (Table 2 only reports full / w-o Fₜ, not w-o ME or w-o MS) — is Semantic Memory ever worth its added complexity for downstream success?
- Does the Selective Foresight threshold τ need per-model/per-environment tuning, and can a learned or self-consistency-based confidence replace token log-prob?
- How does WorldEvolver behave under genuine non-stationarity (environment dynamics changing mid-deployment) vs. the mostly-static ALFWorld/ScienceWorld?
- Can the framework extend to multi-step rollouts and to multimodal / tool-use / web agents where observation factorization is harder?
- Would episodic retrieval with dense (semantic-embedding) similarity beat Jaccard on action tokens for environments with high action vocabulary drift?
