M Methodology

How a frozen world model gets smarter at test time — three non-parametric levers + an alignment loop.

The agent πθ and world model Wθ stay fully frozen. Each step, Wθ predicts the next observation ôt+1; WorldEvolver revises only the contextual evidence fed to that prediction — never the weights — sidestepping the cost and catastrophic-forgetting risk of online gradient updates at LLM scale.

ME
ExploitEpisodic Memory. Stores realized transitions (o,a,o′); retrieves the kME nearest by action-token Jaccard to ground next-state prediction. Records append only after execution, so retrieval never leaks the future.
MS
ExploreSemantic Memory. Factorizes a prediction–observation mismatch into object/relation/action tuples, then an LLM critic transcribes failures into persistent heuristic rules carrying evidence scores. Only rules with ei > 0 enter context.
Ft
AbstainSelective Foresight. Confidence qt = geometric-mean token probability of the prediction. Foresight reaches the agent only when qt ≥ τ; untrustworthy simulations are dropped.
↻
AlignPrediction–action loop. If the executed action differs from the draft, Wθ is re-queried with the executed action, so Semantic Memory learns from the mismatch that actually mattered.

+ Strengths

  • Genuinely training-free & frozen on both agent and world-model weights — lightweight, broadly applicable.
  • Clean, interpretable separation of instance-level (episodic) vs. rule-level (semantic) knowledge, plus an explicit abstention gate; all memory is auditable prompt-level evidence.
  • Honest, informative baselines — showing strong methods (RAWM-φ, ITP-I) can harm planning makes the case for filtering, not just better prediction.
  • Consistent prediction wins across three backbones, with strong planning gains on the weaker backbone.
  • Insightful analysis (oracle study, hyperparameter sensitivity, cumulative-replanning curves) explaining why it works.

− Limitations

  • Only two text environments (ALFWorld, ScienceWorld) — no web navigation, code, robotics, or multimodal agents.
  • Single-step foresight only; behavior under multi-step rollouts is untested.
  • Confidence = token log-prob, unavailable from some closed APIs; correlation with accuracy can vary.
  • Benefit shrinks with stronger planners (GPT-5.4-mini gains mixed, two net-negative cells).
  • Semantic Memory net contribution over Episodic alone is sometimes marginal (e.g. Qwen3.5-9B SW EM 28.92→29.82).
  • Best-of-L evaluation inflates headline success rates vs. single-trial deployment.

? Questions for Further Exploration

Open threads the paper leaves — where the argument is weakest or the work could extend.

  • Module-level planning effect: Table 2 reports full / w-o Ft but not w-o ME or w-o MS — is Semantic Memory ever worth its complexity for downstream success?
  • Threshold sensitivity: Does τ need per-model/per-environment tuning? Could a learned or self-consistency-based confidence replace token log-prob?
  • Non-stationarity: How does it behave when environment dynamics change mid-deployment vs. the mostly-static ALFWorld/ScienceWorld?
  • Breadth: Can it extend to multi-step rollouts and multimodal / tool-use / web agents where observation factorization is harder?
  • Retrieval similarity: Would dense semantic-embedding similarity beat Jaccard on action tokens under high action-vocabulary drift?
