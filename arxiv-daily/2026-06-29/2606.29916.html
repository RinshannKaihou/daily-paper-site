EVAF: A Test-Retest Protocol for Selective Parametric Consolidation in Large Language Models · Haoliang Han· China Pharmaceutical University · 2026-06-29

The Key Finding

A valence×surprise gate couples a fast spherical working-memory prefix to a slow LoRA store — a falsifiable Test-Retest protocol certifies the writes as genuine parametric consolidation.

一句话总结：EVAF（Echo-Valence Attractor Field）由 valence×surprise 双门控驱动、将 LoRA 作为慢存储、把前缀隐状态约束在高维球面上，实现"选择性参数化巩固"，并配以可证伪的 Test-Retest 协议。

01

Methodology

Two stores on different timescales, joined by a dual gate and a paired surprise test.

Two coupled stores. A trainable prefix s(t) is hard-projected each step onto a per-token sphere of radius √D (thin-shell concentration) — the fast working memory. Its update mixes a surprise gradient with a repulsive contrastive term away from a slow-gated EMA, which sustains a pre-consolidation limit cycle; flipping the sign to attraction kills both the cycle and the freeze signature (EMA-sign control).

A rank-32 LoRA on attention q/k/v/o is the slow store. The dual write gate P_write = σ(k_v·(V−τ_v))·σ(k_s·(S−τ_s)) admits a sample only when both valence (external oracle) and epistemic surprise are high. A capacity-4 buffer then fires a 5-step AdamW inner loop with experience replay (K=4) and an EWC-style L2 anchor, prefix held fixed so the stores decouple.

Test-Retest protocol. Each event measures ΔS_buf / ΔS_rep / ΔS_global and a step-angle dynamical-freeze z-score (limit cycle → fixed point). Null-optimizer counterfactuals peg a verifiable zero at +0.000.

System 1 · fast working memory

Spherical prefix, per-token product manifold. Repulsive contrastive inertia; rate slides fast when surprise high, near-frozen when low. Pre-consolidation limit cycle ≈ 17.7°/step (ρ₁₀ = −0.87).

System 2 · slow store

Rank-32 LoRA, attention q/k/v/o. 5-step AdamW on CE(B∪R) + λ‖θ−θ₀‖², B=4 admitted + R=4 replay. Prefix frozen during the inner loop.

Dual gate · §3.3

P_write = σ(k_v(V−τ_v))·σ(k_s(S−τ_s)); admit iff P > 0.55. Product form rejects single-factor samples.

03

Strengths & Limitations

A thoughtfully-hedged protocol — inventive, but preliminary as a practical advance.

Strengths

  • Novel combination of spherical attractor prefix + dual V×S gate + falsifiable harness, clearly delineated from prior CLS / replay / EWC / Larimar work.
  • Exceptional self-criticism: ablations designed to break its own metrics (C1 noise-fitting trap), null-optimizer zero-points, held-out ΔS_H separating write from rote-fit.
  • Honest reframe of V9 as "selectivity & stability" once multi-seed data forced a correction of an earlier 2-seed pilot.
  • Cross-architecture reproduction (4 attention families, ×57 scale) of all qualitative signatures.
  • Clean concept: memory access vs. consolidation depth as distinct operations, with a retrieval / EVAF orthogonality argument.

Limitations

  • All experiments on controlled / synthetic corpora; no open-domain agent conversations or natural noisy text.
  • Statistical power limited (n=4–5); headline productive-recall gain non-significant with huge per-seed variance.
  • Scaling rules are mostly fitted / conjectural (§5.3, §5.4), not predictive for unseen models.
  • One designed persona task family; no external benchmarks (LongMemEval, LoCoMo, MemoryArena) — explicitly deferred.
  • Oracle-conditioned valence (heuristic or DistilBERT-SST2); no autonomous salience mechanism.
  • Single author, no peer review, non-ML affiliation; "dynamical-freeze" is operational, not a proven phase transition.
04

Open Questions

Where the protocol is weakest — and what would move it from inventive to validated.
Does EVAF survive open-domain noisy conversation, where a fatter surprise tail could explode firing rate and break the λ_reg rule?
Can a 5th–7th model (Llama-3-8B / Qwen3-7B) discriminate the §5.3 capacity-exponent form and the §5.4 lrLoRA conjecture?
Would training-time prefix evaluation (not end-of-stream) reveal V9's selectivity edge that the 600-turn test missed?
Has an adversarial-filler stream (high-S, low-V decoys) been run to test whether the dual gate beats naive every-step LoRA?
Is "parametric = typology, retrieval = verbatim" universal, or an artifact of 124M–1.1B models (R2 token floor was 0/50)?
Can the valence oracle be replaced by a learned salience head to make EVAF deployable autonomously — and does bit-for-bit v9.0 reproduction hold outside the author's environment?
