LoRA-Muon: Spectral Steepest Descent on the Low-Rank Manifold

Franz Louis Cesista, Katherine Crowson, Cédric Simal, Stella Biderman

2026-06-11 · arXiv 2606.12921 · EleutherAI / Ateneo de Manila / UNamur

一句话总结：LoRA-Muon 将 Muon 优化器的谱最陡下降规则推广到低秩流形，通过投影器形式的因子更新、分裂权重衰减与规范不变性，实现了跨秩、宽度、深度和因子缩放的学习率迁移，并在 TinyShakespeare 上以 rank-32 超过了稠密基线。
Key Finding
Gauge-invariant spectral steepest-descent on the low-rank manifold lets a rank-2 proxy recover dense Muon's optimal learning rate — and that rate transfers across rank, width, depth, and rescaling.
Key Metric
1.776 val loss ± 0.002  vs.  1.789 ± 0.002 dense baseline
Why This Matters
Ultra-cheap hyperparameter search: tune at rank 2, deploy at rank 32+ with the same learning rate — no costly full-rank sweep needed.
Critical Edge
All evidence is from TinyShakespeare. No validation on 7B+ LLMs, vision, or multimodal models. Scale-up is an open question.

Methodology

LoRA-Muon reframes LoRA fine-tuning as optimization on the rank-r matrix manifold Mr, rather than treating factors A and B as independent coordinates.

Trust-region → projector-form update

The tangent space TWMr = {ΔA BT + A ΔBT} is split evenly. For spectral norm, the Linear Minimization Oracle is the matrix sign function:

ΔA* BT = (η/2) msign(Gt PB)
A ΔB*T = (η/2) msign(PA Gt)

Split weight decay

With s = √(1 − λη), update both factors so decay acts on the composed weight W = ABT, not on A and B independently:

At+1 = s·At + s−1·ΔAt
Bt+1 = s·Bt + s−1·ΔBt

Gauge invariance

Any invertible rescaling (A, B) → (AR, BR−T) leaves the induced update on W unchanged. This property uniquely distinguishes LoRA-Muon from Spectron, which fails under scalar gauge rescaling.

Results

Across rank 2/4/8/16/32, LoRA-Muon consistently selects lr = 0.1 — the same optimum as dense Muon. Only rank-1 fails.

Efficiency

Rank-2 proxy uses only ~4.3% trainable params while supporting 9.2× more optimisation steps within the same compute budget (1M tokens).

Strengths

  • +Clear geometric framing — lifts LoRA optimisation from heuristic coordinate descent to manifold steepest descent
  • +Gauge invariance is a strong, falsifiable theoretical property that cleanly separates LoRA-Muon from Spectron
  • +Learning-rate transfer across rank is practically valuable for cheap hyperparameter search
  • +No QR factorization or second-moment storage — implementation-friendly
  • +Multi-axis sweep design (rank, width, depth, gauge) tests the core claim thoroughly

Limitations

  • −TinyShakespeare only — tiny model, single-task; no 7B+ or vision evidence
  • −Split weight-decay proof is asymptotic; no finite-step quantitative error bound
  • −No comparison with AdamW-8bit, Adafactor, GaLore, or other memory-efficient baselines
  • −Code not confirmed open-source; reproducibility relies on appendix details
  • −Only learning-rate transfer validated — other hyperparameters untested

Open Questions

  1. How does LoRA-Muon scale to 7B+ language models and vision-language models? Does lr transfer hold at scale?
  2. Can the split weight-decay rule be derived from a variational objective with finite-step error bounds?
  3. Wall-clock and memory comparison vs. AdamW-8bit, Adafactor, GaLore?
  4. Does gauge invariance survive QLoRA (quantized base models)?
  5. Sensitivity to momentum β, warmup, and layer-wise lr scaling at scale?
  6. Can the projector-form LMO extend to nuclear norm or Ky Fan norms for different inductive biases?
