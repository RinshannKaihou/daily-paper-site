arXiv 2606.12879 · Network Alignment

Diffusion-Network Alignment: An Efficient Algorithm and Explicit Probability Bounds

Ziao Wang · Lei Ying — University of Michigan, Ann Arbor
Polynomial-time two-pass algorithm aligns diffusion trees to correlated networks with explicit depth-dependent per-vertex correct-matching probability lower bounds and a global no-false-positive guarantee — the first algorithmic result bridging independent-cascade observation asymmetry with sparse graph matching theory.
Key Metric
√q·s² > α ≈ 0.338
Otter's constant sets the computational threshold. Root success lower bound ≈ 0.4125 vs. information-theoretic optimum ≈ 0.7149.
n2+o(1) time · O(n−ε/3) error floor
Why This Matters
First rigorous algorithm for aligning cascade-observed diffusion trees to partially correlated networks
Enables contact tracing, rumor source identification, and blockchain deanonymization where only one side of the graph is observable through cascades.
Critical Edge
No empirical validation · Recall unquantified
Purely theoretical. Assumes sparse ER graphs, known IC parameters (λ, s, q), and supercritical cascades (λsq > 1). Algorithm may leave many vertices unmatched.

Algorithm — Two-Pass Propagation

The algorithm operates on a rooted diffusion tree TIC generated by an Independent Cascade process on correlated Erdős-Rényi graph G₁, and a permuted observation G̃₂. It recovers the vertex correspondence π* using local tree correlation tests propagated in two passes.

Matching Criteria

Criterion 1 · Root bootstrap
Three children each have a dangling neighbor in G̃₂ whose BFS subtree passes the correlation test.
Criterion 2 · Mixed evidence
One child already matched + a second child's subtree passes correlation test with a neighbor of ṽ.
Criterion 3 · Parental chain
Two children already matched to two distinct neighbors of ṽ in G̃₂.
Criterion 4 · Downward pass
Parent matched to a neighbor of ṽ, and own subtree passes correlation test with ṽ.

Propagation order

↑ Upward pass

  • Starts at leaves (depth dmax), moves toward root
  • Applies Criteria 1–3
  • Matches propagate upward as strong evidence

↓ Downward pass

  • Starts at depth 1, moves toward leaves
  • Applies Criterion 4
  • Parental matches help align hard-to-match children

Main Result (Theorem 1)

P(Mv = {π*(v)} | v ∈ Vd) ≥ pd' − O(n−ε/3)
where d' = min(l+, dmax − d); bound improves closer to root.
P(M1 = {π*(1)}) ≥ pl+ − O(n−ε/3)
Root guarantee.
P(Mu ⊆ {π*(u)}, ∀u) ≥ 1 − O(n−ε/3)
Global correctness: no false positives with high probability.

Model Setup

  • Correlated ER pair: Base G ~ ER(n, λ/ns), subsampled with probability s → G₁, G₂; G₂ permuted by π*
  • IC diffusion: From source vertex 1, each activated vertex tries to activate each inactive neighbor with probability q
  • Observation: Algorithm sees diffusion tree TIC and permuted network G̃₂; knows λ, s, q
  • Parameters: l = ⌊log n / log(λq)⌋, dmax = ⌊(1−ε)log n / log(λq)⌋

Strengths · Limitations

Strengths

  • Clean problem formulation capturing information asymmetry in real diffusion data
  • Polynomial time with interpretable local matching criteria
  • Explicit per-vertex probability lower bounds — not just asymptotic existence
  • Global no-false-positive property with concrete error decay
  • Honest comparison to information-theoretic limits

Limitations

  • No empirical validation — practical performance unknown
  • Recall unquantified — many vertices may remain unmatched
  • Downward pass improves results but is not rigorously bounded
  • Strong model assumptions: sparse ER, known IC params, supercritical cascade
  • Recursive likelihood-ratio computation may be expensive at scale

Open Questions

  • Can the downward pass be rigorously analyzed to tighten bounds?
  • Efficient estimation of IC parameters (λ, s, q) from observations alone?
  • Robustness to model misspecification — non-IC diffusion, community structure?
  • Extension to dynamic/temporal networks?
  • What is the exact information-theoretic limit — and can poly-time algorithms reach it?

Relevance 实际意义

对社交网络去匿名化、传染病接触者追踪、谣言溯源以及区块链交易分析等场景具有理论指导意义；其双向传播与逐顶点概率下界的分析框架也可启发其他非对称图匹配问题的算法设计。
