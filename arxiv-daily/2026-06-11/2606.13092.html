2606.13092 World Models Equivariance Certified Prediction Lyapunov Spectrum 2026-06-11

Scale Buys Interpolation, Structure Buys a Horizon

Structure—not scale—buys a computable, tight, calibration-free certified prediction horizon for equivariant world models. The certificate is exclusive to equivariance: no non-equivariant model has it at any scale.

Key Metric · Lyapunov Spectrum Recovery
R² = 0.9940-D Lorenz-96

Equivariant cyclic-convolution net recovers the full Lyapunov spectrum.
Identically trained dense MLP: R² < 0, λ₁ inflated ~3.4×.

Why This Matters

Enables calibration-free deployment monitoring and sensing-budget scheduling for equivariant world models. The equivariant certificate yields 8–16% violation vs 61–65% for dense — requiring zero rollout calibration data and 2.7–3.5× fewer observations to match performance.

Critical Edge

Certificate is vacuous when λ₁ ≈ 0 (near-neutral dynamics) — covers important control regimes. At foundation scale (V-JEPA 2-AC, 1B) the model never enters the linearization neighborhood. The lift to learned models assumes C¹-closeness, verified only by one-step L² error.

Methodology Pipeline
  1. Equivariant Latent Setup
    Encoder E, predictor f, dynamics Φ with group G acting via orthogonal representation ρ. Assumptions A1–A5 encode encoder, predictor, and dynamical symmetry.
  2. Orbit-Constant Error (Theorem A)
    T-step rollout error is identical across every orbit point — checking k generators certifies the exponentially large generated monoid.
  3. Channel-Stratified Horizon (Theorem B)
    Each latent channel j certified to Tj(ε) ~ log(1/ε)/λj. Channels with λj ≤ 0 get unbounded horizons — the Noether hinge.
  4. Tightness (Proposition 6)
    A single encoder defect of size ε produces orbit-error variation exactly ε·eλT — matching Theorem B up to constants.
  5. Executable Procedure (Algorithm 1)
    Check generator equivariance → estimate predictor spectrum → report whether (w, T, ε) lies in the certified region. Finite-sample intervals given.
Key Results
E2 · Spectrum Recovery
On 40-D Lorenz-96, equivariant net: R² = 0.98–0.99, Liouville identity matched to 0.0%. Dense MLP: R² < 0. Phase transition at N=40: only equivariant survives.
E2 · Horizon Staircase
Certified-horizon staircases on Lorenz, Hénon, Rössler are linear in log(1/ε) with R² = 0.975–0.995, recovering textbook Lyapunov exponents to 1–12%.
E12 · Sensing Budget
Equivariant: 8–16% violation. Dense: 61–65%. Dense needs 2.7–3.5× observations to catch up; recalibration costs rollout data, equivariant needs none.
E14 · Public Model Audit
TD-MPC2 audited training-free: calibrated where expansive, correctly abstaining where contracting. Across 1M–317M ladder, calibration does not improve monotonically with scale.
E16 · Foundation Scale
V-JEPA 2-AC (1B): reads expansive (λ₁ ≈ 0.178), but measured log-slope 0.03 ≪ λ₁ — model never enters linearization neighborhood. Cross-check overrides raw spectrum.
E15 · Deployed Monitor
Sensor-only cheetah monitor replicates bench scope out-of-sample, detects frozen-actuator faults with recall 1.00 on 3/3 seeds.
Core Bound (Theorem B)
Orbit-error variation ≤ Σj cj · (mεmax + Tδ) · eλjT

Channel j certified to   Tj(ε) ~ log(1/ε) / λj   when λj > 0
Channel j certified to   Tj = ∞   when λj ≤ 0   ← Noether hinge

εmax = encoder residual  ·  δ = per-step predictor error  ·  cj = 1/sin(θj)  ·  λj = channel Lyapunov exponent
Strengths / Limitations
Strengths
  • Formal, computable, two-sided certificate — not just empirical error bars
  • Connects DL symmetries to classical dynamics: Lyapunov, Oseledets, Noether
  • Proves exclusivity: the certificate characterizes equivariance (Lemma 2)
  • Training-free auditing of public pretrained world models
  • Honest scope reporting: inconclusive results and failure modes disclosed
  • Actionable: sensing-budget scheduling and deployment monitoring
Limitations
  • Exact guarantees require genuine dynamical symmetry; approximate degrades gracefully but is horizon-limited
  • Prefactor cj = 1/sin(θj) can be large for near-tangential splittings
  • Horizon law vacuous for near-neutral dynamics (λ₁ ≈ 0) — important control regimes
  • Small-scale experiments; raw-pixel accuracy poor for all architectures at reported scale
  • Foundation-scale certificate can be vacuous (V-JEPA 2-AC)
  • Single-author preprint; peer review and independent replication pending
