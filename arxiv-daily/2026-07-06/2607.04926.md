> **一句话总结：** 在一个可全枚举的微型世界（MicroGround）中，作者用约6–10K参数的极小Transformer证明：信息足够（Bayes上界=1.0）时，无论信息以符号token、干净的逐因子"oracle"码还是纠缠感知向量进入模型，零样本组合绑定在所有路径下都收敛到查表式失败（≤随机水平0.25）；而一旦泄漏少量样本，少样本效率只由两个平凡因素决定——输入路径的参数共享和码的可读性——"groundedness"本身并未带来任何独立收益。

> **TL;DR:** On a fully-enumerable testbed (MicroGround, 128–256 states) with ~6–10K-parameter transformers, the paper shows that no informative input pathway (symbolic token / factored-oracle / weak- or strong-entangled perceptual / one-hot-shared) achieves converged zero-shot compositional binding—all collapse to lookup despite an exact Bayes ceiling of 1.0, even after a ~25× capacity increase. When a few-shot fraction is leaked, sample efficiency is governed only by (i) input-pathway parameter sharing and (ii) the practical readability of the code: the maximally clean per-factor oracle is the *least* sample-efficient readable route (+0.057 deficit vs. one-hot-shared, p=0.0020), and the poorly-readable strong-entangled code saturates near 0.48 while readable routes reach 0.82–0.96 at f=0.2.

## 核心贡献

- **MicroGround可全枚举测试床与方法学**：构建128–256状态、~6–10K参数Transformer的有限因子世界，对所有路由进行全输入空间评估（零采样方差），并为每个路由计算精确Bayes上界，从而把"失败"明确分类为信息缺失 vs. 归纳偏置 vs. 训练可识别性问题。统计协议采用10–20种子、配对Wilcoxon检验、Holm–Bonferroni校正、bootstrap置信区间与Cliff's δ，并在探索/确认（exploratory/confirmatory）划分下报告。

- **端点不变性（Endpoint Invariance）负结果**：在留出的组合绑定查询上，没有任何信息足够的路由实现收敛的零样本组合——全部在或低于随机水平（0.25），即使Bayes上界=1.0，且经容量（~25×参数）、学习率（10×范围）、权重衰减和更长训练检验仍稳健。这是关于小规模下零样本组合性的一个异常受控的否定结论。

- **少样本绑定的双因素解释**：当少量留出样本泄漏进训练后，样本效率主要由两个平凡变量预测——输入路径参数共享（共享投影迁移到未见查询类型；逐因子嵌入表则不能）和码的可读性。最干净的逐因子oracle反而是最差的可读路由，给出"解耦不是充分条件"的明确机制（参数共享）。

- **双重分离（Double Dissociation）**：训练早期，分布式码（符号/弱感知）经历短暂的零样本迁移高峰后塌缩为记忆化，而索引类码从不离开随机水平——这一轨迹现象由码格式决定；而少样本效率由路径共享决定。零样本失败模式并不预测少样本能力。

- **失效解剖（Failure Anatomy）**：通过控制任务探针加输入干预，分离出三种失效模式——符号路由在readout位置丢失答案（表征死亡）、索引路由系统性错绑定（答案在残差中仍可线性解码但输出因果依赖于错误槽位，错绑率高达87%）、纠缠路由继承而非改善输入的可读性（残差可解码性0.589≈输入原始0.58）。

## Key Contributions

- **MicroGround testbed and methodology (§3–§5):** Fully-enumerable factored worlds with route-factored conditions (symbolic / factored-oracle / weak- & strong-entangled perceptual / one-hot-shared, plus floor and scrambled controls), exact per-route Bayes-optimal solvability ceilings, exhaustive evaluation with zero sampling variance, and a rigorous statistical protocol (converged-accuracy metric, 5–20 seeds with 10–20 for headline comparisons, paired Wilcoxon with Holm–Bonferroni, bootstrap CIs, Cliff's δ) under an explicit exploratory/confirmatory split.

- **Endpoint Invariance (§6):** A strong, unusually well-controlled negative result: no informative route achieves converged zero-shot compositional binding—every route ends at or below chance (0.25) despite an exact Bayes ceiling of 1.0. This is robust to a ~25× parameter increase (6K→153K), a 10× learning-rate range, weight decay, and training length. A transition-holdout diagnostic (after Hu et al., 2024) shows the same memorization pattern (0.000–0.058 vs. a 0.833 blind-majority ceiling).

- **Two-factor account of few-shot binding (§9):** Once a fraction f∈{0.02,0.05,0.1,0.2} of the held-out query type is leaked, sample efficiency is best predicted by (i) input-pathway parameter sharing and (ii) practical readability. The clean per-factor oracle is the *worst* readable route—an information-matched, exactly parameter-matched instance of "parameter sharing aids systematic generalization" (Csordás et al., 2021) and a candidate mechanism for "disentanglement is not sufficient" (Montero et al., 2021).

- **A double dissociation (§7):** Early in training, distributed codes (symbolic, weak-perceptual) transiently rise above chance on the never-trained query type (e.g., 0.358 peak for symbolic) before collapsing into memorization, while index-like codes never leave chance. This transient tracks **code format**; few-shot efficiency tracks **pathway sharing**—two orthogonal axes.

- **Failure anatomy with causal input intervention (§8):** Control-task probes plus a direct input-intervention separate three failure modes—representation death at the readout position (symbolic), systematic mis-binding with the answer decodable yet the output causally depending on the wrong slot (index routes, mis-binding up to 0.866), and no readability gained over the input (entangled route).

## Methodology

**Testbed — MicroGround.** Finite factored worlds: a single-object world with four categorical factors (colour, shape, position, size; cardinalities 4,4,4,2 → 128 states) and a two-object world (left/right positional roles, each with colour+shape → 4⁴=256 states). Both are enumerated in full, so every evaluation runs over the entire query space (zero sampling variance).

**Tasks.** (i) *Attribute* (report a queried factor — a lookup sanity ceiling); (ii) *counterfactual* (report the successor (v+1) mod k); (iii) *binding* (report one attribute of the object at a queried position, e.g., "shape of right"). The two-object binding task has four query types, 256 queries each.

**Input routes (conditions).** Each condition factors into a text form plus a state encoder, so the same exhaustive query set is realized under every route:
- `text_only` (symbolic): description+question in tokens.
- `state_factored` (oracle): per-factor index → per-factor embedding tables.
- `state_onehot_shared`: concat one-hot → shared linear (16→24).
- `state_perceptual` (weak): fixed 2-layer tanh mix, d=16 → shared linear (raw linear dec. 0.95).
- `state_perceptual_hard` (strong-entangled): fixed 3-layer tanh mix, d=8, gain 2.5 → shared linear (raw linear dec. 0.58).
- Floors: `text_minimal`, `uninformative_state`. Control: `scrambled_state` (bijective relabeling).
Every informative route is verified injective (Bayes ceiling 1.0). `state_onehot_shared` and `state_perceptual` share an identical 16→24 linear pathway (exactly matched parameter count).

**Generalization splits.** Random 80/20; *binding holdout* (remove an entire (position, attribute) query type); *k-shot binding* (leak fraction f back in); *transition holdout* (remove 30% of successor transitions, after Hu et al., 2024).

**Model & optimization.** 1-layer (2-layer in the capacity arm) pre-norm decoder-only transformer, hidden 24 (~6–10K params), AdamW, LR 10⁻³, batch 32, cross-entropy on the final-token logits. State embedding added to every token position.

**Exact Bayes ceilings.** For every (task, split, route), group test queries by exact input identity and predict within-group majority—an a priori, model-agnostic upper bound. This separates information insufficiency (channel capacity) from inductive-bias failures.

**Evaluation discipline.** Converged accuracy (mean over last 10% of evaluations) rather than peak; balanced accuracy always reported against exact chance and majority baselines; 10–20 seeds with bootstrap 95% CIs, paired Wilcoxon signed-rank, Holm–Bonferroni correction, Cliff's δ. Exploratory holdout ("shape of right") was re-tested at 20 seeds and replicated on a fresh confirmatory holdout ("shape of left").

**Failure probes.** Control-task linear probes (Hewitt & Liang, 2019) subtracted from 5-fold logistic probes on the final-token residual (selectivity = probe − control); a direct causal input-intervention that varies the correct vs. wrong object slot and reads the output fraction that matches.

## Results & Highlights

1. **Zero-shot endpoint invariance (Table 2, n=20, chance=0.25, ceiling=1.0).** No informative route generalizes at convergence: `state_perceptual_hard` ends at **0.146** [0.11, 0.18], `state_factored` (oracle) at **0.198** [0.16, 0.23], `text_only` at **0.227** [0.19, 0.26], all at or below the chance baseline of 0.25, while every informative route memorizes the three trained query types to ≈1.0. Capacity scaling to 153K parameters (≈25×) leaves converged held-out binding within [0.15, 0.24] for every route swept.

2. **Few-shot dose-response (Fig. 3, n=20).** Readable routes rise smoothly from chance to **0.82–0.96 at f=0.2**, while the strong-entangled route saturates near **0.48**—pooled disadvantage vs. oracle −0.090 (p=3.8×10⁻⁶ exploratory) and −0.112 (p=5.7×10⁻⁶ confirmatory). The strong code's factors are recovered by an MLP decoder at only 0.56 (vs. 0.95/0.98 linear/MLP for the weak code), so the deficit is practical readability, not information content (the code is injective, ceiling 1.0).

3. **Pathway sharing dissociated from readability (Table 6, n=20, pooled ∆ vs. oracle).** The shared one-hot route beats the modular oracle by **+0.057** (p=0.0020 exploratory) and **+0.062** (p=1.1×10⁻⁴ confirmatory), and is statistically indistinguishable from weak-perceptual (p=0.52 / 0.083). So the maximally clean per-factor code is the *least* sample-efficient readable route. The ordering is: shared readable pathways > modular readable oracle ≫ shared low-readability code.

4. **Graded readability at fixed dimension (Table 5, d=16, n=10).** Four shared-projection codes at raw linear decodability {0.95, 0.82, 0.76, 0.53} give few-shot binding accuracy {0.57, 0.48, 0.47, 0.32} at f=0.1 and {0.93, 0.67, 0.57, 0.34} at f=0.2 (Spearman ρ=1.0 at both doses)—few-shot binding is a graded, monotone function of readability at fixed input dimension, ruling out an input-dimension confound.

5. **Transient compositional phase (Fig. 1, n=20).** Early in training the symbolic route reaches a peak of **0.358** [0.33, 0.39] on the never-trained type before collapsing (collapse 0.131), weak-perceptual peaks at 0.308, while index-like routes never leave chance (peaks 0.22–0.25). Collapse magnitude separates by code format: text_only vs. state_factored p_adj=7.8×10⁻⁴, δ=+0.71.

6. **Three-object stress check (Table 8, 729 states, n=10).** The two core effects replicate: shared one-hot beats oracle by +0.066 (p=0.037); strong-entangled collapses at −0.348 (p=0.002), saturating near 0.51 while readable routes exceed 0.94. The one effect that does *not* carry over is the weak-perceptual edge over the oracle (−0.001, p=0.77), flagged as testbed-specific.

7. **Failure anatomy (Table 3, n=10).** Index routes keep the answer near-perfectly decodable (probe 0.952–0.963, selectivity ≈0.69) yet mis-bind on up to **0.866** of discriminating states; the symbolic route's selectivity decays from 0.215 (epoch 5) to ≈0. The causal intervention (Table 4) confirms that on the held-out query the output is more sensitive to the wrong slot (0.71–0.86 for index routes) than the correct one (0.15–0.24, i.e. chance), whereas on a trained query it depends on the queried slot (0.67–1.00).

## Authors & Credibility

- **Yoshiyuki Ootani**, independent researcher (single author). The work is methodologically unusually careful: explicit exploratory/confirmatory split, pre-registered predictions in the confirmatory arm, honest disclosure of two hypotheses that did *not* survive the protocol (a three-way route ordering shrank to two-group at 20 seeds; the headline "groundedness induces composition" hypothesis was falsified by the strong-entangled control). All code, manifests, and per-seed logs are released at github.com/otanl/microground for exact reproduction, and every table/figure re-aggregates from JSONL manifests without retraining. The single-author, synthetic-world scope keeps credibility high for the claims actually made, while naturally limiting the reach of extrapolation.

## Critical Assessment

- **Novelty:** Individually the ingredients are *not* new—parameter sharing aids systematic generalization (Csordás et al., 2021), disentanglement is not sufficient (Montero et al., 2021), small transformers memorize under weak compositional pressure, and encoding form governs generalization (Kazemnejad et al., 2023). The genuine contribution is *confound removal*: matching information exactly (verified by per-route Bayes ceilings), enumerating the whole input space, and factoring the input pathway into sharing vs. code-format vs. readability so that a single parameter-matched extra cell (`state_onehot_shared`) settles a two-way mechanism question. That is a real but incremental methodological advance rather than a new phenomenon.

- **Rigor:** High. Exhaustive evaluation, exact ceilings, 10–20 seeds on headline comparisons, paired nonparametric tests with Holm–Bonferroni, bootstrap CIs, Cliff's δ, and a properly separated exploratory/confirmatory split. The honesty about the n=10 mechanism arms and the symbolic route's borderline confirmatory significance (p=0.053) is commendable.

- **Reproducibility:** Strong. JSONL manifests record config/seeds/trajectories; the repo claims exact re-aggregation; determinism caveat (CPU-deterministic up to library versioning) is disclosed. This is exemplary for the scale.

- **Significance:** Bounded and the authors say so. The central positive claim (two-factor few-shot account) replicates across two held-out shape query types and a three-object stress check, but the weak-perceptual/oracle ranking does *not* carry over (explicitly flagged testbed-specific), and both held-out types query *shape*. The negative endpoint-invariance result is only as strong as its sweep: a training objective for which pure lookup is a low-loss solution cannot reward composition, and the authors themselves name the missing test (more held-out types to make lookup costly) as future work.

- **Red flags / specific weaknesses:**
  - **The strongest baseline is "chance," not a competitive method.** There is no comparison to existing compositional-generalization or binding models; the negative result is "no route beats chance at convergence," which, given the tiny world and lookup-sufficient training objective, is arguably the expected default rather than a surprising pathology. The transition-holdout result is acknowledged as possibly meaningless (only three visible transitions per factor may not reward the successor rule).
  - **The readability story has an untested cell.** The authors cannot construct a code that is linearly unreadable yet nonlinearly (MLP-)recoverable with a fixed tanh mix, so they explicitly cannot rule out "linear readability gates binding even when information is nonlinearly recoverable"—the decisive test is open work.
  - **Causal handle is coarse.** The input intervention is clean but coarse; internal readout-swap or activation-patching would localize the dependence. The authors flag this.
  - **Same-author exploration and confirmation.** The confirmatory arm fixed predictions/configs before running, but exploration and confirmation were done by the same authors on the same testbed—a soft threat to the "confirmatory" label.
  - **Scale.** ~6–10K parameters, 128–729 states. The authors are explicit that nothing should be extrapolated beyond small transformers without new evidence, but readers interested in large-model grounding claims will find the leap unsupported here.

## Strengths

- Exceptional experimental control: information matched exactly across routes and verified by exact per-route Bayes ceilings, eliminating the confounds (dataset/capacity/difficulty) that plague modality comparisons.
- Exhaustive, zero-variance evaluation turns a transient training phenomenon into a measurable object and turns "failure" into a three-way classification (information- vs. inductive-bias- vs. identifiability-limited).
- Genuine falsifiability discipline: the paper reports its own falsified headline hypothesis (groundedness → composition) and the orderings that shrank at higher seed counts.
- Causal input-intervention goes beyond correlational probes to show the converged output functionally depends on the *wrong* slot.
- Full release (code, manifests, per-seed logs) with exact re-aggregation.

## Limitations

- Worlds of only 128–729 states and 1–2-layer models; claims are explicitly not extrapolable to larger transformers.
- Entanglement is operationalized by two main routes plus a dimension-matched control; the decisive "high-information, linearly-unreadable, MLP-recoverable" cell is untested and left as open work.
- Both held-out query types query *shape*—attribute- and modality-general replication is future work, and the weak-perceptual/oracle ranking does not replicate in the three-object world.
- The training objective makes pure lookup a low-loss solution, so the world does not strongly reward composition; "endpoint invariance" is a bounded observation ("no knob in this box"), not a universal limit.
- The transition-holdout diagnostic may be uninformative (the tiny world may not reward the successor rule at all), so the authors correctly refuse to lean on it beyond the binding conclusion.
- Exploration and confirmation performed by the same authors on the same testbed.
- The strongest comparative baseline is chance (0.25), not a competing compositional-generalization or binding model.

## Questions for Further Exploration

- Does the two-factor account (sharing + readability) predict adapter-/prompt-style transfer in large models, where "shared vs. modular input pathways" has direct analogues (e.g., shared embedding tables, Mixture-of-Experts routing)?
- Can a code be constructed that is injective and high-information yet linearly unreadable but nonlinearly recoverable—the decisive untested cell for the readability claim?
- If more query types are held out (making pure lookup costly in training), does endpoint invariance break, or do routes merely shift along the same two-factor axes?
- Does the transient above-chance phase generalize beyond tiny models, or is it capacity-fragile (the paper already shows the peak decays 0.385→0.258 from 6K→153K params)?
- Would internal activation-patching or readout-swap interventions localize the wrong-slot dependence to a specific circuit, strengthening the causal claim beyond coarse input intervention?
- Can the mis-binding failure mode (intact representation, wrong-slot readout) be detected as a silent failure pattern in production-scale models, where it would be a form of reliability/SDC risk?
