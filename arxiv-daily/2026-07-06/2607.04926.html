Input Pathways Shape Few-Shot, Not Zero-Shot, Binding in Tiny Transformers — A Fully-Enumerable Study · Yoshiyuki Ootani

Zero-shot binding works on no pathway.
Few-shot success is set by two things only: parameter sharing & code readability.

一句话总结：在可全枚举的微型世界（MicroGround）中，~6–10K 参数的极小 Transformer 证明——信息足够（Bayes 上界 = 1.0）时，零样本组合绑定在所有输入路径下都收敛到查表式失败（≤ 随机水平 0.25）；一旦泄漏少量样本，少样本效率只由输入路径的参数共享和码的可读性决定，"groundedness" 本身并未带来任何独立收益。

Key metric · few-shot ceiling

0.48

The strong-entangled route's few-shot binding saturation — while readable routes reach 0.82–0.96 at f=0.2. The code is injective (Bayes ceiling 1.0): the deficit is practical readability, not information.

p = 3.8×10⁻⁶ · Cliff's δ deficit −0.090

Why this matters

Dissects failure modes via control-task probes and causal input intervention, and uses exact Bayes ceilings to classify every failure as information- vs. inductive-bias — a controlled interpretability paradigm.

Critical edge

The strongest baseline is chance itself: with a tiny, lookup-sufficient world, "no route beats chance" is arguably the expected default, not a surprising pathology.

Results & highlights

7 findings · n=10–20 seeds · paired Wilcoxon + Holm–Bonferroni
1Zero-shot endpoint invariance — a robust negative result.

No informative route generalizes at convergence: every route ends at or below chance ≤ 0.25 despite an exact Bayes ceiling of 1.0, while memorizing trained query types to ≈1.0. Robust to a ~25× capacity jump (6K→153K params), a 10× LR range, weight decay, and longer training.

2Few-shot dose-response splits on readability.

Readable routes rise smoothly to 0.82–0.96 at f=0.2; strong-entangled saturates near 0.48. An MLP decoder recovers the strong code's factors at only 0.56 vs. 0.95/0.98 for the weak code — deficit is readability, not content.

3Cleanest oracle is the least sample-efficient.

The shared one-hot route beats the modular per-factor oracle by +0.057 (p=0.0020 exploratory, +0.062 confirmatory) and is indistinguishable from weak-perceptual. A parameter-matched instance of "sharing aids systematicity."

4Readability is a graded, monotone lever.

Four shared codes at fixed d=16 with linear decodability {0.95, 0.82, 0.76, 0.53} yield binding {0.57, 0.48, 0.47, 0.32} at f=0.1 — Spearman ρ = 1.0. Rules out an input-dimension confound.

5A transient compositional phase, then collapse.

Early in training the symbolic route peaks at 0.358 on the never-trained type before collapsing (−0.131); index-like routes never leave chance. The transient tracks code format — distinct from the few-shot axis.

6Three-object stress check replicates core effects.

729 states: shared one-hot beats oracle by +0.066; strong-entangled collapses at −0.348. The weak-perceptual/oracle ordering does not carry over (−0.001, p=0.77) — flagged testbed-specific.

7Failure anatomy — the answer is there, read from the wrong slot.

Index routes keep the answer near-perfectly decodable (probe 0.952–0.963) yet mis-bind on up to 0.866 of states. Causal input intervention confirms the held-out output depends on the wrong slot (0.71–0.86) over the correct one (0.15–0.24 ≈ chance), while the symbolic route's selectivity decays 0.215 → ≈0 (representation death).

Methodology

route-factored · exhaustive · zero sampling variance

Testbed & design

MicroGround — finite factored worlds, enumerated in full so every evaluation covers the entire query space. Single-object world: four factors (colour, shape, position, size; 4·4·4·2 = 128 states). Two-object world: 4⁴ = 256 states.

Tasks: attribute (lookup ceiling), counterfactual (successor mod k), binding (attribute of the object at a queried position). Generalization splits: random 80/20, binding holdout (remove an entire query type), k-shot (leak fraction f), transition holdout (Hu et al., 2024).

1-layer pre-norm decoder-only transformer, hidden 24 (~6–10K params; 2-layer in the capacity arm), AdamW, CE on final-token logits. Exact Bayes ceilings computed per (task, split, route) separate information insufficiency from inductive-bias failure.

Strengths & limitations

honest falsifiability vs. bounded reach
+

Strengths

  • Information matched exactly across routes and verified by per-route Bayes ceilings — eliminates the confounds that plague modality comparisons.
  • Exhaustive, zero-variance evaluation turns a transient phenomenon into a measurable object and classifies "failure" three ways (information / inductive-bias / identifiability).
  • Genuine falsifiability: reports its own falsified headline hypothesis (groundedness → composition) and orderings that shrank at higher seed counts.
  • Causal input-intervention goes beyond correlational probes — shows the output functionally depends on the wrong slot.
  • Full release: code, JSONL manifests, per-seed logs, exact re-aggregation without retraining.
–

Limitations

  • Worlds of only 128–729 states, 1–2-layer models — explicitly not extrapolable to larger transformers.
  • The decisive cell — high-information, linearly-unreadable, MLP-recoverable — is untested and left as open work.
  • Both held-out query types query shape; attribute/modality replication is future work; weak-perceptual/oracle ranking does not replicate in 3-object world.
  • The training objective makes pure lookup a low-loss solution, so "endpoint invariance" is a bounded observation, not a universal limit.
  • Strongest comparative baseline is chance (0.25), not a competing binding model.
  • Exploration and confirmation performed by the same author on the same testbed.
