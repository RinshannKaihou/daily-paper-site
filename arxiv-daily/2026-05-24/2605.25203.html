Methodology

Before quantization, each linear layer's weight matrix is transformed by an orthogonal rotation Q (fixed Walsh-Hadamard) and a diagonal scale D (spectral-energy adaptive):

x → x_pad · Q · D⁻¹   |   W^T → D · Q^T · W^T_pad   |   s_ℓ = (1 + d_pad · ρ_ℓ)^0.5

High-energy Walsh-basis channels get proportionally more INT2 grid resolution. At dequantization, Q and D unfold exactly — relative error < 5×10⁻⁷. Three architecture-specific extensions handle Qwen3 q_norm/k_norm, RoPE rotation pairs, and MoE gate absorption.

Strengths

  • Clean, math-invariant recipe — no per-model hyperparameter tuning
  • Deliberate W4 negative control proves method is W2-specific, not universal
  • Exceptional intellectual honesty: explicit disclaimers, acknowledged gaps
  • Architecture-specific fixes generalize beyond tested models (Qwen3, MoE, RoPE)
  • Hardware portability verified: PPL invariant across NPU, Arc, CPU (±0.1)

Limitations

  • No comparison with strong W2 baselines (QuIP#, AQLM, OmniQuant)
  • Scale capped at 1.5B — no evidence of transfer to 7B+
  • Evaluated on 8,192-token wikitext-2 subset only, no downstream tasks
  • Single-seed results; no multi-seed significance testing
  • No FP16 reference column — gap-to-FP16 metric absent
  • Theoretical grounding (Boolean influence → spectral energy) is qualitative
