Why This Matters

Any VLM can become interpretable by reusing existing textual SAE ecosystems — a major practical win for mechanistic interpretability. No costly vision-specific SAE training or labeling pipeline needed.

Critical Edge

Object removal steering succeeds on only 47 / 100 test images. The LLM fills semantic voids with plausible but inaccurate hallucinations (e.g., removed cat → hallucinated person).

Methodology

Total loss Ltotal = LCE + LSAE
SAE loss  LSAE = (1/|M|) Σ ∥SAEl(evl) − evl∥²
  1. Frozen vision encoder (DINOv2 / CLIP / I-JEPA) produces tokens v
  2. Trainable linear projector Pθ maps v into LLM embedding space
  3. SAE reconstruction loss forces projected tokens onto the text SAE manifold
  4. Two-stage training: 595 k Conceptual Captions → 665 k LLaVA-1.5 fine-tune
  5. Vision encoder & LLM remain frozen throughout

Strengths

  • Minimal modification — one auxiliary loss term added to standard LLaVA training
  • Zero cost for vision SAE training or labeling; reuses existing textual SAE infrastructure
  • Strong causal validation through localized patch-level steering, not just correlational metrics
  • Systematic encoder comparison reveals local-objective encoders (DINOv2) far outperform global ones (CLIP)
  • Disproves prior assumption of universal delayed alignment — scientifically honest
  • Demonstrates both concrete (object) and abstract (emotion, danger) concept manipulation

Limitations

  • Requires pre-trained text SAEs for the target LLM — not available for all models yet
  • DINOv2 essentially required for reliable spatial steering; CLIP/I-JEPA fall short
  • Removal steering creates semantic voids filled with plausible hallucinations
  • SAE constraints cause performance regression on smaller LLMs (Gemma-2-2B)
  • Cannot retroactively apply to already-trained VLMs without re-alignment
  • Steering evaluation limited to single-concept removal/replacement

Questions for Further Exploration

  1. Can VISTA work on already-trained VLMs via projector fine-tuning alone?
  2. How does it scale to frontier VLMs (LLaVA-OneVision, InternVL, Qwen-VL) with larger LLM backbones?
  3. Can we trace causal visual→textual computation paths through the LLM using VISTA-aligned features?
  4. Would a hybrid approach (lightweight vision SAE + textual SAE transfer) capture visual-only concepts?
  5. Does the SAE reconstruction loss impair abstract visual reasoning (spatial relations, counting)?
  6. How does steering scale to video inputs where temporal consistency matters?
