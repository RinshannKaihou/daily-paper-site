> **一句话总结：** MultiHaluDet 提出四阶段堆叠框架，通过动态层采样、多尺度注意力和折外集成元学习来探测冻结 LLM 全层隐藏状态轨迹，实现多语言幻觉检测，在英语基准上达到 98.55% AUROC，并在法语/孟加拉语/阿姆哈拉语上展示强跨语言泛化。

> **TL;DR:** A four-stage stacking framework probes frozen LLM hidden state trajectories across all layers via dynamic layer sampling, multi-scale attention, and out-of-fold ensemble meta-learning for multilingual hallucination detection, achieving up to 98.55% AUROC on English benchmarks and strong cross-lingual transfer to French, Bangla, and Amharic.

## 核心贡献

- 提出四阶段框架：动态特征提取 → 多尺度注意力编码 → 折外深度特征生成 → 集成元学习，从冻结 LLM 的隐藏状态轨迹中捕获幻觉信号
- 设计动态层采样策略，将任意深度的 LLM 映射到固定 K=32 层序列，无需模型特定配置
- 多尺度注意力模块同时处理多种时间分辨率，通过可学习门控机制聚合，捕获隐藏状态中的局部语义偏移
- 折外（OOF）堆叠将深度表示输入 6 种异构分类器（XGBoost、LightGBM、RF、SVM、MLP、LR），用逻辑回归元学习器融合
- 在法语（高资源）、孟加拉语（中资源）、阿姆哈拉语（低资源）上验证跨语言泛化，无需语言特定微调

## Key Contributions

- Introduces a four-stage pipeline: dynamic feature extraction, multi-scale attention encoding, out-of-fold deep feature generation, and learned ensemble meta-learning for hallucination detection from frozen LLM hidden states
- Proposes dynamic layer sampling that maps any transformer depth to a fixed K=32 sequential representation, enabling architecture-agnostic processing across Mistral-7B and LLaMA2-7B
- Designs a multi-scale attention module that processes hidden state trajectories at multiple temporal resolutions with learned position-wise gating to capture localized semantic shifts
- Employs 5-fold OOF stacking with 6 diverse classifiers (XGBoost, LightGBM, RF, SVM, MLP, LR) fused by an ℓ2-regularized logistic meta-regressor, avoiding data leakage and producing calibrated probabilities
- Achieves state-of-the-art AUROC: 98.55% (HaluEval/LLaMA2-7B), 98.43% (HaluEval/Mistral-7B), 98.30% and 98.26% on TriviaQA
- Demonstrates cross-lingual generalization to French (96.2%), Bangla (89.1%), and Amharic (78.5%) on HaluEval with Mistral-7B, all substantially outperforming baselines

## Methodology

MultiHaluDet operates in four stages:

**Stage 1 — LLM-Based Feature Extraction.** Question-answer pairs are formatted as prompts and passed through a frozen, half-precision LLM (Mistral-7B or LLaMA2-7B) in a single forward pass. Dynamic layer sampling maps the model's L transformer layers to K=32 uniform indices via interpolation (Eq. 1), handling shallow/deep models without architecture-specific configuration. For each sampled layer, two complementary views are extracted: last-token representation and sequence mean. From these, per-layer descriptors (s^(l)) are computed comprising ℓ2 norm, mean, std, extremal values, activation sparsity, near-zero mass, kurtosis, and MAD (Eq. 2). Four anchor layers at proportional depth fractions provide consistent cross-architecture features. A global feature vector aggregates top-k next-token probabilities, logit entropy, layer-wise norm trajectory statistics, anchor descriptors, and cross-feature interactions.

**Stage 2 — Projection and Multi-Scale Attention.** The sequential input S ∈ R^(K×ds) is projected to hidden dimension H=384 with layer normalization and GELU. A multi-scale attention module processes the sequence at multiple scales via local average pooling → scale-specific projection → nearest-neighbor upsampling, combined through learned position-wise gates (Eq. 4). The fused sequence is modulated by learnable softmax-normalized layer importance weights and positional embeddings, then encoded by 6 Pre-LN Transformer encoder layers (8 heads). Self-attention pooling (Eq. 6) replaces mean pooling with a learned MLP that assigns scalar relevance scores for input-adaptive depth selection. A global branch processes the global feature vector through a 2-layer MLP, and gated fusion (Eq. 7) combines sequential and global representations.

**Stage 3 — Out-of-Fold Stacking.** 5-fold stratified cross-validation trains fresh MultiHaluDet instances on K−1 folds and extracts gated fusion representations for the held-out fold, producing a complete OOF training matrix without data leakage. Test representations are averaged across all fold models at the representation level.

**Stage 4 — Ensemble Meta-Learner.** The OOF features feed into 6 diverse classifiers (XGBoost, LightGBM, GradientBoosting, RF, SVM, LR). Their log-odds outputs are fused by an ℓ2-regularized logistic meta-regressor (Eq. 10) trained on a secondary hold-out split, with classification threshold set via Youden's J statistic.

**Multilingual Adaptation.** English benchmarks are translated to French, Bangla, and Amharic via Gemini 2.5 Flash, with human QA (96% accuracy, remaining 4% regenerated). Translated inputs are processed natively through the frozen LLM in each language's token space.

**Training.** Composite loss (BCE + focal + asymmetric + contrastive with label smoothing), data augmentation (Mixup + CutMix), AdamW optimizer (lr=2e-4, weight decay=6e-5), ReduceLROnPlateau, 45 epochs with early stopping (patience=15). Hardware: AMD Ryzen 5 8500G + RTX 5060 Ti 16GB.

## Results & Highlights

- **English benchmarks:** 98.55% AUROC (HaluEval/LLaMA2-7B), 98.43% (HaluEval/Mistral-7B), 98.30% and 98.26% on TriviaQA — consistent across architectures
- **Cross-architecture robustness:** unlike all baselines which degrade from Mistral to LLaMA2, MultiHaluDet maintains near-identical performance (slightly higher on LLaMA2 for HaluEval)
- **Cross-lingual performance (Mistral/LLaMA2):** French 96.2%/95.8% (HaluEval), Bangla 89.1%/88.4%, Amharic 78.5%/76.2% — all substantially above best baselines
- **Baselines fail dramatically:** P(True), AvgProb, AvgEnt near random chance (~42-50%); surface-level methods (Unigram, NLI) plateau at ~60%; strongest prior (Neural CDEs) reaches 95.4% on Mistral HaluEval but drops to 91.4% on LLaMA2
- **Ablation confirms all components necessary:** removing OOF stacking causes ~10% drop (most critical); removing multi-scale attention causes ~7% drop; removing trajectory probing (using only final layer) causes ~5% drop

## Critical Assessment

- **Novelty:** The individual components (hidden state probing, multi-scale attention, OOF stacking, ensemble meta-learning) are well-established. The primary novelty is their specific combination applied to multilingual hallucination detection. The dynamic layer sampling strategy is a practical contribution for cross-architecture compatibility.
- **Rigor:** The paper provides thorough ablation studies and compares against 13 baselines spanning 4 method categories. However, the 98.55% AUROC on HaluEval is remarkably high — this may reflect benchmark characteristics (HaluEval hallucinations may have clear internal-state signatures) rather than general detection difficulty. The multilingual evaluation is translation-based rather than using naturally occurring non-English data, which limits ecological validity.
- **Reproducibility:** Code is publicly available. Hyperparameters, training details, and hardware specifications are clearly documented. The use of frozen LLMs eliminates fine-tuning variability.
- **Significance:** Addressing hallucination detection in multilingual and low-resource settings is practically important. The finding that internal state signals transfer across languages without fine-tuning is noteworthy for the field.
- **Potential concerns:** (1) The translation-based multilingual evaluation may not reflect naturally occurring hallucination patterns in non-English languages. (2) Performance degrades substantially for low-resource languages (78.5% for Amharic), suggesting the approach inherits limitations of the base LLM's multilingual representations. (3) White-box access requirement limits applicability to proprietary models.

## Strengths

- No language-specific fine-tuning required — leverages frozen LLM representations
- Architecture-agnostic dynamic layer sampling works across different LLM depths
- Comprehensive evaluation: 13 baselines, 2 benchmarks, 2 LLM architectures, 4 languages, ablation study
- OOF stacking provides principled protection against overfitting
- Multi-scale attention captures both fine-grained and coarse-grained hallucination signals across layers
- Code publicly available, enhancing reproducibility
- Ablation study clearly demonstrates the necessity of each component

## Limitations

- Requires white-box access to LLM internal hidden states — incompatible with proprietary models (GPT-4, Claude)
- Extracting full-depth hidden states incurs non-trivial memory overhead and requires a complete forward pass
- Multilingual evaluation uses translated benchmarks rather than naturally occurring non-English data — cultural and linguistic nuances may not be fully captured
- Performance degrades significantly for low-resource languages (78.5% for Amharic vs 98.55% for English), inheriting base LLM limitations
- Binary hallucination/non-hallucination setup may not capture nuanced hallucination types (partial errors, numerical inaccuracies)
- Computational cost of the four-stage pipeline compared to simpler methods is not analyzed

## Relevance

本文通过探测 LLM 内部隐藏状态实现免微调的多语言幻觉检测，对关注 LLM 可靠性、跨语言泛化及模型内部表示分析的研究者具有直接参考价值，尤其是其发现内部状态幻觉信号可跨语言迁移。

## Questions for Further Exploration

- How does the framework perform on more nuanced hallucination types beyond binary classification (e.g., partial factual errors, numerical fabrications, logical contradictions)?
- What is the computational overhead of the full four-stage pipeline compared to single-pass hidden state probing methods?
- Can the approach be adapted for real-time detection during autoregressive generation rather than post-hoc analysis?
- How does performance change with larger LLMs (e.g., LLaMA-70B, Mistral-8x22B) — does dynamic layer sampling scale effectively?
- Would training on naturally occurring multilingual datasets (rather than translated ones) close the performance gap for low-resource languages like Amharic?
- Can the multi-scale attention mechanism be interpreted to identify which LLM layers carry the strongest hallucination signals, and does this vary by language?
