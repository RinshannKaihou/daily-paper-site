Causal Tongue-Tie: LLMs Can Encode Causal Direction, But Their Yes/No Outputs Fail to Express

A frozen linear probe recovers the evidence-aligned answer from LLM hidden states with ~97% accuracy — yet the model's spoken Yes/No output reverts to commonsense at ~50%. This Causal Tongue-Tie gap persists across 8 models from 0.5B to 72B parameters.

The accuracy gap between linear-probe readout and spoken Yes/No output on anti-commonsense causal items. Remarkably constant across all 8 models and all scale tiers.

Why This Matters

Output-only benchmarks are fundamentally broken for causal reasoning

Answering correctly does not prove understanding; answering incorrectly does not prove inability. Every causal reasoning benchmark that judges only token output may be measuring verbal compliance, not internal competence.

Critical Edge

Probe transfer is template-conditioned; base models untested

Cross-template probe transfer averages only 0.672. Anti-commonsense items are synthetic. Base (pre-RLHF) checkpoints not tested — cannot separate interface failure from instruction-tuning policy. The gap between “encodes direction” and “understands causation” remains large.

Methodology

Probe Design & Measurement

  • Frozen ℓ&sub2; logistic regression probe fit once on commonsense CLadder items' hidden states, reused unchanged on all subsets
  • Counterfactual subspace Vcs extracted via truncated SVD on paired hidden-state differences
  • Anti-commonsense items where in-prompt evidence contradicts everyday knowledge (e.g., “smoking prevents lung cancer”)
  • Same forward pass: reads both the linear probe prediction from an intermediate layer and the Yes/No token probabilities from the final lm_head
  • 5 answer interfaces tested to localize where the failure occurs

2×2 Decomposition

Redefining “wrong on a causal benchmark”

Signal Absent → Output Wrong

True failure

Model genuinely lacks internal representation of the causal direction. Probe and output both fail. Benchmark verdict is correct.

This is what current benchmarks assume is always happening.

Signal Present → Output Wrong

Causal Tongue-Tie

Model encodes correct direction internally but cannot express it verbally. Probe succeeds, output fails. Benchmark verdict is misleading.

This is the novel failure mode the paper identifies and characterizes.

Open Questions

  1. Does Causal Tongue-Tie manifest in naturally occurring causal disagreements (scientific debates, policy)?
  2. Would auditing base checkpoints reveal whether RLHF introduces the interface failure?
  3. Can multi-layer or low-rank interventions close the remaining sufficiency gap?
  4. How does CoT interact with Tongue-Tie — alternative channel or same gap masked?
  5. Would nonlinear probes transfer better across paraphrases?
