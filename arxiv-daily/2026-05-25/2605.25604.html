DVAO: Dynamic Variance-adaptive Advantage Optimization for Multi-reward Reinforcement Learning

Methodology — DVAO Formulation

DVAO replaces static combination weights with dynamic variance-adaptive weights that up-weight high-variance objectives (stronger learning signal) and suppress low-variance noisy ones within each rollout group.

w̃k = wk · σki / Σl wl · σli    →    ADVAO(i,j) = Σk w̃k · Ak(i,j)
Proposition 1 — RC vs AC
RC advantage squared magnitude ≥ AC. Magnitude explosion occurs unless all reward pairs are perfectly correlated.
Proposition 2 — Boundedness
DVAO advantage is pointwise bounded by RC advantage, preventing magnitude explosion entirely.
Proposition 3 — Regularization
Gradient of each objective modulated by overall multi-objective performance — implicit cross-objective regularization that AC lacks.
Core Contributions
  • Exposes theoretical flaws in both Reward Combination (magnitude explosion) and Advantage Combination (static weights, objective isolation).
  • Fully dynamic, hyperparameter-free weighting — up-weights high-variance objectives automatically.
  • Three formal propositions with proofs rigorously characterize why DVAO works.
  • State-of-the-art across math reasoning (4 benchmarks) and tool-use (BFCL-v4) with Qwen3 and Qwen2.5 models.
Results & Highlights
  • Qwen3-4B: 42.19% avg accuracy with ~100% length compliance. GDPO matches compliance but at ~13% accuracy.
  • Qwen3-8B: 47.49% avg accuracy — highest across all methods while maintaining near-perfect compliance.
  • Qwen2.5-7B BFCL-v4: 87.06% accuracy + 95.11% format compliance — best on both dimensions simultaneously.
  • Pareto frontier: DVAO dominates across weight sweeps (w₁ ∈ {0.1–0.9}); baselines show RC saturation, AC instability, GDPO incoherence.
  • Training dynamics show higher reward mean, lower variance, faster convergence, and dramatic variance collapse.

Strengths
  • Clean theoretical framework connecting variance-adaptive weighting to bounded advantages.
  • Only method dominating both accuracy and compliance across all model scales.
  • Hyperparameter-free — no manual weight tuning.
  • Comprehensive evaluation: 2 task domains, 4 model sizes, 7 benchmarks.
Limitations
  • Only 2-objective scenarios tested — behavior with 3+ conflicting objectives is unexplored.
  • Variance-adaptive mechanism may falter when all objectives have similar variance profiles.
  • Oscillatory response lengths during training — acknowledged but not fully addressed.
  • No computational overhead analysis for dynamic weight computation per rollout group.
  • Tool-use evaluation limited to BFCL-v4; generalization to other agentic tasks not shown.
Open Questions
  1. How does DVAO behave with 3+ objectives having varying correlation structures?
  2. Can variance-adaptive mechanism combine with curriculum learning for progressive objective introduction?
  3. Sensitivity to group size G — does DVAO degrade with smaller rollout groups?
  4. How does DVAO interact with GRPO variants like DAPO or GSPO?
  5. Is implicit cross-objective regularization sufficient to prevent reward hacking?
