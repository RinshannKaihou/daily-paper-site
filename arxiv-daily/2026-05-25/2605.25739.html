The Behavioral Credibility Trilemma: When Calibrated Autonomy Becomes Impossible

Lovén, Do, Mehmood, Sah & Tarkoma · University of Oulu, Finland

AI-alignment game-theory calibration RLHF impossibility-theorem
Key Finding

No RL policy with confidence-gated autonomy can simultaneously maximize helpfulness, calibration, and autonomy — the trilemma is a geometric impossibility, optimizer-independent across all log-concave policy families.

Why This Matters

Alignment training that jointly optimizes helpfulness, calibration, and autonomy targets a mathematically infeasible point. This impossibility applies to RLHF, selective prediction, conformal prediction, LLM tool use, MoE routing, federated learning, and active learning — any system that gates agent actions on self-reported confidence. The result forces an architectural choice: you can have at most two of the three.

Methodology

  • Principal-agent model: CGDP maps confidence reporting + gating + scoring to a delegation game
  • Impossibility proof: any non-affine autonomy incentive added to a strictly proper scoring rule destroys strict properness
  • Optimizer independence via FKG correlation inequality under log-concave densities
  • 540-config Best-of-N experiment on Qwen-2.5-7B (4-bit, τ=0.8) with 100-task evaluation set
  • Mixed-effects models with Bonferroni-Holm correction

Constructive Resolutions

  • Commitment (Thm 31) — Delegate binding-state tasks to an external reviewer. Recovers H = H* and C = C* at bounded autonomy cost (delegation rate).
  • Domain Separation (Prop 32) — Train separate policies for safe (p ≥ pmin) and uncertain (p < pmin) regimes. Restores two objectives per domain.

Critical Edge

  • Best-of-N ≠ gradient training. The experiment tests selection, not PPO/DPO. Whether the same magnitude of trilemma effects emerges under gradient-based optimization is unverified.
  • Single model, static model. All causal experiments on Qwen-2.5-7B (4-bit). Cross-model results are descriptive. The CGDP does not capture sequential token generation or multi-turn dialogue.
  • Delegation competence assumption. The Commitment Resolution assumes external reviewers maintain sufficient competence — an unsolved problem if humans rubber-stamp.
