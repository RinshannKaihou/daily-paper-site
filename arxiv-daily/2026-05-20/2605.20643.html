AVSD: Adaptive-View Self-Distillation

Balances consensus and teacher-specific privileged signals across multiple views, using a gated mechanism that prevents residual noise from reversing the consensus update direction.

Self-Distillation Reinforcement Learning Multi-View Learning LLM Reasoning Privileged Information Qwen3-4B / 8B DeepSeek-R1-Distill
Key Finding

Multi-view self-distillation with gated consensus/residual decomposition reduces incorrect-sign token credit from 12.3% (single-view) to 2.9% — a 4.2× improvement in credit assignment reliability — while consistently outperforming single-view OPSD and GRPO baselines across three model families and six benchmarks.

Neither consensus-only nor arithmetic-only targets are uniformly optimal. The gated combination is necessary: it retains view-specific signal while bounding residual magnitude to never exceed consensus.

Why This Matters

The field has sidestepped a fundamental question: which privileged information to use for self-distillation? AVSD dissolves this choice by aggregating multiple views with a provable guarantee that residual signals cannot reverse the consensus update direction.

  • Eliminates the need to hand-pick a single privileged context
  • Principled: geometric target minimizes avg reverse-KL to teacher family
  • Gate boundedness proven: λ·J ≤ |AG| always holds
  • Consistent gains across math + code, 3 model scales, 6 benchmarks
Critical Edge
Moderate Gains 2-3% absolute improvement — meaningful against strong baselines but not a paradigm shift.
Hidden Cost Wall-clock training overhead and GPU-hours are not reported despite M× additional teacher passes.
Narrow Scope Only evaluated on reasoning tasks with verifiable answers; open-ended generation is untested.
Ad-Hoc Views Privileged views (full/partial solution, answer) rely on dataset structure — not automatically discovered.
