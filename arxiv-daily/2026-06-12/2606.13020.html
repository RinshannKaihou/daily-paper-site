SciR: A Controllable Benchmark for Scientific Reasoning in LLMs

Pierre Beckmann · Marco Valentino · Andre Freitas  —  Idiap / EPFL / Sheffield / Manchester
SciR is the first multi-paradism scientific-reasoning benchmark with parametric control over both inference complexity and premise obfuscation — revealing that even neuro-symbolic pipelines see accuracy roughly halve when premises are buried in realistic scientific text.
首个在演绎、归纳、因果溯因三种推理范式上同时提供参数化难度控制的基准，系统揭示神经符号管线在前提混淆下准确率腰斩，证明形式化步骤本身需要真正的推理能力。
Key Finding
Premise obfuscation degrades even neuro-symbolic pipelines — the formalization step itself requires genuine reasoning, not just symbolic solving.
Key Metric
90.4 → 42.6
GPT-4o NS accuracy across four cells (NL Easy → Obf Hard)
Why This Matters
Neuro-symbolic methods were assumed robust to surface variation. This proves they aren't — directly impacting how we build scientific AI systems.
Critical Edge
Single-run at n=200 with no error bars. Rendering depends on o3-mini, a proprietary model that may change. One domain per paradigm.

Methodology — Three Tracks, Two Axes

Each track maps a reasoning paradigm to a real scientific domain, generating formal objects with verifiable ground truth before rendering them into multi-document scientific discourse.

TrackDomainTaskSolver
DeductionDevelopmental BiologyFOL syllogism validityProver9 + Mace4
InductionDrugBank DDIsRule discovery from observationsPopper (ILP)
Causal AbductionSachs Signalling NetworkEdge recovery for protein XYZGIES

Two-Axis Difficulty Control

  • Inference complexity — depth of deduction trees, number of distractor rules, subgraph size
  • Premise obfuscation — chunk count, genre diversity, rendering LLM capability
  • Each task rendered into N chunks across 8 document genres (scRNA-seq reports, FDA drug labels, wet-lab notebooks, etc.)
  • Cross-validated invertibility contract ensures information preservation

Results — Accuracy Collapse Under Dual Pressure

Both axes hurt every model and effects compound. GPT-4o CoT: 40.7 → 32.7 → 29.2 → 17.7 across the four difficulty cells. Neuro-symbolic accuracy roughly halves under obfuscation.

Assessment

Strengths

  • Two-axis decomposition tells you why a model fails
  • Invertibility contract is rigorous for information preservation
  • Pipeline regenerable as models saturate — forward-looking defense
  • Formaliser analysis provides actionable mechanistic insight

Limitations

  • One scientific domain per paradigm — generalization untested
  • No error bars (single run, n=200)
  • Rendering LLM (o3-mini) is a moving proprietary target
  • Six models — no Claude, Gemini, or smaller models

Takeaways

Core finding
Obfuscation degrades even NS pipelines — formalisation itself needs reasoning, not just symbolic solving.
Diagnostic value
Extraction-vs-inference profiling reveals model-specific failure modes: deduction is extraction-hard, causal is inference-hard.
Forward-looking
Pipeline can regenerate at higher difficulty as models saturate, making it contamination-resistant.

Relevance

首次实现推理能力与信息提取能力的独立量化，对理解模型在科学场景中的瓶颈具有重要价值。NS管线混淆下降直接启示提升科学AI鲁棒性的方向。

Open Questions

  1. How do RAG or multi-pass extraction strategies change the profile?
  2. Can the invertibility contract train better formalizers?
  3. Does rendering with stronger LLMs make extraction harder?
  4. Correlation with real-world scientific tasks?
  5. Extend profiling to analogical or counterfactual reasoning?
  6. NS degradation: formalization errors vs. extraction errors?
