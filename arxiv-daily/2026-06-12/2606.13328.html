8-bit sign-voting eliminates all floating-point MAC while surviving 90% weight truncation.

Key Metric
90%

Truncation sparsity sustained with near-perfect MSE reconstruction—confirming holographic storage properties.

Why This Matters

8-bit integer manifold mapping could reshape edge AI hardware—if it can move beyond 15×15 patch reconstruction to real-world tasks.

Critical Edge

Evaluates only reconstruction MSE on 15×15 patches with zero comparison to any existing quantization or binary network method.

Methodology

Input (15×15 grayscale) ├─► Spatial Stream → 8,192 nodes (over-complete coordinate grid) │ · Sensitive to sub-pixel shifts & rotation │ └─► Structural Stream → 4,096 nodes · 4-orient Gabor quadrature pairs (0° 45° 90° 135°) · DC luminance removal → 5×5 overlapping pool · Smooth topological tolerance │ ▼ Z-Matrix W ∈ ℤ^(M×N) [int8_t, clamped -127…+127] │ ▼ sign-voting accumulation (no MAC) local_sums[j] = Σ sign(W_ij) · 𝟙(|W_ij| ≥ 8.0) Y_j = 𝟙(local_sums[j] > 2.0) │ ▼ Sparse binary code {0,1} │ Learning: ΔW = ±1 bounded, stochastic noise injection
  • Replaces every floating-point MAC with sign-voting & conditional accumulation at fixed thresholds (θreject=8.0, θcut=2.0).
  • Cache-optimized inference via linear pointer offsets and boolean masks.
  • Bounded ±1 weight updates clamped to [-127, 127] with stochastic noise for regularization.

Results & Highlights

  • 20% random node destruction causes almost no perceptible MSE degradation.
  • 4× memory reduction vs. FP32 in footprint and cache utilization.
  • Spatial stream: rapid orthogonal decay under sub-pixel shifts.
  • Structural stream: excellent topological tolerance via Gabor pooling.
  • Core ops reduce to AND, OR, shift registers—native FPGA / neuromorphic fit.

Strengths

  • Elegant sign-voting replacement for MAC operations.
  • Clean biological motivation (V1/V2 simple & complex cells).
  • Holographic resilience experiments are visually compelling.
  • Valid points on memory bandwidth and edge computing constraints.

Limitations

  • No standard benchmarks—no ImageNet, CIFAR, MNIST.
  • No baselines—zero comparison to INT8, BNN, ternary networks.
  • Scalability unproven beyond 15×15 patches.
  • No energy measurements despite "low-energy AI" framing.
  • Learning rule is essentially a clipped Hebbian ±1 update.
