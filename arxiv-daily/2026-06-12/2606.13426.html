Accelerating Speculative Diffusions via Block Verification

Alexander Soen (KTH / Google Research) · Hisham Husain (Google Research) · Valentin De Bortoli (Google DeepMind) · Arnaud Doucet (Google DeepMind / Oxford)

Block verification—the state-of-the-art LLM speculative decoding technique—is successfully adapted to continuous diffusion models via an orthogonal decomposition that reduces high-dimensional residual sampling to a tractable 1D inverse-CDF problem. Combined with a training-free Free Drafter, it yields up to 6.3% additional speedup over existing speculative diffusion with zero quality degradation.

Key Finding
Block verification works for diffusions—cheaper drafting beats better drafting when overhead dominates.
Joint verification of entire draft blocks maximizes acceptance probability over per-sample checks.
Key Metric
6.3%
Max additional wall-clock speedup over existing speculative diffusion; 1.9–3.6× overall vs vanilla DDPM.
Why This Matters
Enables the mathematically correct speculative mechanism for diffusions, opening the door to compounding future gains.

Critical Edge
Only works with stochastic samplers (ε > 0). Untested on modern DiT architectures.
Incompatible with DDIM / probability flow ODEs. Memory scales ×(γ+1).

How It Works

  1. Orthogonal decomposition — Project the d-dimensional residual sampling problem onto the mean-difference direction e, reducing it to a 1D inverse-CDF solved via bisection. O(d) time, one target model evaluation.
  2. Block verification — Compute joint acceptance probabilities hⱼ for the entire draft block (γ=7 steps) using a recursive α parameter and the CDF of the 1D residual. Maximizes accepted steps per round.
  3. Free Drafter — Reuse cached score evaluations from the previous verification round instead of calling the target model at draft time. Marginal drafting cost drops to negligible.
  4. Exact correction — The stochastic residual ensures exact sampling from the target distribution—no approximation error accumulates.

Key Contributions

  • Decomposition algorithm reduces high-dimensional residual sampling to a deterministic 1D bisection + orthogonal Gaussian (Proposition 3.1, Algorithm 5)
  • Impossibility result — proves no deterministic invertible correction (incl. reflection coupling) works for block verification when γ ≥ 2 (Proposition 3.2)
  • Free Drafter formalization — proves Frozen Drafter needs ~2× the block efficiency just to match Free Drafter's wall-clock (Corollary 3.5)
  • Comprehensive evaluation — 6 datasets, 4 churn parameters, 5 step counts, wall-clock + FID + block efficiency all reported

Open Questions

  • Does block verification transfer to DiT/Transformer architectures at 1024² resolution?
  • Can decomposition-based residual sampling extend to non-Gaussian kernels (flow matching, rectified flow)?
  • Memory-cost vs. speedup tradeoff on consumer GPUs (A100 vs. RTX 4090)?
  • Adaptive draft sizes (varying γ per denoising step) for further gains?
  • Free Drafter principle applied to speculative decoding in other domains?

Verdict

  • Novelty High
  • Rigor Excellent
  • Reproducibility Good
  • Significance Moderate

Strengths

  • No training, no extra parameters, no quality loss—purely algorithmic
  • Exact sampling guarantee preserved
  • Elegant math: d-D → 1D + (d−1)-D separation
  • Clear theoretical proof why reflection coupling fails
  • Counterintuitive insight: cheaper > better drafting

Limitations

  • Incompatible with DDIM / deterministic solvers
  • Memory scales ×(γ+1) — problematic for large models
  • Speedup shrinks when combined with fewer-step methods
  • Only tested on U-Net ≤ 256²
  • 6.3% ceiling is modest in absolute terms

Relevance

Diffusion inference acceleration with methodology borrowed from LLM speculative decoding. Direct relevance is to image generation; indirect methodological value for inference optimization research.
