Training Infinitely Deep and Wide Transformers

The first rigorous mean-field training framework that treats attention coupling as a neural PDE — not a neural ODE — proving gradient flow converges to global minima at a linear rate.

Raphaēl Barboni · Maarten V. de Hoop · Takashi Furuya · Gabriel Peyré
Bocconi · Rice · Doshisha / RIKEN AIP · CNRS / ENS Paris

Key Finding
Training infinitely deep & wide transformers is a neural PDE optimal-control problem with an explicit Wasserstein gradient and guaranteed convergence to global minima.
Key Metric
L(ρt) ≤ L(ρ0)·e−CN−1λ0t
Linear rate · independent of token count
Why This Matters
Attention coupling transforms the ODE structure of ResNets into a PDE structure — the first framework handling both infinite depth and width simultaneously.

⚙ Methodology

  • Dual mean-field: Continuous depth s∈[0,1] × probability measure over attention params θ = (Q, q, V), yielding a neural PDE via continuity equation.
  • Adjoint sensitivity: Explicit Wasserstein gradient via backward-ODE adjoint variables mρ(j)(s) — the continuous-depth analogue of backpropagation.
  • COT metric: Conditional Optimal Transport enforces layer-wise parameter structure; gradient flow exists & is unique via Minimizing Movements + Grönwall.
  • NTK injectivity: Characterized by linear independence of cumulant generating functions (log-sum-exp) modulo affine functions — verified for discrete, uniform, Laplace, and Gaussian-mixture token distributions.

★ Results & Highlights

  • Forward well-posedness (Prop. 1): Unique flow-map solution in C0(S, ℝd) via Carathéodory theory.
  • Gradient formula (Thm. 1): Explicit conditional Wasserstein gradient with strong upper-gradient property.
  • Gradient flow (Thms. 2–3): Existence (De Giorgi scheme) and uniqueness (Grönwall) in COT metric space.
  • Linear convergence (Thm. 4): Rate O(e−CN−1λ0t) when NTK injective at init & small loss — no spurious local minima.
  • Token-count independence: Convergence guarantees do not depend on the number of input tokens.

✓ Strengths

  • First mean-field training framework purpose-built for transformers, with the key insight: attention = PDE (not ODE).
  • Flow-map perspective on C0(S, ℝd) yields quadratic (vs. exponential) Lipschitz dependence.
  • Complete NTK theory: necessary & sufficient conditions, verified for practical token distributions.
  • Convergence independent of token count — a significant advantage over prior work.
  • COT framework naturally preserves layer structure during training.

✗ Limitations

  • Infinite depth & width idealization with no quantitative finite-approximation bounds.
  • Impractically small initial-loss threshold: L(ρ0) < CN−3λ03.
  • FixUp initialization (V=0) is non-standard; may affect real training dynamics.
  • No experimental validation whatsoever — purely theoretical.
  • Simplified attention: K = Id removes key matrix; supplementary bias q is non-standard.
  • Highly specialized mathematical machinery limits audience.

Open Questions for Further Exploration

  1. How many layers and heads suffice for mean-field predictions to hold within ε?
  2. Does NTK injectivity hold for real-world token embeddings (high-dimensional, heavy-tailed, clustered)?
  3. Can the PDE perspective inspire PDE-aware learning rate schedules or architectures?
  4. What happens under large-initial-loss regimes where the PL condition fails?
  5. Can the framework extend to causal (decoder-only) transformers with masked attention or normalization layers?
  6. Does the supplementary bias q suggest a practical benefit of query biases in attention?
