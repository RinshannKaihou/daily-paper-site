Low-Loss Valleys in LoRA Bayesian Inference via Bezier Curves

LoRA-Curve constructs segmented Bezier paths through LoRA weight space, proving that independently fine-tuned optima can be connected by continuous low-loss valleys — enabling Bayesian model averaging over a continuous subspace rather than discrete ensemble points.
Headline Claim
LoRA optima are not isolated — they live in connected low-loss valleys bridgeable by Bezier curves.
Anchor Fact
980+ W&B runs across ~2,500 GPU-hours on A100/L40S, evaluated on Qwen2.5 7B
So What
Continuous mode connectivity captures functional diversity that both single-mode VI and discrete deep ensembles miss.
Critical Edge
Only one base model tested. FLC shows a prominent train-test generalization gap.

F Free LoRA-Curve (FLC)

  • All control points jointly optimized from scratch
  • Initialization: Bi = 0 for all points, so initial weights = W0 everywhere
  • Dynamically shifts anchors to circumvent loss barriers
  • Smoother training-loss valleys but exhibits generalization gap
  • Special case: MAP = FLC(1, 0)

A Anchored LoRA-Curve (ALC)

  • Anchor points frozen to independently fine-tuned LoRA optima
  • Only intermediate handles optimized for low-loss paths
  • Configuration: ALC(N, m) — N anchors, m handles per segment
  • Matches or outperforms Deep Ensembles across datasets
  • Special case: DE(N) = N anchors with m = 0

P Pipeline — How LoRA-Curve Works

1

Bezier Parameterization

Each segment defined by Bernstein polynomials with m+2 control points. Curve parameter t maps to LoRA config via zeta_theta(t), directly determining continuous A_theta(t) and B_theta(t).

2

Segmented Construction

For N anchors with N-1 segments, adjacent segments share boundary points ensuring C0 continuity. Global parameter t in [0, N_seg]. Enables localized bending around loss barriers.

3

JSD Regularization

Samples t_1 uniformly, computes t_2 = t_1 + N_seg/2 (mod N_seg) for maximal separation. JSD between predictive distributions clipped at tau=0.05, strength lambda=0.2.

4

Flat-LoRA Perturbation

Noise injection at rho=0.25 forces paths into broader loss regions. Excessive noise (rho=0.5) degrades performance.

5

Bayesian Model Averaging

M equispaced grid points with temperature scaling. Uniform averaging (T → inf) performs best — justified by a double-counting argument: the curve was already optimized on D.

1 Proposition 1 — Pathwise Continuity

The output distribution varies continuously along the Bezier curve.

t → p(y|x, W(t)) is continuous in t

Intermediate models remain highly performant. Class probability evolution shows smooth transitions.

2 Proposition 2 — Lipschitz Regularity

Loss along the curve cannot jump; changes are bounded by gradient norm and curve speed.

|L(W(t1)) - L(W(t2))| ≤ C · ||∇L|| · |t1 - t2|

Establishes that the loss cannot jump along the curve — controlled by gradient norm and curve speed.

M Mutual Information Metric (BALD)

MI(t; y) = H[y|D] - E_t[H[y|W(t), D]]

Epistemic uncertainty quantified as total predictive entropy minus expected data uncertainty across M grid points. Higher MI = more functional diversity captured by the curve.

+ Strengths

  • Elegant theoretical framework with formal propositions and proofs
  • First proof that LoRA space supports mode connectivity
  • Unified notation subsumes MAP, DE, linear connectivity
  • Two complementary variants (FLC for flexibility, ALC for control)
  • Honest reporting of FLC's generalization gap
  • Comprehensive ablations with clear takeaways
  • Thoughtful reporting of negative results in Appendix E

– Limitations

  • Single base model only (Qwen2.5 7B)
  • FLC train-test generalization gap
  • No calibration analysis (ECE, Brier score) centrally reported
  • Uniform BMA weighting likely fails in higher dimensions
  • No comparison vs Laplace approximation or MC-dropout
  • Classification benchmarks only; generation tasks untested
  • No linked code repository
