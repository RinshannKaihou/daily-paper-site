Attention outputs beat hidden states for semantic similarity — uncovered via multilingual consensus

DySem extracts cross-linguistically consistent dimensions from LLM internal states to compute STS. No training required. Roughly 1,000 dimensions instead of 4,096+. Across 10 models and 7 benchmarks, it consistently outperforms all baselines.

Kaijie Zheng, Weiqin Wang, Yile Wang, Hui Huang Shenzhen University LLaMA / Qwen / Phi

Why This Matters

The default practice of extracting last-layer hidden states from LLMs for text similarity is suboptimal. DySem shows that attention accumulations carry stronger semantic signal, and that multilingual invariance can isolate it with no training. This reframes how we should think about what LLMs internally encode about meaning.

Critical Edge
  • O(m) inference cost: m translations + m forward passes per text pair. With 10+ languages, this is 10x the cost of single-pass methods.
  • Pair-wise only: Produces similarity scores, not reusable embeddings. Unsuitable for indexing, retrieval, or clustering at scale.
  • No embedding baseline: Explicitly excludes trained embedding models (SimCSE, BGE, E5), leaving the practical performance gap unmeasured.

1

Multilingual Consensus

Translate text into m languages, extract cumulative attention outputs for each, identify dimensions that are positive across all language versions. Retain top-k as semantic component set S(x).

2

Joint Semantic Set

For a pair (x,y), construct U(x,y) = S(x) ∪ S(y). This preserves dimensions important to either text, capturing both overlap and difference.

3

Subset Cosine Similarity

Index into U(x,y) for both texts, compute cosine similarity on this dynamic subset. No training, no fine-tuning.

S Assessment

Strengths

  • Elegant core idea: multilingual invariance as a probe for semantic content
  • Strong empirical results across 3 model families, base and instruct
  • Thorough ablations: representation, language, prompt strategy, k value
  • Training-free with meaningful dimensionality reduction
  • Key finding: attention outputs > hidden states for STS
  • Code released, high reproducibility

Limitations

  • 10x+ inference cost from multi-language forward passes
  • Pair-wise scores only, not reusable embeddings
  • External translator dependency, uncontrolled variable
  • No downstream task evaluation (retrieval, clustering)
  • No comparison with dedicated embedding models
  • Optimal language pool varies per model
