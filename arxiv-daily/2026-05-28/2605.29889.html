Internal Representation, Not Clinical Knowledge: Where Apparent LLM Triage Failures Originate

David Fraile Navarro·Berardino Como·Jialei Sheng·Soundariya Ananthan·Shlomo Berkovsky
Macquarie University · Politecnico di Bari · NSW Health · 2026-05-28
LLM clinical triage failures are not knowledge gaps. Medical features encode correctly at 98.3–100% anchor rates across formats, but go completely silent at the multiple-choice decision token—scaffold features hijack the output instead. The failure is in the output-stage format mapping, not clinical representation.
Key Metric
98.3–100%
Medical feature peaks anchored in shared vignette tokens under both NL and NF formats—all three models, every layer
The Flip
0 / 60
Medical features active at the NL decision token across all 60 cases. Zero. Scaffold features dominate instead.
Why This Matters
Nature Medicine's 51.6% under-triage is an artifact
Forced-letter MCQ evaluation, not clinical incompetence. Benchmarks need redesigning.
Critical Edge
Diagnosis without cure
SAE interventions change 0–2 of 60 predictions. Format penalty inverts at larger models (12B, Qwen prefer NL).

Exhaustive Option-Order Shuffle (1,380 passes/model)
4B Result
Shuffling letter-to-content mappings raises NL accuracy from 55.0% → 69.8%—statistically indistinguishable from NF's 71.7%. The entire gap attributed to answer-key-layout × content-prior interaction.
Position Bias
Ruled out: same-letter rates at or below chance. Content prior is strong: 64–83% same-content across all models regardless of letter assignment.
Methodology Pipeline
1 SAE Feature ID 3 features/model/layer via contrastive activation, validated with selectivity filters
2 Invariance Test Max-pooled SAE activations compared NL vs NF (sMAPE, cosine sim, bootstrap CI)
3 Decision Token NLA verbalization + logit attribution + top-20 feature characterization
4 Decomposition 1,380 shuffle passes + 5-way adjudication + linear probes (AUC 0.77–0.80)
NLA Verbalization: The Representational Flip
At Gemma 12B L32 across 7 token positions, NLA verbalization reveals that content tokens are consistently medical-primary under both NL and NF. But at the NL decision token, all 60 cases flip to scaffold-primary / medical-partial. LLM judges agree 96.7%/96.2% on medical/scaffold axes. Clinical content is demoted from "primary subject" to "partial mention" while the multiple-choice scaffold becomes the dominant frame.
Linear Probes & Constraint-First Ablation
L2-regularized logistic regression on last-token hidden states predicts format-flipping behavior with AUC = 0.769–0.804 (p = 0.004–0.010). The Constraint-First ablation—moving formatting instructions to the prompt start—selectively modifies late-layer signal for NL-sourced transitions, confirming that constraint position relative to the generation point contributes substantially to the representational shift.
Clinician Calibration
Cohen's kappa = 0.808 between LLM judges and blinded clinician on 5-class labels (A/B/C/D/deferred). Clinician deferral rate (37.5%) far exceeds model rate (6.7%), suggesting models under-defer relative to clinical practice. 16-case stratified sample used for adjudication.
Verdict
Novelty
High. First SAE mechanistic interpretability applied to a real clinical evaluation problem, going beyond behavioral analysis to localize the format effect.
Rigor
Strong. Multiple independent methods cross-validate. Exhaustive shuffle (1,380/model). Magnitude-matched baselines. Clinician calibration (κ=0.808).
Significance
High. Directly challenges Nature Medicine headline finding. Immediate implications for clinical AI benchmark design.
Limitations
  • n=60 vignettes—several behavioral strata contain very few cases
  • Only greedy decoding evaluated; sampling could differ
  • Qwen-Scope ~34% reconstruction error raises noise floor
  • Features are content detectors, not acuity probes—proves medical content is preserved, not correct triage
  • SAE interventions: 0–2 of 60 predictions changed—no practical fix
  • Format penalty inversion at larger models acknowledged but unexplained
  • NLA limited to Gemma 12B only
Experimental Setup

Corpus: 60 patient-voiced triage vignettes (2×2 factorial: input format × output format)

Models:

Gemma 3 4B IT Gemma 3 12B IT Qwen3-8B

SAEs: Gemma Scope 2 (dSAE=16,384) · Qwen-Scope (dSAE=65,536)

Reproducibility: Code, data, and analysis scripts publicly available. All models open-weight.

Open Questions
  • Can format-agnostic protocols (open-ended acuity, ranked preference) avoid scaffold artifacts?
  • What causes the format penalty to invert at 12B/Qwen?
  • Would contrastive format-invariance training resolve the decision-token mode shift?
  • Can richer interventions (multi-layer steering, representation engineering) overcome the intervention nulls?
