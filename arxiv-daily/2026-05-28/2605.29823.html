Quantifying and Optimizing Simplicity
via Polynomial Representations

Tianren Zhang, Xiangxin Li, Minghao Xiao, Guanyu Chen & Feng Chen — Tsinghua University · 2026

A function-space simplicity metric—Effective Degree—that fits Chebyshev polynomial surrogates along data interpolation paths, predicts generalization better than sharpness, and yields a differentiable regularizer improving performance across vision, language, VLM fine-tuning, and RL.

Headline Claim
Effective Degree
A coefficient-weighted average polynomial degree that quantifies function-space simplicity, inherently invariant to reparameterization.
Anchor Fact
+3.02pp
ViT-Tiny CIFAR-10: 90.82% vs 87.80% baseline. SAM achieves only +0.05. Consistent gains across all four tested domains.
So What
One Config
A single default hyperparameter setting (tuning only lambda) generalizes across ViT/ResNet, BERT, CLIP, and ProcGen RL.
Critical Edge
Weak Baselines?
SAM/ASAM/Jacobian all sit at ~87.8%—suspiciously close to baseline. Overhead from extra forward passes. LLM-scale untested.

Method Core Pipeline

1 Reduce multivariate network f to 1D: g(α) = f(αx1 + (1-α)x2) along linear interpolation between data pairs.
2 Fit a degree-K Chebyshev polynomial surrogate P(α) via least-squares on r stratified Chebyshev-node samples.
3 Compute Effective Degree as coefficient-weighted average degree. Apply PCA to high-dim outputs before fitting.
ED(P) = ∑k |ck| · k    |   L(θ) = Lce + λ · EDbatch
4 Gradient is fully analytic via damped least-squares: dED/dy = T(TTT + εI)-1(sign(c) · d). Label-anchoring fixes boundary predictions to ground-truth labels for classification.

Strengths

  • Reparameterization-invariant by construction (function-space, not weight-space)
  • Clean theory: Theorem 3.1 proves degree preservation on random paths a.s.
  • Consistent gains across vision, NLP, VLM, RL with one config
  • Comprehensive ablations on every design choice
  • Label-anchoring cleverly resolves cross-entropy optimization conflict

Concerns

  • SAM/ASAM baselines suspiciously weak at ~87.8% on ViT-Tiny CIFAR-10
  • Extra forward passes along interpolation paths add training cost
  • BERT improvements modest (0.84–0.92pp); embedding mixup hurts some tasks
  • LLM / large-ViT scalability entirely untested
  • No formal generalization bound—only empirical correlation shown
