Relational Rank Geometry in Transformers

Detecting and steering hidden-state relation frames via Plucker sign entropy across Llama 8B – 405B. A single-author work that bridges algebraic geometry to mechanistic interpretability.

Detection — Plucker Sign Entropy

  1. Extract hidden states, project to 64-dim analysis subspace via SVD (seed 42).
  2. Enumerate ordered token tuples for r-argument relations using argument-only (rank k=r) and predicate-plus-argument (k=r+1) constructors.
  3. Compute determinant sign of each tuple’s projected matrix — encodes subspace orientation.
  4. Plucker sign entropy H_pm(T) measures sign consistency across tuples.
  5. Statistic D(r,k) = H_pm(scrambled) − H_pm(true): positive ⇒ true tuples have more consistent orientation.
  6. Diagonal margin compares D at expected rank vs. max over other ranks; held-out validation with even/odd prompt splits.

Steering — Edge-Grid Intervention

8×8 grids with A-row/B-column tokens and YES/NO edge markers. Clean prompts use identity map; corrupt prompts use duplicate-target map sharing the same scaffold.

  • Patch layer 5, readout at layer 35.
  • 21 alpha points in [0, 1], 32 prompts, proj dim 64.
  • Recovery: R_beh (logit gap), R_res (blade cosine), R_coup = R_beh × R_res.

Limitations & Open Questions

Scope
  • 32 prompts, synthetic 8×8 edge-grid only
  • Fixed patch layer 5 / readout layer 35, no sweep
  • 70B and 405B run under 4-bit quantization
  • 8B fails competence threshold (boundary case)
Open Questions
  • Does this transfer to natural-language reasoning?
  • What upstream circuits construct relation frames?
  • Relation to sparse autoencoder features?
  • Non-Llama architecture transfer?

Significance is contingent on generalization beyond the edge-grid paradigm. The paper explicitly does not localize upstream/downstream circuits.
