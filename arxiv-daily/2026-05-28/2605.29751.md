> **一句话总结：** DySem提出一种免训练的STS计算框架，通过多语言共识从LLM内部提取与语义最相关的注意力累积向量维度，并为每个文本对动态构建联合语义维度子集来计算余弦相似度，在多个LLM和STS基准上一致超越现有方法，且仅使用约1000维而非全量隐藏维度。

> **TL;DR:** DySem is a training-free framework that computes semantic textual similarity (STS) by identifying cross-linguistically consistent dimensions in LLM internal states. It translates each text into multiple languages, extracts the cumulative attention outputs ($A_n^L$), identifies dimensions co-activated across all translations (multilingual consensus), selects the top-k as a per-text semantic component set, then constructs a pair-specific joint semantic set via union and computes cosine similarity over this dynamic subset. Across 10 LLMs and 7 STS benchmarks, DySem consistently outperforms PromptEOL, AlignedWVA, and other baselines while using roughly 1,000 dimensions instead of full hidden states (e.g., 4096). A key finding is that cumulative attention outputs $A_n^L$ outperform final-layer hidden states $h_n^L$ in 61 out of 70 comparisons for semantic similarity tasks.

## 核心贡献

- **多语言共识提取语义维度：** 利用"文本核心语义在翻译后不变"的直觉，将文本翻译为多种语言，通过LLM前向传播提取各语言版本的内部表示，找出在所有语言中一致激活的维度作为语义相关的维度子集。
- **动态样本特异性相似度计算：** 为每对文本动态构建联合语义维度集合（两个文本各自语义维度的并集），在该子集上计算余弦相似度，避免了全量隐藏维度的冗余和噪声。
- **发现注意力累积输出优于隐藏状态：** 系统比较了最终层隐藏状态 $h_n^L$、累积FFN输出 $F_n^L$、累积注意力输出 $A_n^L$ 三种表示，发现 $A_n^L$ 在70个对比中61次表现最优，揭示了注意力表示在语义相似度计算中的优势。
- **免训练且维度更低：** 整个框架无需任何模型训练或微调，仅使用约1000维即可获得优于全维基线的效果。

## Key Contributions

- **Multilingual consensus for semantic component extraction:** Translates text into m target languages, extracts internal state vectors for each, and identifies dimensions consistently activated (positive) across all language versions, filtering out language-specific surface artifacts.
- **Dynamic, pair-specific joint semantic sets:** For each text pair, constructs $U(x,y) = S(x) \cup S(y)$ -- the union of each text's top-k semantic dimensions -- and computes cosine similarity only on this flexible subset, enabling sample-adaptive similarity computation.
- **Attention outputs outperform hidden states for STS:** A systematic ablation across 10 models shows cumulative attention outputs ($A_n^L$) beat final-layer hidden states ($h_n^L$) in 61/70 comparisons and FFN outputs ($F_n^L$) never rank first, challenging the default practice of using last-layer hidden states.
- **Training-free with reduced dimensionality:** No model training or fine-tuning required. Achieves strong results using approximately 1,000 dimensions on average versus the full hidden size (e.g., 4096 for 8B models).

## Methodology

DySem operates in three stages:

**Stage 1 -- Extracting Semantic Components via Multilingual Consensus (Section 3.1):**
- Given text $x$, translate it into $m$ target languages $\mathcal{L} = \{\ell_1, \dots, \ell_m\}$ using a translator, yielding $x^{(\ell)} = T_\ell(x)$.
- Wrap each language version in a prompt template and pass through the LLM to extract internal state vectors $\mathbf{s}^{(\ell)}(x) \in \mathbb{R}^d$. Three internal components are considered: final hidden states $h_n^L$, cumulative FFN outputs $F_n^L$, and cumulative attention outputs $A_n^L$.
- Define the multilingual consensus set as dimensions positive across all languages: $\mathcal{I}(x) = \{j \mid s_j^{(\ell)}(x) > 0, \forall \ell \in \mathcal{L}\}$.
- Rank dimensions by mean activation across languages and retain top-k: $\mathcal{S}(x) = \text{TopKIdx}(\{\bar{s}_j(x)\}_{j \in \mathcal{I}(x)})$.

**Stage 2 -- Constructing the Joint Semantic Set (Section 3.2):**
- For a text pair $(x, y)$, construct the joint semantic set as the union: $\mathcal{U}(x,y) = \mathcal{S}(x) \cup \mathcal{S}(y)$.
- This preserves dimensions important to either text, capturing both overlap and difference.

**Stage 3 -- Calculating STS by Joint Semantic Dimensions (Section 3.3):**
- Construct text-specific joint semantic representations by indexing into $\mathcal{U}(x,y)$: $\mathbf{v}_{\mathcal{U}}(z) = \mathbf{v}(z)|_{\mathcal{U}(x,y)}$.
- Two vector variants: source-language vector or multilingual mean vector.
- Compute cosine similarity: $\text{sim}(x,y) = \cos(\mathbf{v}_{\mathcal{U}}(x), \mathbf{v}_{\mathcal{U}}(y))$.

**Prompt strategies:** English Prompt (same English template for all languages) and Language-Specific Prompt (template translated to each target language).

**Models tested:** 10 LLMs from three families (LLaMA, Qwen, Phi) in both base and instruction-tuned variants.

**Benchmarks:** 7 standard STS benchmarks (STS12-STS16, STS-B, SICK-R), evaluated using Spearman correlation.

## Results & Highlights

- **Consistent superiority across models:** DySem outperforms all baselines (PromptEOL, LLM2Vec, AlignedWVA, SemPa, ContrastivePrompting) across both base and instruction-tuned models. On LLaMA3.1-8B-it, DySem achieves 80.59 average, outperforming AlignedWVA by 5.24 points and PromptEOL by 8.03 points. On Qwen3-8B-it, DySem scores 80.16, surpassing AlignedWVA by 6.17 points.
- **Larger gains on instruction-tuned models:** Average improvement over AlignedWVA is 4.55 points on instruct models vs. 3.82 on base models, suggesting DySem better filters non-semantic variation introduced by instruction tuning.
- **Semantic components vs. random components:** DySem's semantic components (avg score 78.24-79.04) vastly outperform random dimension selection (avg 52.70) across all 70 model comparisons, confirming gains come from meaningful semantic identification.
- **$A_n^L$ dominates other representations:** Cumulative attention outputs win in 61/70 comparisons. $F_n^L$ never ranks first. The only exceptions come from Qwen2.5 family where $h_n^L$ is occasionally competitive.
- **English is the strongest individual language** for consensus, with French and Spanish typically near the top. Language pool selection is model-dependent.
- **Dimensionality reduction:** DySem achieves competitive results using roughly 1,000 dimensions on average, compared to the full hidden size (e.g., 4096 for 8B models).

## Critical Assessment

**Novelty (High):** The idea of using multilingual consensus to identify semantic dimensions within LLM internal states is genuinely novel. The discovery that cumulative attention outputs outperform final hidden states for STS is a significant finding that challenges prevailing practice. The dynamic, pair-specific dimension selection is a clean conceptual contribution.

**Rigor (Moderate-High):** The paper conducts thorough ablations across 10 models, 7 benchmarks, and multiple design choices (representation type, language selection, prompt strategy, k value). However, several concerns remain:
- The reliance on an external translator adds an uncontrolled variable -- the quality of translations directly affects the consensus set, but this is not systematically analyzed.
- The O(m) inference overhead (m translations + m forward passes per text) is a significant practical cost that limits scalability, acknowledged but somewhat downplayed.
- Comparison with dedicated embedding models (e.g., SimCSE, BGE) is explicitly excluded, which is understandable given the different setting but leaves a gap in understanding relative performance.
- The paper does not evaluate on downstream tasks (retrieval, clustering), only on direct STS benchmarks.

**Reproducibility (High):** Code is released. The method is training-free and the experimental setup (models, benchmarks, metrics) is standard. The hyperparameter k and language set selection are well-documented.

**Significance (Moderate-High):** The finding that attention outputs are better than hidden states for semantic similarity could influence future work on LLM-based text representation. The multilingual consensus mechanism is elegant. However, the practical applicability is somewhat limited by the inference cost and the focus on pair-wise STS rather than embedding generation.

**Red flags:**
- The O(m) inference cost is a real concern for practical deployment. With m=10+ languages, this is 10x the cost of single-pass methods.
- No comparison with trained embedding models means we cannot assess the gap between this training-free approach and state-of-the-art embeddings on STS.
- The method produces pair-wise similarity scores rather than reusable embeddings, limiting its applicability to tasks like search/indexing.

## Strengths

- Elegant and well-motivated core idea: multilingual invariance as a probe for semantic dimensions.
- Strong empirical results across diverse model families (LLaMA, Qwen, Phi) and both base/instruct variants.
- Thorough ablation studies covering representation selection, language strategies, prompt strategies, and k values.
- Training-free: no additional model training or fine-tuning required.
- Meaningful dimensionality reduction (~1,000 dims vs. 4,096+) with improved performance.
- Important finding that cumulative attention outputs $A_n^L$ consistently outperform final hidden states for STS.
- Code is publicly available.

## Limitations

- **Computational overhead:** Requires m translations and m LLM forward passes per text, making it expensive for large-scale applications. The authors acknowledge this but frame it as primarily an offline cost, which may not always hold.
- **Pair-wise only:** The method computes similarity between text pairs but does not produce reusable single-text embeddings, making it unsuitable for tasks requiring pre-computed indexes (e.g., large-scale retrieval).
- **External translator dependency:** Relies on a translation system whose quality and language coverage affect results. No analysis of how translation quality impacts performance.
- **Limited task scope:** Evaluated only on STS benchmarks, not on downstream tasks like retrieval, clustering, or classification.
- **No comparison with embedding models:** Explicitly excludes comparison with dedicated embedding models (SimCSE, BGE, E5, etc.), leaving an open question about relative performance.
- **Language selection is model-dependent:** The optimal language pool varies across models, adding a tuning burden.

## Relevance

本文对LLM内部语义表示的探索提供了一个新颖视角，特别是利用多语言一致性来识别语义维度的方法具有启发性。对于关注LLM内部机制解释、注意力机制功能分析以及语义表示质量评估的研究者具有较高参考价值。累计注意力输出优于最终隐藏状态的发现也为LLM表示学习研究提供了新的方向。

## Questions for Further Exploration

- How does translation quality affect the multilingual consensus? Would noisy or paraphrase-based alternatives (instead of proper translation) yield similar semantic dimensions?
- Can the identified semantic dimensions be distilled into a fixed projection that approximates DySem's performance without requiring m forward passes at inference time?
- How do the semantic components identified by DySem relate to features discovered by sparse autoencoders or other mechanistic interpretability methods?
- Does the superiority of cumulative attention outputs $A_n^L$ over hidden states $h_n^L$ extend to other semantic tasks beyond STS (e.g., natural language inference, paraphrase detection)?
- Can DySem's dynamic dimension selection be adapted to produce single-text embeddings rather than only pair-wise similarity scores?
- What is the minimal number of languages needed to achieve most of the consensus benefit? Is there a sweet spot between coverage and cost?
