Understanding Geometric Representations in Self-Supervised Vision Transformers via Subspace Intervention

arXiv: 2607.01987

Date: 2026-07-03

https://arxiv.org/abs/2607.01987

一句话总结： 本文提出受控子空间干预框架 Geosubprobe——对收敛后的线性探针权重做 SVD 提取任务对齐低秩子空间，再投影特征到对齐/随机/正交残差子空间并用固定线性头评估，从而把"几何信息是否存在"与"是否线性可解码"解耦；在 DINOv2-L/iBOT-L/MAE-L 的 NYU Depth V2 单目深度（SA-δ1）评测中发现：DINOv2 几何信号高度对齐可压缩（Linear 0.9157，Local Entangl 仅 +0.0168），MAE 则分散（Linear 0.6033，Spatial Frag. +0.0632），且几何精度在中间层峰值、末层让位于语义抽象。
TL;DR: The paper introduces a controlled subspace intervention framework to diagnose how self-supervised ViTs encode dense geometric information. It decomposes converged linear-probe weights via SVD to isolate task-aligned low-rank subspaces, then projects frozen features onto (i) the aligned subspace S_k, (ii) a random subspace R_k, and (iii) the orthogonal residual S_k⊥, evaluating each via a fixed linear head — decoupling "is the geometry present" from "is it linearly readable." Three-tier probing (Linear / 1×1 MLP / DPT) quantifies a readability gap split into Local Entanglement (point-wise non-linear folding) and Spatial Fragmentation (global-context reliance). On NYU Depth V2 (SA-δ1 ↑) surface-normal estimation: DINOv2-L Linear 0.9157 / MLP 0.9325 / DPT 0.9483 (Local Entang +0.0168, Spatial Frag +0.0158 — highly aligned); iBOT-L 0.8198 / 0.8376 / 0.8524; MAE-L 0.6033 / 0.6390 / 0.7022 (Local Entang +0.0357, Spatial Frag +0.0632 — dispersed). Key findings: (1) pre-training objective determines encoding (DINOv2 aligns, MAE disperses); (2) explicit geometric representations are highly compressible into low-rank subspaces across all paradigms; (3) geometric precision peaks at intermediate layers, yielding to semantic abstraction in final layers.

核心贡献

  1. 受控子空间干预框架：对收敛的线性探针权重做 SVD 提取任务对齐基 V_k，把特征投影到对齐子空间 S_k / 随机子空间 R_k / 正交残差 S_k⊥，用固定线性头评估——把"信息存在"与"线性可解码"解耦。
  2. 三层探针可读性差距：Linear → 1×1 MLP（局部非线性折叠 Local Entanglement）→ DPT（全局空间碎片化 Spatial Fragmentation），分别量化逐点非线性缠绕与跨 token 空间聚合需求。
  3. 预训练目标决定编码格式：DINOv2（自蒸馏）把几何对齐到高度可压缩的低秩坐标系；MAE（生成式掩码重建）为满足像素级重建把几何信号分散到更宽维度。
  4. 几何精度中间层峰值 + 高可压缩性：所有范式的显式几何表示都高度可压缩到低秩子空间；层间任务亲和性显示几何在中间层最强、末层让位于语义抽象。

Key Contributions

  • Controlled subspace intervention framework: SVD on converged linear-probe weights isolates task-aligned directions; projecting frozen features onto aligned (S_k) / random (R_k) / orthogonal-residual (S_k⊥) subspaces and evaluating via a fixed linear head isolates the geometric signal from black-box output metrics.
  • Three-tier probing readability gap: Linear → 1×1 MLP → DPT splits the gap into Local Entanglement (point-wise non-linear folding) and Spatial Fragmentation (reliance on global spatial context), quantifying alignment vs dispersion.
  • Pre-training objective determines encoding: self-distillation (DINOv2) aligns geometry into compressible low-rank coordinates; masked reconstruction (MAE) disperses it across broader dimensions to satisfy pixel-level constraints.
  • Compressibility + layer-wise task affinity: explicit geometry is highly compressible into low-rank subspaces across all paradigms; geometric precision peaks at intermediate layers and yields to semantic abstraction in final layers.

Methodology

Diagnostic framework (§2). Two stages: (A) readability gap — three-tier probing on frozen backbone features $Z^{(l)} \in \mathbb{R}^{N\times D}$ (N patches, D feature dim): Tier-1 Linear Probe (1×1 conv, $Z^{(l)} W^{(l)\top}$, explicit isolated geometry); Tier-2 1×1 MLP Probe (point-wise non-linearity with GELU, same 1×1 token-wise receptive field → isolates local non-linear folding); Tier-3 DPT Decoder (multi-scale feature aggregation + global spatial receptive field → absolute latent geometric upper bound). Gaps: Linear→MLP = Local Entanglement; MLP→DPT = Spatial Fragmentation. (B) Subspace intervention — apply SVD to the global concatenated probe weight matrix $W_{global}$ to extract task-aligned basis $V_k$ of rank k; project features onto S_k (aligned), R_k (random), S_k⊥ (orthogonal residual); evaluate each via the fixed linear head.

Setup (§3.1). Three pre-training paradigms, ViT-Large: DINOv2-L (self-distillation), iBOT-L (hybrid), MAE-L (masked image modeling). Primary task: monocular depth estimation on NYU Depth V2 (AbsRel metric; reported as SA-δ1). Parallel probing for 40-class semantic segmentation (mIoU) and surface normal estimation to study layer-wise task affinities. Supplementary adds depth on KITTI, depth/normal on NAVI, and ViT-Base variants.

Analytical roadmap (§3.3). (1) readability gap via tiered decoders; (2) SVD compressibility + cross-layer energy routing; (3) rank sensitivity via single-layer subspace interventions; (4) geometric vs semantic probing (capacity loss vs task-affinity transition); (5) subspace stability across random probe initializations.

Results & Highlights

  • Three-tier readability gap (Table 1, NYU Depth V2, SA-δ1 ↑):
  • DINOv2-L: Linear 0.9157 / MLP 0.9325 / DPT 0.9483 — Local Entanglement +0.0168, Spatial Fragmentation +0.0158 (geometry highly aligned, minimally entangled).
  • iBOT-L: 0.8198 / 0.8376 / 0.8524 — Local +0.0178, Spatial +0.0148.
  • MAE-L: Linear 0.6033 / MLP 0.6390 / DPT 0.7022 — Local Entang +0.0357, Spatial Fragmentation +0.0632 (geometry dispersed; needs broad spatial context — DPT recovers much more than linear).
  • Pre-training objective drives encoding format: DINOv2 (self-distillation) → aligned, compressible, low-rank; MAE (masked reconstruction) → dispersed across broader dimensions (pixel-level reconstruction constraint scatters geometry); iBOT (hybrid) sits between.
  • High compressibility across all paradigms: explicit geometric representations concentrate into low-rank task-aligned subspaces — dense predictive heads could be constrained to low-rank subspaces with minimal performance loss (motivating lightweight decoder design).
  • Layer-wise task affinity: geometric precision (surface normal / depth) peaks at intermediate layers, then weakens in final layers as semantic abstraction (segmentation) takes over — a capacity-vs-task-affinity transition, not pure capacity loss.
  • Subspace stability: the extracted task-aligned subspaces are stable across random probe initializations (§4.5), supporting their interpretability as genuine task-aligned directions rather than probe artifacts.

Critical Assessment

  • The DINOv2-vs-MAE encoding-format contrast is the strongest finding, and the readability-gap decomposition (Local Entanglement vs Spatial Fragmentation) is a clean diagnostic that genuinely distinguishes "geometry absent" from "geometry present but dispersed" — addressing the paper's stated motivation. However, the SA-δ1 (surface-normal) numbers are reported as the primary metric while depth (AbsRel) is mentioned as the primary task in §3.1 — a slight metric-task inconsistency the reader should disentangle when citing specific numbers.
  • The "highly compressible to low-rank subspaces" claim is well-supported by the SVD energy-routing analysis, but the paper does not report the exact rank k at which performance saturates for each paradigm in the parsed sections — the qualitative "highly compressible" finding would be tighter with a per-paradigm rank-vs-performance table (the figure shows the curve but the parsed text does not extract specific k values).
  • The layer-wise peak-at-intermediate-layers finding is consistent with prior ViT interpretability work (feature granularity vs semantic abstraction tradeoff), so it is conceptually expected rather than surprising; the contribution is the geometric-specific instantiation and the task-affinity-vs-capacity-loss distinction.
  • Three ViT-L backbones only (DINOv2/iBOT/MAE); whether the encoding-format taxonomy (aligned vs dispersed) generalizes to other paradigms (e.g., MoCo v3, BEiT, DINOv3, or non-ViT backbones like MambaViT) is unverified.
  • The framework is diagnostic, not prescriptive: it tells you how geometry is encoded but does not propose a new pre-training objective or decoder that exploits the finding (lightweight low-rank decoder design is suggested as future direction, not delivered).

Strengths

  • Clean SVD-on-converged-probe-weights subspace intervention — a principled way to isolate task-aligned directions without retraining, going beyond black-box probe accuracy.
  • The Local-Entanglement-vs-Spatial-Fragmentation decomposition is a genuinely useful diagnostic axis.
  • DINOv2-vs-MAE encoding-format contrast is well-motivated and empirically clean.
  • Public code (Geosubprobe) and standard benchmark (NYU Depth V2) support reproducibility.

Limitations

  • Metric-task inconsistency (SA-δ1 surface normal as primary reported, AbsRel depth as primary task stated) — disentangle when citing.
  • Exact rank-k saturation points per paradigm not in the parsed text; compressibility claim would be tighter with a table.
  • Layer-peak finding is conceptually expected; contribution is the geometric-specific instantiation.
  • Three ViT-L backbones only; generalization to other paradigms/backbones unverified.
  • Diagnostic only; no new pre-training objective or decoder delivered (suggested as future work).

Questions for Further Exploration

  1. SVD 子空间干预框架能否迁移到 LLM 隐藏状态——对线性探针权重做 SVD 来分离"任务对齐方向"与"正交残差"，量化表征的线性可压缩性？
  2. DINOv2 对齐 vs MAE 分散的编码格式差异，在 DINOv3 / BEiT / MoCo v3 等更新的自监督范式上是否仍然成立？是否存在"对齐度"随预训练目标设计的演化趋势？
  3. 几何可压缩性的精确 rank-k 饱和点（每范式）是多少？能否据此给出"低秩解码器不损性能"的 rank 上界建议？
  4. 中间层几何峰值的层位置在不同 ViT 深度（ViT-B vs ViT-L vs ViT-H）上是否对应同一相对深度？还是随深度漂移？
  5. Local Entanglement vs Spatial Fragmentation 的分解能否用于指导解码器架构选择（高 Local Entang → MLP 够用；高 Spatial Frag → 必须 DPT 式全局聚合）？

  7. 该诊断框架能否用于监测预训练过程中几何表征的"对齐化"动态——即几何何时从分散收敛到对齐？
