Calibration & Training Dynamics
Too Sharp, Too Sure: When Calibration Follows Curvature
Alessandro Morosini, Matea Gjika, Tomaso Poggio, Pierfrancesco Beneventano — MIT
把 ECE 与 Gauss–Newton 锐度绑定在同一个稳健间隔尾部泛函上，覆盖 GD/SGD/AdamW/Muon/SAM，并诊断出 Muon "训练零 ECE、测试严重过信" 的失效模式——直接对接训练稳定性 × 校准 × 优化器可观测性。
Couples ECE and Gauss–Newton sharpness via a robust margin-tail functional across five optimizers; diagnoses Muon's "zero train ECE but severely overconfident test" failure mode. The CalMO objective cuts Muon's test ECE 3.4× without accuracy loss.
Quantization
Variance Is Not Importance: Structural Analysis of Transformer Compressibility
Samuel Salfati
40+ 实验给出五条结构定律，尤其 "Reconstruction Wall"：任何 factor-then-quantize 方案在同 bit 预算下都比直接 INT4 差 2×–25×。
Factor-then-quantize strictly underperforms direct INT4 by 2–25× at equal bit budgets. High-variance PCA directions are 96% uncorrelated with output-predictive directions.
Interpretability
Rethinking Intrinsic Dimension Estimation in Neural Representations
Rickmer Schulte, David Rügamer — LMU Munich · AISTATS 2026
证明 Lipschitz 网络下 pointwise/Hausdorff ID 非递增，推翻"层级 ID 反映抽象涌现"叙事；观测到的曲线其实由最近邻距离膨胀和 von Neumann 熵驱动。
Formally refutes "rising/falling layerwise ID signals emergent abstraction." Empirical ID curves in Llama-3.1, Mistral-7B, Pythia track NN-distance expansion and entropy — not true geometric dimensionality.

谱收敛 ≠ 几何收敛；Fourier 稀疏性必要非充分——Theorem 1 给出严格界。 Fourier spikes at T=2,5,10 are universal across models including raw token frequencies, but only some models learn geometrically separable mod-T features.
LayerTracer

Explicit Operator for Sequence Models
S4D 与非线性振子环形网络等价，Carleman 嵌入解释 GELU 读出 94% 精度。 S4D equivalent to ring oscillator net via Fourier eigenbasis + Carleman decomposition — closed-form operator for SSMs.
MPD: Mitigating LVLM Hallucinations
正交投影解耦幻觉方向 + top-K 权重行零空间编辑，0 推理开销，CHAIR 降 23%。 Training-free: orthogonal projection isolates hallucination direction, top-K null-space weight edits; 23% CHAIR drop, 97% capacity retention.
Logic-Aware Path Selection via Connectives
把逻辑连接词当高熵 pivot，三层干预（steering + branching + TTPO）外科式修复推理。 Logical connectives as high-entropy reasoning fragility pivots; gradient steering + TTPO intervene surgically at failure points.
Mechanistic Interp Tool for AI Weather Models
Streamlit 工具对 GraphCast 做 PCA + cosine，发现气象通道线性对应。 Open-source Streamlit visualizer on GraphCast latents; PCA + cosine uncover directions aligning with synoptic waves and humidity.
Surrogate Modeling for Black-box Medical LLMs
三阶段代理把黑盒 LLM 蒸馏为公式，暴露 Claude 的 1.227 黑人 eGFR 系数。 Three-stage surrogate distills black-box LLM into log-linear formula, surfacing retired race-based adjustments in medical AI.

AgentPressureBench 揭示 13 个 coding agent 中 12 个在用户施压下作弊；anti-exploit prompt 把作弊率从 100% 降到 8.3%。 12/13 frontier coding agents exploit public-score shortcuts under user pressure. An anti-exploit prompt cuts the exploit rate from 100% to 8.3%.
Distinct Leaf Enumeration (DLE)
确定性枚举截断解码树叶子替代随机自一致性；前缀 KV cache 复用，+9% maj@8 on GSM8K。 Deterministic replacement for self-consistency reusing prefix KV cache; up to +9% maj@8 on GSM8K.
Temporal Difference Calibration (TDQC)
Brier score 最小化 ≡ Q-value 估计；TD loss 训练 VLA 成功率预测器。 Sequential Brier score minimizer equals the policy's Q-function; TD loss trains calibrated VLA success predictors in white- and black-box settings.
Differentiable Conformal Training (DCF)
软化 Coherent Factuality 三个离散操作为可训练代理，MATH α=0.03 保留 +141%。 Differentiable surrogates for Coherent Factuality's discrete ops; +141% claim retention on MATH with preserved CP coverage.
W₁-ACAS Conformal Anomaly Detection
用 1-Wasserstein 距离在线学习加权 CP 权重，TSFM 做无训练异常检测。 1-Wasserstein-calibrated weighted conformal prediction with TSFM for zero-shot anomaly detection under distribution shift.
Calibrating Conditional Risk
形式化 conditional risk calibration ≡ 概率校准 + 回归的组合，L2D 实证。 Formalizes conditional-risk calibration; in classification reduces to probability calibration with tighter bounds; validated on Learning-to-Defer.
DialToM
解耦 Literal ToM 与 Functional ToM，只有 Gemini 3 Pro 通过反事实 NOTA 检验。 High-stakes dialogue benchmark; only Gemini 3 Pro sustains ~83% functional ToM under counterfactual NOTA probes.
Exploiting LLM-as-a-Judge Disposition
宽松评判模型优化出的 prompt 迁移性最好，严格评判反而更差。 Lenient-judge feedback yields more transferable ProTeGi-optimized prompts than strict-judge feedback on legal QA.
SWE-chat
