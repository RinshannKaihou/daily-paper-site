Seeing Further and Wider: Joint Spatio-Temporal Enlargement for Micro-Video Popularity Prediction

arXiv: 2604.20311v1

Date: 2026-04-22

https://arxiv.org/abs/2604.20311v1

一句话总结： 作者提出 STAP 框架，用帧打分 + SSM 稠密/稀疏扫描 + 内容自适应稀疏注意力处理 100 帧长视频（Temporal Enlargement），并构造按 popularity 分层、按 topic 聚类的 Topology-Aware Memory Bank（Spatial Enlargement）——在 MicroLens、SMPD-video、Informs 三个基准上相较 11 个强基线分别提升 nMSE 2.15%/35.79%/14.66%。
TL;DR: STAP tackles micro-video popularity prediction by jointly enlarging temporal context (up to 100 frames via Frame Scoring + SSM dense/sparse scan + adaptive Sparse Attention + gated fusion) and spatial retrieval (Topology-Aware Memory Bank with P×C prototype grid trained via load-balance + DPPO ranking loss). Consistent wins on MicroLens, SMPD-video, and Informs; complexity O(Tω̄+PC) vs O(TT_w+N) for traditional pipelines.

核心贡献

  • 提出同时建模长时 (100 帧) + 拓扑结构历史检索的统一流程 STAP。
  • Temporal Enlargement 用 SSM+稀疏注意力解耦稠密/稀疏关注，配合 anchor 驱动的 ∆_t 自适应步长。
  • Spatial Enlargement 使用 P×C 原型网格内存 bank，不随训练样本线性膨胀，支持 cold-start。
  • 提出 DPPO (Dynamic Pairwise Preference Optimization) 对齐检索偏好与 SRC 排序指标。

Key Contributions

  • First unified spatio-temporal enlargement framework for MVPP breaking the negative coupling between temporal encoder and flat retrieval bank.
  • Frame Scoring Module (lightweight MLP) producing importance weights and a global anchor.
  • Load Balance + DPPO blocks mitigating memory-slot collapse and aligning routing with ranking supervision.
  • Complexity O(Tω̄+PC) with space O(PC).

Methodology

Three-stage pipeline: (1) Temporal Enlargement: Frame Scoring → selective-scan SSM block (dense/sparse ∆_t adaptation) + Sparse Attention block (adaptive highlight windows) + gated fusion; (2) Spatial Enlargement: query fused with Topology-Aware Memory Bank M ∈ R^{P×C×d_m}, top-K=5 routing via Prototype-Grid, refined by Load Balance (KL to power-law/uniform priors) and DPPO (pairwise ranking); (3) Popularity Prediction via bi-directional cross-attention and lightweight MLP.

Results & Highlights

  • MicroLens: nMSE 0.7148 vs ICPF 0.7305 baseline, +2.15% improvement.
  • SMPD-video: nMSE 0.4887, +35.79% over MMRA.
  • Informs: nMSE 0.6460, +14.66%.
  • Ablations: removing frame scoring, SSM block, sparse attention, visual branch all degrade; removing visual branch is largest drop.
  • Cold-start (users with ≤3 videos) Top-5 routing outperforms MMRA Top-10 and ICPF Top-10 on MAE and SRC.

Authors & Credibility

Huazhong University of Science and Technology team (Dali Wang, Yunyao Zhang, Junqing Yu, Zikai Song corresponding) with collaborators at La Trobe (Y.-P. Phoebe Chen, known in data mining) and BICTA. Mid-to-high credibility. Arxiv preprint status.

Critical Assessment

  • Novelty: Mid-high. Combines SSM + sparse attention + memory bank ideas — none individually new, but the coordination (anchor-resonance scan, DPPO ranking) is thoughtful.
  • Rigor: 11 baselines across 3 datasets with p-values. Ablations cover both temporal and spatial blocks.
  • Reproducibility: Code not linked in extracted pages. CLIP ViT-L/14 visual encoder and UAE-Large-V1 text encoder are standard.
  • Significance: Strong practical gains on one representative benchmark (SMPD-video), smaller on others. Framework may port to other long-context multimodal tasks.
  • Red flags: The 35.79% gain on SMPD-video vs 2.15% on MicroLens is inconsistent — suggests dataset-specific tuning or baseline weakness. No comparison to recent frontier multimodal LLMs. DPPO adds complexity; ablation "w/o DPPO" shows only modest drop.

Strengths

  • Addresses a realistic two-sided problem (temporal sparsity + flat retrieval).
  • O(PC) memory bank scales better than O(N).
  • Cold-start results are a meaningful stress test.

Limitations

  • Only micro-video popularity; no transfer to other long-context multimodal tasks.
  • Ablation gains on MicroLens and Informs modest.
  • 100-frame ceiling still insufficient for hour-long content.
  • Relies on CLIP encoder — performance coupled to that backbone.
