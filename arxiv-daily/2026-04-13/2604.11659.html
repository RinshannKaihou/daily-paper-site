GPU Acceleration of Sparse Fully Homomorphic Encrypted DNNs

一句话总结： 提出基于AMD GPU的全同态加密DNN矩阵乘法优化方法，利用操作数稀疏性将密文矩阵乘法性能提升至CPU的3.0倍，并将时间复杂度从三次降为半线性。
TL;DR: Optimized sparse ciphertext matrix multiplication on AMD GPUs using FIDESlib, achieving up to 3.0x speedup over CPU and reducing time complexity from cubic to semi-linear for FHE DNN inference.

核心贡献

  • 针对AMD GPU优化的全同态加密DNN矩阵乘法
  • 利用双操作数稀疏性将复杂度从O(n³)降至半线性
  • 基于FIDESlib开源FHE库实现

Key Contributions

  • GPU-accelerated sparse ciphertext matmul for FHE DNNs
  • 3.0x speedup over CPU counterpart
  • Cubic to semi-linear complexity reduction via sparsity exploitation

Methodology

Targets matrix multiplication in FHE DNNs. Uses FIDESlib (open-source GPU FHE library) and exploits sparsity in both operands of the ciphertext multiplication.

Results & Highlights

  • Up to 3.0x speedup over CPU
  • Time complexity reduced from cubic to semi-linear

Authors & Credibility

Multi-institution team: University of Glasgow, University of Murcia, Northeastern University, and AMD. David Kaeli at Northeastern is well-known in GPU architecture research. AMD collaboration provides hardware expertise.

Critical Assessment

  • Novelty: Solid systems contribution — sparse FHE matmul on GPUs is practically important.
  • Significance: Relevant for privacy-preserving ML inference. Direct application to secure computing.

Strengths

  • Practical GPU optimization for FHE
  • Strong industry collaboration (AMD)
  • Open-source library integration

Limitations

  • Focused on matmul only, not full DNN inference pipeline
  • Limited to AMD GPUs
