S3: Structured Sparsity Specification

一句话总结： 提出结构化稀疏规范（S3）代数框架，通过View（张量重排）、Block（原子剪枝单元）和Scope（稀疏决策范围）三个组件统一描述从N:M细粒度到通道剪枝的多种稀疏结构，并集成OBD/OBS方法。
TL;DR: S3 is an algebraic framework for defining, composing, and implementing structured sparse patterns through View (tensor reshape), Block (atomic pruning unit), and Scope (sparsity decision), integrating with OBD and OBS for structured pruning.

核心贡献

  • 提出统一的结构化稀疏规范代数框架（View + Block + Scope）
  • 支持跨张量耦合实现协调稀疏化
  • 与OBD和OBS无缝集成，在输出重构上超越传统二阶启发式方法

Key Contributions

  • Unified algebraic framework for structured sparsity via View, Block, Scope components
  • Cross-tensor coupling for coordinated sparsification
  • Integration with OBD/OBS, surpassing established second-order heuristics

Methodology

S3 specifies sparsity through: View (tensor reshape via layout composition), Block (atomic pruning unit definition), Scope (sparsity decision range). Supports coupling across tensors. Formally validated and experimentally tested with structured OBS and OBD implementations.

Results & Highlights

  • Surpasses established second-order heuristics on output reconstruction
  • Expressive enough for diverse patterns from N:M to channel pruning
