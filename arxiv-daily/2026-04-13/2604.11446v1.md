# Low-rank Optimization Trajectories Modeling for LLM RLVR Acceleration

> **一句话总结：** 本文发现RLVR训练中模型的秩1子空间呈非线性演化规律，提出NExt框架通过非线性外推低秩参数轨迹来加速训练，减少约37.5%的计算开销，同时保持与多种RLVR算法和任务的兼容性。

> **TL;DR:** Proposes NExt, a nonlinear extrapolation framework for low-rank parameter trajectories during RLVR training that models rank-1 subspace evolution nonlinearly, reducing computational overhead by ~37.5% while remaining compatible with diverse RLVR algorithms.

## 核心贡献

- 发现RLVR训练中模型的秩1子空间呈非线性演化，且在LoRA训练中其对原始参数的主导性进一步增强
- 提出NExt（Nonlinear Extrapolation）框架，通过训练预测器来建模参数更新轨迹的非线性外推
- 实现约37.5%的计算开销减少，同时保持与多种RLVR算法和任务的兼容性
- 代码已开源：https://github.com/RUCAIBox/NExt

## Key Contributions

- Empirical finding that rank-1 subspace of model parameters evolves nonlinearly during RLVR training
- NExt framework: nonlinear extrapolation of low-rank trajectories using a trained predictor
- ~37.5% reduction in computational overhead while maintaining compatibility
- Open-source code release

## Methodology

The authors train LLMs using LoRA and extract the rank-1 subspace of parameter differences at multiple training steps. A predictor is trained to model the trajectory of parameter updates, then used to perform predict-extend extrapolation of model parameters. This skips intermediate training steps, achieving acceleration. The approach is compatible with various RLVR algorithms (GRPO, PPO, etc.).

- **Significance:** Practically useful for reducing RLVR training costs, which is a significant bottleneck. However, the quality-cost trade-off needs more thorough evaluation.
