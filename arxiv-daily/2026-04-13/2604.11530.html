SVD-Prune: Training-Free Token Pruning For Efficient Vision-Language Models

一句话总结： 提出基于SVD的无训练token剪枝方法，通过统计杠杆分数选择对全局方差贡献最大的视觉token，在极端剪枝比（16-32 tokens）下仍保持较强性能。
TL;DR: Training-free SVD-based token pruning for VLMs using statistical leverage scores to select tokens contributing most to dominant global variance, maintaining strong performance even at 16-32 vision tokens.

核心贡献

  • 提出基于SVD的无训练即插即用视觉token剪枝方法
  • 使用统计杠杆分数选择对主导全局方差贡献最大的token
  • 在极端视觉token预算（16和32个token）下持续优于现有剪枝方法

Key Contributions

  • Training-free, plug-and-play SVD-based token pruning for VLMs
  • Statistical leverage scores preserve tokens contributing to dominant global variance
  • Outperforms prior methods under extreme token budgets (16-32 vision tokens)

Methodology

Decomposes the vision token feature matrix via SVD and selects top-K tokens using statistical leverage scores. Unlike attention-score or norm-based heuristics, this captures global variance contribution and avoids positional bias.

Results & Highlights

  • Consistently outperforms prior pruning methods under extreme budgets
  • Maintains performance with only 16-32 vision tokens
  • Code available

Authors & Credibility

Authors from IBISC (Universite Paris-Saclay) and CEA List. These are established French research institutions with reasonable ML/systems research presence.

Critical Assessment

  • Novelty: Applying SVD leverage scores to token pruning is a clean, well-motivated approach. The global variance perspective is more principled than local heuristics.
  • Significance: Practical for reducing VLM inference cost. The training-free aspect is valuable.

Strengths

  • Principled global variance-based selection
  • Training-free and plug-and-play
  • Strong performance at extreme compression ratios

Limitations

  • Focused on vision tokens only
  • SVD computation adds overhead (though amortized)
