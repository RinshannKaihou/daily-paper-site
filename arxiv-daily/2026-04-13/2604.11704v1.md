# Fairness is Not Flat: Geometric Phase Transitions Against Shortcut Learning

> **一句话总结：** 通过拓扑审计器隔离垄断梯度的特征，发现"容量相变"现象——线性捷径被剪枝后网络被迫使用更高几何容量来弯曲决策边界，将反事实性别脆弱性从21.18%降至7.66%。

> **TL;DR:** Uses a topological auditor to isolate gradient-monopolizing features, discovering a capacity phase transition where pruning linear shortcuts forces networks to use higher geometric capacity, reducing counterfactual gender vulnerability from 21.18% to 7.66%.

## 核心贡献

- 提出基于几何先验的拓扑审计方法，无需人工干预即可隔离垄断梯度的特征
- 发现"容量相变"现象：线性捷径被剪枝后网络被迫使用更高几何容量弯曲决策边界
- 方法优于L1正则化（会坍缩为人口统计偏见），计算成本仅为JTT等方法的一小部分

## Key Contributions

- Geometric a priori methodology using topological auditor for gradient-monopolizing feature isolation
- Capacity Phase Transition discovery: pruning shortcuts forces higher geometric capacity utilization
- Outperforms L1 regularization and JTT at fraction of computational cost
- Reduces counterfactual gender vulnerability from 21.18% to 7.66%

## Methodology

Zero-hidden-layer (N=1) Topological Auditor mathematically isolates features that monopolize the gradient. When linear shortcuts are pruned, networks must curve decision boundaries using higher geometric capacity (N>=16).

## Results & Highlights

- Counterfactual gender vulnerability: 21.18% → 7.66%
- Outperforms L1 Regularization (which collapses into demographic bias)
- Fraction of computational cost of JTT
