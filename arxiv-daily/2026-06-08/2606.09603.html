⚠️ Abstract-only summary

一句话总结： 提出 Corpus-Grounded Feature Diffusion (CGFD) 低资源微调流程，用 25 个种子样本扩散到 582 个训练样本微调 Breeze-7B，在繁体中文 IEP 生成上超越 GPT-5.4 和 DeepSeek-V3.2。
TL;DR: Proposes CGFD pipeline for low-resource Traditional Chinese IEP generation — 25 seed transcripts → 582 training samples via feature diffusion → QLoRA fine-tune of Breeze-7B — achieving BERTScore F1=0.779, beating GPT-5.4 (0.726) while running fully local and air-gapped.

核心贡献

  • 提出 Corpus-Grounded Feature Diffusion (CGFD) 低资源微调流程
  • 25 个双专家高分种子通过 tau 阈值和 flag-aware 评分上限选择
  • FeatureProfile 提取句长、结构、量化模板注入 LLM 提示
  • 意外发现：Grammar-Constrained Decoding 在繁体中文 token 预算下适得其反 — 无 GCD 路径实现 100% schema 通过率和 34% 更低延迟
  • 本地推理 Breeze-7B 超越 GPT-5.4、DeepSeek-V3.2、Gemini-3-Flash 等大模型

Key Contributions

  • CGFD pipeline: 25 dual-expert seed transcripts → 567 diffusion samples → 582-sample training set
  • FeatureProfile extraction (sentence length, structure, quantification templates) for diversity control
  • Counter-intuitive finding: GCD is counterproductive under Traditional Chinese token budgets — no-GCD achieves 100% schema pass at 34% lower latency
  • Breeze-7B + QLoRA beats GPT-5.4, DeepSeek-V3.2, Gemini-3-Flash, Llama-4-Maverick zero-shot

Methodology

25 dual-expert high-score seed transcripts are selected via tau threshold. A FeatureProfile is extracted and injected into LLM prompts with Verbalized-Sampling diversity control to drive diffusion. 15 expert gold seeds serve as diffusion anchors. The resulting 582-sample training set fine-tunes Breeze-7B with QLoRA. Schema-constrained inference uses Grammar-Constrained Decoding (GCD), though ablation shows no-GCD is superior.

Results & Highlights

  • BERTScore F1 = 0.779 (no-GCD path), exceeding:
  • GPT-5.4: 0.726
  • DeepSeek-V3.2: 0.703
  • Gemini-3-Flash-Preview: 0.703
  • Llama-4-Maverick: 0.700
  • 100% schema pass rate (no-GCD), 34% lower median latency than GCD
  • Fully local, air-gapped inference
