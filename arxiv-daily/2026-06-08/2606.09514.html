一句话总结： BUDDY 提出预算驱动的动态深度路由框架，通过轻量级决策模块在解码时动态选择执行的 Transformer 层，支持严格预算控制和单模型多预算配置。
TL;DR: Introduces BUDDY — a budget-driven dynamic depth routing framework for LLM inference that uses a lightweight Decision Module to score layers and deterministically execute top-k, supporting strict budget control and decode-time rerouting with a single trained model.

核心贡献

  • 预算驱动的动态深度路由：轻量级 Decision Module 根据输入为中间层打分，确定性执行 top-k 层
  • 解码时重新路由：复用首层 KV cache 作为低成本全局上下文，与最新 token 表示池化后做路由决策
  • 可选 Budget Predictor：在无明确预算时自动估计输入相关的计算级别
  • 单一训练模型支持多预算配置

Key Contributions

  • Budget-driven dynamic depth routing with lightweight Decision Module
  • Decode-time rerouting using first-layer KV cache as global context
  • Optional Budget Predictor for input-dependent compute level estimation
  • Single trained model supports multiple budgets
  • Competitive with strong static pruning baselines while uniquely supporting strict budget control

Methodology

BUDDY uses a lightweight Decision Module to score intermediate layers conditioned on input. Top-k layers are deterministically executed to satisfy a given budget. For decode-time adaptation, it reuses the first-layer KV cache as a low-overhead global context source and pools it with the newest token representation before each routing decision. An optional Budget Predictor estimates input-dependent compute levels when no explicit budget is given.
