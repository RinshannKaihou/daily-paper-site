Preserving Plasticity in Continual Learning via Dynamical Isometry

arXiv: 2606.09762

Date: 2026-06-08

https://arxiv.org/abs/2606.09762

⚠️ Abstract-only summary

一句话总结： 本文从动态等距（dynamical isometry）角度解释持续学习中的可塑性丧失问题，提出AdamO优化器和等距正则化方案，在持续学习基准上匹配或超越现有方法。
TL;DR: Links plasticity loss in continual learning to deviations from dynamical isometry (layer-wise Jacobian singular values near one). Revisits near-isometric universal Lipschitz networks, proposes an efficient isometry-promoting regularization that reactivates dormant ReLU units, and introduces AdamO — an Adam-style optimizer that decouples isometry regularization from gradient updates. Shows prior plasticity methods only achieve partial isometry. Consistently matches or outperforms existing approaches on supervised and RL continual-learning benchmarks.

核心贡献

  • 首次将持续学习中的可塑性丧失（plasticity loss）与经验神经正切核（empirical NTK）及动态等距（dynamical isometry）建立理论联系
  • 重新审视一类几乎处处等距且仍为通用 Lipschitz 函数逼近器的网络，证明近动态等距与强表达能力的非线性表征兼容
  • 提出高效的等距促进正则化方案，并发现其能重新激活休眠 ReLU 单元的新机制
  • 提出 AdamO 优化器，以 AdamW 解耦权重衰减的方式将等距正则化与梯度更新解耦
  • 从动态等距视角重新解释现有可塑性保持方法，揭示它们仅部分度量等距

Key Contributions

  • Establishes a theoretical link between plasticity loss in continual learning and the empirical Neural Tangent Kernel (NTK), identifying dynamical isometry as a key mechanism for preserving plasticity
  • Revisits near-isometric networks that remain universal Lipschitz function approximators, demonstrating compatibility of near-dynamical isometry with expressive nonlinear representations
  • Proposes an efficient isometry-promoting regularization scheme and identifies a novel mechanism for reactivating dormant ReLU units
  • Introduces AdamO, an Adam-style adaptive optimizer that decouples isometry regularization from gradient updates (analogous to how AdamW decouples weight decay)
  • Reinterprets prior plasticity-preserving methods through the dynamical isometry lens, showing they target only partial measures of isometry
  • Demonstrates consistent matching or outperforming of existing approaches across supervised and RL continual-learning benchmarks designed to induce plasticity loss
