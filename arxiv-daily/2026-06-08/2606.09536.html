一句话总结： HadamardNet 将 Hadamard 编码引入目标检测和语义分割的输出表示，利用码字冗余实现单次前向传播中的对抗攻击/扰动检测，同时保持干净数据上的参考性能。
TL;DR: Introduces Hadamard-coded output representations for object detection and semantic segmentation, deriving optimal decoding via probability simplex projection and exploiting codeword inconsistency for single-pass adversarial/disturbance detection at SOTA performance.

核心贡献

  • 首次将 Hadamard 编码输出表示应用于目标检测和语义分割
  • 推导出基于概率单纯形投影的最优类概率解码过程
  • 利用码字不一致性作为对抗攻击和扰动检测信号
  • 单次前向传播即可实现检测，无需额外推理步骤

Key Contributions

  • First application of Hadamard codes to object detection and semantic segmentation output representations
  • Novel optimal decoding via projection onto probability simplex
  • Exploits intrinsic codeword redundancy for adversarial attack and disturbance detection
  • Single detection pass — no additional inference overhead
  • HadamardNet framework supporting both tasks

Methodology

The authors replace standard one-hot output encodings with Hadamard codes. They derive a novel decoding procedure that projects onto the probability simplex for optimal class-wise probabilities. The intrinsic redundancy of Hadamard codewords provides a measure of prediction inconsistency, which is exploited for detecting adversarial perturbations and disturbances in a single forward pass.

Results & Highlights

  • State-of-the-art perturbation detection for both object detection and semantic segmentation
  • Single detection pass (no additional inference)
  • Equivalent or close-by reference performance on clean data
  • Addresses both adversarial attacks and natural disturbances
