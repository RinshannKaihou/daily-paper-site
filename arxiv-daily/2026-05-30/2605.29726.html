SLAD: Shared LoRA Adapters for Task Specific Distillation
Bensaid, Bendou, Gripon, Leduc-Primeau · 28 May 2026
Vision Transformer Knowledge Distillation Parameter-Efficient FT
Key Metric
3.71%
CKA drop with shared LoRA, vs 9.4% for full fine-tuning and 5.1% for LoRA alone. Feature alignment preserved.
Why This Matters
Fine-tuned teachers paradoxically worsen distillation because feature alignment collapses; SLAD fixes this with a simple, faster one-step pipeline.
Critical Edge
Only evaluated on DINOv2 ViT; the dimensionality-subset sharing constraint limits applicability to architecturally compatible model pairs.

Methodology

SLAD starts from a striking observation: when a DINOv2 teacher is fully fine-tuned for a downstream task, its feature alignment with a smaller student (measured by CKA) drops by 9.4%—so a stronger teacher actually transfers worse.

The fix is shared LoRA adapters:

  • Apply rank-16 LoRA to QKV attention matrices in both teacher and student.
  • The student reuses the first ds dimensions of the teacher's LoRA A/B matrices.
  • Joint single-step training with task loss + KL distillation (T=2).
  • Even block mapping bridges depth mismatches (e.g. 24→12 layers).

Strengths / Limitations

Strengths

  • CKA analysis gives a crisp, mechanistic explanation for the "stronger teacher, worse distillation" paradox.
  • Method is conceptually simple—no extra projection layers or multi-stage training.
  • Improves both student and teacher simultaneously (unusual and practical).
  • Consistent gains across classification, segmentation, and multiple model size pairs.
  • Thorough ablations: rank, temperature, loss weights, block mapping, and PEFT alternatives.

Limitations

  • Scope limited to DINOv2 ViT; no evidence for CNNs, Swin, NLP, or multimodal settings.
  • Datasets are relatively small (CUB, Aircraft, DTD, Cityscapes)—no ImageNet or ADE20K scale.
  • LoRA's low-rank ceiling may matter more on harder tasks where full fine-tuning shines.
  • Baseline re-implementation ("Probing*") is unverified; original code was unavailable.
  • No empirical comparison with intermediate feature distillation (e.g. FitNets).

Questions for Further Exploration

  1. Can SLAD scale to multi-teacher distillation with multiple adapter-sharing paths?
  2. Does shared LoRA transfer across architectures (e.g. ViT teacher → CNN student)?
  3. How does the performance gap vs full fine-tuning evolve on large-scale benchmarks like ImageNet?
  4. What is the optimal student-to-teacher size ratio for adapter sharing?
  5. Would adding feature-level hints (FitNets-style) further boost SLAD?
  6. Is shared LoRA equally effective for LLM distillation in NLP?
