MF-Diffuser lifts diffusion-based offline RL planning to thousands of agents through mean-field trajectory diffusion.
Mean-Field Diffuser: Scaling Offline MARL to Thousands of Agents · Wenhao Li, Xiangfeng Wang, Bo Jin · 2026-05-28 · cs.LG Offline RL Mean-Field Diffusion
Key Metric
N ≥ 1000
Agents scaled with offline distribution shift bounded independently of population size.
Why This Matters
Provides a theoretically-grounded paradigm for scaling diffusion planners to massive multi-agent systems with population-independent error bounds.
Critical Edge
Requires agent homogeneity — real-world systems with heterogeneous roles and capabilities are not covered.

Methodology

Mean-field trajectory SDEs. Each agent's trajectory is a particle in R^D. The interaction operator separates individual dynamics A_θ from population interaction B_θ, with coupling via kernel K_θ decomposed across MDP steps.

Value-weighted chaotic entropy (MF-VSM). Unifies score matching and value-gradient alignment at the chaotic limit, with a 1/√(ND_T) prefactor providing natural robustness to large N.

Hierarchical coarse-to-fine planning. Start from N₀ ~ √N noise trajectories. At designated denoising steps, branch N_k → b·N_k via operator Ψ^b. Training aggregates K sub-problems with b⁻ᵏ weighting — an implicit fine-tuning bias.

Inference. Reverse denoise with value-gradient bias η · ∇V(τ) for high-return plans; enforce observed initial state via inpainting; execute only the first action before re-planning.

Assessment

Strengths

  • Addresses genuinely hard scaling problem with tractable joint action space.
  • Exceptionally strong theory: explicit bounds, Nash convergence, distribution-shift independence.
  • Propagation-of-chaos insight (only O(√N) agents needed) is elegant and computationally dramatic.
  • Four-term error decomposition provides clear guidance for future improvements.

Limitations

  • Homogeneity assumption — real-world heterogeneous agents not covered.
  • O(H²/√N) bound depends quadratically on horizon; compounds multiplicatively.
  • Moderate-N gap: 10–50 agent regimes may not benefit from mean-field approximation.
  • Benchmark scope limited to mean-field RL tasks; real-world gap untested.
  • Behavior policy (MFQ) may represent a favorable data-collection scenario.

Questions for Further Exploration
