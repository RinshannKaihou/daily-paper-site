Derives precise BBP-type phase transitions for when linear classification via conjugate-kernel eigenvectors becomes feasible on XOR data.

Eigen-Spike Emergence and Quadratic Equivalents for Conjugate Kernels on Nonlinearly Separable Data Cranston, Wang, Kemp, Mahoney 2026-05-28 Theory Random Matrix Theory
Key Metric
[no headline metric reported]
This is a theoretical paper deriving phase-transition conditions rather than reporting empirical numerical results.
Why This Matters
Extends random matrix theory tools from linear to nonlinear settings, revealing when spectral structure emerges for nonlinear classification.
Critical Edge
Restricted to the XOR problem — generalization to richer nonlinear structures remains unproven.

Methodology

  • Object: Conjugate kernel (CK) of a feedforward net under the canonical XOR dataset.
  • Novel tool: A quadratic deterministic equivalent for the spiked CK matrix — going beyond standard linear RMT surrogates.
  • Question asked: Do eigenvectors of emergent informative spikes asymptotically align with XOR labels?
  • Framework: Generalizes classical BBP phase transitions to the nonlinear setting.

Results & Highlights

  • Precise BBP-type phase transitions derived for CK-eigenvector linear classification feasibility.
  • Analysis spans practical knobs: sample complexity, SNR, activation choice, pretrained features.
  • Deterministic equivalence tools from RMT shown applicable to nonlinearly separable data for the first time.
Strengths
  • Rigorous theoretical framework with precise phase-transition results.
  • Clear connection between RMT tools and practical ML parameters.
  • Addresses a well-defined gap in the deterministic-equivalence literature.
Limitations
  • Restricted to XOR — richer nonlinear structures are not addressed.
  • Quadratic equivalent may not capture full complexity of modern deep networks.
  • Practical guidance for practitioners is limited; primarily a theory contribution.

Questions for Further Exploration

  1. Can the quadratic equivalent framework extend beyond XOR to more general nonlinearly separable data (e.g., concentric spheres, manifold-structured data)?
  2. How do the derived phase transitions scale with dimensionality and depth of modern neural networks?
  3. What are the implications of the quadratic equivalent for activation-function design in practice?
  4. How does this framework relate to recent work on feature learning in the NTK regime?
  5. Can BBP-type transitions inform practical sample-size decisions for nonlinear classification tasks?
