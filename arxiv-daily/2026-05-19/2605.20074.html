Towards Distillation Guarantees under Algorithmic Alignment for Combinatorial Optimization

Thien Le & Melanie Weber 2026-05-19 Abstract-only summary
arXiv Page PDF Markdown

★ Key Finding

Knowledge distillation in combinatorial optimization is provably efficient when the target GNN is algorithmically aligned with a dynamic programming solver and the source model satisfies the Linear Representation Hypothesis (LRH). The distillation problem's complexity is then bounded by the DP transition function's decision-tree complexity — turning an open empirical question into a tractable learning-theoretic guarantee.

μ Key Metric

O(DTψ)
Distillation complexity is parameterized by the decision-tree complexity of the DP transition function ψ — concrete sample and computational bounds, not just existential results.

→ Why This Matters

  • First rigorous bridge between algorithmic alignment, representation richness, and distillation success for graph-structured problems.
  • Extends prior decision-tree distillation theory (Boix-Adsera 2024) to GNN/DP settings with combinatorial structure.
  • Provides a principled answer to when and why structured distillation works — not just that it can.

! Critical Edge

  • LRH may not hold for real-world trained models.
  • Requires perfect algorithmic alignment — robustness to misalignment is untested.
  • Zero experimental validation; all results are purely theoretical.
  • Limited to DP-solvable tasks with DT-representable transitions.

M Methodology

The framework builds on two theoretical pillars, then derives distillation guarantees by connecting them through decision-tree representations:

Algorithmic Alignment

GNN architecture mirrors the structure of a DP algorithm for the target combinatorial problem (Xu et al., 2019). This structural correspondence is the key enabler for provable distillation.

Linear Representation Hypothesis

Source model representations linearly encode task-relevant features (Elhage et al., 2022; Park et al., 2024). This richness assumption ensures the source has "enough signal" for the target to extract.

The DP transition function is represented as a decision tree, and learning-theoretic results from Boix-Adsera (2024) bound the sample and computational complexity of distillation in terms of that tree's complexity parameters.

+ Strengths

  • Clean theoretical framework linking alignment, representations, and distillation.
  • Extends DT distillation results to practically relevant graph-based problems.
  • Provides concrete complexity bounds, not just existence proofs.

− Limitations

  • LRH assumption may be violated in practice.
  • No experimental validation of theoretical guarantees.
  • Scope restricted to DP-solvable, DT-representable problems.

C Core Contributions

  • Establishes a distillation guarantee framework for combinatorial optimization, linking it to DP complexity parameters.
  • Proves that algorithmic alignment + LRH suffice for efficient distillation into GNN targets.
  • Generalizes decision-tree distillation theory from Boix-Adsera (2024) to graph-structured settings.

? Open Questions

  1. How do guarantees degrade under approximate LRH?
  2. Can the framework extend beyond DP — e.g., greedy or branch-and-bound?
  3. Practical implications for NP-hard problems where exact DP is intractable?
  4. How do theoretical bounds compare with empirical distillation on benchmarks?

R Relevance

Provides distillation-level theoretical guarantees for the algorithmic alignment framework — directly relevant to understanding GNN learning mechanisms in combinatorial optimization, structured prediction, and graph learning theory.
