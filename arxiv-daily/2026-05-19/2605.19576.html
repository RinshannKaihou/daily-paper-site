Library Drift: Diagnosing and Fixing a Silent Failure Mode in Self-Evolving LLM Skill Libraries

Xing ZhangYanwei CuiGuanghui WangZiyuan LiWei QiuBing ZhuPeiyang He
arXiv PDF Self-Evolving Agents Skill Libraries Library Drift Governance
Key Finding
Self-evolving LLM skill libraries suffer a silent failure — "library drift" — where unbounded skill accumulation degrades retrieval and injects false positives, stagnating performance.

M Methodology

Three-pillar approach to diagnosing and fixing library drift in self-evolving LLM skill libraries:

  • Reproducible triggers — Ablation experiments isolate drift: disabling injection yields +0.002 (flat floor); premature retirement causes −0.019 (active harm).
  • Trace-level diagnostics — Append-only evidence log with per-skill contribution scores, attribution verdicts, and router engagement metrics. Surfaces drift before end-task metrics degrade.
  • Governance recipe — (a) outcome-driven retirement, (b) bounded active-cap on simultaneous skills, (c) meta-skill authoring prior for quality control.
