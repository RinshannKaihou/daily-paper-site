Post-Training is About States, Not Tokens: A State Distribution View of SFT, RL, and On-Policy Distillation
Dong Nie · 2026-05-21

Key Metric
0.95 vs 0.83
Retention ratio at identical MMD drift — state source, not drift magnitude, drives forgetting.
Why This Matters
Unifies SFT, RL, and distillation under one lens — and tells you how to improve a target task without destroying general capabilities.
Critical Edge
Tested only on Qwen3-0.6B on a single GPU — unclear if findings survive at production scale (7B+).

Methodology

Define state s = (prompt, prefix) → Decompose into state source q(s) + signal source S(s) → Map SFT / RL / OPD onto 2D axes
The core framework treats autoregressive generation as inducing a state visitation distribution dπ(s). Every post-training method is then characterized by two questions: where do the training states come from? (dataset → teacher → student) and what signal is applied? (labels → rewards → teacher logits). Experiments use Qwen3-0.6B-Base with LoRA on a single RTX 3090. GSM8K is the target task; TruthfulQA and MMLU measure retention. Conditions: mild SFT (1 epoch, lr 1e-4), stress SFT (5 epochs, lr 5e-4), OPD from both teachers, and lightweight GRPO-style RL. Drift is measured via rollout-state MMD with RBF kernel on lexical features.

Key Findings

Stress SFT is doubly harmful: degrades GSM8K below base (0.420 vs 0.448) while destroying retention (0.826).
OPD recovers from a bad teacher: student beats stress-SFT teacher on every metric, using it as sole supervision.
RL is remarkably gentle: +2.4 GSM8K with only 0.002 mean forgetting.
One-step OPD collapses: GSM8K drops to 0.040 — next-token logit matching is too weak for reasoning tasks.

Strengths

  • Clean conceptual framework that genuinely unifies SFT, RL, and distillation
  • Striking result: OPD student beats degraded teacher on all metrics
  • Honest about scale limitations; does not overclaim
  • Practically actionable: continuation-based OPD > one-step logit matching
  • Well-controlled experimental design with clear baselines

Limitations

  • Single 0.6B model; transfer to 7B+ is unknown
  • Only TruthfulQA and MMLU subset for retention
  • Lexical-feature MMD is a crude drift proxy
  • Minimal RL (GRPO-style, not PPO); may not represent production RL
  • No computational cost comparison between methods
  • Mild vs. stress boundary not systematically explored

Questions for Further Exploration

  1. Does the state-distribution view hold at 7B/70B scale, or do larger models absorb off-policy stress differently?
  2. How does OPD behave when the teacher is stronger than the student (e.g., larger model)?
  3. What is the optimal continuation length for OPD between one-step (catastrophic) and full-trajectory?
  4. Can the framework predict which retention tasks are most affected, or only aggregate magnitude?
  5. How do DPO and other preference-optimization methods fit into the state-distribution axes?
