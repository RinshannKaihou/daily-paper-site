Relational Linear Properties in Language Models: An Empirical Investigation

Giovanni Valer, Luigi Gresele, Marco Bronzini, Emanuele Marconato | 2026-05-21 | U. Trento / FBK / U. Copenhagen
mechanistic-interpretability linear-representations relational-probing llm-internals
Key Metric
KL-RP achieves F1 ≥ 0.89 on tense and truthfulness for Llama-3.1-8B, with dKL ≤ 0.10.
Why This Matters
Reveals how LLMs linearly encode relational knowledge across layers — directly relevant to mechanistic interpretability and representation engineering.
Critical Edge
Only two small quantized models tested (8B, 2B); no evidence this scales to frontier models (70B+). Code not released.

Methodology

The authors formalize relational linearity: for a fixed query q and token subset YP, the conditional distribution p(y | s ˑ q; YP) can be exactly reconstructed from context-only embeddings via a linear classifier with softmax.

KL-RP trains weights wy ∈ Rd and biases by to minimize KL divergence between the linear probe output and the model's true next-token distribution. No annotated triplets required.

Evaluated across four relations — language identification (20 langs, 9.5k samples), tense (10k), subjectivity (8k), truthfulness (9.1k).

Results & Highlights

  • TENSE and TRUTH show strong linearity. Llama-3.1: F1(LLM) ≥ 0.89, dKL ≤ 0.10. Gemma-2: TENSE F1=0.95, TRUTH F1=0.90.
  • KL-RP crushes LRE on LANG for Llama-3.1: 0.98 vs 0.36 F1(LLM). LRE's Jacobian approximation breaks on diverse prompts.
  • Layer dynamics mirror the NLP pipeline: surface features (language, tense) decode from early layers; abstract features (truthfulness) peak in middle layers.
  • Paraphrase impact is model-dependent: cross-lingual query rewrite drops Gemma-2 by ~12.5 F1 pts but Llama-3.1 by only ~6.7.
  • SUBJ defeats both models — below majority-class baseline. Gemma-2 collapses to near-trivial distributions (CSS ≈ 0.91).

Strengths & Limitations

Strengths

  • Clean formalization with explicit definitions (Definition 2.1).
  • CSS metric is a practical contribution — flags trivial probing scenarios prior work ignores.
  • Comprehensive layer-wise analysis across four diverse relations yields actionable insights on information flow in transformers.
  • Transparent about failure cases (SUBJ) and setup differences from the original relational linearity formulation.

Limitations

  • Only two relatively small models (8B, 2B); unclear if results generalize to 70B+ or frontier models.
  • 8-bit quantization may alter representation geometry vs. full-precision.
  • System-prompt setup differs from the original s ˑ q formulation without empirical comparison.
  • Limited to classification-style relations; open-ended or multi-hop relations untested.
  • No public code release.

Open Questions

  1. Does relational linearity hold for larger models (70B+), and does it strengthen or weaken with scale?
  2. Can the identified linear relational subspaces be used for activation steering to control model outputs on specific relations?
  3. How does relational linearity change during training — gradual emergence or phase transition?
  4. Does it extend beyond classification-style relations to compositional ones (e.g., “X is the capital of the country where Y was born”)?
  5. What is the relationship between relational linearity and Sparse Autoencoder features — same information?
