AMUSE: Anytime Muon with Stable Gradient Evaluation

Time-varying interpolation merges Muon's speed with Schedule-Free stability for schedule-free anytime LLM training.

Jueun Kim, Baekrok Shin, Jihun Yun, Beomhan Baek, Minhak Song, Chulhee Yun KAIST 2026-05-21
optimization muon-optimizer schedule-free LLM-pretraining anytime-training
Key Metric 1.51× Fewer steps to match Muon final performance on 720M LLM pretraining
Why This Matters Eliminates learning rate scheduling and optimizer instability — the two biggest pain points in LLM pretraining.
Critical Edge Zero experiments beyond 1B parameters; frontier-scale (70B+) transfer is completely unverified.

Methodology

AMUSE builds on two pillars:

River-Valley Landscape Analysis

Decomposes the loss Hessian into a small number of high-curvature dominant directions (valley walls) and a much larger bulk subspace (the flat river). Muon's Newton-Schulz orthogonalization increases bulk-ratio (accelerating progress) but polar normalization strips magnitude info — tiny dominant components get constant-magnitude updates, causing oscillation.

Time-Varying Schedule-Free Muon

The standard SF framework maintains three sequences: base iterate zt, evaluation point yt, and averaged sequence xt. AMUSE interpolates with a time-varying coefficient:

βt = 1 − ((T₀ − 1)/(t − 1))ρ · (1 − β₁)    for t ≥ T₀

Small βt early (fast adaptation near Muon sequence), gradually increasing toward averaging to suppress valley-wall oscillation. Independent of total training steps.

Applies Muon to matrix-valued layers; SF-AdamW/SF-SGD to non-matrix parameters. Linear warmup but no learning rate decay.

Results

1.51× fewer steps — 720M LLM pretraining (FineWeb-100B)
3.08× fewer steps — ImageNet ViT-B/16 fine-tuning (MAE)
1.12× fewer steps — ImageNet ResNet-50
Outperforms AdamW, SF-AdamW, Muon on CIFAR-10/100, SVHN, ISIC 2018
1B LLM: continues to outperform all baselines at scale
Memory = AdamW (one extra state copy zt); negligible wall-clock overhead

Strengths

  • Principled Hessian-based analysis connecting Muon behavior to loss landscape geometry
  • Elegant time-varying βt naturally transitions from exploration to stabilization
  • Eliminates LR decay — significant practical simplification
  • Anytime training: valid checkpoints at every step
  • Same memory footprint as AdamW
  • 6 image tasks + 3 LLM scales + extensive baselines

Limitations

  • No experiments beyond 1B; 1.51× speedup may not transfer to frontier scale
  • 1B run reuses 720M hyperparameters — potential underperformance
  • Not fully schedule-free: linear warmup still required
  • Hybrid optimizer (Muon + SF-AdamW) adds implementation complexity
  • Concurrent Muon variants (MOUSSE, Adam+Muon) only in appendix

Relevance / 相关性

AMUSE通过河-谷损失景观分析揭示了Muon正交化的双面性，并利用时变插值系数βt将Muon的快速收敛与Schedule-Free的轨迹平均稳定性结合，实现无需学习率调度的随时训练。

Directly addresses LR scheduling and optimizer instability — the two core pain points in LLM pretraining. The 1.51× convergence acceleration has major practical value for compute-intensive training runs. Schedule-free property and Muon improvements are both central topics in current ML systems optimization research.
