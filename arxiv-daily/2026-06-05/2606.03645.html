Key Finding · ICML 2026 · Mechanistic Interpretability

LLM arithmetic errors are geometric slippages of a continuous Carry Potential across integer quantization boundaries in residual stream space — not random failures.

Why This Matters
Provides a unified, falsifiable geometric explanation for arithmetic fragility and a practical training-free correction method reusable across interpretability research.
Critical Edge
Single model (Qwen3-4B), single task — and the correction still misses the majority of errors (TP correction rate only 30–44%).
Methodology

Task & Data: Three-term 10-digit integer addition, N=10,000 problems, left-to-right generation. Model: Qwen3-4B (36 layers, single-digit tokenization).

Probing: Six lightweight probes (logistic regression / MLP) decode ground truth, model output, correctness, raw sum mod 10, input carry, and Carry Potential Φ from final-layer hidden states.

Geometry: UMAP (cosine distance) applied to residual stream activations reveals Iso-Raw-Sum Trajectories (IRSTs) — continuous manifold fibers stratified by carry value.

NQM: Perceived Carry Potential Φ̂ = Φ + ε (ε ~ N(0,σ²)). Error occurs when noise pushes Φ̂ across an integer boundary. Fitted to empirical bathtub distribution: R²=0.80, σ≈0.05.

Correction: Dual-stream protocol compares a local raw-sum probe and a global carry-potential probe; inconsistencies trigger output overwrite: ŝ_new = (r̂_p + ⌊Φ̂_p⌋) mod 10.

Strengths
  • IRST framework is theoretically coherent, grounded in UMAP, probing, causal steering, and NQM curve fitting simultaneously.
  • Provides a unified explanation for arithmetic fragility, probe versatility variation, and the off-by-one error pattern.
  • NQM is a rare quantitative, predictive mechanistic model — falsifiable, not post-hoc.
  • ICML 2026 publication with public code (github.com/RL-MIND/Shape-of-Addition) — high reproducibility.
  • Dual-stream correction is practical and training-free, demonstrating engineering value beyond pure analysis.
Limitations
  • Single model (Qwen3-4B), single task (3-term 10-digit addition) — BPE-model generalization is unverified.
  • Correction misses the majority of errors: TP correction rate only 30–44%.
  • UMAP as primary visualization introduces projection artifacts; geometric claims rest on 2D embeddings.
  • Cognitive noise σ fit to one setting; dependence on task complexity and model scale is an open question.
  • No comparison to lightly finetuned baselines; no analysis of multi-step reasoning chains.
Open Questions
  • Q1 Does IRST geometry and the Noisy Quantization Model generalize to BPE-tokenized models (GPT-4, Llama), where digit grouping introduces representational entanglement?
  • Q2 How does cognitive noise level σ scale with operand count, digit length, and model size — is there a phase transition where carry propagation becomes catastrophically unreliable?
  • Q3 Can the carry-potential probe be used proactively during beam search re-ranking rather than reactively post-hoc, and what is the accuracy ceiling?
  • Q4 At which intermediate layer does IRST structure crystallize? This could inform targeted fine-tuning or activation patching strategies.
  • Q5 What is the relationship between the helical/trigonometric representations (Kantamneni & Tegmark 2025) and IRST fiber structure — compatible descriptions at different scales, or competing accounts?
