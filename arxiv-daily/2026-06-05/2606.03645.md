> **一句话总结：** 本文在 Qwen3-4B 的残差流中发现了"等原始和轨迹"（IRST）几何结构，证明加法错误本质上是连续进位势（Carry Potential）跨量化边界的随机噪声滑移，并据此设计了一种无需重训练的双流一致性推理时自纠错方法，将 token 准确率从 86.26% 提升至 89.56%。

> **TL;DR:** This ICML 2026 paper dissects why LLMs make "off-by-one" errors in multi-operand addition. Using Qwen3-4B on 10,000 three-term 10-digit addition problems, the authors discover that last-layer residual stream activations organize into **Iso-Raw-Sum Trajectories (IRSTs)**—continuous fibers stratified by input carry. Arithmetic errors are not random: they are **geometric slippages** where Gaussian neural noise pushes a latent continuous **Carry Potential** Φ across an integer quantization boundary. This Noisy Quantization Model fits the observed periodic "bathtub" error distribution with R²=0.80, extracting an internal noise level σ≈0.05. A practical dual-stream inference-time correction—enforcing consistency between a local raw-sum probe and a global carry-potential probe—achieves 89.56% token accuracy and 44.39% true-positive error correction with 98.13% false-positive preservation, outperforming re-prompting, linear steering, and hard replacement baselines. Code is public at https://github.com/RL-MIND/Shape-of-Addition.

## 核心贡献

- **IRST 几何结构发现：** 通过 UMAP 可视化，在最终层激活空间中识别出以"等原始和轨迹"（Iso-Raw-Sum Trajectory，IRST）为骨架、以输入进位纤维为纹理的层次几何流形，揭示了 LLM 内部编码加法状态的精确几何方式。
- **噪声量化模型（NQM）：** 提出连续进位势 Φ 作为 LLM 内部推力表示，将算术错误建模为感知进位势 Φ̃=Φ+ε（ε~N(0,σ²)）跨整数边界的阈值越过，推导出周期性"浴缸形"误差率公式，与实验数据高度吻合（R²=0.80，σ≈0.05）。
- **探针多功能性几何解释：** 证明轻量探针（逻辑回归/MLP）能从单一激活向量同时解码真值数字（94.85%）、模型输出（98.81%）、正确性（82.41%）、原始和（98.60%）、输入进位（96.84%）、进位势（92.08%），并从几何拓扑角度解释了不同探针精度差异的根本原因。
- **双流推理时自纠错：** 提出无需重训练的双流一致性协议，在错误被检测时将输出覆写为 ŝ_new=(r̂_p+⌊Φ̂_p⌋) mod 10，最优配置（δ=0.1）达到 89.56% token 准确率，真正例纠错率 44.39%，显著优于所有基线。

## Key Contributions

- **IRST Geometry:** Residual stream activations at the final layer (Qwen3-4B, position p=4) organize into **Iso-Raw-Sum Trajectories**—continuous manifold fibers of constant raw sum, stratified into parallel sub-fibers by input carry c_p ∈ {0,1,2}. UMAP (cosine distance) makes this hierarchy visually explicit.
- **Noisy Quantization Model (NQM):** A theoretically grounded model positing that LLMs maintain a continuous **Carry Potential** Φ_p = Σ r_{p+j}/10^j (Eq. 3), corrupted by Gaussian cognitive noise ε~N(0,σ²). Arithmetic errors occur when noise pushes Φ̂_p across an integer boundary. The resulting error probability formula (Eq. 6, Q-function based) predicts the observed periodic bathtub error distribution with R²=0.80, and calibrates σ≈0.05.
- **Probe Versatility Explained:** Six probes trained on the same activation vector achieve different accuracies (82%–99%); the paper explains this as a direct function of the geometric separability of the underlying manifold topology, not information extraction difficulty.
- **Trajectory-Level Causal Validation:** A steering vector v_steer = (μ_b − μ_a)/‖μ_b − μ_a‖ is applied to final-layer pre-norm activations. States near quantization boundaries flip under smaller |α| than deep-basin states, confirming the causal role of IRST geometry in carry determination (Figure 6).
- **Dual-Stream Inference-Time Correction (Section 8):** Decoding a local raw-sum probe (r̂_p) and a global carry-potential probe (Φ̂_p), the protocol flags inconsistencies and overwrites the output logit. The δ=0.1 tolerance variant achieves 89.56% token accuracy, 44.39% TP correction, 98.13% FP preservation—best overall in Table 2.

## Methodology

**Task and Data:** Multi-operand addition of three 10-digit integers, generating outputs left-to-right (most significant to least significant digit). N=10,000 problems; dataset balanced per probe target. The model studied is **Qwen3-4B** (36 transformer layers), which uses single-digit tokenization enabling direct alignment between visual digits and internal states.

**Probing Framework:** At each generation position p, hidden states h_p^(L) are extracted from the final layer L. Six lightweight probes (logistic regression / MLP) are trained on balanced datasets to decode: Ground Truth digit s_p, Model Output ŝ_p, Correctness (s_p≠ŝ_p), Raw Sum mod 10 (r_p), Input Carry c_p, and Carry Potential Φ_p (via regression + flooring).

**Geometric Analysis:** UMAP with cosine distance is applied to h_p^(L) vectors. Digit anchor vectors (output unembeddings for 0–9) are included as semantic reference. The resulting 2D projection reveals the macroscopic backbone (10 digit basins) and microscopic carry fibers.

**IRST Formalization:** An IRST T_r is defined as the continuous manifold of all internal states with the same raw sum r_p = r (Eq. not numbered, Sec. 4). Each trajectory threads through adjacent digit anchors as carry increments. Geometric slippages = off-by-one errors localized to transition zones between adjacent digit basins along an IRST.

**Noisy Quantization Model:** Perceived Carry Potential Φ̂_p = Φ_p + ε, ε~N(0,σ²). Predicted carry ĉ_p = ⌊Φ̂_p⌋. Error probability P(Error|Φ) = Q(δ(Φ)/σ) + Q((1−δ(Φ))/σ) where δ(Φ) = Φ mod 1 (fractional part). Fitted to empirical error distribution with σ≈0.05 (R²=0.80).

**Dual-Stream Self-Correction (Sec. 8):** Two orthogonal probes decode (1) local raw sum r̂_p and (2) global carry potential Φ̂_p. Plausible carries K_p(δ) = {⌊φ⌋ | φ ∈ [Φ̂_p−δ, Φ̂_p+δ]}. If the model's output ŝ_p fails the modular consistency check ŝ_p ≡ (r̂_p + c) (mod 10) for some c ∈ K_p(δ), the output is overwritten as ŝ_new = (r̂_p + ⌊Φ̂_p⌋) mod 10.

## Results & Highlights

**Probing Accuracy (Table 1, position p=4):**
- Model Output ŝ_p: **98.81%** (near-perfect; mimics Voronoi basin of predicted anchor)
- Raw Sum mod 10: **98.60%** (confirms distinct IRST structure)
- Input Carry c_p: **96.84%** (disentangles parallel fibers)
- Carry Potential Φ_p: **92.08%** (continuous regression + flooring)
- Ground Truth s_p: **94.85%** (lower than model output—confirms geometric slippage)
- Correctness: **82.41%** (lowest; manifold is continuous with no sharp correct/incorrect boundary)

**Noisy Quantization Model Fit:** The periodic bathtub error distribution—error spikes at Φ≈1.0, 2.0 (integer boundaries = metastability zones), troughs at Φ≈1.5 (robust plateaus)—is fit with R²=0.80, σ≈0.05.

**Ablation (Table 3):** R+TC (Raw Sum + True Carry) reaches 96.0% token accuracy, confirming that the carry signal is the main bottleneck; TR+C (True Raw Sum + Carry Probe) reaches only 90.5%, confirming the raw sum representation is latently robust even when final output is wrong.

**Correction Baselines (Table 2):**
| Method | Token Acc | TP Corr | FP Pres |
|---|---|---|---|
| Original | 86.26% | — | — |
| Re-Prompting | 79.90% | 0.08% | 99.98% |
| Steering | 88.27% | 30.58% | 96.97% |
| Replacement | 89.13% | 31.73% | 97.65% |
| Ours (δ=0) | 87.27% | **44.39%** | 94.07% |
| **Ours (δ=0.1)** | **89.56%** | 30.46% | 98.13% |

The δ=0.1 variant balances correction rate and false-positive preservation to achieve the best overall token accuracy.

**Causal Steering:** Boundary-adjacent states (4,5,1) and (5,4,2) flip under |α|≈0.1 and 0.3 respectively (small perturbations), while deep-basin states (5,5,2) and (4,4,1) require |α|≈0.5, directly confirming that proximity to the quantization boundary governs flip susceptibility.

## Critical Assessment

**Novelty:** High within mechanistic interpretability of arithmetic. The IRST framework is a genuine conceptual advance over prior work: Nanda et al. 2023 studied modular addition and found clock/helix representations; this paper extends to multi-operand integer addition and identifies the carry fiber structure. The Noisy Quantization Model with a quantitative σ estimate and bathtub prediction is new and falsifiable. The "probe versatility from manifold topology" explanation is elegant and more principled than prior hand-wavy appeals to "information content."

**Rigor:** Good but circumscribed. The NQM achieves R²=0.80—a solid but not exceptional fit; the authors acknowledge approximations (Appendix F). The UMAP visualizations are compelling but inherently lossy 2D projections; Appendix B reportedly validates with other reduction methods, which is good practice. Causal steering results are clean. The ablation in Table 3 is informative and correctly isolates the carry bottleneck.

**Reproducibility:** Strong. Single model (Qwen3-4B), clearly defined dataset (N=10,000, 3-term 10-digit addition), public code, full mathematical formalism for Φ and the NQM.

**Significance:** Meaningful for the mechanistic interpretability community and anyone studying LLM arithmetic. The inference-time correction achieves a real (3.3 percentage point) accuracy gain without retraining. The insight that "hallucinations are quantization failures of a correct continuous potential" is a memorable and actionable framing.

**Red Flags / Limitations to Note:**
- **Single model, single task:** All core results are on Qwen3-4B doing 3-term 10-digit addition. The paper acknowledges BPE-model generalization is a "frontier for future research." It is unclear whether the IRST geometry holds for GPT-family or other architectures, or for tasks beyond addition (multiplication, multi-step reasoning).
- **UMAP interpretability caveat:** The 2D embedding is visually persuasive but the paper relies heavily on it as evidence. The geometric conclusions depend on dimensionality reduction fidelity that is hard to fully verify.
- **Correction gain is modest:** 89.56% vs. 86.26% original—a 3.3pp gain. The TP correction rate of 30–44% means the method misses the majority of errors.
- **σ≈0.05 is a single-task estimate:** Cognitive noise level likely varies with model size, context length, and operand count; these dependencies are deferred to Appendix G.
- **No comparison to finetuning:** The correction method is compared only to inference-time baselines; no comparison to a lightly finetuned version of the same model is provided.

## Strengths

- The IRST geometric framework is theoretically coherent and empirically grounded in multiple complementary analyses (UMAP, probing, causal steering, NQM curve fitting).
- Provides a unified explanation for three previously disconnected phenomena: arithmetic fragility, probe versatility, and the off-by-one error pattern.
- The Noisy Quantization Model is a rare example of a quantitative, predictive mechanistic model in interpretability—not just a post-hoc description.
- ICML 2026 publication with public code makes this a high-credibility, reproducible contribution.
- The dual-stream correction method is practical and training-free, demonstrating that the geometric insight has engineering value beyond pure analysis.

## Limitations

- Results are demonstrated on one model (Qwen3-4B) and one task (3-term 10-digit addition); generalization to other architectures, tokenization schemes (BPE), and arithmetic operations is unverified.
- The inference-time correction corrects fewer than half of errors (TP Corr 30–44%), limiting practical impact.
- UMAP as the primary visualization tool introduces projection artifacts; high-dimensional geometric claims rest on 2D embeddings.
- The Cognitive Noise Level σ is fit empirically to one setting; its dependence on task complexity, operand count, and model scale is an open question.
- Multi-step reasoning or chained arithmetic (where carry errors propagate) is not studied; the framework's applicability to longer reasoning chains is unclear.
- No analysis of whether the IRST geometry emerges from training dynamics or is an artifact of single-digit tokenization.

## Relevance

与 LLM 机制可解释性和算术推理研究直接相关，尤其适合关注"模型内部如何编码结构化计算"的研究者。对于研究 LLM 推理可靠性、推理时干预方法或表示几何的学者，本文提供了可复用的分析框架（IRST + NQM）和可直接借鉴的推理时纠错策略。

## Questions for Further Exploration

1. Does the IRST geometry and Noisy Quantization Model generalize to models with BPE tokenization (e.g., GPT-4, Llama), where digit grouping introduces additional representational entanglement?
2. How does the Cognitive Noise Level σ scale with number of operands, digit length, and model parameter count—is there a phase transition where carry propagation becomes catastrophically unreliable?
3. Can the carry-potential probe be used proactively during generation (e.g., beam search re-ranking) rather than reactively for post-hoc correction, and what is the accuracy ceiling?
4. Does the IRST structure appear in intermediate layers, and if so, at which layer does it crystallize? This could inform targeted fine-tuning or activation patching strategies.
5. What is the relationship between the helical/trigonometric representations found by Kantamneni & Tegmark (2025) and the IRST fiber structure—are they compatible descriptions at different scales, or genuinely competing accounts?
