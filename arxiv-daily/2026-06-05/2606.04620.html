Methodology

Three-Stage PTQ Pipeline

A
Network Analysis
Parse architecture → identify Common Attention Blocks (CABs). Embedding and output layers stay at FP16.
B
Block Compression
Per-block sensitivity profiling (2–16 bit candidates). Precision lockout rejects configs exceeding T_loss. Activation scaling via per-CAB map M.
C
Config Selection
Evaluate remaining valid configs on WikiText-2/103. Select minimum memory footprint within T_perf = 5% PPL budget.
  • Block-0 to Block-2 consistently most sensitive across all four models
  • Middle blocks tolerate 6–8 bit; only single A100 40GB needed
  • T_loss = 2% and 5% converge to same final config — threshold robustness confirmed

Strengths & Limitations

What QuBLAST Gets Right

  • CAB abstraction handles hybrid SSM (Falcon H1R-7B) — most PTQ papers ignore non-standard attention
  • Cross-entropy loss (not perplexity) as sensitivity metric avoids over-inflation of small differences
  • No retraining: any pre-trained HuggingFace checkpoint is compatible
  • Threshold robustness: 2% and 5% lockout thresholds produce identical selected configurations

Where It Falls Short

Critical edge

Zero downstream task evaluation — perplexity preservation does not guarantee MMLU/instruction-following preservation. Activation scaling map M derivation is underspecified (not reproducible from the paper alone). No latency or throughput reported. All models are 7–8B; 13B+ and MoE architectures untested.

Open Questions

Further Exploration

  1. Does 40–45% memory savings translate to actual inference speedup, or do mixed-precision layers hurt kernel efficiency?
  2. How is the activation scaling map M derived — learned, calibrated via percentile clipping, or heuristic?
  3. Does the CAB abstraction hold for MoE models where individual experts may have divergent sensitivity profiles?
  4. What is the actual calibration cost (forward passes, wall time) vs. GPTQ or AWQ at equivalent quality?
  5. Can sensitivity profiles calibrated on Llama3-8B transfer to Llama3-70B, saving re-calibration cost?
