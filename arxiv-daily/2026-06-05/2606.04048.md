> **一句话总结：** 本文首次为 Gated Delta Network（GDN）严格推导了 Maximal Update Parametrization（μP），通过坐标尺度分析揭示门控权重矩阵需要非标准的 Θ(1/√d) 学习率缩放，实验证实该配置在 AdamW 和 SGD 下均实现零样本学习率迁移，而标准参数化彻底失效。

> **TL;DR:** This paper derives the first complete μP formulation for Gated Delta Networks (GDN), a promising sub-quadratic linear recurrent architecture. The key finding is that GDN's data-dependent gating mechanisms require non-standard learning-rate scalings (Θ(1/√d) for gating weight matrices and Θ(√d) for scalar gating parameters under SGD) that deviate from standard μP. Pre-training experiments on models from 21M to 342M parameters confirm zero-shot learning-rate transfer across widths under both AdamW and SGD, while standard parametrization fails to transfer.

## 核心贡献

1. **GDN 的完整 μP 推导**：本文是首个为 Gated Delta Network 建立 μP 一致参数化的工作，填补了线性递归架构超参数迁移理论的空白。此前唯一相关工作 Vankadara et al. (2024) 仅处理对角 SSM（如 Mamba），而 GDN 的全矩阵隐状态更新（通过外积 delta 规则）使该工作的分析完全不适用。

2. **门控的非标准缩放规律**：发现 GDN 中门控权重矩阵 **W_α** 和 **W_β** 在 SGD 下需要 Θ(1/√d) 的学习率缩放（而非标准 μP 的 Θ(1)），标量门控参数 a_log 则需要 Θ(√d) 缩放，这两者均偏离标准 μP 配置。

3. **AdamW 与 SGD 的统一处理**：AdamW 的坐标归一化使门控权重的梯度尺度差异被消除，因此 AdamW 下 GDN 的 μP 配置与标准架构一致；而 SGD 下门控权重的不同梯度量级需要单独处理，是本文理论推导的核心难点。

4. **实验验证**：在 FineWeb-Edu 100B 数据集上，跨 d ∈ {256, 512, 1024, 1536}（对应 21M–342M 参数）进行预训练，所提出的 μP 配置在两种优化器下均实现学习率零样本迁移，而 SP 和原版 μP 均失败。

---

## Key Contributions

1. **First complete μP derivation for GDN**: This is the first work to establish a μP-consistent parametrization for Gated Delta Networks, closing a gap in hyperparameter transfer theory for linear recurrent architectures. The only prior related work (Vankadara et al., 2024) handled only diagonal SSMs (e.g., Mamba); GDN's full-matrix hidden state updated via outer-product delta rules makes that analysis inapplicable.

2. **Non-standard scalings from gating**: The paper discovers that gating weight matrices **W_α** and **W_β** require Θ(1/√d) learning-rate scaling under SGD (vs. Θ(1) in standard μP), while scalar gating parameters a_log require Θ(√d) scaling — both deviating from the standard μP law.

3. **Unified treatment of AdamW and SGD**: AdamW's coordinate-wise gradient normalization absorbs the magnitude differences from gating, making the GDN μP configuration identical to standard architectures under AdamW. SGD lacks this normalization, requiring explicit non-standard scalings for gating weights — this is the core theoretical challenge of the paper.

4. **Empirical validation**: Pre-training on FineWeb-Edu 100B across d ∈ {256, 512, 1024, 1536} (21M–342M non-embedding parameters) confirms that the proposed μP achieves zero-shot learning-rate transfer under both optimizers, while SP and original μP both fail.

## Methodology

### 背景：Gated Delta Net 架构

GDN 的隐状态更新规则为：

**S_t = S_{t-1}(α_t(I - β_t k_t k_t^T)) + β_t v_t k_t^T**

其中 α_t ∈ (0,1) 是数据依赖的门控尺度，β_t ∈ (0,1) 是写入强度，**q_t, k_t, v_t** 是经过短卷积和 L2-归一化后的查询/键/值向量。该架构在 delta 规则基础上增加了类似 Mamba-2 的数据依赖门控，是一种全矩阵隐状态的线性 Transformer 变体。

### μP 前向传播分析（坐标尺度估计）

**关键结论链条**：
- 经 L2-归一化的查询 **q_t** 和键 **k_t** 具有 Θ(1/√d) 坐标尺度
- 写入矩阵 U_t = β_t v_t k_t^T 的每个元素量级为 Θ(1/d)
- 由几何级数稳态分析，累积隐状态 **S_t** 的坐标尺度为 Θ(1/√d)
- 读出 o_t = S_t q_t 的量级为 Θ(1/√d)，需插入 √d 乘子后再接 RMSNorm，使后续投影权重的标准 μP 配置得以适用

**门控机制的非高斯性挑战**：α_t 和 β_t 是有界 sigmoid 激活，不服从高斯分布，标准 μP Tensor Programs 框架不能直接适用。论文通过一阶分析证明，在 μP 初始化下，∂α_t/∂z_{α,t} 和 ∂β_t/∂z_{β,t} 均为 Θ(1)，使坐标尺度分析对门控权重仍然有效。

### SGD 下的后向传播分析

各权重矩阵的梯度量级和对应 SGD 学习率（以 n_{ℓ-1} 表示宽度）：

| 参数类别 | 梯度量级 | SGD 学习率 |
|---------|---------|-----------|
| 投影权重 W_q, W_k, W_v | Θ(1/d) | Θ(1) = n_ℓ / n_ℓ |
| 门控权重 W_α, W_β | Θ(1/√d) | **Θ(1/√d) = Θ(1/√{n_{ℓ-1}})** |
| 标量门控 a_log, b | Θ(1/√d) | **Θ(√d) = Θ(√{n_{ℓ-1}})** |

**W_α 的梯度推导要点**：∂L/∂α_t = Tr((∂L/∂S_t)^T · ∂S_t/∂α_t)。由于 S_{t-1} q_t 和 S_{t-1} k_t 均为 Θ(1/√d) 量级，最终 ∂L/∂α_t = Θ(1/√d)。对应的 W_α 梯度 ∂L/∂(W_α)_j 也是 Θ(1/√d)，而预激活 z_α 对 W_α 的总变化量为 Δz_α = η_α · Θ(√d)，需令 η_α = Θ(1/√d) 才能满足特征学习条件 Δz_α = Θ(1)。

**标量参数的相反缩放**：a_log 作为乘性标量进入 α_t，其梯度同样为 Θ(1/√d)，但由于 a_log 直接作为加性标量影响预激活（不经过 d 维求和），Δb = η_b · Θ(1)，需令 η_b = Θ(√d)。

### AdamW 下的简化

AdamW 通过坐标方向归一化有效实现符号下降：Δθ_j ≈ -η_θ · sign(g_j)。由于梯度量级（Θ(1/√d)）远大于 Adam 的 ε = 10^{-8}，AdamW 在尺度不变区域运行，消除了原始梯度尺度差异。因此门控权重和投影权重的学习率缩放规律相同，均为 Θ(1/n_{ℓ-1})，与标准 μP 一致。

## Results & Highlights

### 主要实验设置

- **数据集**：FineWeb-Edu 100B，20k 步，序列长度 1024，全局 batch size 480（约 9.83B tokens）
- **模型规模**：8 层，6 注意力头，宽度 d ∈ {256, 512, 1024, 1536}（AdamW）或 {256, 512, 768, 1024}（SGD）
- **硬件**：1 块 NVIDIA H100 80GB HBM3
- **基线框架**：nanoGPT（Karpathy, 2022）
- **评估指标**：Shifted validation loss（各宽度验证损失减去该宽度最优损失，以去除宽度间的绝对损失差异）

### 核心结果

**AdamW（图1）**：
- 标准参数化（SP）下，最优学习率随宽度显著变化（从 d=256 到 d=1536 有明显漂移）
- 本文 μP 配置下，4 个宽度的最优学习率曲线高度一致，实现零样本迁移
- 最优学习率约为 8×10^{-3}

**SGD（图2）**：
- SP 和原版 μP 配置下，最优学习率均随宽度漂移
- 本文 μP 配置下，最优学习率在所有宽度下完全一致（约为 4×10^{-1}）

**动态探针验证（附录 C/D）**：
- 前向传播坐标尺度：本文配置下 q̃/√d_head × √d 和 k̃/√d_head × √d 跨宽度近似常数，SP 和原版 μP 则不然
- 后向传播梯度缩放：∂L/∂h 在本文配置下严格保持 Θ(1/d) 量级，SP 下退化为 Θ(1/√d) 导致不稳定
- 门控稳定性：β_t 的均值跨宽度稳定在 0.5 附近，且有大量 token 触发强写入（β_t > 0.5），验证了一阶近似的有效性

## Critical Assessment

### Strengths

1. **理论严谨性**：论文对坐标尺度的传播进行了完整的前向和后向分析，对 GDN 架构的每一个组件（投影权重、门控权重、标量参数、RMSNorm 位置、L2-归一化雅可比）都给出了显式的量级估计，推导链条清晰完整。

2. **非平凡的理论发现**：揭示了门控权重和投影权重需要不同学习率缩放的根本原因——门控路径的梯度量级与标准投影路径不同，这一发现对所有含类似门控机制的线性架构均有启示意义。

3. **AdamW vs SGD 的差异化处理**：明确指出 AdamW 因梯度归一化而"吸收"了门控权重的非标准缩放，而 SGD 不具备此性质。这一分析深化了对两类优化器在非标准架构下行为差异的理解。

4. **实验验证充分**：不仅报告验证损失，还提供大量动态探针图（前向坐标尺度、后向梯度尺度、门控稳定性、隐状态相关性），既验证了理论假设，也提供了实际训练健康度的诊断工具。

5. **开源代码**：有完整的 GitHub 实现，且专门适配了 Amazon Trainium 硬件，实用价值高。

### Limitations

1. **短记忆假设的局限**：论文的梯度分析（Assumption 5.1）假设有效记忆长度 L_eff = O(1)，即门控值 α_t 衰减足够快。当模型用于长上下文场景（α_t → 1，L_eff = O(T)）时，μP 学习率处方需相应缩小 1/L_eff，论文未深入处理此情形。

2. **SiLU 激活的近似性**：短卷积后的 SiLU(x) = x/(1+e^{-x}) 引入非高斯统计，论文的坐标尺度分析假设将其抑制，并以 Remark 4.1 说明这是"一阶近似"。实际影响有多大在实验中未被单独量化。

3. **规模仍然有限**：最大模型仅 342M（非嵌入参数），在 1 块 H100 上训练 20k 步。μP 的核心卖点是从小代理模型迁移到实际大模型的能力，论文并未展示向数十亿参数级别迁移的实验。

4. **仅对 GDN 架构有效**：论文明确指出分析不适用于其他线性架构（如 Mamba、RWKV 等），每种架构需要单独推导，通用性受限。

5. **计算开销未讨论**：论文的 μP 配置需要对不同参数类别设置不同学习率，在实际工程实现中需要分组处理，但论文未讨论实现细节和额外开销。

- **LLM 训练基础设施**：随着 GDN 类架构在长上下文推理场景的应用增加，本文提供的稳定训练配置具有直接工程价值。

## Questions for Further Exploration

1. **长上下文情形**：当 α_t → 1 使 L_eff = O(T) 时，论文建议将 μP 学习率缩小 1/L_eff。这个修正是否在实际长文档训练中有显著影响？是否存在自适应的 L_eff 感知学习率调度？

2. **其他线性架构的 μP**：Mamba-2、RWKV、RetNet 等架构的 μP 推导是否可以复用本文的分析框架？门控机制的存在是否是引发非标准缩放的充分条件？

3. **规模外推**：论文在 342M 以内验证了零样本迁移。在 1B、7B 甚至更大规模下，本文的 μP 配置是否依然有效？GDN 与 Transformer 在同等参数下的最优学习率是否在同一量级？

4. **混合架构**：越来越多的 LLM 使用 Transformer + SSM 混合层（如 Jamba、Zamba）。当 GDN 层与标准注意力层混合时，如何统一两者的 μP 配置？是否存在层间学习率不一致的问题？

5. **超越语言建模**：本文的实验仅限于语言模型预训练。对于序列到序列任务、强化学习等场景，GDN 的 μP 配置是否仍然有效？门控行为在不同任务分布下是否保持稳定？
