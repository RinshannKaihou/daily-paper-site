Methodology

Three-Stage Differentiable Pipeline

  • Continuous parameterization: Single scalar β ∈ [0,2] per layer defines format as E(2+β)M(1+β), collapsing MXFP4/MXFP6/MXFP8 into one real axis — the quantizer remains differentiable w.r.t. β
  • Temperature annealing: Sigmoid β = 2/(1+e^{−T·(β̂−0.5)}) starts near-linear (T=small, explore) and sharpens to step function (T=400, commit) — avoids STE oscillation artifacts
  • Target-aware regularizer: Penalty max(0, λ·(b̄_current − b̄_target)) is zero below budget, linear above — lets users specify a bit-width budget directly without tuning λ magnitude
  • SpinQuant integration: Jointly optimizes rotation matrices and bit-widths on 3200 FineWeb calibration samples, 400 steps

Strengths & Limitations

Strengths

  • First gradient-driven bit-width assigner for OCP MX floating-point format family
  • Single scalar β parameterization: combinatorial search → 1D continuous optimization
  • Temperature annealing eliminates STE oscillation between FP4 and FP8
  • 400 calibration steps only — far faster than QAT; seamlessly layers onto SpinQuant
  • Closed-form gradients ∂L/∂m, ∂L/∂e verified against PyTorch autograd (Appendix A)

Limitations

Critical edge

1–1.7B only — layer sensitivity differences at 7B+ may require different temperature schedules. No real AMD GPU latency data. Shared β for weights and activations ignores their different sensitivity. λ search space spans 5 orders of magnitude despite "robustness" claim.

Open Questions

Further Exploration

  1. Does temperature annealing schedule need retuning at 7B / 70B where layer sensitivity gradients are steeper?
  2. Can weights and activations each get independent β — and what's the accuracy-complexity trade-off?
  3. Does the framework extend to INT4 + FP8 mixed formats, or is it constrained to the MX format family?
  4. How much does FineWeb calibration domain drift affect bit-width policy learned for code or math tasks?
  5. What actual inference latency change appears on AMD CDNA/RDNA hardware running mixed MXFP layers?
