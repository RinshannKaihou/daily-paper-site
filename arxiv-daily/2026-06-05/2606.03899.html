Key Finding — 2606.03899 · 2026-06-05

Momentum in Muon is a spectral low-pass filter: applying it before orthogonalization is provably optimal — denoise first, orthogonalize later

arxiv.org/abs/2606.03899 ↗ Li, Zhang, Liu, Bao · ISM / SOKENDAI / NII / CUHK / RIKEN AIP
Key Metric
~3.0
LLaMA 350M val loss (Pre-polar, step 3000) — vs ~3.4 Post-polar, ~4.5 Polar-only. Lower is better.
Why This Matters
Prior theory predicted heavy momentum hurts in matrix optimizers. This paper proves it helps — giving the mechanism that resolves the contradiction and a design rule applicable to Shampoo, SOAP, and any optimizer ending in a spectral step.
Critical Edge
Analysis excludes Nesterov momentum — which is what practical Muon implementations actually use. The BVMZOS noise model and time-invariant signal are idealizations. Theorem 3 restricted to rank-1 Gaussian.
Methodology
Core Insight: Noncommutativity
Momentum and orthogonalization do not commute. The polar factor O(·) replaces all singular values with 1, erasing the amplitude gap between signal and noise. If applied first (Post-polar), momentum has no structure to exploit. If momentum acts first (Pre-polar), it suppresses noise before O(·) discards singular values — so the polar factor sees a matrix already aligned with the signal.
Pre-polar (standard Muon)
ΔW_t ∝ O(M_t) [momentum → polar]
Momentum filters noise first; orthogonalization sees denoised signal subspaces. Provably dominant.
Post-polar (Orthogonal-SGDM)
M̃_t = β M̃_{t-1} + (1-β)O(G_t) [polar → momentum]
Polar erases amplitude structure before momentum can exploit it. No denoising benefit.
Polar-only (no momentum)
ΔW_t ∝ O(G_t) [polar of raw gradient]
No filtering at all. Signal alignment vanishes as m→∞ in low-SNR regime (Theorem 3).

Gradient Model

Gradients modeled as G_t = G_sig + Ξ_t where G_sig is a coherent low-rank signal (rank r, time-invariant) and Ξ_t is BVMZOS noise (bounded variance, mean-zero, pairwise orthogonal in expectation). Effective window: T = 1/(1−β).

Key Theorem (Theorem 1)

Spectral gap between signal and noise singular values in momentum buffer M_t grows with T. Noise singular values bounded above by O((2T−1)^{−1/4}). Subspace alignment error decays as O(T^{−1/4}). Larger β = better denoising — opposite of classical predictions.

Strengths / Limitations
Strengths
  • Resolves a real theory-practice contradiction: prior theory predicted heavy momentum hurts; this paper proves it helps with the correct mechanism
  • "Denoise first, then orthogonalize" is a memorable, actionable design rule for any optimizer ending in a spectral step
  • Theorems 1–3 have direct empirical counterparts validated via stationary and trajectory probes
  • Quantitative prediction (2T-1)^{-1/4} floor verified in real NanoGPT gradient spectra
  • LLaMA 350M end-to-end validation is non-trivial and shows consistent improvement throughout training
Limitations
  • Analysis excludes Nesterov momentum (ν M_t + κ G_t) — what practical Muon code actually uses
  • BVMZOS noise and time-invariant signal: idealized model; real LLM gradients drift significantly
  • Theorem 3 restricted to rank-1 spiked Gaussian; general rank-r extension deferred
  • Time-varying β schedules (warmup ramp) not analyzed
  • O(T^{-1/4}) convergence rate is slow — bound may be loose given empirical results
  • No wall-clock comparison: does Pre-polar beat Polar-only per compute unit?
