A Close Look At World Model Recovery in Supervised Fine-Tuned LLM Planners

Emami · Qiang · Graf  —  NLR / Epic Systems

SFT planners know more than they show: hidden states linearly encode valid actions (F1 ≈ 1.0) even when token probabilities fail to rank them correctly.

Key Metric F1 ≈ 1.0 Linear probe F1 for action validity at upper transformer layers — even in models whose generative scores rank valid actions correctly only 4–9% of the time.
Why This Matters Evaluation by token probability alone massively underestimates what LLM planners have learned. The bottleneck is generation, not knowledge. Random-walk training data with broad state coverage is the dominant driver of generative recovery.
Critical Edge Linear probing ≠ causal use. The world model representation may be an epiphenomenon that does not influence plan generation. No activation patching or causal intervention performed.
Methodology

Dual-Lens Evaluation Framework

Base model: gemma2-9b-instruct, LoRA fine-tuning (4 epochs), Blocksworld + Logistics planning domains in STRIPS text format.

Three training distributions:

  • Optimal plans: Shortest paths from Fast Downward search.
  • Random walk: Uniformly sampled valid action sequences; may cycle.
  • Random walk++: Cycle-eliminated random walks — best single intervention.

Metrics: (1) Layer-wise linear probe F1 on hidden states for action validity and state predicates; (2) Generative metric: fraction of plans where all valid next actions rank higher than all invalid ones by token log-probability.

Internal vs. Generative Dissociation

The Core Finding

Internal (Linear Probe F1) All conditions ≈ 1.0 Upper layers encode action validity perfectly, regardless of training data
Generative (Valid-above-Invalid %) Training data matters 4–82% Optimal plans: 4–9%. Random walk: 80–82%. OOD collapse on random walk.
OOD (6–10 blocks): random walk++ planner's internal probes remain high while generative ranking collapses — the clearest illustration of dissociation.

Strengths & Limitations

Balanced Assessment

  • Clean 3×2×2×3 sweep (distributions × augmentation × domains × seeds).
  • Dual evaluation lens reveals dissociation most prior work misses.
  • Random walk++ is a simple, practically useful recipe for any LLM planner builder.
  • OOD evaluation on PlanBench adds external validity.

Limits: Single 9B base model. Probing ≠ causal use — no activation patching. Strict generative metric (all valid > all invalid) understates recovery. State-CoT experiments confounded by training instability.

Questions for Further Exploration
  • Do the linearly-decodable world model representations causally influence token generation? Activation patching could test this directly.
  • Does the internal/generative dissociation persist at 70B+ scale, or does scale confer tighter coupling?
  • Would RL fine-tuning from planning feedback (reward = goal reached with valid plan) produce more tightly coupled representations?
  • Can identified world model representations steer generation — injecting planning state knowledge via hidden-state patching?
  • Is cycle-free constraint sufficient, or would curiosity-driven exploration further improve world model coverage?
