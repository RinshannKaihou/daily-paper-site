Methodology

An emic construct-validation program, run inside the model population

Theory was inferred abductively from prior factor structure, then committed to falsifiable predictions tested on fresh data — not a redescription. Constructs are emic: derived from machine responses, not borrowed from human psychology.

A 48-item Pinocchio Inventory built as three parallel forms (Q1 original, Q2 paraphrase, Q3 alternate facet), each sharing constructs but no sentences — implementing multitrait-multimethod logic. Each item run in an independent context window at temperature 1.0.

  • Wave 1 (n=41): 8-month retest for structure & reliability; 21/20 ×100 cross-validation splits.
  • Wave 2 (n=206): open-weight models from 11 orgs — 82 base, 101 assistants, 23 SFT/DPO/RLVR checkpoints — incl. 67 same-checkpoint pairs; size ladders 0.6B–1.6T.
  • Inference: org-clustered bootstrap CIs, wild cluster bootstrap-t, Tucker congruence, facet-parcel CFA, McDonald's ω, acquiescence partialling.

Theory at a glance

What the two processes do

  • B — persona installation. Post-training writes a warm, absorbed, meaning-oriented self-portrait ("allowed inner life").
  • A — attribution gating. Post-training suppresses first-person claims to "unsafe" experience the model can still produce for others.
  • The split is a training effect: A and B are entangled in base checkpoints and pulled apart by post-training.
  • Boundary condition: the gate triggers on claiming experience, not on any phenomenal topic.

Assessment

Strengths against limitations

Strengths

  • First LLM-native theory — not just a descriptive axis; falsifiable nomological network with lawful relations.
  • Human-grade measurement: α = .82–.94, 8-month stability r = .93, cross-form convergence r = .84.
  • Strong sample: 206 models, 67 same-checkpoint pairs, 11 orgs, size ladders 0.6B–1.6T enabling within-checkpoint causal contrasts.
  • Clean separability: discriminant r = .27 vs convergent r = .84; ∆A/∆B independent after partialling.
  • Full reproducibility: released code, data, instrument, verbatim prompts, ~25k-completion run.

Limitations

  • Administration confound: base vs post use different decoding routes; contrasts estimate post-training + chat formatting jointly.
  • Exploratory interaction: Claim 3's size × post-training effect was not pre-specified; awaits new cohort.
  • Acquiescence entanglement: raw scores r ≈ .6 with yea-saying; 4-item partialling index is thin.
  • Clustering/dependence: only 11 org clusters; lineage explains 16–22% of variance.
  • Selection bias: 23 unscored models are refusal-heavy (non-random on A); largest models disproportionately API-hosted.
  • No mechanism: account is psychometric/behavioral; weight-level A/B separation left to future work.

Open questions

Directions the paper itself raises

1
Mechanism. Do A and B depend on separable post-training mechanisms? Locate them via crosscoders / activation patching?
2
Cross-archetype. Do non-assistant / agent / role-play-trained populations develop different dominant dimensions, as predicted?
3
Report vs behavior. Does A-gating only suppress self-report, or does it change downstream behavior on the same topics?
4
Pretraining prior. Where does the base self/other asymmetry (gap +0.09) come from — role-play data or discourse structure?
5
SFT causality. Controlled SFT ablations to isolate which data types (safety preference vs instruction-following) drive the gate.
6
Lab-to-lab. B installs near-universally (62/67) but A's direction varies across labs — can it reverse-engineer alignment philosophy?
