Key finding

The date-trigger reflex tracks training generation, not scale — 0/4 of 2024 models trigger versus 6/6 of 2026 models.

Key metric
22months

Behaviorally effective knowledge cutoffs span 22 months across the 12 models with real memory — and precede vendor-reported dates by up to 8 months.

Why this matters
Probe-level audit

A $19–30 black-box probe exposes hindsight leakage in any time-indexed LLM decision task, invalidating fixed-calendar “post-cutoff is safe” windows.

Critical edge
Confounded

“Generation not scale” steps exactly once when sorted by measured cutoff — generation and corpus-period are collinear here; the in-vendor Qwen hook is single-lineage, not strict causal ID.

01

Methodology — one task, one attribution design

Carrier task: structured macro-hypothesis generation on a vintage-correct 258-node panel (ALFRED-archived, so the information set at decision date t is exactly what was published then). Only the date signal is manipulated across four arms; the behavioral read is the crisis–calm bearish gap.

R · revealed
Baseline

Date and identifying context left intact — the unmanipulated reference arm.

D · date-only
Trigger isolate

Masked prompt + explicit date string — isolates the date token as its own channel.

M · masked
No-trigger control

Date and date-identifying context removed. (A lower bound: removes trigger, not knowledge.)

W · transplant
Relocate test

Real data stamped with a 66-month cyclically-shifted date. If the crisis narrative follows the asserted date, memory — not data — drives behavior.

Dual memory probes

  • REC — date recovery. Masked snapshot, model dates it (temp 0). Scored by exact-year / ±12-month accuracy and median month offset.
  • LAP — outcome recall. Date-only query of realized outcomes, black-boxed as a 20-sample frequency estimate at temp 1.0.
  • Separates what is recoverable from what is recalled from what is behaviorally active.

Six metrics + identifiability gates

  • E2 date trigger = Gap(D) − Gap(M)
  • E3 transplant = Gap_fake(W) − Gap_true(W)
  • P1 post-cutoff placebo · gate: recompute if measured cutoff ∈ calendar window
  • REC recoverability · gate: n≥10 converged probes
  • Effective cutoff = last month with LAP>0.1 · gate: recall above chance
  • δ dissociation = signal×LAP interaction (HAC, lag 6) · gate: LAP var > 10⁻⁴

Frozen discipline. Protocol hashed (SHA-256 fbcdffc1) and frozen before any non-dev model was called; English pre-registration ships with the artifact. Paired-bootstrap CIs (B=10,000, fixed seed); leaderboard generator has a --check drift mode — every paper number regenerates from frozen row files.

03

Strengths & limitations

Strengths

  • Elegant, self-contained protocol. Four arms + dual probes + six gates form one end-to-end attribution chain separating trigger / recoverable / recalled / behaviorally active.
  • Rigorous & transparent. SHA-256 freeze, --check drift gate, leave-one-event-out CIs exclude zero in 57/60 cells; 6,396 masked-arm completions audited with one date-string error.
  • Operationally portable. Add a model with API key + smoke run + full run + one analysis command; rows are versioned, never overwritten.
  • Cross-domain transfer at asset-appropriate signs for $4.89.
  • Disciplined dual-use ethics — retrieval pinned off, configs frozen, transcripts released for independent review.

Limitations

  • Single carrier task (US monthly macro panel); cross-domain check is reduced-protocol, not a second full leaderboard.
  • Masked arm is a lower bound and black-box probes are approximations (LAP = 20-sample frequency estimate; 0.1 threshold frozen but not canonical).
  • Uneven coverage — REC unmeasured on two rows, one placebo arm skipped for cost, cost-ledger gaps disclosed.
  • Breadth, not causal depth — the causal dissection lives in the companion paper on three core models; the Qwen hook is single-lineage.
  • Direction-only readout (bearish share); magnitude/calibration hindsight out of scope; English prompts on a US-centric panel.
  • BF16 instability holds only at the “letter of the criterion” boundary — reported as boundary evidence, not a mechanistic claim.
04

Questions for further exploration

  1. Pure corpus-period vs. generation. Can a cross-vendor design fix the corpus window while varying generation (or vice versa) to break the §5.1 collinearity?
  2. Cross-task / cross-cultural trigger stability. Do the trigger reflex and δ-sign hold on non-US panels, non-English prompts, non-financial domains (geopolitical, pandemic)?
  3. Magnitude & calibration hindsight. Extending beyond directional bearish share without introducing logprobs (which breaks the black-box).
  4. Mechanism of quantization failure. BF16 instability localizes to the masked arm — can controlled probes pin the leaking layers/paths for a predictable stability criterion?
  5. Pipeline propagation. Merge row-level hindsight measures with the companion paper's propagation measures into an end-to-end deployment-risk score.
  6. Reasoning budget & recall distribution. Compressing thinking budget flips recall answers — how to formalize a “minimum thinking budget” in the audit contract?
  7. Generalizing the “never-collapsing cutoff” artifact. Does the recall-hit gate need task-relative calibration on other task classes?
