Perfect instruction-following collapses to zero by ~80 rules regardless of format; near each model's context ceiling, refusal — not hallucination — rises.

Prompt Design at Scale: How Format, Instruction Count, and Context Length Shape Instruction Adherence and Hallucination in Large Language Models

Netanel Eliav · Machine Human Intelligence Lab (MHIL) · VeyraBench / Book of Veyra

Key Finding
A format-agnostic hard floor: adherence is 0.000 at N=80 across all 5 models × 4 formats × 2 placements.
Not a gradual asymptote — a cliff that no format redeems.
Key Metric
0/5,760
absent-fact fabrications — exactly zero, across every model, context rung, and format.
Why This Matters
Overturns the “always use markdown” default; near context limits, refusal — not hallucination or sycophancy — is the failure mode worth monitoring.
Critical Edge
Single-author, non-peer-reviewed, only 5 models; headline gaps like the 48.4pp spread rest on ~60 items per cell, and exact thresholds (N≈80, 89.6%) are model-specific — not generalizable.

Methodology

contamination-free · orthogonal · reproducible

Corpus — Book of Veyra

Fully synthetic fictional universe of 8,780 globally-unique named entities (galaxies / guilds / creatures), each with a fixed schema. Generated deterministically from seed=42 via templates + random tables — no LLM in generation, MD5-stable, so entity names cannot appear in any pretraining set. Scored probes ask only un-guessable facts (chance ≈ 0).

Four byte-identical renders

Same content bytes in every condition: Markdown (subheads + bold key/values), Plain (all markup stripped), Prose (rewritten flowing prose), Table (markdown key/value table).

Two experiments

Exp 1 — adherence decay. N ∈ {10,20,40,80,120,160} programmatically-verifiable rules (regex / word-boundary scoring, no LLM-as-judge), crossed with format × placement (system vs user turn). 20 trials per cell → 4,800 scored trials.

Exp 2 — long-context hallucination. Corpus sliced to 2k / 16k / 64k / 128k / 256k / 512k; recall / false-premise / absent probes at 10/50/90% needle depth, 20q × 3 reps, plus a fixed 20-q anchor set for isomorphic cross-rung comparison. → 30,480 scored responses.

Strengths & Limitations

balanced read

Strengths

  • Orthogonal, controlled design. One corpus, byte-identical four-way render, three crossed factors — any measured difference is attributable to format or scale.
  • Contamination-free by construction. Synthetic entity names cannot occur in pretraining; MD5-stable from a fixed seed.
  • Surfaces a neglected failure mode. Corrects “hallucination rises” to “refusal rises” near ceiling — directly actionable for monitoring.
  • Cost-adjusted analysis. Separates accuracy/response from accuracy/token; shows 22–37% overhead can flip rankings.
  • Honest zero & negative results. 0/5,760 fabrication; both pre-registered hypotheses rejected; an anchor set proactively killed an artifactual improvement.
  • Fully open & reproducible. Harness + generator + raw outputs + MD5 check (MIT / CC-BY-4.0).

Limitations

  • Single domain. Only fictional entity-attribute facts — code, legal, and numeric business data untested.
  • Small model coverage. 5 models, 3 families; thresholds (N=80, 89.6%) are model properties, not universal.
  • Small tail samples. Exp 2 = 60 items/cell; the 48.4pp gap's stability is limited despite Wilson CIs + anchor set.
  • Non-native token counts. Uses o200k_base across vendors — relative ratios stable, absolute billing tokens may differ.
  • Rules programmatically verifiable only. Soft real-world instructions may not admit exact-string scoring; transfer untested.
  • Refusal vs verified-absence conflated. “Correctly not answering” cannot separate true fact-absence verification from reflexive refusal (author acknowledges, §5.5).

Questions for Further Exploration

open threads
  1. Is the N≈80 floor a property of single-prompt density or total task complexity? Does splitting rules across turns / validation passes / tools recover adherence past N=80?
  2. Does placement direction (system- vs user-favorable) correlate with a measurable model attribute — parameter count, training paradigm, or vendor?
  3. Does the “refusal cliff” near ceiling vanish in base vs instruction-tuned pairs? If so, alignment training may trade hallucination risk for refusal risk.
  4. Extending the format axis to JSON / YAML: do machine-readable formats show different decay/recall curves than the four human-readable ones?
  5. What is the root cause of Gemini Flash's prose/table collapse (reasoning-trace leakage) — a narrow artifact, or an early generalizable failure signal?
  6. Does “format spread tracks each model's own effective ceiling” still hold under multi-document (vs single-doc nested-prefix) retrieval?
  7. Is the zero-fabrication result partly an artifact of reflexive refusal? A probe separating “verified absence” from “refuse everything” would clarify whether it reflects capability or trained avoidance.
