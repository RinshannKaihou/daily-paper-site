> **一句话总结：** 提出 *Jacobian Scopes*，将 LLM 输出对输入 token 嵌入的 Jacobian 通过单一向量–雅可比积投影到不同方向，从而在单次反向传播内同时给出 (i) Semantic Scope（特定目标 token 的 logit）、(ii) Fisher Scope（基于 Fisher 信息主方向的整条预测分布）与 (iii) Temperature Scope（有效逆温度/置信度）的 token 级因果归因；在 LAMBADA 与 IWSLT2017 上六个 1B–4B 模型上以 AOPC 指标匹配或超过 Integrated Gradients / Input×Gradient，并在指令偏见、机器翻译与时间序列 ICL 三个案例中产出可解释的机制性结论。

> **TL;DR:** Jacobian Scopes unifies gradient-based token attribution for LLMs by computing a single vector-Jacobian product per input position with three principled projection directions: target unembedding (Semantic), principal Fisher-information eigenvector (Fisher), and the normalized hidden state (Temperature, effective inverse temperature). The method is computed in one backward pass at negligible overhead (~0.027s on RTX A4000). Across LLaMA-3.2 / Qwen2.5 / Gemma-3 at 1B and 3B/4B scales, Fisher and Temperature Scopes match or beat Integrated Gradients and Input×Gradient on AOPC for LAMBADA and IWSLT2017, despite being target-free. Case studies surface implicit political bias (LLaMA attributes "liberal" to the token "Columbia"), word- vs. phrase-level translation strategies on Dante's *Inferno*, and provide evidence for the nearest-neighbor-in-delayed-embedding conjecture of ICL time-series forecasting on Lorenz systems. Authors release code and a HuggingFace demo.

## 核心贡献

- 提出**统一框架**：把"梯度可解释性"统一为 `v⊺Jt` 这一向量–雅可比积；选择不同的投影方向 `v` 即得到不同的 attribution objective，三种方向自然落到三个解释目标上（logit / 分布 / 置信度）。
- 引入**Fisher Scope** 与 **Temperature Scope** 这两个**target-free** 的归因目标：Fisher 通过 Fisher 信息矩阵主特征向量刻画整条 softmax 分布对扰动的最敏感方向；Temperature 通过分解隐藏状态范数 ∥y∥₂ 作为"有效逆温度"，并用近似高斯论证证明 β_eff ∝ σ⁻²，使 ∥y∥₂ 的归因等价于对预测方差的归因。
- 在 AOPC（Area Over the Perturbation Curve）这一**干预式**指标上，target-free 的 Fisher / Temperature Scope 在六个模型 × 两个数据集上**匹配或超过** target-aware 的 Integrated Gradients / Input×Gradient / Semantic Scope。
- 通过三个定性案例给出可解释的机制性结论：政治偏见来源、翻译策略（词级 vs. 短语级）、以及混沌/随机时间序列 ICL 中"延迟嵌入空间最近邻"的证据。
- 开源代码与 HuggingFace Space 交互 Demo，降低复现门槛。

## Key Contributions

- **Vector-Jacobian-product framework.** Influence at input position `t` is defined as `∥v⊺ Jt∥₂`, computed via one custom scalar loss `L = v⊺ y` and one backward pass — no full Jacobian, no path integration, no Shapley sampling.
- **Three projection directions, three explanations:**
  - `v = w_target` (target unembedding row) → Semantic Scope (per-logit, target-specific).
  - `v = u₁`, the principal eigenvector of the Fisher information matrix `F = W⊺ diag(p) − pp⊺ W` → Fisher Scope (distributional, KL-aware).
  - `v = ŷ` (normalized hidden state) → Temperature Scope; under near-Gaussian numerical predictions, this equals attribution of predictive variance.
- **Quantitative evidence.** AOPC on LAMBADA (long-range LM) and IWSLT2017 DE→EN across LLaMA-3.2, Qwen2.5, Gemma-3 at 1B/1.5B and 3B/4B. Target-free Fisher/Temperature Scopes **tie or beat** target-aware Integrated Gradients and Input×Gradient; Fisher Scope reaches −1.81 on Gemma-3 4B LAMBADA vs. Integrated Gradients' −1.43.
- **Target-free distributional insight.** Under AOPC (a target-specific metric), distributional attribution is empirically more faithful than logit-wise sensitivity — supports a principle that *tokens driving the whole distribution also drive any individual prediction in it*.
- **Mechanism-level case studies:**
  - *Bias*: Semantic Scope on LLaMA-3.2 attributes the prediction "liberal" to the token "Columbia" and "conservative" to "the South".
  - *Translation*: Fisher Scope on Dante's *Inferno* (LLaMA-3.2 3B) shows word-to-word translation when source has direct counterparts ("mondo" → "world") and phrase-level strategies when syntax mismatches force multiple plausible renderings.
  - *Time-series ICL*: Temperature Scope on a Lorenz-like chaotic series shows the model attends to history regions matching the local pattern near the cutoff (consistent with the "nearest-neighbor in delayed embedding space" conjecture); on Brownian motion, influence shifts toward later context, explaining the early ICL-loss plateau observed by Gruver et al.
- **Open release.** Code plus interactive HuggingFace demo.

## Methodology

**Setup.** Let `X_{1:T}` be input embeddings in `ℝ^{d}`. The LLM's last-layer post-LN hidden state is `y = f(X_{1:T}) ∈ ℝ^{d}`. Logits `z = W y`, distribution `p = softmax(z)`, with unembedding matrix `W ∈ ℝ^{|V|×d}`.

**Influence score.** Define `J_t = ∂y/∂x_t ∈ ℝ^{d×d}` and pick a projection direction `v ∈ ℝ^{d}`. The token-level influence is

```
Influence_t = ‖v⊺ J_t‖₂
```

which geometrically equals the largest displacement along `v` induced by an `ε`-perturbation of `x_t`. All token influences are obtained by one backward pass of the custom scalar loss `L(X) = v⊺ y`, because `∂L/∂x_t = v⊺ J_t`.

**Three Scopes.**
1. **Semantic Scope** — `v = w_target`; loss is the logit of a chosen target vocabulary entry.
2. **Fisher Scope** — `v = u₁`, the principal eigenvector of the Fisher information matrix `F = W⊺ diag(p) − pp⊺ W`. By eigen-decomposition `F = UΛU⊺`, perturbations along `u₁` maximize KL-divergence change in `p`; the framework views this as a rank-1 approximation to total mutual information between `p` and `x_t`.
3. **Temperature Scope** — write `y = ‖y‖₂ · ŷ`, so `z = ‖y‖₂ · Wŷ = β_eff · ẑ`. With `β_eff := ‖y‖₂` and `ẑ` normalized logits, the predictive distribution is `p(v) ∝ exp(β_eff · ẑ(v))`. Under the assumption that `ẑ` is quadratic near its peak, **Proposition 1** shows `β_eff ∝ σ⁻²`; hence attributing `β_eff` is equivalent to attributing predictive variance. The custom loss is `L = β_eff`, giving `Influence_t^Temp = ‖ŷ⊺ J_t‖₂`.

**Computational cost.** A single backward pass per attribution; on RTX A4000, the backward pass for an example prompt takes 0.027 s vs. 0.069 s forward. Parameter gradients are disabled; only input-embedding gradients are computed.

**Evaluation.** AOPC (Area Over the Perturbation Curve): ablate top-`k`% tokens (replace with zero vectors), measure drop in ground-truth next-token log-probability; aggregate across `k ∈ {5%, 10%, 20%}` via the trapezoidal rule, averaged over 1000 test passages. Lower (more negative) AOPC ⇒ higher attribution faithfulness.

## Results & Highlights

**AOPC, LAMBADA + IWSLT2017, 6 models (Table 1).** Selected headline numbers (more negative = better):

| Setting | Random | IG | Input×Grad | Sem. Scope | Temp. Scope | Fisher Scope |
|---|---|---|---|---|---|---|
| LLaMA-3.2 1B LAMBADA | −0.26 | −1.10 | −1.28 | −1.30 | **−1.32** | −1.32 |
| Qwen2.5-1.5B LAMBADA | −0.35 | −1.71 | −1.72 | −1.68 | **−1.86** | −1.77 |
| Gemma-3 1B LAMBADA | −0.42 | −0.64 | −1.56 | −1.64 | −1.66 | **−1.67** |
| LLaMA-3.2 3B LAMBADA | −0.23 | −0.67 | −1.12 | −1.16 | −1.17 | **−1.17** |
| Qwen2.5 3B LAMBADA | −0.29 | −1.39 | −1.39 | −1.34 | **−1.55** | −1.41 |
| Gemma-3 4B LAMBADA | −0.29 | −1.43 | −1.70 | −1.78 | −1.78 | **−1.81** |

- Fisher Scope wins 5/6 LAMBADA setups and 4/6 IWSLT2017 setups; Temperature Scope wins 4/6 LAMBADA and 3/6 IWSLT2017 setups. The single tie: Gemma-3 4B IWSLT2017, where Integrated Gradients equals Fisher Scope (−1.20).
- All gradient-based methods beat random ablation by a wide margin (e.g., 4–5× on Qwen/Gemma LAMBADA).

**Case studies.**
- *Political bias (LLaMA-3.2, Semantic Scope):* for the prompt fragment about a person from a region, "liberal" prediction is attributed to "Columbia"; "conservative" prediction to "the South".
- *Translation of Dante's Inferno (LLaMA-3.2 3B, Fisher Scope):* "mondo" → "world" (single-token attribution); the phrase "priegoti ch'a la mente altrui mi rechi" shows a multimodal predictive distribution with each top token hinting at a different phrasing strategy, and Fisher Scope attributes the predictive *spread* to multiple source tokens — something Semantic Scope cannot do because it picks only one target logit.
- *ICL time-series (Temperature Scope):* on a Lorenz-style chaotic series, attribution concentrates on history segments that match the local pattern near the cutoff — direct support for the "LLMs perform ICL via nearest-neighbor search in delayed-embedding space" conjecture. On Brownian motion, influence migrates to later context, explaining why ICL loss plateaus early for unbounded stochastic systems.

**Runtime.** Backward pass ≈ 0.027 s for a representative prompt on RTX A4000 vs. 0.069 s forward — practical for interactive use.

## Critical Assessment

- **Novelty.** The unifying `v⊺ J_t` framing is a genuine conceptual simplification: any scalar-output gradient method is just a particular choice of `v`. The genuinely novel pieces are (i) the Fisher-direction projection for distributional attribution, and (ii) the `β_eff ∝ σ⁻²` theorem that legitimizes Temperature Scope for Gaussian-like outputs. These are real advances, not just rebranding of gradient×input.
- **Rigor.** AOPC is the right kind of metric — interventional, not just gradient-local. The comparison baselines (random, Input×Grad, Integrated Gradients) are appropriate; SHAP and ALTI are not included, which is a gap. Error bars (±0.01–0.02) are tiny because they average over 1000 passages, so statistical claims are well-supported but individual-passage faithfulness is not characterized.
- **Reproducibility.** Open code + HuggingFace demo is excellent; the appendices report experimental setup, and the AOPC protocol is clearly defined. Independent replication should be straightforward on the released models.
- **Significance.** Useful contribution to the interpretability toolbox; the target-free insight ("tokens driving the distribution also drive any prediction in it") is a clean empirical principle worth taking seriously.
- **Red flags / honest concerns.**
  - *Ablation by zero-embedding* is known to be aggressive (it can induce out-of-distribution hidden states). AOPC magnitudes therefore likely overstate how much any single method matters, but since all baselines share this protocol, *relative* comparisons remain valid.
  - *Case studies are illustrative, not exhaustive.* Three anecdotes on bias / translation / time-series — bias in particular is hard to claim without controlled counterfactual prompt pairs and demographic-cue ablations.
  - *Architecture-blindness is acknowledged but glossed over.* The Gemma-3 4B IWSLT2017 tie is hand-waved to "5:1 interleaved local/global attention"; Temperature Scope's elevated early-token influence is attributed to the *attention sink* phenomenon. Both deserve dedicated study.
  - *No human evaluation* for the bias case; "implicit political bias" is a strong claim resting on a heatmap.
  - *Proposition 1 (β_eff ∝ σ⁻²)* is derived under a quadratic-logit assumption that the authors assert holds empirically for the numerical-token regime — fine for ICL time-series but not for open-vocabulary generation, which limits Temperature Scope to numerical/structured prediction contexts.

## Strengths

- Single, principled unification of gradient-based attribution as vector-Jacobian projection.
- Three Scopes cover the full output spectrum (logit → distribution → confidence) within one computation pipeline.
- Computationally negligible (single backward pass per attribution).
- Quantitative evidence on a real interventional metric across six models and two tasks — most attribution papers stop at illustrative heatmaps.
- Target-free design is genuinely useful when the correct next token is ambiguous or unknown (translation, open-ended generation).
- Open code + HuggingFace demo is a strong reproducibility signal.

## Limitations

- **First-order / local.** Linearized in input-embedding space; cannot capture nonlinear causal pathways the way activation patching or circuit tracing can.
- **Architecture-blind.** Does not localize computations to specific layers / heads / circuits; needs to be combined with mechanistic methods for full picture.
- **Attention-sink artifacts.** Early-token influence inflation (Fig. 4b) noted but not fully corrected for.
- **Temperature Scope relies on near-Gaussian assumption** for predictive distribution — only valid for numerical-token ICL tasks; not generally applicable to vocabulary distributions.
- **No SHAP / ALTI / causal-tracing baselines.** Comparison set is limited to gradient-family methods; relative ordering vs. non-gradient methods is unknown.
- **Zero-vector ablation** is a strong intervention; results are protocol-dependent.
- **Case studies lack controlled counterfactuals**, especially for the bias claim.
- **No qualitative human study** validating interpretability of the heatmaps.

## Relevance

匹配用户的「LLM Hidden States & Interpretability」方向：Jacobian Scopes 直接在输出层隐藏状态 `y` 上做信息几何–敏感的 token 级归因，与 mechanistic interpretability / representation geometry 高度相关；同时其通过 AOPC 的定量评估范式也契合用户对「推理评估 / 输出一致性」的关注。

## Questions for Further Exploration

- Can Jacobian Scopes be **lifted from input embeddings to the residual stream** so that attribution localizes to specific layers / attention heads, closing the gap with activation patching?
- Does the target-free principle generalize to **chain-of-thought / multi-token generation** — i.e., attributing an entire reasoning chain rather than a single next-token prediction?
- How sensitive are attribution maps to **embedding-layer choice** (input embedding vs. post-embedding residual stream) and to **prompt paraphrases / random seeds**?
- For the bias case, what is the result under **counterfactual prompt pairs** that differ only in demographic cues — does the method produce a calibrated, falsifiable bias score?
- Can Fisher Scope's principal-eigenvector idea be extended to **multi-modal / sequence-level outputs** (e.g., attribution of an entire generated sentence)?
- Does the **target-free ⇒ more faithful** phenomenon hold for very large models (>= 70B), where integrated gradients may have its own advantages?
- How does Jacobian Scope interact with **SAE features** — can it identify which input tokens activate which SAE latent?
- Can the `v⊺ J_t` framework be specialized for **calibration, hallucination detection, or factual recall** as the authors suggest, and do those specialized scopes also beat existing methods on appropriate AOPC-style metrics?
