> **一句话总结：** 本文提出FaReG框架，通过变分编码器-解码器学习潜在表征空间中遭受算法偏见的子群，在保持预测集紧凑的同时实现对非线性组合特征定义的不公平群体的自适应等化覆盖保证，发表于ICLR 2026。

> **TL;DR:** FaReG is a fair conformal prediction method that uses a variational encoder-decoder to identify unfairly treated subgroups in a learned latent representation space, achieving adaptive equalized coverage for groups defined by nonlinear feature combinations — something prior methods like AFCP cannot capture — while keeping prediction sets compact and efficient.

## 核心贡献

- 提出FaReG框架，通过变分自编码器将原始特征映射到潜在表征空间，在该空间中自适应识别遭受算法偏见的子群，突破了现有方法（如AFCP）只能处理线性/单特征定义群体的局限性。
- 引入深度变分信息瓶颈（Deep VIB）优化目标，联合最大化编码Z与不公平指标S、原始特征X的互信息，同时压缩个体身份信息，确保模型既能发现不公平群体又具有可解释性。
- 提出WSC+非线性条件覆盖度量指标，用二次函数替代原WSC中的线性映射，能更准确地发现由非线性特征关系定义的低覆盖群体，比WSC提升最高达9.89%。
- 提供了自适应等化覆盖的理论保证（Theorem 1），证明通过联合伯努利采样构造的预测集并集满足对所选群体的条件覆盖要求。
- 在合成数据和真实数据（Nursery）上，FaReG是唯一始终达到有效条件覆盖（≥0.9）的方法，同时预测集大小显著小于AFCP2。

## Key Contributions

- Proposes FaReG, a representation-learning approach to fair conformal prediction that encodes raw features into a latent space via a variational encoder-decoder, enabling identification of unfairly treated subgroups defined by nonlinear feature combinations (e.g., XOR-type interactions) that prior work (AFCP) cannot detect.
- Introduces a Deep Variational Information Bottleneck (VIB) objective that jointly maximizes mutual information between the encoding Z and the unfairness indicator S as well as the input X, while compressing individual identity information — balancing group discovery with interpretability through input reconstruction.
- Proposes WSC+, a nonlinear extension of the Worst-Slab Coverage (WSC) metric, replacing linear projections with quadratic functions to more accurately uncover groups with poor conditional coverage — achieving up to 9.89% improvement in detecting the worst-coverage subgroups.
- Proves that FaReG satisfies adaptive equalized coverage (Theorem 1): the union of prediction sets constructed from T sampled groups guarantees conditional coverage on the identified subgroups, even when groups are defined by complex nonlinear feature combinations.
- Demonstrates on both synthetic and real-world (Nursery) datasets that FaReG is the only method consistently achieving valid conditional coverage (≥0.9) while maintaining prediction set sizes significantly smaller than AFCP2.

## Methodology

FaReG operates within the split conformal prediction framework. Given a calibration set D = {(Xi, Yi)} and a pre-trained classifier, the method proceeds in three main stages:

**1. Learning Representation-Based Groups.** The core innovation is encoding raw features X into a latent representation Z = f(X) via a stochastic encoder (Gaussian parameterized by two MLPs for mean and variance). The optimization objective is derived from the Deep Variational Information Bottleneck:

$$L = L_{CC} + L_{MSE} - \beta L_{KL}$$

- **L_CC (Conditional Coverage Loss):** Minimizes the expected empirical conditional coverage of the selected group Ĝ, where group membership is determined by a logistic regression decoder operating on Z, with a minimum group size constraint (≥ δ proportion). Solved via Projected Gradient Descent to enforce the size constraint.
- **L_MSE (Reconstruction Loss):** Standard mean squared error for reconstructing X from Z, ensuring interpretability — users can trace back which input features contribute to group membership via perturbation analysis (Beta-VAE style).
- **L_KL (KL Divergence):** Regularizes the encoder toward a standard normal prior, compressing irrelevant individual-level information.

**2. Constructing Adaptive Prediction Sets.** After training, T group vectors are sampled from the learned joint Bernoulli distribution. For each sampled group, a conformal prediction set is constructed using only the calibration points in that group. The final prediction set is the union of all these group-specific sets plus the standard marginal conformal set:

$$C(X_{N+1}) = C_m(X_{N+1}, D) \cup \bigcup_{t=1}^{T} C_m(X_{N+1}, \hat{G}_{s_t})$$

**3. WSC+ Metric.** To evaluate conditional coverage more rigorously, WSC+ replaces the linear projection v^T X in traditional WSC with a quadratic function π(x) = x^T W x + v^T x, enabling detection of poorly-covered groups defined by nonlinear feature interactions.

The overall time complexity is O(EN(|θ|+|ϕ|+|φ|) + T(N+M)), which is effectively O(N+M) — linear in both calibration and test set sizes.

## Results & Highlights

- **Synthetic data (XOR-type bias):** FaReG is the only method achieving valid conditional coverage (≥ 0.9) across all sample sizes (500–4000). AFCP1 fails because it cannot detect XOR-type groups. AFCP2 achieves coverage at large sample sizes but produces prediction sets 2–3x larger than FaReG.
- **Nursery dataset:** FaReG consistently outperforms all baselines on both Group Coverage and WSC+ metrics. It maintains marginal coverage close to 0.9 while achieving superior conditional coverage compared to Marginal, Partial, AFCP1, and AFCP2.
- **Efficiency:** FaReG produces significantly smaller prediction sets than AFCP2 (which also tries to capture multi-feature groups), making the uncertainty information more actionable.
- **Running time:** FaReG is substantially faster than AFCP methods. Most time is spent on encoder-decoder training (e.g., 161.3s training vs 0.8s inference for n=2000), and training time scales linearly with sample size.
- **Feature selection:** Perturbation analysis shows FaReG correctly identifies target sensitive features (X[0] and X[1] in the synthetic experiment) with increasing accuracy as sample size grows, demonstrating that the reconstruction pathway provides meaningful interpretability.
- **WSC+ metric:** On synthetic data, WSC+ discovers groups with 5–10% lower conditional coverage than the standard WSC metric, confirming that nonlinear feature combinations harbor hidden algorithmic biases.

## Critical Assessment

**Novelty: High.** The key insight — using representation learning to discover unfair groups in latent space rather than enumerating over raw features — is genuinely novel and addresses a real limitation of prior work. The WSC+ metric is a meaningful contribution. The theoretical analysis (Proposition 1, Theorem 1) is rigorous.

**Rigor: Strong.** The paper provides formal guarantees (Theorem 1 for adaptive equalized coverage, Proposition 1 for coverage estimation bounds, Proposition 2 for WSC+ lower bound). The experimental design includes synthetic data with controlled bias mechanisms and real-world benchmarks. The VIB objective derivation is sound.

**Reproducibility: Good.** Code is public. Hyperparameters and training details are provided in the appendix. All experiments run on a single RTX 3090.

**Significance: High.** Fairness in conformal prediction is an important and growing area. The practical limitation of prior methods (inability to capture nonlinear group definitions) is well-motivated by the XOR example (Fig. 1). FaReG offers a principled solution with theoretical backing.

**Red flags:**
- The interpretability claim relies on perturbation-based feature attribution (Beta-VAE style), which is known to be approximate and sometimes unreliable for complex encoder architectures. The paper acknowledges this as a limitation but could be more forthright about failure modes.
- The minimum group size constraint δ is a critical hyperparameter. Setting it too small risks selecting tiny, noisy groups; setting it too large may miss genuinely disadvantaged minorities. The sensitivity analysis in Appendix B.2 helps but the interplay between δ and fairness guarantees deserves more discussion.
- Evaluation is limited to two datasets (one synthetic, one relatively small real-world dataset). Testing on larger-scale, real-world high-stakes applications (e.g., medical diagnosis, credit scoring) would strengthen the claims.
- The reconstruction-based interpretability adds a form of regularization that may not always align with the group discovery objective — there could be tension between faithful reconstruction and compact group identification.

## Strengths

- Addresses a fundamental limitation of prior fair CP methods by enabling nonlinear group discovery.
- Principled theoretical framework with formal coverage guarantees.
- Strong experimental results: only method achieving valid conditional coverage across all settings.
- Significantly more efficient than AFCP, with linear time complexity.
- WSC+ metric is a useful standalone contribution for the community.
- Code available, supporting reproducibility.

## Limitations

- Interpretability of discovered groups is indirect (via perturbation analysis) and may be unreliable for high-dimensional or complex feature spaces.
- Only evaluated on one synthetic and one small real-world dataset; generalization to large-scale, truly high-stakes domains is unverified.
- The minimum group size constraint δ requires careful tuning and interacts with the fairness-efficiency trade-off.
- The encoder-decoder architecture introduces additional hyperparameters (β, architecture choices, learning rate schedule) compared to simpler baselines.
- The method assumes access to sensitive features as input; in practice, identifying which features are sensitive may itself be challenging.

## Questions for Further Exploration

1. How does FaReG scale to datasets with hundreds or thousands of features (e.g., genomics, large language model outputs)?
2. Can the encoder-decoder framework be extended to regression tasks with continuous outcomes and equalized coverage requirements?
3. What happens when unfair groups are defined by highly complex, high-order feature interactions beyond quadratic (XOR)?
4. How sensitive is FaReG to the choice of the pre-trained classifier — could a biased classifier mislead the group discovery process?
5. Is there a principled way to automatically determine the minimum group size δ based on calibration set statistics rather than treating it as a hyperparameter?
