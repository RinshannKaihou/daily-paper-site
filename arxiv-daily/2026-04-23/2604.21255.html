When Agents Look the Same: Quantifying Distillation-Induced Similarity in Tool-Use Behaviors

Chenghao Yang, Yuning Zhang, Zhoufutu Wen, Tao Gong, Jiaheng Liu, Qi Chu, Nenghai Yu

arXiv: 2604.21255v1

Date: 2026-04-23

https://arxiv.org/abs/2604.21255

一句话总结： 本文提出RPS（响应模式相似度）和AGS（动作图相似度）两个指标，量化LLM智能体工具调用行为因蒸馏产生的同质化现象，发现Kimi-K2的行为模式与Claude Sonnet 4.5高度相似（AGS=82.7%），并通过受控实验证实蒸馏是导致这一现象的因果机制。
TL;DR: RPS and AGS are two complementary metrics for measuring behavioral similarity between LLM agents in tool-use tasks; applied to 18 models across 8 providers, the metrics reveal that Kimi-K2 (thinking) exhibits the strongest Claude-like tool-use behavior (AGS=82.7%, RPS=3.65), that within-family models are 5.9pp more similar than cross-family models, and that a controlled LoRA fine-tuning experiment on 200 Claude trajectories causally confirms distillation as the similarity mechanism.

核心贡献

  • 提出RPS（Response Pattern Similarity）：通过5阶段对话标注（认证/引导/执行/验证/通知）和LLM评判打分（风格/结构/对齐，1-5分），量化智能体响应模式的语义相似度
  • 提出AGS（Action Graph Similarity）：构建动作流图G=(V, E_s, E_d)，通过节点一致性S_node（可选工具调用匹配率）、序列习惯S_seq和依赖模式S_dep三个维度度量工具调用策略的结构相似度
  • 以Claude Sonnet 4.5（thinking模式）为参考，对8家供应商的18个模型进行系统评测，发现Kimi-K2（thinking）AGS=82.7%为非Anthropic模型中最高，揭示商业模型蒸馏的普遍性
  • 通过受控蒸馏实验（200条Claude轨迹LoRA微调Qwen2.5-14B-Instruct）提供因果证据：微调组AGS向教师模型提升+0.13，对照组AGS下降-0.05

Key Contributions

  • RPS (Response Pattern Similarity): A metric that evaluates how an agent communicates across a tool-use interaction. Five conversation stages are annotated (Authentication, Elicitation, Execution, Verification, Notification), and an LLM judge scores three dimensions per stage: Style (lexical/tonal), Structure (format/length), and Alignment (information completeness). Final RPS is the mean score across stages × dimensions (range 1-5). Inter-rater Cohen's κ = 0.70 confirms reliable LLM judging.
  • AGS (Action Graph Similarity): A metric that evaluates what an agent does structurally. An Action Flow Graph G = (V, E_s, E_d) is built from the tool call sequence, where V = tool invocations, E_s = sequential edges (A called before B), E_d = dependency edges (A's output required by B). AGS decomposes into three sub-scores: S_node (fraction of optional tool calls that agree with reference), S_seq (sequential ordering similarity), S_dep (dependency pattern similarity).
  • Large-scale behavioral benchmark: 18 LLMs from 8 providers evaluated on τ-Bench and τ²-Bench (realistic multi-turn tool-use scenarios). Claude Sonnet 4.5 (thinking) selected as reference based on its state-of-the-art tool-use performance.
  • Distillation causal experiment: Qwen2.5-14B-Instruct fine-tuned with LoRA on 200 Claude Sonnet 4.5 tool-use trajectories. AGS vs. teacher increases from baseline to +0.13 relative to control group's -0.05 — a 0.18 AGS point separation attributable purely to trajectory distillation.
  • Within-family homogenization finding: Models within the same provider family (e.g., Qwen-7B and Qwen-72B) show 5.9 percentage points higher AGS than cross-family pairs on average, quantifying the provider-specific behavioral fingerprint.

Methodology

τ-Bench and τ²-Bench: Realistic agentic benchmarks requiring multi-step tool use to complete service-like tasks (airline booking, retail, etc.). τ²-Bench is an extended version with harder multi-agent coordination scenarios. The paper uses these as the evaluation environment rather than designing new tasks.

Reference model selection: Claude Sonnet 4.5 (thinking) is chosen as the reference because it achieves highest overall task completion. Similarity is therefore measured as "similarity to the best known agent" — this is a design choice that makes the metrics interpretable but also anchors the framing around Claude specifically.

RPS computation:

  1. Stage segmentation: Conversation turns are labeled into 5 stages using an LLM annotator (given the task context and turn content)
  2. LLM judging: For each stage, GPT-4 or equivalent judge compares the target agent's response to the reference response, scoring Style, Structure, and Alignment on a 1-5 Likert scale
  3. Aggregation: Mean across stages and dimensions
  4. Validation: Human annotators re-score a subset; Cohen's κ = 0.70 confirms reliability

AGS computation:

  1. Action graph extraction: Parse tool call sequences into G = (V, E_s, E_d)
  2. S_node: For optional (non-mandatory) tool calls, compute Jaccard similarity of the called tool sets vs. reference
  3. S_seq: Compare sequential orderings using a normalized sequence distance (edit distance variant)
  4. S_dep: Compare dependency edge structures using graph kernel or Jaccard over edge sets
  5. AGS = weighted combination of S_node, S_seq, S_dep

Controlled distillation experiment:

  • Base model: Qwen2.5-14B-Instruct (selected for being a widely-used open-source model)
  • Teacher trajectories: 200 complete tool-use interaction traces from Claude Sonnet 4.5 on τ-Bench
  • Training: LoRA fine-tuning (r=16, α=32), 3 epochs, standard cross-entropy on teacher responses
  • Control: Same Qwen2.5-14B-Instruct fine-tuned on 200 random (non-Claude) trajectories of equal length
  • Measurement: AGS of both fine-tuned models vs. Claude Sonnet 4.5 reference

Results & Highlights

  • Kimi-K2 (thinking): AGS = 82.7%, RPS = 3.65 — highest among non-Anthropic models, substantially above second place
  • Within-family AGS advantage: 5.9pp higher within-provider pairs vs. cross-provider pairs (Table 2), demonstrating that training heritage leaves measurable behavioral fingerprints
  • Distillation causal evidence: Fine-tuned model AGS → teacher: +0.13; control model AGS → teacher: -0.05; delta = 0.18 AGS points, p < 0.05
  • Mandatory vs. optional tool calls: S_node focuses on optional tools (where the agent has discretion). Mandatory tools show near-universal agreement (>95%) and are thus uninformative for similarity analysis
  • RPS inter-rater reliability: Cohen's κ = 0.70 (substantial agreement) for the LLM judge vs. human evaluators, validating the automated scoring pipeline
  • Model family clustering: Qwen family models cluster together, Anthropic models cluster together; the AGS-based clustering visually separates families in a 2D projection, suggesting AGS captures genuine distributional fingerprints

Critical Assessment

Novelty: High. While behavioral similarity between LLMs has been discussed qualitatively, this is among the first papers to formalize it with concrete, reproducible metrics (RPS + AGS) and provide causal distillation evidence. The action graph formalism is particularly novel.

Rigor: Moderate. The inter-rater agreement for RPS (κ=0.70) is good. However, the AGS sub-component weights are not well-justified — S_node, S_seq, S_dep are combined but their relative importance to the overall similarity perception is not ablated. The distillation experiment uses only 200 trajectories and a single base model, limiting generalization.

Reproducibility: Moderate. RPS requires an LLM judge (GPT-4 or equivalent) which introduces cost and API dependency. AGS requires tool call parsing from agent transcripts. The paper does not release code or the 200 Claude trajectories used in the distillation experiment (likely cannot, due to Claude's terms of service).

Significance: High for AI safety and governance. If commercial models can be behaviorally fingerprinted as distilled from specific teachers, this has implications for attribution, copyright, and ecosystem diversity. The finding that Kimi-K2 closely resembles Claude in tool-use behavior is commercially sensitive.

Red flags:

  • Using Claude Sonnet 4.5 as the sole reference conflates "similar to Claude" with "high quality" — these are not the same thing. Models could converge to Claude-like behavior for reasons unrelated to distillation (e.g., both trained on similar data)
  • The distillation experiment used 200 trajectories — this is small enough that the AGS difference might reflect fine-tuning overfitting to specific task patterns rather than genuine behavioral transfer
  • The paper cannot definitively prove Kimi-K2 was distilled from Claude — high AGS is consistent with but does not prove distillation. The authors acknowledge this but the framing is suggestive
  • S_node focuses on optional tools, but the definition of "optional" vs. "mandatory" may be task-specific and is not formalized rigorously

Strengths

  • First formal metrics (RPS + AGS) for quantifying tool-use behavioral similarity
  • Real causal evidence from controlled distillation experiment
  • Large-scale evaluation (18 models, 8 providers)
  • Within-family homogenization finding is empirically clean and replicable
  • Action Flow Graph formalism is extensible to other agentic tasks beyond tool use

Limitations

  • Reference model bias: all similarity measured relative to a single commercial model (Claude Sonnet 4.5)
  • Cannot prove causation for commercial models (Kimi-K2 vs. Claude) — only correlation
  • RPS requires expensive LLM judging; not easily scalable to continuous model monitoring
  • Limited to tool-use tasks; generalization to other agent behaviors (planning, reasoning without tools) not addressed
  • Distillation experiment uses LoRA on 200 examples — may not represent the scale of real commercial distillation

相关性

该工作与LLM可解释性和推理评估高度相关：RPS/AGS框架可扩展为AI系统卡（system cards）的行为指纹组件，帮助追踪模型训练来源和行为同质化风险；对于关注ML系统可靠性的读者，模型行为同质化可能带来单点失效的系统性风险，值得警惕。

Questions for Further Exploration

  1. Can RPS and AGS detect behavioral similarity when the teacher model is not a frontier model — e.g., if an open-source model is distilled into another open-source model?
  2. How stable are AGS scores across different task domains? If a model has high AGS on τ-Bench, does it also have high AGS on coding agent benchmarks?
  3. Could AGS be used as a training signal to deliberately increase or decrease behavioral diversity among model families in an ecosystem?
  4. What is the relationship between AGS/RPS similarity and downstream task performance? Are models that are more similar to Claude also better at tool-use tasks, or is similarity orthogonal to performance?
  5. How would the metrics behave on adversarially designed agents that mimic surface-level Claude patterns without genuine capability distillation?
