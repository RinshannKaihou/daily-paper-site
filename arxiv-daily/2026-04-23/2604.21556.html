Probabilistic Verification of Neural Networks via Efficient Probabilistic Hull Generation

["Jingyang Li", "Xin Chen", "Hongfei Fu", "Guoqiang Li"]

arXiv: 2604.21556v1

Date: 2026-04-23

https://arxiv.org/abs/2604.21556

一句话总结： 本文提出基于回归树引导的概率性凸包生成框架，通过边界感知采样和迭代细化高效构建安全/不安全概率包，在 ACAS Xu 和火箭着陆器基准上比枝界法实现高达 10× 加速，同时获得有保证的安全概率区间。
TL;DR: This paper proposes a regression tree-guided probabilistic verification framework for feedforward DNNs with Gaussian-distributed inputs. By generating safe and unsafe probabilistic hulls via boundary-aware subdivision rather than uniform state-space partitioning, the method avoids wasted subdivisions in purely safe/unsafe regions, achieves up to 10x speedup over branch-and-bound baselines, and computes a guaranteed [Lₛ, Uₛ] interval for the true safe probability—applicable to any DNN architecture (not limited to ReLU).

核心贡献

  • 提出概率包（Probabilistic Hull）框架：通过构造有限个轴对齐的安全/不安全凸包集合，给出真实安全概率的保证上界和下界，将不可计算的积分问题转化为可处理的有界优化。
  • 提出边界感知采样策略：通过随机采样识别输入空间中的安全边界，再以回归树自适应细分输入空间，将分割集中在边界附近而非均匀覆盖整个空间，避免均匀划分在纯安全/纯不安全区域的无效计算。
  • 引入基于概率优先级的迭代细化机制：始终优先处理 R_unknown 中概率最大的凸包，避免密集采样全空间的禁止代价。
  • 框架通用性：仅将 DNN 作为黑箱使用（用 CROWN 验证安全性），不依赖 ReLU 激活函数，可适用于任意前馈 DNN。

Key Contributions

  • Introduces a probabilistic hull representation: a closed, bounded box-shaped set in input space whose probability under Gaussian input can be efficiently computed via the error function (erf). Safe hulls (all outputs satisfy Γ) give a lower bound Lₛ; unsafe hulls give an upper bound Uₛ = 1 − Σ P(Hᵢᵘ).
  • Proposes boundary-aware subdivision via regression trees: random samples are drawn and labeled safe/unsafe via CROWN; a regression tree is built on these samples and its leaf nodes define the subdivision boxes. This concentrates partitions near the decision boundary, avoiding exponential uniform subdivision.
  • Develops probabilistic prioritization: the main loop always selects Rₙₑₓₜ = argmax P(R) from R_unknown, refining the highest-probability undecided regions first and terminating when Σ_{R∈R_unknown} P(R) < ε.
  • Architecture-agnostic: uses DNN as a black box with CROWN as the safety verifier; applicable beyond ReLU to networks with any activation function.
  • Provides Theorem 1: formal soundness guarantee that if H₁ˢ,...,HₙNˢ are safe and H₁ᵘ,...,HₘMᵘ are unsafe, then Uₛ ≥ Lₛ and the actual safe probability lies in [Lₛ, Uₛ].

Methodology

Problem formulation: Given feedforward DNN f: ℝᵈ → ℝᵐ, safety property Γ(y): Cy ≥ a (affine output constraints), and Gaussian input x ~ N(μ, Σ), estimate P(Γ(y) | x ~ N(μ, Σ)).

Probabilistic Hull: A hull H = [a₁,b₁] × ... × [aᵈ,bᵈ] is a box in input space. For diagonal Σ = diag(σ₁²,...,σᵈ²), P(x ∈ H) = Π_{1≤i≤d} ½(erf((bᵢ−μᵢ)/(√2σᵢ)) − erf((aᵢ−μᵢ)/(√2σᵢ))). For full-rank Σ, transform to diagonal. H is safe if all f(x) for x ∈ H satisfy Γ; unsafe if all violate Γ.

Main algorithm loop:

  1. Initialize: R_unknown = {[Space]}, R_safe = R_unsafe = ∅. The whole state space is bounded to a large box with probability ≥ 99.999%.
  2. Select Rₙₑₓₜ with highest P(R) from R_unknown; remove from R_unknown.
  3. Apply boundary-aware subdivision to Rₙₑₓₜ: (a) draw random samples, label safe/unsafe via CROWN; (b) build regression tree on labeled samples; (c) new probabilistic hulls = tree leaves.
  4. Verify each new hull: add safe ones to R_safe, unsafe ones to R_unsafe, mixed to R_unknown.
  5. Terminate when Σ_{R∈R_unknown} P(R) < ε.
  6. Output: [Lₛ, Uₛ] where Lₛ = Σ P(Hᵢˢ), Uₛ = 1 − Σ P(Hᵢᵘ).

Safety verification oracle: CROWN (linear bound propagation) verifies whether a given box hull is safe or unsafe; if verification is inconclusive, hull remains in R_unknown.

Key insight vs. uniform subdivision: Uniform subdivision splits regions regardless of the decision boundary, wasting computation on purely safe/unsafe areas. Boundary-aware subdivision focuses splits precisely where the decision boundary lies (Figure 2 in paper), achieving the same coverage with fewer subdivisions.

Results & Highlights

  • Evaluated on ACAS Xu (airborne collision avoidance neural network, multiple safety properties) and a rocket lander controller.
  • Shows "an obvious advantage over the state of the art" including: up to 10× speedup compared to basic branch-and-bound approaches; superior accuracy (tighter [Lₛ, Uₛ] intervals) vs. set-based techniques (β-CROWN family) and ProbStar.
  • Boundary-aware subdivision significantly outperforms uniform sampling in experiments.
  • The incremental probabilistic prioritization mechanism avoids the curse of dimensionality that plagues existing methods requiring exponential uniform subdivisions.
  • Framework handles non-ReLU networks where ProbStar (relying on linear region enumeration) cannot be applied.

Strengths

  • Formal soundness: Theorem 1 provides a rigorous guarantee on the interval [Lₛ, Uₛ].
  • Architecture-agnostic: works for any feedforward DNN, not limited to ReLU networks like ProbStar.
  • Boundary-aware sampling addresses the curse of dimensionality more directly than uniform subdivision.
  • Probabilistic prioritization ensures computational budget is spent on high-probability regions.
  • Clean integration with CROWN as a black-box verifier.

Limitations

  • Relies on CROWN's ability to verify hull safety; for hulls near the boundary where CROWN is loose, the region remains in R_unknown, potentially not converging to tight bounds within budget.
  • Regression tree quality is sample-dependent; in high-dimensional spaces, boundary identification may require many samples.
  • Gaussian input assumption—real-world noise may be non-Gaussian (heavy-tailed, correlated with state).
  • Only feedforward (not recurrent or convolutional) DNNs are addressed in the formulation.
  • Full experimental section not available in the 5-page excerpt; claims need validation against the complete paper.
