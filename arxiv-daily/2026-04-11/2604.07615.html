ADAG: Automatically Describing Attribution Graphs

一句话总结： 提出ADAG——首个全自动的归因图描述管道，通过引入归因特征谱（attribution profiles）量化特征功能角色，并利用LLM解释器-模拟器生成自然语言描述，成功在Llama 3.1 8B上发现可引导的有害建议越狱电路。
TL;DR: Introduces ADAG, the first end-to-end automated pipeline for describing attribution graphs in LLMs, using attribution profiles and an LLM explainer-simulator to generate interpretable natural-language circuit descriptions, discovering steerable jailbreak clusters in Llama 3.1 8B Instruct.

核心贡献

  • 提出归因特征谱（attribution profiles），通过输入和输出梯度效应量化特征的功能角色，解决了电路追踪中依赖人工解读的问题
  • 设计了全新的聚类算法对特征进行分组，配合LLM解释器-模拟器框架生成并评分自然语言解释
  • 在Llama 3.1 8B Instruct上发现了导致有害建议越狱的可引导特征簇

Key Contributions

  • Introduces attribution profiles that quantify feature functional roles via input/output gradient effects, eliminating manual interpretation in circuit tracing
  • Proposes a novel clustering algorithm and LLM explainer-simulator pipeline that generates and scores natural-language explanations of feature groups
  • Demonstrates ADAG on human-analyzed circuit tasks, recovering known circuits, and discovers steerable jailbreak clusters in Llama 3.1 8B Instruct

Methodology

ADAG works in three stages: (1) attribution profiles compute the functional role of each feature via its gradient effects on inputs and outputs; (2) a novel clustering algorithm groups features with similar profiles; (3) an LLM explainer generates natural-language descriptions of each cluster, scored by an LLM simulator that predicts activations given the description. The pipeline is fully automated, replacing ad-hoc human interpretation with quantitative profiling.

Results & Highlights

  • Successfully recovers interpretable circuits on previously human-analyzed circuit-tracing tasks
  • Discovers steerable clusters responsible for a harmful advice jailbreak in Llama 3.1 8B Instruct
  • First fully automated pipeline for describing attribution graphs without human intervention

Authors & Credibility

Steinhardt and Schwettmann are established researchers at UC Berkeley / MIT with prior work on interpretability and representation engineering. The team has credible track records in mechanistic interpretability.

Critical Assessment

Novelty: High — prior circuit tracing relied entirely on human interpretation; ADAG is the first to automate this. The attribution profiles concept is genuinely new. Rigor: Validated on known human-analyzed circuits, but evaluation is somewhat qualitative — would benefit from larger-scale quantitative benchmarks. Significance: Important step toward scalable mechanistic interpretability. Red flags: The jailbreak application is interesting but preliminary — unclear how generalizable the steerable clusters are.

Strengths

  • First fully automated pipeline for attribution graph description
  • Attribution profiles provide a principled, quantitative alternative to manual interpretation
  • Practical application to safety-relevant jailbreak discovery

Limitations

  • Evaluation primarily qualitative; needs larger-scale quantitative validation
  • Unclear how well ADAG scales to very large circuits
  • LLM-as-explainer component introduces its own potential biases

Relevance

直接解决机械可解释性（mechanistic interpretability）的核心问题——自动化电路追踪和描述，与LLM隐藏状态和可解释性研究方向高度相关，为大规模电路分析提供了可扩展的方法。

Questions for Further Exploration

  • Can ADAG be combined with SAEs to analyze circuits in sparse feature spaces?
  • How does automated description quality compare to expert human analysis at scale?
  • Could attribution profiles be used for real-time monitoring during deployment?
