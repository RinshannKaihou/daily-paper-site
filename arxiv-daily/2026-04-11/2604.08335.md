# Dead Weights, Live Signals: Feedforward Graphs of Frozen Language Models

> **一句话总结：** 本文将多个异构冻结 LLM（Llama-3.2-1B、Qwen2.5-1.5B、Gemma-2-2B 编码 + Phi-3-mini、Mistral-7B 精炼）组织成前馈图，仅训练 17.6M 投影参数（12B 冻结）通过共享潜在空间通信，在 ARC-Challenge 上达 87.3%（比最佳单模型 +11.4pp）、OpenBookQA 82.8%、MMLU 67.2%。

> **TL;DR:** This paper composes heterogeneous frozen LLMs into a feedforward graph where nodes communicate through a shared continuous latent space via learned linear projections (17.6M trainable params against ~12B frozen). Three small frozen encoders (Llama-3.2-1B, Qwen2.5-1.5B, Gemma-2-2B) project into a shared d=1024 space, whose aggregate signal is injected into two larger frozen refiners (Phi-3-mini, Mistral-7B) via residual-stream hooks, feeding a cross-attention output node. On ARC-Challenge it reaches 87.3% vs the best single model Phi-3-mini's 75.9% (+11.4pp) and a parameter-matched learned head's 78.2% (+9.1pp); on MMLU 67.2% vs Mistral-7B's 66.0% (+1.2pp) and the learned head's 60.5% (+6.7pp). Gradient flow through frozen boundaries is measured at 13% of output-node signal strength.

## 核心贡献

- 提出冻结 LLM 前馈图架构：异构冻结 LLM 作为计算节点，通过学习到的线性投影在共享潜在空间（d_s=1024，刻意中性化以避免恒等映射）中通信，仅训练投影矩阵与轻量跨注意力输出节点（共 17.6M 参数）。
- 验证跨冻结 LLM 边界的梯度流可行：在两节点最小图中，投影矩阵 W 接收约 13% 输出头梯度信号，远高于灾难性坍缩；置换对照 R²=−0.243（vs Ridge R²=0.299）证明对齐是真实语义几何而非分布伪影。
- 在 ARC-Challenge、OpenBookQA、MMLU 三个多选基准上超过最佳单模型 11.4/6.2/1.2pp，并超过参数匹配的冻结单模型学习分类头 9.1/5.2/6.7pp。
- 观察到输出节点自发涌现选择性路由：无任何路由监督下，跨注意力对 Phi-3-mini（W4）的梯度范数比 Mistral-7B（W5）高 5–30×，归因于 Phi-3-mini 合成语料产生的几何规整表示。

## Key Contributions

- A feedforward graph architecture where heterogeneous frozen LLMs serve as computational nodes communicating through a shared continuous latent space (d_s=1024, deliberately neutral to force genuine geometric work) via end-to-end learned linear projections, with only 17.6M trainable parameters against ~12B frozen.
- Empirical validation that gradients flow tractably through frozen LLM boundaries: in a minimal two-node graph the projection matrix W receives ~13% of the output head's gradient signal (norm ratio 0.130), far above catastrophic collapse, and a permutation control (R²=−0.243 vs Ridge R²=0.299) confirms the alignment captures genuine semantic geometry rather than distributional artifacts.
- Strong benchmark performance: 87.3% on ARC-Challenge (+11.4pp over best single model, +9.1pp over a parameter-matched learned head), 82.8% on OpenBookQA (+6.2pp, +5.2pp), and 67.2% on MMLU (+1.2pp, +6.7pp), with the gains attributable to the multi-node communication mechanism rather than the classifier.
- Emergent selective routing: without any routing supervision, the cross-attention output node develops asymmetric attention favoring Phi-3-mini (W4 gradient norms exceed W5 by 5–30×), attributed to the geometric regularity of Phi-3-mini's synthetic-corpus representations.

## Methodology

**Graph definition.** A feedforward graph G=(V,E) where each node v is a frozen pretrained LLM and each directed edge (u,v) carries a learned projection W_uv ∈ R^{d_s×d_u}. Only the projection matrices and a lightweight output node are updated; all LLM weights θ_v stay frozen.

**Layer 1 — diverse encoding.** Three heterogeneous frozen LLMs — Llama-3.2-1B (d=2048), Qwen2.5-1.5B (d=1536), Gemma-2-2B (d=2304) — each receive the input with a distinct analytical framing prefix ("analyze the factual content", "analyze the reasoning structure", "analyze the language and framing"). Hidden states are extracted at relative depth l_T=0.90 (deeper layers yield more resolved semantic representations) at the final token position and L2-normalized: h_i = hidden(x⊕p_i, θ_i, l_T)/‖·‖₂.

**Shared latent space.** Each hidden state is projected and averaged: z₁ = (1/3)Σ W_i h_i, W_i ∈ R^{d_s×d_u}, d_s=1024. d_s=1024 is deliberately neutral (not equal to any constituent model's hidden dimension) so every projection must perform genuine geometric work rather than approximate an identity mapping.

**Layer 2 — injection and refinement.** z₁ is injected into the residual stream of each layer-2 node — Phi-3-mini (d=3072) and Mistral-7B (d=4096) — at relative depth l_S=0.75. Since d_s=1024 ≠ d_j, z₁ is linearly interpolated to the target dimension (differentiably, preserving gradient flow to layer-1 matrices). The blend uses magnitude-rescaling: h′ = (1−α)·h_node + α·(z̃₁·‖h_node‖₂), with α=0.25 (prior work shows reasoning representations become fragile for α≥0.8). Each layer-2 node completes its forward pass; its hidden state is projected: z₂^(j) = W_{j+3}·h′_j.

**Output node.** A cross-attention output node aggregates both layer-2 representations: o = LayerNorm(MHA(q, [z₂^(4), z₂^(5)], [z₂^(4), z₂^(5)])), ŷ = softmax(W_cls·o), where q ∈ R^{d_s} is a learned query, MHA uses 4 heads, and W_cls ∈ R^{4×d_s} maps to the four answer choices. Attention weights implicitly route between the two layer-2 nodes.

**Training.** Cross-entropy loss L = −log ŷ_{c⋆}. Gradients flow backward through the output node, through W4/W5, through the differentiable injection operations, and into W1/W2/W3; frozen LLM weights receive no gradients. Optimizer: AdamW with η_W=10⁻³ for projection matrices and η_out=10⁻⁴ for the output node, weight decay 10⁻⁴, gradient clip 1.0, effective batch size 8, cosine annealing. All models loaded in bfloat16 via HuggingFace; the full graph fits on a single 80GB A100. Injection hooks registered immediately before each forward pass and removed afterward to prevent cross-example contamination. Epochs: 4 (MMLU, ARC); 3 (OBQA).

**Total trainable parameters: 17,580,036** (~0.15% of ~12B frozen): W1 (2,098,176), W2 (1,573,888), W3 (2,360,320), W4 (3,146,752), W5 (4,195,328), output node (4,205,572).

**Baselines.** Single-model greedy decoding (each constituent model individually, zero-shot); parameter-matched learned head (MLP classifier on the frozen hidden states of the strongest single model at budget matched to 17.6M — 16.8M for Mistral-7B on MMLU; 22.8M for Qwen2.5-1.5B on ARC/OBQA, slightly over budget). Hidden states extracted at l_T=0.90 and L2-normalized, identical to the graph's protocol.

## Results & Highlights

- On ARC-Challenge, the frozen LLM graph reaches **87.3%** vs the best single model (Qwen2.5-1.5B-Instruct) at 75.9% (+11.4pp) and the parameter-matched learned head at 78.2% (+9.1pp).
- On OpenBookQA, the graph reaches **82.8%** vs the best single model (Qwen2.5-1.5B-Instruct) at 76.6% (+6.2pp) and the learned head at 77.6% (+5.2pp).
- On MMLU, the graph reaches **67.2%** vs the best single model (Mistral-7B-Instruct-v0.3) at 66.0% (+1.2pp) and the learned head on Mistral-7B at 60.5% (+6.7pp).
- Gradient flow validation (Llama-1B→Qwen-1.5B minimal two-node graph): the projection matrix W receives a gradient norm ratio of **0.130** (~13% of the output head's signal) — attenuated but well above catastrophic collapse; the permutation control R²=−0.243 vs Ridge R²=0.299 confirms genuine semantic alignment; skip connections produce a 1.00× improvement (neutral, hence omitted).
- Training dynamics: ARC-Challenge converges rapidly to **80.7% after just 50 gradient steps**; OpenBookQA crosses the best single-model baseline (76.6%) before the end of epoch 1 and peaks at 85.4% validation mid-epoch 2; MMLU peaks at 67.2% at epoch 2 under cosine annealing.
- Emergent routing: on MMLU, W4 (Phi-3-mini) gradient norms exceed W5 (Mistral-7B) by **10–30× in early training, settling to 5–10× by epoch 4**; the asymmetry is larger on ARC than MMLU. W5 remains active throughout (an order of magnitude above zero).
- All five projection matrices receive nonzero gradients throughout training (no dead nodes); layer-1 matrices W1–W3 maintain nearly identical gradient norms despite distinct framing prefixes (they do not specialize — diversity derives from model heterogeneity).

## Critical Assessment

The paper's central empirical claim — that frozen LLMs can be composed into a differentiable graph with only 17.6M trainable params — is supported by the gradient-flow validation (13% signal retention, permutation control), but several weaknesses undercut the strength of the results. First, the MMLU margin over the best single model is only **+1.2pp** (67.2% vs 66.0%), and the authors themselves flag in Section 6 that "all results use one run per benchmark; the MMLU +1.2pp margin should be interpreted cautiously" — no error bars, no multiple seeds, and cosine-annealing interacts poorly with checkpoint resumption causing oscillating MMLU validation accuracy, so the MMLU win may not be statistically significant. Second, the parameter-matched learned head baseline is oddly constructed: on ARC and OpenBookQA it is given 22.8M parameters (larger than the graph's 17.6M, which the authors call "conservative"), but on MMLU it gets only 16.8M — and critically, the learned head on Mistral-7B underperforms Mistral's own greedy decoding on MMLU (60.5% vs 66.0%), which the authors hypothesize but do not resolve; this suggests the baseline may be unfairly weak rather than the graph unfairly strong. Third, the layer-1 projections W1–W3 "do not specialize" (Section 5.2) — they converge to nearly identical gradient norms despite distinct analytical framing prefixes, meaning the three distinct prefixes provide no learned differentiation; the authors concede "representational diversity derives from the heterogeneity of the frozen models themselves rather than from learned projection specialization," which undermines the premise that the framing prefixes add value. Fourth, the evaluation is restricted to four-choice multiple-choice questions (ARC, OBQA, MMLU) with no test of open-ended generation, longer contexts, or non-English tasks — generalization is untested. Fifth, the offline Ridge R² of 0.299 is markedly lower than the R²≈0.50 reported in the authors' own prior work for similar-scale pairs, which they attribute to smaller validation sets and cross-family pairing; while they argue R² does not correlate with behavioral correction, the weaker alignment raises questions about how robust the linear-projection assumption is across more diverse model families. Finally, the paper is explicitly a "work in progress" preprint with no code release, limiting reproducibility.

## Strengths

- Clean, novel architectural idea: treating frozen LLMs as composable graph nodes communicating through a learned shared latent space, with only 0.15% of parameters trainable.
- The gradient-flow validation experiment (Section 5.1, Appendix B) is rigorous: it measures concrete gradient norms, includes a permutation control that rules out distributional artifacts, and tests the skip-connection hypothesis (finding it neutral at this depth).
- The parameter-matched ablation isolates the communication mechanism from the classifier — the 5.2–9.1pp margins over the learned head confirm the gains are not just from a bigger classifier.
- Honest about the non-specialization finding: rather than hiding that W1–W3 converge to similar norms, the authors analyze it and trace it to the averaging operation.
- The emergent routing observation (Phi-3-mini preferred without supervision) is intriguing and consistent with the geometric-regularity hypothesis from their prior work.
- Practical deployment story: the full graph fits on a single 80GB A100 despite ~12B frozen params, and only 17.6M params need to be trained/stored per task.

## Limitations

- MMLU margin (+1.2pp) is within plausible noise; single run per benchmark, no error bars, and cosine-annealing instability on MMLU validation.
- Layer-1 projections do not specialize — the distinct framing prefixes add no learned differentiation; diversity comes entirely from the frozen models themselves.
- Parameter-matched baseline is inconsistently budgeted (22.8M on ARC/OBQA vs 16.8M on MMLU) and the MMLU learned head underperforms greedy decoding, suggesting a possibly weak baseline.
- Evaluation limited to four-choice multiple-choice (ARC, OBQA, MMLU); no open-ended generation, long-context, or multilingual tests.
- Offline Ridge R² (0.299) is substantially lower than prior work's R²≈0.50 for similar-scale pairs, raising questions about cross-family linear-projection robustness.
- "Preprint. Work in progress." with no code release; reproducibility is limited.
- No comparison against other composition methods (model stitching, MoE routing, ensemble token voting) at matched budgets — only single-model greedy and a learned MLP head.
- The injection depth (l_S=0.75) and blend coefficient (α=0.25) are fixed from prior work without ablation; the sensitivity to these choices is unreported.

## Questions for Further Exploration

- Does the layer-1 non-specialization persist if average pooling is replaced with learned attention-weighted pooling (as the authors propose), or do W1–W3 still converge?
- How does the graph behave with deeper topologies (3+ layers) where gradient attenuation compounds — does the 13% ratio per boundary hold, or does it decay geometrically?
- Can the cross-architecture linear projection assumption survive more diverse families (e.g., Mamba/SSM models, or decoder-vs-encoder-only), or does R² degrade below usable thresholds?
- The emergent routing favors Phi-3-mini — is this stable across seeds and benchmarks, or an artifact of its synthetic corpus that would reverse on a different task distribution?
- Could the frozen-graph approach extend to open-ended generation by replacing the cross-attention classifier with a trained LM head, or does the injection-magnitude fragility (α≥0.8) prevent coherent long-form output?
- Is the +1.2pp MMLU margin statistically significant over multiple seeds, and does a warmup-then-constant schedule (as the authors suggest) stabilize the oscillating validation accuracy?
