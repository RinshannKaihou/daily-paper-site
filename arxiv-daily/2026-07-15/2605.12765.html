Inference-Time Machine Unlearning via Gated Activation Redirection

Key finding

GUARD-IT routes each query through a similarity gateway and applies a norm-preserving activation rotation — training-free, gradient-free, weights untouched.

一句话总结 · 把遗忘建模为推理期的输入相关几何旋转：离线聚类出原型引导向量，推理时按相似度网关按需路由并施加保范数旋转，不改任何权重。

Why this matters

A training-free, auditable, quantization-robust unlearning path directly serves GDPR / EU AI Act deletion requests and post-deployment safety intervention.

Critical edge
Suppresses retrieval — does not erase influence. The cosine-similarity gateway is untested against paraphrase, cross-lingual, or suffix attacks that could evade routing, and the rotation approximates rather than provably removes the data's distributed statistical footprint.
01

Two decoupled phases, zero weight edits

offline ≈ 2 min / RTX A6000 (8B) · online = 1 cosine + 1 vector op

OFFLINE · once per forget corpus

Sentence-Transformer embed → cluster → extract
1Embed & L2-normalize every doc in forget set D_f.
2Pick k* by max silhouette; k-Means partitions D_f into semantic clusters with centroids {c_k}.
3At an intermediate layer ℓ, mean-pool residual-stream activations per cluster → Prototype Steering Vectors h̄_f^k + mean norm ρ̄.
4One retain reference PSV h̄_r computed analogously. Best ℓ sits near the first quartile of the stack.
→ output: an auditable library of PSVs (add-only, never mutates weights)

ONLINE · per query

gate → orthogonalize → rotate
1Cosine-gate forms active set K(x) = {j : sim(c_j, φ(x)) ≥ T}, default T=0.55.
2If K(x) empty → no-op. Retain-distribution queries bypass entirely (mechanically explains utility preservation).
3Mean-aggregate active PSVs; project orthogonal to h̄_r to strip shared retain direction.
4Apply norm-preserving rotation: ‖h′‖ = ‖h‖ exactly — only direction changes, LayerNorm / attention undisturbed.
→ α is a continuous deployment-time knob: 0 = no-op, 1 = full forget displacement
Why fluency survives by construction. Because the rotation preserves hidden-state magnitude, the perturbation leaves normalization and attention scaling untouched — so coherent output is structurally guaranteed even at aggressive α, unlike parameter edits that drift under quantization.

03

Structural strengths vs. where the rigor slips

read sideways — not a verdict, a trade map
● Strengths
  • Collapse-immune by construction. ‖h′‖ = ‖h‖ guarantees fluency even at aggressive α.
  • Gateway mechanically explains utility preservation — forget-irrelevant queries are never touched; α becomes a per-query, deployment-time knob.
  • Truly training- / gradient-free. No fine-tuning pipeline, no GPU pairs for 8B, offline < 2 min.
  • Cleanest continual unlearning in the set — appending PSVs sidesteps the interference that dooms sequential gradient updates.
  • Quantization-robust by construction — no weights change, so base and quantized twin share parameters up to the cast.
  • Honest, detailed appendices — layer/token sweeps, BM25-vs-ST gate parity, orthogonal-vs-diff-means, tuned-config tables.
● Limitations
  • Depends on the linear representation hypothesis; may weaken for non-linearly encoded or distributed low-magnitude concepts.
  • Benchmark scope is narrow — only TOFU & MUSE, both entity-level; no procedural, reasoning, multimodal, or adversarial forget sets.
  • TOFU's retain/forget overlap flatters the gateway; real corpora are expected to be harder, not easier.
  • Privacy-leakage is weak under defaults (67.37 on 1B/forget01 vs tuned 54.38) — a tension between maximizing Avg and minimizing exposure.
  • Reversibility is a security liability — anyone with the vector store can restore forgotten content or misuse the gate for censorship; paper recommends but does not provide access control.
  • No adversarial-elicitation evaluation (jailbreaks, paraphrases, cross-lingual) that could evade the cosine gate.
  • Suppresses retrieval, not provably erases distributed influence — an approximation under strict retrain-equivalence.
04

Open questions for further exploration

jump non-linearly — each stands alone
Q1Does the similarity gateway hold against adversarial elicitation — paraphrased, cross-lingual, or suffix-attack queries designed to evade the cosine threshold while still extracting forget-set content?
Q2Can it extend beyond entity-level facts to procedural / skill knowledge, multi-step reasoning chains, and multimodal forget targets where the linear representation hypothesis is less obviously valid?
Q3How does it behave on real corpora with strong retain/forget separation (the opposite of TOFU's overlap) — does Mem improve at constant Util, or does the gateway over-fire?
Q4A head-to-head vs other inference-time interventions (CAST, SADI, ITI) on the same benchmarks — is it the clustered-PSV + rotation design, or input-conditional steering in general, that drives the gains?
Q5Can the privacy-leakage weakness under defaults be closed without sacrificing Avg — e.g. a learned gate, an ST + BM25 ensemble, or a per-cluster α schedule?
Q6What is the forget-set size ceiling before the growing PSV library makes K cluster comparisons a measurable latency cost — and can the gate be approximated with ANN retrieval over centroids?
