RLVR · Web Agents · Negative Result

ALearning-Rate-Gated Failure of GRPO in a Small Web Agent: A Controlled Null and Its Mechanism

By Chengguang Gan, Zhixi Cai, Yunhao Liang, Hanjun Wei, Shiwen Ni, Qinghao Zhang · arXiv 2607.12640

在 18 组受控网格 + 配对 McNemar + TOST 等价检验下，没有任何 GRPO 配置在已掌握任务上可信超越监督基线（49.1%）；中等学习率损毁、高学习率击穿智能体，权重嫁接因果定位损伤到 attention/MLP 块。

KV Cache · Tensor Methods

JoLT: Near-Lossless KV Cache Compression via Joint Tucker and JL-Residual Allocation

By Rahul Krishnan & Volker Schulz · Universität Trier · 2607.12550

KV cache 视为三阶张量，谱分析证明 head/layer 轴不可压缩、K/V 不对称；partial Tucker + JL 残差 + 单一 Lagrangian 联合分配，在 2–3× 区间重建误差比 int4/xKV 低约一个数量级。

Treats the KV cache as a third-order tensor; a partial Tucker plus JL-rotated residual, jointly allocated by one Lagrangian dual, lands roughly an order of magnitude below int4/xKV reconstruction error in the 2–3× free zone.
