OOD Detection · Interpretability / arXiv 2607.12094 / 13 Jul 2026 / arxiv.org/abs/2607.12094

Sparse autoencoders make out-of-distribution detection interpretable.

Ayush Karmacharya* · Luke Luschwitz* · Lucia Romero · Yanan Niu · Joseph Campbell  — Purdue University · EPFL   (*equal contribution)

Key finding

Training top-k sparse autoencoders on a frozen vision backbone's intermediate layers, then scoring by cosine similarity to class prototypes, beats seven post-hoc baselines — while exposing which concepts an OOD input fails to activate.

Key metric · ViT-B/16 semantic shift

92.4 mean AUROC across 9 far-OOD datasets
▲ +3.0 AUROC vs strongest baseline MDS (89.4)

Also leads covariate shift: 83.5 vs VIM 81.0. On ResNet-50 the margin narrows to +0.3 on semantic shift.

Why this matters

It imports mechanistic-interpretability tooling (sparse autoencoders) into vision OOD detection — yielding both higher scores and concept-level explanations, not just a scalar flag.

Critical edge

The interpretability claims rest entirely on GPT-5.5 as an uncalibrated LLM judge — no human ground truth, no code release, so the concept-labeling stage is non-reproducible without paid API access.

Methodology — three stages, backbone untouched

1
Frozen backbone + per-layer SAE training. Pass the ID training set through a pretrained classifier f (ViT-B/16 or ResNet-50); for each selected intermediate layer train an independent top-k sparse autoencoder (MSE reconstruction, no L1 — sparsity enforced by the top-k function). ViT uses latent dim 2048 with k=128 at layers {4,7} and k=384 at {10,13}; ResNet uses 1024 with k=64/{10,22}, k=96/{40,49}. 30 epochs, Adam 1e-4, batch 4096, A100 40GB.
2
Similarity-based scoring. For each class c and layer l precompute the mean latent μ_{c,l}. At test time the predicted class ĉ drives s_l = cos(z_l, μ_{ĉ,l}); the score S(x) is the uniform average over layers and a sample is OOD when S(x) < τ. Uniform averaging and cosine beat Hamming, Mahalanobis, and RMDS-style distances — sparsity makes the relative activation pattern matter more than magnitude.
3
VLM concept annotation. For each SAE latent, retrieve the top-25 activating ID images and prompt GPT-5.5 to name the shared visual theme. A 400-image probe (200 ID / 200 OOD) then measures concept agreement — fraction of top concepts semantically consistent with the image.

Strengths vs Limitations — side by side

+ Strengths

  • Consistent ViT gains across both shift types — the harder covariate setting included.
  • Intermediate-layer story backed by UMAP + per-layer curves + a 12-dataset ablation.
  • Compact inference: one mean vector per class per layer vs KNN's full reference set.
  • Interpretable by design — concept-level diagnostics, not just a scalar score.

− Limitations

  • Vision-only; language/multimodal transfer is future work, not shown.
  • Fixed-k top-k ignores that simple images need few concepts, complex ones many.
  • Per-layer SAE training is a compute barrier for very large backbones.
  • Thin margins on ResNet-50 — SOTA claim is strongest only for ViT.

Core contributions — what's new

  • SAID framework — post-hoc detector; one top-k SAE per selected intermediate layer, backbone frozen.
  • Similarity score — per-layer cosine to the predicted class mean latent, averaged and thresholded.
  • Intermediate layers carry structure the final layer discards; optimal layer varies by dataset.
  • VLM concept pipeline — GPT-5.5 labels each latent from its top-25 images; ID inputs activate more consistent concepts, with the ID–OOD gap widening in deeper layers.

Open questions — for further work

  1. Would an adaptive / learned sparsity schedule (varying k by input complexity) lift both concept quality and AUROC?
  2. Does SAID transfer to language / multimodal backbones (CLIP, LLMs), and does the "later-layer gap" finding hold there?
  3. Can VLM concept labels be validated against human annotations to calibrate the LLM judge?
  4. Is there a principled layer-selection criterion rather than hand-picked {4,7,10,13} / {10,22,40,49}?
  5. Does SAID stack with training-time OOD methods, or does the post-hoc ceiling plateau?
  6. Against activation-shaping (ASH) and other intermediate-feature baselines not included, where does SAID rank?
