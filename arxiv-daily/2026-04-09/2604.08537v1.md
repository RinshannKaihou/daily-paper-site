## Methodology

BrainCoDec consists of a two-stage hierarchical inference pipeline built on transformer architectures:

**Stage 1 -- Voxel-wise encoder parameter estimation.** Building on the prior BrainCoRL work [115], the model takes a context of n image-brain activation pairs for a single voxel and outputs the voxel's encoding function weights via an in-context transformer (T_theta). This is repeated independently for each voxel of interest in higher visual cortex.

**Stage 2 -- Contextual functional inversion.** Each voxel is represented as a context token c_k = [omega_k, beta_k], concatenating its estimated encoder parameters (omega_k from Stage 1) with the measured activation (beta_k) from a novel stimulus. A second transformer (P_gamma) with variable-length voxel contexts performs aggregated functional inversion to predict the image embedding. Log-scaled attention is adopted to handle variable voxel counts, and positional embeddings are omitted for order invariance.

**Training strategy** follows a three-stage pipeline inspired by LLM practices: (1) pretraining on synthetic data (random Gaussian weights with noise), (2) context extension with variable voxel counts (200--4000), and (3) supervised fine-tuning on real fMRI measurements. The training objective combines cosine embedding loss with InfoNCE contrastive loss.

The model is evaluated with multiple visual backbones: CLIP (ViT-B, 55.70M params), DINOv2 (ViT-B, 88.76M params), and SigLIP (ViT-B, 157.35M params).

## Results & Highlights

**Cross-subject decoding (NSD).** On held-out NSD subjects (leave-one-subject-out), BrainCoDec-200 (using only 200 of 9000 support images) achieves a mean Top-1 retrieval accuracy of 22.7% and Top-5 of 54.0% across four unseen subjects, compared to 3.90%/9.81% for MindEye2 (with anatomical alignment) and 0.82%/3.09% for TGBD. This represents a roughly 6x improvement in Top-1 accuracy over the best prior method.

**Cross-scanner generalization (BOLD5000).** A model trained on all four NSD subjects (7T scanner) generalizes directly to BOLD5000 subjects (3T scanner, different voxel size, different stimulus timing) using just 20 in-context images. CLIP backbone achieves 31.45% Top-1 accuracy (chance = 5%), demonstrating strong cross-site transfer without any retraining.

**Context scaling.** Performance scales positively with both image context size (Stage 1) and voxel context size (Stage 2). Remarkably, with only 200 images and 4000 voxels, the model approaches the performance of using the full context (~9000 images and all higher-visual-cortex voxels).

**Robustness.** Masking category-selective ROIs (e.g., removing PPA/OPA/RSC for scenes) leads to minimal performance degradation for most categories, indicating distributed rather than localized decoding. Only scene-selective regions show some vulnerability.

**Interpretability.** Attention analysis reveals that the model learns to allocate attention consistent with known functional organization: face stimuli activate FFA/EBA voxels, place stimuli activate PPA/OPA/RSC voxels. UMAP projections of attention weights show clear semantic clustering.

## Critical Assessment

**Strengths:**
- The hierarchical in-context learning formulation is elegant and principled, cleanly separating the "understand this subject's brain" step from the "decode this image" step.
- The zero-shot cross-subject and cross-scanner results represent a substantial advance over prior methods that require per-subject fine-tuning or anatomical alignment.
- Strong interpretability through attention analysis, with learned attention patterns matching known cortical functional organization.
- The synthetic-data pretraining stage is a clever way to bootstrap without consuming limited fMRI data.
- Code and models are promised to be publicly available.

**Weaknesses:**
- The method still requires a non-trivial calibration set (200 images with brain recordings) from each new subject. While less burdensome than full fine-tuning, this is not truly "zero-shot" -- it is few-shot, and the paper could be more transparent about the practical cost of acquiring these calibration data.
- The evaluation is restricted to semantic retrieval (nearest-neighbor in embedding space) rather than pixel-level image reconstruction. While the authors argue that image synthesis is a separate (and already well-studied) problem, the gap between retrieval accuracy and perceptual quality of reconstructed images remains an open question.
- Training requires leave-one-subject-out model variants (5 models per backbone, 15 total), which is still somewhat cumbersome. The "foundation model" claim would be more convincing if a single model worked universally.
- Performance on non-NSD subjects (S3/S4/S6/S8 with incomplete scanning) shows lower accuracy (14--20% Top-1), suggesting sensitivity to data quality and quantity that is not fully analyzed.
- The comparison with MindEye2 may be slightly unfair, as MindEye2 is applied with anatomical alignment but was originally designed for within-subject scenarios. A more controlled comparison with a cross-subject variant of MindEye2 would strengthen the claims.
- The method is evaluated only on visual decoding from higher visual cortex. Generalization to other brain regions or cognitive tasks remains unexplored.
