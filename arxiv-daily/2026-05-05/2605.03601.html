Most ReLU Networks Admit Identifiable Parameters

arXiv: 2605.03601

Date: 2026-05-05

https://arxiv.org/abs/2605.03601

一句话总结： 本文通过加权多面体复形框架证明，几乎所有ReLU网络架构（输入和隐藏层宽度≥2）都存在参数可识别的开集，精确给出功能维度 = 参数数 - 隐藏神经元数，同时揭示了最小表示仍可有连续参数冗余，并建立了通用深度不可替代性结果。
TL;DR: This paper proves that for every ReLU network architecture with input and hidden layer widths ≥ 2, there exists an open set of identifiable parameters (recoverable from the realized function up to scaling/permutation). This yields the exact functional dimension = #parameters − #hidden neurons. The authors further show that minimal network representations can still have positive-dimensional fibers (continuous non-trivial redundancies), refuting a conjecture by Shahverdi et al. (2026a), and establish that for generic parameters, depth cannot be traded for width — deeper architectures realize functions that shallower ones generically cannot.

核心贡献

  • 统一的多面体框架：引入加权多面体复形（weighted polyhedral complexes）、断点复形（breakpoint complex）和依赖图（dependency graph）来描述ReLU网络的函数-参数映射关系，将函数的非线性几何结构与参数空间分离
  • 可识别性结果：对于所有输入和隐藏层宽度≥2的ReLU网络架构，证明了存在参数可识别的开集，即从函数可以唯一恢复参数（仅差缩放和排列对称性），大幅推广了Grigsby et al. (2023)（要求nℓ≥d）和Phuong & Lampert (2020)（要求金字塔结构）的结果
  • 精确功能维度：作为推论，确定了几乎所有ReLU架构的功能维度 = 参数数量 − 隐藏神经元数量（即去掉正缩放对称性后的维度）
  • 最小性≠可识别性：构造了一个开集的最小但不可识别的参数（正维纤维），证明即使没有神经元可以被移除，连续参数冗余仍然存在，反驳了Shahverdi et al. (2026a)的猜想
  • 通用深度层级：证明了对于大多数架构，存在开集参数使得其实现的函数无法被任何更浅的网络用通用参数表示——即通用参数下深度不可用宽度替代

Key Contributions

  • Unified polyhedral framework introducing weighted polyhedral complexes, the breakpoint complex (a functional invariant separating visible function geometry from latent parameter geometry), and a dependency graph encoding the layer hierarchy of nonlinearities
  • Identifiability for almost all architectures: For every ReLU architecture with input/hidden width ≥ 2, there exists a nonempty open set of identifiable parameters — substantially generalizing prior width-dependent results (Grigsby et al. 2023 required nℓ ≥ d; Phuong & Lampert 2020 required pyramidal structure)
  • Exact functional dimension: dim(µ_A(Θ_A)) = #parameters − #hidden neurons for nearly all ReLU architectures (Theorem 4.11)
  • Minimality ≠ identifiability: Constructs an open set of minimal parameters with positive-dimensional fibers, showing that continuous parameter redundancies persist even when no neurons can be removed (Proposition 5.23, refuting a conjecture of Shahverdi et al. 2026a)
  • Generic depth hierarchy: For generic parameters, depth cannot be traded for width — an open set of parameters in a deeper network cannot be generically represented by any shallower network (Proposition 5.24)

Methodology

The paper develops a three-part technical framework:

  1. Weighted polyhedral complexes (Section 3): The authors define tropical weights on facets of the canonical polyhedral complex that record how the linear part of the function changes across each facet (Lemma 2.1). They introduce generic parameters as an open dense set satisfying generic hyperplane arrangements and maximal-rank conditions on matrix products. For generic parameters, they prove supertransversality (codimension-k faces lie in exactly k bent hyperplanes) and cancellation-freeness (tropical weights vanish iff an entire layer is inactive). Transparency (at least one neuron active per input) is shown to imply the Linear Region Assumption (LRA).
  1. Slab-layer construction (Section 4): To construct identifiable parameters, the authors introduce oriented slab layers — bounded polytope-based constructions where n nearly-parallel hyperplanes partition a region into n+1 cells with controlled activation patterns. The construction proceeds inductively: each layer is designed to be transparent on the image of the previous layers and to satisfy TPIC (Transverse Pairwise Intersection Condition) with the preceding layer's bent hyperplanes. Transparency + genericity + TPIC + LRA ⇒ identifiability (Theorem 3.13, strengthening Rolnick & Kording 2020 by resolving the neuronwise sign ambiguity via generic rank conditions).
  1. Configuration varieties for fiber analysis (Section 5): The breakpoint complex (a subcomplex of the canonical complex retaining only facets visible in the function) and its dependency graph provide functional invariants. The configuration variety V_{(ϕ,s)} is an algebraic variety encoding the polynomial constraints that parameters must satisfy to match the visible geometry. The fiber is contained in a finite union of projections of such varieties, enabling both the minimality-without-identifiability construction and the depth hierarchy proof.

For the minimality ≠ identifiability result (Proposition 5.23), the construction composes an identifiable parameter (from Theorem 4.10) with a last hidden layer containing "bending neurons" plus two "linear neurons" (always active, contributing only an affine map). The GL₂(ℝ) freedom in the linear neurons creates a 2-dimensional family of equivalent parameters, yet the rank condition (Lemma 5.22) prevents any strict submodel from realizing the same function.

For the depth hierarchy (Proposition 5.24), the dependency graph of the constructed parameter contains a chain v₁ → v₂ → ⋯ → v_L spanning all L hidden layers. Any generic shallower network with L' < L layers cannot accommodate a strictly increasing sequence of L layer indices, creating an obstruction.

Results & Highlights

  • Theorem 4.10: For every architecture A = (n₀, …, n_L, m) with n_i ≥ 2 for i ∈ {0,…,L}, there exists a nonempty open subset U ⊆ Θ̃_A of identifiable generic parameters
  • Theorem 4.11 (Functional Dimension): dim(µ_A(Θ_A)) = Σ_{ℓ=1}^{L+1} n_ℓ(n_{ℓ-1}+1) − Σ_{ℓ=1}^{L} n_ℓ for all architectures with n_i ≥ 2
  • Proposition 5.23: For architectures with last hidden layer width ≥ 4, there exists an open set of minimal parameters with positive-dimensional fibers
  • Proposition 5.24 (Generic Depth Hierarchy): For any architecture with n_i ≥ 2, there exists an open set of parameters whose realized functions cannot be represented by any shallower network with generic parameters
  • Appendix A: Identifies a gap in the construction of Grigsby et al. (2023) — their inductive construction does not guarantee LRA for the final realized function, as later layers may cancel visible nonlinearities from earlier layers. The present paper fixes this via explicit transparency and cancellation-freeness conditions.

Critical Assessment

  • Novelty: Very High. The results substantially advance the state of the art on ReLU network identifiability. Prior results required either pyramidal architectures (Phuong & Lampert 2020) or hidden layer widths ≥ input dimension (Grigsby et al. 2023). This paper removes both constraints simultaneously. The minimality ≠ identifiability result is unexpected and the identification of the gap in Grigsby et al. (2023) is intellectually honest and valuable.
  • Rigor: Strong. The proofs are detailed and self-contained. The polyhedral framework provides clean abstractions. The construction is explicit (oriented slab layers with bounded polytopes), not merely existential. The identification of the gap in prior work (Appendix A) and the fix via transparency/cancellation-freeness demonstrate careful scholarship.
  • Reproducibility: Good. Theoretical work with explicit constructions. The slab-layer construction is algorithmic and could be implemented. All conditions (TPIC, LRA, transparency, genericity) are precisely defined with open/dense characterizations.
  • Significance: High. The functional dimension formula is the definitive answer for nearly all ReLU architectures. The minimality ≠ identifiability result has implications for understanding loss landscape geometry, neural network pruning, and the theoretical limits of parameter recovery. The generic depth hierarchy is complementary to existing depth-separation results and uses a fundamentally different proof technique.
  • Red flags / weaknesses:
  • The identifiability result is existential (open set), not quantitative — the Lebesgue measure of the identifiable set vs. non-identifiable set is unknown (acknowledged as Open Question 1).
  • The construction requires width ≥ 2 in all non-output layers; the width-1 case remains open (Open Question 4).
  • The slab-layer construction is quite specialized (nearly parallel hyperplanes) and may not reflect typical trained network parameters.
  • The minimality ≠ identifiability construction requires the last hidden layer width ≥ 4, which is a mild but non-trivial constraint.
  • No experimental validation or comparison with trained networks is provided (though this is standard for theoretical work in this area).

Strengths

  • Resolves the functional dimension question for nearly all ReLU architectures with a clean, exact formula
  • The weighted polyhedral complex framework unifies and clarifies several prior approaches (tropical weights, bent hyperplanes, linear regions)
  • The breakpoint complex and dependency graph provide new structural tools for analyzing parameter fibers
  • Identifies and fixes a genuine gap in the influential Grigsby et al. (2023) construction
  • The minimality ≠ identifiability result is surprising and has foundational significance — it shows that the parameter-function mapping is more complicated than previously understood
  • The generic depth hierarchy proof is elegant: a chain in the dependency graph creates an obstruction that no shallower architecture can overcome

Limitations

  • The measure of the identifiable parameter set is not characterized — it could be very small relative to the full parameter space
  • Results are limited to ReLU activation; extension to leaky ReLU, GELU, or smooth activations is not discussed
  • No connection to practical training dynamics, optimization landscapes, or generalization is established
  • The width-1 case (architectures with bottleneck layers) remains open
  • Identifiability over finite data (rather than the full function) is listed as an open question with no partial results
  • The specialized nature of the slab-layer construction means the identifiable parameters may be far from typical random initialization or trained parameters

Relevance

本文对ReLU网络参数可识别性的完整刻画和功能维度的精确公式，为理解神经网络参数化与函数空间的关系提供了严格的数学基础。加权多面体复形框架和最小性≠可识别性的发现，与过度参数化、损失景观几何和深度-宽度权衡等基础理论问题直接相关，是深度学习理论的重要进展。
