Realizable Bayes-Consistency for General Metric Losses

arXiv: 2605.03823

Date: 2026-05-05

https://arxiv.org/abs/2605.03823

一句话总结： 本文在可实现设定下刻画了任意（可能无界）度量损失下存在分布无关强贝叶斯一致学习规则的充要条件——假设类 H 不包含无穷非递减 (γ_k)-Littlestone 树（γ_k→∞）；并给出构造性学习规则，在无此障碍时风险几乎必然收敛到 0。
TL;DR: The authors resolve the realizable case of the open problem of Tsir Cohen & Kontorovich (2022) by giving a sharp hypothesis-class characterization of realizable strong universal Bayes-consistency (RSUBC) for general (possibly unbounded) metric losses. Under Polish-space assumptions and a compact-parameterization condition (Lemma 4.3), H admits a distribution-free learning rule with risk → 0 a.s. for every realizable distribution if and only if H contains no infinite non-decreasing (γ_k)-Littlestone tree with γ_k → ∞ (Theorem 4.5). The lower bound (Theorem 5.1) shows that if such a tree exists, every learner A suffers E_{S_n}[R(A(S_n))] = ∞ for every n; the upper bound (Theorem 6.5) constructs an explicit rule (drive a Gale-Stewart game on half the sample, partition X into countably many bounded-range cells, run MedNet per cell). A counterexample (Section 3) proves the finiteness condition R* < ∞ is insufficient.

核心贡献

  • 给出可实现设定下一般（可能无界）度量损失的强通用贝叶斯一致性（RSUBC）的完整假设类刻画：H 可学习 ⟺ H 不含无穷非递减 (γ_k)-Littlestone 树（γ_k→∞）（Theorem 4.5）。
  • 下界（Theorem 5.1）：若存在此树，则对任意学习算法 A，存在可实现分布 μ̄ 使 E_{S_n}[R(A(S_n))]=∞ 对所有 n 成立——即"灾难性稀有事件"使风险无法消失。
  • 上界（Theorem 6.5）：若无此障碍，构造显式分布无关学习规则——在半样本上驱动 Gale-Stewart 博弈并稳定，将 X 划分为可数个有界直径单元，每单元运行 MedNet（Tsir Cohen & Kontorovich 2022）并聚合，风险几乎必然收敛到 0。
  • 反例（Section 3）：证明候选条件 R<∞ 不足以保证一致性——构造 X=(0,1)、Y=ℕ_0、ℓ_1 损失、H 在区间 I_k 上取 {0, 2^{2k+1}} 的可实现类，其中 R=0 但任意学习器风险无穷。
  • 通过 Lemma 4.3（桥接引理）在紧参数化假设（紧 Θ、h 在 θ 连续）下将有限前缀可实现性自动提升为无穷路径可实现性，并论证该假设温和且覆盖几乎所有实际场景。

Key Contributions

  • A complete hypothesis-class characterization of realizable strong universal Bayes-consistency (RSUBC) for general (possibly unbounded) metric losses: H is RSUBC-learnable if and only if H admits no infinite non-decreasing (γ_k)-Littlestone tree with γ_k → ∞ (Theorem 4.5), under Polish-space assumptions and the compact-parameterization condition (Lemma 4.3).
  • Lower bound (Theorem 5.1): if such a tree exists, then for every learning algorithm A there is a realizable distribution μ̄ such that E_{S_n∼μ̄^n}[R(A(S_n))] = ∞ for every n — catastrophic rare-event failures driven by Borel-Cantelli on unseen tree depths where label gaps γ_{k_m} ≥ m² force per-depth conditional risk ≥ m²/2.
  • Upper bound (Theorem 6.5): in the obstruction-free regime, an explicit distribution-free learning rule achieves R_μ̄(f̂_n) → 0 almost surely — split the sample, drive a Gale-Stewart game to stabilization (K∞ < ∞ a.s.), partition X into countably many cells C_{K,j} with bounded label regions Y_{K,j} (diam ≤ 4γ_{K+1}), run MedNet per cell, and aggregate.
  • Counterexample (Section 3) proving the candidate finiteness condition R < ∞ is insufficient: on X=(0,1), Y=ℕ_0, ℓ_1, with H = {f : ∀k, f(x) ∈ {0, 2^{2k+1}} for x ∈ I_k=(2^{-k}, 2^{-(k-1)})} and a coin-flip construction, R(μ̄_B)=0 yet every learner suffers R_μ̄_B(h_n)=∞ a.s. because infinitely many unseen intervals contribute ≥ 2^k each.
  • Bridging Lemma 4.3: under compact Θ and h continuous in θ, finite-prefix realizability (Definition 4.1) automatically implies infinite-path realizability (Definition 4.2) via the finite-intersection property of compact spaces; the assumption is shown to be mild (covers finite/compact Y, bounded-weight NNs, Lipschitz functions on compact domains) and does not rule out unlearnability (the Section 3 counterexample satisfies it).

Methodology

Setting. Instance space (X, ρ) and label space (Y, ℓ) are Polish (separable, complete metric) spaces; ℓ may be unbounded. Hypothesis class H ⊆ Y^X satisfies the measurability condition of Bousquet et al. (2020, Def. 3.3) with a compact parameter space Θ and evaluation map h: Θ × X → Y continuous in θ. Realizable means inf_{h∈H} R_μ̄(h) = 0; under compactness this equals strict realizability (existence of f∈H with Y=f(X) a.s.) by attainment of the infimum.

Obstruction (Definition 4.1–4.2). A non-decreasing-(γ_k)-Littlestone tree for H is a rooted full binary tree whose depth-k internal nodes carry instances x_{k,i}∈X, whose two outgoing edges carry labels y_{k,i,1}, y_{k,i,2}∈Y with ℓ(y_{k,i,1}, y_{k,i,2}) ≥ γ_k, and such that every finite root-to-leaf path is realizable by some h∈H. An infinite such tree is realizable if every infinite path is realized by a single h∈H. Lemma 4.3 bridges the two notions under compact-parameterization.

Lower-bound construction (Theorem 5.1). Select depths k_1 < k_2 < … with γ_{k_m} ≥ m²; assign probability mass p_m ∝ 1/m (normalized) to depth-k_m nodes. A random root-to-leaf path is drawn by independent fair coins B_{k_m}; the tree's realizability ensures μ̄_B is realizable by some f*_B ∈ H. A sample of size n touches ≤ n depths, so all but finitely many k_m are unseen; at each unseen depth the learner's prediction is independent of the true coin, so by the triangle inequality E[ℓ(h_n(x_{k_m}), Y) | S_n] ≥ m²/2. Borel-Cantelli (second lemma) forces infinitely many "bad" outcomes → R_μ̄_B(h_n) = ∞ a.s.; Fubini over paths yields a single μ̄ with infinite expected risk at every n.

Upper-bound construction (Theorem 6.5, four steps). (1) Encode the tree obstruction as a Gale-Stewart game (Section 6.1): adversary PA picks (ξ_k, η_{k,1}, η_{k,2}) with gap ≥ γ_k; learner PL picks a label; PL wins if PA has no legal move at a finite round. Absence of an infinite tree ⟺ PL has a measurable winning strategy σ. (2) Drive the game on an i.i.d. sample (Section 6.3): it stabilizes after finitely many advances a.s. (K∞ < ∞), yielding per-x history-conditional label sets H_{K∞}(x) of diameter ≤ γ_{K∞+1} (Lemma 6.2) that contain the true label a.s. (Lemma 6.3). (3) Partition X into countably many cells C_{K,j} via a countable dense subset {q_j}⊆Y, each with bounded label region Y_{K,j} of diameter ≤ 4γ_{K+1} (Lemma 6.4). (4) On each cell, run MedNet (Tsir Cohen & Kontorovich 2022) on the restricted BIE subproblem and aggregate; risk decomposes as Σ p_j R_{μ̄_j}(f̂_{n,j}) → 0 by countable intersection of probability-one events.

Constructive learning rule (Section 6.5). Split S_n into two halves; run stabilization on the first half to get (K, τ_K); compute H_K(·), j_K(·), cells C_{K,j}, regions Y_{K,j}; run MedNet on each cell's second-half subsequence; output f̂_n(x) = f̂_{n,j_K(x)}(x).

Results & Highlights

  • Main characterization (Theorem 4.5): Under Polish assumptions and compact-parameterization (Lemma 4.3), the following are equivalent: (1) there exists a distribution-free learning rule A with R_μ̄(A(S_n)) → 0 a.s. for every realizable μ̄; (2) H admits no infinite non-decreasing-(γ_k)-Littlestone tree for any non-decreasing gap sequence γ_k → ∞.
  • Lower bound (Theorem 5.1): If H contains a realizable infinite non-decreasing-γ_k-Littlestone tree with γ_k → ∞, then for any learning algorithm A there exists a realizable distribution μ̄ with E_{S_n∼μ̄^n}[R(A(S_n))] = ∞ for all n. The construction selects depths with γ_{k_m} ≥ m² and mass p_m ∝ 1/m; unseen depths force conditional risk ≥ m²/2; Borel-Cantelli yields infinite risk a.s.
  • Upper bound (Theorem 6.5): If no such tree exists, the explicit learning rule of Section 6.5 satisfies R_μ̄(f̂_n) → 0 almost surely. The proof bounds the aggregated risk by Σ_{j≤J} p_j·ε + Σ_{j>J} p_j·4γ_{K+1} ≤ ε + 4γ_{K+1}·ε, with γ_{K+1} < ∞ on the fixed sample path and ε arbitrary.
  • R < ∞ is insufficient (Section 3 counterexample): X=(0,1), Y=ℕ_0, ℓ_1, H = {f : ∀k, f(x) ∈ {0, 2^{2k+1}} for x ∈ I_k=(2^{-k},2^{-(k-1)})}. For coin-flip f_B, R(μ̄_B)=0 < ∞, yet conditional on any sample infinitely many unseen intervals I_k contribute ≥ 2^k each → R_μ̄_B(h_n)=∞ a.s.
  • Bounded-diameter lemma (Lemma 6.2): For every realizable history τ_k and x ∈ X, diam(H_k(x)) ≤ γ_{k+1}, ensuring the stabilization yields bounded label sets.
  • True-label containment (Lemma 6.3): With probability one over S_∞, for the terminal history τ_{K∞}, μ̄{(x,y) : y ∉ H_{K∞}(x)} = 0.
  • Cell-wise bounded range (Lemma 6.4): For every k, j and x ∈ C_{k,j}, H_k(x) ⊆ Y_{k,j} where Y_{k,j} = {y : ℓ(y, q_j) ≤ 2γ_{k+1}}; hence conditional loss within each cell is bounded by 4γ_{K+1}.
  • Example 1 (not learnable, Appendix A.1): The Section 3 class admits a realizable tree with γ_k = 2^{2k+1} → ∞; the disjoint intervals allow independent binary choices at geometrically increasing loss scales.
  • Example 2 (learnable, Appendix A.1): L-Lipschitz functions on [0,1] with ℓ = absolute loss have unbounded range but admit no infinite unbounded-gap tree: after the depth-1 commitment, all subsequent gaps are bounded by 2L, so the adversary has no legal move once γ_k > 2L.

Critical Assessment

  • Realizable setting only; agnostic case open: The characterization assumes inf_{h∈H} R_μ̄(h) = 0. The authors explicitly flag (Appendix A.5) that the agnostic extension is blocked because the stabilization procedure relies on Y_i = f*(X_i) ∈ V_k(X_i) (true label feasibility); under agnostic distributions sample points may not be legal game moves, so the upper-bound argument breaks. This is the practically important case, since real data is rarely realizable.
  • Compact-parameterization assumption is load-bearing: The equivalence between finite-prefix (Definition 4.1) and infinite-path (Definition 4.2) trees requires compact Θ and h continuous in θ (Lemma 4.3). Appendix A.4.2 gives a counterexample (Θ = ℕ_0 discrete, h(n, x_k) = k-th binary digit of n · k) where a Definition 4.1 tree coexists with RSUBC — so the main theorem's lower bound cannot be proved from Definition 4.1 alone without the assumption. The authors argue the assumption is mild, but it is a genuine restriction whose removal is posed as an open question (A.4.6).
  • The learning rule is "explicit" only relative to a non-constructive strategy: The upper bound requires a measurable winning strategy σ for PL in the Gale-Stewart game, whose existence is established non-constructively via Bousquet et al. (2020, Theorem B.1) determinacy. The authors acknowledge (Appendix A.2) that σ "can be described explicitly via transfinite induction on the ordinal dimension" for countable-ordinal Littlestone dimension, but this is a theoretical, not practical, construction.
  • No finite-sample rates: The paper proves almost-sure convergence R_μ̄(f̂_n) → 0 but provides no finite-sample convergence rates; the universal-learning program's rate results (Bousquet et al. 2020; Hanneke et al. 2023; Attias et al. 2024b) are not extended to the unbounded-metric-loss realizable setting here.
  • Counterexample is somewhat pathological: The Section 3 negative example uses Y = ℕ_0 with ℓ_1 and labels growing as 2^{2k+1} on disjoint intervals — a constructed worst case. Whether the obstruction arises in more natural hypothesis classes (beyond the two appendix examples) is not systematically surveyed.

Strengths

  • A complete necessary-and-sufficient characterization — the gold standard in learnability theory — resolving a stated open problem.
  • The unbounded-gap Littlestone tree is a clean, intuitive combinatorial obstruction that cleanly generalizes the bounded-loss Littlestone tree to the metric-loss setting, with the scale-divergence ingredient capturing precisely why unbounded losses break the bounded-loss intuition.
  • Both directions are constructive in mechanism: the lower bound gives an explicit adversarial distribution (coin-flip on tree paths with γ_{k_m} ≥ m²); the upper bound gives an explicit four-step learning rule (Gale-Stewart → stabilize → partition → MedNet per cell).
  • The R* < ∞ counterexample (Section 3) is valuable: it definitively rules out the most natural candidate condition, sharpening what the "right" characterization must be.
  • Honest treatment of assumptions: Lemma 4.3 and Appendix A.4 carefully justify the compact-parameterization condition, show where it is necessary (A.4.2), and confirm it does not rule out unlearnability (A.4.3, via the Section 3 counterexample which satisfies it).

Limitations

  • Realizable setting only; the agnostic extension is open and blocked by a genuine proof obstacle (Appendix A.5).
  • The compact-parameterization assumption (compact Θ, h continuous in θ) is required for the lower bound; without it, Definition 4.1 and Definition 4.2 trees can diverge (Appendix A.4.2).
  • The learning rule depends on a non-constructively-obtained measurable winning strategy σ; it is "explicit" only given σ, and σ's concrete description requires transfinite induction on ordinal Littlestone dimension.
  • No finite-sample convergence rates; only almost-sure asymptotic consistency.
  • Pure learning theory with no experiments; practical applicability is indirect (the obstruction is a structural diagnostic, not an algorithmic tool).
  • The negative examples are somewhat pathological (ℕ_0 labels with exponentially growing gaps); the survey of natural hypothesis classes is limited to two appendix examples (one positive: L-Lipschitz on [0,1]; one negative: the Section 3 class).

Questions for Further Exploration

  • Can the characterization be extended to the agnostic setting, and what new obstruction would replace the realizable Gale-Stewart stabilization (given that sample labels may not be feasible game moves)?
  • What finite-sample convergence rates are achievable in the obstruction-free regime, and do they depend on the tree-free structure beyond mere absence of the obstruction?
  • Can the compact-parameterization assumption (Lemma 4.3) be removed, or is there a genuinely different characterization for non-compact Θ (as the Appendix A.4.2 counterexample suggests)?
  • Is there a computationally efficient approximation to the measurable winning strategy σ, avoiding transfinite induction, for common hypothesis classes (NNs with bounded weights, Lipschitz functions)?
  • How does the unbounded-gap Littlestone tree relate to online learnability under unbounded losses (Attias et al. 2024a's scaled Littlestone dimension), and can the realizable and online characterizations be unified?
  • Does the "catastrophic rare-event" failure mode (R* = 0 but E[R] = ∞) have analogues in SDC or numerical-reliability settings, where rare high-magnitude errors can dominate expected loss?
