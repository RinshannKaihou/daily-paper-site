Provable Data Scaling Law for Meta Learning via Complexity Minimization

arXiv: 2606.02008

Date: 2026-06-03

https://arxiv.org/abs/2606.02008

一句话总结： 本文提出"复杂度最小化"元表示学习框架，通过Lepski自适应方法估计各源域的最优模型复杂度并以最坏情况最小化为目标学习特征提取器，首次从端到端理论上证明了元训练数据量增长时下游少样本回归的误差率会系统性改善。
TL;DR: This paper introduces complexity minimization, a meta-representation learning framework that uses Lepski's method to estimate per-domain optimal model complexities and learns a feature extractor by minimizing the worst-case complexity across source domains, providing the first end-to-end theoretical proof that downstream few-shot regression error improves systematically as meta-training data scales.

核心贡献

  • 提出复杂度最小化（complexity minimization）元表示学习框架：元学习器选择特征提取器，使得各基学习器报告的域内最优模型复杂度的最坏情况被最小化。
  • 构造基于Lepski方法的新型最优复杂度估计器，无需事先知道底层复杂度参数即可从样本中自适应地识别最优复杂度级别。
  • 提供从元训练到下游学习的端到端理论分析，证明下游超额误差满足数据缩放规律：误差率指数随元训练样本量$m$增大而趋近理想值$\beta_P^*$。

Key Contributions

  • Introduces complexity minimization, a meta-representation learning framework where the meta-learner selects a feature extractor by minimizing the worst-case best model complexity across observed source domains.
  • Constructs a novel estimator of the best model complexity using Lepski's method, enabling adaptive model selection without prior knowledge of underlying complexity parameters.
  • Provides an end-to-end theoretical analysis spanning meta-training through downstream regression, proving that the downstream excess error exhibits the data scaling law: the convergence exponent approaches the ideal $\beta_P^*$ as meta-training sample size $m$ grows.
  • Empirically validates that adding a spectral-norm-based complexity regularizer to four standard meta-learning algorithms (first-order MAML, second-order MAML, Prototypical Networks, R2-D2) consistently improves downstream sample efficiency on CIFAR-10 after Mini-ImageNet pre-training.

Methodology

The paper addresses a fundamental gap: empirical scaling laws show that pre-training on more data improves downstream performance, yet existing theoretical frameworks for meta-learning yield downstream error rates that are independent of pre-training data size.

Complexity Minimization Framework. The framework follows a meta-learner/base-learner architecture. Each base-learner is associated with one observed domain $d$ and assesses a feature extractor $\phi$ by its best model complexity $J_n^{\dagger(d)}(\phi)$, rather than by raw regression error. The best model complexity is the complexity level $J$ that minimizes the downstream excess error for domain $d$:

$$J_n^{\dagger(d)}(\phi) = \arg\min_{J \in \mathbb{N}} \bar{\mathcal{E}}_\phi(f_{n,J}; X^{(d)}, Y^{(d)}).$$

The meta-learner then selects $\phi$ to minimize the worst-case complexity across all $D$ observed domains:

$$\min_{\phi \in \Phi} \max_{d=1,\dots,D} J_n^{\dagger(d)}(\phi).$$

Lepski's Method for Complexity Estimation. Since $J_n^{\dagger(d)}(\phi)$ depends on the unknown downstream distribution, each base-learner estimates it from pre-training samples using Lepski's method — an adaptive model selection procedure from nonparametric statistics. For each candidate complexity $J$, a sieved least-squares estimator $\hat{f}_{m,J}$ is computed. The estimated complexity $\hat{J}^{(d)}(\phi)$ is the smallest $J$ such that the empirical distance between estimators at $J$ and any larger $J' \leq m/\ln m$ does not exceed a variance-dependent threshold $\rho V_{m,J'}$. The meta-learner then solves:

$$\hat{\phi} = \arg\min_{\phi \in \Phi} \max_{d \in [D]} \hat{J}^{(d)}(\phi).$$

Downstream Learning. At downstream time, Lepski's method is applied again with the learned extractor $\hat{\phi}$ to select the final head-function complexity $\hat{J}_n$ from $n$ downstream samples.

Theoretical Instantiation. The analysis uses a polynomial approximation-error assumption: $\mathcal{A}_J(\phi; X, Y) \asymp J^{-2\alpha(\phi;X,Y)}$, which yields an ideal downstream convergence exponent $\beta_P^* = \sup_{\phi \in \Phi} \inf_{(X,Y) \in P} 2\alpha/(2\alpha+1)$. Additional assumptions concern the metric entropy of the feature-extractor class $\Phi$, near-uniform domain sampling, and sub-Gaussian noise.

Results & Highlights

Main Theorem (Theorem 2). Under Assumptions 1–4, with meta-training sample size $m$, downstream sample size $n$, and $D$ domains satisfying $\ln^{1+\nu\gamma} m \lesssim D$ and $\ln D \lesssim \ln m$, the downstream excess error satisfies with high probability:

$$\bar{\mathcal{E}}_{\hat{\phi}}(\hat{f}_n) \lesssim \left(\frac{n}{\ln n}\right)^{-\beta_P^* + O(\frac{\ln^\gamma m}{\ln n})}.$$

This is the first end-to-end theoretical result establishing the data scaling law for pre-training: as $m \to \infty$, the exponent approaches $\beta_P^*$, meaning the downstream error decays faster with $n$ when more meta-training data is available.

Empirical Validation. The authors augment four representative meta-learning algorithms with a spectral-norm regularizer on model parameters as a proxy for complexity minimization:

  • First-order MAML (FO)
  • Second-order MAML (SO)
  • Prototypical Networks
  • R2-D2

Models are pre-trained on Mini-ImageNet in the 5-way 1-shot setting with varying meta-training sizes $m \in \{8{,}000, 16{,}000, 24{,}000, 32{,}000, 40{,}000\}$, then fine-tuned on subsets of CIFAR-10 with varying $n$. Results show that complexity regularization yields clear and consistent improvements in downstream test error across all algorithms and pre-training sizes. Additional experiments (in the appendix) confirm that regularizing other norms (e.g., $\ell_1$) and testing on other datasets also improves sample efficiency.

Critical Assessment

  • Novelty: High. The paper closes a significant theoretical gap. Prior meta-learning theory proved reduced downstream sample complexity, but the derived bounds were independent of pre-training data size — contradicting the empirically observed scaling law. This is the first end-to-end framework that provably links meta-training scale to downstream convergence rate.
  • Rigor: Strong. The theoretical analysis is carefully structured with explicit assumptions, clear theorem statements, and connections to established results in nonparametric statistics (Lepski 1991; Hayakawa & Suzuki 2020; Schmidt-Hieber 2020). The authors honestly flag that constructing concrete feature-extractor families satisfying Assumption 2 remains an open problem.
  • Reproducibility: Good. The experimental setup is standard (Mini-ImageNet → CIFAR-10, 5-way 1-shot, well-known algorithms). The paper mentions JAX-based implementations and provides sufficient experimental detail in the appendix. However, no public code repository is cited in the main text.
  • Significance: High. A provable data scaling law for meta-learning has broad implications for understanding why foundation models improve with scale and could guide the design of future pre-training algorithms.
  • Red flags: The main theorem's error rate includes an $O(\ln^\gamma m / \ln n)$ term in the exponent, which means the scaling benefit materializes only when $n$ is exponentially larger than $m$ (or when $m$ is very large). The practical regime where this term is small may be limited. The empirical validation uses a spectral-norm proxy rather than the full Lepski-based algorithm, so the experiments test a simplified version of the theory.

Strengths

  • Identifies and addresses a real theoretical gap between empirical scaling laws and existing meta-learning theory.
  • Introduces a principled, mathematically grounded framework with a clear optimization objective (worst-case complexity minimization).
  • Leverages classical statistical tools (Lepski's method) in a novel meta-learning context.
  • Provides honest discussion of limitations (open problem on constructing suitable $\Phi$, regression-only analysis).
  • Empirical results are algorithm-agnostic and consistent across multiple baselines.

Limitations

  • Assumption 2 (metric entropy conditions on the feature-extractor class $\Phi$) is abstract; the authors explicitly note that exhibiting a concrete class satisfying it remains an important open problem.
  • The theoretical analysis is restricted to regression; extension to classification and structured prediction is left as future work.
  • The empirical experiments use a proxy (spectral-norm regularization) rather than the full Lepski-based complexity minimization algorithm, so the gap between theory and practice is not fully closed.
  • The scaling-law benefit depends on the $O(\ln^\gamma m / \ln n)$ term becoming negligible, which may require very large downstream sample sizes $n$.
  • The framework assumes a shared common representation across all domains in $P$; domains with fundamentally incompatible structures may violate this.

Relevance

Questions for Further Exploration

  1. Can concrete families of feature extractors (e.g., specific neural network architectures with controlled depth/width) be shown to satisfy Assumption 2's metric entropy conditions?
  2. How does the framework extend to classification settings, where the notion of "best model complexity" and Lepski's method require non-trivial adaptation?
  3. Does the empirical spectral-norm proxy capture the same mechanism as the theoretical Lepski-based algorithm, or is there a systematic gap?
  4. How does complexity minimization compare with concurrent theoretical approaches to scaling laws (e.g., feature-learning regimes, mean-field analysis) in terms of assumptions and tightness?
  5. Can the framework be extended to settings where domains are not i.i.d. sampled but instead arrive sequentially or with distribution shift?
