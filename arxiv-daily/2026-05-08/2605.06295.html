Attributions All the Way Down? The Metagame of Interpretability

["Hubert Baniecki", "Przemyslaw Biecek", "Fabian Fumagalli"]

arXiv: 2605.06295v1

Date: 2026-05-08

Tags: ["interpretability", "feature-attribution", "shapley-values", "second-order-interactions", "transformers"]

https://arxiv.org/abs/2605.06295

一句话总结： 本文提出 METAGAME 框架，将任意一阶特征归因方法本身视为合作博弈，通过对其求 Shapley 值来提取"元归因"（meta-attribution），从而以方向性、可分解的方式量化二阶特征交互，并在大语言模型 token 交互分析、视觉-语言编码器相似度解释及扩散模型概念解释三类任务上验证了有效性。
TL;DR: The paper introduces METAGAME, a framework that wraps any existing first-order attribution method (Shapley values, integrated gradients, AttnLRP, etc.) into a new cooperative game, then applies the Shapley value to that game to produce "meta-attributions" — directional, second-order interaction scores that quantify how much the attribution of feature i changes when feature j is present. Unlike serial methods (Integrated Hessians, serial Shapley) or set-based interaction indices (Shapley-Taylor, SOP), meta-attributions cleanly separate pure individual effects from interaction effects by construction, satisfy a hierarchical decomposition theorem, and are shown to be directional extensions of STII and SOP. Empirically, METAGAME consistently outperforms existing interaction baselines on ImageNet-1k interaction recognition across CLIP, SigLIP-2, and MetaCLIP-2, and substantially improves ConceptAttention on Pascal VOC and MS COCO segmentation for diffusion transformers.

  • 层次分解定理：证明任意一阶归因值可被精确分解为"纯个体效应" φᵢ→ᵢ 与所有交互效应 φⱼ→ᵢ (j≠i) 之和（层次效率，Corollary 1），从根本上解决了串行方法中的"交互泄漏"问题。

Key Contributions

  • Method-agnostic meta-attribution: METAGAME wraps any first-order attribution φᵢ(f, x) as a cooperative game with players = features (or tokens) and game value = the attribution itself, then computes a Shapley value over that metagame to yield directional interaction scores φⱼ→ᵢ(f, x), requiring no architectural changes or second-order gradients.
  • Hierarchical decomposition (Corollary 1): Proved that φᵢ(f, x) = Σⱼ φⱼ→ᵢ(f, x), i.e., any first-order attribution decomposes exactly into a pure individual effect (diagonal) plus all directional interactions (off-diagonal), cleanly avoiding the interaction leakage that plagues serial methods.
  • Unification of existing interaction indices (Theorem 2): Meta-Shapley value is the directional variant of STII; Meta-IG is the directional variant of SOP. This positions METAGAME as a generalizing principle over the existing interaction literature.
  • Scalable computation: Exact computation for d < 20 via vectorized Shapley; for larger games, the framework supports SHAP-IQ Monte Carlo and regression-based approximators from the shapiq library.
  • Empirical gains across modalities: Achieves state-of-the-art interaction recognition on ImageNet-1k (e.g., Attention + METAGAME: .90 at 4 objects vs. .25 for plain Attention on CLIP ViT-B/16), and Pareto-dominates ConceptAttention on both Pascal VOC and MS COCO for FLUX.1 diffusion models.

Methodology

Core formalism. For a first-order attribution method φᵢ(f, x) that quantifies feature i's influence on predictor f at input x, the METAGAME defines a new cooperative game ν: 2^([d]\{i}) → ℝ with characteristic function ν(S) = φᵢ(f(S ∪ {i}; x); x), i.e., the attribution of i when only subset S of remaining features is known. The meta-attribution φⱼ→ᵢ(f, x) is then the Shapley value of player j in this (d-1)-player metagame, computed as:

φⱼ→ᵢ(f, x) = Σ_{S ⊆ [d]\{i,j}} [1 / ((d-1)·C(d-2,|S|))] · [φᵢ(S ∪ {i,j}; f; x) − φᵢ(S ∪ {i}; f; x)]

The pure individual effect is the boundary value φᵢ→ᵢ(f, x) = φᵢ({i}; f; x) — the attribution when no other features are present.

Why this avoids interaction leakage. Serial methods (e.g., applying Shapley to the output of Shapley) jointly vary both xᵢ and xⱼ when computing interactions, entangling individual and joint effects. METAGAME fixes xᵢ and varies only xⱼ in the presence of a coalition, capturing the directional effect of j on i's attribution without contamination.

Instantiations. The framework is instantiated as:

  • Meta-Shapley value — takes φᵢ = Shapley value; equals STII/2 per dimension by Theorem 2.
  • Meta-IG — takes φᵢ = integrated gradients; yields SOP's directional decomposition.
  • Meta-G×I — takes φᵢ = gradient×input; asymmetric by construction.
  • Meta-AttnLRP — takes φᵢ = AttnLRP token attribution; captures token-token synergies in LLMs.
  • Meta-Grad-ECLIP — takes φᵢ = Grad-ECLIP patch attribution; explains cross-modal similarity.
  • Meta-ConceptAttention — takes φᵢ = ConceptAttention in diffusion transformers; reduces context-dependence.

Computational scaling. For small games (d < 20, typical for captions or concept sets), exact Shapley is feasible in a single vectorized pass. For LLM prompts with many tokens, SHAP-IQ Monte Carlo or regression-based approximators are recommended. Crucially, Meta-ConceptAttention requires only a single model forward pass by caching embeddings.

Results & Highlights

Token interactions in LLMs (Gemma 3 via Meta-AttnLRP). Meta-AttnLRP correctly identifies known connotation/negation interactions (e.g., "is → vegan", "honey → butter", "pulmonary → embolism") that AttnLRP's first-order attribution misses. Recall of human-labeled interactions increases steeply with METAGAME compared to plain AttnLRP across both Gemma-3-18-IT and Gemma-3-27B-IT.

Interaction recognition in vision-language encoders (ImageNet-1k). Across all tested models (CLIP ViT-B/16, MetaCLIP-2 ViT-H/14, SigLIP-2 variants) and all base attribution methods (Attention, MaskCLIP, Grad-ECLIP), applying METAGAME consistently and significantly improves interaction recognition rates — especially in multi-object settings (3–4 objects). METAGAME+Attention reaches .90 recall@4objects on CLIP, surpassing even the specialized black-box FIxLIP baseline (.79) which is orders of magnitude more expensive to compute.

Concept interpretation in diffusion transformers (FLUX.1 on Pascal VOC + MS COCO). Meta-ConceptAttention Pareto-dominates ConceptAttention on all metrics (Accuracy, mIoU, mAP) for both single-concept and multi-concept settings. On Pascal VOC multi-concept: .803/.553/.663 vs. .612/.476/.622 for ConceptAttention. Performance is maintained or improves as more distractor concepts are added to context — the opposite of what ConceptAttention exhibits.

Critical Assessment

Novelty: High within the interaction attribution literature. The key insight — treating the attribution map itself as a cooperative game value function — is elegant and unifying. The claim that it generalizes STII and SOP (Theorem 2) is non-trivial and well-proved. The directional decomposition avoiding interaction leakage is a genuine improvement over serial methods.

Rigor: Strong theoretical grounding. All main claims are supported by formal theorems with proofs in the appendix. The illustrative toy example (f(x) = x₁ + x₁x₂²) makes the interaction leakage problem concrete and traceable. However, the "separation problem" is only demonstrated analytically on a 2-feature toy function; more complex cases with higher-order interactions are not analyzed theoretically.

Reproducibility: Good. Open-source code is provided. The shapiq library is leveraged for approximation, a mature and publicly available tool. Hyperparameters and experimental setup are documented in Appendix D. The FLUX.1 experiment relies on a publicly available model (Black Forest Labs). The LLM experiment uses the Gemma 3 model family and acknowledges its illustrative (non-exhaustive) nature.

Significance: Moderate-to-high for the XAI community. METAGAME is a principled and practical meta-wrapper that costs little (one attribution call per subset evaluation) and consistently improves downstream interpretability tasks. The vision-language encoder results are particularly convincing because they use a rigorous interaction recognition benchmark.

Red flags:

  • The LLM token interaction analysis is explicitly described as illustrative ("a few exemplary prompts") — no systematic faithfulness evaluation (e.g., insertion/deletion curves) is provided for the language model setting, which limits how strongly one can conclude about real-world LLM interpretability gains.
  • Computation cost for large token sequences (e.g., full LLM prompts) is a genuine concern; the paper acknowledges this but the approximation quality at scale is not empirically benchmarked in the main text.
  • The framework assumes baseline masking/imputation as the principled removal mechanism, which is the standard but contested choice in XAI; results may vary significantly with other imputation strategies.
  • The asymmetry of meta-attributions (φⱼ→ᵢ ≠ φᵢ→ⱼ for gradient-based methods) is a feature for gradient methods but could complicate interpretation for practitioners who expect symmetry.

---

Strengths

  • Elegant unification: a single framework recovers and extends STII, SOP, Integrated Hessians, and serial Shapley as special cases.
  • Provably avoids interaction leakage that affects all serial methods — a known pain point in the literature.
  • Works with any black-box first-order attribution method, requiring no model modification or gradient access beyond what the base method already needs.
  • Empirical evaluation is broad (3 modalities, multiple architectures, multiple base methods) and uses established benchmarks.
  • Meta-ConceptAttention achieves a practical win: reducing context-dependence without any additional compute cost (single forward pass).

---

Limitations

  • Theoretical analysis is limited to pairwise (second-order) interactions; higher-order extensions are discussed in Appendix C.1 but not experimentally validated.
  • The LLM experiment is primarily qualitative; no faithfulness metric is reported for Meta-AttnLRP on language tasks.
  • Scaling to sequences with d >> 20 tokens requires approximation, and the error bounds of those approximations under METAGAME are not studied.
  • The baseline masking assumption (replacing absent tokens with a [unk] baseline) may not generalize well across domains.
  • Cognitive burden for end-users: a d × d interaction matrix is harder to communicate than a single attribution vector, and this practical usability gap is not addressed.

Questions for Further Exploration

  1. How does METAGAME's performance degrade as the number of tokens grows beyond 50–100 in LLM prompts, and is the regression-based approximator in shapiq reliable enough for production use?
  2. The paper shows Meta-AttnLRP captures directional token interactions — but AttnLRP is already a post-hoc approximation to gradient×input. Does layering METAGAME on a faithfully grounded method (e.g., exact integrated gradients on a small model) yield qualitatively different interaction patterns?
  3. Could METAGAME be used inversely — to identify which input tokens are causally responsible for attribution failures (e.g., hallucinated attributions) rather than correct attributions?
  4. The directional asymmetry (φⱼ→ᵢ ≠ φᵢ→ⱼ) is valuable but could it serve as a proxy for causal direction between concepts in multimodal models?
  5. How sensitive are METAGAME results to the choice of baseline (zero vector vs. mean input vs. Gaussian noise), particularly for vision-language encoders where the baseline determines what "absent" means?
