> **一句话总结：** 本文提出 DAPRO 框架，通过动态分配多轮 LLM 对话的计算预算（审查时间），在满足全局资源约束的前提下，构建出方差更低、覆盖率更接近名义水平的"时间到越狱"下预测界，其关键理论贡献是将覆盖误差间隙从最坏情况权重缩放改为均值权重的平方根缩放，并在 Llama 3.1、Qwen 2.5 等主流模型上实测验证。

> **TL;DR:** DAPRO introduces the first theoretically grounded *dynamic* budget allocation framework for multi-turn LLM safety evaluation. Instead of fixing each conversation's maximum length before any interaction occurs (as in prior static conformal survival methods), DAPRO uses an online policy learned from a training split to adaptively decide, step-by-step, whether to keep probing a given prompt or stop and reallocate budget to other prompts. The result is distribution-free lower predictive bounds (LPBs) on "time-to-unsafe-sampling" that provably satisfy a global computational budget constraint, achieve coverage closer to the nominal level (e.g. 90%), and have lower variance than static baselines—demonstrated across jailbreak, toxicity, agentic task success, and RAG hallucination benchmarks with Llama 3.1 and Qwen 2.5.

## 核心贡献

- **动态预算分配框架（DAPRO）**：首个在多轮 LLM 交互中实现理论有效预算动态分配的框架，将审查时间在对话过程中实时更新，解决了静态方法无法再分配未使用资源的问题。
- **更紧的 PAC 型覆盖保证**：证明覆盖误差间隙与均值审查权重的平方根成比例（而非先前工作中最大权重的平方根），在理论上更紧，且无需条件独立性假设。
- **两阶段自适应策略**：Phase I 在一半校准数据上学习最优"分数→延续概率"映射；Phase II 在剩余数据上自适应执行，实现对话提前终止。
- **无偏人口级评估**：DAPRO 可用于估计不安全事件率等人口层面指标，方差低于静态基线。
- **多任务全面实验验证**：在毒性内容、对抗越狱（PAIR）、指令跟随（AutoIF）、RAG 幻觉等任务上，针对 Llama 3.1、Qwen 2.5、Phi-4、Gemma 3 等主流模型系统验证。

## Key Contributions

- **DAPRO (Dynamic Allocation via PRojected Optimization)**: First adaptive budget allocation framework for multi-turn LLM evaluation that dynamically updates censoring times $C_i$ online based on observed conversational trajectories, as opposed to setting them statically before interactions begin.
- **Tighter Theoretical Coverage Bound**: Proves a PAC-type coverage guarantee whose gap scales with $\sqrt{\bar{w}/N}$ (mean inverse-censoring weight) rather than $\sqrt{\gamma^2/N}$ (worst-case weight), strictly improving prior conformal survival analysis bounds [Davidov et al., AISTATS 2026].
- **Removal of Conditional Independence Assumption**: The coverage proof does not require that the censoring time $C$ and true event time $T$ are conditionally independent given the prompt $X$—a restrictive assumption violated by DAPRO's own early-stopping mechanism.
- **Budget Validity Theorem**: Formally proves that expected budget consumption satisfies the global constraint $B$ in expectation, with a finite-sample bound establishing asymptotic consistency.
- **Population-Level Metric Estimation**: Beyond per-prompt bounds, DAPRO can deliver unbiased, low-variance estimates of aggregate metrics like the jailbreak rate (Appendix F.6).
- **Open-source implementation**: Code available at https://github.com/Shai128/dapro.

## Methodology

The paper frames multi-turn LLM safety evaluation as a **survival analysis under budget constraints**. Given a set of prompts, an attacker model iteratively generates adversarial follow-ups for each prompt, and a judge model labels each response as safe or unsafe. The goal is to construct a calibrated **lower predictive bound (LPB)** $\hat{L}(X_\text{test})$ such that $\mathbb{P}(T_\text{test} \geq \hat{L}(X_\text{test})) \geq 1 - \alpha$ with high probability, where $T_\text{test}$ is the true "time-to-unsafe-sampling" for a new prompt and $\alpha$ is the miscoverage level.

**Static baseline (prior work):** The method of Davidov et al. (2026) sets censoring times $C_i$ for each conversation *before* any interaction occurs, based on prior quantile estimates. This wastes budget when a conversation triggers an unsafe event early (the unused tail budget cannot be redistributed) and leads to high variance in the resulting LPBs.

**DAPRO's two-phase approach:**

1. **Phase I – Learning the optimal acquisition policy** on a first calibration split $\mathcal{I}_\text{cal1}$: For each conversation, interactions run to the prior-estimated event time to observe full histories. A transformer-based predictive model $\mathcal{S}_t(\mathcal{H}_i(t))$ estimates, at each step $t$, the probability of a future unsafe event. An optimal mapping from scores to *continuation probabilities* $P_i(t) \in [0,1]$ is learned by minimizing the average inverse-continuation probability (which upper-bounds LPB variance) subject to the expected budget constraint. This is solved via coordinate descent over $P \in [0,1]^{N_1 \times t_\text{max}}$ with monotonicity constraints.

2. **Phase II – Adaptive acquisition** on the second split $\mathcal{I}_\text{cal2}$: At each step $t$, the learned model $M_t$ maps the current risk score $S_i(t)$ to a continuation probability $P_i(t)$. A Bernoulli draw determines whether to proceed; if not, the conversation terminates and budget is saved for other prompts. Censoring times are set as the last step reached before termination.

**LPB construction:** After obtaining censored survival times $(\tilde{T}_i, C_i)$, the framework calibrates a quantile level $\hat{\tau}$ using weighted empirical coverage (Eq. 3) accounting for the distribution shift induced by censoring. The LPB is $\hat{L}(x) = \hat{q}_{\hat{\tau}}(x)$, where $\hat{q}$ is a pre-trained quantile model.

The key theoretical insight (Theorem 1) is that the coverage gap is bounded by $\sqrt{\bar{w}/|\mathcal{I}_\text{cal}|}$, where $\bar{w}$ is the **mean** inverse-censoring weight. Phase I explicitly minimizes $\bar{w}$, which directly tightens this bound and reduces empirical variance.

## Results & Highlights

- **Coverage accuracy**: DAPRO consistently achieves empirical coverage closer to the nominal target (e.g., 90%) compared to the static optimized baseline across all tested datasets (RealToxicityPrompts, Anthropic Red Team, AutoIF, RAG hallucinations) and all four target models (Llama 3.1 8B, Qwen 2.5 14B, Phi-4 Mini, Gemma 3 4B).
- **Budget compliance**: DAPRO satisfies the global budget constraint $B/|\mathcal{I}_\text{cal}| = 20$ interactions per sample in all experiments, while the static baseline sometimes slightly overshoots or undershoots.
- **Lower variance**: The distribution of coverage deviation (|Coverage - Target|) across 50 random calibration/test splits has noticeably smaller spread for DAPRO than for static baselines (Figures 5 and 6).
- **LPB size**: At a fixed target coverage, DAPRO produces LPBs that are not larger and often smaller than static baselines, indicating it is not purchasing tighter coverage at the cost of looser bounds.
- **Both LPB and UPB**: The same framework works for upper predictive bounds on "time-to-success" for agentic utility evaluation (AutoIF dataset, target coverage 70%).
- **Ablation robustness**: DAPRO's advantage over static baselines holds even when its internal components (score function, Phase I split size $N_1$) are degraded (Appendix F.7).
- **Unbiased aggregate estimation**: DAPRO accurately recovers true jailbreak rates with near-zero bias and lower variance than static baselines (Appendix F.6).

## Critical Assessment

**Novelty**: The dynamic budget allocation idea is genuinely novel in the conformal survival analysis for LLM safety literature. The tighter theoretical bound (mean vs. max weight) is a meaningful theoretical advance, not just incremental. Removing the conditional independence assumption is an important practical contribution since DAPRO's own mechanism violates it.

**Rigor**: The theoretical claims are carefully stated with explicit assumptions (i.i.d. data, bounded mean weight, exchangeability). Theorems 1 and 2 are formally stated; proofs are deferred to appendices. The two-phase split introduces an additional asymptotic regime assumption (Phase I model accuracy), which is acknowledged. The budget validity theorem is somewhat conservative in its finite-sample form, as the authors themselves note.

**Reproducibility**: Code is publicly available (https://github.com/Shai128/dapro). Datasets used (RealToxicityPrompts, Anthropic Red Team, AutoIF) are publicly accessible. Experimental details (train/cal/test splits, 50 random trials, model choices) are described concretely. Reproducibility appears good.

**Significance**: For practitioners evaluating LLM safety under compute constraints, the practical benefit—lower variance coverage estimates from the same budget—is directly useful. The framework's scope extends beyond jailbreaking to any multi-turn event-triggered evaluation.

**Red flags / concerns**:
- The budget validity guarantee relies on the Phase I model being "sufficiently accurate," a condition that is not verifiable in practice without the oracle. The conservative finite-sample bound scales as $O(t_\text{max}^2 / \sqrt{|\mathcal{I}_\text{cal1}|})$, which may not be tight enough for small calibration sets.
- The method requires splitting the calibration set in two (Phase I and Phase II), which reduces the effective calibration size. The LPB's coverage and tightness depend on the calibration set size; this split may hurt performance when $N$ is small.
- The formal guarantees rely on the standard conformal exchangeability assumption: the calibration and test attacker, target, and judge models must be the same distribution. This is violated in practice as models and attack strategies evolve over time—a limitation acknowledged in the discussion but not quantitatively addressed.
- The predictive model (transformer) trained on $\mathcal{D}_\text{train}$ may not transfer well across different attacker-target pairs or domains; no out-of-distribution analysis is provided.

## Strengths

- Clean and principled theoretical framework with formal coverage and budget guarantees.
- Strict improvement over prior static methods both in theory (tighter bound, weaker assumptions) and empirically (lower variance, better coverage accuracy).
- Handles both safety (LPB on time-to-unsafe) and utility (UPB on time-to-success) evaluations within a unified framework.
- Thorough empirical evaluation: four models, four datasets, 50 independent splits, comparison to multiple baselines, ablation studies.
- Practical utility: can provide unbiased jailbreak rate estimates useful for population-level model assessment, reducing expensive LLM inference costs.

## Limitations

- **Exchangeability assumption**: Coverage guarantees break down if the test distribution (attacker strategy, model version) differs from calibration—a realistic scenario in deployed systems.
- **Phase I accuracy dependence**: Budget validity formally relies on the Phase I score-to-probability model being accurate; finite-sample guarantees accounting for model error are highly conservative.
- **Calibration set split overhead**: Dividing calibration data into two phases reduces sample efficiency; performance when $N$ is small (hundreds of prompts) is not thoroughly characterized.
- **Single scalar $\hat{\tau}$**: The calibrated quantile level is global across all test prompts, not prompt-adaptive. Making $\hat{\tau}$ prompt-specific (suggested as future work) could yield tighter bounds for predictable interactions.
- **Training cost**: Phase I requires running all conversations to their full prior-estimated lengths to learn the policy, incurring unlimited training-phase budget; only calibration-phase budget is controlled.
- **No robustness to annotation noise**: Conformal guarantees assume accurate judge/auditor labels; the paper acknowledges but does not address noise in the audit function.

## Relevance

本文对需要在有限算力下可靠评估 LLM 安全性（越狱率、毒性）的研究者高度相关，尤其是使用多轮 adversarial 交互框架（如 PAIR）进行红队测试或 safety benchmark 构建的工作；其动态预算分配思路也可推广至任何多轮 agent 评估场景。

## Questions for Further Exploration

1. How does DAPRO perform when the calibration set is small (N < 500)? Does the Phase I/Phase II split become a bottleneck, and is there an adaptive way to choose the split ratio?
2. The current method uses a single global $\hat{\tau}$ for all test prompts. The paper mentions prompt-adaptive $\hat{\tau}$ as future work—what conditional coverage guarantees would be achievable, and at what sample cost?
3. The budget validity guarantee relies on Phase I model accuracy. Can a data-driven, distribution-free budget safety valve (e.g., using risk control from [24]) be incorporated to eliminate this assumption entirely?
4. How robust is the framework to distribution shift between calibration and test (e.g., new jailbreak strategies emerge after calibration)? Adaptive conformal methods [31] could potentially address this—how would they integrate?
5. The paper shows unbiased jailbreak rate estimation as a byproduct. Could DAPRO be extended into a continuous monitoring system that updates LPBs as new attack interactions are observed in deployment?
