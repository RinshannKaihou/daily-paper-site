> **一句话总结：** 本文提出"源-锚定特征反演"(source-grounded feature inversion)，将特征反演从"对目标表示的预图像搜索"重构为"沿生成该特征的样本局部计算DAG的上游状态估计"，用闭式矩阵 Wiener 修复把 VJP 伴随信号转成上游状态估计，单次标定即可跨新输入/深度/通道复用，无需逐查询优化。

> **TL;DR:** The paper reframes feature inversion from per-query target-preimage optimization into a calibrated, closed-form matrix Wiener repair of VJP/JVP signals along the source sample's local computational DAG; on a matched 16-image ResNet18 target it reaches input-instance pixel cosine 0.939 (vs 0.035 for 2000-step iterative preimage search) at 0.139 s/query (a 176x online-time reduction), with one ResNet50 calibration reused across 1024 images, 4 depths, 4 channels and one 8-channel subset without refitting.

---

## 核心贡献

1. **重新定义特征反演对象。** 论文将查询从仅给出目标张量 `H_T(x)`，扩展为四元组 `(f, x, T, S)`——即把"生成该特征的样本所决定的局部前向激活、非线性工作区、分支状态、注意力路由和局部 Jacobian"作为反演对象的一部分。这把"目标一致"的欠定预图像问题，转成一系列"源-局部上游状态估计"任务。
2. **闭式矩阵 Wiener 修复 (Closed-Form Matrix Wiener Repair)。** 在每个张量边界节点标定一个频率对角的、全通道的零截距矩阵 Wiener 映射 `G_v`，把均种 VJP 观测量映射为上游状态估计；再用第二个 Wiener 映射 `D_v` 修复 JVP 前向一致性残差。所有映射仅由二阶标定统计量与一个标量岭参数决定，无梯度训练。
3. **可复用的层级/通道级解释接口 + 决策接地。** 单次标定的局部映射族支持新输入、深度、通道与通道组的查询；并构造"预测条件化特征图谱"，把可视化特征组与对原始内部表示的独立干预(校准均值/零替换)挂钩，跨残差CNN、归一化重型CNN、Transformer 三族均成立。

## Key Contributions

1. **Problem reformulation.** Source-grounded feature inversion treats the query as `(f, x, T, S)` rather than a target tensor alone, so the inverse is conditioned on the source-local forward point (activations, routing, local Jacobians) instead of being selected by an image prior among an equivalence class of preimages.
2. **Closed-form two-stage DAG repair.** A local matrix Wiener map `G_v(ω) = S_v^{HR}(ω)(S_v^{RR}(ω)+λI)^{-1}` converts a mean-seed VJP into an upstream-state estimate; a second Wiener map `D_v` predicts the residual state error from a pulled-back JVP child-consistency cotangent. Composed through the boundary DAG in one finite reverse pass. No gradient-based training; only second-order statistics and a scalar ridge (`ρ=0.01`).
3. **Reusable interpretation interface + decision grounding.** One calibrated map family supports new inputs, depths, channels, and channel groups without refitting; prediction-conditioned feature atlases link visualised feature groups to independent intervention effects on the model decision.

## Methodology

- **DAG 局部 cotangent 修复。** 把冻结可微模型 `f` 暴露的张量级计算 DAG 记为 `G=(V,E)`，对选定的边界节点 `v` 取非重叠子前沿 `C_v`。局部源是"通道平均子能量"诱导的 VJP：`R_v(x) = Σ_{u∈C_v} J_{u←v}(x)* · H_u(x)/C_u`。标定目标是同节点的前向激活 `H_v(x)`。
- **唯一张量规则。** 沿坐标轴做 k 维 Fourier 变换，在每个坐标频率 ω 上施加一个全 `C×C` 复矩阵。通道可在固定频率上任意混合，而不同频率之间不被修复算子耦合（用于统计可处理性与计算效率，而非断言原网络是平移等变）。
- **闭式标定。** 每节点/频率解岭回归 `G_v*(ω) = S_v^{HR}(ω)(S_v^{RR}(ω)+λI)^{-1}`，岭 `λ_eff = ρ·max{mean_ω,c S_{v,cc}^{RR}(ω), ε_ridge}`，`ρ=0.01`。
- **JVP 前向一致性修正 (JVP-FC)。** 第一阶段估计 `y_v^0 = G_v·R_v` 后，在真实前向点计算 JVP 得到子预测 `Ĵ_{u|v} = J_{u←v}·y_v^0`，残差 `e_{u|v} = H_u - Ĵ_{u|v}`，把它拉回得到第二观测量 `C_v`，再用同形式 Wiener 映射 `D_v` 修复 `H_v - y_v^0`。残差精确分解为基项 `b_{u|v} = H_u - J_{u←v}H_v`（非齐次非线性/偏置项）与 `J_{u←v}·Δ_v`（源-特定误差响应）。
- **拓扑尊重的反演。** 反演时从目标到输入依序复合：终端 seed `s_T^{on} = y_T/C_S`，每个已反演非终端子节点用 `y_u/C_u`，逐节点计算 `R_v^{on} → y_v^0 → C_v^{on} → y_v`。分支、残差加法、拼接、一多复用按 autograd 原生语义处理。
- **理论链。** 从 MSE-最优条件均值出发，在线性高斯参考模型下把条件均值等同为协方差诱导度量下的自然伴随 `J^♯ = Σ_v J* Σ_u^{-1}`；再用图-局部欧氏 VJP 观测量 `R_v` 作为代理源，以闭式分布 Wiener 投影逼近；最后给有限 DAG 复合一个稳定性界 `max_v ‖δ_v‖ ≤ max_{v≠T}‖ε_v‖/(1-q)`。
- **边界模式与架构无关轴。** 支持 Full-DAG、Coarse/stage 块、Custom list 三种边界实例化；通道轴由用户指定或推断，其余轴作坐标轴走 `F_k`。覆盖 CNN 特征、ViT token 流、向量/GAP/MLP、注意力张量四类。

## Results & Highlights

- **可复用规模。** 单次 ResNet50 标定跨 1024 张图、4 个目标深度、4 个单通道与 1 个 8 通道终端子集复用，四个全状态深度的像素余弦保持在 **0.928–0.945**，更换图像/深度/通道掩码均不重拟合。
- **闭式 vs 迭代预图像搜索。** 在相同 16 张 ResNet18 图与终端目标上：迭代预图像优化 (3 个初始化、每图 2000 步 Adam) 达到目标重编码余弦 0.958 但输入实例像素余弦仅 **0.035**；闭式方法输入实例像素余弦达 **0.939**，在线用时 **0.139 s/query** vs 优化法 **24.379 s/query**，**176×** 在线时间缩减。跨混合负载闭式为 0.110–0.141 s/query，优化法为 21.728–23.578 s。
- **跨骨干与张量分量。** 9 个 ImageNet 骨干（ResNet18/50、DenseNet121、MobileNetV2、EfficientNet-B0、RegNetY-1.6GF、ConvNeXt-B、Swin-T、ViT-B/32）均用同一 4096/1024 协议。JVP-FC 相对 `G_v`-only 的像素余弦增益从 ResNet18 的 **0.037** 增到 ViT-B/32 的 **0.391**；Swin-T 像素余弦 0.504 但 SSIM/高频对齐 (0.519/0.614) 强于 ViT-B/32 (0.314/0.248)。
- **拓扑 vs 全局对照。** 单个全局终末端映射 G 像素余弦仅 0.072 (G+D 为 0.067)，原始 VJP 近零 (0.0002)，而局部 `G_v` 达 0.901、最终局部方法 0.938——证明增益来自拓扑尊重的局部复合。
- **算子族消融。** Gamma-only 全架构近零；逐频对角 0.253/0.096/0.062；rank-8 通道修正在 ResNet18 有效 (0.834) 但 ViT-B/32 失效 (0.050)；最终"全通道+逐频"最高 (0.9375/0.8450/0.7021)。
- **目标-算子对照。** ResNet18 上 `H_A with J_B` 像素余弦对 B 为 0.939、对 A 为 0.034；零目标恒产生零输出；通道间平均成对像素余弦 ResNet18 0.497、ViT-B/32 0.230、Swin-T 0.099。
- **决策接地。** 在 16 张 CUB 图 × 4 深度 × k∈{1,4,8,16} 的设置下，校准均值替换协议中 top-positive 通道组的预测 margin 下降在全部 **48/48** 个 model–layer–k 组合中超过 within-image 随机对照组，每个配对 bootstrap 区间都排除零；零替换同样复现 48/48；12 个终端层 top-negative 对照呈现预期反号效应。审计覆盖 36,864 干预行 / 3,072 组反演 / 1,536 条删除曲线。
- **标定需求与部署成本。** ViT-B/32 像素余弦随标定图数变化：512→0.129, 1024→0.363, 2048→0.648, 4096→0.702, 8192→0.724；ResNet18 在 c512 全 DAG 标定把像素余弦从 0.929 提到 0.963 但拟合节点数从 9 增到 49。在线应用耗时从 ResNet18 的 58.4 ms/image 到 ConvNeXt-B 的 369.7 ms/image；ConvNeXt-B 映射存储达 15.51 GiB。

## Critical Assessment

- **方法学严谨度高。** 把特征反演重构成"沿源-局部 DAG 的状态估计"是有意义的视角转变：传统迭代预图像搜索用图像先验在等价类里选成员，而本文把"生成该特征的样本局部计算几何"作为反演对象的一部分，并由零截距映射保证不存在与目标无关的可加模板（零目标→零输出，逐通道验证）。这一性质通过零目标、shuffle、checkpoint mismatch 等多重组照得到支撑。
- **对照设计强。** Table 4 的拓扑/阶段/算子三联消融很有说服力：全局单映射仅 0.072 vs 局部 DAG 0.938，直接定位增益来源；算子族消融显示"全通道+逐频"是必要的（gamma-only 近零，rank-8 在 ViT 失效），把"为什么需要全秩逐频矩阵"这件事讲清楚。
- **诚实区分信息集。** 论文明确不把"闭式 0.939 vs 迭代 0.035"当作严格的估计器比较：迭代法从噪声搜索目标一致预图像，本文方法额外使用源-局部几何，两者反演的是不同的 inverse object。这种区分避免了夸大。
- **真正薄弱处。** (1) 结果几乎完全依赖像素余弦等"输入实例对齐诊断"作为代理，而论文反复强调这"不是正确性度量"。缺少一个真正独立于输入的"上游状态恢复"金标准（虽有 projected local-oracle 实验 C.6，但作者自己说它不能给出稳定全局误差排序）。(2) Transformer 上 ViT-B/32 仅 0.702、Swin-T 仅 0.504，且高度依赖大标定集（ViT-B/32 从 512 到 8192 才到 0.724），说明频率对角 + 全通道映射对 token 架构可能不是最优算子族——作者也承认"高效样本特定度量/状态估计"是下一方向。(3) 单作者 + 未提及正式同行评审；理论体量大但部分关键桥梁（如 `R_v` 作为 `H_v` 的代理源的可预测性）是经验建模假设而非代数恒等式。
- **决策接地实验的价值与边界。** 48/48 的方向一致性是强证据，但作者也声明"特征反演图本身不被当作因果刺激重新输入"——因果证据来自对原始表示的干预。这一边界划得清楚，是优点。

## Strengths

- 把特征反演从"逐查询优化"变成"标定一次、复用多次"的部署形态，176× 在线加速与无重拟合的通道/深度查询是实打实的工程价值。
- 双阶段闭式 Wiener 修复有清晰的理论动机（条件均值 ↔ 自然伴随 ↔ 分布 Wiener 投影），且 JVP-FC 针对样本依赖算子（归一化、注意力、路由）给出可解释的残差修正。
- 架构无关的张量轴规则（一条规则覆盖 CNN/ViT/向量/注意力四类张量），12 个正式 component run 全部通过验证器无需手工逆规则。
- 控制实验充分且预声明（fixed sample indices、predeclared depths/groups），区分诊断指标与正确性度量，避免过度声明。

## Limitations

- **Transformer 上效果有限。** ViT-B/32 (0.702) 与 Swin-T (0.504) 像素余弦偏低；ViT-B/32 的 online-child oracle gap 大；标定需求随架构剧烈变化（ViT 需 4096+ 才稳定）。
- **频率对角限制。** 修复算子不允许跨坐标频率混合，论文承认这是为可处理性引入的建模决定，其充分性是经验问题而非保证。
- **Checkpoint 特异性。** 映射编码 checkpoint 特定局部几何，换 checkpoint 需重标定。
- **存储与内存。** ConvNeXt-B 映射族 15.51 GiB、评估峰值 16.78 GiB，部署成本不低。
- **证据范围。** 当前实验限于视觉 CNN/Transformer；内部注意力张量、更广架构（含语言模型）未验证。
- **缺乏独立正确性金标准。** 主要指标（像素余弦、SSIM、LPIPS）是输入对齐诊断，作者自己反复强调它们不定义反演正确性。

## Relevance

- **对可解释性/机制可解释性研究有直接意义。** 提供了一种"在任意层/通道上、以样本为条件、可视化模型提取了什么输入结构"的可复用接口，可与因果干预、概念方向、网络解剖等方法互补。
- **对 dagger (DAG) 视角的反演/逆向工程有理论启发。** 把"VJP 是伴随传输而非上游状态估计"这一观察，通过 Wiener/自然伴随的桥梁转化为可部署的闭式修复，思路可迁移到其他需要"把 cotangent 修成 state"的场景。
- **对大模型可视化潜在应用。** 若频率对角限制能在 token/语言模型上被验证或放宽，可为 LLM 的内部表示提供输入域可视化工具（作者明确将其列为重要验证域）。
- **工程价值。** 标定一次、复用多次的形态适合需要反复查询同一模型多层多通道的解释/调试 pipeline。

## Questions for Further Exploration

1. 频率对角 + 全通道的算子族是否是 Transformer token 流的最优选择？引入轻量跨频耦合（如分块/低秩跨频）能否显著提升 ViT-B/32 的 online-child oracle gap？
2. JVP-FC 的残差修复在多大程度上等价于"在线估计样本特定度量 `M_v, M_u`"？能否用一个小的在线头直接预测度量，替代固定分布投影？
3. 零截距映射保证"零目标→零输出"，但谱乘法可在坐标空间扩散能量。能否给出一个支持非扩张性的可验证界，或在反演图上做稀疏性验证？
4. 决策接地实验用 calibration-mean/zero 替换作为干预；若改用更精细的因果干预（如激活路径编辑、meddling），48/48 的方向一致性是否仍然成立？
5. checkpoint 特异性要求重标定。能否构造"checkpoint-不变的局部几何特征"，使映射在小幅微调后仍可迁移？
6. 方法对内部注意力张量（softmax simplex 约束）是否需要专门的非线性逆规则，还是当前线性松弛已足够？论文把它列为开放问题。
7. 在语言模型上，token 流的"坐标轴"是序列位置——此时频率对角的语义是什么？是否对应某种句法/语义的频段分解？
