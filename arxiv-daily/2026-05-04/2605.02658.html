Deciphering Shortcut Learning from an Evolutionary Game Theory Perspective

arXiv: 2605.02658v2

Date: 2026-05-04

https://arxiv.org/abs/2605.02658

一句话总结： 本文从演化博弈论视角建模深度学习中捷径特征的形成机制，将数据样本视为博弈参与者、神经正切特征视为策略，严格证明全批量梯度下降收敛到捷径策略而随机梯度下降收敛到核心特征策略，并通过随机微分方程揭示了数据噪声加剧捷径偏差、优化噪声抑制捷径偏差的双重效应。
TL;DR: This paper formally defines core vs. shortcut features and models shortcut bias formation as an evolutionary game among data samples, proving that full-batch GD converges to shortcut-dominated stochastically stable states while mini-batch SGD converges to core-feature states. A stochastic differential equation analysis further shows that data noise amplifies shortcut bias whereas optimization stochasticity mitigates it, providing theoretical grounding for why mini-batch training learns more robust features.

Key Contributions

  • Provides the first rigorous formal definition distinguishing core features from shortcut features based on mutual information with labels
  • Models neural network training as an evolutionary game where data samples are players and neural tangent feature directions are strategies, assuming the existence of separate core and shortcut subnetworks within the network
  • Proves that full-batch GD leads to shortcut-dominated stochastically stable states (Theorem 11), while mini-batch SGD leads to core-feature stochastically stable states (Theorem 12), providing a game-theoretic explanation for why SGD generalizes better
  • Derives a stochastic differential equation for training intensity dynamics and proves that data noise amplifies shortcut bias while optimization noise suppresses it (Theorem 13)
  • Uses Davis-Kahan theorem and random matrix theory to show that shortcut features dominate early training due to higher variance in the input data
  • Validates the subnetwork hypothesis on Colored-MNIST: L1-regularized probing reveals that distinct neuron groups specialize for core (digit) vs. shortcut (color) features

Methodology

The paper develops a multi-layered theoretical framework:

  1. Formal definitions (Section 2): Core features are defined as features $V^{\text{core}}$ satisfying $I(V^{\text{core}}; Y) = 1$ (complete label information), while shortcut features $V^{sc}$ satisfy $I(V^{sc}; Y) < 1$. A partial order $\ll$ ranks features by shortcut severity based on intra-class vs. inter-class prediction consistency.
  1. Initial bias analysis (Section 3.1): Using the neural tangent kernel (NTK) framework, the authors decompose the kernel matrix into a signal component (core features) and a noise/shortcut component. Applying the Davis-Kahan sin-Θ theorem, they bound the perturbation of top eigenvectors, showing shortcut features dominate early representations when they explain more data variance. A random matrix theory analysis (Wigner semicircle law) further characterizes signal recovery conditions.
  1. Evolutionary game model (Section 3.2): The network is partitioned into three edge sets — $E_1$ (core subnetwork), $E_2$ (shortcut subnetwork), and $E_3$ (interference). Data samples are classified as core-strategy or shortcut-strategy players based on their neural tangent feature alignment. A payoff matrix is derived from the gradient descent dynamics, and the evolution of the proportion of core-strategy samples ($z_t$) follows Darwinian dynamics with mutation. Under full-batch GD, the transition matrix analysis (via z-tree mutation energy counting, following Kandori-Math-Young techniques) proves the shortcut state $z=0$ is stochastically stable. Under mini-batch SGD, the batch-wise payoff computation using multivariate hypergeometric distributions reverses the stability: $z=N$ (all-core) becomes the stochastically stable state when the core payoff advantage exceeds a threshold.
  1. Continuous-time SDE model (Section 3.2.4): The training intensity difference $w_2 - w_1$ between subnetworks is modeled via a coupled stochastic system with reflection boundaries. The data noise parameter $\tau$ causes exponential decay of effective learning signals, while the optimization noise $\sigma$ enters through Brownian motion. The Fokker-Planck stationary distribution analysis proves $\tau$ monotonically increases shortcut bias while $\sigma$ decreases it.
  1. Experimental validation (Section 4): Colored-MNIST experiments confirm (a) initial variance bias toward color, (b) full-batch vs. mini-batch divergence in neural tangent projections, (c) subnetwork specialization via L1 probing, and (d) SDE simulation matching theoretical predictions for varying $\tau$ and $\sigma$.

Results & Highlights

  • On Colored-MNIST (90% color-label correlation), color features explain 25.43% of total variance vs. 16.61% for digit features, confirming the initial variance bias
  • Full-batch GD achieves 97.10% test accuracy while mini-batch SGD achieves 98.06%, with qualitatively different neural tangent feature representations: full-batch preserves color separation, mini-batch preserves digit separation
  • L1-regularized probing of a 32-neuron hidden layer reveals only 4 active neurons: neurons #1, #10, #27 for core features and neuron #14 for shortcut features, validating the subnetwork hypothesis
  • SDE simulations with $w(0) = (0.02, 0.4)$ show that low data noise ($\tau=0.3$) allows the core subnetwork to eventually overtake the shortcut subnetwork, while high data noise ($\tau=0.8$) locks in shortcut dominance
  • Increasing optimization noise $\sigma$ from 0.1 to 0.4 shifts the strategy equilibrium from shortcut-dominated ($\alpha \to 0$) to core-dominated ($\alpha \to 1$)

Critical Assessment

Novelty (High): The evolutionary game-theoretic framing of shortcut learning is genuinely novel. While prior work has studied shortcut learning empirically or in linear models, this paper provides the first formal game-theoretic treatment with rigorous convergence proofs distinguishing GD from SGD outcomes. The formal definition of core vs. shortcut features via mutual information is a useful contribution.

Rigor (High): The proofs are technically sound, leveraging Kandori-Math-Young stochastic stability techniques, Davis-Kahan perturbation bounds, and Fokker-Planck analysis. The three main theorems are non-trivial and correctly established. However, the subnetwork assumption (that distinct neuron groups independently model different features) is a strong simplification, and the assumption that $w_1, w_2$ remain approximately constant within an epoch is an approximation whose impact is not fully quantified.

Reproducibility (Moderate): The Colored-MNIST experiments are straightforward to replicate, and the SDE simulations use standard numerical methods. However, several hyperparameter choices (e.g., 32-neuron hidden layer, specific noise levels, the choice of $\gamma$ values) are not exhaustively explored. The theoretical results depend on specific conditions (e.g., $B$ divisible by $N$, $w_1 + w_2 \leq 1 - \beta$) whose practical tightness is unclear.

Significance (Moderate-High): The theoretical insight that SGD's stochasticity is mechanistically responsible for mitigating shortcut bias — not just a practical convenience — is important. The identification of data noise vs. optimization noise as opposing forces on shortcut bias is actionable. However, the gap between the simplified two-feature, binary-classification setting and real-world deep learning is substantial. Extension to modern architectures (transformers), multi-class settings, and high-dimensional features is acknowledged as future work.

Red flags: The experimental evaluation is limited to Colored-MNIST, a relatively simple benchmark. No experiments on natural image datasets (CIFAR, ImageNet) or NLP tasks are provided. The evolutionary game model requires knowing or assuming which features are "core" vs. "shortcut," which limits its direct applicability to settings where shortcuts are unknown — a point the authors acknowledge but do not fully resolve.

Strengths

  • Provides the first rigorous game-theoretic framework for understanding shortcut bias dynamics during training
  • The distinction between GD and SGD as converging to different stochastically stable states is a clean, impactful theoretical result
  • The SDE model elegantly separates the roles of data noise and optimization noise
  • Complete mathematical proofs in the appendix make the results verifiable
  • The subnetwork hypothesis validation via L1 probing is a clever experimental design
  • The discussion section provides principled connections to existing mitigation techniques (data augmentation, disentangled representations, ETF-Debias)

Limitations

  • Experimental validation is restricted to Colored-MNIST; no evaluation on complex datasets or modern architectures
  • The two-feature, binary-classification setting is a strong simplification of real-world scenarios
  • The subnetwork assumption (orthogonal feature subnetworks) may not hold in overparameterized networks where features are distributed across neurons
  • The interaction coefficient $\gamma$ between subnetworks is treated as given rather than derived from network architecture
  • The framework is limited to supervised learning; extension to self-supervised, contrastive, or RL settings is unexplored
  • The paper does not propose new debiasing algorithms, only theoretical insights

Relevance

本文从演化博弈论角度严格分析了深度网络中捷径偏差的形成动力学，与用户关注的"ML可靠性、鲁棒性与数学基础"方向高度相关。特别是其关于优化噪声如何帮助模型学习鲁棒特征的理论分析，为理解训练稳定性和泛化可靠性提供了新的数学视角。

Questions for Further Exploration

  • Can the evolutionary game framework be extended to transformer architectures where attention heads may serve as feature subnetworks?
  • How does the interaction coefficient $\gamma$ relate to architectural choices (width, depth, residual connections)?
  • Does the theoretical prediction that optimization noise suppresses shortcut bias hold for adaptive optimizers (Adam, AdaGrad) where gradient noise is modulated?
  • Can the framework explain why some shortcut features are harder to unlearn than others (the "irreversibility" issue mentioned in Section 3.2.4)?
  • Is there a principled way to estimate the optimal batch size $B$ for shortcut bias mitigation in a given dataset without trial-and-error?
