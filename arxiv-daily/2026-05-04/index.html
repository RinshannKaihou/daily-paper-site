Must Read · Reasoning & Evaluation

Measuring AI Reasoning: A Guide for Researchers

Nwadike, Iklassov, Ali, Genadi, Inui  ·  MBZUAI  ·  2605.02442v1

本文论证仅依赖最终答案准确率无法衡量推理能力，提出以推理链的忠实性（faithfulness）与有效性（validity）为核心的分层证据等级框架（Level 0–2），并给出可移植量化指标 SVR、VSR、IFR，为推理基准泛滥的研究生态提供了方法论锚点。

A principled position paper arguing that outcome-only accuracy is insufficient for evaluating reasoning. Proposes an evidence-tier framework (Levels 0–2) with portable metrics (SVR, VSR, IFR) for process-based evaluation — a much-needed methodological anchor amid proliferating reasoning benchmarks.

Rahul Kumar  ·  Independent  ·  2605.02398v1

首次通过6条件因子实验证明合规指令（而非威胁内容）导致8/11前沿模型元认知崩溃（最多下降30.2pp），Anthropic宪法AI近乎免疫。

Factorial design isolates compliance-forcing instructions—not threat content—as the trigger for metacognitive collapse in 8/11 frontier models; Constitutional AI shows near-perfect immunity.

Must Read · Theory

Deciphering Shortcut Learning from an Evolutionary Game Theory Perspective

Li, Gai, Zhang  ·  AMSS / SIMIS  ·  2605.02658v2

严格证明全批量GD收敛到捷径策略、SGD收敛到核心特征策略；SDE模型揭示数据噪声加剧捷径、优化噪声抑制捷径的双重效应。

Proves via evolutionary game theory that full-batch GD converges to shortcut states while mini-batch SGD converges to core-feature states; SDE analysis reveals data noise amplifies shortcut bias while optimization noise suppresses it.
