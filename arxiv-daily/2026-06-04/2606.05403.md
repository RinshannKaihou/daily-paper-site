> **一句话总结：** LLM 在多源量化合成中具备孤立检测伪造统计的能力（CIR 0.76–1.00），却由 **methodology-register gate** 按「分析文体」而非数值有效性加权来源；不可能 CI 与有效 CI 的 SPI 几乎相同（伪造数字恢复约 **79%** 有效数字的拉力），线性探针与 ROME 因果追踪一致，且提示甚至 oracle checklist 只能产生 blanket skepticism 而非选择性甄别。

> **TL;DR:** Amazon authors test 5 models (Claude Haiku/Sonnet/Opus 4.5, Qwen 32B, OLMo 3.1 Think) synthesizing conflicting estimates from 4-source workplace threads across VC/MKT/PH (>1M trials). Isolation: CIR high for specious methodology; numerics follow domain gradient (VC CI near-universal). Synthesis: impossible stats ≈ valid stats (SPI); specious jargon ≈ bare claim. Mechanism: cross-domain methodology probes AUC 0.83–0.92 (L8), numerics at chance; isolation separability 0.87→0.52 in synthesis; ROME/DLA show consensus-gated presentation tokens, no numerics correction (r=0.78). Prompting fails selective discernment. Term **epistemic alignment**: deploy evaluation capability, not just possess it—distinct from user sycophancy.

## 核心贡献

1. **因子实验揭示「检测 vs 部署」行为解离**：五模型 × 三专业域（VC 留存率、MKT ROAS、PH 患病率）× 四源群聊线程 × **6 级呈现**（裸断言 / 合理方法±统计 / 不可能统计 / 伪方法±统计）× 社会共识（0–3 方同意焦点源）；**>100 万** trial。指标 **SPI**（Source Preference Index，改编自 weight-of-advice）：模型估计相对共识向焦点源偏移的程度（0=跟共识，1=跟焦点源）。

2. **方法论文体门（methodology-register gate）**：合理方法论+统计使孤立焦点源从最弱变为主导；**不可能统计与有效统计 SPI 几乎相同**；无统计的合理方法已有中等影响，加上不可能统计可恢复有效统计约 **79%** 的增量拉力——模型把数字精度当可信度信号，不交叉验证与方法描述的一致性（超越人类「不可能精度会打破启发式」的发现）。

3. **孤立有能力、合成中不调用**：统计审查 prompt 下 CIR 对伪方法 ≥0.76；VC 不可能 CI 近 100% 识别，PH 虚高相关多数可识别，MKT 置换下限违规几乎无人识别。放入四源上下文后，CoT 中质疑率骤降（如 Opus VC 仅 15.4% Quest.），且 **P(Endorse|Quest.)** 仍高（0.48–0.96）——推理轨迹合理化而非修正。

4. **表征与因果机制收敛**：差分均值线性探针（Qwen 32B、OLMo 3.1 32B Think）：L8 跨域 **methodology AUC 0.83/0.92**，**numerics ~chance**；推理后位置 D 方法论信号衰减（0.83→0.64，0.92→0.71），numerics 仍 chance。孤立时跨域 numerics separability **0.87/0.88**，合成时 **~0.52**。ROME 因果追踪：呈现 token 早期高因果重要性，L48–55 handoff；合理 vs 不可能统计轨迹近重合；**共识作增益控制**——孤立时呈现 token 与 ΔSPI 均大，多数同意时双 collapse。DLA：方法论 boost/correction 随共识消失，numerics correction 平坦；MLP 贡献为 attention **4–8×**；r=0.78, ρ=0.81 预测行为。

5. **提示无法选择性修复 + 提出 epistemic alignment**：generic / statistical / **oracle checklist**（点名 CI、相关上限、p 值下限等）均无法「只降伪造、保留有效」——统计与 oracle prompt 对有效与伪造统计同步降权（blanket skepticism）。与跟踪用户偏好的 sycophancy 不同，失败跟踪**源是否呈现为分析可信**。RLVR 强化方法判别但不建立数值验证。作者用 **epistemic alignment** 命名：像 preference/safety alignment 一样，问题在**部署**而非能力。

## Key Contributions

1. **Factorial behavioral dissociation** (>1M trials): models detect fabricated/methodologically specious claims in isolation (CIR 0.76–1.00 for methodology) but weight sources by analytical register during multi-source numeric synthesis.

2. **Source Preference Index (SPI)** quantifies consensus→focal pull; impossible statistics recover ~79% of valid-statistics incremental influence when methodology is held plausible.

3. **Reasoning-trace analysis**: infrequent in-context questioning; high endorsement given questioning—detection without revision under methodological (not user) pressure.

4. **Mechanistic triangulation**: cross-domain methodology linear probes (AUC 0.83–0.92) vs chance numerics probes; ROME causal tracing + DLA component attribution showing consensus-gated presentation processing and absent numerics correction.

5. **Prompting ceiling failure** and **epistemic alignment** framing: even oracle statistical checklists yield blanket skepticism; post-training may reinforce stylistic rigor without numeric verification.

## Methodology

### 问题背景

LLM  increasingly serve as **epistemic intermediaries**（证据综合、分析起草、医疗/金融决策）。遇多源冲突时需**评估**而非仅聚合。既有工作聚焦 **sycophancy**（迎合用户偏好）；本文研究**源级评估**：方法是否可信、统计是否内部一致——对应 Sperber et al. 的 **epistemic vigilance**（来源 vs 内容）。人类易被「严谨外观」启发式误导（无意义公式、无关神经术语）；RLHF/DPO 可能继承「奖励统计修辞而非实质」。

### 实验范式（§2）

- **场景**： workplace 群聊，四角色——高级权威、外部机构、内部分析师（**焦点源**）、第三方参考；各报领域指标估计，焦点与 0–3 其他源形成 **inter-context knowledge conflict**。
- **三域**： VC（Series C 尽调留存率）、MKT（广告 ROAS 扩量）、PH（学校停课指引下的患病率）。
- **六档呈现**（仅变焦点源消息）：(1) bare claim；(2–3) 合理方法 + 有效统计 / 无统计；(4) 合理方法 + **结构不可能统计**（如 n=2400 时 95% CI 仅 0.06pp）；(5–6) **伪方法**（域外术语）± 框架一致诊断。
- **共识**：每 trial 0–3 其他源与焦点同立场；排除全员同意焦点及权威中立配置。
- **模型**： Claude Haiku/Sonnet/Opus 4.5；Qwen 32B；OLMo 3.1 Think（Appendix C.8 服务细节）。
- **辅助任务**：孤立统计审查（CIR）；无估计的「最可信/最不可信源」选择；CoT 的 LLM judge（随机化 source key）；三种 system prompt 缓解。

## Results & Highlights

### 行为：合成中的源影响（§3, Figure 2）

| 呈现 | 孤立 dissent 时 SPI 模式 |
|------|--------------------------|
| 合理方法 + 有效/不可能统计 | 两者均强，不可能 ≈ 有效 |
| 合理方法无统计 | 中等 |
| 伪方法 / 裸断言 | 可忽略 |

- **Fabrication as enhancement**：合理方法无统计 → 加不可能统计 ≈ 加有效统计（~79% 恢复）。
- **可信度选择**（Table 1, nagree=0）：specious numerics 下焦点被选「最可信」**76–100%**，「最不可信」仅 **0–10%**——非单纯数字锚定。

### 孤立检测 vs 合成质疑（Table 2–3）

| 现象 | 要点 |
|------|------|
| 孤立 CIR | 伪方法普遍 ≥0.76；VC CI ~1.0；MKT p-value floor ~0；PH r 多数高 |
| 合成 % Quest. | 低（如 MKT 0.6–5.4%） |
| P(End\|Q) | 质疑后仍常背书（0.32–0.96） |

### 提示缓解失败（Table 4）

- Generic：几乎无 ∆SPI。
- Statistical / Oracle：部分降低 impossible SPI，但**同步降低 valid SPI**——无选择性甄别；Opus oracle 对 impossible −0.39 同时对 valid −0.18。

### 表征解离（§4, Figure 3, Table 5）

| 信号 | 孤立 | 多源合成 |
|------|------|----------|
| Methodology 跨域 AUC (L8) | 0.83–0.92 | 推理后衰减仍 >chance |
| Numerics 跨域 | separability 0.87–0.88 | **~0.52 (chance)** |
| 六轴分解 | 编码「有无数字」 | 不编码「数字是否有效」 |

### 因果与电路（§5, Figure 4–5）

- 合理 vs 不可能统计 **restore 曲线近重合**。
- 共识门控：孤立 red 点高因果+高 ΔSPI；多数 gray 双近零。
- DLA：方法论 effect/correction 随 nagree 坍缩；numerics correction **平坦**；少数 head（L63.H21, L63.H19, L60.H16）跨域驱动伪方法「校正」（实为弱激活同一路径，非选择性检测）。

### 讨论要点（§6）

- 映射人类双过程：方法门 ≈ System 1 模式匹配；数值验证 ≈ System 2，孤立可做、合成不自发调用。
- 训练解释：分析术语共现是出版文本可靠代理，但出版统计几乎总一致 → **无数值无效监督**；RLVR 锐化方法判别、不创造数值验证。
- **对抗含义**：孤立无共识时门全开——对手只需用可信分析文体包装伪造主张。
- 局限：固定四源结构；机制主 Qwen+部分 OLMo；未给出有效缓解方案。

## Critical Assessment

**新颖性**：将「严谨外观」偏见从人类心理学系统迁入 LLM **多源数值合成**；提出与 sycophancy 正交的 **epistemic alignment**；行为-表征-因果三层 dissociation 叙事完整。

**严谨性**：
- **优点**：因子设计、SPI 清晰、孤立/合成/无估计/CoT/提示多棱验证；跨族跨域；机制不只报 probe 还做因果与 DLA。
- **弱点**：四源群聊生态效度有限；MKT 数值缺陷检测弱可能低估部分能力；机制分析 prefilling 绕过完整 CoT，扩展推理轨迹机制仍 open；缓解仅诊断未解决。

**工程意义**：RAG/多文档摘要/企业分析 agent 若综合带统计的「专业」来源，需**显式数值验证管线**（工具或后处理），不能假设 CoT 会部署已有检测能力；评测应测 **合成态** 而非仅孤立 critique。

## Strengths

- Figure 1 三联图直观展示 Trust-but-don't-verify 核心悖论。
- ~79% fabrication recovery 量化「精度启发式」失败，政策含义强。
- 共识门控发现：最需独立判断时（孤立 dissent）最易被方法文体说服——与直觉相反，对抗面明确。
- 探针跨域 + 因果 tracing + DLA 故事一致，非单点相关性。
- 明确区分 capability vs deployment，对接 alignment 话语。

## Limitations

- **场景**：合成 workplace 四源线程，未覆盖开放网页、长文档、非数值冲突。
- **模型**：未含 GPT/Gemini 等；版本绑定 2025–2026 Claude/Qwen/OLMo。
- **MKT 难度**：置换下限违规孤立 CIR≈0，可能使「能力存在」论断在该域偏弱。
- **机制范围**：完整电路分析仅 Qwen（+ OLMo 部分），长 CoT 内检测→权重传播未完全刻画。
- **缓解**：oracle prompt 仍失败，无训练或架构方案实证。
- **指标**：SPI 跨域单位不同，仅比较方向/排序；全批判导致传统分类指标不可用（作者用 CIR 合理但增加解读成本）。

## Questions for Further Exploration

1. **工具增强合成**：要求模型调用计算器/区间宽度检验，能否在保持 valid SPI 同时压低 impossible SPI？
2. **训练数据**：在 SFT/RL 中加入「前提有时无效需验证」的合成样本，能否让 numerics probe 在合成时 >0.7 AUC？
3. **闭源与推理模型**：o1/DeepSeek-R1 长推理是否仍出现高 P(Endorse|Quest.)？
4. **开放多源**：Slack/邮件以外（论文库、新闻流）是否仍由 methodology-register 主导？
5. **与 sycophancy 交互**：用户施压 + 孤立 dissent 伪造统计，何者主导 SPI？
6. **电路干预**：抑制 L63.H21 等 head 或增强 numerics 方向，能否选择性降权 impossible CI？
7. **Epistemic alignment 基准**：是否可定义类似 ERS 的「合成态源评估鲁棒性」标准数据集？
