Epidemiology of Model Collapse: Bilayer SIR Dynamics for Synthetic Data Contamination

KEY FINDING: Cross-model synthetic contamination is a coupled epidemic on data corpora and AI models—all calibrated scenarios are supercritical (R0>1); detection rate γD is the highest-leverage intervention, not ecosystem diversity.

Key metric
R₀ = 2.62 baseline · optimistic 1.10 · pessimistic 6.63 · P(R₀>1)=98.2%
Why this matters

First systems-level vocabulary for ecosystem-wide cross-model contamination—actionable priority: filter/detect synthetic text and cut mixture α.

Critical edge

Illustrative calibration only; GPT-2 124M; source-diversity effect borderline (p=0.047) and null at α=0.5.

Headline

WikiText PPL 33.5→126.9 at α=1; Distinct-2 0.68→0.38; γD Sobol ST=0.324 tops all parameters.

Bilayer epidemic framework

Data layer SD clean · ID contaminated · RD filtered · ΛD, γD, μD
βD (model→data) ↑↓ βM (data→model)
Model layer SM clean-trained · IM contaminated · RM retrained · ΛM, γM, μM
R₀ = √(βDβM / [(γD+μD)(γM+μM)])  ·  SIRS waning δD,δM → same R₀, damped oscillations

Data → model (βM)

  • Contaminated corpora train downstream models
  • Mixture fraction α operationalizes dose
  • Turnover μD ≈ 0.02 (corpus refresh)

Model → data (βD)

  • Model outputs re-enter web / crawls
  • ~74% AI-modified newly indexed pages (cited)
  • βD ≈ 0.217 from 2023–2025 prevalence trend

Validation stack

  • NGM + Sotomayor proofs (200 configs)
  • Bipartite ABM: R²>0.96 dense, drops sparse
  • 1,280 GPT-2 fine-tunes as qualitative bridge
