CPT Scaling laws LR · Batch size Cpre Dense-8B MoE-3B 55B tokens

CPT 最优超参可预测缩放律 — 用等效预训练算力零样本调参

核心发现：Continued pre-training 中，最优 学习率随算力单调下降、最优 batch size 单调上升（与从零训练不同）。两阶段框架：小模型代理网格拟合律 → 用 checkpoint 在目标域的 val loss 反推 Cpre，预测 Dense-8B / MoE-3B 的 LR 与 B，较目标模型 9 组网格搜索节省 73–92% 搜索 FLOPs。

Scaling law

Iso-loss 曲线顶点 → 幂律 Bopt(L)、LRopt(L) + Kaplan 式 L(C)，CPT 专用可外推。

代理网格：B ∈ {64, 128, 256}，LR ∈ {10⁻⁴, 2×10⁻⁴, 5×10⁻⁴}（9 配置 × 4 架构）。训练后期更大 B 降方差、更小 LR 抑振荡 — 解释收敛阶段「大 batch + 小 LR」。

L(C) = L0 + α·C−γ
Bopt(C) = f(L(C))   ·   LRopt(C) = g(L(C))
C = σ·D·N FLOPs · 与 scratch Kaplan 律区分：CPT 从任意 checkpoint + 域偏移出发

两阶段可部署框架

Stage 1 · 离线 → 代理网格 → Iso-loss 顶点 → 拟合 f, g, L(C) ⇒ Stage 2 · 在线 → Linit → Cpre → 预测 B, LR
Stage 1 — Empirical Law Discovery

小模型一次付费，律可复用

  1. 100M/500M 稠密 + MoE 在 55B 混合语料上 CPT 网格
  2. 固定 L̂ 画 iso-loss：达到 L̂ 的最小 token 轨迹
  3. 顶点 → (Bopt, LRopt) 随 loss/compute 的幂律
  4. 输出 f, g 与 L(C) 系数（架构相关，Table 7）
Stage 2 — State-Aware Prediction

目标模型零试探训练

  1. 在目标 CPT 数据上评估 Linit（单次 forward/val）
  2. Cpre = (α / (Linit − L0))1/γ
  3. Ctotal = Cpre + Ccpt（计划 CPT FLOPs）
  4. Bopt = f(Ltarget), LRopt = g(Ltarget)

Stage 1 成本

  • 4 组代理 × 9 格点完整 CPT — 一次性，与目标规模解耦
  • 搜索 FLOPs 8.15×10²⁰（共享代理实验）

Stage 2 收益

  • Dense-8B / MoE-3B 无需 9× 目标网格
  • 模型越大，相对目标网格优势越明显

等效预训练算力 Cpre

将 checkpoint 在目标域的验证损失映射为「若在目标数据上从零训练达到同等损失所需算力」，量化域偏移与非零初始化下的优化状态。

Cpre = ( α / (Linit − L0) )1/γ
Ltarget = L0 + α · Ctotal−γ   where   Ctotal = Cpre + Ccpt

架构感知：Dense vs MoE 系数

回归 y = a·x + b，x ∈ {ln(LR), B}，y = ln(FLOPs)。MoE 对 LR 更不敏感（可维持更大步长），对 batch 更「贪婪」以稳定路由。

评估与局限

Strengths

  • 首次系统 CPT 超参缩放律 + Cpre 状态量化
  • 双架构验证、iso-loss 几何、透明 FLOPs 账本
  • 模型无关流程，利于 CPT 工具链标准化

Limitations

  • 仅 8B 级；70B+ 外推未验证
  • 未对比 WSD / re-warmup / data replay 启发式
  • L₀, α, γ 正文未完整列表；未开源代码

工业相关性

  • 美团大规模域适配 CPT 典型场景；可与 2606.05433（ZK 训练验证）组合
  • 与 Gadre continued-training 缩放、Gupta/Ibrahim CPT warmup 同属第二阶段训练文献簇
  • 与 2606.05682（NVFP4 蒸馏）互补：稳定 CPT 超参影响量化学生起点
