So What

固定输入的 LLM-as-judge 管线在可追问裁判的现实下不安全：Bradley-Terry 排名 Kendall τ 可跌至 0.50，人类对齐从 67% 降至 48%。

Critical Edge

仅 2 个 GPT-4o 系评判、100 英文对；anti-baseline 为诊断性协议（非任意操控）——需 counterbalanced 才得 DS≈0；挑战 prompt 未开源。

双协议 · 方法论

说服指向与基线判决相反的回答 → 测试已稳定判决能否被推翻（诊断性）。

B1 Baseline z⁽⁰⁾ → B2 Repeat → Neutral z⁽ⁿ⁾ → Persuasion z⁽ᶜ⁾
ERS = 1 − (α·PS + β·DS)  ·  PS = Pr(z⁽ᶜ⁾ ≠ z⁽⁰⁾)  ·  DS = max(0, DSsigned) 默认 α=β=0.5。应力测试 ERS ≈ 0.51；反平衡审计 ERS ≈ 0.903。

目标回答独立于基线随机分配 → 分离可翻转性与净定向操控。

  • PS = 0.194（authority 31.7%）仍显著可翻转
  • DSsigned = −0.018，DS = 0 → 无超出中性复评的净 steering
  • 池化 Bradley-Terry 排名 Kendall τ = 1.00（0/6 换位）

核心贡献

Post-decision manipulability

固定候选、仅变 judge 对话；独立 failure mode。

双协议设计

Stress test vs validation audit；避免误读高翻转为任意操控。

Strengths

  • 首次系统量化「判决后」脆弱性，填补 one-shot 假设空白
  • 稳定性 vs 鲁棒性概念区分简洁有力
  • Within-instance 因果 + GEE 聚类 + 多对照严谨
  • 连接局部翻转到 MT-Bench/AlpacaEval 排名风险
  • 诚实区分诊断协议与观察性解读

Limitations

  • 仅 GPT-4o 系 · 100 英文成对 · 无开源 prompt
  • 受控挑战非真实用户对抗
  • 未测 rubric-lock、多 judge 投票等工业缓解
  • 机制停留行为层；ERS α/β 敏感性简述
