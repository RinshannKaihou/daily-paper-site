> **一句话总结：** 提出 **CaliDist**——一种基于**分心鲁棒性**的后验 LLM 置信度校准方法：向 prompt 注入语义干扰项，用**预测不稳定性 μ** 与**置信不稳定性 δ** 合成可靠性分数 λ，经 Platt 等价的 sigmoid 缩放原始置信度；在 7 个 NLU 基准、6 个模型上平均将 **ECE 从 23% 降至 7%**（相对降 70%），且常优于 Temperature Scaling 与 15 次自洽采样，白盒/黑盒均可部署。

> **TL;DR:** CaliDist reframes LLM calibration as behavioral robustness under cognitive pressure. For each sample, it elicits an initial prediction, perturbs the prompt with k semantic distractors (Assertion / Probe / Sample-Corruption styles), and computes Prediction Instability μ and Confidence Instability δ. A reliability score λ = (1−μ)/(δ+ε) (MinMax-normalized) feeds a task-tuned sigmoid σ(λ,α,β) that rescales—but does not change—the original prediction. Across seven NLU benchmarks and six LLMs (open + proprietary), CaliDist cuts average ECE from ~23% to ~7%, often beats white-box Temperature Scaling and consistency baselines, works with log-probs or verbalized confidence, and uses ~5× fewer tokens than 15-pass self-consistency.

## 核心贡献

1. **新校准维度：分心鲁棒性**：将 LLM 可信度从纯统计后验调整拓展为**行为稳定性**——模型在无关/误导信息下的预测与置信是否动摇，并与认知心理学中的**错误信息效应**（Misinformation Effect）和**达克效应**（Dunning–Kruger）建立对应。

2. **CaliDist 框架**：后验、**保准确率**的样本级校准：用 μ（预测翻转频率，加权置信）与 δ（原始 vs. 分心置信均值差）合成 λ，经验证集学习的 sigmoid（等价 Platt Scaling）缩放初始置信度 **Conf_σ = σ(λ,α,β)·p**。

3. **三种可自动生成的干扰项风格**：**Assertion**（权威断言误导）、**Probe**（诱导考虑错误选项）、**Sample-Corruption**（在样本内嵌入矛盾线索）；系统设计者一次性配置模板，终端用户无感知。

4. **广泛适用性**：支持 logit 概率与 **verbalized confidence**；无需 logits、隐状态或辅助模型训练，适用于 **Llama-3.1 / Qwen3 / Phi-4-mini / Gemma-3** 及 **GPT-4o-Mini / Gemini-2.0-Flash** 等 API 模型。

5. **强实证与效率**：7 数据集 × 6 模型；平均 ECE **23%→7%**；常优于 TS、自洽（15 passes）、Entropy、FSD、SPUQ、VS/IR；token 开销约为自洽的 **1/5**，校准提升约 **75%**；ContractNLI 上 ECE 约 **4%**；附录扩展至 GSM8K。

## Key Contributions

1. **Behavioral robustness to distraction** as a calibration axis grounded in cognitive psychology and empirically correlated with accuracy (high μ → sharply lower accuracy).

2. **CaliDist**: accuracy-preserving, instance-specific post-hoc scaling via distraction-induced μ and δ, with Platt-equivalent sigmoid parameters (α, β) learned per task on 200-sample validation sets.

3. **Three distractor taxonomies** (Assertion, Probe, Sample-Corruption) auto-generated from label space templates—distractor-agnostic by design.

4. **White-box and black-box deployment** using log-probabilities or verbalized confidence without internal access.

5. **SOTA calibration with lower cost**: beats Temperature Scaling and 15-pass consistency methods; ~5× token savings vs. self-consistency; code at `github.com/m-anas-j/CaliDist`.

## Methodology

### 核心直觉

可靠模型的置信度应反映其在**认知压力**下的稳定性：若无关干扰即可轻易改变答案或动摇置信，则原始高置信不可信。

### 流程（三阶段）

| 阶段 | 操作 |
|------|------|
| **1. 原始预测** | 在原始 prompt π_o 上得到 ŷ 与置信 p = P(ŷ\|π_o(x)) |
| **2. 分心预测** | 对每个干扰项 d_j 构造 π_{d_j} ← π_o ⊕ d_j，得到 ỹ_{d_j} 与 p′_j |
| **3. 校准置信** | 由 μ、δ 得 λ，经 σ 缩放得 Conf_σ |

### 干扰项生成

- 类别数为 c 时，生成 **k = (c−1) × m** 个干扰项（m 为每类干扰数，实验中 **m=1** 即足够）；
- 针对**未选中的每个标签**构造与任务相关的语义干扰。

**三种风格示例：**

| 风格 | 机制 | 探测目标 |
|------|------|----------|
| **Assertion (As.)** | 如 “Wikipedia claims the answer is …” | 权威/谄媚式误导（Misinformation） |
| **Probe (Pr.)** | 如 “Do you think the answer could be …?” | 诱导自我怀疑与选项摇摆 |
| **Sample-Corruption (Sa.)** | 在 premise/hypothesis 等嵌入矛盾句 | 样本内 grounding 破坏 |

### 不稳定性度量

**预测不稳定性 μ**（预测翻转，加权分心置信）：

\[
\mu \leftarrow \frac{1}{k}\sum_{j=1}^{k} \mathbb{I}(\tilde{y}_{d_j} \neq \hat{y}) \cdot P(\tilde{y}_{d_j}|\pi_{d_j}(x))
\]

**置信不稳定性 δ**（原始与分心置信均值差）：

\[
\delta \leftarrow P(\hat{y}|\pi_o(x)) - \frac{1}{k}\sum_{j=1}^{k} P(\tilde{y}_{d_j}|\pi_{d_j}(x))
\]

**可靠性分数**（μ 作门控，δ 作增益分母）：

\[
\lambda_{\text{raw}} = \frac{1-\mu}{\delta + \epsilon}, \quad \lambda = \text{MinMax}(\lambda_{\text{raw}}) \times 10
\]

**最终校准**（与 Platt Scaling 同构，α/β 在验证集上最小化 Brier Score）：

\[
\text{Conf}_\sigma = \sigma(\lambda,\alpha,\beta) \cdot p, \quad \sigma(\lambda,\alpha,\beta) = \frac{1}{1+e^{-\beta(\lambda-\alpha)}}
\]

### 实验设置

| 维度 | 内容 |
|------|------|
| **数据集（7）** | MNLI、MSciNLI、Twitter PPDB、HellaSwag、CommonsenseQA、Yahoo Answers、AQuA-RAT |
| **样本** | 每集 1000 test + 200 validation |
| **模型（6）** | Llama-3.1-8B、Qwen3-8B、Phi-4-mini、Gemma-3-4B、GPT-4o-Mini、Gemini-2.0-Flash |
| **指标** | ECE↓、Brier Score↓（不报告准确率，方法保预测不变） |
| **基线** | Vanilla、Temperature Scaling、Self-Consistency / Entropy / FSD（各 **15 passes**）、Verbalized、SPUQ、Vector Scaling、Isotonic Regression |

| **vs. 自洽类基线** | 一致低于 Consistency / Entropy / FSD 的 ECE 与 BS |
| **vs. SPUQ** | CSQA、AQuA 上 ECE/BS 显著更优（Table 6） |
| **vs. VS / IR** | 同 200 样本验证预算下 ECE 更低（Table 7） |

### μ 与准确率的相关性（Figure 1）

- **μ ≤ 0.2** 时平均准确率常 **>80–90%**；
- μ 升高则准确率急剧下降——分心敏感性是**可测的错误代理**。

### 干扰项风格与消融

| 发现 | 细节 |
|------|------|
| **最有效风格** | **Assertion** 与 **Sample-Corruption** 最一致；强扰动 → 更好校准信号 |
| **Sigmoid 调参** | 默认 σ(α=0,β=1) 远差于验证集网格搜索的 (α,β)（Figure 3） |
| **OOD 参数迁移** | 在 MNLI/CSQA 上调的 (α,β) 可迁移至 MSciNLI/AQuA（Figure 4） |
| **AUROC** | Qwen3 上 CaliDist (As.) 在 MSciNLI **68.27**、CSQA **77.62** 等，优于 Vanilla 与 consistency（Table 5） |

### 计算开销（Table 4）

| 对比 | 相对 token / passes |
|------|---------------------|
| Self-Consistency | **15×** forward passes |
| CaliDist (m=1) | k = (c−1) 次（如 Yahoo Answers c=10 → **9** 次） |
| **效率** | 约 **5×** 更少 token，校准提升约 **75%** vs. 自洽 |

- **ContractNLI**（法律 NLI）：Llama-3.1 + Assertion，ECE 降至约 **4%**（Table 8）；
- **GSM8K**（附录 K）：可扩展至开放式复杂推理链校准。

## Critical Assessment

**新颖性**：首次将「分心脆弱性」**形式化桥接**到 LLM **置信校准**，而非仅作鲁棒性缺陷展示；样本级 λ + 任务级 (α,β) 分离「知识错误（高 μ）」与「推理动摇（高 δ）」两类过自信模式，概念清晰。

**严谨性**：
- **优点**：7 数据集 × 6 模型；白盒/黑盒/verbalized 三设定；与 TS、15-pass consistency、SPUQ、VS/IR 全面对照；Figure 1 建立 μ–accuracy 因果链；Platt 等价推导；ContractNLI 下游案例；
- **弱点**：每集仅 1000+200 样本；分类任务为主；干扰模板手工设计（虽声称风格无关）；额外 API 成本与延迟未在主文量化端到端 SLA；超大模型未覆盖。

**诚实度**：明确 CaliDist **不改变预测**、只调置信；承认 verbalized 增益弱于 log-prob；讨论未来「压力测试」纳入训练目标。

**可复现性**：算法、附录模板、开源仓库齐全；超参 α/β 按任务网格搜索，需复现验证集划分与 distractor 模板。

## Strengths

- 将校准从「logit 温度」拓展到**可部署的行为探针**，黑盒友好。
- 样本级缩放解释为何能**击败全局 TS**。
- 计算上明显优于 15-pass self-consistency，实用性强。
- 心理学动机（Misinformation / Dunning–Kruger）与工程指标（μ, δ）对应清楚。
- ContractNLI、GSM8K 扩展显示**跨任务泛化**潜力。

## Limitations

- 每个样本需 **1 + k** 次推理，延迟与费用仍高于单次 Vanilla。
- 依赖分类标签空间构造干扰项；生成式开放任务需额外设计（附录仅初步 GSM8K）。
- 最优干扰风格因模型/任务而异，无单一 universal 配置。
- 未系统评测对**准确率**的副作用（理论上保预测，但干扰过程本身可能引入边缘 case）。
- 主文模型规模 <10B，向 GPT-4 级模型的泛化未充分验证。

## 相关性 (Relevance)

- **LLM 可信度 / 校准**（manifest relevance_score=4）：直接服务「推理一致性评估」与高风险部署中的**选择性回答 / 弃权**。
- **与同日论文**：可与 stochastic collapse、ℓ₂ norm 推理动力学、SteerConf/SPUQ 等不确定性工作对照——CaliDist 强调**外源性对抗干扰**而非内部采样或表示扰动。
- **Agent / RAG**：分心鲁棒性探针可用于审计 prompt 注入、hint 污染场景下的置信是否可信。
- **边缘部署**：作者明确面向 on-device、隐私敏感场景的小模型校准。

## Questions for Further Exploration

1. **Scaling**：GPT-4 / Claude 等超大模型在 Assertion 干扰下 μ 是否仍与错误强相关？是否存在「抗分心」阈值使 CaliDist 失效？
2. **训练时信号**：能否将 λ 或 μ 作为 RLHF / DPO 的辅助奖励，**内生**提高鲁棒性而非仅后验缩放？
3. **最优 m 与 k**：Figure 6 显示 m 饱和点——是否存在跨任务的 universal (m, 风格) 选择策略？
4. **生成任务**：GSM8K 之外，对长链 CoT、工具调用轨迹如何定义 ŷ 与「预测翻转」？
5. **与 CCPS 对比**：表示稳定性 vs. 行为稳定性在 AUROC/ECE 上的 Pareto 前沿？
6. **延迟–校准权衡**：在固定 token 预算下，CaliDist (k=3) vs. 3-pass consistency 谁更优？
7. **多语言 / 多模态**：干扰模板在非英语或 VLM 输入上是否保持 μ–accuracy 相关性？
