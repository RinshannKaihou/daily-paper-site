UltraVR — diagnostic ultra-resolution VQA for evidence-grounded reasoning

KEY FINDING: Frontier VLMs peak at 44.9% macro accuracy on four-choice evidence VQA; oracle intermediate facts (S5) lift the mean to 80.2% (+49.4 pp vs. S4)—failures concentrate in GND and PER, not generic CoT or language-side inference.

Oracle lift (S5)

Mean 80.2% when GT prefixes supply visual facts—bottleneck is acquiring evidence, not formatting reasoning.

Why this matters

Turns ultra-res VQA from leaderboard accuracy into a traceable visual→decision pipeline for CCTV, RS, pathology, and anomaly QA.

  • Most trajectories fail first at GND or PER

  • S5: QUA/INT/INF near ceiling once visual facts are given

Four-domain diagnostic benchmark

Design: visually grounded QA only · evidence tiny / distant / cross-scale · verifiable intermediate facts · strict QC (leakage, single answer, expert WSI review).

PANDA · gigapixel surveillance

  • Dense crowds, small targets, occlusion, long-range layout
  • Reasoning: comparison · logical verification · relational inference
  • GPT-5.5 domain overall: 51.8%

DOTA 1.5 · aerial

  • Small objects across wide layouts; cross-region comparison
  • GPT-5.5 RS overall: 44.2%
  • Gemini-3.1-Pro CCTV Logic subtask up to 60.0% in places

TCGA-BRCA · whole-slide imaging

  • Multi-scale morphology; ROI navigation; pathologist-reviewed
  • GPT-5.5 WSI overall: 52.7%
  • V3 local evidence = ROI crops (not full slide)

MVTec LOCO AD · synthetic 16k grid

  • 10×10 → ultra-res grid; rare logical/structural anomalies in repetition
  • Hardest domain: GPT-5.5 AD 30.8%
  • V3 = random 5×5 native cells from source 10×10 (not answer-local)

Structured GT-CoT (annotation schema)

GND → PER → QUA → INT → INF

Per-step annotation

  • Step questions + verifiable intermediate answers
  • Operation label per step (cross-domain shared)
  • Not model-generated rationales—diagnostic ground truth

Reasoning taxonomy

  • CCTV/RS/WSI: comparison · logic verify · relational
  • AD: logical anomaly · structural anomaly
  • All tasks: 4-choice closed answers

vs. prior benches

  • XLRS · RSHR · UR-Bench · HR-Bench · V*
  • UltraVR = evidence candidates + functional chain + op tags
  • Positioned as diagnostic, not SOTA leaderboard

Strengths

  • First systematic GND/PER localization for ultra-res VQA
  • GT-CoT + S5 oracle cleanly splits perception vs. inference
  • Four real-world domains + strict QC & expert WSI review
  • Engineering signal: local pipelines > generic CoT prompts

Implications

  • Adaptive visual search · multi-scale memory
  • Process supervision on GND/PER steps
  • Agent tool-use / zoom-in vs. fixed V3 crops
