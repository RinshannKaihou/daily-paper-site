Hybrid predict & reconstruct reshapes embedding geometry while GLUE probes stay flat

Headline: Joint JEPA latent prediction + MLM token reconstruction on one shared encoder yields dramatically more uniform, spectrally richer embeddings—yet linear-probe GLUE accuracy barely moves.

Anchor fact
7× uniformity gap · attention pool · SST-2

Hybrid −0.448 vs MLM −0.055; 20/20 dataset–pooling cells favor hybrid uniformity (≤ −0.16 vs ≤ −0.05).

So what

Linear-probe accuracy alone is a blind spot—uniformity, spectral rank, and probe-gap metrics reveal objective differences invisible to GLUE scores.

Critical edge

3 epochs WikiText only; GLUE ~50–69%; no significance tests; single-author limited scale—geometry claims, not task SOTA.

Mechanism

MLM alignment ≈ 0 (tight clusters) cancels hybrid uniformity gains at probe time—matched accuracy, different geometry.

Hybrid pre-training architecture

L = λ LJEPA + (1 − λ) LMLM  ·  λ = σ(w) learnable  ·  LJEPA = 1 − mean cos(pred, EMA target)
Shared encoder (512-d) → JEPA: block mask [0.65–0.85] + MLM: 15% token mask → 6-layer predictor → EMA target f̄θ (m ∈ [0.996, 1])

Controlled setup

  • WikiText · bert-base-uncased · max len 512
  • 3 epochs · batch 64 · single H100 · bfloat16
  • AdamW peak 5×10⁻⁵ · cosine · wd 0.05
  • Identical compute vs MLM-only baseline

Evaluation protocol

  • 5 GLUE tasks · frozen encoder · linear probe
  • 4 pooling: mean · max · weighted · attention
  • 6 metrics: spec. entropy · eff./stable rank · alignment · uniformity · probe gap
  • ≤ 2,000 validation samples per task

Core contribution

  • First systematic geometry study: GLUE × pooling × metrics
  • Learnable λ—no manual loss weighting
  • Open: aymen-000/predict-reconstruct-language-models

Alignment–uniformity trade-off

MLM pulls same-class points together (alignment ≈ 0); hybrid spreads the hypersphere (uniformity ↓). Linear probes see both effects—and cancel.

Critical assessment

Strengths

  • Clean controlled ablation (objective only)
  • Rich 5×4×6 metric grid rarely reported for SSL
  • Mechanistic alignment–uniformity explanation
  • Attention pooling as JEPA diagnostic

Novelty & significance

  • Hybrid objective is incremental; evaluation protocol is novel
  • Warns practitioners against accuracy-only pretraining selection
  • CoLA MCC hints block masking aids syntax

Read before citing

  • 3 epochs WikiText—geometric gains may shrink at BERT-scale training
  • No bootstrap CIs; λ trajectory not reported in main text
  • No SimCSE / DeCLUTR / contrastive baselines
  • Linear probes only—retrieval & MLP probes left to future work
  • Uniformity may hurt uses needing tight semantic clusters

Open questions

Does uniformity persist at BookCorpus + Wikipedia, 40+ epochs?
Scale-up experiment the authors flag as highest priority.
Retrieval & non-linear probe translation
Hypothesis: geometric gains appear only beyond linear GLUE probes (STS12–16, MLP heads).
Curriculum λ: MLM-heavy → JEPA-heavy
May improve both accuracy and geometry vs. fixed learnable blend.
Alternative JEPA losses (VICReg, Barlow Twins)
Pareto frontier for alignment–uniformity without sacrificing MLM recovery.
