Minimizing the Hidden Cost of Scales: Graph-Guided Ultra-Low-Bit Quantization for Large Language Models

Why This Matters

超低比特 PTQ 常被未计入的 scale/LUT 元数据吃掉收益——SAGE 在单卡 L40 上 LLaMA-2-70B 解码约 1.5×、GPU 显存较 BiLLM 约 −50%，单卡大模型推理叙事有力。

Critical Edge

仅 weight-only，激活仍 FP16/未压缩；谱聚类离线成本高于纯 GPTQ；与 QuIP#/SpinQuant 等强 multibit 基线对比有限；代码尚未开源（正文承诺 soon）。

Methodology — SAGE-PTQ Pipeline

针对 PB-LLM/BiLLM 的 rigid 分组、隐藏 scale >1 bit、启发式显著性——五阶段图引导 PTQ。

Probit 显著性 → 稀疏 KNN + 谱聚类 → 双模式量化 → Brent 阈值 → bit-packed LUT
Ωsall = { w ∈ Ωl | |w| > βl + γl Φ−1(ζsall/2) } Mode 1：显著多比特 + 矩阵 per-channel scale · Mode 2：非显著二值/三值 + 每组标量 scale · Brent 最小化 Frobenius 重建误差 J(ζ)
  • 图引导组数： 子采样非显著权重 → 稀疏 KNN → Silhouette 最大化选 Noptimaluns,l → 分位数 mask。
  • 自适应显著性： LLaMA-7B 约 50% 更低 PPL 且 ~30% 更少显著权重比特（vs 固定阈值）。
  • 运行时： LLUTl 放 host 层间流式加载；lookup 成本较 BiLLM 低约 25%。

核心贡献

显著性 + 图引导划分： 分布统计 + 谱聚类/Silhouette 替代固定 block 启发式。
双模式极低 scale： 矩阵级 salient scale + 组标量 unsalient，0.004 scaling bits。
Brent 自适应阈值： 凸重建目标逐矩阵优化 ζsal，更少显著比特、更低 PPL。
跨族 1.3B–70B 实证： LLaMA/OPT/Vicuna/DeepSeek/Qwen，PPL + 零样本 + 显存 + 延迟。

Critical Assessment

  • 新颖性： 揭示 ultra-low-bit 收益被未计入 metadata 吃掉；数据驱动组数 + 显式比特会计差异化清晰。
  • 严谨性： 10+ 模型族、同时报告 weight/scale/PPL/零样本/显存/延迟；BiLLM lookup 取 best-case 仍占优。
  • 工程意义： PPL 55.8→6.74 量级跃迁 + 半显存，memory-bound 单卡 70B 部署有实际价值。

Strengths

  • 系统纳入 scaling + LUT bits，暴露 BiLLM 有效比特率远高于名义值
  • 图聚类 + 分位数 mask 兼顾非高斯 early-layer 与可计算子采样
  • 双模式 scale 极简，~1 bit 总 footprint 级部署
  • 评估维度完整：PPL、6 项零样本、显存、量化时间、70B 延迟分解
  • 附录权重统计为方法提供可解释动机

Limitations

  • 仅权重量化，端到端受 activation memory 限制
  • 谱聚类多阶段 PTQ 离线成本仍高于纯 GPTQ
  • MoE、多模态、长上下文 KV 未评估
  • 与 3–4 bit AWQ/GPTQ 帕累托全面对比不足
  • 代码未发布；零样本仅 3 个模型
