Strengths

  • Clear three-RQ structure with matched deletion protocol across explanation types
  • Honest trade-off analysis: accuracy (PLM) vs score range (LLM) vs explanation faithfulness (SHAP)
  • Cross-model experiments go beyond single-model eraser benchmarks
  • QoF dimension choice justified (less skewed CLASS axis, text-manifest indicators, pedagogical importance)

Limitations

  • **Scope:** QoF only; unclear generalization to less discourse-heavy CLASS dimensions (e.g., Productivity)
  • **Modality:** Text transcripts ignore prosody, timing, video—standard CLASS practice uses richer cues
  • **Labels:** Single expert per segment; no inter-rater reliability; extreme scores rare → PLM compression
  • **Evaluation gap:** No human audit whether top sentences instantiate rubric constructs

TL;DR

On 6k NCTE transcript segments scored for CLASS Quality of Feedback (1–7), a sentence-level framework compares hierarchical fine-tuned PLMs (SHAP on sentence embeddings) against few-shot LLM scorers plus zero-shot top-10 sentence rankings. Fine-tuned PLMs win on MAE (~0.96–1.00 vs best LLM 1.02) but compress labels to mid-scale; deletion tests show SHAP induces larger, more coherent score shifts and transfers across architectures, whereas LLM rationales overlap weakly (Jaccard≈0.085, Spearman≈0.062) and perturb PLMs poorly—arguing Shapley attributions over persuasive LLM prose for high-stakes rubric scoring.
