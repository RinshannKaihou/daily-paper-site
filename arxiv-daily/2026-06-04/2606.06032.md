> **一句话总结：** 在 Split CIFAR-100 + ResNet-18 的纯顺序训练（无 replay/正则/CL 方法）下，Task 0 准确率从 54.8% 跌至 0%，但线性探针仍保留 checkpoint 水平的 **76%** 表征信息，冻结骨干仅重训分类头可恢复 **75.7%** 原性能；遗忘集中在 Layer 4 与 readout，早期层甚至略有提升——作者提出「存储–表征–可访问性」三层框架与 **Accessibility Gap**，将灾难性遗忘重新定义为**可访问性崩溃**而非表征擦除。

> **TL;DR:** On Split CIFAR-100 with ResNet-18 trained sequentially on 10 tasks (no replay, regularization, or CL method), Task 0 accuracy collapses from 54.8% to 0% while linear-probe retention stays at 76% of checkpoint level and a frozen-backbone classifier reset recovers 75.7% of original performance. Forgetting concentrates in Layer 4 and the final classifier; Layers 1–3 retain or improve probe accuracy (100–104%). The authors formalize the **Accessibility Gap** (AG = LP − ACC = 0.468) and **Projection Energy**, show distributed (not low-rank) representation persistence despite ~79° mean principal-angle drift, and report transparent negative results for parameter-space geometric recovery—framing catastrophic forgetting as **accessibility collapse**, not representational erasure.

## 核心贡献

- **三层知识框架**：区分 **Knowledge Storage**（checkpoint 仍达原精度）、**Knowledge Representation**（冻结特征上线性探针）、**Knowledge Accessibility**（端到端任务准确率）；核心发现是表征层与行为层之间的巨大裂隙。
- **Accessibility Gap（AG）**：AG = LP_final − ACC_final；Task 0 上 AG = 0.468 − 0.000 = **0.468**，量化「表征仍可读、原生决策路径已失效」。
- **纯遗忘基准**：10 任务 Split CIFAR-100、ResNet-18、SGD+交叉熵，**无** replay/正则/架构隔离；Task 0 准确率 **0.548 → 0.000**，建立干净、可复现的极端遗忘设定。
- **分类头重置（Classifier Reset）**：冻结全骨干、仅重训 Task 0 线性头 → **0.415**（原 0.548 的 **75.7%**），证明 backbone 仍承载大量可恢复任务信息。
- **层级可恢复性层级**：Layer 1 **122%**、Layer 2 **121%**、Layer 3 **110%**、Layer 4 **85%**、全骨干 reset **75.7%**；早期层在后续任务训练后甚至**更**利于 Task 0 线性可分。
- **Projection Energy + 主角度**：PE(K) 随 K 单调上升（K=5→0.054，K=300→0.583），高探针精度与低主导子空间重叠并存 → **分布式高维编码**；checkpoint vs final 子空间平均主角度 **79°**（最大 ~90°），仍保留 76% LP。
- **参数空间恢复负结果**：θ_rec = θ_final + α(θ_i − θ_0) 在所有配置下接近 chance；漂移与 LP 保留的 Pearson r 仅 **−0.236 / −0.339**——透明报告失败结论。
- **Accessibility Collapse Hypothesis**：灾难性遗忘主要是**末层 readout 的功能可访问性崩溃**，而非全网表征结构性销毁；倡导 continual learning 从「预防」转向「修复」范式。

## Key Contributions

- **Three-level knowledge model:** Separates storage (checkpoints retain learned accuracy), representation (frozen-feature linear probes), and accessibility (end-to-end task accuracy)—with a large empirical gap between the latter two.
- **Formal diagnostics:** Defines **Accessibility Gap** AG = LP_final − ACC_final and **Projection Energy** PE(K) = ‖U_K U_K^T H_f‖_F² / ‖H_f‖_F², plus principal-angle analysis for subspace drift.
- **Controlled pure-forgetting benchmark:** Split CIFAR-100, ResNet-18, 10 sequential tasks, no CL interventions; Task 0 accuracy 0.548 → 0.000.
- **Stronger-than-probe recovery test:** Classifier reset (frozen backbone, new head) recovers 75.7% of original Task 0 performance from a model scoring 0.000.
- **Layer-wise retention and recoverability hierarchies:** Early layers retain/improve (103–104% LP retention); degradation concentrates in Layer 4 and classifier; independent per-layer readouts exceed full-backbone reset.
- **Distributed encoding evidence:** Rising PE(K) with K, high LP despite low dominant-subspace overlap and large principal angles.
- **Negative result transparency:** Weight-space interpolation toward checkpoints does not restore accuracy.
- **Accessibility Collapse Hypothesis:** Synthesizes evidence into a repair-oriented reframing of continual learning.

## Methodology

**Setup.** Split CIFAR-100 → 10 tasks × 10 classes; ResNet-18 from scratch; standard cross-entropy + SGD; checkpoint θ*_i saved after each task; analysis on final model θ_N vs stored checkpoints.

**No CL tricks.** Deliberately excludes replay, regularization (EWC/LwF/SI), and architectural isolation to study unconstrained forgetting.

**Metrics.**
- **Task accuracy:** held-out per-task classification.
- **Linear probe (LP):** train linear classifier on frozen layer features.
- **LP Retention:** LP_Final / LP_Checkpoint.
- **Accessibility Gap:** AG = LP_final − ACC_final.
- **Projection Energy:** fraction of final activation energy in top-K checkpoint principal directions.
- **Principal angles:** via SVD of U_1^T U_2; σ_i = cos(θ_i).

**Experiments (7 strands).**
1. Baseline forgetting (Task 0 & 9 accuracy).
2. Checkpoint persistence (storage level).
3. Final-layer linear probe (representation level).
4. Layer-wise LP retention (ResNet blocks 1–4 + classifier).
5. Classifier reset + layer-wise recoverability readouts.
6. PE(K) sweep + principal angles + drift–LP correlations.
7. Parameter-space recovery (negative).

**Positioning.** Extends Davari et al. (CVPR 2022) probing with layer resolution, formal metrics, three-level model, and classifier reset; engages van de Ven et al. critique via triangulation (checkpoint + probe + reset + geometry).

## Results & Highlights

| Result | Detail |
|--------|--------|
| **Task 0 accuracy** | Checkpoint **0.548** → Final **0.000** (complete behavioral forgetting) |
| **Task 9 accuracy (final)** | **0.751** (new task learned successfully) |
| **Linear probe (Layer 4)** | Checkpoint **0.616** → Final **0.468** (**76%** retention) |
| **Accessibility Gap** | **0.468** (LP − ACC) |
| **Classifier reset** | **0.415** = **75.7%** of original 0.548 |
| **Layer LP retention** | L1 **103%**, L2 **104%**, L3 **100%**, L4 **76%**, classifier **~0%** |
| **Layer recoverability** | L1 **0.669 (122%)**, L2 **0.662 (121%)**, L3 **0.603 (110%)**, L4 **0.468 (85%)** |
| **Checkpoint persistence** | Task 0/5/9 checkpoints retain exact learned acc (e.g. 0.630, 0.758, 0.751) post full training |
| **Projection Energy** | K=5: **0.054**; K=100: **0.327**; K=300: **0.583** (monotone with K) |
| **Principal angles** | Mean **79°**, max **~90°**; LP still 76% of checkpoint |
| **Param-space recovery** | Failed; acc near chance; drift–LP r ≈ **−0.24 / −0.34** |

**Interpretive synthesis.** Functional metrics (accuracy) dramatically under-report internal knowledge. Forgetting is **localized** to late layers/readout; early representations may be **refined** by later tasks. Knowledge persists as **distributed** high-dimensional structure, not a preserved low-rank dominant subspace.

## Critical Assessment

**Novelty (moderate–high for framing).** The three-level decomposition and AG metric provide a crisp vocabulary; classifier reset beyond probing is a meaningful empirical upgrade. Distributed-encoding interpretation via PE(K) vs LP dissociation is insightful. Not a new CL algorithm—rather a **diagnostic/reframing** paper.

**Rigor (solid for exploratory study).** Controlled pure-forgetting setup, systematic layer hierarchy, geometry analysis, and honest negative results. Gaps: limited scale (CIFAR-100 only), single architecture, unclear multi-seed variance, no comparison to standard CL baselines (by design).

**Reproducibility (moderate).** Protocol and metrics are clearly specified; code availability not stated in PDF extract—implementation details (epochs, lr, probe training protocol) would need full paper/appendix for replication.

**Significance (high for CL theory & practice).** If accessibility collapse generalizes, continual learning should invest in **adaptive readouts**, routing heads, and repair mechanisms rather than only parameter insulation. Implications for evaluation: reporting accuracy alone may mislead about retained knowledge.

**Red flags:**
- Generalization beyond Split CIFAR-100 + ResNet-18 unproven
- Early-layer recoverability > original end-to-end performance is striking but needs broader validation
- Parameter-space recovery failure may be method-specific (α schedule, interpolation path not fully detailed in summary tables)

## Strengths

- Compelling empirical dissociation: 0% accuracy vs 76% LP vs 75.7% reset recovery
- Layer-wise story is consistent across retention and recoverability experiments
- Formal metrics (AG, PE) enable cross-paper comparison
- Repair-oriented discussion challenges prevention-only CL orthodoxy
- Engages representation-probing critique seriously (triangulation + geometry)
- Negative result on weight interpolation reported openly

## Limitations

- **Scope:** Split CIFAR-100, ResNet-18, 10-task sequential classification only
- **No CL baselines:** By design—but limits direct comparison to SOTA mitigation performance
- **Probe semantics:** High LP does not alone prove "meaningful" knowledge (authors acknowledge; reset partially addresses)
- **Single readout repair:** Classifier reset is one lightweight fix; no learned routing/multi-head system demonstrated
- **Parameter recovery:** Tested interpolation fails; alternative geometric repair paths unexplored
- **Statistics:** Tables show point estimates; seed variance / confidence intervals not prominent in extracted text
- **Scale:** Modern large models, NLP/RL continual settings not evaluated

## Relevance

若你关心**灾难性遗忘的机理**、表征探测（linear probe）、或 continual learning 的**评测指标设计**，本文值得精读：它用极简训练设定把「准确率归零」与「表征大量残留」并置，并给出可操作的 **AG** 与层级诊断。对设计**任务特定 readout、adapter、路由头**、以及「遗忘后修复」路线（相对 EWC/replay 的预防路线）有直接概念启发。

## Questions for Further Exploration

1. **Does the AG / 76% LP / 75.7% reset pattern replicate on ImageNet-R, CORe50, or transformer backbones** with multi-seed confidence intervals?
2. **Can layer-selective or multi-layer readouts beat full-backbone classifier reset** given that L1–L3 individually exceed 0.415 recovery?
3. **What training interventions minimize AG** without full replay—e.g., context-dependent heads, hypernetwork readouts, or sparse routing?
4. **Does distributed PE(K) growth imply specific CL regularizers should target readout alignment** rather than backbone drift?
5. **Why does parameter-space interpolation fail while representation-space reset succeeds**—can mode connectivity / linear mode connectivity explain viable repair paths in weight space?
