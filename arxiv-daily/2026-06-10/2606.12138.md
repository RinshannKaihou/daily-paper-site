> **一句话总结：** SAE 不稳定特征个体上不可复现，但几何上集中于跨种子可复现的低秩子空间，反映的是激活空间共享区域内的基底模糊，而非随机噪声；稳定特征以 4.5× 更高的自动解释命中率承载绝大多数重建与预测信号。

> **TL;DR:** Unstable SAE features are not noise: they individually fail to reappear across seeds but collectively span reproducible low-rank subspaces, reflecting basis ambiguity within shared activation-space regions. Stable features (with 4.5× more perfect auto-interpretation scores) carry almost all reconstruction and prediction signal; pooling unique cross-seed features builds more stable SAEs without sacrificing explained variance.

## 核心贡献

- 提出**逐特征稳定性概率** $\hat{p}_i$：在 $N+1$ 个独立训练的 SAE 中，通过余弦相似度匹配（阈值 $\theta=0.7$）估计每个特征重新出现的概率，形成可扩展的无监督信号，将特征分为稳定（$\hat{p} \geq 0.95$）与不稳定（$\hat{p} \leq 0.05$）两类。
- 大规模实证揭示**功能不对称性**：稳定特征激活频率更高（0.44% vs 0.18%）、幅度更大，承载绝大多数重建解释方差（EV）和下游 next-token 预测信号，而不稳定特征的边际贡献极弱——遮蔽 $4N$ 个不稳定特征的 EV 损失仍远小于遮蔽 $N$ 个稳定特征。
- **语义分析**：不稳定特征由低频表层触发（子串、大小写、标点），稳定特征对应短语、句法角色、高级语义概念；GPT-5 仅凭自动解释文本即可以 0.88 准确率区分两类特征。
- **几何发现**：不稳定特征解码器子矩阵的有效秩（ER/d）为 0.59–0.65，显著低于稳定特征的 0.80–0.81（降低 20–27%）；但不稳定特征所张成的低秩子空间在不同种子间高度一致（跨种子 SVD 子空间传输接近同种子水平）。
- **受控合成实验**验证机制：在 $d=32, r=2, k=8$ 的低秩合成字典中，全秩特征跨种子近完美复现，低秩特征（均值 $\hat{p}\approx0.25$）则不然——但其子空间仍可恢复，从而以可控方式明确了"基底模糊"机制。
- **跨种子特征池化构建稳定 SAE**：汇聚多个独立训练 SAE 的解码器特征，去重（余弦阈值 0.7），从高概率特征初始化新的 $F=16384$ 维 SAE，经简短调优后既提升特征稳定性又保持 EV（在 SAEBench 上与标准 TopK SAE 竞争力相当）。

## Key Contributions

- **Per-feature stability metric**: estimates $\hat{p}_i = X_{0,i}/N$, the fraction of $N$ independently trained SAEs that contain a feature matching anchor feature $i$ at cosine similarity $\geq 0.7$. Endpoint categories: *unstable* ($\hat{p} \leq 0.05$) and *stable* ($\hat{p} \geq 0.95$); main experiments use 96 TopK SAEs on GPT-2 layer 7, $F=2^{14}$, mean $\hat{p}\approx 0.75$.
- **Functional asymmetry (empirical)**: masking $4N$ unstable features degrades EV only marginally, while masking $N$ stable features causes substantially larger drops. Same asymmetry holds for next-token cross-entropy under activation patching.
- **Token-structure and auto-interpretability gap**: stable features cover phrases, syntactic roles, semantic groupings; unstable features are dominated by surface-form triggers (substring, capitalization, punctuation). SAEBench auto-interpretation: stable features have 4.5× more perfect-score features; GPT-5 classifies stable vs. unstable from explanation text at 0.88 accuracy.
- **Geometric analysis**: unstable feature decoder submatrices have 20–27% lower effective rank than stable ones, yet their subspaces transfer cross-seed as accurately as within-seed, confirming reproducible low-rank structure despite individual non-reproducibility.
- **Synthetic validation**: controlled low-rank toy model ($d=32, r=2$) confirms full-rank features are individually identifiable across seeds, while low-rank features are not — but their shared subspace is recoverable.
- **Feature-pool construction**: cross-seed pooling of high-stability features builds SAEs with dramatically reduced unstable fractions without an apparent EV penalty; the most-probable construction quickly dominates as pool size grows.

## Methodology

**SAE variants studied.** Five variants are evaluated: Vanilla SAE (ReLU + $\ell_1$), TopK, BatchTopK, HierarchicalTopK, and JumpReLU. Training uses 1B token activations from FineWeb's sample-10BT split. Decoder columns are $\ell_2$-normalized so cosine similarity reduces to dot product.

**Stability estimation.** Train $N+1$ SAEs with the same data order but different random seeds. Designate one as anchor ($k=0$). For each anchor feature $e_i^{(0)}$, count how many of the $N$ other SAEs contain any feature with cosine similarity $\geq \theta=0.7$:
$$X_{0,i} = \sum_{k=1}^{N} \mathbf{1}\!\left\{\max_{j} \cos(e_i^{(0)}, e_j^{(k)}) \geq \theta\right\}, \quad \hat{p}_i = X_{0,i}/N.$$
This many-to-one matching is computationally cheap and, as a robustness check, yields nearly identical results to one-to-one Hungarian matching (IoU = $0.978 \pm 0.001$).

**Functional impact — frequency-matched masking.** To control for the lower activation frequency of unstable features, the authors mask $N$ stable features and $4N$ unstable features to roughly equalize expected active mass, then measure EV and next-token CE degradation.

**Geometric analysis.** For each feature set (stable/unstable), the decoder submatrix $X_\mathcal{I}^{(s)} = W_\text{dec}[:, \mathcal{I}^{(s)}]$ is formed; effective rank (ER) and SVD-explained variance across seeds are computed. Cross-seed subspace transfer $\text{EV}_\text{SVD}^{(a\to s)}(r)$ measures how well the top-$r$ singular subspace from seed $a$ explains the feature subspace in seed $s$.

**Synthetic model.** A dictionary $W = [D; UV]$ mixes $n_\text{full}$ full-rank rows with $n_\text{low}$ rows lying in a rank-$r$ shared subspace ($r < d$). Activations are generated by sampling $k$ dictionary rows and summing with unit coefficients; TopK SAEs are trained from multiple seeds.

**Cross-seed pooling.** Pool decoder vectors from several SAEs, greedily deduplicate at $\theta=0.7$ cosine similarity, producing unique feature pool $\mathbb{U}$; initialize new $F=16384$-latent SAEs from the most-probable, least-probable, or uniform subsets, then briefly tune.

## Results & Highlights

- **Bimodal stability distribution**: the empirical $\hat{p}$ distribution is strongly U-shaped (mean $\approx 0.75$), with most features either recurring in nearly every seed or almost none.
- **Activation statistics**: unstable features activate at mean frequency 0.18% vs. 0.44% for stable features; stable features have a heavier high-magnitude tail.
- **EV asymmetry (Figure 3)**: masking $4N$ unstable features barely changes EV; masking $N$ stable features causes a substantially larger drop. Next-token CE patching shows the same pattern.
- **Auto-interpretability**: SAEBench detection scores are higher for stable features; stable explanations use phrase/semantic keywords (phrase: 4.1% → 32.0% in stable explanations), unstable explanations use surface keywords (substring: 38.9% vs 11.3% in stable).
- **Geometric (Table 5)**: ER/d = 0.59–0.65 for unstable vs. 0.80–0.81 for stable (20–27% reduction), consistent across 96 seeds. Cross-seed SVD subspace transfer plateaus at ≈ 0.73 for $\varepsilon=0$ and 0.67 for $\varepsilon=0.1$.
- **Synthetic (Figure 6)**: full-rank features have near-perfect reappearance and cosine similarity to matched ground truth; low-rank features have $\hat{p} \approx 0.25$ and poor one-to-one recovery but recoverable subspace.
- **SAE variant comparison (Table 1)**: TopK at k=64 achieves EV 0.892 with 9.8% unstable, 52.2% stable. Vanilla ReLU+$\ell_1$ at sparsity 62 is extremely stable (0.3% unstable, 81.0% stable) but EV only 0.833 — clear stability–reconstruction tradeoff across variants. JumpReLU improves stability over TopK at comparable EV.
- **Training budget (Figure 7)**: unstable fraction decreases in early training then plateaus at ≈8% at 10B tokens; more training alone does not eliminate instability.
- **Dead-salmon baseline (Figure 8)**: SAEs on randomly initialized GPT-2 collapse to near-zero stable fraction, confirming stability reflects genuine model structure; crucially, auto-interpretation alone does not flag this failure (plausible explanations arise even for random-model features).
- **Feature-pool construction (Figure 4)**: with a modest number of source SAEs, the most-probable construction is dominated by high-stability features and achieves near-baseline EV after tuning, competitive with standard TopK on SAEBench metrics.

## Critical Assessment

**Novelty:** High within the mechanistic interpretability sub-field. While prior work (Paulo & Belrose 2025, Leask et al. 2025) established that SAEs trained on the same data with different seeds learn substantially different features, this paper is the first to provide: (1) a scalable per-feature stability probability signal, (2) a geometric account distinguishing basis ambiguity from pure noise, (3) a controlled synthetic model making the mechanism explicit, and (4) a practical pooling construction. The conceptual reframe — instability as basis ambiguity within reproducible subspaces rather than failure — is meaningful.

**Rigor:** Strong. The main experiment uses 96 SAE runs (N=95 comparisons per anchor), providing robust statistical estimates. A Hungarian-matching robustness check (IoU 0.978) validates the cheaper argmax matcher. The frequency-matched masking protocol ($N$ stable vs. $4N$ unstable) carefully controls for the confound that unstable features activate less frequently. The dead-salmon control (random-model SAEs) is an important sanity check. The synthetic model provides a mechanistic causal validation.

**Reproducibility:** Positive signals: clear hyperparameter reporting (θ=0.7, ε=0.05, F=2^14, k=64, 1B tokens from FineWeb sample-10BT), five SAE variants described precisely, appendices with formal definitions and full dataset/batching details. Code and data availability are not explicitly mentioned in the visible pages — a standard gap for preprints from this community.

**Significance:** Directly actionable for interpretability practitioners: the stable/unstable distinction provides a principled filter for which SAE features to trust in downstream analyses. The dead-salmon finding is particularly important: it shows that auto-interpretability metrics can appear high even when features are not reproducible, undermining a common quality-assessment shortcut. The pooling construction provides an immediately practical recipe.

**Red flags:** (1) The no-stability–EV tradeoff result is qualified to the feature-pool construction setting, not to SAE training objectives in general — this caveat is stated clearly but could be missed. (2) The low-rank subspace analysis is described as "evidence for a mechanism" rather than proof — other mechanisms could also produce the observed instability. (3) Results are primarily on GPT-2; generalization to frontier-scale models is plausible but untested. (4) Threshold sensitivity (θ, ε) is acknowledged as a limitation; the endpoint fractions depend on these choices.

## Strengths

- Principled, scalable per-feature stability signal that avoids manual inspection and works with standard SAE training infrastructure.
- Large-scale multi-axis empirical study (96 seeds × multiple models × layers × dictionary sizes × 5 SAE variants) builds strong confidence in the generality of findings.
- The geometric analysis and synthetic model together provide both correlational and causal evidence for the basis-ambiguity mechanism.
- Dead-salmon control is methodologically sophisticated: shows auto-interpretability is insufficient as a quality metric and that stability provides complementary information.
- Practical contribution: cross-seed pooling offers an immediately usable recipe for more stable SAEs without sacrificing downstream interpretability benchmarks.
- Authors are transparent about limitations and scope conditions.

## Limitations

- Primary results on GPT-2 (layer 7); broader model coverage is shown for stability patterns but not all analyses (e.g., pooling construction) are replicated across all models.
- The no-stability–EV tradeoff holds for the pooling construction but not necessarily for single-run SAE training objectives — the paper is careful about this, but the takeaway can be over-generalized.
- Threshold choices (θ=0.7 for matching, ε=0.05 for endpoint labeling) affect the precise sizes of stable/unstable sets; the qualitative conclusions appear robust to reasonable variations but exact quantitative claims are threshold-conditional.
- The subspace reproducibility finding is consistent with basis ambiguity but does not rule out all alternative mechanisms for instability in real LLMs.
- The pooling construction is validated primarily in the GPT-2 / F=16384 setting; computational scalability to very large dictionaries or frontier models is not demonstrated.
- Code and data release status not confirmed in available pages.

## Relevance

本文直接针对稀疏自编码器（SAE）在机制可解释性中的核心可靠性问题，对关注大语言模型内部表征结构与可解释性工具可信度的研究者高度相关；"自动解释分数高不代表特征可复现"的发现尤其值得警惕，是评估任何 SAE 实验结论时必须考量的系统性误差来源。

## Questions for Further Exploration

1. The dead-salmon control shows auto-interpretability scores can be high even for random-model SAEs — do any existing interpretability benchmarks (e.g., SAEBench) have a stability-conditioned variant that would be more reliable?
2. The low-rank subspace containing unstable features is reproducible across seeds — do these subspaces correspond to known geometric structures in transformers (e.g., self-attention anisotropy directions, dimensional collapse subspaces)?
3. The pooling construction improves stability without apparent EV penalty in the tested setting — would the same hold if applied as an initialization strategy for much larger SAEs (e.g., F = 2^17) on frontier models?
4. The paper notes that increasing TopK sparsity k monotonically increases the unstable fraction while improving EV. Is there a principled way to choose k that optimizes the stability–EV Pareto frontier rather than treating it as a hyperparameter?
5. Post-training shifts some features toward lower reappearance probabilities, suggesting lower-stability directions can be reconstruction-useful. Can a training objective be designed that explicitly encourages recovery of individual features within reproducible low-rank subspaces (i.e., breaks the basis ambiguity during training)?
