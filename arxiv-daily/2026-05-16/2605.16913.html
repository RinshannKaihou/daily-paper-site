A Fourier Perspective on the Learning Dynamics of Neural Networks

Fabiola Ricci · Claudia Merger · Sebastian Goldt  —  SISSA, Italy  |  stat.ML / cond-mat.dis-nn / cs.LG  |  2026-05-16

Neural networks learn Fourier amplitude (pixel correlations) before phase (edges, higher-order structure), and natural images' power-law spectra act as an invisible scaffold that polynomially accelerates the otherwise-exponentially-hard task of learning phase features.

Anchor Fact — Sample Complexity
n ≫ N³  →  n ≫ N · log² N
Phase learning on isotropic inputs requires cubic steps of SGD. Power-law spectra (α = 1, as in natural images) reduce this to near-linear — a polynomial speedup of order N².
Why This Matters
Data geometry IS the curriculum
The spectral structure of natural images does not encode labels — yet it fundamentally reshapes the loss landscape so SGD finds phase features efficiently. This explains why deep learning works on natural data despite the curse of dimensionality.
Critical Edge
Theory covers two-layer nets only
All formal results assume a two-layer network with online SGD. Whether amplitude-before-phase order persists in transformers or very deep networks is an open question. Cyclic model assumes exact translation-invariance, which real images satisfy only approximately.

1 Amplitude-Before-Phase Learning

Amplitude
Accuracy: ~80% (few epochs)
Pixel-pair correlations (2nd-order statistics) are learned almost immediately. On CIFAR10/ResNet-18, amplitude test accuracy on phase-randomized images reaches ~80% within the first epochs. Same pattern on CIFAR100 and ImageNet.
Phase
Accuracy: ~65% (after ~10 epochs)
Edges, corners, higher-order structure are learned slowly. Phase test accuracy starts at chance and only improves after the network has already exploited amplitude features. The gap narrows with data complexity (CIFAR100 < CIFAR10) but the order is consistent.

2 The Power-Law Scaffold

Accelerated Phase Learning

The key insight: the amplitude spectrum is uninformative about labels (amplitude accuracy = chance when labels depend on phase), yet it provides a non-trivial structural scaffold that accelerates learning of the features that DO carry label information. This is not a direct effect but a geometric one — the data distribution sculpts the optimization landscape.

3 Formal Results

Theorem 1 — Phase Lower Bound
For isotropic (flat-spectrum) inputs with labels in phases only, online SGD needs n ≥ N³ steps to learn anything. Information-theoretic proof: gradient signal per sample is O(1/N3/2), requiring cubic accumulation to overcome noise.
Theorem 2 — Phase Upper Bound
With n ≫ N³ log²N samples, two-layer network (width ≥ 2) achieves non-trivial accuracy. Gap from lower bound is only polylogarithmic — tight by theoretical standards.
Theorem 3 — Power-Law Speedup
When spectrum follows ak ∝ k-α with α > 1/2, complexity drops to n ≫ N3-2α log²N. For α = 1 (natural images): n ≫ N log²N.

4 Experimental Evidence

  • CIFAR10 + ResNet-18: Amplitude accuracy ~80% in ~2 epochs; phase accuracy ~65% after ~10 epochs. Gap confirmed.
  • CIFAR100 + ResNet-18: Same learning order; gap narrows due to harder label structure.
  • ImageNet + ResNet-18: Pattern persists at scale. Amplitude-before-phase is a universal phenomenon across depth and data complexity.
  • Synthetic cyclic model: Phase accuracy at chance for n ≪ N³ (flat spectrum); begins learning at n ~ N with power-law spectrum.
  • DTD textures: Unwhitened textures (natural spectra) show clear amplitude-then-phase. Whitened textures (flat spectra) show much slower phase learning — directly confirming Theorem 3.

Critical Assessment

Novelty High — Fourier decomposition of simplicity bias is new; power-law scaffolding is genuinely surprising
Rigor Strong — Three formal theorems, complete proofs, tight upper/lower bound gap
Reproducibility Good — Cyclic model fully specified; ResNet-18 on standard datasets; Fourier swapping needs care
Significance High — Principled explanation for efficient learning on natural images; implications for data augmentation
  • Two-layer limitation: Theory does not formally cover deep networks or transformers. Whether the learning order persists in ViTs is open.
  • Exact translation-invariance: Cyclic model is an idealization; sensitivity to violations not analyzed.
  • Online vs. epoch-based SGD: Theory assumes fresh samples; connection to practical finite-sample training is mentioned but not rigorous.
  • Phase metric caveat: Randomizing amplitudes changes signal scale, which may interact with nonlinear activations in unexpected ways.
  • No comparison with NTK or mean-field frameworks that might predict similar or different learning orders.

+ Strengths

  • Elegant framework connecting Fourier analysis, statistical physics, and deep learning practice
  • Near-matching upper/lower bounds on sample complexity (only polylog gap)
  • Power-law scaffolding is a genuine, surprising insight with practical implications
  • Extensive validation: synthetic + textures + CIFAR + ImageNet
  • Cyclic data model is a reusable theoretical tool for future work

! Open Questions

  • Does amplitude-before-phase hold in vision transformers (weaker frequency bias)?
  • Can power-law scaffolding inform frequency-domain data augmentation strategies?
  • Extension to multi-channel (color) images and cross-channel Fourier interactions?
  • Does pre-training retain the scaffold during fine-tuning?
  • What happens with adversarial spectra rather than natural power-law?
