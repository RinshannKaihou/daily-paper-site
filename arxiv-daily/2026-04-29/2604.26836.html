Uncertainty-Aware Predictive Safety Filters for Probabilistic Neural Network Dynamics

arXiv: 2604.26836v1

Date: 2026-04-29

https://arxiv.org/abs/2604.26836v1

一句话总结： 本文提出UPSi框架，通过从概率集成（PE）神经网络动力学模型中推导出具有理论保证的可达集过度逼近，并结合显式确定性约束防止模型在高认知不确定性区域被利用，从而在保持与标准MBRL相当性能的同时大幅提升了强化学习探索过程的安全性。
TL;DR: The paper introduces UPSi (Uncertainty-Aware Predictive Safety Filter), the first predictive safety filter that provides provably valid overapproximations of reachable sets from probabilistic ensemble (PE) neural network dynamics models. It uses a Kalman-gain-based certainty constraint to prevent model exploitation in regions of high epistemic uncertainty, and integrates seamlessly into Dyna-style model-based RL (MBPO). Evaluated on Pendulum, Cartpole, and Drone benchmarks, UPSi substantially reduces constraint violations compared to the prior PE-based PSF (XMPSC) while maintaining returns on par with standard MBPO. The main practical limitation is slow MPC solve time, which currently prevents real-time deployment on higher-dimensional systems.

Key Contributions

  • Derives the first provably valid overapproximation of robust reachable sets from probabilistic ensemble neural network models, using a first-order Taylor propagation with explicit linearization error compensation (Eq. 8) and an ellipsoidal outer approximation via Minkowski sums.
  • Introduces a certainty constraint based on the normalized trace of the Kalman gain matrix (Eq. 5), which separates aleatoric from epistemic uncertainty and prevents the PSF from relying on predictions in poorly modeled regions.
  • Proposes a sampling-based terminal set expansion scheme (Eq. 14) inspired by Learning MPC, which monotonically grows the safe operating region across episodes as the model improves.
  • Demonstrates seamless integration with MBPO where UPSi reuses the same PE dynamics model already employed for model-based policy optimization, requiring no additional model training.

Methodology

UPSi operates within the standard predictive safety filtering paradigm but replaces first-principles or GP models with probabilistic ensemble neural networks. The methodology proceeds as follows:

  1. Uncertainty Decomposition: Using ideas from sensor fusion (Covariance Intersection), the method decomposes PE predictions into aleatoric variance (representing true process noise, propagated forward) and epistemic variance (representing model uncertainty, used to define the certainty set). The nominal dynamics are obtained as a covariance-weighted average across ensemble members.
  1. Reachable Set Computation: Starting from the current state, the mean trajectory is propagated through the nominal dynamics (Eq. 7) while the covariance is propagated via linearized dynamics with explicit linearization error compensation (Eq. 8). This compensation inflates the covariance using minimal-trace outer ellipsoids to account for higher-order terms, ensuring the epsilon-level set of the propagated Gaussian overapproximates the true robust reachable sets (Theorem 1).
  1. Certainty Constraint: A Kalman gain matrix G(s,a) quantifies the ratio of aleatoric to total variance. The certain set C is defined as state-action pairs where the average dimension-wise Kalman gain exceeds a threshold xi (Eq. 5). The PSF enforces that the entire reachable tube remains within this certain set (Eq. 10), preventing exploitation of unreliable model predictions.
  1. PSF Optimization Problem: The complete UPSi optimization (Eq. 11) minimizes deviation from the RL action subject to state/action constraints tightened by the reachable set ellipsoids, terminal set containment, and the certainty constraint. It is compatible with standard robust MPC solvers.
  1. Terminal Set Expansion: Between episodes, feasible trajectories are used to expand the terminal set via sampling-based estimation (polytope or ellipsoid), enabling progressive exploration of larger safe regions.

In practice, the authors set all Lipschitz constants to zero (neglecting formal error bounds), use no ancillary controller, and handle infeasibilities via a backup scheme. Despite these simplifications breaking formal guarantees, empirical performance remains strong.

Results & Highlights

  • Reachable Set Validation: On Cartpole with 100,000 ground truth trajectories, UPSi's predicted ellipsoidal tubes consistently contain the true reachable sets, verifying Theorem 1. Even the naive tube (without linearization error compensation) captures the vast majority of trajectories.
  • Safety Improvement: Across three benchmarks (Pendulum/S=2, Cartpole/S=4, Drone/S=12), UPSi achieves substantially fewer accumulated constraint violations compared to XMPSC (Gronauer et al., 2024), the only prior PE-based PSF. The improvement is most pronounced in Cartpole and Drone, where higher dimensionality exacerbates XMPSC's lack of model exploitation prevention.
  • Performance Preservation: UPSi maintains control returns comparable to both MBPO (without safety filtering) and SAC baselines, demonstrating that safety enforcement does not significantly degrade learning performance.
  • Filtering Behavior: UPSi filters more frequently than XMPSC (higher filter rate) but finds feasible solutions more often (lower infeasibility rate), indicating more active but reliable safety intervention.
  • Computational Cost: Control refresh rates reach up to 25 Hz for low-dimensional environments (Pendulum), but real-time feasibility remains elusive for higher-dimensional systems (Drone), identified as the main practical limitation.

The two co-first authors (Frauenknecht and Kesper) build directly on Frauenknecht et al. (2025, ICLR), which established the uncertainty decomposition framework for PE models. The senior author, Sebastian Trimpe, is well-established in the learning-based control and MPC communities. Code is publicly available at https://github.com/Data-Science-in-Mechanical-Engineering/upsi. The paper provides extensive theoretical proofs in the appendix (Lemmas 2-10, Theorem 9, Theorem 11 on persistent feasibility), lending mathematical rigor to the claims.

Critical Assessment

  • Novelty: The paper makes a genuine contribution by being the first to derive provably valid reachable sets from PE models with rigorous uncertainty handling. The certainty constraint via Kalman gain is an elegant solution to the model exploitation problem. However, the practical implementation discards all the formal guarantees (sets Lipschitz constants to zero, no ancillary controller), making the gap between theory and practice noteworthy.
  • Rigor: Theoretical development is thorough with detailed proofs in appendices. The assumptions (2-5) are clearly stated and justified. However, Assumptions 2 and 3 (consistent and unbiased estimator within the certain set) are strong and their empirical validation relies on the authors' prior work (Frauenknecht et al., 2025). The Lipschitz continuity assumption (Assumption 5) for neural networks is technically unbounded but the authors acknowledge this and treat it pragmatically.
  • Reproducibility: Good -- code is publicly available, hyperparameters are provided in tables, and environments are standard Gym benchmarks. The paper clearly states practical simplifications and their consequences.
  • Significance: This work meaningfully bridges the gap between scalable MBRL and rigorous safety filtering. It opens the door to applying PSFs to a much broader class of systems where first-principles models or GPs are impractical. The integration with MBPO is natural and requires no additional model training.
  • Red Flags: (1) The practical version uses zero Lipschitz constants, which formally invalidates all theoretical guarantees. The paper is transparent about this but it means the "rigorous" guarantees are aspirational rather than realized. (2) The computational bottleneck (SQP solver for each time step) limits applicability to real-time systems. (3) Benchmarks are limited to low-to-moderate dimensional systems (up to 12D state); scalability to truly high-dimensional robotics remains unclear. (4) No comparison with GP-based PSF methods, which would have strengthened the evaluation.

Strengths

  • First principled derivation of reachable sets from PE models with formal overapproximation guarantees.
  • Elegant uncertainty decomposition using Kalman gain to separate aleatoric from epistemic uncertainty.
  • The certainty constraint directly addresses the critical failure mode of prior PE-based PSFs (model exploitation under high epistemic uncertainty).
  • Seamless integration with existing MBRL frameworks (MBPO) without additional model requirements.
  • Strong empirical safety improvements while preserving task performance.
  • Transparent discussion of theory-practice gap and computational limitations.

Limitations

  • The practical implementation discards formal guarantees (zero Lipschitz constants, no ancillary feedback), creating a significant theory-practice gap.
  • Computational cost is prohibitive for real-time control of higher-dimensional systems (main acknowledged limitation).
  • Requires an initial data-generating controller and certain set, limiting cold-start applicability.
  • Benchmarks are limited to simulated environments with relatively low dimensionality (max 12D state).
  • No comparison with GP-based PSF methods or other safe RL approaches (CBFs, HJR).
  • The terminal set expansion uses heuristic approximations (k-NN outlier removal, convex hull estimation) that require manual verification.
  • Assumptions about Lipschitz continuity of neural network models are technically unbounded for standard architectures.

Relevance

Questions for Further Exploration

  • How can the computational bottleneck of the SQP solver be addressed to enable real-time feasibility for higher-dimensional systems? Would differentiable MPC or learned surrogates help?
  • Can the formal guarantees be partially recovered through data-driven Lipschitz constant estimation rather than setting them to zero?
  • How does UPSi scale to systems with state dimensions beyond 12D, such as manipulator robots or autonomous vehicles?
  • What is the sensitivity of UPSi's performance to the certainty threshold xi and the noise bound epsilon? Is there a principled way to set these?
  • Could UPSi be extended to handle model non-stationarity (e.g., domain shifts) where the certain set might shrink rather than grow over time?
  • How does the approach compare empirically with GP-based PSFs on systems where both PE and GP models are feasible?
