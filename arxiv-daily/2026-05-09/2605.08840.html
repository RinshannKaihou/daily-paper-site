ReST-KV: Robust KV Cache Eviction with Layer-wise Output Reconstruction and Spatial-Temporal Smoothing

Yongqi An, Chang Lu, Kuan Zhu, Tao Yu, Chaoyang Zhao, Hong Wu, Ming Tang, Jinqiao Wang

Reformulates KV cache eviction as a layer-wise output reconstruction problem. A closed-form indicator captures attention redistribution effects that raw weights miss. Combined with spatial-temporal smoothing, it achieves strong gains across LongBench, RULER, and Needle-in-a-Haystack—with 10.61x decoding speedup at 128k context.

Key Finding
KV cache eviction can be modeled as output reconstruction. The derived indicator It[n] captures attention redistribution—how remaining tokens compensate when one is removed—revealing importance that raw attention weights cannot see.
Key Metric

Why This Matters
KV cache is the core bottleneck for deploying LLMs at long context. This method makes 128k+ inference practical with minimal quality loss and negligible overhead—directly applicable to real deployments.
Critical Edge
Eviction happens only during prefill. No evaluation of dynamic eviction during decoding—a gap for multi-turn dialogue, streaming, and growing-context scenarios where importance must be reassessed continuously.

01 The Core Indicator

Starting from minimizing the reconstruction error of the MHA output when each KV pair is removed, the authors derive a closed-form importance score:

I_t[n] = A_t[n] / (1 - A_t[n]) * ||MHA(x_t, <K_T, V_T>) - v_n * W_O||_2

Two factors: nonlinear reweighting (amplifies high-competition pairs) and redistribution sensitivity (measures whether remaining values can compensate for the removed one). Computed over a small observation window (S_w=32), same complexity as SnapKV.

03 Spatial-Temporal Smoothing

  • EMA temporal smoothing (alpha=0.3): Dampens short-term importance fluctuations over recent query steps within the observation window.
  • Adaptive window spatial smoothing: Splits observation window into front/rear halves, computes average position of top-B KV pairs in each, uses the positional difference to adaptively adjust smoothing window size and shift. Captures vertical-slash attention structures.
  • Recent S_w tokens always preserved (assigned max importance). Eviction performed once during prefill.
