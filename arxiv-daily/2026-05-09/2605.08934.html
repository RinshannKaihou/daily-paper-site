From Mechanistic to Compositional Interpretability

Ward Gauderis, Thomas Dooms, Steven T. Holmer, Kola Ayonrinde, Geraint A. Wiggins

A category-theoretic framework that redefines mechanistic interpretability as commuting syntax–semantics pairs, proving that compressive refinement — the shared abstraction behind SAEs, transcoders, and crosscoders — provably yields more concise explanations when the coding distribution is comonotonic with interpretation complexity.
Key Metric
L = Lrep + Lint
Compositional description length decomposes explanation cost into representation complexity (wiring + component semantics) and interpretation complexity (human-alignment cost), bounding minimal description length (Prop 3.3).
Why This Matters
Gives mechanistic interpretability its first formal mathematical foundation — turning "find sparse features" into a proved optimization problem with explicit faithfulness and complexity tradeoffs.
Critical Edge
Zero empirical validation. The entire framework is theoretical; its central conjecture — the Weak Principle of Explanatory Optimism — remains untested. The distance from string diagrams to practical tools for billion-parameter LLMs is substantial, and coding-distribution subjectivity could undermine the objectivity it seeks.

Methodology

  1. Compositional model (Sec 2). A model M = (D, S, C, [[-]]) consists of a string diagram D in a monoidal category S (syntax), a semantic category C, and a strict monoidal functor [[-]]: S → C. Interpretations require both IS: S → H (syntax → human concepts) and IC: H → C (concepts → semantics) such that the diagram commutes — ensuring component-level semantic reconstruction.
  2. Faithfulness & complexity (Sec 3). Explanation quality is formalized via MDL as a rate-distortion problem. Compositional faithfulness requires d([f], IC(IS(f))) ≤ ε for all morphisms. Complexity is bounded by Lrep(M) + Lint(M, IC).
  3. Compressive refinement (Sec 4). A syntactic functor R: S → S′ that preserves semantics while reducing representation cost. The parsimony criterion (Thm 4.2) gives the exact condition: refinement helps iff the drop in Lrep exceeds any rise in Lint.
  4. Unification (Sec 5). Positions the framework relative to causal abstraction, symmetry-based interpretability, and measurable interpretability, arguing it addresses the antecedent decomposition step that others assume given.

Key Results

The paper is entirely theoretical with no experiments. Its contributions are formal theorems:

Proposition 3.3
Compositional description length upper-bounds MDL, providing a tractable optimization surrogate.
Theorem 4.2 · Parsimony
Refinement improves explanation iff ΔLrep ≥ ΔLint — making the representation vs. interpretation tradeoff explicit.
Corollary 4.4 · Comonotonic
If Prep aligns with interpretability, compressive refinement is guaranteed to help. Operationalizable via Bayesian priors.
Theorem 4.5 · Sparsity
Deep networks of polynomial size efficiently approximate compositionally sparse functions, bounding diagram growth.

Strengths

  • Exceptional conceptual clarity — the four-category model (D, S, C, H) cleanly separates wiring, syntax, semantics, and human concepts.
  • Honest about assumptions: Explanatory Optimism is clearly labeled a conjecture, not a fact.
  • The unification of SAEs, transcoders, and crosscoders as points on a refinement spectrum is genuinely illuminating.
  • Reframes interpretability from heuristics to principled optimization (minimize CDL subject to faithfulness).
  • Constructive rather than dismissive — builds on existing work instead of attacking it.

Limitations

  • No experiments, no case studies, no worked examples on real networks.
  • Gap to practice is large — no scalable algorithms specified; tensor networks are suggested but undeveloped.
  • Explanatory Optimism conjecture is untested; if model regularities are not human-understandable, the guarantee collapses.
  • Coding distribution Prep choice introduces subjectivity the framework aims to eliminate.
  • Heavy category-theory notation creates a significant barrier to entry for practitioners.
  • No discussion of computational complexity for finding optimal refinement.

Open Questions

  1. What concrete algorithms can implement compressive refinement at scale — would a tensor-network-based SAE work?
  2. Can Explanatory Optimism be empirically tested? What would a falsification look like?
  3. How does the framework handle transformer-specific architectures (attention, residual streams)?
  4. Is finding the optimal compressive refinement NP-hard in general?
  5. Can the framework accommodate distributional or approximate compositionality rather than exact commutativity?
  6. What coding distributions Prep best encode human priors (sparsity, hierarchy)?
  7. Can it support multi-level explanations at different abstraction granularities?
  8. How does compositional faithfulness relate to causal scrubbing and other validation methods?
