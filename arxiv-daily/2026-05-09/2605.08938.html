Can We Formally Verify Neural PDE Surrogates?

SMT Compilation of Small Fourier Neural Operators — compiling trained FNOs into Z3 to obtain the first sound formal proofs for neural PDE operators.

Ali Baheri  ·  Ignacio Laguna Peralta
Key Finding
Fixed-weight FNO spectral convolution is a piecewise-linear map, enabling exact compilation into Z3’s linear real arithmetic — yielding the first sound formal proofs for neural PDE operators.
Key Metric
1.1–1.6× more severe counterexamples
Z3 found violations worse than gradient-based falsification on 7 of 10 models. Against Monte Carlo (5K samples): 1.2–2.5× more severe.
Why This Matters
Establishes that neural PDE surrogates are in principle amenable to formal verification and maps the practical soundness–scalability boundary for the first time.
Critical Edge
Exact encoding handles only 85–117 parameters on grids 8–32. ReLU positivity proofs time out due to exponential activation-pattern case splits. Models use random-feature regression, not end-to-end training.

Core Insight

Once the grid and trained weights are fixed, an FNO’s spectral convolution — DFT, mode-wise scaling, IDFT — is a composition of linear maps. An explicit spectral matrix Wspec can be constructed by applying this operation to each basis vector. Combined with ReLU (piecewise-linear), the entire forward pass becomes a piecewise-linear function representable in Z3’s linear real arithmetic without approximation. Validation against the FFT code path shows discrepancies below 10−15.

Two Encodings

  • Exact encoding — compiles spectral matrix into dense matrix multiplication. O(N²H²L) terms. UNSAT = formal proof; SAT = verified real counterexample. Sound for both proofs and counterexamples.
  • Frozen encoding — evaluates spectral path at a reference input and substitutes as constant. O(NH²L) terms. Sub-second checks at N=64. Verifies a frozen surrogate, not the original FNO.

Limitations

  • Production FNOs (N≥128, H≥32) would require spectral matrices of dimension 4096×4096 or larger — far beyond current capability.
  • ReLU positivity proofs time out even at grid size 8–16; fundamental limitation of piecewise-linear encoding with general-purpose SMT.
  • Random-feature regression (frozen hidden layers, least-squares projection) rather than end-to-end training — verified models may not represent deployed FNOs.
  • Only 1D advection-diffusion-reaction tested; 2D/3D would dramatically increase matrix sizes.
  • Frozen encoding gap to original FNO is not bounded (no Lipschitz error analysis).
  • No comparison with specialized verifiers (alpha-beta-CROWN, Marabou, ERAN).

Path Forward

  • Integration with specialized verifiers (alpha-beta-CROWN, Marabou) for ReLU models
  • Lipschitz-based bounds for frozen encoding approximation quality
  • Spatial decomposition for 2D/3D spectral matrices
  • End-to-end trained FNO verification
  • Extension to U-FNO, SFNO, and non-piecewise activations
