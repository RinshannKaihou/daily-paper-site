LBI: Parallel Scan Backpropagation via Latent Bounded Interfaces

Shaun Christopher Lee, Sangeetha Abdu Jyothi
University of California, Irvine
parallel-training backpropagation sequence-models mamba efficient-computing
Key Finding
Restricting backprop inter-region communication to a latent space of r=16 dimensions reduces per-combine Jacobian cost by ~1016x at 1B scale, while preserving exact gradients (cosine similarity > 0.99999).
Key Metric
+0.16 – +0.35 cross-entropy gap vs. dense baselines across 4 architectures
At r=16 with 47–61M params on FineWeb-Edu. Gap is roughly constant for r∈{16,32,64}.
Why This Matters
Transforms sequential backprop depth from O(K) to O(log K) via parallel suffix scan — the algorithmic foundation for region-parallel training of sequence models.
Critical Edge
All experiments are single-GPU at ~60M params. No distributed training demonstrated. Per-step cost is 5–13x slower on one device. Scale-up entirely unproven.

Three-Phase Backward Pass

Phase 1
Jacobian Construction
Each region independently builds Jk ∈ Rr×r via r perturbation directions. Fully parallel across regions. Cost: Θ(r · Fk) per region.
Phase 2
Interface Adjoint Scan
Compose all Jk via parallel suffix scan to compute suffix products Pk, then interface adjoints. The sole sequential bottleneck: Θ(r3 log K) span.
Phase 3
Region-Local Backward
Each region independently computes parameter gradients using the interface adjoints and cached forward activations. Fully parallel across regions.

The canvas mechanism broadcasts the D-dimensional token embedding to every region, so the latent interface only carries inter-region refinements — explaining why small r suffices.

Spectral Stability

Individual region Jacobians have spectral norms 1.3–1.9 (slightly amplifying), but suffix-composed Jacobians converge to spectral norms < 1 (contractive).

For r=64, the spectral norm (1.901) is ~2.5x the RMS singular value (0.749), indicating effective communication concentrates in a low-dimensional subspace.

This contraction explains why training remains stable despite locally expanding maps: composition is a regularizer.

Strengths

  • Principled algorithm with formal proofs and exact gradient preservation (no approximation)
  • Three-phase decomposition cleanly isolates the sequential bottleneck from parallel work
  • Model-agnostic: validated on Mamba-2, Mamba-3, Transformer, and Hybrid without architecture-specific changes
  • Honest about limitations: explicitly acknowledges lack of distributed experiments
  • Spectral analysis provides mechanistic insight into why small interfaces suffice
  • Canvas mechanism elegantly explains the sufficiency of small r

Limitations

  • No distributed training validation — all experiments are single-GPU
  • Scale is very small: ~60M params, 20.48M training tokens, no downstream benchmarks
  • LBI runs are 5–13x slower per-step on a single device
  • Encoder/decoder adds 10–18% parameter overhead without proportional quality benefit
  • No comparison to existing parallelism methods (pipeline, tensor, sequence) on wall-clock time
  • r=16 sufficiency may not hold at larger scales — scaling behavior is untested
  • Missing ablations on how r should scale with model width, depth, or sequence length

Cost Analysis at Scale

At ~1B scale (r=64, d~1.26×107): full-rank scan requires Θ(d3) ≈ 2.0×1021 FLOPs per combine vs. Θ(r3) ≈ 2.6×105 for LBI. The suffix scan itself is negligible (<10-4 of Jacobian construction cost). Jacobian construction increases total arithmetic by (r+1)x over standard reverse-mode AD.

Total compute across all experiments: ~124 H100-hours. Code: github.com/shaunlee8/latent-bounded-interfaces
