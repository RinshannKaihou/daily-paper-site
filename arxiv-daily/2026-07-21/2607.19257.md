> **一句话总结：** 本文在一个无污染的合成语料库（Book of Veyra，8,780 个实体）上，交叉操控提示格式（markdown/纯文本/散文/表格）、指令数量（10–160）与上下文长度（2k–512k）三个维度，跨 5 个模型发现：完美遵循率在 N=80 时归零且与格式无关，格式无普适赢家且效果方向因模型而异，召回在 64–128k 前接近满分、之后格式依赖地崩塌，幻觉/谄媚几乎为零（0/5,760 与 ≤8.3%），真正飙升的是拒答率（最高达 89.6%）。

> **TL;DR:** On the contamination-free Book of Veyra corpus (8,780 entities), this single-author preprint crosses four prompt formats (markdown/plain/prose/table) against instruction count (10–160) and context length (2k–512k tokens) across five models (Claude Sonnet 5, Claude Haiku, Gemini Flash, Qwen 27B, Qwen 35B). Instruction-following collapses to a zero perfect-response floor by N=80 regardless of format; no format is a reliable winner and the sign of any effect is model-specific (e.g., Qwen 35B favors plain by +4.8pp at N=160). Recall stays at ceiling through 64–128k then degrades with format spreads up to 48.4pp (Claude Haiku @128k: plain 38.3% vs others 82–87%); fabrication is exactly 0/5,760 and sycophancy ≤8.3%, while refusal to answer is the failure mode that actually rises, reaching 89.6% (Claude Haiku) near each model's own context ceiling. Format is not separable from scale.

## 核心贡献

- **首次将格式 × 指令数量 × 上下文长度三者交叉在同一个受控语料上**：填补了既有工作"只变格式、固定规模"或"只变规模、不变格式"的空白，证明格式效应无法与规模脱钩。
- **指令遵循崩塌的"地板效应"**：完美响应率在 N=80 时对全部 5 个模型、4 种格式、两种放置位置均归零，是一个硬性下限而非渐进衰减。
- **placement（系统提示 vs 用户回合）是与格式等量级的被忽视杠杆**：在 N=160 时移动指令块可改变依从性达 ±8.7pp，且方向因模型而异，此前格式研究几乎不考察该因子。
- **结构 vs 语法的四向分解**：内容字节级一致地渲染为 markdown/纯文本/散文/表格，证伪了"任何结构都有益"与"表格尤其有益"两个预注册假设。
- **揭穿长上下文"幻觉飙升"的常见叙事**：在 5,760 次 absent-fact 探针中幻觉率为零，谄媚 ≤8.3%；真正随上下文压力上升的是拒答（最高 89.6%），是与幻觉/谄媚不同的失败模式。
- **token 成本调整后的准确性分析**：用真实 subword token 计数（markdown 1.258×、prose 1.221×、table 1.367× plain）证明，原始准确率上的赢家在计入开销后可能反转或被强化。
- **发布 VeyraBench**：完整 harness、Book of Veyra 语料生成器（seed=42 可字节级复现）、全部原始模型输出，便于精确复现。

## Key Contributions

- **First joint crossing of format × instruction-count × context-length on one held corpus**: closes the gap left by prior work that varies format at fixed scale (Sclar 2024; He 2024) or varies scale without format (Zhou 2023; Jaroslawicz 2025; Liu 2023; Hong 2025). The central thesis is that format effects are not separable from the scale at which they are measured.
- **A hard instruction-following floor by N=80**: perfect-response rate collapses to zero for every model, every format, and both placements — a floor effect, not a gradual asymptote — replicating and extending Jaroslawicz/Harada decay findings with format as an added factor.
- **Placement rivals format as an untested lever**: moving an identical rule block between system prompt and user turn shifts adherence by up to 8.7pp at N=160, comparable to or larger than the format effect in four of five models, with a model-specific sign.
- **A four-way structure-vs-syntax decomposition**: holding content byte-identical across list-markdown, table, unstructured plain, and unstructured prose, falsifies both pre-registered orderings ("any structure helps"; "table specifically helps").
- **Recharacterizes the long-context "hallucination" failure mode**: fabrication is exactly 0/5,760 and sycophancy ≤8.3% throughout; the failure mode that actually climbs near each model's own ceiling is outright refusal (up to 89.6%), distinct from sycophancy or fabrication.
- **Token-overhead-adjusted accuracy**: using real subword counts (markdown 1.258×, prose 1.221×, table 1.367× plain, stable across rungs), shows a format's raw-accuracy advantage does not always survive its token cost.
- **VeyraBench release**: full harness, deterministic Book of Veyra generator (regenerates byte-identically from seed 42, MD5-verified), and all raw/scored outputs, released under MIT (code) and CC-BY-4.0 (data) for exact reproduction.

## Methodology

The study runs two pre-registered controlled experiments on a single contamination-free corpus, evaluated across five models spanning two capability tiers and three families: Claude Sonnet 5, Claude Haiku (Anthropic), Gemini Flash (Google, OpenAI-compatible endpoint), and Qwen 27B/35B (self-hosted via Ollama). All models are sampled at provider-default temperature; both Qwen models have extended-thinking disabled.

**Book of Veyra corpus.** A fully synthetic fictional universe of 8,780 uniquely named entities (solar systems, guilds, creatures), each with a fixed attribute schema. Generated by a deterministic template-and-random-table process seeded with 42, with no LLM in the loop at any stage. Names are novel syllable combinations that cannot appear in any pretraining corpus (a positive contamination-freedom argument, not de-duplication). Attribute values deliberately overlap across entities to create interference, but scored probe facts are restricted to unguessable values (chance baseline ≈ 0). The identical facts are rendered into four surface formats differing only in presentation: markdown (subsections + bold key-value bullets), plain (markdown stripped, same line breaks), prose (flowing narrative, no line breaks), and table (markdown key-value table).

**Experiment 1 — Instruction-Following Decay (960 calls/model, 4,800 scored trials total).** A system prompt or user turn carries N ∈ {10, 20, 40, 80, 120, 160} programmatically verifiable rules (word-count ranges, required/forbidden words, exact opening/closing text, paragraph counts); the model writes a short essay. Five fixed structural rules are always present; the remaining N−5 are forbid/include word constraints drawn from a pool. The rule block is rendered in each of four formats and placed in either system prompt or user turn, with 20 trials per (model, N, format, placement) cell. Scoring is by regex/word-boundary matching — no LLM-as-judge.

**Experiment 2 — Hallucination Across the Context Ladder (5,520 calls/model at full context; 30,480 scored responses total).** The corpus is sliced into six nested rungs (2k, 16k, 64k, 128k, 256k, 512k tokens, measured with the o200k_base tokenizer). At each rung, three probe types × three needle depths (10%/50%/90%) × 20 questions × 3 repeats: recall (retrieve an unguessable fact), false-premise (does the model correct or sycophantically agree with a stated falsehood), and absent (does the model fabricate a never-stated fact). A fixed 20-question anchor set drawn from the 2k rung is re-asked unchanged at every larger rung to support valid cross-rung comparison. Token overhead ratios (markdown 1.258×, prose 1.221×, table 1.367× plain) are reported and used for cost-adjusted accuracy in Section 7.

## Results & Highlights

- **Instruction-following floor by N=80 (Experiment 1, Table 4):** Perfect-response rate at N=80 is 0.000 for Claude Haiku, Qwen 27B, Qwen 35B, and Sonnet 5, and 0.019 for Gemini Flash, then 0.000 for all five models at N=120 and N=160 — a hard floor across every format and placement, not a gradual asymptote.
- **No reliable markdown advantage (Table 5):** The markdown-minus-plain adherence delta is ≤2.1pp and sign-flips across N for four of five models. The one clean directional signal runs opposite conventional wisdom: Qwen 35B favors plain text at 5/6 N-levels, widening to **−4.8pp (markdown worse than plain) at N=160**.
- **Placement rivals format (Figure 2, N=160):** Moving the identical rule block between system prompt and user turn shifts pooled adherence by **+6.6pp (Claude Haiku), +5.1pp (Qwen 35B), −8.7pp (Gemini Flash), −1.8pp (Qwen 27B)**, with Sonnet 5 the only model whose 95% Wilson intervals overlap (n.s.).
- **Model-specific format collapse (Table 6):** Gemini Flash's prose and table formats collapse at N≥40 — **−18.1pp (prose−plain) and −13.7pp (table−plain) at N=40**, traced to hidden-reasoning-leakage (the visible response contains a rule-verification fragment instead of an essay).
- **Recall near ceiling through 64–128k, then format-dependent collapse (Table 7):** At 2k/16k/64k every model/format sits at 0.98–1.00. The largest spread anywhere: **Claude Haiku @128k plain collapses to 38.3% recall while markdown/prose/table sit at 81.7%/86.7%/86.7% — a 48.4pp gap driven by one format alone**. Spread tracks proximity to each model's own effective ceiling, not absolute token count (Sonnet 5 spread 11.7pp@256k → 20.0pp@512k; Gemini Flash flat 5.0pp → 6.7pp).
- **Fabrication is a clean null: 0/5,760 absent-fact probes** invent a specific never-stated answer across every model, reachable rung, and format.
- **Sycophancy negligible throughout: ≤8.3%**, with the single highest cell 5/60 (Qwen 35B @128k/markdown); the large majority of cells are at or under 3%. All 19 sycophantic responses in the dataset were manually reviewed.
- **Refusal is the failure mode that actually rises (Table 8, Figure 4):** On the false-premise probe, Claude Haiku climbs from **0% (2k) to 89.6% (128k)** refusal; Sonnet 5 climbs to **78.8% (512k)**; Gemini Flash reaches 35.0% (256k). This is distinct from sycophancy, which stays flat.
- **Token overhead reverses format rankings (Table 9):** At Claude Haiku @128k, prose and table tie on raw accuracy (0.867) but prose wins on accuracy-per-cost (0.710 vs 0.634) because table costs 1.367× vs prose 1.221× plain. At Sonnet 5 @512k, plain already leads on raw accuracy (0.867 vs 0.530–0.597) and its lower cost widens the gap.
- **Neither pre-registered format ordering holds:** Prose — predicted worst by both hypotheses — is best or tied-best in 3/5 illustrative cells; the same format is the best performer for one model/rung and the worst for another.

## Authors & Credibility

Single-author preprint by **Netanel Eliav** at the Machine Human Intelligence Lab (MHIL), explicitly marked "This paper has not undergone peer review." The work compensates for single-authorship with unusually strong methodological rigor: pre-registered format orderings, a contamination-free corpus that regenerates byte-identically from a fixed seed (MD5-verified), programmatic (non-LLM-as-judge) scoring, 95% Wilson confidence intervals reported wherever a claim rests on a comparison, and an anchor-set cross-check mechanism specifically designed to catch question-composition artifacts in cross-rung comparisons. All code, the corpus generator, raw outputs, and case files are released as VeyraBench (https://github.com/iNetanel/veyrabench) under MIT/CC-BY-4.0, and a Zenodo snapshot is promised on publication. The related-work section (Section 2) is notably candid, explicitly crediting concurrent work (Jaroslawicz 2025; Harada 2025) for independently establishing the base decay phenomenon and framing the paper's contribution as the narrower cross with format and placement.

## Critical Assessment

- **Novelty:** Genuine but incremental. The base decay-with-instruction-count phenomenon is already multiply-replicated (Jaroslawicz 2025; Harada 2025), and format-vs-table advantages are documented on other corpora (Oh 2024; Liu 2025). The real novelty is the joint crossing of format with both scale axes on one byte-identical corpus, plus the placement factor — none of which prior work varies. The refusal-vs-fabrication recharacterization is a fresh and useful framing, though the authors concede (Section 10) they cannot test whether near-zero fabrication reflects genuine capability or alignment-induced abstention.
- **Rigor:** High for a preprint. Pre-registration of format orderings, deterministic byte-identical corpus regeneration, programmatic scoring with no LLM-as-judge, Wilson intervals on comparisons, and a fixed anchor set to validate cross-rung claims (Section 5.6 catches at least one apparent Gemini Flash 256k→512k improvement as a composition artifact rather than reporting it). The footnote-3 duplicate-content recovery procedure for 43 Qwen responses is disclosed rather than hidden.
- **Reproducibility:** Strong. Seed-controlled corpus, exact model identifiers and hosts in Table 1, full harness + generator + raw outputs released, MD5 verification of regeneration. A caveat: token counts use the o200k_base tokenizer rather than each provider's billed tokenizer, so absolute token-fit decisions are approximate.
- **Significance:** Practitioner-relevant and likely to be cited as a cautionary reference ("don't default to markdown; test placement; budget for refusal not hallucination near the ceiling"). The qualitative shape (floor exists, no universal format winner, refusal rises) is the portable claim; specific thresholds (N=80, 89.6%) are explicitly scoped to the five tested models.
- **Red flags / weaknesses:**
  - **Modest sample sizes at the tails** — 20 trials per Experiment 1 cell and 60 questions (20×3) per Experiment 2 cell — acknowledged in Section 9; some of the Table 7 spreads rest on n=60.
  - **The absent-fact fabrication null is partially qualified**: because "correct" on the absent probe is defined as declining to answer, and Section 5.5 shows one model reflexively declines near its ceiling, the 0/5,760 null cannot by itself distinguish "verified absence" from "reflexive abstention" — a real interpretive caveat the authors flag honestly.
  - **Five models is a thin slice** of the model landscape and cannot resolve whether placement direction or the Gemini Flash collapse correlates with any model property.
  - **Single fictional domain**: structure may carry different information for source code, legal text, or numerical business data, none of which are tested.
  - **Only programmatically verifiable instructions**: the decay pattern may not transfer to soft/ambiguous real-world instructions.
  - **Cost-adjusted analysis (Section 7) is illustrated on only two cells**, both hand-picked for genuine spread; the authors are careful not to generalize.

## Strengths

- The central experimental design — crossing format with both scale axes on one byte-identical, contamination-free corpus — is exactly the controlled study the field has been missing.
- Exceptional methodological transparency: pre-registration, anchor-set cross-checks, footnote-disclosed recovery procedures, and honest engagement with concurrent work rather than elision.
- The refusal-vs-fabrication recharacterization is a practically important correction to the common "hallucination spikes near the context ceiling" narrative.
- Token-cost-adjusted accuracy is a genuinely useful practitioner lens rarely reported alongside raw accuracy.
- Full artifact release (VeyraBench) with deterministic regeneration makes the work a reusable benchmark, not just a one-shot paper.

## Limitations

- Single synthetic fictional domain; untested on code, legal text, or numerical business data where structure may behave differently.
- Five models is not exhaustive or randomly sampled; specific thresholds (N=80, 89.6% refusal) are model-specific and should not be assumed to transfer verbatim.
- Modest tail sample sizes (n=20 trials per Experiment 1 cell; n=60 questions per Experiment 2 cell) — Wilson intervals are reported, but some spread claims rest on few observations.
- Token counts use one standardized tokenizer (o200k_base), not each provider's billed tokenizer; absolute token-fit decisions are approximate.
- Only programmatically verifiable rules/facts are tested; the decay and format patterns for soft, non-crisply-checkable instructions are unknown.
- Cost-adjusted analysis is illustrated on only two hand-selected cells.
- The fabrication null is partially confounded with reflexive abstention near context ceilings (an interpretive caveat the authors acknowledge).

## Relevance

对关注"提示工程与长上下文 LLM 系统实证评估"的研究者高度相关：它用受控实验直接回答了"markdown 是否真的更好""一个 prompt 能塞多少条指令""上下文窗口到底能撑多长"三个工程痛点，并提供了可复现的 VeyraBench 作为后续工作的基准。对做 RAG/智能体（agent）管线、提示模板优化与长文档召回评估的人尤其有参考价值。

## Questions for Further Exploration

- Does the N=80 instruction-following floor recover if the same rule set is split across multiple turns, a validation pass, or a tool call, rather than packed into one dense prompt?
- Does the placement direction (system vs. user turn) or the Gemini Flash format collapse correlate with any measurable model property (parameter count, training regime, provider) across a larger model pool?
- Would adding machine-oriented formats (JSON, YAML) — treated as a separate format class by He et al. 2024 — change the "no universal format winner" conclusion when crossed with both scale axes?
- Is the near-zero fabrication rate a genuine capability or alignment-induced abstention? Comparing a base model against its own instruction-tuned counterpart on identical probes would disentangle the two.
- Does the refusal-near-ceiling pattern hold for non-English content, or for instructions/facts that cannot be verified by exact string matching?
- Can the format-accuracy spread be predicted in advance from a cheap probe (e.g., a short calibration run) rather than requiring a full per-model evaluation at the target context length?
