Prompt Design at Scale: How Format, Instruction Count, and Context Length Shape Instruction Adherence and Hallucination in Large Language Models

Netanel Eliav · Machine Human Intelligence Lab (MHIL) · 2026-07-21 · VeyraBench · Book of Veyra
prompt-engineering long-context instruction-following hallucination empirical-benchmark 5 models · 35,280 trials
Key finding

Instruction-following hits a hard zero floor by N=80, no format is a reliable winner, and the failure mode that climbs near the context ceiling is refusal — not hallucination.

Key metric · fabrication vs refusal
0/5,760
absent-fact fabrications  ⟷  refusal up to 89.6%

Across every model, reachable rung, and format, not a single absent-fact probe invented an answer — while Claude Haiku's refusal on the false-premise probe climbs from 0% at 2k to 89.6% near its own context ceiling.

Why this matters

Directly answers three engineering pain points with controlled evidence and a reproducible benchmark: does markdown actually win, how many rules can one prompt hold, and how far does a context window really stretch — before recall collapses.

audience: RAG · agents · prompt-template & long-doc recall evaluation
Critical edge

One synthetic fictional domain and five hand-picked models — the specific thresholds (N=80, 89.6%) are explicitly model-specific and should not be assumed to transfer verbatim.

untested: code · legal text · numerical business data

M / 01 Methodology

two pre-registered controlled experiments · one contamination-free corpus · five models

Two pre-registered experiments run on a single contamination-free corpus, evaluated across five models spanning two capability tiers and three families: Claude Sonnet 5, Claude Haiku (Anthropic), Gemini Flash (Google), and Qwen 27B / 35B (self-hosted via Ollama). Provider-default temperature; extended-thinking disabled on both Qwen models.

Book of Veyra. A fully synthetic universe of 8,780 uniquely named entities (solar systems, guilds, creatures), each with a fixed attribute schema. Generated by a deterministic template-and-random-table process seeded with 42, with no LLM in the loop at any stage. Names are novel syllable combinations that cannot appear in any pretraining corpus. The identical facts are rendered into four surface formats differing only in presentation: markdown, plain, prose, table.

Exp. 1 — Instruction-Following Decay (4,800 scored trials). A system prompt or user turn carries N ∈ {10,20,40,80,120,160} programmatically verifiable rules; the model writes a short essay. 20 trials per (model, N, format, placement) cell. Scored by regex / word-boundary matching — no LLM-as-judge.

Exp. 2 — Hallucination Across the Context Ladder (30,480 scored responses). Corpus sliced into six nested rungs (2k · 16k · 64k · 128k · 256k · 512k). At each rung: three probe types × three needle depths (10/50/90%) × 20 questions × 3 repeats. A fixed 20-question anchor set from the 2k rung is re-asked unchanged at every larger rung for valid cross-rung comparison.

pre-registered format orderings seed=42 · MD5-verified no LLM-as-judge 95% Wilson intervals anchor-set cross-check placement: system vs user turn

S / 03 Strengths

  • Exactly the controlled study the field was missing — crossing format with both scale axes on one byte-identical, contamination-free corpus.
  • Exceptional transparency — pre-registration, anchor-set cross-checks, footnote-disclosed recovery procedures, candid credit to concurrent work.
  • Refusal-vs-fabrication recharacterization corrects the common "hallucination spikes near the context ceiling" narrative.
  • Token-cost-adjusted accuracy — a practitioner lens rarely reported alongside raw accuracy.
  • VeyraBench release with deterministic regeneration makes it a reusable benchmark, not a one-shot paper.

L / 04 Limitations

  • Single synthetic fictional domain — untested on code, legal text, or numerical business data where structure may behave differently.
  • Five models, not randomly sampled — thresholds (N=80, 89.6%) are model-specific; should not transfer verbatim.
  • Modest tail sample sizes — n=20 per Exp.1 cell, n=60 per Exp.2 cell; some spread claims rest on few observations.
  • One standardized tokenizer (o200k_base), not each provider's billed tokenizer; absolute token-fit is approximate.
  • Only programmatically verifiable rules tested — decay for soft / ambiguous real-world instructions unknown.
  • Cost-adjusted analysis illustrated on two hand-selected cells; authors are careful not to generalize.
  • Fabrication null partially confounded with reflexive abstention near ceilings (acknowledged).

Q / 05 Questions for Further Exploration

  1. Does the N=80 floor recover if the same rule set is split across multiple turns, a validation pass, or a tool call — rather than packed into one dense prompt?
  2. Does the placement direction (system vs user turn) or the Gemini Flash format collapse correlate with any measurable model property across a larger model pool?
  3. Would adding machine-oriented formats (JSON, YAML) change the "no universal format winner" conclusion when crossed with both scale axes?
  4. Is the near-zero fabrication rate a genuine capability or alignment-induced abstention? Comparing a base model against its instruction-tuned twin would disentangle the two.
  5. Does the refusal-near-ceiling pattern hold for non-English content, or for instructions/facts not verifiable by exact string matching?
  6. Can the format-accuracy spread be predicted in advance from a cheap calibration probe, rather than requiring a full per-model evaluation at the target context length?
