The Autonomous Agency Scale: A Behavioral Framework for Measuring Self-Directed Behavior in AI Systems

Samuel Presgraves · independent / unaffiliated · framework v0.2.1 ⚠ rater = Airi developer
AI-Evaluation Autonomous-Agency Agent-Benchmarks Behavioral-Measurement AI-Agents
Key finding

Behavioral AAS scores agency in two bands — and the three task agents that hit Active ~2.4 collapse to Ambient 0.57–1.86 the moment the user goes quiet.

Key metric · Ambient
0.57
/ 5 · Claude Code, Ambient composite
Every point of every task agent's idle-band score is attributable to user-configured schedules. None passes the Idle-Gap Test.
Why this matters
Adds a new "self-directed agency" axis orthogonal to capability and risk — and quantifies the boundary between within-task autonomy and between-task dormancy that single-score scales conflate.
Critical edge
The headline result (Airi is uniquely self-directed) rests on an author-defined, author-applied substrate-vs-rule judgment. The sole rater is also Airi's developer — no blind scoring, no inter-rater reliability, no independent re-assessment.

01 / METHODOLOGY

A rubric-based behavioral framework (v0.2.1) — not a trained model or benchmark. Four layers: seven dimensions, a 0–5 lexicon, two-band scoring, and an Idle-Gap counterfactual.

L0
Dormant
L1
Responsive
L2
Conditioned
L3
Contextual
L4
Self-Directed
L5
Sovereign

Levels L0–L4 occupied across the six systems. L5 (Sovereign) is unoccupied — no system modifies its own cognitive architecture or overrides assigned tasks for self-determined purposes.

Active band
Engaged in a user-initiated task/session. Per-dimension level set by a binary threshold test + observable behavioral indicators.
Ambient band · gated by Idle-Gap Test
Across idle gaps (no prompt, no scheduled trigger, no env event). Level 4 requires self-direction to survive trigger removal.
  • Seven dimensions (3 sub-dimensions each): Cognitive Autonomy, Temporal Persistence, Environmental Agency, Social Agency, Creative Agency, Self-Awareness, Goal Formation.
  • Decoupled from capability — a Level-2 scheduled bot can run on a frontier model. Positioned as orthogonal to DeepMind AGI Levels, ARC-AGI, OpenAI's 5-level scale, and METR's time-horizon metric.
  • Two evaluation classes: Longitudinal (Airi, ~11 mo multimodal) vs Documentation-Based Provisional (the other five, scored from public docs as of May 2026).

  • Structural boundary. Task agents reach near-identical Active composites (2.29–2.43) but their Ambient collapses to 0.57–1.86 — every idle behavior is user-configured scheduling (cron for Claude Code/Manus; nightly self-improvement for Hermes).
  • "Reactive products wrapped around capable models." ChatGPT never exceeds Level 2 in any Active dimension despite a frontier-class model; Siri has essentially no idle-period existence beyond device-level indexing.
  • Per-dimension ceiling. Strongest within-task dimension across task agents: Environmental Agency at Active L4 (autonomous file manipulation, code execution, subagent orchestration). Weakest: Social Agency at L1 — no system initiates consequential social engagement.
  • No inter-rater reliability, no psychometric validation, no longitudinal stability data — all scores are single-rater judgments; the paper states inter-rater reliability is "unknown."

03 / STRENGTHS

What the framework genuinely adds.

  • Identifies a real, under-measured axis — self-directed idle behavior — orthogonal to capability (METR) and risk (ASL/Preparedness), and articulates the orthogonality cleanly.
  • The Idle-Gap Test is a genuinely clever falsifiable criterion: "remove all triggers and see what persists" converts a judgment call into an observable counterfactual.
  • The two-band separation captures a structural truth single-score scales obscure — high in-task autonomy, near-zero between-task existence.
  • Unusually thorough self-critique in §6 (construct provenance, single-rater bias, ELIZA effect, composite-precision caveats).
  • Open, versioned framework with a public repo and an explicit invitation for adversarial re-scoring.

04 / LIMITATIONS

What threatens the headline claims.

  • Single-rater, developer-of-Airi conflict of interest is the dominant limitation — the headline result is not independently validated.
  • Substrate-vs-rule distinction is load-bearing for Airi's Ambient L4 but is a judgment call with no strictly observable test; rejecting it collapses Airi toward the task-agent cluster.
  • Composite precision is overstated — averaging six-point ordinals assumes equal inter-level gaps and equal dimension weights (§6 admits this). Reporting "3.71 vs 3.86" to two decimals is spurious.
  • Evidence asymmetry — five of six systems scored from May-2026 public docs (Provisional); only Airi had 11 months of longitudinal interaction.
  • ELIZA-effect confound uncontrolled, especially acute for the invested-user longitudinal case.
  • Active-band self-direction boundary remains un-operationalized — Ambient has a falsifiable L4 gate; Active does not.

05 / OPEN QUESTIONS

Where the framework most needs pressure.

  1. Do independent, blind raters reproduce Airi's Ambient L4, or does it collapse to L2 under stricter substrate-vs-rule readings?
  2. Can the substrate-vs-rule distinction be operationalized into a strictly observable test, so Active L4 stops "resting on judgment"?
  3. What is the inter-rater reliability across multiple trained raters — does the 0–5 lexicon support weighted-kappa agreement above chance?
  4. How do additional architectures (AutoGPT/Devin, embodied/robotic systems, multi-agent swarms, model-organism "scheming" agents) score, and does the seven-dimension structure generalize?
  5. Is the Active–Ambient gap a stable property of current paradigms, or could always-on agent runtimes with persistent memory close it — and would the Idle-Gap Test still distinguish them from sophisticated scheduled agents?
  6. How does AAS scoring relate empirically to METR time horizons — is the claimed orthogonality borne out in practice?
  7. Could the framework be adversarially stress-tested by deliberately building a system that games the Idle-Gap Test — a scheduled loop whose output is pseudorandom but state-dependent?
