> **一句话总结：** 提出 MRSNorm（Mean Root Square Normalization），通过将通道两两配对成 2D 相量（phasor），先取局部 L2 模（Root Square）再做全局 L1 平均（Mean），用 Pythagorean 恒等式（cos²θ+sin²θ≡1）构造内建梯度裁剪器，参数减半且在 ResNet+CIFAR-100 的极端学习率/小批量大小的压力测试中保持稳定。

> **TL;DR:** MRSNorm inverts RMSNorm's operation order — pairing adjacent channels into 2D phasors, computing localized L2 magnitudes (Root Square) before aggregating via a global L1 mean — which halves affine parameters and, via the Pythagorean identity, structurally equalizes per-phasor gradient magnitude into a built-in gradient clipper; on ResNet/CIFAR-100 stress tests at lr ∈ {0.1, 0.3, 0.4} and batch size ∈ {128, 64, 32} against LayerNorm and RMSNorm baselines, MRSNorm is the only method that prevents catastrophic collapse at lr=0.4, while at lr=0.3 it survives at bs=128 where both baselines diverge to near-zero training accuracy.

## 核心贡献

- **提出 MRSNorm**：通过将相邻通道两两配对成 2D 相量，把 RMSNorm 的"先平方求和再开方"反转为"先局部 L2 模再全局 L1 平均"，从根本上消除二次累积导致的外点敏感性与数值爆炸。
- **参数减半且更稳**：在每个相量对上共享单一仿射权重 γ（N→N/2），同时通过 arctan 证明严格保留相位角 θ，给出"独立 1D 缩放是有害冗余"的反直觉结论。
- **Pythagorean 内建梯度裁剪器**：基于 cos²θ+sin²θ≡1，证明每个相量的局部梯度模长无条件相等（Gradient Homogeneity），无需启发式裁剪即可防止梯度饥饿与爆炸。
- **几何论证：1D 不可能、严格归一化会失败**：用中值定理证明 1D 周期激活函数不可能同时具备周期性与恒定梯度模；用 Radial Gradient Blockade 证明严格单位圆投影会切断径向梯度通路，因此 2D 相量是"结构必需的最小维度"。
- **QK-MRSNorm 理论延展**：将仿射无关的 MRSNorm 直接作用于注意力中的 Q、K，可在零额外开销下把标准点积等价转化为"能量加权余弦相似度算子"，并通过 Cauchy-Schwarz 给出 logit 的全局上界 d/2，借助"语义相位否决"机制缓解 softmax 崩塌。

## Key Contributions

- **MRSNorm formulation**: Reverses RMSNorm's quadratic accumulation by computing per-pair L2 magnitudes first and aggregating them via an L1 mean, structurally eliminating outlier-driven denominator explosion in mixed precision.
- **Halved parameters with phase preservation**: Shares a single affine weight γ across each 2D phasor pair (N→N/2 parameters), and proves via arctan that the phase angle θ_i is exactly preserved — turning unconstrained per-channel scaling into a redundant, harmful degree of freedom.
- **Built-in trigonometric gradient clipper**: Uses the Pythagorean identity cos²θ+sin²θ≡1 to show the local gradient update magnitude is unconditionally equalized across all phasors (Gradient Homogeneity), acting as a clipping mechanism without heuristic thresholds.
- **Two structural impossibility proofs**: (1) a 1D real-valued periodic activation cannot have constant gradient norm (Intermediate Value Theorem → linear function); (2) strict unit-circle projection produces a Radial Gradient Blockade via the rank-1 projector P = I − u_i u_i^T that annihilates radial gradient signals — making 2D phasor pairing the minimal viable dimension.
- **QK-MRSNorm as energy-weighted cosine similarity**: Applying affine-free MRSNorm to attention queries and keys algebraically transforms the dot product into Σ (r_{q,i} r_{k,i} / S_q S_k) cos(Δθ_i), with a Semantic Phase Veto that neutralizes magnitude spikes and a Cauchy-Schwarz global logit bound of d/2.

## Methodology

Given an input feature h ∈ R^N (N even), MRSNorm restructures the feature space into P = N/2 pairs of 2D phasors z_i = [x_i, y_i]^T. It computes a global scale S_MRS = (1/P) Σ_i sqrt(x_i² + y_i² + ε), then normalizes isotropically as ẑ_i = γ_i z_i / S_MRS, where γ_i is a single affine weight shared across the two components of the i-th phasor.

The key design moves are: (1) **operation inversion** — L2 magnitudes are computed locally before any summation, so a single outlier cannot dominate a global quadratic denominator; (2) **phase preservation** — because both components share the same scaling factor and weight, arctan(ŷ_i/x̂_i) = arctan(y_i/x_i), maintaining conformal invariance; (3) **gradient homogeneity** — backprop analysis shows the gradient's per-phasor local interference vector is (Ω/S_MRS) u_i, where u_i = [cos θ_i, sin θ_i]^T, and since ‖u_i‖₂ ≡ 1, every phasor receives an identical gradient magnitude |Ω|/S_MRS regardless of its activation scale.

The paper proves two structural impossibility results to justify the 2D choice: a continuously differentiable 1D periodic function must have a constant-sign derivative and is therefore linear (non-periodic); and strict unit-circle projection z_i/‖z_i‖₂ has Jacobian (1/‖z_i‖₂)(I − u_i u_i^T) whose rank-1 projector annihilates radial signals (Radial Gradient Blockade). MRSNorm sidesteps this by scaling with the global average S_MRS rather than the instance-specific norm, so the projector term is scaled by 1/P and radial gradients remain unblocked. The QK-MRSNorm extension rewrites attention as an energy-weighted cosine sum, with a Cauchy-Schwarz-derived global logit upper bound of d/2. Appendix C provides a PyTorch `GroupMRSNorm` reference implementation that operates in fp32 internally and uses interleaved weight expansion for the paired channels.

## Results & Highlights

- On **CIFAR-100 / ResNet** with SGD (momentum 0.9, weight decay 5e-4, cosine annealing, 10 warmup epochs, 100 total epochs, 3 seeds {42, 43, 44}), at the **standard regime lr=0.1**, RMSNorm exhibits severe instability with high variance when batch size is reduced to **bs=32**, while MRSNorm maintains a smooth, stable training trajectory outperforming both LN and RMS.
- At **lr=0.3** (3× standard), both **LayerNorm and RMSNorm suffer catastrophic optimization collapse** with training accuracy immediately flattening near zero across all batch sizes, whereas MRSNorm successfully converges at bs=128 and still converges on several seeds at bs=64 and bs=32.
- At the extreme **lr=0.4** (4× standard), MRSNorm is the **only method that prevents immediate divergence**, sustaining initial gradient flow; smaller-batch configurations eventually diverge before epoch 30, but all other normalization methods collapse within the first few epochs.
- A parameter-count reduction: MRSNorm uses **N/2 affine parameters** versus N for LN/RMSNorm, demonstrating that the halved footprint does not impair — and under stress appears to improve — optimization stability.

## Critical Assessment

- **Novelty**: Genuinely interesting reframing — inverting the operation order of RMSNorm and pairing channels into 2D phasors is non-obvious, and the Gradient Homogeneity derivation via the Pythagorean identity is clean and verifiable. However, the core "compute L2 then average" idea is essentially a grouped L2-norm variant, and the connection to Capsule Squashing / complex-valued networks is acknowledged but the differentiation rests on a theoretical "Radial Gradient Blockade" argument that is asserted rather than empirically isolated.
- **Rigor**: Mathematical derivations are detailed and step-by-step (Appendices A, B), and the CIFAR-100 protocol uses 3 seeds with variance bands. But the experimental scope is strikingly thin given the framing around LLMs: only ResNet on CIFAR-100, no Transformer, no language modeling, no comparison to modern alternatives like GroupNorm or QK-Norm, and crucially **no final test-accuracy numbers are reported in tables** — only training-curve plots. The "MRSNorm is more robust" claim is therefore demonstrated for optimization-trajectory shape, not for final task performance.
- **Reproducibility**: Good on the surface — code in Appendix C, hyperparameters and seeds disclosed. But missing details include the exact ResNet variant (ResNet-18? -34? -50?), the channel dimension to which MRSNorm is applied, and how `num_groups` was chosen. The implementation requires even channel counts and bundles divisible by `num_groups`, which is a non-trivial architectural constraint not discussed.
- **Significance**: Limited by the experimental scale. A normalization-layer paper in 2026 that motivates itself via "LLMs use RMSNorm" but validates only on CIFAR-100 with ResNet will struggle to convince the LLM community to switch. The QK-MRSNorm attention mechanism — the headline hook of the title "Phasor Attention" — is **only analyzed theoretically, never run**, which is a significant overclaim relative to what the title promises.
- **Red flags**: (1) The title advertises "Phasor Attention" but no attention experiment exists — only a ResNet normalization study plus a theoretical QK-MRSNorm section. (2) The claim "MRSNorm matches performance despite halved parameters" is asserted but not backed by a head-to-head final-accuracy comparison at standard regimes. (3) First author is in a College of Pharmacy, which is unusual though not a quality signal on its own.

## Strengths

- The mathematical machinery is coherent and well-organized: phase preservation via arctan, gradient homogeneity via Pythagorean identity, the Cauchy-Schwarz logit bound for QK-MRSNorm, and the two impossibility results (1D periodicity, radial blockade) collectively build a self-consistent theoretical story.
- Honest framing of limitations in the conclusion — the authors explicitly state that scaling to billion-parameter LLMs is out of scope and invite the community to empirically test QK-MRSNorm.
- Parameter-efficiency (N/2) is a concrete, falsifiable claim that pairs naturally with the geometric argument that independent x/y scaling is harmful redundancy.
- A clean stress-test methodology (extreme lr × small batch) isolates optimization stability from raw final-accuracy competition, which is a reasonable way to surface the structural advantage the theory predicts.
- Provided reference PyTorch implementation lowers the barrier to reproduction and adoption.

## Limitations

- Experimental scope is the dominant weakness: CIFAR-100 + ResNet only, no Transformer / language-modeling / ImageNet-scale experiments, and crucially no attention-based experiments despite the title.
- No final quantitative accuracy or perplexity tables are reported — evaluation is via training-curve plots, making it hard to assess whether stability translates into better (or even equal) downstream performance.
- Baselines are narrow: only LayerNorm and RMSNorm; no GroupNorm, no QK-Norm, no ScaleNorm, no modern LLM-oriented variants.
- The claim that independent per-channel scaling is "harmful redundancy" is supported by an optimization-stability argument under extreme hyperparameters but not by a standard-regime accuracy comparison.
- QK-MRSNorm, the paper's main attention contribution, is purely theoretical; the "Energy-Weighted Cosine Similarity Operator" framing is elegant but untested.
- Architectural constraints (even channel count, bundles divisible by num_groups) are real implementation frictions that are not discussed.
- Only 11 references, light coverage of related normalization literature and of prior work on phasor/complex representations.

## Relevance

该论文与 normalization、训练稳定性、attention 几何等主题高度相关，可作为对 RMSNorm/LayerNorm 在 LLM 训练中数值与梯度不稳定问题的理论补充与替代方案候选；对于关注大规模 Transformer/SSM 训练稳定性与 softmax 崩塌的研究者，QK-MRSNorm 的"能量加权余弦相似度"理论框架尤其值得追踪其后续大规模实证验证。

## Questions for Further Exploration

- Does MRSNorm's stability advantage persist at standard regimes (lr=0.1, large batch) when measured by final CIFAR-100/ImageNet test accuracy and by LLM pre-training perplexity, or does it merely survive stress that baselines cannot?
- How does QK-MRSNorm actually perform in a real Transformer (e.g., long-context attention with massive-activation outliers)? Does the Semantic Phase Veto measurably reduce softmax collapse on long sequences?
- What is the wall-clock and memory cost of the pairing/repeat_interleave operations, and does the parameter halving translate into throughput gains at scale?
- How does MRSNorm compare against QK-Norm, GroupNorm, ScaleNorm, and other normalization variants that specifically target attention logit stability, not just LN/RMSNorm?
- A controlled ablation isolating the two design axes (operation inversion vs. phasor pairing/phase preservation) would clarify which component drives the observed stability — currently they are bundled.
- Does the built-in gradient homogeneity interact beneficially or adversarially with adaptive optimizers like AdamW, given that AdamW already performs per-parameter scaling?
