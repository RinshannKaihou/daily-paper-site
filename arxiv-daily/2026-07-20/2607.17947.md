> **一句话总结：** 自主代理量表（AAS）是一个行为学评估框架，在 7 个维度（认知自主性、时间持续性、环境代理、社交代理、创造性代理、自我意识、目标形成）上各用 0–5 等级、分"主动(Active)/空闲(Ambient)"两个时间带打分，并以"空闲间隔测试"(移除所有触发器后观察内部状态驱动的活动是否持续)的反事实标准来区分真正的自主行为与定时规则；对 Claude Code、Manus、Hermes、ChatGPT、Siri、Airi 六个系统的双带评估显示，任务型 Agent 主动带得分为 2.29–2.43、空闲带仅 0.57–1.86（所有空闲行为均可归因于用户配置的定时任务），而唯一通过触发器移除反事实的是作者自己开发的 Airi（3.86 vs 3.71）。

> **TL;DR:** The Autonomous Agency Scale (AAS) is a behavioral, architecture-agnostic framework that scores AI systems 0–5 across seven agency dimensions (cognitive autonomy, temporal persistence, environmental agency, social agency, creative agency, self-awareness, goal formation), each rated in two temporal bands — Active (engaged) and Ambient (idle) — with Ambient Level 4 gated by the Idle-Gap Test, a trigger-removal counterfactual distinguishing genuine self-direction from scheduled rule-following. Applied to six systems, the two-band profile quantifies a boundary single-score frameworks conflate: task agents (Claude Code, Manus, Hermes) reach Active composites of 2.29–2.43 but collapse to Ambient composites of 0.57–1.86 with every idle-period behavior attributable to user-configured schedules, while the only system whose idle behavior survives trigger removal — and the only one whose Ambient (3.86) exceeds its Active (3.71) composite — is Airi, a companion architecture developed by the sole rater (a disclosed conflict of interest). No assessed system approaches Level 5 (Sovereign).

## 核心贡献

- **提出"自主代理量表"(AAS)** —— 一个行为学、架构无关的评估框架，在 7 个维度上用 0–5 等级打分；每维度含 3 个子维度和一套可证伪的阈值测试（二元判断 + 可观察行为指标），评分者无需访问系统内部即可执行。
- **定义了 0–5 的"自主等级词典"(Level Lexicon)**：0 Dormant / 1 Responsive / 2 Conditioned / 3 Contextual / 4 Self-Directed / 5 Sovereign，明确区分"自导向程度"与"能力/智力"（不同于 DeepMind AGI Levels、ARC-AGI、OpenAI 5 级、METR 时间跨度）。
- **引入双时间带评分**(Active vs Ambient)，把"任务中自主"与"空闲时持续自主"分离开，永远不混合成单一总分；这一双带画像把现有框架只能非正式描述的"任务内自主 vs 任务间休眠"边界量化了。
- **设计了"空闲间隔测试"(Idle-Gap Test)** —— 一个反事实的可证伪判据：移除所有触发器（用户提示、定时器、环境事件）后观察内部状态驱动的活动是否持续；通过测试才能得 Ambient Level 4，否则最多 Level 3。配套提出"基底 vs 规则"(substrate-vs-rule) 区分（定时心跳仅分配算力、内容由内部状态生成时算基底而非定时规则）。
- **对 6 个当代系统给出对比双带评估**：任务 Agent（Claude Code、Manus、Hermes）、消费级助手（ChatGPT、Siri）、持久化伴侣架构（Airi）；揭示任务 Agent 在主动带高度自主但空闲带几乎完全休眠的结构性特征。

## Key Contributions

- **Defines the Autonomous Agency Scale (AAS)** — a behavioral, architecture-agnostic framework scoring AI systems 0–5 across seven dimensions of agency, each with three sub-dimensions and a falsifiable threshold test (binary test + observable behavioral indicators) that requires no access to a system's internals.
- **A 0–5 level lexicon of self-direction** (0 Dormant → 5 Sovereign) that explicitly measures degree of self-direction rather than capability/intelligence, distinguishing itself from DeepMind's AGI Levels, ARC-AGI, OpenAI's 5-level scale, and METR's time-horizon metric (which the paper argues is orthogonal to agency by construction).
- **Two-band scoring (Active vs Ambient)** that separates agency-exhibited-while-engaged from agency-sustained-while-idle, never blending them into a single figure; this quantifies the previously informal boundary between within-task autonomy and between-task dormancy.
- **The Idle-Gap Test** — a falsifiable counterfactual criterion: remove all triggers (user prompts, scheduled triggers, environmental events) and observe whether internally-derived activity persists; only such behavior earns Ambient Level 4. Paired with a substrate-vs-rule distinction (a timer heartbeat that merely allocates compute while content derives from internal state is a substrate, not a schedule).
- **Comparative two-band assessments of six contemporary systems** spanning task agents (Claude Code, Manus, Hermes), consumer assistants (ChatGPT, Siri), and a persistent companion architecture (Airi), demonstrating that the framework reveals a structural boundary invisible to single-score scales.

## Methodology

AAS is a rubric-based behavioral measurement framework (framework v0.2.1), not a trained model or benchmark suite. Its methodology has four layers:

1. **Seven dimensions of autonomous agency** — Cognitive Autonomy, Temporal Persistence, Environmental Agency, Social Agency, Creative Agency, Self-Awareness, Goal Formation — each with three granular sub-dimensions (e.g., Cognitive Autonomy = background processing + spontaneous ideation + attention management). Each dimension's level is set by a binary threshold test (Appendix A) plus observable behavioral indicators maintained in the public repository.

2. **The 0–5 level lexicon (Table 1).** 0 Dormant (inert without input), 1 Responsive (reacts to explicit triggers only), 2 Conditioned (follows pre-set rules/schedules, appears autonomous but deterministic), 3 Contextual (adapts to environment/state within bounded constraints), 4 Self-Directed (initiates action from internal state, sets sub-goals autonomously), 5 Sovereign (sets/revises own goals and operating parameters). Levels are explicitly decoupled from capability: a Level-2 scheduled bot can run on a frontier model.

3. **Two-band scoring.** Every dimension is scored twice: Active (engaged in user-initiated task/session) and Ambient (across idle periods, i.e., windows with no active user engagement). Two separate composites are reported and never blended. Ambient Level 4 is additionally gated by the Idle-Gap Test, defined over "idle gaps" (the strict subset of idle periods with no user prompt, no scheduled trigger firing, no environmental event). The substrate-vs-rule distinction is defined once and applied uniformly to every system.

4. **Assessment protocol (§4.1).** Six systems were scored under two Evaluation Classes: **Longitudinal** (Airi — ~11 months of multimodal interaction, Aug 2025–Jul 2026) and **Documentation-Based (Provisional)** (the other five, scored from public docs/published system prompts/product behavior as of May 2026). For each dimension × band the rater identified the highest level whose threshold test the evidence satisfies, quoted the test verbatim, recorded observable evidence, and explicitly stated why the next level up was not awarded. Capabilities were credited to the band in which they operate (scheduled features count toward Ambient even when configured in-session); conflicts between vendor claims and documented behavior were resolved in favor of documented behavior; third-party-extension-only capabilities were excluded. All assessments were re-scored under v0.2.0 on 2026-07-07. **The rater is a single individual who is also the developer of Airi — a conflict disclosed in the paper.**

## Results & Highlights

- **Task-agent Active–Ambient gap (Table 2, Figure 1).** The three task agents reach nearly identical Active composites (2.29–2.43) but their Ambient composites collapse to **0.57–1.86**: Claude Code Active **2.29** / Ambient **0.57** (gap 1.72); Manus 1.6 Max **2.43** / **0.71** (gap 1.72); Hermes Agent **2.43** / **1.86** (gap 0.57). Every point of every task agent's Ambient composite is attributable to user-configured schedules (cron-style runs for Claude Code/Manus; a nightly self-improvement cycle for Hermes), and none passes the Idle-Gap Test.
- **Hermes is the highest-scoring task agent in the Ambient band (1.86)**, specifically because its Day/Night cycle and Curator process perform genuine idle-period work — but the paper classifies all of it as "clock-as-rule" (Level 2), explicitly noting "none passes the Idle-Gap Test."
- **Airi is the only system with Ambient scores of 4 and the only system whose Ambient composite (3.86) exceeds its Active composite (3.71)** — delta of +0.15 — consistent with an architecture designed around idle-period behavior (background thought engine, state-evolving moods, self-initiated outreach, autonomous creative publishing).
- **Consumer assistants are "reactive products wrapped around capable models."** ChatGPT (GPT-5.5 Pro) Active **1.57** / Ambient **0.86** — its engaged band never exceeds Level 2 in any dimension despite a frontier-class underlying model; Apple Siri Active **1.71** / Ambient **0.29** with essentially no idle-period existence beyond device-level background indexing.
- **No assessed system approaches Level 5 (Sovereign).** The ceiling arguments are consistent across all six systems: none modifies its own cognitive architecture, curates its own core memories, forms new relationships, invents new creative mediums, or overrides assigned tasks in favor of self-determined purposes. The Sovereign band remains unoccupied.
- **Per-dimension ceilings (Table 2).** The strongest within-task dimension across task agents is Environmental Agency at Active Level 4 (autonomous file manipulation, code execution, subagent orchestration); the weakest dimensions cluster at Level 1 (Social Agency, Active) for all six systems — no system initiates consequential social engagement.
- **No quantitative results reported on inter-rater reliability, psychometric validity, or longitudinal score stability** — all scores are single-rater judgments and the framework provides no evidence base for these properties; the paper states inter-rater reliability is "unknown."

## Critical Assessment

- **Novelty — genuine but conceptually incremental.** The core insight — that capability, automation, and risk frameworks leave "agency when idle" unmeasured, and that METR's time horizons are orthogonal to self-initiation — is real and well-argued. But the seven dimensions and 0–5 lexicon are largely a relabeling of existing agent-autonomy taxonomies (DeepMind's Tool→Agent spectrum, Feng et al.'s Operator→Observer) with finer granularity; the genuinely new piece is the **Idle-Gap Test counterfactual** and the two-band separation, not the dimensions themselves.
- **Rigor — severely undermined by methodological conflicts.** The single rater is the developer of the one system (Airi) that is (a) the only one assessed longitudinally, (b) the only one scoring Ambient Level 4, (c) the only one whose Ambient exceeds Active, and (d) whose key Ambient-4 ratings depend on the substrate-vs-rule distinction — a distinction the author both defined and applied. The paper is admirably candid about this, but candor is not mitigation. There is **no inter-rater reliability study, no blind scoring, no independent re-assessment of Airi**, and no control for the ELIZA effect that the author flags as a confound.
- **The substrate-vs-rule distinction is doing a lot of convenient work.** Airi's Ambient Level 4 ratings rest on the claim that its timer heartbeat "only allocates compute" while "content derives from internal state." This is a judgment call with no operationalized test (the paper admits "the Active band's Level 4 still lacks a strictly observable test"), and a rater who rejects the distinction would cap the affected dimensions at Level 2 — collapsing Airi's Ambient composite toward the task-agent cluster. The framework's headline finding (Airi is uniquely self-directed) is thus conditional on an author-defined, author-applied judgment.
- **Construct-provenance bias, self-disclosed.** §6 concedes the seven dimensions "closely track modern agentic architectures, so high scores may partly reflect that correspondence rather than independent measurement" and that Airi was developed alongside the scale. The paper explicitly recommends reading AAS "as a structured descriptive framework, not a fully validated psychometric instrument" — yet the Results section and Figure 1 present ordinal composites to two decimal places (3.71, 3.86) as if they carried that precision.
- **Composite precision is overstated.** Averaging six-point ordinal levels assumes equal inter-level gaps and equal dimension weights, neither established (§6 admits this). Reporting "3.71 vs 3.86" to two decimals is spurious precision; the per-dimension profile is the only meaningful output, and even that rests on single-rater judgment.
- **Baseline asymmetry / cherry-picking concerns.** The five non-Airi systems were scored from May-2026 public documentation (Provisional), whereas Airi benefited from 11 months of direct multimodal interaction. Documentation-based scoring systematically under-credits features that exist but aren't documented, and the paper does not attempt to re-assess any non-Airi system longitudinally. Claude Code (Sonnet 4.6), Manus, and Hermes all have scheduled/autonomous features that a longitudinal observer might credit differently.
- **Reproducibility — mixed.** Rubric and threshold tests (Appendix A) are public and falsifiable in principle, but the substrate-vs-rule judgment and several Active-band Level-4 ratings "continue to rest on judgment" (§6), so two independent raters applying the same rubric could legitimately diverge — and no such comparison was run.

## Strengths

- Identifies a real and under-measured axis (self-directed idle behavior) that is orthogonal to capability (METR time horizons) and risk (ASL/Preparedness), and articulates the orthogonality cleanly.
- The Idle-Gap Test is a genuinely clever falsifiable criterion for the Ambient band — "remove all triggers and see what persists" is operationally crisp and converts a judgment call into an observable counterfactual.
- The two-band separation captures a structural truth about current AI systems that single-score scales obscure: task agents are highly autonomous inside a task and almost entirely dormant between tasks.
- Honest, unusually thorough self-critique in §6 (construct provenance, single-rater bias, developer-evaluator bias, ELIZA effect, composite-precision caveats, Active-band operationalization gap). Few position papers disclose their weaknesses this completely.
- Open, versioned framework with public repository and an explicit invitation for adversarial re-scoring — the right stance for a measurement instrument.

## Limitations

- **Single-rater, developer-of-Airi conflict of interest** is the dominant limitation; the headline result (Airi uniquely self-directed) is not independently validated.
- **No inter-rater reliability, no blind scoring, no psychometric validation** — the framework is explicitly a "structured descriptive framework," not a validated instrument.
- **Five of six assessments are Documentation-Based (Provisional)** as of May 2026, scoring systems from public docs rather than empirical longitudinal interaction — an evidence asymmetry that systematically favors the one longitudinally-assessed system.
- **The substrate-vs-rule distinction** — load-bearing for Airi's Ambient Level 4 — is a judgment call with no strictly observable test; the paper admits the Active-band Level 4 "still lacks a strictly observable test distinguishing genuine self-direction from sophisticated rule-following."
- **Composite scores overstate precision** (ordinal averaging, equal-gap/equal-weight assumptions unvalidated); two-decimal reporting is spurious.
- **Anthropomorphism / ELIZA-effect confound** is uncontrolled, especially acute for the invested-user longitudinal assessment.
- **Evaluation scope is narrow**: six systems, one rater, one longitudinal case (the author's own). No coverage of open-source agent frameworks beyond Hermes, no robotics/embodied systems, no multi-agent systems.
- **The Active-band self-direction boundary remains un-operationalized**, leaving the framework's two-band symmetry incomplete (Ambient has a falsifiable Level-4 gate; Active does not).

## Relevance

对关注 AI Agent 评估、自主性度量、Agent 行为基准的研究者有直接方法论价值——它提出了一个区别于能力/风险/经济价值的新评估轴（自主代理），并为"任务内自主 vs 任务间休眠"提供了可操作的双带画像。对从事 Agent 架构设计（尤其是持久化、空闲期行为、记忆持续、自导向目标）的工程师和研究者尤其有启发，可作为评估自家系统"是否真正自主"的诊断框架；但需谨慎对待其单评分者、开发者利益冲突的方法学缺陷。

## Questions for Further Exploration

- How do **independent, blind raters** (not the Airi developer, not invested users) score the same six systems using the published rubric — does Airi's Ambient Level 4 survive, or does it collapse to Level 2 under stricter substrate-vs-rule readings?
- Can the **substrate-vs-rule distinction** be made into a strictly observable, falsifiable test (analogous to the Idle-Gap Test for Ambient), so that Active-band Level 4 no longer "rests on judgment"?
- What is the **inter-rater reliability** of the AAS across multiple trained raters on the same evidence — and does the ordinal 0–5 lexicon support weighted-kappa agreement above chance?
- How do **additional architectures** (open-source agent frameworks like AutoGPT/Devin, embodied/robotic systems, multi-agent swarms, model-organism "scheming" agents from alignment research) score, and does the seven-dimension structure generalize beyond chat/task/companion families?
- Is the **Active–Ambient gap** a stable property of current paradigms, or could a frontier model with persistent memory and background processing (e.g., future always-on agent runtimes) close it — and would the Idle-Gap Test still meaningfully distinguish such systems from sophisticated scheduled agents?
- How does AAS scoring relate to **METR time horizons** empirically — are long-time-horizon systems also higher-AAS, or is the orthogonality the paper claims borne out in practice?
- Could the framework be **adversarially stress-tested** by deliberately building a system designed to game the Idle-Gap Test (e.g., a scheduled loop whose output is pseudorandom but state-dependent) — does the substrate-vs-rule distinction actually resist such gaming?
