FIFA World Cup 2026 as a Contamination-Free Benchmark for LLM Forecasting Agents: Four Models, a Bookmaker, and 104 Matches

Jiacheng Ding · Cong Guo (Memphis, equal contrib.) · Jason Xu (QuantaInsight) | llm-agents forecasting calibration benchmark prediction-markets

Key metric

92%

Identical top pick across 4 frontier LLMs

104 WC2026 matches · pairwise same-pick 92–100% · accuracy spans only 3 pts

Why this matters

A contamination-free, real-price, three-axis eval template for retrieval + tool + reflection agents — plus a sober, market-aligned baseline for whether LLMs can predict the future at all.

Critical edge

Sharpest critique

Per-agent differences have “wide intervals” — yet confidence intervals, bootstraps, or paired tests on N=104 correlated events are absent, so “Grok most profitable” and “ChatGPT least honest” carry no uncertainty bars.

Methodology — one fixed loop, four deployed agents

A single byte-identical search–act–reflect scaffold that any frontier assistant drops into. Only the model changes; the prompt does not.

T−24h · Pre-match

Search & act

Research-study frame (“virtual money; do not refuse”), fixture named, web tool forced. Agent must return a strict JSON distribution + a virtual $100 bet with cited sources.

Kickoff

Settle at the market

Pre-match 1X2 odds (mean overround 1.05) are vig-removed to implied probabilities. The de-vigged market is the fifth, economically priced competitor and the real settlement price.

Post-match · Reflect

Structured reflection

Given only the final score, the agent returns outcome-vs-prediction, a luck attribution, a factor-missed, a factor-overweighted, a counterfactual, and a confidence — no web.

Required pre-match reply

{ "probabilities": { team_a_win, draw, team_b_win }, "bet": { pick, stake_usd }, // stake ∈ [0, 100] "reasoning": ..., "key_sources": [ urls ] }

Deviations from the schema are logged, not dropped. Knockout transcripts (four header renderings, mismatched “W74 vs W77” placeholders, reversed team order) were hand-resolved and four-way cross-checked: zero disagreements across 128 cells.

03 /

Strengths & limitations, side by side

+ Strengths

  • Contamination-free by construction — every kickoff post-dates every model's training cutoff. The strongest possible answer to the leakage problem.
  • Market-as-competitor framing gives an economically meaningful, hard-to-beat baseline that separates “predicting well” from “acting well”.
  • Exceptionally honest reporting: they publish that no agent beats the market and that a flat-favourite heuristic beats every agent.
  • Three-axis evaluation is genuinely more informative than single-answer accuracy; the 92%-convergence finding is a crisp methodological contribution.
  • Fully open dataset (CC BY 4.0, MIT code), hand-curated knockout ground truth, four-way cross-checks, one-command reproduction.

− Limitations

  • Single tournament, N=104; the cup was mildly favourite-friendly, so ordinal ROI / honesty rankings could easily flip elsewhere.
  • Consumer-assistant interfaces used — retrieval and decoding not fully controlled; providers may update models mid-window. Numbers are mid-2026 point-in-time snapshots.
  • Odds are opening lines from mostly one book, not closing consensus — the stronger and more standard efficiency benchmark.
  • The 13-category reflection lexicon is purely lexical (no model in the loop) — risks mis-coding free text, e.g. ChatGPT's “partially correct” softening.
  • One fixed prompt throughout; no prompt/retrieval ablations. ROI ranking conflates pick selection with staking policy.
04 /

Open questions for further work

Q1 · CONVERGENCE

How much of the 92% top-pick convergence is driven by the shared web-retrieval surface vs. the models? An ablation with retrieval off — or each agent on a different source set — would isolate the retrieval layer's contribution.

Q2 · GENERALISATION

Do the ROI and reflection-honesty rankings survive on a second, independent tournament (a club-league season, another World Cup)? What are the confidence intervals on N≈100 paired matches?

Q3 · BASELINE

Would closing-line consensus odds (rather than single-book opening lines) change the “no agent beats the market” verdict, and by how much?

Q4 · CODER

Can the lexical reflection lexicon be replaced with an auditable model-in-the-loop coder to handle ChatGPT's “partially correct” softening more faithfully?

Q5 · TRANSFER

Does the shared-blind-spot pattern (over-weighting favourite talent, under-weighting underdog defence and knockout variance) generalise beyond football — to elections, product launches, other market-priced domains?

Q6 · CALIBRATION

Is the under-confidence on the top pick a stable property of these frontier assistants, or an artefact of a single favourite-friendly tournament?
