RQ-MoE: Residual Quantization via Mixture of Experts for Efficient Input-Dependent Vector Compression

Date: 2026-05-14

⚠️ Abstract-only summary

一句话总结： RQ-MoE 将残差量化与混合专家机制结合，通过双流量化实现输入相关的动态码本构建，在重建和检索任务上达到SOTA性能的同时解码速度比先前方法快6-14倍。

  • 双流架构解耦指令与量化：将码本适配过程分为指令流和量化流，使码本构建可以并行解码，突破了 QINCo 等方法的严格串行依赖瓶颈。
  • 两级 MoE 实现输入相关码本：通过混合专家机制动态生成码本，使量化器能适应异构数据几何结构，显著提升表达能力。
  • 理论统一框架：证明标准残差量化和 QINCo 均为 RQ-MoE 在特定约束下的特例，并推导了专家维度设置的指导准则。
  • 大幅加速解码：在保持SOTA或同等性能的同时，解码速度提升6-14倍，解决了动态量化器的实际部署瓶颈。

Key Contributions

  • Proposes RQ-MoE, a framework that combines a two-level Mixture-of-Experts (MoE) with dual-stream quantization to enable input-dependent codebook adaptation for efficient vector quantization.
  • Decouples instruction computation from quantization, enabling parallel decoding and eliminating the sequential bottleneck present in prior dynamic quantizers like QINCo.
  • Provides theoretical analysis showing that standard Residual Quantization and QINCo are constrained special cases of RQ-MoE, and derives a guideline for setting expert dimensionality.
  • Achieves state-of-the-art or on-par performance in reconstruction and retrieval tasks while providing 6x–14x faster decoding than prior vector quantization methods.

Methodology

RQ-MoE introduces a two-level Mixture-of-Experts architecture integrated with residual quantization. The key innovation is a dual-stream design that separates the instruction stream (which adapts codebooks to input) from the quantization stream (which performs the actual quantization). This decoupling removes the strict sequential dependencies that plague QINCo-style dynamic quantizers, enabling parallel codebook construction across residual steps.

The MoE component allows the system to dynamically specialize codebook entries based on input geometry, addressing the expressiveness limitation of static multi-codebook methods. The framework is general enough to recover both standard Residual Quantization (when experts degenerate) and QINCo (under specific architectural constraints) as special cases.

Results & Highlights

  • Achieves state-of-the-art or competitive performance in both reconstruction quality and retrieval accuracy benchmarks.
  • Provides 6x–14x faster decoding compared to prior vector quantization methods, making dynamic quantization practical for deployment.
  • Demonstrates that input-dependent codebook adaptation through MoE can match or exceed the expressiveness of sequential adaptive methods while being dramatically faster.
  • Implementation is publicly available at https://github.com/KDEGroup/RQ-MoE.

  • Significance: High practical impact potential. The 6x–14x decoding speedup addresses a real deployment bottleneck for dynamic quantizers in large-scale retrieval systems.

Strengths

  • Clean architectural design that solves a real bottleneck (sequential decoding) in dynamic vector quantization.
  • Strong theoretical grounding with provable connections to prior methods.
  • Significant practical speedups (6x–14x) without sacrificing reconstruction quality.

Limitations

  • The MoE component introduces additional parameters and memory overhead compared to static codebook methods; the exact trade-off is unclear from the abstract alone.
  • The "on-par" (rather than strictly superior) performance on some benchmarks suggests the speed advantage may come at a minor quality cost in certain regimes.
  • Evaluation is limited to reconstruction and retrieval tasks; applicability to other domains (e.g., generative modeling, feature compression) remains to be seen.
