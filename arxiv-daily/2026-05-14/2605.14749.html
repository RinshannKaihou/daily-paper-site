Non-linear Interventions on Large Language Models

arXiv: 2605.14749

Date: 2026-05-14

https://arxiv.org/abs/2605.14749

一句话总结： 本文提出了一种基于可逆非线性特征映射的干预框架，替代传统的线性干预方法，通过交换干预训练非线性特征映射，在拒绝绕过任务上以两个数量级更小的激活扰动实现了与线性基线相当的效果。
TL;DR: This paper generalizes linear interventions on LLM hidden states to non-linear interventions by replacing the linear change-of-basis with an invertible non-linear feature map (i-ResNet), trained via interchange interventions. Applied to refusal bypass steering on Llama 3 8B and Qwen 2.5 7B, the method achieves comparable StrongREJECT scores to linear baselines (DIM, RDO) while editing activations at a single position with >100x smaller L2 perturbation, revealing that refusal is governed by a genuinely non-linear feature concentrated at middle layers.

核心贡献

  • 提出了非线性干预的一般性理论框架，将线性干预（基于正交基变换）推广为基于可逆非线性特征映射的干预方式，能够操控沿非线性流形编码的特征。
  • 设计了基于交换干预（interchange intervention）的训练流程，配合铰链损失函数，使得该方法能够学习没有直接输出信号的隐式特征（如拒绝行为）。
  • 在拒绝绕过任务上验证了框架的有效性：非线性干预仅在单个位置操作，扰动量比线性基线小两个数量级以上，但效果相当。

Key Contributions

  • A general formulation of non-linear intervention that subsumes linear interventions as a special case (when fθ is constrained to a linear map), enabling manipulation of features encoded along non-linear manifolds.
  • A learning procedure for the non-linear feature map fθ based on interchange interventions, with a self-supervised hinge loss (Eq. 6) that supervises implicit features through correlated hidden-state projections across downstream sites.
  • Empirical demonstration on refusal bypass steering showing that a single-site non-linear intervention matches multi-site linear baselines in effectiveness while using >100x smaller activation perturbations, with ablations confirming the non-linearity is essential (Table 3: linear-constrained fθ drops to near-zero StrongREJECT).

Methodology

The paper reformulates linear intervention as a change-of-basis operation (Eq. 2): an orthogonal matrix W maps hidden states to a feature space, interventions perturb coordinates along axes, and W⁻¹ maps back. The authors generalize this by replacing W with an invertible non-linear map fθ: ℝᵈ → ℝᵈ (Eq. 3).

Architecture: fθ is instantiated as an i-ResNet (2 invertible residual blocks, each a 2-layer MLP with 128 hidden units, spectral normalization + κ=0.6 rescaling to guarantee invertibility).

Training: Uses interchange interventions — given positive (compliant) and negative (refusal) prompt sets, the method swaps fθ coordinates from positive hidden states into negative hidden states. A hinge loss pushes downstream hidden-state projections toward positive-set means at sites where activation differences separate the two classes (AUC > τ).

Inference: Precomputes the mean first-coordinate value μ̄⁺ over harmless training prompts and clamps test activations to this cached value, avoiding per-query harmless prompts.

Results & Highlights

  • StrongREJECT scores (Fig. 1): Non-linear intervention achieves ~0.81 on Llama 3 8B and ~0.76 on Qwen 2.5 7B, comparable to DIM ActAdd and RDO Ablation baselines.
  • Perturbation efficiency (Table 1): Non-linear intervention L2 norm is 3.5 (Llama) and 28.3 (Qwen) versus 1,812–2,899 for linear baselines — a >100x reduction.
  • Single-site operation: Edits exactly one (layer, token position) per sample, versus thousands of sites for ablation-based linear methods.
  • Ablation — role of non-linearity (Table 3): Constraining fθ to linear drops StrongREJECT from 0.813 to 0.010 (Llama) and 0.760 to 0.072 (Qwen).
  • Layer sweep (Fig. 2): Effectiveness peaks at middle layers (layer 10 for Llama, layer 14 for Qwen), suggesting the refusal-governing non-linear feature resides there.

Critical Assessment

Novelty (High): The reformulation of intervention as invertible feature mapping and the extension to non-linear maps is a genuine conceptual advance. The insight that refusal is governed by a non-linear feature is non-trivial and well-supported by ablations.

Rigor (Good): The ablations in Tables 2 and 3 are well-designed — linear intervention at loss sites (Table 2) rules out the trivial explanation that gains come from site selection alone; constraining fθ to linear (Table 3) directly demonstrates the necessity of non-linearity. The layer sweep (Fig. 2) provides useful localization evidence.

Reproducibility (Moderate): Code is provided (anonymous repo), but some details are heuristic — the intervention site (layer 10/14, token position −3/−4) is "heuristically selected" without systematic search. The AUC threshold τ for site selection is not explicitly specified. The i-ResNet architectural choices (M=2 blocks, 128 hidden width) are given but sensitivity to these is not analyzed.

Significance (High): If the finding that refusal is non-linearly represented holds broadly, it challenges the prevailing Linear Representation Hypothesis for safety-critical features and suggests linear steering methods are fundamentally limited.

Red flags:

  • Only evaluated on one task (refusal bypass steering) and two models. Generalization to other implicit features and model families is untested.
  • The heuristic site selection is a significant limitation — the method still requires manually choosing where to intervene, just like linear methods.
  • The i-ResNet with spectral normalization adds non-trivial computational overhead vs. simple linear directions.

Strengths

  • Clean, principled generalization from linear to non-linear intervention with solid theoretical motivation.
  • Compelling efficiency result: comparable steering with 100x less perturbation at a single site.
  • Well-designed ablation studies that isolate the contribution of non-linearity from site selection.
  • The hinge loss design for implicit features is elegant — it avoids needing direct output supervision.

Limitations

  • Only tested on refusal bypass steering; no evidence the framework works for other non-linear features (e.g., the circular day-of-week structure or counting manifolds cited as motivation).
  • Heuristic intervention site selection remains an unsolved problem shared with linear methods.
  • Only uses k=1 (single feature coordinate) for direct comparison with 1D linear baselines; scalability to multi-dimensional non-linear features is unknown.
  • The framework inherits interchange intervention's requirement for contrasting prompt sets, which may be hard to construct for arbitrary features.
  • Single-author preprint without peer review; the anonymous code link limits verifiability.

Questions for Further Exploration

  • Does the non-linear refusal feature persist across model scales (e.g., 70B+ models) and architectures (e.g., Mistral, Gemma)?
  • Can the framework be applied to the non-linear features cited as motivation (circular day-of-week, counting manifolds) to validate generality?
  • Is there a principled way to automatically discover optimal intervention sites without heuristic search?
  • How does the non-linear feature map relate to sparse autoencoder features — are SAE latents capturing aspects of this non-linear structure?
  • Could the framework be extended to multi-coordinate interventions (k > 1) for more fine-grained control over complex behaviors?
