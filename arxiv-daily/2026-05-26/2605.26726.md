> **一句话总结：** 本文提出 resilience——一种无需重新训练、无需修改架构的不确定性度量方法，通过扰动NCA的最终状态并观察其是否能恢复原预测来量化分割结果的可信度，在五个医学图像分割基准上表现优于现有基线。

> **TL;DR:** The authors propose resilience, a training-free uncertainty measure for Neural Cellular Automata (NCA) that probes prediction stability by perturbing the final automaton state with Gaussian noise and checking whether the system recovers to the same segmentation. Across five medical segmentation benchmarks, resilience consistently outperforms baselines in selective prediction and failure detection.

## 核心贡献

- 首次系统性地研究NCA医学图像分割中的不确定性估计问题
- 提出 resilience 方法：对NCA最终状态施加高斯扰动，再运行若干步观察是否恢复原预测，用 1-IoU 度量不确定性
- 无需修改模型架构、无需重新训练，直接利用NCA固有的迭代动力学特性
- 在五种医学成像模态（内窥镜、显微镜、皮肤镜、组织学）上统一评估，覆盖分布偏移场景
- 代码完全开源

## Key Contributions

- First systematic study of uncertainty estimation for NCA-based medical image segmentation across four imaging modalities
- Introduces resilience, a perturb-and-recover uncertainty measure that probes the stability of NCA learned dynamics by injecting Gaussian noise into the final state and running additional NCA steps
- Training-free and architecture-agnostic: requires no retraining, no dropout layers, no architectural changes
- Unified evaluation protocol across five benchmarks with selective prediction (ΔDice@90, AURC) and failure detection (AUROC, AUPRC) metrics
- Publicly available implementation at github.com/marrlab/resilience

## Methodology

The method views NCA as a dynamical system where convergent attractors correspond to confident predictions. The pipeline is:

1. **Run NCA for T steps** to obtain final state $S_T$ and segmentation mask $m$.
2. **Perturb the state**: $\tilde{S}_T = S_T + \varepsilon$, where $\varepsilon \sim \mathcal{N}(0, \sigma^2 I)$ with $\sigma = 0.02$. Only latent state channels are perturbed; the input image remains fixed.
3. **Relaxation**: Run additional $T' = 12$ NCA update steps from $\tilde{S}_T$ to get recovered prediction $m'$.
4. **Resilience score**: $u_{\text{res}} = 1 - \text{IoU}(m, m')$. Low values indicate stable attractors (confident predictions); high values indicate unstable dynamics (uncertain predictions).

Compared against five baselines: single-pass entropy, stopping-time disagreement, late-stage drift (stability), binary flicker, and test-time augmentation (TTA). All methods produce a scalar uncertainty score per image aggregated from pixel-wise maps over a thin boundary band around the predicted mask.

## Results & Highlights

- **Selective prediction (ΔDice@90):** Resilience achieves the highest gains on 3/5 datasets: ClinicDB (8.13%), ISIC 2017 (7.61%), NuInsSeg (7.63%), and Kvasir-SEG (9.26%). Several baselines produce negative gains on some datasets.
- **Failure detection (AUROC/AUPRC):** Resilience achieves best or near-best AUROC on DSB 2018 (0.972) and ISIC 2017 (0.910), and best AUPRC on DSB 2018 (0.914) and ISIC 2017 (0.755).
- **AURC (lower is better):** Resilience achieves the lowest AURC on 4/5 datasets, indicating superior risk-coverage tradeoffs.
- The strongest competing baseline varies by dataset (stoptime on DSB 2018, TTA on Kvasir-SEG, flicker on NuInsSeg), suggesting complementary uncertainty cues across different failure modes.
- Qualitative analysis shows high-uncertainty cases involve ambiguous boundaries, low contrast, artifacts, or atypical anatomy.

## Critical Assessment

- **Novelty (moderate):** The perturb-and-recover idea is conceptually simple and resembles techniques from dynamical systems theory (Lyapunov stability). The contribution is adapting this to NCA specifically, but the core idea is not fundamentally new.
- **Rigor (strong):** Unified evaluation across 5 datasets, 5 baselines, 4 metrics, 3 runs with standard deviations. No dataset-specific tuning of the NCA model ensures fair comparison. Distribution shift scenarios are included.
- **Reproducibility (strong):** Code publicly available, all hyperparameters specified (σ=0.02, T'=12, C=64, fire rate 0.5), deterministic splits described.
- **Significance (moderate-high):** Practical value for deploying NCA models in clinical settings. The method is lightweight and easy to integrate. However, the paper only evaluates binary segmentation; extension to multi-class is not discussed.
- **Red flags:** Only one perturb-and-recover rollout per image is used. Ablation on σ and T' is minimal. The NCA model performance on some datasets is mediocre (Kvasir-SEG mIoU 0.43), which may inflate uncertainty improvements relative to stronger base models.

## Strengths

- Training-free: no architectural changes or retraining required
- Conceptually elegant: directly probes dynamical stability as a proxy for trustworthiness
- Consistently strong across diverse benchmarks and metrics
- Readout-agnostic: applicable beyond segmentation to other NCA tasks
- Lightweight computation overhead (only 12 additional NCA steps)

## Limitations

- Only evaluated on binary segmentation tasks; multi-class extension is unclear
- Single perturbation per image may miss unstable directions not captured by one random noise vector
- NCA base model uses identical training settings across all datasets, which is suboptimal for segmentation performance
- Ablation studies on perturbation magnitude σ and relaxation length T' are limited
- Comparison only against NCA-specific baselines; no comparison with uncertainty methods for encoder-decoder networks (e.g., MC dropout on U-Net) to contextualize absolute performance

## Relevance

本文针对NCA这一轻量级分割范式提出了实用的不确定性量化方法，对关注医学图像分割可靠性和可信赖AI部署的研究具有参考价值。Resilience方法的设计思想——利用模型内在的动力学特性而非额外训练——对其他迭代式模型也有启发意义。

## Questions for Further Exploration

1. How does resilience scale to multi-class segmentation with multiple competing attractors?
2. Would ensembling multiple perturbations (different noise realizations) improve robustness of the uncertainty estimate?
3. Can resilience be combined with complementary baselines (e.g., flicker + resilience) for a hybrid uncertainty signal?
4. How does resilience compare to MC dropout or deep ensembles on a standard U-Net baseline for the same datasets?
5. Is there a theoretical connection between resilience and Lyapunov exponents of the learned NCA dynamics?
