Why This Matters

Rigorous analysis reveals fundamental limits of SVD-free clipping for Muon-family optimizers — useful for design theory, but unvalidated in any actual training run.

⚠ Critical Edge

No experiments — the paper frames itself around LLM training but contains zero training results, no convergence plots, and no baseline comparisons.

Methodology

Purely theoretical/numerical analysis — no empirical component. Six-stage derivation:

  • Problem setup: Replace Muon's polar direction UVT with clipped direction U diag(min{σi, τ})VT. Can clipping avoid full dense SVD?
  • SpectralP scaling: Situates MuCon within the CompleteP width-depth regime for Transformers — learning rates, weight decay, and ε scale with model dimensions.
  • Low-rank correction: Clipping = rank-k> correction, motivating partial SVD when few singular values exceed τ.
  • Polar / absolute-value identity: M = QH, then fτ(H) = ½(H + τI − |H − τI|). Exact, but sign determination ill-conditioned near threshold.
  • Rational Newton filter: Spectral functional-calculus iteration. Quadratic convergence away from σ = τ, linear at the double root. Linear solve degenerates near threshold.
  • Regime split guideline: Few violations → partial SVD / Lanczos. Many violations → global matrix-function methods with stable polar/square-root primitives.

Strengths

  • Rigorous handling of edge cases — rank-deficient and rectangular matrices
  • Honest negative result: shared numerical obstruction clearly identified
  • SpectralP scaling recipe (Table 1) is a practical reference for Muon-family implementations

Limitations
  • Purely theoretical — no training experiments, convergence plots, or baseline comparisons
  • Negative-leaning conclusion: both SVD-free paths face the same obstruction, limiting practical takeaway
  • Does not demonstrate that clipping outperforms full orthogonalization in any setting
  • SpectralP details depend heavily on external work (CompleteP, SteptronOSS)
  • Rational Newton section is incremental — standard scalar Newton applied via functional calculus
