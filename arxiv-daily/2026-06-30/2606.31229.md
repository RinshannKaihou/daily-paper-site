> **一句话总结：** 本文提出 Agentic-Ideation 框架，通过 Oracle 引导的多智能体数据合成（Dual-Thought 机制）与工具结果掩码的 SFT 训练，让 Qwen3-8B 学会自主科研构想；在 300 篇 NeurIPS 2025 论文的评测中整体质量较 SOTA 工作流基线提升 11.91%，并将高质量轨迹合成从平均 12 次 rollout 降到 1 次（10× 以上效率提升）。

> **TL;DR:** Agentic-Ideation trains an agentic LLM (Qwen3-8B) for scientific ideation via an Oracle-Guided multi-agent trajectory synthesis pipeline (a Planner + Controller + 6 tool agents with a Dual-Thought mechanism) and SFT with tool-result masking, beating SOTA workflow baselines by 11.91% overall while cutting trajectory-synthesis rollouts from ~12 to 1 (>10× sample-efficiency gain).

## 核心贡献
- **首次将训练式 agentic LLM 应用于科研构想（scientific ideation）**，作为对现有固定工作流方法（ResearchAgent、SciPIP、SciMON、VirSci）的范式替代，使模型能自适应地导航复杂的想法搜索空间。
- **提出 Oracle-Guided 多智能体数据合成框架**：以高质量"参考想法"作为预言指引，通过 Dual-Thought 机制（私有想法用 oracle 做差距分析、公共想法仅基于可见历史进行推理），将无方向的试错采样转化为有向的轨迹重构，解决开放答案空间下拒绝采样效率极低的问题。
- **定义混合工具空间**：3 个外部工具（Search、Get_References、Get_Cited）+ 3 个认知工具（Analyse_Gap、Ideation、Reflection），兼顾外部检索与内部推理。
- **引入工具结果掩码的 SFT 训练**：在损失计算中掩去 `<Result>` 标记内的工具返回内容，只对智能体自身的决策（思考与工具调用）回传梯度，避免模型记忆确定性的环境反馈。
- **全面实验验证**：在 NLP/CV/Other 三类共 300 篇 NeurIPS 2025 论文上由 4 个 SOTA 评判模型（GPT-5.2、Gemini-3-Pro、Claude-Sonnet-4.5、Qwen3-Max）打分，整体质量提升 11.91%；双盲人工评估进一步确认新颖性、意义、可行性均领先；合成效率提升 10× 以上。

## Key Contributions
- **Pioneers training-based agentic LLMs for scientific ideation**, offering a flexible, autonomous alternative to rigid workflow-based "AI Scientist" systems that depend on hand-crafted prompts.
- **Oracle-Guided multi-agent data synthesis** with a Dual-Thought mechanism: a Planner + Controller + six specialized tool agents reconstruct the reasoning path toward a high-quality "reference idea" in a single pass, converting stochastic rejection sampling into directed trajectory generation.
- **A hybrid tool space** combining three external tools (Search, Get_References, Get_Cited) for academic-DB access with three cognitive tools (Analyse_Gap, Ideation, Reflection) for internal reasoning, modeling the iterative explore-then-synthesize nature of ideation.
- **Tool-result masking during SFT**, so gradients flow only through the agent's own thoughts and tool calls—not through deterministic `<Result>` feedback—forcing the model to learn decision-making logic rather than memorize returns.
- **Comprehensive evaluation**: SOTA across all metrics on 300 NeurIPS-2025 anchor papers scored by a 4-LLM judge panel (+11.91% Overall vs. best baseline), corroborated by a double-blind human study (+9.88% Overall vs. strongest baseline), plus a >10× synthesis-efficiency gain (12 → 1 rollouts per valid sample).

## Methodology
Agentic-Ideation has three integrated stages.

**1. Agentic Tool Space Definition.** The tool space is formalized as T = T_ext ∪ T_think over interaction history H.
- *External tools* (T_ext) interface with an academic database. Only `Search(q)` returns full metadata (abstract, citation count, year); `Get_References(d_id)` and `Get_Cited(d_id)` return title lists only to keep context shallow and simulate realistic constraints.
- *Thinking tools* (T_think) model latent cognitive steps: `Analyse_Gap(H)` identifies a "research void"; `Ideation(H, o_gap)` generates a novel idea; `Reflection(H, o_idea)` critiques and triggers refinement.

**2. Oracle-Guided Agentic Data Synthesis.** A hierarchical multi-agent system operates on T. An "Anchor Paper" is selected and its bibliography seeds the input H_0.
- *Planner (A_plan)* generates a high-level research roadmap P = A_plan(H_0).
- *Controller (A_ctrl)* executes step-by-step, producing a reasoning thought and an action at at each step.
- **The core efficiency trick is the Dual-Thought mechanism.** Naive rejection sampling is prohibitively wasteful (an unguided agent rarely converges on a specific target idea). So the anchor paper's core idea—*Reference Idea (I_ref)*—is injected as oracle guidance. At each step the controller simultaneously emits two streams:
  - *Private Thought (th_priv)* — the "Oracle View": uses I_ref to compute the optimal next tool a_t to bridge the gap to the target, collapsing the search space with directional foresight.
  - *Public Thought (th_pub)* — the "Researcher View": rationalizes the action a_t using only visible history H_t, with I_ref masked, so the reasoning trace mimics natural autonomous exploration suitable for training.
- Synthesis ends when `Reflection` is invoked; trajectories are kept only if the generated idea semantically recovers I_ref. This yields a **valid trajectory in a single rollout** versus ~12 for rejection sampling.

**3. Agentic Supervised Fine-Tuning.** Traces are linearized with special tokens (`<Tool_Name>Query</Tool_Name>`, `<Result>Output</Result>`, `<Answer>Final Idea</Answer>`). Only the Public Thought is retained (Private Thought and oracle guidance are discarded) so the trained model can't "cheat." Crucially, a binary mask M_t zeroes out loss on tokens inside `<Result>...</Result>` and system instructions, back-propagating only through the agent's own decisions.

**Implementation.** Synthesis uses Qwen3-235B-A22B-Instruct-2507 for all agents (Planner, Controller, tools). The target agent is Qwen3-8B, fully fine-tuned on 4,646 synthesized trajectories, 4× A100 (80GB), ~10.5 hours, 5 epochs, LR 5e-5, cosine schedule.

## Results & Highlights
- **Main result (Table 1, 4-LLM judge panel, 300 NeurIPS-2025 papers):** Agentic-Ideation scores **6.67 Overall** vs. best baseline 5.96 (SciPIP), a **+11.91%** improvement, with gains on Novelty (+11.40%), Significance (+11.14%), and Feasibility (+8.70%). All deltas are statistically significant (p < 0.05). Gains are consistent across NLP, CV, and Other domains.
- **Synthesis efficiency (Table 2):** Rejection sampling needs **12 rollouts** on average for one valid sample (14 for NLP, 9 CV, 13 Others); Agentic-Ideation needs **1**—a >10× efficiency gain that makes large-scale agentic data synthesis tractable.
- **Ablation (Table 3):** Removing any single tool degrades all metrics. `Search` removal gives the lowest Overall (5.99)—without broad external queries the agent is confined to initial references. Removing `Analyse_Gap` causes the largest Novelty drop (5.86), confirming its role in surfacing a non-trivial "research void."
- **Human study (Table 4, double-blind):** Overall 6.45 vs. strongest baseline SciPIP 5.87 (**+9.88%**); largest gain in Significance (+12.28%), and +8.62% Feasibility, aligning with automated metrics.
- **Qualitative trajectory (Fig. 4):** The agent explores diffusion-model error compounding, chains `Get_Cited` and `Search` to connect "Self-Guidance," "ReMDM," and "Corrective Diffusion," and crucially uses `Reflection` to *reject* an initial "Latent-Space Remasking" draft as too close to RePaint, iterating to a more novel final proposal—demonstrating self-correction.

## Critical Assessment
- **Novelty (moderate-high):** The two headline ideas—(a) oracle-guided trajectory reconstruction via a Dual-Thought mechanism, and (b) tool-result masking during SFT—are each grounded in prior work (teacher-forcing/dagger-style distillation; FireAct-style action masking). The genuine novelty is the *application-specific synthesis pipeline* that adapts these to the open-ended ideation setting where no ground-truth answer signal exists. It is a sound engineering synthesis rather than a fundamentally new learning paradigm.
- **Rigor (good):** Headline numbers carry statistical significance (p<0.05), the metric rubric is published, baselines are reimplemented on the *same* backbone and engine (GPT-4o) for fairness, and a human study corroborates automated scores. Caveat: all baselines use GPT-4o while Agentic-Ideation's *trained* model is Qwen3-8B—a reasonable design (it isolates the agentic-training contribution) but means the comparison is "trained 8B open model vs. GPT-4o-driven workflows," not strictly model-controlled.
- **Reproducibility (mixed):** Compute, hyperparameters, dataset construction, and metric rubrics are disclosed. However, **no code/data/model release is mentioned** in the paper body. The 4,646 synthesized trajectories, the academic database behind the three external tools, and the GPT-5-mini paper classification step are not obviously reproducible without those artifacts. The "semantic alignment" filter for keeping trajectories (idea must recover I_ref) is described qualitatively without a threshold.
- **Significance (high for the niche):** Directly attacks the dominant practical bottleneck—cost of synthesizing agentic training data in open-ended domains. The 12→1 rollout reduction is the most compelling and generalizable claim. If the recipe transfers beyond ideation, it could inform agentic data synthesis broadly.
- **Red flags / concerns:**
  - **Circularity risk:** The same group curates the oracle reference ideas (from ICLR/ICML/NeurIPS 2025 papers) *and* defines quality—possibly favoring ideas structurally similar to the training distribution. The NeurIPS-2025 test set overlaps in venue/era with the training anchor papers (ICLR/ICML/NeurIPS 2025), risking mild distributional leakage between synthesis and evaluation.
  - The >10× efficiency claim is clean (12 vs 1 rollouts) but is measured against *their own* rejection-sampling implementation, not a strong third-party method.
  - "Overall quality" is ultimately an LLM/human Likert judgment on ideas that were never *executed*—no downstream experimental validation that the ideas actually work.
  - Synthesis relies on Qwen3-235B; whether the recipe holds with weaker synthesizers is untested.

## Strengths
- Targets the real, expensive bottleneck of agentic training-data synthesis in open-ended domains, with a clean and well-motivated mechanism (oracle foresight via Dual-Thought).
- Strong, multi-pronged evaluation: 4-LLM judge panel + double-blind human study + statistical significance + per-domain breakdown + tool ablations.
- The 12→1 rollout efficiency result is concrete and arguably the paper's most transferable contribution.
- Fair baseline treatment: all workflow baselines run on the same GPT-4o engine; backbone (Qwen3-8B) zero-shot is included to isolate agentic-training gains.
- Transparent about limitations (Qwen3-8B scale; retrieval-only, non-executable tool space) in Appendix A.3.

## Limitations
- **No execution/empirical validation of generated ideas**—quality is judged only by LLM/human raters, never by running the proposed experiments, so "high-quality idea" ≠ "idea that works."
- **Train/test distribution proximity:** both synthesis anchors and test anchors are recent top-ML-conference papers (ICLR/ICML/NeurIPS 2025), with NeurIPS 2025 used for testing—potential mild leakage and unclear generalization to other fields (biology, chemistry, social science).
- **Backbone scale:** Qwen3-8B limits reasoning depth; authors acknowledge scaling may be needed.
- **Tool space is retrieval-only**—no executable tools (code, simulators), so the agent cannot prototype or validate its own ideas, capping Feasibility assessment at the reasoning level.
- **Oracle dependence:** requires a curated set of high-quality reference ideas; sourcing and selection bias of these seed ideas is not discussed.
- **No code/data/model release mentioned**, limiting independent reproduction of the >10× efficiency and quality claims.

## Relevance
本文聚焦于训练可自主进行科研构想的智能体 LLM，并提出可显著降低开放域智能体训练数据合成成本的方法，对关注 AI for Science、agentic LLM 训练范式以及低成本轨迹数据合成的读者具有直接参考价值。

## Questions for Further Exploration
- How robust is the >10× efficiency gain if the synthesizer is downgraded from Qwen3-235B to a smaller open model, or if reference ideas are lower-quality?
- Does the NeurIPS-2025 test set overlap in authorship/topic with the ICLR/ICML/NeurIPS-2025 training anchors, and how do results change on out-of-distribution fields (e.g., biomedicine, economics)?
- Can the generated ideas pass an *execution* test—e.g., are any of them implementable and do experiments confirm the predicted gains?
- Would replacing the masking strategy with a reinforcement-learning second stage (rewarding grounded, verifiable ideas) further improve Feasibility beyond +8.70%?
- How does Agentic-Ideation compare to recent training-based agents (Search-R1, WebSailor) adapted to ideation, rather than only to workflow baselines?
- What is the failure mode of the Dual-Thought mechanism—if the Public Thought can't rationalize the oracle-driven action, does the trajectory become incoherent, and how often is such a trajectory rejected?
- Does releasing the 4,646 trajectories and tool-DB enable reproduction of the 11.91% gap, and is the gap stable under judge-panel perturbation (e.g., removing one judge)?
