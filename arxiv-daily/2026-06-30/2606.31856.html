Low-dimensional topology of deep neural networks

Junyu Ren  ·  Lek-Heng Lim — University of Chicago

↗ code ICML 2026 · PMLR 306 linking-number network-expressivity topology-of-neural-networks

Key finding

A monotonic-activation feedforward net cannot unlink linked data no matter how deep it is — but ResNet skip connections, attention, and nonmonotonic activations fold the representation to escape the barrier.

01 Key metric

100%

GELU Hopf-link separation vs ReLU ceiling ≤ 92.8%

On the Hopf link (30 seeds × 12 depths), nonmonotonic GELU reaches 100% best-run at depths 5–12 and breaks the predicted ceiling; ReLU plateaus and degrades with depth.

02 Why this matters

A topological theory ranking ResNet ≈ transformer > monotonic FFN > flow/ODE models — and a unified explanation for why skip connections, attention, and GELU consistently beat alternatives.

!

03 Critical edge

Expressivity ≠ optimization. The theory says what a net can represent, not what gradient descent finds — real CIFAR-10 gaps conflate the topological barrier with optimization dynamics the theory does not model.

§ 01 — 02

How the argument is built & what it shows

Fix each layer's representation to ℝ³ (then generalize to ℝd). The entry point is the Hopf link — two interlocked circles with linking number ±1. A separating hyperplane forces each class into a convex half-space, so linear separability requires the linking number to be zero. Classifying linked data means unlinking it first.

link(X,Y) = (1/4π) ∬ (x−y)·(dx×dy) / ‖x−y‖³

The engine is a layer-by-layer preservation argument, then a trio of escape routes — all of which hinge on a coordinate fold:

  1. AInvertible affine layers are homeomorphisms → preserve linking number up to sign.
  2. BMonotonic activations (ReLU, sigmoid, tanh) preserve it via a simultaneous straight-line homotopy on both components.
  3. CRank-deficient layers force intersections whenever link ≠ 0 (SVD → projection).
  4. ∴Impossibility (Thm 4.7): to drive link → 0 you need rank-deficient layers, but those force the classes to intersect — so no monotonic width-d FFN, CNN, or bottleneck autoencoder separates linked data, at any depth.
  5. ×The fold: |x| = x + 2·ReLU(−x) breaks the homotopy. ResNet builds it from ReLU + skip (Thm 5.2); two-token attention builds a smoothed V-shape (Thm 5.3); GELU/Swish/Mish supply it directly (Thm 5.1).

§ 03

Strengths & honest limitations

＋ Strengths

  • Genuinely original, mathematically rigorous reframing of relative architecture expressivity — usually settled only empirically.
  • Clean comparative verdict: ResNet ≈ transformer > monotonic FFN > flow/ODE; nonmonotonic activation is the single lever that elevates FFNs.
  • Pins activation nonmonotonicity as the precise, designable lever — |x| = x + 2·ReLU(−x) is a memorable identity.
  • Works in a visualizable space (ℝ³) and rigorously generalizes to d > 3.
  • The extrinsic (ambient-embedding) framing sharply distinguishes it from prior intrinsic-TDA work.
  • Candid about limitations; code released; ICML-reviewed.

－ Limitations

  • Expressivity-only theory — no model of optimization, trainability, or generalization dynamics.
  • The d = 3 / width-d regime is far from large practical nets; the d > 3 generalization's empirical tightness is unverified at scale.
  • CIFAR-10 results are correlational, measured on PCA-3D projections (not the native 3072-D manifold), with modest effect sizes (+1.2% on deer–dog).
  • Only pairwise linking number is treated; richer invariants (Milnor μ̄, knot types) deferred to a companion paper.
  • No new architecture or benchmark — the payoff is interpretive / heuristic.
  • The "transformer = ResNet" equivalence rests on a minimal two-token, single-head construction, not a full trained transformer.
§ 04

Open questions for further work

  1. Does the expressivity hierarchy quantitatively survive as width grows from d = 3 toward the thousands of real networks?
  2. Can the linking-number framework become a quantitative design metric — a "linking-aware" loss or NAS objective that measurably improves architecture search?
  3. How do the deferred invariants (Milnor μ̄, link groups, knot types, ambient cobordisms) refine or overturn the current ordering?
  4. Do nonmonotonicity's gains on synthetic links translate to large-scale tasks beyond CIFAR-10 binary pairs?
  5. How does this topological axis correlate — or conflict — with optimization landscape, trainability, and generalization? Can the theory describe what gradient descent actually finds?
  6. Does the two-token attention fold scale to realistic multi-head, multi-layer, trained transformers — or does richer attention introduce different topological power?
