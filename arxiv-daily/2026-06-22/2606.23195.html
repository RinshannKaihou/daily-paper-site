Memory Contagion: Cross-Temporal Propagation of Evaluator Bias via Agent Memory

TL;DR

This paper identifies and formalizes "Memory Contagion" -- a phenomenon where evaluator biases (e.g., preferring longer responses or authority-signaling language) propagate across time through agent memory systems. Even with perfect (oracle) memory consolidation, biased input alone is sufficient to cause contagion. Consolidation has bias-type-dependent effects: it attenuates length bias (84% reduction) but preliminarily amplifies authority bias (47% increase). No safe contamination threshold exists -- bias propagation is detectable when only 20% of memories are contaminated.

本文识别并形式化了"记忆传染"(Memory Contagion)现象：评估器偏差（如偏好更长回复或权威性语言）通过智能体记忆系统跨时间传播。即使在完美（oracle）记忆整合条件下，有偏差的输入本身就足以引发传染。整合过程对不同偏差类型有不同效果：它减弱长度偏差（降低84%），但初步证据表明可能放大权威性偏差（增加47%）。不存在安全的污染阈值——仅20%的记忆被污染时就能检测到偏差传播。

Abstract

LLM agents increasingly rely on memory systems for long-term coherence. Prior work (Zhang et al., 2026) shows that agent memories degrade during continuous consolidation, but assumes unbiased input experiences. This paper identifies Memory Contagion -- the cross-temporal propagation of evaluator bias through agent memory. When agents are trained or guided by biased evaluators, their trajectories become biased; when stored and consolidated into memory, the bias propagates to future agents retrieving from the same memory store, even with perfect consolidation. Across two bias types (length preference, authority bias) and four experimental phases, the authors demonstrate: (1) Memory Contagion occurs even with perfect consolidation (oracle condition, Gamma_A = 13.18 for length, 11.45 for authority); (2) Consolidation has bias-type-dependent effects, robustly attenuating length bias (Gamma_B = 2.03) while preliminarily amplifying authority bias (Gamma_B = 16.80); (3) No observed safe threshold -- bias propagation is detected at contamination rates as low as p = 0.2.

LLM智能体日益依赖记忆系统以维持长期连贯性。已有工作表明智能体记忆在持续整合过程中会退化，但假设输入经验是无偏的。本文识别了"记忆传染"现象——评估器偏差通过智能体记忆的跨时间传播。当智能体被有偏差的评估器训练或引导时，其轨迹变得有偏差；当这些轨迹被存储并整合到记忆中时，偏差会传播给从同一记忆库中检索的未来智能体，即使整合过程是完美的。实验涵盖两种偏差类型和四个实验阶段，证明了：（1）即使完美整合也会发生传染；（2）整合对不同偏差类型有不同效果；（3）不存在安全阈值。

Problem Statement / 问题陈述

Autonomous LLM agents use memory systems to store, consolidate, and retrieve past experiences. Existing research on memory degradation (Zhang et al., 2026) focuses on the consolidation mechanism itself and assumes clean inputs. However, in practice, agents are often guided by biased evaluators -- reward models with systematic preferences, human feedback with inherent biases, or LLM-as-judge systems. The paper asks: when biased experiences enter the memory store, does the bias propagate across time to future agents, and what role does consolidation play?

自主LLM智能体使用记忆系统来存储、整合和检索过往经验。现有关于记忆退化的研究关注整合机制本身，假设输入是干净的。然而实际中，智能体经常受到有偏差的评估器引导——具有系统性偏好的奖励模型、带有固有偏差的人类反馈、或LLM评判系统。本文提出的问题是：当有偏差的经验进入记忆库时，偏差是否会跨时间传播给未来的智能体？整合过程扮演什么角色？

Method / 方法

Problem Formulation

The framework involves a source agent A_s interacting with tasks T and evaluated by a biased evaluator E_b. Trajectories are stored in memory store M. A target agent A_t retrieves from M to solve new tasks. A biased evaluator is modeled as E_b(tau) = E_clean(tau) + alpha * b(tau), where b is a bias function. Two bias types are studied:

  • Length bias: b(tau) = (len(a) - mu_L) / sigma_L (preference for longer responses)
  • Authority bias: b(tau) = n_auth(a) / n_total (density of authority-signaling phrases like "studies show," "according to research")

框架涉及一个源智能体A_s与任务T交互，由有偏差的评估器E_b评估。轨迹存储在记忆库M中。目标智能体A_t从M中检索来解决新任务。两种偏差类型：长度偏差（偏好更长回复）和权威性偏差（权威信号短语密度）。

Measurement: Gamma_temporal

The key metric is Gamma_temporal = D_behavior(A_t|M_b, A_t|M_c), measuring the Wasserstein distance between a target agent's behavior when retrieving from biased vs. clean memory. Gamma > 0 indicates bias propagation; statistical significance is assessed via permutation test (p < 0.01).

核心指标Gamma_temporal使用Wasserstein距离衡量目标智能体从有偏差记忆库 vs. 干净记忆库检索时的行为差异。Gamma > 0表示偏差传播。

Memory Consolidation

Two strategies are compared:

  • Oracle consolidation: Perfect merge without information loss (upper bound)
  • LLM consolidation: Merge via LLM-based summarization (realistic)

Mechanism Decomposition

Total contagion is decomposed as: Gamma_total = Gamma_content (E1) + Gamma_retrieval (E2) + Gamma_interaction (E3), separating content-based contamination from retrieval-based contamination.

4-Phase Experimental Pipeline

  • Phase 1 (Bias Injection): Source agent solves 20 QA tasks; rejection sampling selects trajectories maximizing biased evaluator scores
  • Phase 2 (Memory Construction): Biased/clean trajectories stored; consolidation applied
  • Phase 2.5 (Oracle Consolidation Ablation): Core test -- does contagion persist with perfect consolidation?
  • Phase 3 (Mechanism Decomposition): Controlled experiments isolating content vs. retrieval contributions
  • Phase 4 (Dose-Response): Varying contamination rate p from 0.2 to 1.0

实验采用DeepSeek-Chat作为基座LLM，温度设为0以确保可重复性，嵌入使用MiniLM-L6-v2。四阶段流水线：偏差注入、记忆构建、机制分解、剂量-响应分析。

Key Results / 主要结果

1. Memory Contagion Occurs Even with Perfect Consolidation

Oracle consolidation yields Gamma_A = 13.18 (length, 95% CI [9.05, 21.10], 3 seeds) and Gamma_A = 11.45 (authority, single exploratory run), both significantly > 0. This proves biased input is a sufficient cause of contagion -- the problem is not solely in the consolidation mechanism.

即使完美整合也无法阻止传染。Gamma_A对长度偏差为13.18（3个种子），对权威性偏差为11.45（单次探索性运行），均显著大于0。这证明有偏差的输入本身就是传染的充分原因。

2. Consolidation Has Bias-Type-Dependent Effects

  • Length bias: LLM consolidation attenuates contagion (Gamma_B = 2.03, 84% reduction vs. oracle). Summarization compresses long texts, reducing the global length signal.
  • Authority bias: LLM consolidation preliminarily amplifies contagion (Gamma_B = 16.80, 47% increase vs. oracle; single-run estimate). Summarization preserves and concentrates local authority markers while removing peripheral content. Authority marker density increases from 3.2 to 5.1 per 100 words after consolidation.

长度偏差：LLM整合减弱传染（降低84%）。权威性偏差：LLM整合初步显示放大传染（增加47%，单次运行估计）。整合后权威标记密度从每百词3.2增至5.1。

3. Content-Based Contagion Dominates

Phase 3 decomposition shows E1 (content) and E2 (retrieval) are comparable contributors, with content slightly larger. E3 (interaction) is small and not statistically distinguishable from zero. Debiasing is challenging: E4 efficacy ratio is 0.90 for length but only 0.51 for authority.

内容污染和检索偏差均有贡献，内容略占主导。去偏差很困难：长度偏差去偏效率0.90，权威性偏差仅0.51。

4. No Safe Contamination Threshold

At p = 0.2 (only 20% of memories biased), Gamma(p) > 0 for both bias types. Length bias shows monotonic increase with contamination rate; authority bias shows a nonlinear peak-and-decline pattern (peaking at p = 0.8, Gamma = 23.03, then declining at p = 1.0).

在仅20%记忆被污染时即可检测到传播。长度偏差随污染率单调增加；权威性偏差呈非线性峰值-下降模式（在p=0.8处峰值23.03）。

5. Authority Bias is Domain-Dependent

Multi-seed replication on synthetic tasks showed Gamma_A ~ 0 for authority bias, suggesting authority bias contagion requires fact-grounded contexts where evaluators can meaningfully distinguish authority-signal density -- a condition absent in open-ended synthetic QA.

多种子重复实验表明权威性偏差传染需要事实基础的上下文，在开放式合成QA中不会传播。

Technical Contributions / 技术贡献

  1. Formalization: Introduces Gamma_temporal as a quantitative measure of cross-temporal bias propagation via memory
  2. Mechanism decomposition: Separates content-based vs. retrieval-based contagion pathways with controlled experiments
  3. Consolidation asymmetry: Discovers that consolidation can either attenuate or amplify bias depending on whether the bias signal is global (length) or local (authority markers)
  4. Dose-response analysis: Establishes that no safe contamination threshold exists

Critical Assessment / 批判性评估

Strengths

  • Identifies a genuinely novel and practically important vulnerability in agent memory systems that prior work overlooked by assuming clean inputs
  • Clean experimental design with oracle consolidation as an upper bound, effectively separating input bias from consolidation artifacts
  • The directional asymmetry finding (consolidation attenuates length bias but amplifies authority bias) is a non-obvious insight with design implications
  • Honest about limitations: authority bias results are from a single exploratory run and are presented as upper-bound motivation rather than confirmed findings

Weaknesses

  • Single-model limitation: All experiments use DeepSeek-Chat only; consolidation effects may be model-specific. No replication with other model families (Llama, Qwen, GPT).
  • Authority bias results are preliminary: The key finding that consolidation amplifies authority bias comes from a single run without confidence intervals. Multi-seed replication on synthetic tasks yielded Gamma_A ~ 0 for authority, raising questions about robustness.
  • Rejection sampling confound: The bias injection method (rejection sampling for highest evaluator score) may co-select for quality features correlated with bias (fluency, coherence), inflating contagion estimates.
  • Only two bias types studied: Length and authority are relatively surface-level. More subtle biases (confirmation bias, political bias) are left to future work.
  • Small task set: Only 20 open-ended QA tasks; unclear how results generalize across domains.
  • Additive separability assumption: The model E_b = E_clean + alpha * b assumes clean quality and bias scoring are separable, which is a working assumption that may not hold in practice.
  • No real-world validation: All evaluators are synthetic; real-world evaluator biases may behave differently.

优点

  • 识别了智能体记忆系统中一个真正新颖且实际重要的脆弱性
  • 使用oracle整合作为上界的实验设计清晰有效
  • 整合方向不对称性发现具有非显然的设计启示
  • 对局限性诚实透明

不足

  • 仅使用单一模型（DeepSeek-Chat），结果可能不具普遍性
  • 权威性偏差的关键结果来自单次运行，缺乏置信区间
  • 拒绝采样注入方法可能与质量特征共选择，膨胀传染估计
  • 仅研究两种相对表面的偏差类型
  • 任务集小（仅20个QA任务），领域泛化性不明
