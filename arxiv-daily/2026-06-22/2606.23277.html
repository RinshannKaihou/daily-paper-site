GIF: Locally Sound Geometric Information Flow Control for LLMs

Summary / 概述

English:

This paper introduces Geometric Information Flow (GIF), a principled semantic framework for tracking how information flows from input token spans to downstream outputs in LLM-based agentic systems. GIF addresses a critical gap in LLM security: while LLMs increasingly mediate interactions between trusted instructions, untrusted inputs, private context, and privileged actions, existing Information Flow Control (IFC) methods suffer from severe taint explosion because any input token can weakly influence any output. GIF formalizes information flow as the Shannon mutual information between local embedding perturbations and the model's next-token distribution, approximated via a local Gaussian surrogate channel built from the LLM's Jacobian geometry. The authors provide a fully mechanized Lean 4 proof that GIF is locally sound -- it never asymptotically undercounts true information flow. Evaluated across three agent benchmarks (AgentDojo, MSB, AgentDAM) and six open-weight models, GIF achieves near-perfect recall for detecting prompt injection and privacy leakage, consistently outperforming attention-based and gradient-based baselines while reducing inference cost by up to 81x compared to GPT-5.5 xhigh judges.

Methodology / 方法论

Core Formulation / 核心公式化

GIF builds on three key ideas:

1. Local Perturbation Model / 局部扰动模型: For a token span $T$ in a prompt $x$, GIF perturbs the span's embeddings by a small isotropic Gaussian noise $U_T \sim \mathcal{N}(0, \tau I_{d_T})$ and measures how much the model's next-token distribution shifts. The local regularity assumption (formalized in Lean 4) states that the hidden-state map $h_T$ is well-approximated by its Jacobian $J_T(x)$ for small perturbations.

2. Gaussian Surrogate Channel / 高斯代理通道: Rather than computing exact mutual information (intractable for large models), GIF constructs a local Gaussian surrogate channel $Y_T = A_T(x) U_T + N$ where $A_T(x) = F_{\text{sm}}(p(x))^{1/2} W J_T(x)$ is the "Fisher factor" -- the composition of the Jacobian, output head, and softmax-Fisher square root. The capacity of this channel is:

$$\text{GIF}_T(\tau; x) := \text{Cap}_T(\tau; x) = \frac{1}{2} \log \det(I_{d_T} + \tau M_T(x))$$

where $M_T(x) = J_T(x)^\top W^\top F_{\text{sm}}(p(x)) W J_T(x)$ is the "Fisher pullback" matrix.

3. Influence as First-Order Proxy / 影响力作为一阶代理: The influence $\text{Inf}_T(x) := \text{tr}\, M_T(x)$ serves as the leading-order rate of the capacity, providing a cheaper linear proxy. It is estimated via Hutch++ randomized trace estimation using only $\ell \geq 2 + 4/(\varepsilon\sqrt{\delta})$ matrix-vector oracle queries per span.

System Architecture / 系统架构

The GIF system has three components:

  • Labeling: Assigns security labels (High/Low) to tokens following standard IFC practice.
  • Flow Tracking: Computes GIF scores between each prefix token and generated sink tokens during autoregressive generation using the Hutch++ approximation.
  • Declassifier: Either an oracle threshold declassifier (based on Policy Source Precision at k) or an LLM-as-a-declassifier that receives GIF-highlighted top-$k$ source spans as compact evidence.

Key Results / 关键结果

RQ1: Policy Signal Quality / 策略信号质量

  • GIF improves average AUROC/AUPRC over GPT-5.5 xhigh by 11.2%/12.0% while reducing attribution cost by 9.3x ($147.90 to $15.98).
  • GIF consistently outperforms attention-based RTBAS and single-logit GIF$^-$ across most model-dataset combinations.

RQ2: LLM Declassification / LLM解密

  • Using only top-$k$ GIF-highlighted evidence, LLM declassifiers match or improve the full-trajectory judge (0.97-1.22x performance) while reducing token cost by up to 3.7x.
  • GIF-3 with Qwen 3 8B can match GPT-5.5 xhigh performance on AgentDojo and MSB despite being 81x cheaper.

RQ3: Robustness / 鲁棒性

  • Hutch++ approximation: Spearman correlation with exact influence reaches 0.957 at rank 1, 0.988 at rank 64. Low-rank Hutch++ (rank 8) is sufficient for declassification.
  • Surrogate transfer: A fixed Qwen 3 0.6B surrogate preserves downstream detection performance almost exactly when analyzing trajectories from models up to 200x larger (GPT OSS 120B).
  • Attribution cutoff $k$: Both AUROC and AUPRC saturate by $k \approx 30$, reaching 95% of total gain.

RQ4: Qualitative Flow Analysis / 定性流分析

  • GIF correctly identifies malicious flow in partial-success prompt injection attacks that benchmark oracles miss (the injected address receives 31% flow vs. RTBAS attributing it to neutral context).
  • GIF assigns zero outflow to private information spans that never surface in outputs, avoiding the over-tainting problem of attention-based methods (RTBAS falsely assigns 9%).

Critical Assessment / 批判性评估

Strengths / 优势:

  • The Lean 4 mechanized proof of local soundness is a genuine contribution -- this is the first formal guarantee connecting Jacobian geometry to information-theoretic flow semantics in LLMs, moving beyond purely heuristic attribution.
  • The Hutch++ estimator makes the framework practical at scale, and the cross-model surrogate transfer result (200x size gap with minimal degradation) is practically significant for deployment.
  • The separation between flow tracking (GIF's contribution) and policy decision (declassifier) is a clean architectural choice that makes the framework composable.

Weaknesses / 不足:

  • The local soundness guarantee holds only asymptotically as perturbation scale $\tau \to 0^+$. The paper does not provide finite-$\tau$ bounds, leaving a gap between theory and the practical perturbation scales used in experiments.
  • The Gaussian surrogate channel is faithful only to second order (Theorem 3.13). For tokens where the model's behavior is highly nonlinear locally, the surrogate may be inaccurate, though the paper does not systematically study when this breaks down.
  • The labeling component uses a simple two-point lattice (High/Low), and the paper explicitly defers more sophisticated labeling policies to future work. In realistic deployments, the boundary between trusted and untrusted content is often ambiguous.
  • Evaluation uses only open-weight models (Qwen, Gemma, GPT OSS). While surrogate transfer is demonstrated, GIF fundamentally requires gradient access to at least a surrogate model, limiting applicability to truly closed-box APIs where no similar-architecture open model exists.
  • The benchmarks (AgentDojo, MSB, AgentDAM) test relatively short agent trajectories. Scalability to very long multi-turn interactions with complex tool chains remains to be demonstrated.

Missing Comparisons / 缺失的比较:

  • No comparison against recent causal mediation analysis or activation patching methods that also provide fine-grained token-level attribution.
  • The paper compares against GPT-5.5 xhigh as a judge but does not compare against specialized prompt injection detectors that use fine-tuned classifiers.

Relevance to Broader Research / 与更广泛研究的相关性

This work sits at the intersection of LLM security, information theory, and formal verification. It provides a principled alternative to the ad hoc attention/gradient-based attribution methods commonly used in prompt injection defense, and the Lean 4 formalization sets a new standard for rigor in this space. The surrogate transfer finding has implications for deploying security monitoring in production systems where the monitored model may be proprietary. The framework is complementary to existing agentic IFC systems (which track flow around the model) by providing the missing piece: tracking flow through the model itself.

Reproducibility / 可复现性

  • The Lean 4 formalization is publicly available at https://geomifc.github.io/.
  • Experiments use open-weight models (Qwen 3, Gemma 4, GPT OSS) on NVIDIA RTX PRO 6000 GPUs.
  • Three public benchmarks (AgentDojo, MSB, AgentDAM) are used.
  • Implementation uses PyTorch 2.12, vLLM 0.22.0, HF transformers 5.9.0.
