Variational Linear Attention: Stable Associative Memory for Long-Context Transformers

arXiv: 2605.11196

Date: 2026-05-13

https://arxiv.org/abs/2605.11196

一句话总结： 本文提出变分线性注意力（VLA），将线性注意力的记忆更新重新建模为在线正则化最小二乘问题，通过Sherman-Morrison公式维护自适应惩罚矩阵实现方向选择性写入，理论上证明了状态范数自限制和Jacobian谱范数为1，在多查询关联回忆任务中将状态范数降低109倍并维持完美检索精度。
TL;DR: VLA reframes linear attention memory updates as online regularised least-squares with an adaptive penalty matrix maintained via Sherman-Morrison updates, achieving 109x lower state norm, perfect associative recall within capacity, and O(T) scaling with stable gradients at arbitrary depth.

Key Contributions

  • Introduces VLA, replacing linear attention's unconditional accumulation with a residual-error update governed by a d×d adaptive penalty inverse matrix A_t maintained via the Sherman-Morrison rank-1 formula.
  • Proves that unit-normalising both the key and gating vector yields a recurrence Jacobian with spectral norm exactly 1 for all sequence lengths and head dimensions (Proposition 2), and that state norm is self-limiting under bounded inputs (Proposition 1).
  • Derives a parallel Blelloch-scan formulation and a fused Triton kernel achieving 14x speedup over sequential Python with O(T) scaling, crossing below softmax attention latency at ~43K tokens.
  • Demonstrates near-perfect exact-match accuracy on MQAR within the per-head memory regime (n < d_h), maintaining 62% accuracy at the capacity boundary while DeltaNet and linear attention collapse to near-random.

Methodology

VLA reformulates the linear attention memory state S_t as the solution to an online regularised least-squares problem. The key insight is to introduce a time-varying penalty matrix M_t that encodes the geometry of previously seen keys, giving the model control over which memory subspaces are protected from overwriting.

Core update mechanism: The penalty inverse A_t = M_t^{-1} is maintained exactly using the Sherman-Morrison rank-1 formula at O(d_h^2) cost per step. A_t accumulates outer products of learned penalty directions u_t, so subspaces that were recently written receive smaller updates — a matrix-valued analogue of DeltaNet's scalar gate. The memory update follows:

  • Compute prediction residual: e_t = v_t - S_{t-1} k_hat_t
  • Compute gating direction: alpha_hat_t = A_t k_hat_t / ||A_t k_hat_t|| (unit-normalised)
  • Update state: S_t = S_{t-1} + e_t alpha_hat_t^T

Parallel formulation: The S_t recurrence is expressed as a linear recurrence (F_t, G_t) that is associative under composition, enabling a Blelloch parallel prefix scan in O(log T) parallel steps. The A_t loop has a data-dependent denominator preventing direct parallelism, so it is fused into a single Triton kernel.

Key normalisation: Both k_hat_t and alpha_hat_t are unit-normalised. This is critical — without normalisation, the Jacobian spectral norm grows as d_h/lambda_0, producing gradient magnification of ~10^32 after 25 steps at d_h=96.

The framework strictly generalises both standard linear attention (setting u_t=0 recovers additive accumulation) and DeltaNet (replacing alpha_hat_t with a scalar gate recovers the error-corrective update).

Results & Highlights

  • State stability: ||S_t||_F is reduced by 109x relative to standard linear attention at T=1,000 (below 15 vs. ~1,630). DeltaNet also remains bounded but with uniform scalar decay rather than direction-selective control.
  • MQAR capacity (n_pairs < d_h=32): VLA achieves 1.000 exact-match accuracy at n=24 while DeltaNet collapses to 0.010 and linear attention to 0.074 (mean over 3 seeds, std=0.000).
  • Capacity boundary (n=d_h=32): VLA retains 62.3% accuracy while all baselines collapse to near-random (~0.056-0.057).
  • Long-context MQAR (n=8, T=64-512): VLA maintains flat 1.000 accuracy at all sequence lengths. All baselines plateau below 0.16.
  • Scaling: VLA-Python is ~3x slower than linear attention due to SM update overhead. The Triton kernel achieves 14x speedup over VLA-Python at T=4,096, crossing below softmax latency at ~43K tokens.
  • Ablation: Removing SM update (fixing A_t=10I) collapses VLA to standard linear attention accuracy. Removing key normalisation causes NaN losses at d>=96 due to Jacobian explosion.
  • OOD generalisation: VLA shows some embedding dependence — OOD accuracy drops from 1.000 (ID) to 0.687 (OOD) at n=24, though still far above random (0.008).

Critical Assessment

  • Novelty: High. The connection between linear attention and recursive least squares is novel and well-motivated. The Sherman-Morrison formulation for adaptive penalty geometry in attention is a genuinely new contribution. The hierarchy (linear attention → DeltaNet → VLA) is cleanly articulated.
  • Rigor: Strong. Formal propositions with proofs, unit Jacobian guarantee, bounded state dynamics. The authors are commendably honest about results that could be overstated — e.g., explicitly noting that DeltaNet and VLA state norms are indistinguishable on the shared scale in Figure 4.
  • Reproducibility: Moderate. Complete pseudocode (Algorithms 1-2) and hyperparameter tables are provided. However, the code is private. The Triton kernel implementation details are not fully specified.
  • Significance: Moderate-to-High. The theoretical contribution is strong. However, the evaluation is limited to synthetic MQAR tasks with a tiny 2-layer model (d=128, 4 heads, d_h=32). No language modelling perplexity, no real-world benchmarks, no scaling experiments beyond d=128. The practical impact depends entirely on whether these results transfer to realistic model scales.
  • Red flags:
  • Evaluation scope is extremely narrow — only synthetic associative recall with a 128-dim, 2-layer model.
  • Per-head capacity of d_h=32 is very small; at realistic head dimensions (d_h=128), the O(d_h^2) state cost becomes significant.
  • OOD generalisation shows substantial degradation (31pp drop at n=24), suggesting the mechanism partially exploits embedding geometry rather than learning purely content-addressable retrieval.
  • The overload regime (n>d_h) shows VLA collapses similarly to baselines, so the advantage is strictly within capacity.
  • No comparison with other efficient attention mechanisms like Mamba, flash attention, or sliding window attention.

Strengths

  • Elegant theoretical framework connecting linear attention to classical RLS adaptive filtering.
  • Clean proofs of state boundedness and unit Jacobian spectral norm.
  • VLA strictly generalises both standard linear attention and DeltaNet, with clear ablation confirming each component's necessity.
  • Honest reporting: authors acknowledge when DeltaNet matches VLA, when results are single-seed, and when overload performance collapses.
  • The Blelloch-scan parallel formulation and Triton kernel show practical engineering consideration.

Limitations

  • Evaluated only on synthetic MQAR with a 2-layer, 128-dim model — no language modelling, no real-world tasks.
  • Per-head associative capacity is fundamentally limited to d_h associations; at realistic dimensions this remains a tight constraint.
  • The O(d_h^2) per-step cost of the Sherman-Morrison update, while asymptotically O(T d_h^2), has a ~5x constant factor overhead over standard linear attention.
  • OOD key generalisation degrades substantially, indicating partial reliance on embedding geometry.
  • No scaling experiments to larger models or longer sequences (max T=512 in MQAR, T=4096 in latency tests).
  • The periodic identity refresh (every 20 steps, +10^{-3}I) is a heuristic without theoretical justification.

Relevance

本文将线性注意力与递归最小二乘（RLS）联系起来，通过Sherman-Morrison更新实现方向选择性记忆管理，对高效长上下文Transformer架构设计有重要启发。理论证明的状态有界性和梯度稳定性为设计数值稳定的常数记忆序列模型提供了新思路。

Questions for Further Exploration

  • How does VLA scale to realistic model sizes (e.g., 1B+ parameters, d_h=128) and does the MQAR advantage transfer to language modelling perplexity?
  • Can the per-head capacity limit (d_h associations) be overcome by cross-head coordination or hierarchical memory?
  • What is the empirical crossover point for VLA vs. flash attention on modern hardware (A100/H100) with realistic batch sizes?
  • How does VLA interact with mixture-of-experts layers or multi-query attention architectures?
  • Does the penalty direction u_t need to be learned, or can a fixed analytically-derived direction (e.g., u_t = k_hat_t) achieve comparable performance?
  • Can the Sherman-Morrison update be approximated with a low-rank factorisation to reduce the constant factor overhead at large d_h?
