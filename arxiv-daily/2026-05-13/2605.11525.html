Key Contributions

  • Introduces OverNaN, a lightweight NaN-aware oversampling framework that generates synthetic samples directly from incomplete feature vectors
  • Reframes missingness as part of the feature space rather than a defect to be eliminated
  • Provides configurable strategies for preserving, propagating, or selectively interpolating missing values during synthetic sample generation
  • Demonstrates that meaningful missingness can be retained during oversampling without introducing artificial certainty

Methodology

OverNaN extends common synthetic oversampling methods (e.g., SMOTE-like approaches) to operate on incomplete feature vectors. Rather than preprocessing missing values through deletion or imputation, the framework allows missing values to be handled according to explicitly defined strategies: they can be preserved as-is in synthetic samples, propagated through the oversampling process, or selectively interpolated. The paper positions the method within the intersection of imbalanced learning, missing-data handling, and NaN-tolerant algorithms. Evaluation uses representative examples bundled with the software.

  • Shows that preserving NaN structure avoids distorting class boundaries and introducing bias that can arise from imputation
