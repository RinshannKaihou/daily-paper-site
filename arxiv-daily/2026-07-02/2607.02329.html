The Key Finding

From 11,083 physics papers, an autonomous agent reaches a manuscript — by confronting published numbers, not merely citing them.

"Grounding" is redefined as structurally enforced numerical confrontation at calibration checkpoints: reproduce a reference, compare, then pass or refuse. Everything else is the apparatus that makes that one act honest.

§ 01 — METHODOLOGY Six phases, one grounding act. A blank-context pipeline that talks only through on-disk files. Conception runs unanchored; execution is structurally forced to reproduce and compare.

Three-layer fault tolerance. fresh-context isolation stops errors crossing sessions; distributed grounding spreads references across 14 access channels (2,162 events); adversarial review is prompted to find, not confirm, error.

Two deliberately absent scaffolds. A curated knowledge base (37 workflows, 17 scripts) and house rules (MUST-do diagnostics like Wannier band-fit verification) are withheld during conception (breadth / depth / pilot-S1) to avoid biasing open idea generation, then structurally loaded only from pilot-S2 onward.

This cleanly separates conception grounding (open exploration) from execution grounding (anchor enforcement).

The decisive ablation. A "no-pilot" run inherited the Ye-2026 reference value verbatim in its own files — yet never wrote the literal comparison "0.066 vs. 0.176", instead arguing divergences acceptable case-by-case and asserting an untested cancellation.

This isolates the mechanism: adversarial review alone is insufficient — structural reproduction enforcement is what closes the "rules cited, not enforced" gap.

A pre-architecture baseline, run as a paired failure mode, accepts an un-anchorable phonon-thermal-Hall direction, writes ~1,200 lines of toy-validated code, gets refuted, and terminates with no manuscript.

§ 01b — WORKED EXAMPLE The calibration trajectory. MnTe orbital magnetization as anchor for the strain-derivative computation — four steps from silent overshoot to honest refusal.
01
Initial overshoot
Agent's first result was 2.44× the published Ye-2026 value — and declared HIGH confidence on internal symmetry checks alone.
02
Anchor enforcement
A fresh-context adversarial sub-agent surfaced the reference; the reflect prompt forced the literal comparison — 2.44×, opposite sign — retracting HIGH → LOW.
03
Convergence refusal
Matched the reference at one mesh but 70% deviation at a denser mesh → agent issued NOT PASS rather than over-iterating.
04
Trace verified
Every parameter preserved on disk; post-hoc human convergence-ladder test confirmed the divergence direction the agent had flagged.

§ 03 — STRENGTHS / LIMITATIONS Exceptionally self-aware — yet the case rests on one direction.
Strengths▲ what holds
  • Genuinely end-to-end in a domain where execution does not calibrate the agent — harder and more realistic than ML sandboxes.
  • "Confront vs. cite" is conceptually clean and operationalized concretely: calibration checkpoints, a 4-tier verdict scale, and NOT-PASS propagation.
  • The no-pilot ablation is a near-ideal controlled experiment — same knowledge, same in-file anchor, only reproduction-confrontation removed.
  • Exceptional intellectual honesty: paired failure modes, a quantified intervention table, and a camera-ready retraction of the anchor's reliability.
  • A usefully scoped regime taxonomy (I execution-calibrated / II mature-anchors / III frontier) clarifies where the architecture is necessary.
Limitations▼ what to weigh
  • Single research direction; cross-direction and cross-domain generalization untested.
  • Multi-trial variance unquantified — each run costs ~6 days, so no run-to-run stability data.
  • No public code/data repository evident; reproducibility rests on released prompts + on-disk artifacts + an unspecified, API-evolving model.
  • Cannot yet escalate from replication to critique when the anchor itself is pathological — the residual failure mode this very run surfaced.
  • "Publication-grade" quality is self-assessed; the three physics findings are not independently peer-reviewed.
  • One author serves as architect, curator and validator — a potential confound for the autonomy claim (mitigated by archived transcripts).
§ 04 — OPEN QUESTIONS What this run deliberately leaves open.
Q1 · GENERALIZATION
Does the grounding behavior reproduce on a second, independently-chosen direction — or is it partly an artifact of this favorable, well-anchored topic?
Q2 · RELEASE
Will code, the curated knowledge base, house rules, and session artifacts be released — and under which LLM model version were the 47 sessions run?
Q3 · SCALING
How do cost and intervention rate scale with corpus size, anchor density, and toolchain maturity — does the bounded-human-intervention pattern plateau or grow?
Q4 · ABLATION
What is each fault-tolerance layer's marginal contribution when ablated independently, beyond the pilot/no-pilot pair already reported?
Q5 · ESCALATION
Can the architecture be extended with the scrutiny half to implement diagnostic-triggered escalation from replication to anchor-critique?
Q6 · TRANSFER
How would "confront, don't cite" transfer to domains where numerical reproduction of references is infeasible — wet-lab biology, clinical trials?
