Key Contributions

  • Introduces Global In-Degree Centrality that aggregates attention patterns across all network layers to identify tokens serving as global information hubs (not just locally salient at one layer)
  • Designs Dynamic Pivot Detection using information-theoretic metrics to adaptively locate the optimal layer for KV cache compression
  • Proposes Structural Propagation and Decoupling that separates computational budget from memory storage budget, allowing structure-preserving compression
  • Accepted to ACL 2026 Findings, indicating peer-reviewed quality

Methodology

StructKV addresses the fundamental limitation of existing KV cache compression methods that rely on single-layer saliency snapshots. It works in three stages: (1) Global In-Degree Centrality computes how often each token is attended to across all layers, identifying tokens that serve as information hubs throughout the network; (2) Dynamic Pivot Detection uses information-theoretic measures to identify which layer provides the most informative compression signal; (3) Structural Propagation and Decoupling preserves the identified hub tokens in memory while allowing non-hub tokens to be compressed differently for computation vs. storage budgets.

Results & Highlights

  • Effective preservation of long-range dependencies compared to single-layer saliency methods
  • Robust retrieval performance on RULER benchmark (needle-in-a-haystack style evaluation)
  • Maintains performance on LongBench (multi-task long-context understanding)
  • The global information hub identification captures tokens that appear dormant at individual layers but serve critical cross-layer roles
  • Decoupling computation and memory budgets allows more aggressive compression without losing structural information

Critical Assessment

StructKV addresses a genuine limitation of existing KV cache compression methods -- the reliance on single-layer saliency snapshots that can miss globally important tokens. The Global In-Degree Centrality idea is intuitive and well-motivated: aggregating attention patterns across layers naturally captures tokens that serve as information hubs throughout the network. The Dynamic Pivot Detection adds adaptivity that fixed-layer methods lack. However, the computational overhead of computing global centrality across all layers during inference is not thoroughly analyzed -- this is critical for practical deployment. The paper would benefit from ablation studies isolating the contribution of each component more cleanly. The LongBench and RULER evaluation is appropriate but could be supplemented with analysis on even longer contexts (1M+ tokens) where KV cache compression matters most.
