ReDAct: Uncertainty-Aware Deferral for LLM Agents

Dzianis Piatrashyn, Nikita Kotelevskii, Kirill Grishchenkov, Nikita Glazkov, Ivan Nasonov, Ilya Makarov, Timothy Baldwin, Preslav Nakov, Roman Vashurin, Maxim Panov

ReDAct: Uncertainty-Aware Deferral for LLM Agents

一句话总结： 提出ReDAct框架，利用小模型的不确定性信号在序列决策中将约15%的关键步骤延迟到大模型，以低成本达到大模型全程使用的性能。
TL;DR: ReDAct defers only ~15% of action decisions from a small LLM to a large LLM based on predictive uncertainty, matching the large model's full performance at a fraction of the cost in sequential embodied environments.

核心贡献

  • 提出ReDAct（Reason-Defer-Act）框架，在LLM代理的序列决策中基于不确定性实现从小模型到大模型的延迟
  • 系统比较了多种不确定性量化方法（信息论、语义熵、语言化置信度），发现基于token概率的信息论方法最适合作为延迟信号
  • 在ALFWorld和MiniGrid文本具身环境中验证，延迟约15%的决策即可匹配大模型全程使用的性能
  • 在ReAct框架基础上自然集成延迟机制，分离推理和行动生成以便精确评估行动的不确定性

Key Contributions

  • Proposes ReDAct, a framework for LLM agents that defers individual action decisions from a small to a large model based on predictive uncertainty
  • Systematically compares multiple UQ methods (Sequence Probability, Perplexity, Mean Token Entropy, Semantic Entropy, Verbalized Confidence) for deferral
  • Demonstrates that ~15% deferral to large models matches full large-model performance on ALFWorld and MiniGrid
  • Integrates deferral naturally into the ReAct framework by separating reasoning and action generation

Methodology

ReDAct extends the ReAct framework with deferral capabilities. At each step, the small model generates a reasoning trace and proposes an action. The uncertainty score is computed at the action-selection stage using token-level probability metrics (Sequence Probability, Perplexity, Mean Token Entropy). If the uncertainty exceeds a calibrated threshold, the action decision is deferred to the large model, which generates its own reasoning from scratch. The framework is evaluated on ALFWorld (compositional household tasks) and MiniGrid (navigation under partial observability) using frontier-size LLMs (70B to 480B parameters).

Results & Highlights

  • Deferring ~15% of decisions to the large model matches its full performance
  • Information-theoretic UQ measures (SP, PPL, MTE) provide the most informative deferral signals
  • Semantic entropy methods are computationally expensive and less practical for step-level deferral
  • ReDAct significantly reduces inference costs while maintaining task success rates

Critical Assessment

The paper addresses a practically important problem: how to balance cost and reliability in sequential LLM agent settings. The key insight -- that uncertainty-based deferral works well with information-theoretic measures and requires deferring only ~15% of steps -- is valuable. However, the evaluation is limited to text-based embodied environments (ALFWorld, MiniGrid), which are relatively constrained compared to real-world agentic applications. The threshold calibration mechanism is not thoroughly analyzed, and the sensitivity of results to the choice of small/large model pair is not explored deeply.

Strengths

  • Practically motivated: addresses real cost-reliability tradeoff in LLM agents
  • Clean integration with the widely-used ReAct framework
  • Systematic comparison of multiple uncertainty quantification methods
  • Demonstrates significant cost savings with minimal performance degradation

Limitations

  • Evaluation limited to text-based embodied environments; generalizability to broader agentic tasks unclear
  • Threshold calibration details and sensitivity analysis are insufficient
  • Does not explore the interaction between deferral and long-horizon error propagation
  • Limited analysis of which types of actions are most commonly deferred

Relevance

本文研究了基于不确定性量化的推理延迟机制，展示了如何利用模型内部概率信号来评估输出可靠性。对于研究LLM推理阶段的不确定性评估和输出一致性具有参考价值。

Questions for Further Exploration

  • How does deferral interact with error propagation in longer episodes with more steps?
  • Can adaptive thresholds that change across the episode further improve cost-effectiveness?
  • How well does this approach transfer to multimodal agentic settings (vision-language)?
