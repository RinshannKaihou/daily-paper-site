# Fine-grained Approaches for Confidence Calibration of LLMs in Automated Code Revision

> **一句话总结：** 针对自动化代码修订任务中LLM置信度校准不佳的问题，提出细粒度局部Platt缩放方法，在3个任务和14个模型上持续降低校准误差，优于传统的全局序列级方法。

> **TL;DR:** Proposes fine-grained local Platt-scaling for LLM confidence calibration in automated code revision tasks, consistently reducing calibration error across 3 tasks and 14 models compared to coarse-grained global approaches.

## 核心贡献

- 系统分析了传统全局序列级Platt缩放在自动化代码修订（ACR）任务中的不足，揭示其粗粒度特性无法捕捉局部编辑决策的校准需求
- 提出三种细粒度置信度分数的局部Platt缩放方法，分别针对代码修订中不同粒度的正确性判断
- 在3个独立的ACR任务（程序修复、漏洞修复、代码优化）和14个不同规模的模型上验证方法有效性
- 发现细粒度置信度分数结合全局Platt缩放可进一步放大校准改进效果

## Key Contributions

- Identifies that coarse-grained global Platt-scaling of sequence-level confidence scores is ill-suited for ACR tasks where correctness is determined by local edit decisions
- Proposes local Platt-scaling applied to three different fine-grained confidence scores tailored for code revision
- Demonstrates consistent calibration improvement across 3 tasks (program repair, vulnerability repair, code refinement) and 14 models of various sizes
- Shows that combining fine-grained scores with global Platt-scaling amplifies calibration gains

## Methodology

The authors propose replacing global Platt-scaling with local Platt-scaling applied to three fine-grained confidence scores: (1) edit-level confidence focusing on the specific tokens changed, (2) token-level confidence for individual edit decisions, and (3) hybrid scores combining both. Each score is calibrated independently using Platt scaling on a held-out validation set. The approach is evaluated using Expected Calibration Error (ECE) and related metrics across probability intervals, on three ACR tasks (program repair, vulnerability repair, code refinement) with 14 models spanning different sizes and families.

## Results & Highlights

- Fine-grained confidence scores consistently achieve lower calibration error across a broader range of probability intervals
- Effect is amplified when global Platt-scaling is additionally applied on top of local calibration
- Evaluated across 14 different models of various sizes and 3 distinct ACR tasks
- The improvement is consistent across all model-task combinations, suggesting the approach is robust
- Conventional global Platt-scaling of sequence-level scores is unreliable for ACR tasks due to sample-dependent miscalibration

## Critical Assessment

The paper addresses a practical and important problem -- providing reliable confidence scores for LLM-generated code revisions. The core insight that code revision correctness is determined by local edit decisions (rather than overall sequence quality) is sound and motivates the fine-grained approach well. The evaluation across 14 models and 3 tasks is thorough. However, the approach is essentially an application of established calibration techniques (Platt scaling) at a different granularity -- the novelty is in the problem formulation rather than the methodology. The paper does not explore more expressive calibration methods (e.g., temperature scaling with binning, isotonic regression) at the fine-grained level. The practical impact depends on how well-calibrated confidence scores change developer behavior, which is not studied.

## Strengths

- Thorough evaluation across 14 models and 3 distinct code revision tasks
- Clear and practical problem formulation (code revision correctness is local, not global)
- Consistent improvements suggest the approach is robust and generalizable
- Actionable for practitioners deploying LLMs in code revision tools

## Limitations

- Methodological novelty is limited (applying Platt scaling at finer granularity)
- Does not explore more expressive calibration methods at the fine-grained level
- No user study evaluating whether better calibration actually improves developer decision-making
- Focus is exclusively on code revision; generalizability to other structured generation tasks is not discussed
