Towards Explainability of SLMs by investigating Token Level Activation

Relevance
- 与 Transformer 可解释性研究直接相关: 为理解 BERT 类模型的内部表征机制提供了新的量化工具 - 对 NLP 从业者有实用价值: 提供了比 attention weights 更语义化的 token 重要性评估方式 - 与 mechanistic interpretability 领域的趋势一致: 从宏观注意力分析向微观激活分析转变，与当前研究社区对模型内部机制的关注方向吻合 - 与 BERTology 研究传统连接: 延续了 Jawahar et al. (2019) 等关于 BERT 层级功能的分析传统，提供了新的分析角度
Methodology
### 模型与工具 - 模型: BERT-Base Uncased (12 layers, 768 hidden units, 12 attention heads, 110M+ parameters) - 实现: Hugging Face Transformers library (Python)，设置 output_hidden_states=True 获取中间层表示 ### 激活提取流程 (Activation Extraction) 1. Tokenization: 使用 bert-base-uncased tokenizer 分词，应用 padding 和 truncation，转换为 PyTorch tensors 2. Forward Pass: 在 evaluation mode (model.eval()) 下前向传播，获取全部 12 层隐藏状态 3. Layer Isolation: 提取 Layer 8 的隐藏状态张量，形状为 (sequence_length, 768) ### Token Activation Strength (TAS) 计算 核心公式： $$S_i = \|\mathbf{h}_i\|_2$$ 将每个 token 的 768 维隐藏状态向量通过 L2 范式降为单一标量值，作为 Layer 8 处的表征优先级度量。 ### Activation Bucket 阈值划分 对每个包含 n 个 token 的句子，基于上四分位数阈值将 token 划分为： - HIGH-activation bucket: 激活强度超过上四分位数边界的 token - LOW-activation bucket: 激活强度低于阈值的 token ### 分析框架特性 - 轻量级
Results & Highlights
- 语义内容词主导 HIGH-activation bucket: 实验一致表明，语义上有意义的内容词占据高激活桶，主导表征激活变化 - 结构辅助 token 贡献较小: 标点符号等功能性 token 的激活贡献相对较小 - Layer 8 的语义整合功能: 该层在 BERT 的信息处理流程中扮演结构信息与语义信息过渡和整合的关键角色 - 相比注意力分析的改进: AFN 方法能有效避免注意力权重被无语义意义的 token (如标点) 主导的问题 ### 关键实验观察 - 隐藏状态范数 (hidden-state norms) 能直接揭示模型的表征优先级 - 提供了一种比 probing accuracy 和 attention weights 更轻量、更语义化的可解释性信号
