# Measuring Cross-Modal Synergy: A Benchmark for VLM Explainability

## 一句话总结

本文揭示了现有 VLM 可解释性评估中因跨模态冗余导致的评估崩塌问题，提出了基于 Shapley 交互指数的 Synergistic Faithfulness 指标来严格量化跨模态协同效应，并在 3 个 VLM 架构和 3 个基准数据集上系统评估了 8 种 XAI 方法，发现注意力机制方法（AttnLRP、Rollout）显著优于 VLM 原生解释器（LLaVA-CAM、TAM）。

## TL;DR

Vision-Language Models often rely on language priors rather than truly using visual information, causing unimodal explanation metrics to produce contradictory rankings (Kendall's tau = -0.06). The authors introduce Synergistic Faithfulness (F_syn), a metric based on the Shapley Interaction Index that isolates the joint Harsanyi dividend between modalities, achieving rho = 0.92 correlation with ground truth at 24x speedup (completing evaluation in ~11 seconds). Evaluating 8 XAI methods across 3 VLM architectures (LLaVA-1.5, Qwen2.5-VL, InternVL-3.5) and 3 benchmark datasets (CVBench, MMStar, RePOPE), the study uses a Linear Mixed-effects Model to statistically isolate explainer performance. Results reveal that attention-based methods (AttnLRP: beta = +0.029, Rollout: beta = +0.028) significantly outperform VLM-native explainers (LLaVA-CAM: beta = +0.008, TAM: beta = +0.008), which over-index on visual salience at the expense of true cross-modal reasoning.

## 核心贡献

1. **揭示评估崩塌现象**: 证明由于多模态数据集中的语言先验和模态偏差，VLM 经常表现出跨模态冗余（即仅用文本就能回答视觉问题），导致视觉和文本的 XAI 排名相互矛盾（Kendall's tau = -0.06）。完美解释器在存在冗余的实例上反而被惩罚。
2. **提出 Synergistic Faithfulness 指标 (F_syn)**: 基于合作博弈论中的 Shapley 交互指数，通过连续扰动轨迹近似计算模态间的联合 Harsanyi 股息，将 O(2^(m+n)) 的精确 Shapley 计算降为 O(K) 次前向传播，保持 rho = 0.92 的高相关性，实现 24 倍加速（约 11 秒完成评估）。
3. **系统性基准评估与 LMM 统计分析**: 在 3 个 VLM 架构、3 个基准数据集上评估 8 种 XAI 方法，使用线性混合效应模型（LMM）将解释器作为固定效应、VLM 架构/数据集/实例作为随机效应，严格隔离算法性能。发现注意力机制方法（AttnLRP: beta = +0.029, Rollout: beta = +0.028）显著优于 VLM 原生解释器（LLaVA-CAM: beta = +0.008, TAM: beta = +0.008）。

## Key Contributions

1. **Identifies evaluation collapse in VLM explainability**: Proves that language priors in multimodal datasets cause cross-modal redundancy, where VLMs can answer visual queries using text alone. This makes unimodal deletion/insertion metrics unable to distinguish faithful explainers from random ones, yielding contradictory rankings (Kendall's tau = -0.06).
2. **Introduces Synergistic Faithfulness (F_syn)**: A scalable metric rooted in cooperative game theory (Shapley Interaction Index) that computes the 2-way Harsanyi dividend between visual and textual features across continuous perturbation trajectories, reducing O(2^(m+n)) combinatorial complexity to O(K) forward passes while maintaining rho = 0.92 correlation with exact SII, achieving 24x empirical speedup (~11 seconds per evaluation).
3. **Comprehensive benchmark with rigorous statistics**: Evaluates 8 XAI methods across 3 VLM architectures (LLaVA-1.5, Qwen2.5-VL, InternVL-3.5) and 3 benchmarks (CVBench, MMStar, RePOPE), using a Linear Mixed-effects Model to isolate algorithmic performance from dataset/model variance, revealing that attention-based methods (AttnLRP: beta = +0.029, Rollout: beta = +0.028) significantly outperform VLM-native explainers (LLaVA-CAM: beta = +0.008, TAM: beta = +0.008).

## Methodology

### Problem Formulation

The paper formalizes VLM inference as a cooperative game where players are input features (visual patches and text tokens). Standard unimodal faithfulness metrics (Deletion AUC, Insertion AUC, SRG) evaluate each modality independently:

- Visual Deletion AUC: integral of model confidence as visual patches are progressively removed
- Visual Insertion AUC: integral of confidence as patches are progressively added
- Text metrics are defined symmetrically

Under cross-modal redundancy (f(I\I_k, T) = 1.0 for all k), the deletion metric assigns worst score to a perfect explainer, and insertion metric assigns perfect score to any explainer -- an evaluation paradox.

### Synergistic Faithfulness Metric

F_syn explicitly computes the 2-way Harsanyi dividend between visual and textual features. Rather than exhaustive enumeration, it approximates continuous perturbation trajectories via Riemann sums over K discrete steps, reducing O(2^(m+n)) exact Shapley complexity to O(K) forward passes. The metric evaluates how masking both visual and textual features simultaneously (at corresponding importance ranks) affects model confidence, isolating the synergistic interaction that neither modality alone provides.

### Experimental Setup

- **VLM Architectures**: LLaVA-1.5, Qwen2.5-VL, InternVL-3.5
- **Benchmark Datasets**: CVBench, MMStar, RePOPE
- **8 XAI Methods**: Random (baseline), GradCAM, Input Gradients, Integrated Gradients, Rollout, Grad x Rollout, AttnLRP, LLaVA-CAM, TAM
- **Statistical Analysis**: Linear Mixed-effects Model (LMM) with Explainer as fixed effect, VLM architecture/dataset/instance as random effects; pairwise Wilcoxon signed-rank tests with Bonferroni correction

### Key Findings

- **AttnLRP achieves highest LMM coefficient** (beta = +0.029, p < 0.001), statistically significantly outperforming all other methods.
- **Rollout is a close second** (beta = +0.028), also highly significant.
- **VLM-native methods barely beat random**: LLaVA-CAM and TAM both achieve only beta = +0.008, marginally above the random baseline.
- **F_syn validity**: Achieves rho = 0.92 Spearman correlation with exact Shapley Interaction Index, validating it as a faithful surrogate.
- **Efficiency**: Average evaluation completes in ~11 seconds (24x speedup over exact computation).

## Critical Assessment

### Strengths

- **Fundamental problem identification**: The evaluation collapse phenomenon is rigorously proven both theoretically (boundary conditions under redundancy) and empirically (Kendall's tau = -0.06).
- **Principled metric design**: F_syn is grounded in well-established cooperative game theory, not ad-hoc heuristics.
- **Strong statistical rigor**: The LMM framework with pairwise significance testing provides far more trustworthy conclusions than simple leaderboard averaging.
- **Counterintuitive and impactful finding**: Challenging the assumption that VLM-native explainers are superior is valuable for the community -- these methods produce visually plausible but mathematically unfaithful explanations.
- **Practical efficiency**: 24x speedup makes large-scale benchmarking feasible.

### Limitations

- **Restricted to VQA format and open-weight models**: The empirical validation covers only Visual Question Answering tasks and mid-scale, open-weight VLMs. Proprietary frontier models (GPT-4V, Gemini) could not be tested as their black-box APIs do not expose logits required by explainers.
- **Computational overhead**: Despite 24x speedup, F_syn remains more costly than traditional unimodal metrics and may not suit hard real-time scenarios.
- **Two-modality assumption**: The metric is designed for vision-language pairs; extension to three or more modalities (e.g., audio-visual-language) is unexplored.
- **The authors acknowledge but do not fully address** whether F_syn could itself be gamed by adversarial explanation methods.
- **Riemann sum approximation**: The discrete K-step approximation of continuous perturbation trajectories may introduce approximation error for very small K values.

## Relevance

This work is highly relevant to researchers working on:
- **VLM interpretability and safety auditing** -- provides a rigorous framework for evaluating whether explanations capture genuine cross-modal reasoning vs. single-modality shortcuts.
- **Multimodal XAI method development** -- challenges developers of VLM-native explainers to optimize for cross-modal faithfulness, not just visual plausibility.
- **Benchmark development** -- establishes a new evaluation paradigm that resolves the fundamental contradiction in unimodal metrics.
- **AI safety and trustworthiness** -- directly addresses the risk of automation bias where plausible-looking but unfaithful explanations could lead to unsafe deployment in high-stakes domains.

## Questions for Further Exploration

1. How does F_syn extend to VLMs with more than two modalities (e.g., vision-language-audio), and does the O(K) approximation still hold?
2. Can the evaluation collapse insight be used to improve VLM training -- for instance, by adding regularization terms that penalize cross-modal redundancy?
3. Is there a correlation between cross-modal synergy scores (F_syn) and downstream task performance (accuracy, robustness)?
4. Could F_syn be used as a training signal or loss component to develop inherently more interpretable VLMs that genuinely fuse modalities?
5. How do proprietary frontier models (GPT-4V, Gemini) behave under this benchmark if access to internal logits/gradients were available?
6. What is the sensitivity of F_syn to the number of discrete steps K in the Riemann approximation, and is there an optimal K for different model scales?
