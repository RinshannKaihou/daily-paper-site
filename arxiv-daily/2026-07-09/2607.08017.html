01

Methodology

Five-stage pipeline turns raw CoT text into causal DAGs, then into a single bounded coherence score.

GRAPHEVAL pipeline

GRAPHEVAL converts sampled Chain-of-Thought paths into analyzable reasoning topologies. Each path becomes a causally-annotated DAG; pairwise distances collapse into one distributional UQ score and a medoid answer is selected by graph consensus rather than majority vote.

Architecturally independent components avoid self-evaluation bias: GPT-4o decomposes, Qwen3-Embedding-0.6B embeds. Correlations use 1,000 bootstrap resamples.

Five stages

  1. Causal DAG construction. Generator emits N=20 CoTs at T=0.7, each [3–6] causally-annotated steps; a distinct Decomposer LLM re-segments them into atomic Subject–Verb–Object facts.
  2. Semantic-Structural GED (SS-GED). All C(N,2) DAG pairs compared via Hungarian matching in O(m³); fair-normalized + terminal-node semantic blend β=0.4, γ=5.0 penalizes "same topology, different conclusion."
  3. GRCS. Aggregates pairwise distances into GRCS = 1/(1+μ(D(q))); mode collapse & confident hallucinations both lower it.
  4. Graph Self-Consistency (GSC). Medoid z* = argmax[α·centrality + (1−α)·consensus], α=0.8, filters unfaithful lucky guesses.
  5. Adversarial medoid ablation. Prompt-poisoning forbids the medoid path; proves it is "load-bearing" when accuracy gains couple with faithfulness drops.

03

Strengths & Limitations

Genuine conceptual novelty — but flagship signal concentrates on smaller models.

Strengths

  • Reframes UQ from answer-level to holistic reasoning-fidelity — useful for mission-critical (medical/legal) deployment.
  • SS-GED's semantic-structural blend is a concrete fix for Topo-UQ's blind spot to "same topology, different conclusion."
  • Decoding analysis (GSC vs SC + faithful/spurious audit) demonstrates practical consequence, not just correlations.
  • Strong statistical reporting: effect sizes, p-values, bootstrap CIs, κ = 0.913.
  • Honest disclosure of where the method breaks (large-capable models, long-horizon).

Limitations

  • O(N²·m³) per question vs SC's O(N) — non-trivial inference cost.
  • Diminishing signal on capable models — GRCS correlation & ablation effect both weaken on DeepSeek R1.
  • Short-horizon only — CoTs capped at 6 steps; agentic / multi-thousand-token scalability unaddressed.
  • Small, single-seed slices for MedQA (100 q, seed 42) + closed DLLM/judge reduce reproducibility in the most safety-relevant domain.
  • Faithfulness proxy depends on Early Answering (Counterfactual Editing only in appendix).
04

Open questions

Where the framework must be stress-tested next.
  1. How does GRCS / GSC behave on genuinely long-horizon reasoning (agentic trajectories, multi-thousand-token CoTs from o1-style models) where n_h ≫ 6?
  2. Can the medoid ablation result be replicated on a frontier dense model (≥100B), or is "load-bearing path" an artifact of distilled / small-model mode collapse?
  3. Is Early Answering a robust enough faithfulness oracle — how do GRCS correlations shift under Counterfactual Editing or human-graded ground truth?
  4. What is the compute cost-benefit frontier of GSC vs SC: at what N does medoid selection stop paying off vs simply running more SC samples?
  5. Could SS-GED be approximated cheaply (embedding-only or learned graph kernels) to bring O(N²·m³) into an inference-time regime?
  6. Does GSC's faithfulness advantage transfer downstream (tool use, code generation) where "correct answer via wrong reasoning" is especially costly?
