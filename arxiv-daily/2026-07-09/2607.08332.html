01

Methodology — a closed-loop research agent

RMA

Report-to-Memory Absorption

External reports chunked through a 3-layer A/B/C taxonomy. A-layer = OHLCV eligibility KEEP/DROP (drops fundamentals, order book, macro, sentiment). B = mechanism family. C = 48 Research Archetypes as cues, not formulas.

MACRO

Planning Brain

Per-cycle theme routing: fixed / coarse-guided / memory-driven (default after 5 warm-up cycles). Picks 1 primary + 3–4 supporting B-layers, narrows C-archetypes, assembles an archetype-guided hypothesis pool.

MICRO

Code Evolution Brain

Hypotheses → executable Python on OHLCV. AST gate, ex-ante TriAlign-Judge (idea / code / financial-plausibility), numerical validation, leakage unit tests. Then mutation / crossover / refinement evolution.

SELECT

Multi-Gate Selection

Winsorized RankIC/IC with rolling OOS windows and a complexity discount. Three tiers: normal parent pool, elite archive, cycle-final library (admission score > 0.75).

CROSS

Feedback Brain

Factor-to-Archetype Attribution (FAA) re-attributes elite factors post-mutation. GOOD/BAD feedback written to archetype / generation / cycle memory, steering the next cycle's themes.

Strengths

  • Clean closed-loop formulation that maps onto how human quants actually work — read → hypothesize → implement → validate → reflect.
  • RMA A/B/C taxonomy with explicit OHLCV eligibility gate keeps report knowledge useful without hard-coding claims into formulas.
  • Ex-ante tri-alignment (idea / code / rationale) plus leakage unit tests — correctness safeguards most alpha miners lack.
  • Honest reporting of gates, thresholds, costs, runtime; real diversity evidence (87.1% < 0.3 corr).

Limitations

  • Single dataset, single horizon — CSI300, 10-day only. No CSI500/CSI1000, no cross-market, no multi-horizon.
  • No quantitative ablation of RMA, routing modes, TriAlign, novelty injection, or FAA — only qualitative traces.
  • Portfolio IR of 1.5853 is large enough to warrant skepticism; no regime / capacity / turnover / window-sensitivity analysis.
  • OHLCV-only data contract excludes fundamentals, order book, macro, sentiment — a known crowded factor space.
  • Ingested report corpus and full prompt set not released — memory not exactly reproducible.
03

Open Questions for Further Exploration

  1. How much of the 1.5853 IR survives on CSI500/CSI1000, US markets, or 1/5/20-day horizons — does the RankIC-vs-portfolio inversion persist or flip?
  2. Quantitative contribution of each component (RMA, routing mode, TriAlign, novelty, FAA) under an identical compute budget?
  3. Is the 3.4× IR gap attributable to factor quality, lower redundancy, the capacity-constrained top-50 universe, a favorable 2022–2025 regime, or target alignment?
  4. Since CogAlpha has higher RankIC but weaker portfolios, what does XALPHA's factor combination contribute beyond single-factor quality — and does it hold for a factor-count-matched baseline?
  5. Sensitivity to the RMA A-layer gate: if fundamentals/order-book/sentiment fragments are re-admitted via proxies, does performance improve or just add noise?
  6. Does the memory loop compound (each cycle strictly better) or saturate — and how many cycles until archetype coverage is exhausted?
