> **一句话总结：** 本文提出 SemGrad，首个面向大语言模型自由文本生成的基于梯度的不确定性量化方法，通过计算输出对"语义保留嵌入"的梯度范数来衡量模型信心，在高歧义性问题（多正确答案）数据集上比现有最优方法 SAR 高出 3.27 AUROC 点，且推理速度显著快于采样类方法。

> **TL;DR:** SemGrad is the first gradient-based uncertainty quantification (UQ) method for LLM free-form generation. Instead of computing gradients in parameter space (which breaks down when multiple valid answers exist), it computes the gradient norm of the output log-likelihood with respect to semantics-preserving hidden-state embeddings. A confident LLM should produce stable output distributions under semantically equivalent input perturbations; a large gradient signals uncertainty. A hybrid variant (HybridGrad) that interpolates between semantic and parameter gradients achieves the best average AUROC across 3 models and 3 QA benchmarks while adding only ~0.25 seconds of overhead per example compared to sampling-based methods that add 0.4–1.9 seconds.

## 核心贡献

- **提出 SemGrad**：首个专为自由文本生成设计的梯度式不确定性量化方法，无需多次采样，计算高效。
- **语义保留分数（SPS）**：提出用于量化隐层表征保留输入语义程度的指标，用于识别最优梯度计算位置（"语义保留 Token"$t^*$），并通过实验验证 SPS 与 UQ 性能正相关。
- **引入熵重加权机制**：对输出 Token 按其熵值加权后计算梯度，缓解语言冗余（停用词等无意义 Token）对梯度计算的干扰。
- **提出 HybridGrad**：通过平均 Token 熵自适应地在 SemGrad 和参数梯度（ParaGrad）之间插值，在高/低歧义性场景下均具鲁棒性，平均 AUROC 达 76.59，优于所有 11 个基线方法。
- **发现并验证语义空间梯度的理论合理性**：在存在多正确答案（高 aleatoric 不确定性）时，参数梯度失效，而语义梯度理论上仍然有效。

## Key Contributions

- **SemGrad (Semantic Gradient)**: The first gradient-based UQ method for free-form generation in LLMs, avoiding the high variance and cost of sampling-based methods.
- **Semantic Preservation Score (SPS)**: A data-driven metric identifying which hidden states (layer $\times$ token position) best preserve input semantics across paraphrase variants, enabling principled selection of the embedding for gradient computation.
- **Entropy-based token reweighting**: Down-weights uninformative tokens (low-entropy stopwords) before computing the gradient, improving signal quality without requiring external importance models.
- **HybridGrad**: An adaptive interpolation between SemGrad and parameter-gradient (ParaGrad) based on sequence-level entropy, achieving the best overall AUROC (76.59 avg.) across models and datasets.
- **Theoretical analysis**: Demonstrates why parameter gradients are unreliable under high aleatoric uncertainty and why semantic-space gradients remain valid regardless of the form of the ground-truth distribution.

## Methodology

The core intuition is that a confident LLM should have a stable output distribution under semantically equivalent input perturbations. Formally, the semantic gradient of the log-likelihood with respect to a semantics-preserving embedding $\mathbf{h}_E(\mathbf{x})$ should be near zero at the optimal parameters if the model is well-calibrated.

**Step 1 — Identify semantic-preserving embeddings via SPS.** For each input query, K=5 paraphrases are generated (using DeepSeek API). For each layer $l$ and token position $t$, the Semantic Preservation Score is computed as the difference between within-paraphrase cosine similarity (semantically equivalent inputs pulled together) and across-query similarity (semantically different inputs pushed apart). Empirically, the token position $t^*$ achieving the highest SPS is consistent within a model across datasets. The upper half of layers at $t^*$ are concatenated into $\mathbf{h}_{t^*}^\dagger$.

**Step 2 — Compute entropy-reweighted gradient (SemGrad).** Given the greedy-decoded response $\hat{\mathbf{y}}$, the SemGrad score is:
$$S_{\text{SemGrad}} = \frac{1}{|\mathbf{h}_{t^*}^\dagger|} \left\| \nabla_{\mathbf{h}_{t^*}^\dagger} \sum_{t=1}^T \omega_t \log p(\hat{y}_t | \hat{y}_{<t}, \mathbf{x}; \mathbf{h}_{t^*}^\dagger(\mathbf{x})) \right\|_1$$
where $\omega_t = H(p(y_t|\hat{y}_{<t}, \mathbf{x}))$ is the per-token output entropy used as an importance weight (detached from the computation graph).

**Step 3 — HybridGrad interpolation.** A token-importance-weighted parameter gradient variant (ParaGrad) is also defined. The average per-token entropy $\bar{\omega}$ serves as a proxy for the sharpness of the predictive distribution. HybridGrad interpolates:
$$S_{\text{HybridGrad}} = (1 - e^{-\bar{\omega}}) S_{\text{SemGrad}} + e^{-\bar{\omega}} S_{\text{ParaGrad}}$$
Low entropy (sharp distribution, single ground-truth likely) biases toward ParaGrad; high entropy (many valid answers) biases toward SemGrad.

The method is white-box (requires gradient access) and requires only a single forward-backward pass, scaling as $O(L \cdot T \cdot D)$ in memory — comparable to a single training step rather than K independent forward passes.

## Results & Highlights

- **HybridGrad achieves the best average AUROC of 76.59** across 3 models (Llama3.1-Instruct-8B, Qwen3-Instruct-4B, Mistral-Nemo-Instruct-12B) and 3 datasets (SciQ, TriviaQA, TruthfulQA), outperforming the previous SOTA SAR (75.33) and all 11 baselines.
- **SemGrad is particularly strong on TruthfulQA** (high aleatoric uncertainty, multiple valid answers): it surpasses SAR by +3.27 AUROC and ExGrad by +6.82 AUROC on average, validating the theoretical argument that semantic gradients are more appropriate when the ground-truth distribution is multimodal.
- **Efficiency advantage**: SemGrad adds only ~0.25 s/example overhead versus ~0.37–1.72 s for sampling-based baselines (SAR, Semantic Entropy, Semantic Density) on Llama3.1-8B on a single A100 80GB GPU.
- **SPS-AUROC correlation confirmed**: A strong positive correlation is observed between the SPS of the hidden state used for gradient computation and the resulting UQ performance, validating the SPS-based embedding selection procedure.
- **Ablation findings**: Entropy reweighting consistently improves performance; $l_1$ norm is marginally better than $l_2$; using the Semantic Preserving Token $t^*$ is important and the top half of layers at $t^*$ achieves the most stable cross-dataset performance.
- **Results are robust** to the choice of correctness metric: rankings under BEM and LLM-as-a-Judge are highly consistent (Table 4).

## Critical Assessment

**Novelty:** The idea of using gradients in semantic/representation space rather than parameter space for UQ is a genuine conceptual advance. The SPS metric as a principled way to select which embedding to differentiate with respect to is also novel. The entropy-reweighting for token importance is simpler than the cross-encoder used in SAR and well-motivated.

**Rigor:** The theoretical justification (why parameter gradients fail under high aleatoric uncertainty while semantic gradients remain valid) is clear and mathematically grounded. Experiments span 3 models and 3 datasets with ablations on all key design choices. Results are verified under two correctness metrics (BEM and LLM-as-a-Judge) and two evaluation metrics (AUROC and AURC).

**Reproducibility:** Code is released; implementation uses standard libraries (HuggingFace, PyTorch autograd). Paraphrase generation relies on the DeepSeek API, which introduces a minor external dependency and cost. The slower runtime of SemGrad on SciQ relative to TruthfulQA (due to torch.autograd constraints requiring gradients through all sequence positions, not just $t^*$) is acknowledged and could be a practical limitation at inference time; the authors note targeted engineering could improve this substantially.

**Significance:** UQ for free-form LLM generation is a high-impact problem. The gradient-based, sampling-free approach is practically attractive. The gains on high-aleatoric-uncertainty settings (TruthfulQA) are the most compelling result.

**Red flags / limitations:**
- White-box requirement is a genuine restriction — closed-source APIs (GPT-4, Claude) cannot benefit from this method.
- Experiments target short-form QA (claim-level). Performance on long-form generation is untested and likely degraded, as gradient signals dilute over many tokens.
- SPS computation itself requires paraphrase generation (5 paraphrases per query, using an external API) as a one-time offline setup per dataset-model pair. This adds an offline cost not fully accounted for in the runtime comparison.
- The HybridGrad balancing is sensitive to the entropy threshold; the default setting works well but may require tuning in new domains.
- Evaluation is limited to English QA; generalization to other languages/tasks is not demonstrated.

## Strengths

- Theoretically grounded: explains precisely when and why parameter gradients fail for UQ in free-form generation, providing a principled motivation for the semantic-space approach.
- Sampling-free and computationally efficient: dramatically lower overhead than SOTA sampling-based methods (SAR, Semantic Entropy, Semantic Density).
- Strong empirical results especially in the challenging high-aleatoric-uncertainty regime, which is the most realistic deployment scenario.
- All design choices are validated through ablation; SPS-AUROC correlation provides interpretable evidence for the embedding selection strategy.
- Code is publicly available; implementation details and prompt templates are fully documented in the appendix.

## Limitations

- **White-box only**: requires access to model weights and gradients; inapplicable to proprietary closed-source models.
- **Short-form generation focus**: performance on long-form outputs is an open question; gradient signal dilution over long responses is a plausible failure mode.
- **SPS offline cost**: identifying the semantic-preserving token $t^*$ requires generating paraphrases and running inference, which is an additional preprocessing step not included in the per-example runtime comparison.
- **Implementation overhead from PyTorch autograd constraints**: computing gradients with respect to intermediate hidden states requires backpropagating through the full sequence, inflating computation compared to what would theoretically be needed.
- **Dataset and language scope**: only English QA benchmarks; no demonstration on dialogue, summarization, or code generation.

## 相关性

本文直接针对 LLM 可靠性与幻觉检测的核心问题，通过分析模型内部语义表征的梯度稳定性来量化不确定性，与 LLM 可解释性（哪些层/位置编码语义信息）及可靠性（如何判断模型何时"不知道"）研究高度契合；其无采样、计算高效的特性也使其具备实际部署价值。

## Questions for Further Exploration

1. Can the SPS-guided embedding selection be applied to other interpretability tasks beyond UQ, such as identifying semantic representations for steering or probing?
2. How does SemGrad perform on long-form generation tasks (e.g., summarization, multi-step reasoning)? Can the claim-level decomposition strategy used in factual evaluation pipelines recover its effectiveness?
3. The semantic-preserving token $t^*$ is consistent within a model across datasets — does this consistency hold across domains (code, math, multilingual)? Could it be identified without running SPS on held-out data?
4. Can the entropy-based token reweighting be replaced or augmented with learned importance weights trained end-to-end, and would that improve performance?
5. Is there a meaningful connection between the SPS of a hidden state and the degree to which it is causally responsible for the model's factual recall, as studied in mechanistic interpretability work (e.g., activation patching)?
