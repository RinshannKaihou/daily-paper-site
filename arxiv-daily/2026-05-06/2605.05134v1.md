> **一句话总结：** 本文将LLM输出建模为动力系统，利用Koopman算子理论拟合幻觉与真实响应的嵌入轨迹，通过差分残差分数（ΔE）实现单次推理、无需外部知识的幻觉检测，在HaluEval、WikiBio、FELM三个基准上与或优于SelfCheckGPT等多采样方法。

> **TL;DR:** This paper treats LLM token embedding sequences as trajectories of a nonlinear dynamical system, fits two linear Koopman-based models (one on hallucinated, one on factual training examples), and classifies new responses by comparing their prediction errors under the two models via a differential residual score. The method requires only a single forward pass and no external retrieval, achieving state-of-the-art black-box hallucination detection on three benchmarks.

## 核心贡献

- **动力系统视角下的LLM幻觉检测**：首次将LLM生成过程建模为离散时间非线性动力系统，认为真实响应和幻觉响应的token嵌入轨迹分别演化在不同的高维流形上，可被区分。
- **差分残差分数（ΔE）**：利用Extended Dynamic Mode Decomposition（EDMD）分别拟合幻觉与真实轨迹的Koopman算子近似，以两者预测误差之差定义ΔE作为分类分数，可在句子级或响应级使用。
- **单次推理、零外部依赖**：无需多次采样、无需外部知识库检索，仅需一次前向嵌入即可判断，大幅降低API调用成本和延迟。
- **用户偏好感知校准机制**：通过少量用户标注的示范样本调整分类阈值η，支持对不同严格程度（如容忍轻微不准确 vs. 严格拒绝）的个性化适配。
- **跨嵌入模型的部分泛化**：实验表明，训练和测试使用不同嵌入模型时，方法仍具备一定的跨模型泛化能力，表明幻觉的动力学特征具有一定普遍性。

## Key Contributions

- **Dynamical system framing of LLM outputs**: Models LLM generation as a discrete-time nonlinear dynamical system where factual and hallucinated responses trace distinct manifolds in the token embedding space, enabling geometric separation without access to internal states.
- **Differential residual score (ΔE)**: Two linear Koopman operators are fitted offline—one on factual, one on hallucinated token embedding trajectories—using Extended DMD. At inference time, prediction errors from each model are subtracted to yield a scalar ΔE that drives binary classification.
- **Single-pass, retrieval-free detection**: Unlike SelfCheckGPT (which requires multiple stochastic samples) or RAG-based methods, the approach classifies any new response in one embedding pass with no external knowledge lookup.
- **Preference-aware threshold calibration**: A small set of user-labeled demonstration examples is used to sweep and select the optimal decision threshold η, allowing the detector to be tuned to user-specified tolerance levels for minor vs. major inaccuracies.
- **Partial cross-embedding generalizability**: Koopman matrices fitted on one embedding model show non-trivial transfer to other embedding spaces, suggesting that hallucination dynamics are partially model-agnostic.

## Methodology

The method proceeds in two phases.

**Phase 1 — Offline dynamical system fitting.** For a reference dataset of labeled factual and hallucinated LLM responses, each response is embedded token-by-token using an external embedding model (e.g., Llama-3, Mistral, Qwen3, F2LLM, or Jina-v5-Nano). The resulting token embedding sequence y_1, ..., y_L forms a trajectory. Each observable is then "lifted" to a higher-dimensional space via polynomial feature functions f_lift, producing z_k = [y_k; f_lift(y_k)]. Snapshot matrices X and X+ are built from consecutive lifted pairs. An Extended DMD pseudoinverse step A = X+ X† yields a finite-dimensional Koopman operator approximation. This is done separately for factual trajectories (yielding A_c) and hallucinated trajectories (yielding A_h). To standardize across embedding models, the observable is projected onto the 500 most dominant SVD modes of the fitting data's covariance matrix.

**Phase 2 — Inference-time classification.** A new LLM response is embedded token-by-token. For each token position k, prediction errors from both fitted operators are computed:
- ε_c,k = ||y_{k+1} - [I 0] A_c z_k||  (error under factual model)
- ε_h,k = ||y_{k+1} - [I 0] A_h z_k||  (error under hallucinated model)

The token-level differential score is ΔE_k = ε_h,k - ε_c,k, and the response-level score ΔE aggregates squared errors across all tokens. A hallucination is declared when ΔE < η (i.e., the hallucinated model fits better). The threshold η is tuned offline to maximize F1 or another target metric. The calibration mechanism allows users to shift η based on labeled demonstrations, personalizing tolerance to minor inaccuracies.

**Datasets and embedding models.** Evaluation covers HaluEval (10K summarization pairs), WikiBio (238 biographies, 1,908 sentences with major/minor/accurate labels), and FELM (4,427 sentences across Math, Science, Logic). Five embedding models spanning 0.03B to 8B parameters are tested. For HaluEval, 2,000 samples are used for fitting and 8,000 for testing; dataset sizes for WikiBio and FELM are substantially smaller.

## Results & Highlights

- **HaluEval summarization**: The DS method achieves classification accuracy that matches or exceeds a wide range of LLMs used as zero-shot hallucination detectors, including GPT-4, Claude-2, and ChatGPT, even with the lightweight 30M-parameter Jina-v5 embedding. Llama-3 embedding yields near-perfect F1/balanced accuracy/AUC across all sequence length thresholds (99.3/99.4/99.9 at L≥150).
- **WikiBio sentence-level**: DS with Llama-3 achieves AUC-PR of 85.3 on the NonFact task, outperforming SelfCheckGPT (81.96). On the harder NonFact* task (detecting major inaccuracies within mixed passages), Mistral embedding achieves 54.1 AUC-PR, significantly surpassing SelfCheckGPT's 44.23.
- **FELM**: DS with Llama-3 embedding reaches balanced accuracy of 68.4, the highest among all methods including GPT-4 (67.1). F1 scores are lower than GPT-4, reflecting the difficulty of this reasoning-heavy benchmark.
- **Sequence length effect**: Detection performance improves consistently as minimum token length L increases, confirming the theoretical prediction that Koopman operator estimation error decreases as O(1/√L). Smaller embedding models benefit more from longer sequences.
- **Cross-embedding transfer**: Several off-diagonal entries in the cross-embedding generalization table exceed 50% balanced accuracy by a non-trivial margin (e.g., Qwen3→F2LLM: 63.0% on HaluEval), suggesting partial universality of hallucination manifold dynamics.
- **Calibration**: Histograms of ΔE scores across WikiBio categories (major inaccurate, minor inaccurate, accurate) show clear distributional separation in means, validating the physical interpretation that factual drift magnitude scales with severity.

## Critical Assessment

**Novelty**: The core framing of LLM token embeddings as Koopman operator observables is genuinely novel and conceptually interesting. Treating hallucination as a manifold-switching phenomenon in embedding space provides a principled theoretical lens that is distinct from all existing black-box approaches. However, the practical implementation reduces to fitting two linear least-squares models offline and computing L2 prediction errors at test time—a relatively simple procedure once the framing is established.

**Rigor**: The paper is largely rigorous. The mathematical formulation is clear, the two-phase pipeline is well-specified, and the appendix discusses assumptions honestly (including the fact that stochasticity/temperature is not modeled). The key assumption—that factual and hallucinated trajectories lie on separate manifolds—is supported empirically but not theoretically proven. Cross-embedding generalization results are presented transparently, including cases where transfer fails.

**Reproducibility**: Moderate. The appendix provides embedding extraction details, lifting function design, and dataset splits, which is helpful. However, several practical details are underspecified: the exact polynomial lifting order varies (up to order 4 for HaluEval), the number of SVD modes retained is fixed at 500 without ablation, and the LLM2Vec wrapper used for Llama-3 introduces a bidirectional attention modification whose sensitivity is not studied. Code is not released.

**Significance**: High for the black-box hallucination detection problem specifically. The single-pass, API-cost-efficient design is practically important. The calibration mechanism is a pragmatic contribution. The method's competitive performance against multi-sample baselines using only one inference call is the most impactful result.

**Red flags**: 
- The fitting sets for WikiBio (206 samples) and FELM (746 samples) are small, raising questions about whether the learned Koopman operators are stable or overfit to quirks of those splits. The paper does not report variance across multiple fitting/test splits.
- The claim that the method "does not require output token probabilities or external information" is true at inference time, but the offline fitting phase requires a labeled dataset of hallucinated vs. factual pairs—a non-trivial data requirement that is somewhat understated.
- The HaluEval results (near-perfect accuracy with Llama-3) seem unusually high and may partly reflect dataset characteristics rather than a general capability; the summarization task in HaluEval has been noted in the literature to be relatively easy for detection.
- No error bars, confidence intervals, or statistical significance tests are reported for any result.

## Strengths

- Principled theoretical grounding in Koopman operator theory, connecting LLM behavior to a well-studied framework from control and dynamical systems.
- Single-pass inference with no external retrieval or multiple sampling—directly addresses the cost bottleneck of existing black-box methods.
- Competitive or superior empirical results across three diverse benchmarks against resource-intensive baselines.
- Preference-aware calibration is a practically useful feature for deployment.
- Transparent reporting of cross-embedding generalization, including failure modes.
- Works with any embedding model, including very lightweight ones (30M parameters), making it accessible on constrained hardware.

## Limitations

- Detection without correction: the method flags hallucinations but provides no factual correction or grounding, limiting its utility in downstream pipelines that need corrected outputs.
- Requires a labeled reference dataset (factual vs. hallucinated pairs) for offline fitting—not truly zero-resource; this data requirement is similar in spirit to what some "retrieval-based" methods need.
- Performance degrades for short token sequences (L<50), limiting applicability to brief LLM responses such as short-form QA.
- The Koopman linear approximation assumes a stable manifold structure; adversarial prompts or highly domain-shifted queries may violate this assumption.
- No formal mechanism for hallucination mitigation (prompt modification, decoding-time intervention) is provided or discussed beyond detection.
- Small fitting sets for two of the three benchmarks limit confidence in the generalizability of learned operators.
- No code release; reproducibility relies entirely on the appendix descriptions.

## 相关性

本文直接针对LLM可靠性中的幻觉检测问题，提出了一种无需访问模型内部状态（如logits、注意力图）即可运行的黑盒方法，与LLM可解释性和可靠性研究高度相关。其动力系统视角为理解LLM生成行为（真实 vs. 幻觉）提供了新的几何解释框架，对关注LLM内部表示及输出可信度的研究者具有参考价值。

## Questions for Further Exploration

1. How sensitive is the Koopman operator fitting to the size and quality of the reference hallucination/factual dataset? Is there a minimum dataset size below which performance collapses?
2. Would incorporating stochasticity (e.g., temperature) into the dynamical system formulation—perhaps via stochastic Koopman extensions—improve performance on short sequences or ambiguous outputs?
3. Can the ΔE score be used not just for binary detection but for uncertainty quantification or confidence calibration of LLM outputs more broadly?
4. The paper shows partial cross-embedding transfer. Would fine-tuning or domain adaptation of the Koopman operators improve cross-embedding generalizability to levels competitive with matched-embedding performance?
5. How does this method perform on structured generation tasks (code, JSON, SQL) where hallucination manifests differently from natural language factual drift?
6. Does the method generalize to detecting other reliability failure modes beyond factual hallucination, such as sycophancy, refusal drift, or instruction non-compliance?
