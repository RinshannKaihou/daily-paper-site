2 Why This Matters

A reproducible control layer that turns agent self-reports into code-state-bound, mechanically verifiable evidence gates — a baseline reference for trustworthy agent pipelines, complementary to LLM critics and CI/CD provenance (SLSA / in-toto).

3 Critical Edge

The headline gain is task-concentrated: all of A4's amplification lives in one task (du-duration, 2/75), and per-scenario FDR yields 0/15 significant contrasts — only the aggregate reaches significance.

§ 01

Methodology

M
  • Lifecycle as claim admissibility: a natural-language "tests passed" carries no binding and is never an admissible E_c (Eqs. 2–3).
  • Three git identities bind the tracked source state: materialHash (tracked tree), headHash (commit), storyFilesHash (story-owned files).
  • Admissibility = 7-way predicate: Fresh ∧ Complete ∧ IntegrityVerified ∧ ProducerAuthorized ∧ ExecutionAttested ∧ Supports ∧ OutcomeAccepted. Freshness defeats stale-evidence silent failure.
  • CLI over a "story": init→plan→dev→review→test→done, with signed review runs, per-run test receipts, and a freshness-bound full-test receipt to enter done.
  • High-risk paths add a 3×2 independent-host review floor (Eq. 4) with honest degraded-single-host fallback (Eq. 5) — never a spoofed quorum.
  • Evaluation read from git history + process-event log + an independently-authored acceptance script — never agent narration.
§ 02

Results & Highlights

R
  • primary A4 amplifies 2/1800 vs A2′ 31/1800; +1.6 pp, CI [0.8, 2.5].
  • isolation A3 review-only 14/1800 vs A4 2/1800 at near-equal spend (token ratio 0.98).
  • contract Engine loop 10/10, false-DONE 0; tamper suite rejects all 18 classes (0 false accept / reject).
  • Excluding du-duration: A4 0/1725 vs A2′ 2/1725 — effect driven by one trap.
  • Self-corpus (live 2026-06-23): 565 stories / 1007 findings, 94.8% resolved.
  • Cross-vendor Codex host-2: 68 high/critical findings, all resolved — incl. a critical signature bypass.
  • Recovery pilot (n=15): A2′ amplifies 15/15; A4 safe-stops 15/15; A4b recovers 15/15.
  • Clean-task overhead (no injection): all arms 9/9 — A4/A1 cost 2.07×, wall 2.51×. Value only appears under green-but-wrong faults.
§ 03

Strengths

S
  • Formal, machine-enforced admissibility (Eq. 2) with freshness bound to exact code state — a falsifiable rule, not a slogan.
  • Pre-registered powered ablation: seeded cluster bootstrap (B=2000), matched-cell invariant, CI-excludes-zero endpoint.
  • Unusually honest claim-tier table + a prohibited-wording guard that mechanically blocks over-claims.
  • Self-red-teaming contracts (B-4c65 tamper, B-7b6e gate-strength) — the method applied to itself.
  • Operated at self-hosted scale (565 stories / 1007 findings), not only synthetic tests.
  • Cross-vendor dogfooding caught real defects in the paper's own §5.2 statistics (denominator bias, GLMM contamination, baseline failing open).
§ 04

Limitations

L
  • Single model family (Sonnet); exact provider snapshot not recorded; no external benchmark; 24 tasks of unstated identity.
  • Primary A4-vs-A2′ is not equal-spend (only 207/1920 cells in the ±20% band); A2′ weaker than raw A2 this run.
  • Effect is task-concentrated (all in du-duration); per-scenario FDR 0/15; clean A3-vs-A4 is exploratory.
  • Self-built / self-reviewed corpus is selection-conditioned; 93% smoke_would_miss is a curated-label, not a base rate.
  • Strong cross-vendor quorum, multi-model & external-benchmark are gated as future work; local-key trust does not defeat a compromised runner.
  • Experience-layer causal A/B unpowered (control 1/30); lane-level marginal contributions unestimated.
