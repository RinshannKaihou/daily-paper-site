PAC Learning in Turn-Based Stochastic Games with Reachability Objectives

A Decentralized, Private Approach via Expected Conditional Distance
Ali Asadi · Krishnendu Chatterjee · Pavol Kebis — ISTA
Key Finding

First positive PAC-learning result for turn-based stochastic games — under decentralized algorithms and private state information.

Why This Matters

A 2026 landmark PAC-RL advance: the first polynomial sample-complexity result for decentralized, private-information game learning — foundational for concurrent games, arbitrary-state sampling, and ECD estimation.

01

Methodology

Purely theoretical (formal-methods / learning-theory). Three conceptual layers, deliberately avoiding estimation of transition probabilities — the key reason it extends to private/decentralized settings.

Layer 1 — ECD generalization (Sec. 3)

  • From the MDP ECD parameter of Svoboda–Bansal–Chatterjee (ICML 2024).
  • Five "obvious" game generalizations ECD₁–ECD₅ provably fail on a 4-state example (Fig. 1): each gives ≤ 2, yet the true needed horizon is ≥ 100.
  • Correct definition fixes the inner quantifier over best-responses of Max: ECD_G = supπ_Min infπ_Max ∈ BR(π_Min) ETR_G → 1001, always finite.

Layer 2 — Reduction (Thm. 14)

  • PAC-RL(ECD) reduced to finite-horizon PAC-RL with horizon H = 2(L+1)/ε, tolerance ε/2.
  • Combines (a) every Min strategy has a Max best-response with ETR ≤ L+1; (b) Markov's inequality truncation (Prop. 13); (c) ε/2-optimality in the finite-horizon game.

Layer 3 — Decentralized private algorithm (Sec. 4)

  • LeTuReGa pair (Alg. 1): built on expanded state–step game G′ (Def. 18), backward induction, Median Elimination best-arm ID (Lemma 17), and treating the unexplored set U_q as an auxiliary target to drive exploration.
  • Termination: U_q shrinks each stage; sim ends when U_q stops shrinking for both players. Lemmas 23, 29–32 give correctness.

Other quantitative claims

  • High-probability guarantee (Lem. 29): "good event" E holds with prob ≥ 1−p via Hoeffding + union bound over |S|²L² state-step/action combos.
  • Termination (Lem. 23): at most |S′| = |S|·L stages; each non-terminating stage removes ≥ 1 state-step from U_q.
  • No experiments reported — consistent with CONCUR / PAC-RL venue.
03

Strengths

+

Crisp, self-contained definitions of decentralized / private PAC-RL for TBSGs that future work can build on directly.

+

Memorable 4-state counterexample (Fig. 1) showing why five natural ECD generalizations fail — rare in learning-theory papers.

+

Algorithm avoids transition-probability estimation entirely — exactly why it extends to private-information settings.

+

Polynomial (not exponential) in all parameters, including ECD — far better than the worst-case exponential-in-|S| min-nonzero-prob bound.

+

The Thm. 14 reduction cleanly separates "ECD is the right horizon bound" from "how to learn finite-horizon reachability" — each part independently reusable.

04

Limitations

—

Pure theory, no experiments. No benchmarks, no synthetic games, no implementation.

—

Steep polynomial exponents — ε¹⁰ (ECD case), L⁷ (both). For ε = 0.1, ε¹⁰ = 10⁻¹⁰: qualitatively informative only.

—

ECD must be supplied externally as input L; no data-driven estimation (acknowledged as future work).

—

Simulator must support joint reset from initial µ for both players; sampling from arbitrary states is left open.

—

Two-player zero-sum turn-based only. Concurrent, n-player, and quantitative objectives (mean payoff, discounted sum) are out of scope.

—

Adversarial learning is provably impossible — "both players cooperate to learn" is a forced, strong assumption.

—

Claim that LeTuReGa is "more efficient" than Daskalakis–Golowich–Zhang (COLT 2023) is asserted without a side-by-side complexity comparison.
