Why this matters

↗ A reusable template for frozen-backbone LLM agents that must adapt online under delayed feedback and distribution drift — no retraining, no within-week leakage.

Critical edge

The +0.068 average gain is the sum of four near-equal contributions (~+0.025 each); no error bars, no seeds, single disease — the system works only because all four fire together.

Methodology

  • Backbone. Qwen3-14B-Base, warm-start fine-tuned through 2021-05-31 (PandemicLLM recipe); weights frozen for all streaming evaluation.
  • Inputs xs,t. Hospitalization/case series, policy text, vaccination %, dominant-variant field, regional context — weekly per state.
  • Output. 5-class ordinal trend (substantial ↓ / moderate ↓ / stable / moderate ↑ / substantial ↑).
  • Memory Mt = (Et, Lt, ρt). Hierarchical episodic store (state / HHS-region / national) + predicate-form rules + regime indicator.
  • Retrieval. Top-N (N=8) by regime-aware cosine similarity; entries fill the <MEMORY> slot.
  • Reflection. Each week, LLM writes one-sentence lesson + candidate rule from delayed labels.
  • Distillation. Every K=4 weeks (and on drift) → consolidates recurring reflections into <RULES> with support counts n and confidence c.
  • Drift detector. Fires when dominant-variant field changes OR cross-region mean ordinal error > warm-start mean + τσ·SD (τσ=2).
  • Streaming loop. Predict all 50 states → batch label reveal → reflect → optionally distill → drift check → advance M.

Strengths  /  Limitations

Strengths

  • Conceptually clean — frozen backbone vs. adaptive memory, isolated cleanly.
  • Rigorous streaming protocol with delayed feedback, no within-week leakage, and a recovery-lag metric.
  • Ablation discipline — every component and every memory tier removed and decomposed.
  • Beats CDC ensemble (.325), streaming fine-tuning (.566), and Claude-ICL (.557).
  • Qualitative case study (Florida, BA.5) makes the mechanism legible.
  • Per-state HHS-region heatmap confirms the regional memory tier is doing real work.

Limitations

  • Single dataset — U.S. COVID, 50 states, 81 weeks; no cross-disease, no cross-country.
  • Coarse target — 5-class trend, not case counts, hospitalizations, or Rt.
  • Post-hoc regime labels aligned with biological events the detector also tracks.
  • No statistical significance — no error bars, no seeds, no tests across boundaries.
  • Rule sensitivity — depends on LLM producing usable predicate rules; no human audit.
  • Memory bloat — behavior on streams > 81 weeks is unreported.

Questions for further exploration

  1. Cross-disease transfer. Does the same architecture generalize to influenza, RSV, dengue without retuning?
  2. Continuous targets. Can the agent move from 5-class trend to actual hospitalization / case counts while keeping recovery-lag gains?
  3. Statistical significance. What do error bars look like across seeds of the backbone fine-tune, the reflection LLM, and τσ?
  4. Rule auditing. Are the predicate-form rules clinically meaningful, or LLM-shaped artifacts?
  5. Learned memory scope. Can the fixed HHS-region partition be replaced by a learned clustering of epidemic trajectories?
  6. Backbone robustness. Is the gain robust across Llama / Mistral / smaller Qwen variants?
  7. Head-to-head vs. Forecast Hub teams on the original quantile MAE.
  8. Longer horizons & active forgetting on 3+ year streams with order-of-magnitude larger memory stores.
  9. Non-biological regime shifts (mask mandates, school closures) — does the statistical trigger catch them?
  10. Cost / latency per week (1 forecast + 1 reflection × 50 states + occasional distillation).
