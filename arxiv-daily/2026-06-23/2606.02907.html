After Ridge-regressing out dataset source one-hot, option count, and response length, reasoning-mode probe accuracy at layer 32 drops from 100% to ≈ 33.5% — the random baseline for 3-way classification.

Why This Matters

High probe accuracy in multi-source reasoning studies reflects task format, not computational structure — format deconfounding should become standard practice.

Three independent evidence streams converge on the same null: (1) residualized probes, (2) trace-mode agreement only 42.5% (chance 33.3%), (3) causal steering p = 0.286 vs. random directions.

Critical Edge · Sharpest Caveat

Conclusions rest on one model (Qwen3-14B), one prompt template, and one token position (last input token) — with thinking mode explicitly stripped. Cross-architecture generalization remains entirely unverified, and residualizing source one-hot may erase real signal.

01 Methodology · The format-confound pipeline Qwen3-14B · 40 layers · d=5120 · DISABLE_THINKING=True

Corpus
750 balanced tasks — Deductive (LogiQA 2.0, 4-choice), Inductive (ARC-Challenge, 4-choice), Abductive (αNLI, 2-choice); 250 each.

1× NVIDIA GH200 480GB · bf16 · batch 8 · model 29.5 GB · Qwen3-4B fallback unused.
Probing
L2 logistic (C=1.0) per layer · 5-fold stratified CV · best layer ℓ* = 32 (~80% depth).
Geometry @ ℓ*
TwoNN intrinsic dim · KNN hull contamination · UMAP · SVD local curvature · separation ratios.
Format deconfound
Ridge regression: f = [source one-hot, n_options, |y|] → residual r = h − ĥ, then re-probe.
Trace-mode
Hand-written behavior anchors (deductive / inductive / abductive) embedded in last-layer states; cosine similarity assignment.
Steering + random baseline
Target v̂ = (μₜ−μₛ)/‖·‖, α* via coherence scan + Otsu; N_rand = 20 random unit vectors, same α*; conflict-injection with 10 random pairs.
Statistics
5,000 permutations · 2,000 bootstrap · empirical p = (k+1)/(N+1) · Cohen's d · all hyperparams data-derived.

03 Strengths what survives scrutiny

  • Pinpoints the right methodological sore spot — the hidden assumption behind multi-source probing that no one is testing.
  • Three converging lines of evidence: residual probes, trace-mode agreement, causal steering — mutually reinforcing nulls.
  • Random-direction baseline in steering-vector work — rare, necessary, with N=20 and Laplace-corrected p-value.
  • Statistical rigor: 5,000 permutations, 2,000 bootstrap, Cohen's d, all hyperparams data-derived.
  • Open code + prompt template + fallback model: reproducibility high.
  • Actionable recommendations: report source-prediction alongside mode-prediction · residualize · random-direction controls · format-controlled benchmarks.

04 Limitations where the claim thins

  • Single model: only Qwen3-14B; Llama, Mistral, GPT-4, Claude replication left to future work.
  • Single prompt + single token position (last input token); other positions, other prompts untested.
  • Format confound itself has structure: αNLI is 2-choice vs. LogiQA/ARC 4-choice — controlled-comparison only partially resolves it.
  • Ridge with source one-hot may be over-conservative — could erase real cross-dataset signal; intermediate analysis (only n_options + length) not run.
  • Trace-mode agreement is coarse: hand-written anchors + cosine may miss fine-grained reasoning differences.
  • No thinking-mode experiment — actively stripped; whether true mode-specific structure emerges with thinking is open.
  • Small steering study: n=15 steered tasks, N_rand=20; p=0.286 is weak, "no functional link" is a strong phrasing for that sample size.
Generalization across model families remains the open question.

05 Questions for Further Exploration eight follow-ups, ranked by leverage

Q01Does the same format-confound decomposition reproduce on Llama-3, Mistral, GPT-4, Claude — or is "uniform strategy" a Qwen3 trait?
Q02Can a format-uniform benchmark (D / I / A with matched options, vocab, length) be built — and what does the probe do on it?
Q03Re-run with thinking mode enabled: is the block itself the format confound the paper strips away?
Q04Intermediate residualization (only n_options + length, not source) — quantify how much each format feature contributes.
Q05Replace anchor + cosine with rule-counting (explicit syllogisms, hypothesis elimination, pattern enumeration) for behavior switching.
Q06Scale N_rand and steered-task count to give "no directionality" a harder statistical backbone with full effect-size CIs.
Q07Multimodal extension: do vision-language hidden states show the same modality-task-format confound?
Q08Design implication for self-evolving agents: if the model is strategy-uniform, is "mode-specific patching" a viable repair path?
