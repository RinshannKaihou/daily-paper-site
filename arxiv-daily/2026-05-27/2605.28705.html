Understanding Generalization and Forgetting in In-Context Continual Learning

arXiv: 2605.28705

Date: 2026-05-27

https://arxiv.org/abs/2605.28705

一句话总结： 本文提出了首个上下文持续学习的理论框架，揭示了Transformer在推理阶段通过注意力机制处理多任务序列时，即使没有参数更新也会因任务间干扰而产生系统性偏差和遗忘现象。
TL;DR: This paper formalizes the first theoretical framework for in-context continual learning (ICCL), showing that shared attention mechanisms in Transformers inevitably induce inter-task interference and forgetting during inference—even without parameter updates—through a bias–variance–interference decomposition of prediction error.

---

核心贡献

  • 首次形式化上下文持续学习： 将预训练Transformer在单一提示中处理多个顺序任务的场景定义为"上下文持续学习"，明确桥接了ICL与持续学习的核心概念（任务顺序、历史积累、干扰）。
  • 偏差-方差-干扰分解： 对线性及掩码线性自注意力机制下的预测误差进行了显式分解，量化了任务相似度、上下文长度和任务顺序如何共同决定历史上下文是产生正迁移还是可证明的负迁移。
  • 遗忘的结构性解释： 证明遗忘源于注意力对任务贡献的重新加权（而非信息丢失），即使上下文长度趋于无穷，当任务均值不对齐时遗忘仍然渐近存在。
  • 真实LLM验证： 在GPT-2和Qwen2.5-1.5B-Instruct上验证了理论预测，包括上下文长度效应、任务相似度效应和任务顺序敏感性。

Key Contributions

  • First theoretical formalization of ICCL: Defines in-context continual learning as a pretrained Transformer processing a sequence of tasks concatenated into a single prompt through shared attention without parameter updates, bridging ICL and continual learning.
  • Bias–variance–interference decomposition: Provides explicit error decomposition for linear and masked linear self-attention, characterizing when historical context yields positive transfer vs. provable negative transfer based on task similarity, context length, and task ordering.
  • Structural explanation of forgetting: Shows that forgetting arises from attention reweighting rather than information loss, and persists asymptotically when task means are misaligned regardless of context length.
  • Empirical validation: Validates theoretical predictions on GPT-2 (12-layer and tiny variants) and Qwen2.5-1.5B-Instruct with linear regression, sparse regression, ReLU networks, and real NLP classification tasks (SST-2, AG News).

Methodology

  • Theoretical framework: Models a pretrained Transformer processing T sequential tasks within a single prompt. Analyzes linear self-attention (LSA) and masked linear self-attention (MSA) where the model aggregates key-value statistics across all preceding tasks.
  • Prediction error analysis (Theorem 4.3): Derives closed-form expected squared prediction error for task t, decomposed into: (1) irreducible optimal linear error, (2) variance from finite in-context samples (decays as O(1/M)), and (3) bias from task misalignment.
  • Forgetting analysis (Theorem 4.4): Quantifies the interference error E[(Ŷ_{t,q} − ŷ_{t,q})²] between the task-t prediction and the final prediction after all T tasks, showing it decomposes into intra-task variance terms and inter-task mean interaction terms.
  • Average interference (Corollary 4.6): Provides time-averaged interference across all tasks with explicit coefficients ᾱ_i and ᾱ_{ij} that depend on task ordering.
  • Experimental validation: GPT-2 (12 layers, 8 heads, dim 256) and tiny GPT-2 (3 layers, 2 heads, dim 64) trained on synthetic linear regression following Garg et al. (2022). Real-world validation on Qwen2.5-1.5B-Instruct with SST-2 and AG News classification tasks.

Results & Highlights

  • Context length M: In single-task settings, increasing M consistently reduces error. In multi-task settings, the effect is non-monotonic — small M amplifies interference-dominated bias; large M enables variance reduction. Variance-induced interference decays as O(1/M) but mean-induced interference persists as a constant.
  • Task similarity: High task similarity (small angle θ between weight vectors) yields positive transfer; dissimilar tasks cause negative transfer. This explains why heterogeneous NLP tasks (mapped to nearly orthogonal representations) systematically suffer interference.
  • Task ordering: Forgetting is inherently order-dependent. Earlier tasks experience greater forgetting. The interference coefficients explicitly encode task position, validated across multiple permutations in experiments.
  • Real LLM validation (Qwen2.5-1.5B): Task A (SST-2) accuracy dropped from 0.934 to 0.472 after introducing Task B (AG News) at M=1 — catastrophic forgetting without any parameter updates. Task B suffered 15.6% negative transfer at M=1, recovering at M=19.
  • Non-linear generalization: Theoretical predictions hold for sparse linear regression and two-layer ReLU networks, where Task 5 MSE catastrophically reached 0.9905 at M=19.

Critical Assessment

  • Novelty: High. First theoretical framework bridging ICL and continual learning. The bias–variance–interference decomposition is elegant and provides new insights into well-known empirical phenomena (order sensitivity, long-prompt degradation).
  • Rigor: Strong theoretical analysis with explicit closed-form error expressions. Proofs provided in appendix. However, the core analysis is restricted to linear and masked linear self-attention — a significant simplification from full softmax attention in real Transformers.
  • Reproducibility: Good. Experimental setup follows established protocols (Garg et al., 2022). Model architectures and training details are described. Real LLM experiments use publicly available Qwen2.5-1.5B-Instruct.
  • Significance: Moderate-to-high. Provides fundamental understanding of why long heterogeneous prompts degrade performance, which is practically relevant for prompt engineering and multi-task inference. However, the linear attention setting limits direct applicability to modern LLMs.
  • Red flags: The gap between linear self-attention theory and full softmax multi-head attention is acknowledged but not fully bridged. The experimental validation on real LLMs (Section 5.5) uses only two tasks and one model, which is relatively thin compared to the theoretical depth.

Strengths

  • Elegant and rigorous mathematical framework with clean closed-form decompositions.
  • Comprehensive analysis of multiple interacting factors (context length, task similarity, ordering, training samples).
  • The forgetting mechanism (reweighting vs. information loss) is a genuinely insightful distinction.
  • Progressive validation from synthetic linear tasks through non-linear tasks to real LLMs.
  • The Table 1 summary provides a clear, actionable reference for practitioners.

Limitations

  • Core theory restricted to linear/masked linear self-attention; full softmax attention with multiple heads is not analyzed.
  • The assumption that tasks are defined by linear weight vectors is a significant simplification of real NLP task diversity.
  • Real-world LLM validation (Section 5.5) is limited to one model (Qwen2.5-1.5B) and two classification tasks.
  • Does not propose mitigation strategies — purely analytical. Practitioners need actionable solutions for reducing interference in long prompts.
  • The training distribution (Gaussian inputs, identity covariance) used in experiments may not capture the complexity of natural language distributions.

Relevance

本文对理解大语言模型在多任务长提示下的性能退化具有重要理论价值，为提示工程和上下文持续学习的设计提供了理论基础。对于关注LLM推理时行为、长上下文处理、以及多任务提示策略的研究者尤为相关。
