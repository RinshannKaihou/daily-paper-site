Sign-Aware Gated Sparse Autoencoders: Modeling Anticorrelated Features with Bi-Jump-ReLU Activations

arXiv: 2605.28149

Date: 2026-05-27

https://arxiv.org/abs/2605.28149

一句话总结： 提出符号感知门控稀疏自编码器SA-GSAE，通过Bi-Jump-ReLU激活函数实现双向门控稀疏性和符号幅度路径，使单个潜在变量编码反相关的两极概念；半宽度SA-GSAE（H=16384）在Pythia-1B和SmolLM3-3B的6个hookpoint上以严格Pareto优势或接近持平的表现超越全宽度Gated SAE（2H=32768），同时将死神经元比例降低100-500倍，且参数量更少。
TL;DR: Standard SAEs waste dictionary capacity by forcing separate latents for anticorrelated concepts (e.g., "pressure too high" vs "pressure too low"). SA-GSAE introduces two-sided gated sparsity via a Bi-Jump-ReLU activation that lets one latent encode both polarities along a shared decoder direction, with a signed-magnitude path avoiding L1 shrinkage and an auxiliary reconstruction loss preventing gate collapse. A half-width SA-GSAE (H=16,384) strictly Pareto-dominates a full-width Gated SAE (2H=32,768) on 3/6 cells and matches R² within 0.025 on the rest, while cutting dead-fraction by ~100x-500x on MLP outputs. All authors are from Amazon Web Services.

核心贡献

  • 指出标准SAE的非负性约束（ReLU）导致反相关特征必须分配两个独立潜在变量（+u_j 和 -u_j），浪费字典容量
  • 提出SA-GSAE架构，核心为Bi-Jump-ReLU激活函数：双向可学习死区 [−δ_i^−, δ_i^+] + 符号幅度路径（避免L1收缩）+ 辅助重构损失（防止门崩溃）
  • 参数效率分析：当反相关特征对比例 p > 8/(d_in+4) ≈ 0.39%（对d_in=2048）时，符号感知方案参数更省；实测 p̂ 范围为 0.039–0.767，远超阈值
  • 推荐对称幅度变体（r_i^+ = r_i^-）作为默认配置，因为与完整模型几乎无差异（|ΔR²| = 0.0015）
  • 半宽度SA-GSAE在Pythia-1B和SmolLM3-3B的MLP输出和残差流hookpoint上严格Pareto优于全宽度Gated SAE

  • 定性分析发现：MLP输出层大多数潜在变量携带双极性信号；注意力层双极结构集中于少量top激活潜在变量

Key Contributions

  • Identifies that non-negativity in standard SAEs (ReLU) forces separate latents for anticorrelated features (e.g., +u_j and -u_j), wasting dictionary capacity
  • Proposes SA-GSAE with Bi-Jump-ReLU: a two-sided learnable dead zone [−δ_i^−, δ_i^+] combined with a signed-magnitude path (avoids L1 shrinkage) and auxiliary reconstruction loss (prevents gate collapse)
  • Parameter efficiency analysis: sign-awareness is parameter-efficient when anticorrelated pair fraction p > 8/(d_in+4) ≈ 0.39%; observed p̂ ranges from 0.039 to 0.767, far exceeding the threshold
  • Recommends the symmetric-magnitude variant (r_i^+ = r_i^-) as default — practically indistinguishable from the full model (|ΔR²| = 0.0015, |ΔMSE| = 0.0002)
  • Half-width SA-GSAE (H=16,384) strictly Pareto-dominates full-width Gated SAE (2H=32,768) on both MLP-output hookpoints and resid-mid/Pythia-1B
  • Ablations confirm: auxiliary loss is load-bearing (removing it collapses LR to 0.27, 98% dead); dead-zone initialization δ_0 exhibits U-shaped effect (recommend δ_0 ∈ [0.1, 1.0])
  • Qualitative analysis: MLP-output gains come from most latents carrying both polarities; attention layers concentrate bipolar structure in a small set of top latents (e.g., latent 22641 contrasting "education-reform astroturf" vs "Murdoch-owned media")

Methodology

Architecture: Bi-Jump-ReLU Activation. Each latent i uses a decoder-aligned projection t_i(x) = D_{:,i}^T(x - b_dec) with gate pre-activation π_i(x) = α_i·t_i(x) + β_i. The activation is:

  • Zero inside learnable dead zone [−δ_i^−, δ_i^+]
  • +ReLU(g_i^+ · t_i(x) + b_mag,i) if π_i(x) > δ_i^+ (positive polarity)
  • −ReLU(−g_i^− · t_i(x) + b_mag,i) if π_i(x) < −δ_i^− (negative polarity)

Both polarities share the same decoder column D_{:,i}, enabling bipolar sharing.

Training Objective. Three components:

  1. Main reconstruction: ||x − Da(x) − b_dec||²
  2. Gate sparsity penalty: λ Σ_i Ω_gate(π_i) — two-sided hinge, zero inside dead zone
  3. Auxiliary reconstruction: λ_aux ||x − D_sg·ReLU_signed(π) − b_sg||² — uses stop-gradient decoder to train gate independently from magnitude

Width Reduction. If fraction p of features are anticorrelated pairs, required width drops to H_± = H(1−p/2). Parameter savings when p > 8/(d_in+4).

Evaluation Setup. 3 mid-depth hookpoints (mlp_out, attn, resid) × 2 models (Pythia-1B layer 8/16, SmolLM3-3B layer 18/36) × 3 seeds. Width-32,768 (full) and width-16,384 (half) SAEs trained on OpenWebText activations (50k sequences, length-128). Metrics: MSE, R², L0, dead-fraction, Loss Recovered (LR). Sparsity coefficient λ swept over 8 log-spaced values.

Baselines. Gated SAE (non-negative, widths H and 2H) and AbsTopK SAE (signed, fixed-k selection). All trained in shared harness with matched cache, optimizer, and schedule.

Synthetic Protocols. Protocol A (signed-axis calibration, d=512, k=128 axes), Protocol B (directional anomaly detection), Protocol C (2D antipodal toy model) — all validate the core mechanism under controlled conditions with 16 seeds.

Results & Highlights

  • Pareto dominance (sweep-level): Half-width SA-GSAE strictly Pareto-dominates full-width Gated SAE over the entire swept L0 overlap on 3/6 cells:
  • mlp_out/Pythia-1B: 100% dominance, median 155× dead reduction
  • mlp_out/SmolLM3-3B: 100% dominance, median 60× dead reduction
  • resid/Pythia-1B: 100% dominance, median 191× dead reduction
  • Near-parity on remaining cells: On attention and resid/SmolLM3-3B, ΔR² ∈ [−0.008, −0.001] while dead-fraction still drops by 0.35–0.62 absolute
  • Dead-fraction reduction: Sweep-geomean ~100×–500× on MLP-output and Pythia-1B resid; ~2×–4× on attention cells and SmolLM3-3B resid
  • Peak ratios: Up to 15,000× dead-fraction reduction at L0 ≈ 125 on resid/Pythia-1B
  • Matched-L0=64 point: SA-half achieves R² = 0.741 vs Gated-full R² = 0.711 on Pythia-1B mlp_out, with 0.8% vs 73.7% dead
  • Protocol A (synthetic): SA-GSAE at width H=128 matches Gated SAE at width 2H=256 in reconstruction (MSE 0.009963 vs 0.009956) with near-ideal calibration (γ+ ≈ 1.04, γ− ≈ 1.04 vs ideal 1.0)
  • Ablations on Pythia-1B mlp_out:
  • No auxiliary loss: catastrophic (LR=0.27, 98% dead, L0 clamps at ~2)
  • Symmetric magnitude (r+=r−): indistinguishable (|ΔR²|=0.0015)
  • δ_0=10⁻³: harmful (R²=0.715, 27% dead vs reference R²=0.741, 0.8%)
  • Hybrid AbsTopK+gated magnitude: dominated (R²=0.724, 41% dead)
  • Full-width collapse: Full-width SA-GSAE on resid/SmolLM3-3B shows reproducible reconstruction collapse (MSE up to 6.0, R² to −4.3) that half-width entirely avoids (R² ≥ 0.985)
  • Qualitative latents: Pythia attn latent 22641 is bipolar — positive side fires on "education-reform astroturf" sentences, negative side on "Murdoch-owned media" tokens

Critical Assessment

  • Novelty: The core insight — non-negativity wastes capacity on anticorrelated pairs — is well-motivated and practically important for the SAE interpretability community. The Bi-Jump-ReLU integration into the Gated SAE framework (with no-shrinkage separation of support and magnitude) is a clean, principled extension. The authors correctly credit prior signed-threshold units (Hahnloser 2003, Han 2023) and position their novelty as the integration with gated no-shrinkage training.
  • Rigor: Strong experimental design: 6 cells × 3 seeds, sweep-level Pareto analysis (not just point comparisons), controlled synthetic protocols (A, B, C) with 16 seeds, and thorough ablations. The half-width vs full-width comparison is the right operational question. The paper is admirably transparent about limitations (collapse at full width, LR saturation, warmup asymmetry).
  • Reproducibility: Excellent. Specific numerical gaps reported, code artifacts mentioned, all training details documented. The sweep-based comparison protocol with log-L0 interpolation is well-specified.
  • Significance: Directly useful for anyone training SAEs on LLM activations. The ~100x-500x dead-fraction reduction at matched reconstruction quality is a substantial practical improvement. The parameter-efficiency argument (p > 0.39% threshold) is elegant.
  • Red flags & concerns:
  • Small models only: Pythia-1B and SmolLM3-3B (both hidden dim 2048). No experiments on 7B+ models where SAE interpretability is most used. Generalization uncertain.
  • Only 3 hookpoints per model: No layer sweep. Unclear if benefits hold uniformly across depth.
  • Half-width vs full-width framing: The Pareto comparison (SA-half at H vs Gated-full at 2H) is the most favorable framing for SA-GSAE. The same-width comparison (SA-half vs Gated-half) shows SA-half is dominated on resid/SmolLM3-3B and trades small R² for dead-fraction on attention — less clear-cut.
  • No downstream interpretability evaluation: Only reconstruction metrics. No SAEBench probing, no causal interventions, no automated interpretability scores. The claim that bipolar latents are "more interpretable" rests on a few qualitative examples.
  • Warmup asymmetry: SA-GSAE uses a 40k-80k step threshold warmup that Gated SAE doesn't have. No matched-warmup ablation. This could partially explain the full-width collapse.
  • Missing baselines: No comparison with Switch SAE or Matryoshka SAE, though the authors acknowledge this.
  • Full-width collapse unexplained: The reproducible collapse on resid/SmolLM3-3B is concerning — the authors don't know why it happens and only note the half-width avoids it.

Strengths

  • Clean problem identification with clear practical motivation — non-negativity waste is a real issue in SAE practice
  • Principled integration into the Gated SAE framework, preserving the no-shrinkage property
  • Thorough ablations isolating each component (auxiliary loss, symmetric magnitude, dead-zone init, hybrid variants)
  • Sweep-level Pareto analysis is more informative than single-point comparisons
  • Multi-model evaluation with proper statistical reporting (mean ± SE, 3 seeds)
  • Transparent about limitations, including the full-width collapse and missing evaluation axes
  • Controlled synthetic experiments (Protocols A-C) with 16 seeds validate the mechanism
  • Parameter efficiency argument with explicit threshold (p > 8/(d_in+4)) is elegant and testable
  • Qualitative latent analysis provides concrete examples of bipolar features

Limitations

  • Only tested on small models (Pythia-1B, SmolLM3-3B, both hidden dim 2048); scalability to 7B+ models unknown
  • Only 3 mid-depth hookpoints per model; no layer sweep or depth analysis
  • No downstream interpretability evaluation (no probing, causal interventions, or SAEBench)
  • Half-width vs full-width comparison is the most favorable framing; same-width comparison is less clear on some cells
  • Full-width SA-GSAE exhibits unexplained reproducible collapse on resid/SmolLM3-3B
  • Warmup asymmetry between SA-GSAE and Gated SAE not ablated; could confound the full-width collapse
  • Missing comparisons with Switch SAE and Matryoshka SAE
  • Qualitative latent analysis limited to a few cherry-picked examples (Table 9)
  • No analysis of how bipolar latents affect circuit discovery or feature attribution downstream
  • Only OpenWebText used; domain transfer to code, math, or multilingual settings untested

Relevance

本文直接改进了稀疏自编码器的架构设计，解决了可解释性研究中SAE容量浪费的实际问题。对于使用SAE进行LLM内部特征分析和机制研究的工作具有直接参考价值，特别是在需要高效利用字典容量和处理反相关特征的场景下。

Questions for Further Exploration

  • How does SA-GSAE scale to larger models (7B, 13B, 70B) where SAE interpretability is most commonly applied?
  • Does the warmup asymmetry explain the full-width collapse on resid/SmolLM3-3B? A matched-warmup ablation is critical.
  • Do bipolar latents improve downstream interpretability tasks (circuit discovery, causal interventions, SAEBench probing)?
  • Can sign-awareness be combined with other SAE improvements (TopK gating, JumpReLU, Matryoshka hierarchy)?
  • How do results transfer across layers (full depth sweep) and to instruction-tuned / RLHF models?
  • Is the parameter-efficiency threshold (p > 0.39%) validated on models with different hidden dimensions?
