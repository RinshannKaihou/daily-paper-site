Random Matrix Theory of Early-Stopped Gradient Flow: A Transient BBP Scenario

arXiv: 2604.18450

Date: 2026-04-21

https://arxiv.org/abs/2604.18450

一句话总结： 本文用随机矩阵理论解析刻画了梯度流中早停现象的瞬态BBP相变机制，证明了输入协方差各向异性导致信号特征值在训练中短暂涌现后又被噪声块吸收。
TL;DR: Provides the first analytically solvable random-matrix model for the transient regime in gradient flow where signal is detectable in a finite time window before overfitting. In a linear teacher-student setting with anisotropic input covariance, learning manifests as an isolated eigenvalue separating from the bulk—a transient BBP transition that depends on signal strength and covariance structure.

Key Contributions

  • Derives the full time-dependent bulk spectrum of the symmetrized weight matrix through a 2x2 Dyson equation in a two-block covariance model
  • Identifies three learning regimes: signal never emerges, persists permanently, or emerges transiently before reabsorption
  • Provides explicit outlier condition via rank-two determinant formula for a rank-one teacher
  • Maps complete phase diagrams parameterized by signal strength and covariance anisotropy
  • Validates theory against finite-size simulations with strong agreement

Methodology

Uses random matrix theory (RMT) applied to gradient flow in a linear teacher-student model. The key analytical tool is the Dyson equation for the resolvent, which tracks the spectral evolution of the student weight matrix over training time. The two-block covariance structure (fast vs. slow eigenvalue directions) creates the anisotropy that drives the transient BBP phenomenon.

Results & Highlights

  • Three distinct phases identified: "no learning," "persistent learning," and "transient learning" (emerge-then-vanish)
  • The transient phase provides a minimal solvable mechanism for why early stopping works: signal is only detectable in a finite time window
  • Covariance anisotropy controls the width of the transient learning window
  • Finite-size simulations confirm theoretical predictions across all phases

Critical Assessment

  • Novelty: High — first analytically solvable RMT model for transient learning dynamics
  • Rigor: Strong — rigorous mathematical derivation with explicit formulas
  • Limitations: Linear teacher-student model is a significant simplification; unclear how results transfer to deep networks with nonlinearities
  • Significance: Provides formal grounding for the empirically observed early-stopping phenomenon

Strengths

  • Clean, analytically tractable framework with closed-form results
  • Explicit phase diagrams provide intuitive understanding
  • Validates theory with simulations

Limitations

  • Limited to linear teacher-student setting; deep networks have fundamentally different dynamics
  • Two-block covariance model may not capture real data structure
  • No experimental validation on actual ML training scenarios
