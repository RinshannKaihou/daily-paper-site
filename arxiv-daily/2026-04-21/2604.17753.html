Evolutionary Negative Module Pruning for Better LoRA Merging

arXiv: 2604.17753

Date: 2026-04-21

https://arxiv.org/abs/2604.17753

⚠️ Abstract-only summary

一句话总结： 本文发现在LoRA合并中存在"负模块"——特定LoRA层会固有地降低合并模型的全局性能，提出ENMP进化搜索方法在合并前定位并剪除这些有害模块。
TL;DR: Uncovers "negative modules" in LoRA merging — specific layers that inherently degrade global performance when merged. Proposes ENMP (Evolutionary Negative Module Pruning), a plug-and-play method that uses evolutionary search to identify and prune detrimental modules before merging. Achieves new SOTA across language and vision domains, boosting existing merging algorithms.

核心贡献

  • 发现LoRA合并中的"负模块"现象：某些层会固有地降低合并后模型的性能
  • 提出ENMP（Evolutionary Negative Module Pruning）方法，在合并前定位并排除有害模块
  • 使用进化搜索策略导航离散、不可微的模块选择空间
  • 即插即用设计，持续提升现有合并算法性能

Key Contributions

  • Discovery of negative modules in LoRA merging that degrade global performance
  • ENMP: plug-and-play evolutionary search for optimal module pruning configurations
  • Navigates the discrete, non-differentiable landscape of module selection
  • New SOTA across language and vision domains when combined with existing merging methods

Methodology

Uses evolutionary search to navigate the combinatorial space of module selection. The key insight is that not all LoRA matrices contribute constructively — some are detrimental. ENMP identifies and prunes these before merging.

Results & Highlights

  • Consistently boosts performance of existing merging algorithms
  • New state-of-the-art in both language and vision domains
  • Accepted to ACL 2026 main conference with code available

Authors & Credibility

Team from Zhejiang University. Accepted to ACL 2026 main conference, code available at https://github.com/CaoAnda/ENMP-LoRAMerging.

Critical Assessment

  • Novelty: High — the "negative module" concept challenges the implicit assumption in all prior merging work
  • Rigor: Good — evolutionary search is appropriate for the discrete optimization problem
  • Limitations: Evolutionary search adds computational cost; unclear how many modules are typically negative
  • Significance: Practical impact on multi-task model deployment efficiency

Strengths

  • Identifies and addresses a fundamental assumption in LoRA merging
  • Plug-and-play design works with existing merging algorithms
  • Strong empirical results across language and vision domains

Limitations

  • Evolutionary search adds overhead to the merging pipeline
  • The "negative module" phenomenon lacks deeper theoretical explanation
  • Unclear how the approach scales to very large numbers of LoRA experts

Relevance

关联LLM训练可靠性和模型合并——发现LoRA合并中的负模块现象揭示了参数融合过程中的潜在风险，为安全的多任务部署提供了实用工具。

Questions for Further Exploration

  • Can negative module detection be done analytically rather than via evolutionary search?
