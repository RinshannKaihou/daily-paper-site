River-LLM: Large Language Model Seamless Exit Based on KV Share

arXiv: 2604.18396

Date: 2026-04-21

https://arxiv.org/abs/2604.18396

⚠️ Abstract-only summary

一句话总结： 提出River-LLM，一种无需训练的LLM早退框架，通过KV-Shared Exit River机制解决decoder-only架构中跳层导致的KV Cache缺失问题，实现1.71-2.16倍的实际推理加速。
TL;DR: A training-free early exit framework for LLMs that solves the KV Cache Absence problem in decoder-only architectures. Introduces a lightweight KV-Shared Exit River that naturally generates and preserves missing KV cache during the exit process, eliminating costly recovery operations. Uses state transition similarity within decoder blocks to predict cumulative KV errors and guide exit decisions. Achieves 1.71-2.16x practical speedup on math reasoning and code generation.

核心贡献

  • 提出KV-Shared Exit River机制，解决decoder-only架构中早退的KV Cache缺失问题
  • 无需训练即可实现token级别早退
  • 使用decoder块内状态转移相似度预测累积KV误差，指导精确的退出决策
  • 在数学推理和代码生成任务上实现1.71-2.16倍的实际推理加速

Key Contributions

  • Training-free token-level early exit framework for decoder-only LLMs
  • KV-Shared Exit River mechanism that bridges the gap between theoretical layer reduction and practical wall-clock speedup
  • State transition similarity for predicting cumulative KV errors
  • 1.71-2.16x practical speedup while maintaining generation quality

Methodology

Introduces a lightweight KV-Shared Exit River that allows the backbone's missing KV cache to be naturally generated and preserved during exit. Uses state transition similarity within decoder blocks to predict cumulative KV errors and guide precise exit decisions. No additional training required.

Results & Highlights

  • Achieves 1.71-2.16x practical speedup on math reasoning and code generation
  • Maintains high generation quality
  • Accepted to ACL 2026
