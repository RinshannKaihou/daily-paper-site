Memory QA Judge score — ObsMem vs. 0.575 (A-mem). Wins on 7/8 metrics.
◆ Why This Matters

Embodied assistants fail when they confuse a routine with an anomaly — typed, provenance-gated memory makes long-horizon state recall auditable instead of lossy.

⚠ Critical Edge

The biggest win comes from an LLM evidence selector (Judge drops 0.699→0.435 without it) — the gain may be bought with ~60% more compute, not a better memory architecture.

Methodology

WorldLines instances: xᵢ = (H<cᵢ, qᵢ, S, Eᵢ, yᵢ★, τᵢ) — visible pre-cutoff history, query, ground-truth state, evidence chain, answer, type.

  1. World Grounding — Habitat/HSSD scenes → semantic + executable scene-graph views.
  2. Project Planning — multi-day life threads (routine, meals, health, family…).
  3. Closed-loop Simulation — Director→Actor proposes acts; deterministic executor validates affordances & preconditions.
  4. Carry-forward — daily notes (changed states, preferences, conflicts) seed the next day.

ObsMem: Observer Gate → 4 typed views (Event / State / Belief / Commitment) → query-time evidence selector composes views by intent (not pure similarity).

✓ Strengths

  • Real gap hit: long-term memory × embodied agents × partial observability in one framework.
  • Evidence-linked, cutoff-controlled evaluation — auditable, no future leakage.
  • Four-view typed memory with provenance gate is clean & modular.
  • Honest diagnosis: baselines have high Session@5 but low Event R@5 — names the bottleneck.
  • Ablations isolate Belief / State / Consolidation / Selector + a Hidden-Judge diagnostic.

✗ Limitations

  • Small samples: 310 QA + 21 planning; Judge variance could blur the 0.142 plan gap.
  • Simulation-only: no real-robot logs, no perception noise, no actuation errors.
  • LLM-judge dependency: GPT-4o at 87.5% human agreement leaves noisy gold judgments.
  • Manual scene curation per family — limits benchmark reproducibility at scale.
  • Belief thresholds (λ) untested; structured-trace assumption needs clean entity IDs.

? Questions for Further Exploration

  1. Does the +0.138 Judge gap survive a backbone swap (GPT-4o-class / open-source 70B)?
  2. Is the selector gain recoverable with a deterministic non-LLM ranker (view-aware BM25 + recency)? — isolates architecture from extra reasoning.
  3. How sensitive is Belief Track to the λ / γ thresholds? A sensitivity sweep is overdue.
  4. Can the WorldLines pipeline scale to hundreds of households without manual scene curation?
  5. Do typed views port to a lighter RAG-with-metadata pipeline to cut the 8.82s latency?
