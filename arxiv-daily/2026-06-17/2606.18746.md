> **一句话总结：** 在"无域标签、仅凭轨迹推断"的潜在域 POMDP 设定下,本文证明:当两个域在共享观测瓶颈处要求不相交的近似最优动作时,任何一致近似最优的策略必须在瓶颈时刻维持可由总变差距离下界区分的记忆分布(Theorem 1);进一步,只要记忆对一组探测目标"价值充分",就能从记忆中解码出局部干预动力学 P_σ(·|s,a)(Theorem 2),并在 ForkWorld 网格世界中通过 6 个智能体的对照实验予以验证。

> **TL;DR:** A theoretical paper that asks *what* memory must encode for generalist agents that must infer their (unobserved) domain from trajectory alone. Theorem 1 proves a control-theoretic separation result: under a "shared-bottleneck, disjoint-action-sets" assumption, uniformly near-optimal goal-conditioned policies must induce memory distributions at the bottleneck that are TV-separated by domain, with a quantitative lower bound `TV ≥ [1 − η_σ − η_σ̃]⁺`. Theorem 2 shows that "value-sufficient" memory for auxiliary probe goals implies an approximate decoder of the local interventional kernel (with bound `dF ≤ 2(ε_val + ε_span)`), supporting approximate planning under a planning-stability assumption. Empirical validation in ForkWorld — a T-junction gridworld with hidden action-permutation domains — confirms that memoryless DQN plateaus at 0.19 success, memory-equipped agents reach 0.75–0.88, and that bottleneck representations become near-perfectly domain-separable after the first arm probe, with a transition decoder reducing next-state error by 41–63%.

## 核心贡献

- **Theorem 1(表征分离定理):** 在 Assumption 1(共享瓶颈 + 不相交动作集)下,任何一致价值近似最优的策略在瓶颈状态处对不同域产生的条件记忆分布必须**不同**,且给出具体的总变差下界 `TV(Λ_σ, Λ_σ̃) ≥ [1 − ε_opt/(ργT ε_gap) − ε_opt/(ργT ε_gap)]⁺`。
- **Theorem 2(可解码动力学):** 若记忆对一组辅助探测目标是 `ε_val`-价值充分的,则在积分概率度量 `dF` 下,**(i)** 同记忆 ⇒ 局部干预核相近;(**ii**) 存在显式解码器 `D_P: M × A → P(S)`;(**iii**) 在 Assumption 3(规划稳定性)下,基于解码核的 Bellman 规划与真实 Q⋆ 的误差由 `S(2(ε_val + ε_span))` 控制。
- **关键定位:** 区别于 Richens & Everitt(2024)、Richens et al.(2025)、Ceriscioli & Mohan(2025)等"已知域/任务索引"的世界模型必要性结果,本文**去除域标签**这一显式输入,智能体只能从自身轨迹推断隐藏域——把研究对象从"任务→策略映射"转向"历史→记忆映射"。
- **可验证的实验:** ForkWorld(T 型路口网格世界 + 隐藏的垂直动作置换域)对两条定理给出直接验证:线性探针测得记忆表征在瓶颈处的域可分性、转移解码器测得可恢复性,均与定理预测一致。

## Key Contributions

- **Theorem 1 (Representation Separation):** Under Assumption 1 (shared bottleneck with disjoint, value-gap-separated optimal action sets), a uniformly value-near-optimal goal-conditioned policy must induce *TV-separated* conditional memory distributions at the bottleneck for any domain pair in `D_inc`. Quantitative bound: `TV(Λ_σ, Λ_σ̃) ≥ [1 − η_σ − η_σ̃]⁺` where `η_ξ = ε_opt / (ρ γ^{t_ξ} ε_gap)`.
- **Theorem 2 (Decoder Existence)** in three parts:
  - (1) Same memory state ⇒ local kernels close in integral probability metric: `dF(P^σ(·|s,a), P^σ̃(·|s̃,a)) ≤ 2(ε_val + ε_span)`.
  - (2) An explicit decoder `D_P: M × A → P(S)` exists with the same dF-error bound.
  - (3) Under Assumption 3 (planning stability), Bellman planning with the decoded kernel yields `|Q̂_ψ,D_P(ht,a) − Q⋆(st,a)| ≤ S(2(ε_val + ε_span))`.
- A **novel latent-domain POMDP formulation** in which the agent receives *no domain index* — structure must be implicitly carried in memory.
- Definitions of **control equivalence** (Definition 1), **goal-induced test class** (Definition 2), and **value-sufficient memory** (Definition 3) that operationalize "domain relevance" and "memory carries enough information."
- A controlled gridworld (ForkWorld) that *provably* satisfies the preconditions of both theorems and is used to empirically validate each prediction (memory necessity, representation separation, decodable transition model).

## Methodology

### Formal Setup

- **Latent-domain POMDP** with state space `S`, action space `A`, discount `γ ∈ [0,1)`. Domain `σ ∈ Σ` is sampled at episode start but never revealed.
- For domain `σ`: initial state `S_0 ∼ μ_σ`, interventional transition kernel `P_σ(B | s, a) := Pr(S_{t+1} ∈ B | S_t = s, do(A_t = a))`.
- Goal `ψ ∈ Ψ` specified by bounded per-transition utility `u_ψ: S × S → R`. Domain-dependence enters only through `μ_σ, P_σ`; rewards are domain-invariant.
- Agent observes history `h_t = (S_0, A_0, …, S_t)` and stores memory `M_t = f(h_t) ∈ M`. Policy acts through a shared head: `π(at | ht; ψ) = π_M(at | M_t, ψ)` — therefore *same memory + same goal ⇒ same action distribution*.
- Standard value definitions: `V^π_σ,ψ(s_0)`, `V⋆_σ,ψ(s_0) = sup_π V^π_σ,ψ(s_0)`, `Q⋆_σ,ψ(s, a)`.

### Key Definitions

- **Control equivalence (Def. 1):** `σ, σ̃` are control-equivalent w.r.t. `Ψ` if for every goal `ψ` and state `s` the near-optimal action sets are identical. *Two domains can differ transitionally yet be interchangeable for control.*
- **Goal-induced test class (Def. 2):** `F ⊆ {ϕ: S → [−1,1]}` is `ε_span`-generated by an auxiliary goal family `Ψ_F` if every `ϕ` is approximated (up to `ε_span`) by the optimal Q-value expectation under some `ψ̄_ϕ ∈ Ψ_F` with proxy `W_ϕ`. *Probe goals act as one-step predictive tests.*
- **Value-sufficient memory (Def. 3):** There exists a value head `q_M: M × Ψ_F × A → R` with `q_M(f(ht), ψ̄, a) − Q⋆_σ,ψ̄(st(ht), a) ≤ ε_val` for every reachable `ht`. *Memory retains everything needed to predict optimal auxiliary values.*

### Assumptions

- **Assumption 1 (Shared bottleneck + disjoint action sets):** For each pair `(σ, σ̃) ∈ D_inc` there exist a goal `ψ`, bottleneck state `s†`, times `t_σ, t_σ̃`, and disjoint action sets `G_σ, G_σ̃` such that (1) value-near-optimal policies reach `s†` with probability ≥ `ρ`, and (2) every action in `G_ξ` is at least `ε_gap` better than every action outside.
- **Assumption 2 (Goal-induced local test class):** Exists `F, Ψ_F` such that `F` is `ε_span`-generated by `Ψ_F`.
- **Assumption 3 (Planning stability under `d_F`):** Exists a non-decreasing `S` with `S(0) = 0` such that `d_F`-close local kernels yield `S(·)`-close optimal action-values after Bellman planning.

### Proof Techniques

- **Theorem 1:** Construct a comparison policy `π̂_ν` that switches to a near-optimal action in `G_ξ` at the bottleneck; bound the value gap this induces via the policy-improvement inequality; integrate to obtain a TV lower bound via the variational characterization of total variation. (Full proof in Appendix A.1.)
- **Theorem 2, Part 1:** For two histories with same memory state `m`, value-sufficiency gives nearly equal `Q⋆` under each `ψ̄_ϕ`; use `ε_span`-generation to substitute `W_ϕ` for `ϕ` and bound `dF` directly. (Appendix A.2.)
- **Theorem 2, Part 2:** Define `D_P(m, a) := P_σ(· | s_m, a)` for a chosen representative history per reachable memory state; appeal to Part 1.
- **Theorem 2, Part 3:** Apply Assumption 3 to the Part 2 bound.

### Empirical Setup — ForkWorld

- T-junction gridworld (Figure 1a): corridor of length `c` to bottleneck `s†`, then upper arm to `G_up` and lower arm to `G_down`. Deterministic movement.
- **Hidden domain `σ ∈ {normal, swapped}`** intervenes on vertical action mapping: `normal: up↦+y, down↦−y`; `swapped: up↦−y, down↦+y`. Goal `ψ ∈ {up, down}` is observable; domain is not.
- All four `(ψ, σ)` combinations yield *disjoint* optimal action sets at `s†` — Assumption 1 satisfied by construction.
- A *memoryless* agent cannot exceed 50% on this task (any-arm ceiling), since for each goal one of {up, down} is correct under each domain.

### Agents Compared

| Agent | Memory type |
|-------|-------------|
| Memoryless DQN | None — sees only `[x, y, ψ_id, 1]` |
| Stacked DQN (k=3, +actions) | Bounded window of previous transitions |
| DRQN (states) | GRU over state sequence |
| DRQN (states + actions) | GRU with previous action concatenated |
| Oracle DQN | Memoryless but given `σ_id` (ceiling without memory) |
| Oracle DRQN | GRU + `σ_id` (ceiling with both) |

Training: 5,000 episodes, Adam (`lr=10⁻³`), `γ=0.99`, ε-greedy decay `1.0→0.05` over 3,000 episodes, 8 seeds.

## Results & Highlights

### Theorem 1: Memory is Necessary for Uniform Competence (Figure 1b, Table 1)

| Agent | Overall success | ∆ Oracle |
|-------|------------------|----------|
| Memoryless DQN | **0.19 ± 0.06** | −0.81 |
| Stacked DQN (k=3, +actions) | **0.88 ± 0.08** | −0.12 |
| DRQN (states) | 0.75 ± 0.09 | −0.25 |
| DRQN (states + actions) | 0.78 ± 0.12 | −0.22 |
| Oracle DQN | 1.00 ± 0.00 | — |
| Oracle DRQN | 0.78 ± 0.12 | — |

- The memoryless DQN *underperforms* the trivial 50% any-arm ceiling — *conflicting gradient signals at the bottleneck prevent even domain guessing*, exactly as Theorem 1 predicts.
- Memory-equipped agents close 70–85% of the gap to Oracle DQN within 5,000 episodes.
- Per-condition breakdown: memoryless DQN is essentially 0.12–0.25 across all `(ψ, σ)` pairs; Stacked DQN reaches a uniform 0.88 across all four conditions, demonstrating cross-domain generalization.

### Theorem 1 (Predictions): Representation Separation at the Bottleneck (Figure 2)

- Linear probe (`σ_id ← M_{t†}`) on bottleneck representations, split by *first visit* vs. *subsequent visits*:
  - **First visit:** all agents at chance (0.48 ± 0.00) — corridor provides no vertical-transition evidence, matching the information-theoretic floor.
  - **Subsequent visits:** DRQN (states) 0.97 ± 0.02, Stacked DQN 0.97 ± 0.03, DRQN (+actions) **1.00 ± 0.00**, Memoryless DQN **0.50 ± 0.00** (stuck at chance). Direct empirical confirmation of TV-separation under near-optimality.

### Theorem 2 (Predictions): Decodable Local Dynamics (Table 2)

- Two-layer MLP decoder trained on `(M_t, onehot(a_t)) → s_{t+1}` from greedy rollouts. Baseline = majority-class next-state predictor ignoring `M_t`.
- Fork-state decoder error:

| Agent | Decoder error | Baseline | Reduction |
|-------|---------------|----------|-----------|
| Memoryless DQN | 0.28 ± 0.08 | 0.28 | **0%** |
| Stacked DQN | 0.33 ± 0.01 | 0.47 | 30% |
| DRQN (states) | 0.17 ± 0.03 | 0.46 | **63%** |
| DRQN (+actions) | 0.28 ± 0.04 | 0.48 | 41% |
| Oracle DQN | 0.00 | 0.50 | 100% |
| Oracle DRQN | 0.00 | 0.52 | 100% |

- Memoryless DQN: 0% reduction — `M_t` carries no transition information at `s†`. Memory-equipped agents: 41–63% reduction — `M_t` supports approximate decoding, consistent with Theorem 2.

## Critical Assessment

- **Novelty:** Strong and well-positioned. The paper occupies a clearly identified gap in the world-model-necessity literature (Richens & Everitt 2024, Richens et al. 2025, Ceriscioli & Mohan 2025 all assume a known domain/task index). Removing that assumption is a natural and non-trivial refinement, and the formal apparatus (Assumption 1 + Theorem 1 + Theorem 2) is the right shape for the question.
- **Rigor:** High. Theorem statements are precise, assumptions are explicit and named, proofs are given in full (Appendix A), and the bounds are quantitative with all constants named. The empirical design *provably satisfies* Assumption 1 — this is unusually clean for an experimental paper.
- **Reproducibility:** High. Standard algorithms (DQN, DRQN), standard hyperparameters, 8 seeds, mean ± SE reported. ForkWorld is a tiny environment (4 `(goal, domain)` conditions, deterministic transitions). One caveat: the linear probe and decoder are auxiliary experiments and the paper does not specify code release.
- **Significance:** Medium-to-high for the theoretical RL community; lower for practitioners. The result formalizes an intuition that practitioners already act on ("memory matters when domains have ambiguous observations"), but it now gives a *citation-worthy theorem* with bounds. Particularly useful as a benchmark result for generalist-agent memory research.
- **Red flags / weaknesses:**
  - **ForkWorld is too clean.** The "50% any-arm ceiling" and the disjoint-action-set structure are *engineered* to satisfy Assumption 1. The paper is honest about this ("we design experiments around a minimal controlled environment that satisfies Assumption 1 by construction"), but it means the experiments are existence proofs for the theorems, not stress tests. How tight are the bounds on noisier, more realistic environments? Unknown.
  - **DRQN (states) ≠ DRQN (states+actions) in an odd way.** Stacked DQN (k=3, +actions) achieves **0.88** overall, exceeding both DRQN variants (0.75, 0.78). The paper notes this without fully explaining why a 3-step window beats a GRU. Possible explanations (shorter credit assignment, fewer parameters, better optimization landscape) deserve at least an ablation.
  - **Oracle DRQN (0.78) ≤ Stacked DQN (0.88).** An oracle-domain recurrent agent underperforms a memoryless-with-window agent. The paper does not comment on this. Suggests instability in DRQN training or interaction between the domain label and the recurrent dynamics; worth flagging.
  - **Theorem 2's `d_F` metric is bounded-test-class dependent.** The result is *not* "the full transition kernel is recoverable"; it is "the kernel is recoverable up to distinctions measured by F." If `F` is small, the guarantee is correspondingly weak. The paper acknowledges this ("if F is too small, only the corresponding coarse aspects of the local transition kernel are identified") but the practical bite depends on choosing a sufficiently rich `Ψ_F`.
  - **The "memoryless DQN plateaus at 0.19" finding is more interesting than the headline number.** The fact that conflicting gradients at the bottleneck *prevent* the trivial 50% baseline is a useful observation about gradient interference in POMDPs, but it deserves more analysis (e.g., gradient magnitude, Q-value spread across domains).
  - **No scaling story.** How do the bounds degrade as the number of domains, the bottleneck time horizon, or the goal family grow? The `η_σ + η_σ̃ < 1` condition is a *sufficient* condition for separation — how tight is it in practice?

## Strengths

- Clean, falsifiable theorem statements with quantitative bounds — not the usual "we argue that…" hand-waving.
- Sharp positioning vs. the existing world-model-necessity literature; the gap (no domain index) is identified and filled.
- Empirical design that *provably* satisfies the theorem preconditions — unusually rigorous alignment of theory and experiment.
- Direct probes (linear probe for domain separation, MLP decoder for transitions) that map onto each theorem's quantitative prediction.
- Theorem 2 is a substantive positive result: not just "memory must distinguish domains" but "the right memory *also gives you a world model*."
- The memoryless-DQN-under-50% observation is a genuinely useful negative result for the deep RL community.

## Limitations

- Theorems are necessary-condition statements about what *must* be true; they do not prescribe architectures, training algorithms, or sample-complexity bounds.
- Assumption 1 (uniform bottleneck reachability + disjoint action sets with value gap) is strong — realistic POMDPs may violate it or satisfy it only in expectation.
- Theorem 2's guarantees scale with `ε_val + ε_span`, both of which are environment-dependent and not directly estimable from data in general.
- Experiments are confined to a single 4-condition gridworld; no image-based, language-conditioned, or large-state-space tests.
- Quantitative tightness of the TV lower bound is not empirically tested (the paper shows *separation* but does not measure the gap between the empirical TV and the theoretical `1 − η_σ − η_σ̃`).
- The result is theoretically beautiful but practically *prescriptive* only in a narrow sense: it tells you memory must separate, but not how much memory, in what form, or how to train it.

## Relevance

直接对应"通用智能体/长视野决策"方向的两个核心问题:**(1)** 何时显式记忆不可省略(在隐藏域导致观测瓶颈冲突时);**(2)** 价值充分的价值表征能否反推出局部世界模型(为基于值的 RL → 世界模型蒸馏 提供形式化背书)。对于做多任务/元 RL/智能体记忆架构(Transformer、RNN、检索记忆)的研究者,本文给出的定理可作为新方法的必要条件验证基准。

## Questions for Further Exploration

- Can Assumption 1 be relaxed to **expected** bottleneck reachability or to action sets that only need to be *value-separated* in expectation? This would broaden the theorems to settings where the bottleneck is stochastic.
- The Oracle DRQN underperforms Stacked DQN — is this a **training instability** in DRQN (e.g., truncated BPTT, target network interactions with recurrence) or a real failure mode? Worth a dedicated ablation.
- How does the TV lower bound `1 − η_σ − η_σ̃` scale with the **number of domains**? Is there a phase transition (e.g., memory must distinguish `|Σ|` modes simultaneously)?
- Can the **probe-goal construction** be operationalized as a regularizer or auxiliary loss for training generalist agents? E.g., "maximize `ε_val`-value-sufficiency of the recurrent state on a constructed probe-goal family."
- Does Theorem 2 give a **constructive recipe** for distilling a world model from a value-trained agent without ever observing the true transition kernel? The decoder `D_P(m, a)` is non-parametric only over memory; can it be approximated by an MLP that is then reusable at deployment time?
- Empirical study on a **richer environment** where Assumption 1 holds approximately (e.g., a partially observed navigation task with hidden goal-conditioned dynamics) — does the bound predict performance gaps between memory-augmented and memoryless agents?
- How does the result interact with **retrieval-augmented memory** rather than RNN memory? Is a frozen-retrieval "memory" sufficient for Theorem 2's value-sufficiency, or does memory need to be updated online?
- Connection to **linear-probe methodology** in mechanistic interpretability: the Theorem 1 validation (linear probe on bottleneck state) is essentially the same setup used to identify "domain neurons" in LLMs. Is there a direct bridge?
