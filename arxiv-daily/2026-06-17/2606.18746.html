Theory · Memory Necessity · POMDP

What Must Generalist Agents Remember?

Khurram Yamin, Namrata Deka, Maitreyi Swaroop, Albert Ting, Jeff Schneider, Bryan Wilder  ·  Carnegie Mellon University · Georgia Tech

Key Finding Near-optimal generalist agents must carry domain-separable memory at observation bottlenecks — and that same memory can decode a usable world model.

separation theorem latent-domain POMDP value-sufficient memory world models ForkWorld

● Key Metric

0.88

Memory-equipped agents hit 0.88 success while a memoryless DQN stalls at 0.19 — below even the trivial 0.50 any-arm ceiling.

● Why This Matters

First necessity theorem that drops the domain-index assumption: it proves memory must separate hidden domains, and shows value-sufficient memory yields a decodable world model — citation-worthy bounds for generalist-agent design.

● Critical Edge

ForkWorld is engineered to satisfy the theorems' preconditions. The experiments are existence proofs, not stress tests — no evidence the bounds survive noisier, realistic POMDPs.

Methodology

  • Latent-domain POMDP. Domain σ ∈ Σ is sampled at episode start but never revealed — the agent must infer it from trajectory alone. Dependence enters only via μ_σ, P_σ; rewards are domain-invariant.
  • Shared policy head: π(a|M_t, ψ) — same memory + same goal ⇒ same action, so all domain knowledge must live in memory M_t = f(h_t).
  • Theorem 1 (Separation). Under a shared-bottleneck + disjoint-action-sets assumption, near-optimal policies induce TV-separated memory: TV ≥ [1 − η_σ − η_σ̃]⁺.
  • Theorem 2 (Decodability). Value-sufficient memory implies an explicit transition decoder D_P: M × A → P(S) with dF ≤ 2(ε_val + ε_span).
  • ForkWorld testbed. A T-junction gridworld with a hidden up/down action-permutation domain, provably satisfying Assumption 1 by construction.

Strengths

  • Quantitative, falsifiable theorems — TV and dF bounds with named constants, not "we argue that…".
  • Sharp positioning: the only necessity result that removes the domain index assumption vs. Richens & Everitt (2024) et al.
  • Empirical setup provably satisfies the theorem preconditions — rare theory/experiment alignment.
  • Theorem 2 is a positive result: the right memory also hands you a world model.
  • Memoryless-DQN-below-50% is a genuinely useful negative result on gradient interference in POMDPs.

Limitations

  • Necessary conditions only — no architecture, training algorithm, or sample-complexity prescription.
  • Assumption 1 (uniform bottleneck reachability + disjoint action sets) is strong; realistic POMDPs may satisfy it only in expectation.
  • Theorem 2's guarantees scale with ε_val + ε_span, which are not directly estimable from data.
  • Single 4-condition gridworld — no image-, language-, or large-state tests.
  • Empirical tightness of the TV bound is never measured — separation is shown, not the gap.

Open Questions for Further Exploration

Relax Assumption 1 to expected bottleneck reachability or value-separation in expectation — would the theorems survive stochastic bottlenecks?
Oracle DRQN (0.78) < Stacked DQN (0.88): is this a DRQN training instability (truncated BPTT, target nets) or a genuine failure mode? Needs an ablation.
How does the bound scale with the number of domains — is there a phase transition forcing simultaneous |Σ|-way separation?
Can the probe-goal construction become a regularizer/auxiliary loss to train value-sufficient memory in practice?
Does Theorem 2 give a constructive recipe to distill a world model from a value-trained agent — reusable at deployment via an MLP decoder?
