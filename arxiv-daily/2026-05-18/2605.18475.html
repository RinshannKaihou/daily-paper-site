GAMMA: Global Bit Allocation for Mixed-Precision Models under Arbitrary Budgets

arXiv: 2605.18475

Date: 2026-05-18

https://arxiv.org/abs/2605.18475

一句话总结： GAMMA 提出了一种量化器无关的混合精度位宽分配框架，通过在纯后训练管线中利用可微分松弛学习逐模块精度偏好，再用整数规划投影为严格满足预算约束的离散分配，单次训练即可服务任意部署目标，在 Llama 和 Qwen 模型上以 2.5-bit 平均精度达到固定 3-bit 的质量水平。
TL;DR: GAMMA is a quantizer-agnostic mixed-precision framework that learns per-module precision preferences via differentiable relaxation in a post-training pipeline, projects them to exact budget-feasible discrete assignments via integer programming, and enables a single training run to serve arbitrary deployment targets—matching fixed 3-bit quality at 2.5-bit average precision on Llama and Qwen models.

核心贡献

  • 提出了 GAMMA 框架，在纯后训练（PTQ）管线中学习逐模块的位宽偏好，无需权重重训练，解决了现有方法要么需要量化感知训练（QAT）、要么依赖静态代理指标的根本限制。
  • 使用增强拉格朗日方法（augmented Lagrangian）精确约束全局平均位宽预算，避免软惩罚方法无法精确收敛到目标预算的问题。
  • 通过整数规划将学习到的软偏好投影为严格满足预算约束的离散分配，并证明了学习到的偏好分数可在不同预算间复用（score reuse），将每次预算适配从数小时缩短至几分钟。
  • 在 Llama3.1-8B、Qwen3-8B、Qwen3-14B 甚至 Qwen3-32B 上，GAMMA 在 2.5–3.0 bit 极低精度区间显著优于固定精度基线和搜索式混合精度方法（AMQ），最高提升达 +12.99 Avg。

Key Contributions

  • Introduces a two-stage pipeline: Stage I learns differentiable bit-selection masks via Gumbel-Softmax relaxation with a teacher-forced layer-wise hidden-state reconstruction objective; Stage II projects soft preferences to budget-feasible discrete assignments via integer linear programming (0–1 knapsack formulation).
  • Demonstrates that the augmented Lagrangian formulation is critical for exact budget convergence—replacing it with a DNAS-style multiplicative penalty degrades performance by up to −3.45 Avg.
  • Shows that the layer-wise reconstruction loss outperforms end-to-end cross-entropy by up to +3.29 Avg., as the task-specific signal overfits small calibration sets.
  • Achieves score reuse: preference scores learned at one target budget transfer to arbitrary budgets by re-solving only the integer program (Pearson correlation between soft scores and discrete assignments consistently exceeds 0.7, often >0.9).
  • Reveals structured allocation patterns: k_proj consistently receives higher precision (errors amplified through attention scoring), while v_proj is kept near minimum; this pattern is stable across budgets (cosine similarity 0.9755) and calibration sets (0.9913).

Methodology

GAMMA operates on a decoder-only Transformer with L layers, each containing linear modules {q, k, v, o, up, gate, down}. Pre-quantized weight candidates at multiple bit-widths (B = {2, 3, 4}) are prepared using an off-the-shelf PTQ method (QTIP or AQLM).

Stage I — Differentiable Score Learning: For each module (i, j), trainable logits parameterize a categorical distribution over candidate bit-widths via Gumbel-Softmax. The mixed-precision weight is a probability-weighted combination of pre-quantized candidates. The objective minimizes a teacher-forced layer-wise hidden-state reconstruction loss (MSE between full-precision and mixed-precision layer outputs) under an augmented Lagrangian constraint that enforces the expected average bit-width equals the target budget. Only the mask parameters are optimized; all model weights remain frozen.

Stage II — Budget-Feasible Discretization: The learned probabilities define assignment scores. A 0–1 knapsack-style integer program selects exactly one bit-width per module to maximize total consistency with the learned scores, subject to the global average bit-width constraint. This is solved efficiently via dynamic programming or standard ILP solvers with negligible overhead.

Score Reuse: Because the learned scores encode a stable sensitivity ranking (not budget-specific weights), changing the target budget only requires re-solving the ILP with the same scores, reducing per-budget adaptation to minutes.

Results & Highlights

  • Qwen3-8B at 3.0 bits: GAMMA achieves 81.16 Avg. vs. AQLM's 68.17 (+12.99) and AMQ's 74.16 (+7.00).
  • Llama3.1-8B-Instruct at 3.0 bits: GAMMA outperforms AMQ on all six benchmarks (78.31 vs. 76.68 Avg.), with gains distributed across reasoning, commonsense, and reading comprehension.
  • Memory efficiency: GAMMA at 2.5 bits matches or exceeds fixed 3-bit QTIP quality on Qwen3-8B (76.77 vs. 76.33) and Qwen3-14B (83.11 vs. 81.21).
  • Scalability: On Qwen3-32B, GAMMA-3.0bit (85.97) even surpasses FP16 (84.83).
  • Score reuse: Preference scores learned at b_target=3.0 transfer competitively to 2.5–3.5 bit targets without retraining, consistently outperforming AMQ.
  • Ternary > binary candidate sets: {2, 3, 4} consistently outperforms {2, 4}, indicating that distributing moderate upgrades across more modules is more effective than concentrating large upgrades on fewer modules.
  • Compute efficiency: ~1 hour for score learning on A100 for Qwen3-8B (comparable to Hessian trace computation, 4× faster than AMQ's 4-hour search). Decoding throughput within 3% of uniform baselines.

Critical Assessment

  • Novelty: High. The combination of teacher-forced hidden-state reconstruction + augmented Lagrangian constraint + ILP projection for LLM mixed-precision is novel. The score reuse property is a genuinely useful practical contribution. However, the individual components (Gumbel-Softmax for NAS, augmented Lagrangian, layer-wise reconstruction) are not new individually—the contribution lies in their synthesis for the LLM quantization setting.
  • Rigor: Strong. Ablation studies cleanly isolate each component's contribution. The paper compares against relevant baselines under identical conditions and reports results across multiple models and bit-width ranges. The score reuse analysis with Pearson correlations and cosine similarities provides quantitative support.
  • Reproducibility: Moderate to good. The method uses only standard components (Gumbel-Softmax, ILP solvers, MSE loss). Calibration set details are provided (1024 sequences, length 2048, from RedPajama). Hyperparameters are deferred to Appendix H. The quantizer-agnostic design aids reproducibility, but the ILP solver choice and exact dynamic programming implementation details could be more explicit.
  • Significance: High. The ability to match 3-bit quality at 2.5 bits has practical deployment implications. The score reuse property addresses a real pain point in multi-target deployment scenarios. The framework is quantizer-agnostic, making it applicable to future PTQ methods.
  • Red flags: The evaluation relies primarily on zero-shot classification benchmarks (ARC, PIQA, HellaSwag, etc.). While Appendix A includes GSM8K and RULER, the main narrative underrepresents generative/long-context evaluation. The 2.5-bit results, while impressive, show some individual benchmark regressions (e.g., WinoGrande drops on certain configurations). The method still requires pre-quantized candidates at all bit-widths, which adds preprocessing cost.

Strengths

  • Quantizer-agnostic design works with any PTQ backend (validated on both QTIP and AQLM).
  • Score reuse eliminates retraining for new deployment budgets—a single run serves all targets.
  • Exact budget compliance via augmented Lagrangian + ILP, unlike soft penalty approaches.
  • Strong empirical gains especially in the extremely low-bit (2.5–3.0 bit) regime where uniform quantization collapses.
  • Clean ablation study demonstrates each component's necessity.
  • Structured allocation insights (k_proj sensitivity, pattern stability) contribute to understanding LLM quantization.

Limitations

  • Requires pre-quantized weight candidates at all candidate bit-widths, adding storage and preprocessing overhead proportional to the number of candidate precisions.
  • Evaluation is heavily weighted toward zero-shot classification; generative and long-context evaluations are relegated to the appendix.
  • The method's effectiveness at higher bit-widths (3.5–4.0 bits) shows diminishing returns, as the most sensitive modules are already well-served.
  • Binary candidate set {2, 4} performs notably worse, limiting flexibility when intermediate quantization kernels are unavailable.
  • No evaluation on MoE architectures or multimodal models; generalization beyond standard dense Transformers is untested.

Relevance

本文提出的混合精度量化框架与 LLM 高效部署的研究方向高度相关。GAMMA 在纯后训练管线中学习位宽分配，并通过分数复用实现快速多目标部署适配，这对于资源受限场景下的模型部署具有直接实用价值。其量化器无关的设计也便于与最新的 PTQ 方法结合使用。
