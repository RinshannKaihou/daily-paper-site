> **一句话总结：** 本文证明了原始递归函数可以通过五种动力学框架等价刻画——固定循环ReLU网络的受限迭代、固定多项式ODE的鲁棒计算、带步长参数的离散多项式映射迭代等——揭示了离散与连续多项式动力学之间的结构性不对称性，并展示了组合操作可以从动力学中自然涌现而非作为闭包规则引入。

> **TL;DR:** This paper proves that the class of primitive recursive functions admits equivalent characterizations across five dynamical frameworks: bounded iteration of a fixed recurrent ReLU network, robust computation by a fixed polynomial ODE, bounded iteration of a fixed polynomial map with an external step-size parameter, bounded-range recurrent rho-activation networks, and the classical definition. Composition emerges from the dynamics rather than being postulated as a closure rule. The equivalences expose a structural asymmetry between discrete and continuous polynomial dynamics: rounding and phase selection are free in continuous time but provably impossible for fixed polynomial maps in discrete time.

## 核心贡献

- 证明了原始递归函数类的五种等价动力学刻画：经典定义、固定循环ReLU网络的受限迭代、固定rho-激活循环网络（带编码和精度控制）、鲁棒多项式ODE表示、带步长控制的离散多项式映射迭代
- 引入"阈值-仿射正规形式"（threshold-affine normal form）作为所有等价性证明的核心枢纽，将LOOP程序编译为单一迭代映射
- 证明了两条不可能性结果：固定的多项式映射无法一致地完成整数取整和精确相位选择，而这两项操作在连续时间ODE中几乎免费
- 揭示了各形式化框架之间的结构性不对称：ReLU门提供精确分支、连续时间提供自主取整和控制、步长参数以离散化精度为代价恢复前两者
- 提出了一种无需显式组合闭包规则的计算观：动力学系统通过时钟、相位选择器和纠错机制塑造轨迹来完成计算

## Key Contributions

- Proves a five-way equivalence theorem for primitive recursive functions: classical primitive recursion, recurrent ReLU computation, recurrent rho-activation computation, robust polynomial ODE computation, and step-size-controlled polynomial simulation are all computationally equivalent
- Introduces the threshold-affine normal form as the central proof-level intermediate object, compiling LOOP programs into bounded iteration of a single map of the form P(y) = A_0(y) + sum_j Theta(q_j(y)) A_j(y) with integer-coefficient affine maps
- Establishes two impossibility results (Proposition 2.4 and 2.6): no fixed polynomial map can uniformly round to the nearest integer or realize exact phase selection in discrete time, operations that are essentially free for polynomial ODEs in continuous time
- Develops a discrete Gronwall shadowing lemma (Lemma 4.22) and integer-faithful approximation framework for controlling error propagation across smooth models
- Demonstrates that composition is redundant: bounded iteration of a single ReLU network generates the full class of primitive recursive functions without any explicit closure under composition

## Methodology

The proof architecture proceeds through a carefully designed cycle of implications centered on the threshold-affine normal form:

1. **Primitive recursion to threshold-affine normal form (Section 5):** A LOOP program computing the target function is compiled instruction by instruction. Arithmetic operations are carried by affine maps with integer coefficients, and zero-testing (for loop control) is realized by the threshold function Theta applied to integer arguments. The resulting one-step transition map has the form P(y) = A_0(y) + sum_j Theta(q_j(y)) A_j(y).

2. **Threshold-affine to recurrent ReLU (Section 6):** Each threshold gate Theta(q_j(y)) is replaced by the ReLU gadget Gate(t) = ReLU(t) - ReLU(t-1), which coincides exactly with Theta on all integer arguments. This yields a fixed feedforward ReLU block whose iterates reproduce the computation.

3. **Recurrent ReLU back to primitive recursion (Section 7):** A fixed ReLU block with rational parameters induces a primitive recursive evolution on rational states, since both affine maps with rational coefficients and ReLU are primitive recursive operations.

4. **Uniform threshold-affine to polynomial ODE (Section 8):** A smooth clock (from c_1' = -c_2, c_2' = c_1 generating cos/sin) organizes compute-hold cycles. Polynomial comparators replace threshold gates, driving registers exponentially toward target values. The discrete Gronwall shadowing lemma controls error propagation along the reference orbit.

5. **Uniform threshold-affine to recurrent rho-computation (Section 9):** Integers are encoded as dyadic codes nu(n) = 2^{-n}. The coded one-step update replaces threshold gates with admissible detector blocks. Basin preservation arguments ensure the coded orbit tracks the reference orbit.

6. **ODE to step-size-controlled polynomial (Section 10):** Euler discretization of the polynomial ODE yields a purely polynomial discrete map, with the step size stored as an additional coordinate serving as an explicit computational resource.

7. **Converse implications (Section 11):** Step-size-controlled polynomial maps and rho-networks are shown to compute only primitive recursive functions, using the fact that bounded iteration of fixed polynomial/rational maps with primitive recursive parameters is itself primitive recursive.

A key structural insight is the uniform-robust characterization (Theorem 4.17): the threshold can be relaxed to Theta(u) = 0 for u <= 1/4 and Theta(u) = 1 for u >= 3/4 without affecting the computation, because gate arguments on the reference orbit are always integers. This margin is essential for bridging to the smooth models.

## Results & Highlights

- **Five-way equivalence (Theorem 4.24):** For f: N^d -> N^e, the following are equivalent: (1) f is primitive recursive; (2) f admits a recurrent ReLU computation; (3) f admits a recurrent rho-computation; (4) f admits a uniform threshold-affine normal form; (5) f admits a robust polynomial ODE representation; (6) f admits a step-size-controlled polynomial representation.

- **No uniform polynomial rounder (Proposition 2.4):** There is no fixed polynomial p and integer N such that iterating p N times rounds every near-integer to its nearest integer. This is proved by showing that p^{[N]}(x) - x would need to be simultaneously bounded and non-constant, a contradiction.

- **No exact polynomial phase selector (Proposition 2.6):** No fixed polynomial map can output 1 for n < C steps and 0 thereafter for all C, proved by showing the n-th iterate as a polynomial in C would have infinitely many roots yet fail at C = n.

- **Discrete Gronwall shadowing (Lemma 4.22):** Provides quantitative control on how one-step errors propagate along an orbit, generalizing the classical Gronwall inequality to discrete time. The key bound is e_n <= epsilon * sum_{k=0}^{n-1} L^k with a tube condition ensuring validity.

- **Raw-input assumption:** All models (except the rho-model due to bounded domain) start from the raw integer input (x, 0, ..., 0) with no auxiliary encoding, ensuring the computational work is done entirely by the dynamics.

## Critical Assessment

- **Novelty: High.** The unification of five distinct computational frameworks under a single equivalence theorem is novel. The key insight that composition is redundant for primitive recursion when viewed through dynamics is surprising and runs counter to the standard assumption in subrecursive function theory. The impossibility results for polynomial maps in discrete time are clean and elegant.

- **Rigor: Very high.** The paper provides complete formal definitions for all five models, detailed proof architectures, and full proofs for every implication. The discrete Gronwall shadowing framework and integer-faithful approximation concepts are carefully developed. The proof-level normal form provides a clean organizational structure.

- **Reproducibility: Good.** All definitions are precise and self-contained. The construction is explicit (compilation from LOOP programs, specific ReLU gadgets, explicit ODE constructions with clock/comparator mechanisms). However, the constructions are primarily of theoretical interest and involve astronomical state dimensions and bounds.

- **Significance: High for theoretical computer science.** This work bridges three major research traditions (implicit complexity, neural computation, analog computation) and provides a unified framework. The observation that different formalisms compensate for each other's limitations (ReLU for branching, continuous time for rounding, step size for discretization) is conceptually important.

- **Red flags:**
  - The paper is purely theoretical with no experimental validation, though this is expected for a computability/complexity theory paper.
  - The constructions involve extremely large state spaces and impractical time bounds, limiting direct practical applicability.
  - The extension to subrecursive hierarchies (Grzegorczyk, FP, FPSPACE) is mentioned as future work but not developed.
  - The paper is quite long (27 pages) and the proof density is high, which may limit accessibility.

## Strengths

- Provides a clean, unified equivalence theorem connecting five seemingly different computational frameworks
- The threshold-affine normal form is an elegant intermediate representation that clarifies the structure of primitive recursive computation
- The impossibility results (Propositions 2.4 and 2.6) are short, direct, and conceptually illuminating
- The raw-input assumption strengthens the results considerably, ensuring no computational work is hidden in encoding
- The paper clearly articulates what each computational resource contributes: ReLU provides branching, continuous time provides rounding and control, step size provides discretization precision
- Opens concrete directions for dynamical characterizations of standard complexity classes (P, PSPACE, etc.)

## Limitations

- The constructions are not intended for practical implementation; the state dimensions and time bounds are astronomically large
- The equivalence is at the level of primitive recursive functions, which is a very large class; the more interesting characterizations of smaller classes (P, FPSPACE) remain future work
- The rho-computation model requires a non-standard encoding (nu(n) = 2^{-n}) and bounded domain, which differs from the raw-input convention of the other models
- The paper does not address the quantitative relationships between the models (e.g., how state dimension, polynomial degree, or network depth relate to the complexity of the computed function)
- No discussion of how these results relate to modern neural network architectures (transformers, etc.) beyond the foundational connection

## Relevance

本文对理解连续动力学系统（如ODE和循环神经网络）的计算能力提供了精确的理论框架。其核心洞察——组合操作可以从动力学中涌现而非显式引入——对于理解神经网络的计算机制和复杂度理论具有重要意义，并为未来通过限制动力学资源来刻画标准复杂度类（P、PSPACE等）提供了清晰的研究路线。

## Questions for Further Exploration

- Can the equivalence theorem be refined to characterize smaller complexity classes (e.g., polynomial time, logarithmic space) by restricting the dynamical resources (iteration count, polynomial degree, state dimension) within the same framework?
- How does the step-size-controlled polynomial model relate to the discrete ODE frameworks of Bournez-Durand and Antonelli-Durand-Kontinen, which take the sign function as a primitive operation?
- What is the minimal polynomial degree needed to compute specific primitive recursive functions under each model, and how does it grow with the function's definition complexity?
- Can the discrete Gronwall shadowing framework be applied to analyze numerical stability and error propagation in practical neural network training?
- Do the impossibility results for polynomial maps extend to broader classes of discrete-time dynamical systems, or are polynomials uniquely limited in this regard?
