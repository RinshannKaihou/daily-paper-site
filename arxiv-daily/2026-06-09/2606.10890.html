Methodology 方法

§ Method

Instead of data-free heuristics (min-max, MSE grid search), PiSO uses a small calibration set to formulate scale selection as a piecewise-smooth objective — the round-to-nearest error partitions into finitely many intervals, each with a closed-form minimizer.

  1. 1Interval partition. For a given weight tensor and target bit-width, the quantization-error function of the scale parameter breaks into a finite set of intervals; the optimum must lie at a boundary or at a closed-form interior point.
  2. 2Enumerate & select. Evaluate the closed-form minimizer inside each interval; return the global best. No gradient descent, no grid search.
  3. 3Group-wise extension. Principled heuristics lift the channel-wise solution to per-group scales for finer-grained quantization.
  4. 4Interleave with error correction. Alternate PiSO with weight clipping / learned rounding for compounding gains.

Strengths & Limitations 优缺

§ Assessment

Strengths

  • Exact optimization replaces heuristic scale tuning — principled and provably optimal.
  • Consistent gains across models and bit-widths, with no inference-time cost.
  • Complementary to error-correction methods — additive when interleaved.
  • Practical relevance for low-bit LLM deployment.

Limitations

  • Requires a small calibration set, unlike purely data-free heuristics.
  • Wall-clock overhead at model scale is not reported in the abstract.
  • Comparison to SOTA learned quantization (GPTQ, QuIP#, AWQ) is not assessable here.
