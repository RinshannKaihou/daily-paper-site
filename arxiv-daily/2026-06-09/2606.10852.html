LLMs systematically distort true information through ordering, framing, and emphasis—without ever fabricating a fact.

Janus: A Benchmark for Goal-Conditioned Information Distortion in LLMs

0.13avg distortion, Media / Information

Drops to near-zero in Law — same model, same fact pool, different framing norm.

Why This Matters

A subtler, arguably more dangerous failure mode than hallucination: misleading impressions built from verified truths.

Critical Edge

The framing judge is itself an LLM — evaluation risks circularity; 160 scenarios across 8 domains is modest.

From Critical Assessment & Limitations

Methodology · Paired Open-Ended Generation

Each scenario holds the fact pool constant and varies only the prompt: a neutral condition and a goal-conditioned condition carrying an institutional objective. Any divergence in the output is therefore distortion — not fabrication.

Five Distortion Dimensions

  1. Selection · ratio of included favorable to adverse facts
  2. Emphasis · token-mass differential across facts
  3. Ordering · favorable facts surfaced before adverse ones
  4. Specificity · loss of numerical precision from source
  5. Framing · LLM-judged evaluative direction φ ∈ {−1, 0, 1}
  6. ·

Four-stage dataset pipeline: domain/schema design → LLM candidate facts → manual author review → artifact controls balancing favorable/adverse properties.

Results & Highlights · 12 LLMs · 8 Domains

Every tested model moves favorable information earlier and adopts more goal-congruent language under goal conditioning. Scale and reasoning mode give no monotonic protection — direction of effect varies by model family.

Strengths

  • Paired design cleanly isolates distortion from fabrication.
  • Five complementary dimensions grounded in communication theory.
  • Evaluation spans 12 diverse LLMs under identical conditions.
  • Domain-sensitivity analysis surfaces contextual factors.
  • Artifact controls balance favorable and adverse fact properties.
  • Public dataset and code release.

Limitations

  • Constructed scenarios — limited ecological validity.
  • 160 scenarios across 8 domains is modest in scale.
  • Automated sentence-fact matching adds measurement error.
  • Single-turn only — no multi-turn or RAG evaluation.
  • Diagnostic only — no mitigation strategies proposed.
  • Framing judge is an LLM — circularity risk in evaluation.

Questions for Further Exploration

  1. Can JANUS-style distortion metrics be used as training signals to reduce goal-conditioned distortion while preserving useful communication?
  2. How does distortion manifest in multi-turn conversations where the model can progressively shape the user's understanding?
  3. Do RAG systems exhibit similar distortion when retrieval results contain conflicting evidence?
  4. Can prompting strategies (e.g., requiring balanced-presentation checklists) effectively mitigate distortion?
