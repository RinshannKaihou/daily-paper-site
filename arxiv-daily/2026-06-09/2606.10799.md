> **一句话总结：** 本文提出了基于(Γ_i, Σ_i, T_i)三元组架构的严格逐步验证框架，将数学证明验证从全局异常检测重构为构造性展开（constructive elaboration），在21个研究级数学证明的诊断套件上实现了零误接受，同时发现剩余的拒绝案例主要源于"学究式过度严格"——即未声明的领域惯例而非真正的逻辑缺陷。

> **TL;DR:** Reframes LLM proof verification from global anomaly detection to constructive step-level elaboration using a tripartite (internal context, external knowledge, background theory) architecture, achieving zero false acceptances on a curated 21-proof diagnostic suite while revealing that remaining rejections stem from unstated domain conventions rather than logical errors.

## 核心贡献

- 提出了(Γ_i, Σ_i, T_i)三元组解释框架：内部上下文（证明文本中的定理/定义）、外部知识（文献引用）、背景理论（平凡重写规则），系统化建模推理步骤的逻辑依赖
- 将验证从直接异常检测重构为构造性展开：如果模型无法为某一步生成有效的展开推导，则怀疑该步存在逻辑缺陷
- 设计了状态驱动的自主验证agent，包含OPEN→VERIFIED、OPEN→FLAWED、FLAWED→OPEN（确认阶段）和动态分解四种状态转换
- 在21个精心策划的研究级证明（来自FirstProof挑战、Aletheia和OpenAI）上实现了零误接受
- 发现"学究式过度严格"（pedantic hyper-rigor）现象：agent因不熟悉特定领域惯例（如实半单Lie群的子群默认为线性的）而拒绝有效证明

## Key Contributions

- Introduces the tripartite explanation framework (Γ_i, Σ_i, T_i) that explicitly traces internal contexts, external knowledge, and background theories for each deduction step
- Reframes verification from black-box anomaly detection to constructive elaboration, where anomaly detection emerges naturally from the generative process
- Designs a state-driven agentic workflow with strict halting criteria: acceptance (all verified), rejection (confirmed flaw), and exhaustion (5 consecutive resumes without progress)
- Achieves zero false acceptances on a curated 21-proof suite including research-level mathematics
- Identifies "pedantic hyper-rigor" as a distinct failure mode: rejection due to unstated domain conventions rather than logical errors, effectively exposing implicit ambiguities in expert benchmarks

## Methodology

The framework treats a mathematical proof $S$ as a string decomposed into steps $(s_1, s_2, \ldots, s_n)$. For each deduction step $s_i$ with assertion $a_i$, the agent produces an elaboration $s'_i$ that must:
1. Conclude with the identical assertion as $s_i$
2. Strictly preserve the original reasoning trajectory
3. Explicitly extract the tripartite dependency $(Γ_i, Σ_i, T_i)$

The agent maintains a Theorem Ledger tracking external claims $Σ_i$. A step transitions to VERIFIED only if every external item is supported by a proof sketch or precise citation. The FLAWED state triggers an adversarial confirmation phase to guard against false positives. Dynamic decomposition allows splitting coarse steps into finer sub-steps.

The diagnostic suite contains 21 proofs: 11 valid (9 from Aletheia, 2 from OpenAI) and 10 invalid (8 from FirstProof baselines, 1 from Aletheia, 1 from OpenAI), drawn from research-level mathematics including topics in algebraic topology, Lie theory, and geometric analysis.

Baselines include: (1) Global GPT-5.4-xhigh (zero-shot LLM-as-judge), (2) Global Codex 5.4-xhigh (sentence-by-sentence reading), and (3) Codex 5.4-xhigh Sentence-by-Sentence ablation (mechanical splitting without tripartite constraints).

## Results & Highlights

- **Global baselines fail on adversarial examples:** Standard global evaluation successfully identified 8/10 invalid proofs but was systematically deceived by the remaining 2, including Aletheia 7A—a formally rigorous but mathematically vacuous pseudoproof.
- **Zero false acceptances:** The step-level agent rejected all 10 invalid proofs, including the adversarial cases that fooled global evaluators, using avg@3 with multi-pass verification.
- **Pedantic hyper-rigor on valid proofs:** Of 11 valid proofs, the agent failed to verify only 3. These failures were not due to reasoning deficits but because the base model lacked familiarity with domain conventions (e.g., subgroups of real semi-simple Lie groups being linear by convention).
- **Ablation confirms tripartite constraints are essential:** The sentence-by-sentence ablation without (Γ, Σ, T) constraints still produced false acceptances, demonstrating that granularity alone is insufficient.
- **Error taxonomy shift:** The framework fundamentally changes failure modes from "logical hallucinations" (accepting flawed proofs) to "pedantic hyper-rigor" (rejecting valid proofs due to unstated conventions).

## Critical Assessment

- **Novelty:** High. The tripartite (Γ, Σ, T) framework for modeling deduction dependencies is novel, as is the constructive elaboration approach that replaces direct anomaly detection. The connection to de Bruijn's Mathematical Vernacular is well-motivated.
- **Rigor:** Strong methodological design with clear formal definitions, state transition rules, and halting criteria. The 21-proof diagnostic suite is small but carefully curated with unambiguous ground truth.
- **Reproducibility:** Code and prompts are publicly available. However, the framework depends on frontier models (Codex 5.4-xhigh) which may not be freely accessible.
- **Significance:** Important for the mathematical reasoning community. The finding that forcing detailed elaboration naturally surfaces logical errors has broad implications for agentic reasoning systems. The "pedantic hyper-rigor" discovery is a genuine insight about the nature of mathematical communication.
- **Red flags:** The sample size (21 proofs) is very small. The framework relies on the base model's mathematical knowledge—if the model doesn't know the relevant domain conventions, it will conservatively reject valid proofs. The external knowledge retrieval component (internet search) introduces variability. The paper is by a single independent researcher, which is unusual for this level of contribution.

## Strengths

- Principled theoretical framework with clear formalization
- Constructive approach that makes verification transparent and auditable
- Zero false acceptances—a strong practical guarantee
- The "pedantic hyper-rigor" concept is a genuine and insightful discovery
- Well-designed ablation study isolating the contribution of each component
- Detailed worked traces showing the agent's verification process

## Limitations

- Very small evaluation set (21 proofs); generalization is unclear
- Depends heavily on the base model's domain knowledge for external knowledge verification
- External knowledge retrieval via internet search is uncontrolled and variable
- The tripartite structure adds significant generation overhead
- Does not compare against formal verification systems (Lean, Coq)
- Single-author work; could benefit from broader validation

## Relevance

本文提出的逐步构造性展开方法为提升LLM在数学推理中的自我纠错能力提供了新范式，对设计可靠的agentic推理系统有直接价值。其关于"学究式过度严格"的发现也揭示了AI与人类专家之间在隐含知识表达上的根本差异。
