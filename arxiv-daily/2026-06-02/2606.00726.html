Latent Reward Steering: An Adaptive Inference-Time Framework that Implicitly Promotes Cognitive Behaviors in Reasoning LLMs

arXiv: 2606.00726

Date: 2026-06-02

https://arxiv.org/abs/2606.00726

一句话总结： 本文提出 Latent Reward Steering (LRS)，一种在推理时通过奖励梯度直接优化稀疏自编码器（SAE）隐状态来自适应地促进认知行为的框架，无需预定义行为标签即可在多个数学与科学推理基准上稳定提升 LLM 的推理表现。
TL;DR: This paper proposes Latent Reward Steering (LRS), an inference-time framework that improves LLM reasoning by reward-guided optimization of sparse-autoencoder (SAE) latent states, adaptively promoting cognitive behaviors without predefined behavior labels and achieving consistent gains across multiple reasoning benchmarks.

核心贡献

  • 将认知行为控制重新定义为隐状态优化：首次将 LLM 推理中的认知行为控制问题建模为隐式隐状态优化，而非显式选择预定义行为（如验证、回溯）。
  • 提出 LRS 框架：设计了一套自适应推理时干预机制，通过训练一个轻量级隐状态奖励模型，利用奖励梯度对脆弱的 SAE 隐状态进行定向修正，并配合奖励-置信度门控实现选择性干预。
  • 无需预定义行为或固定方向：不同于已有的表征级干预方法（如 SEAL、CREST、RISER），LRS 不依赖预定义的认知行为标签、固定转向方向或有限向量库。
  • 跨模型与基准的稳定提升：在 Open-Reasoner-7B 和 Open-Reasoner-1.5B 上，于 MATH-500、AIME24/25、GPQA-Diamond、AMC23、INEQ MATH 等六个基准上均取得一致的性能提升。
  • 隐式促进有益认知行为：事后分析表明，LRS 干预后的推理轨迹更频繁地展现出纠正错误（Course Correction +20%）、约束检验（Constraint Grounding +10%）和答案验证（Solution Verification +10%）等有益行为。

Key Contributions

  • Reframes cognitive behavior control as latent-state optimization: For the first time, cognitive behavior control for LLM reasoning is modeled as implicit latent-state optimization rather than explicit selection of predefined behaviors.
  • Proposes the LRS framework: An adaptive inference-time mechanism that trains a lightweight latent reward model and uses reward gradients to correct fragile SAE latent states, with reward-and-confidence gating for selective intervention.
  • Eliminates dependency on predefined behaviors or fixed directions: Unlike prior representation-level steering methods (SEAL, CREST, RISER), LRS does not rely on predefined cognitive behavior labels, fixed steering directions, or finite vector libraries.
  • Consistent gains across models and benchmarks: Achieves consistent improvements on Open-Reasoner-7B and Open-Reasoner-1.5B across six benchmarks: MATH-500, AIME24, AIME25, GPQA-Diamond, AMC23, and INEQ MATH.
  • Implicitly promotes useful cognitive behaviors: Post-hoc analysis shows LRS-steered traces exhibit significantly higher rates of course correction (+20%), constraint grounding (+10%), and solution verification (+10%).

Methodology

The paper formalizes the problem as an inference-time intervention on a frozen reasoning model's latent states during autoregressive generation.

SAE-based Intervention Space. At generation step t, the hidden state h_t is encoded into a sparse latent vector z_t using a pretrained Sparse Autoencoder (SAE): z_t = f_enc(h_t). The SAE provides a low-dimensional, interpretable sparse code (dz = 10 for Open-Reasoner-7B, dz = 5 for Open-Reasoner-1.5B) that serves as the intervention space.

Latent Reward Training (Stage 2). A lightweight Transformer reward model R_θ is trained on sequences of SAE latent codes derived from reasoning traces, labeled only by final-answer correctness (binary). The model is trained with binary cross-entropy to predict p_i,t, the probability that a latent prefix z_i,s_i:t belongs to a correct trace. To handle class imbalance, weighted random sampling is used.

Online Reward-Guided Latent Correction (Stage 3). During inference:

  1. The current activation h_t is encoded to z_t^(0).
  2. The reward model evaluates the current latent: r_t^(0) = R_θ(z_t^(0)).
  3. If the reward-confidence gate triggers (I_steer(t) = 1), the latent is optimized via normalized gradient ascent for K steps: z_t^(k+1) = z_t^(k) + α * ∇R / (||∇R||_2 + ε).
  4. Only the latent residual Δz_t = z_t^(K) - z_t^(0) is decoded back to activation space: Δh_t = Δz_t * W_dec, and added to the original hidden state: h'_t = h_t + Δh_t.

Selective Reward and Confidence Gating. Steering is triggered when either:

  • The reward score is low: r_t < τ_r, or
  • The reward is acceptable but decoding confidence is low: r_t ≥ τ_r AND c_{t-1} < τ_c.

This selective mechanism prevents disrupting already-healthy reasoning states, which is critical because the ungated variant (LRS BASIC) degrades performance on some benchmarks.

Results & Highlights

Main Results (RQ1). Full LRS consistently improves over standard zero-shot decoding on all six benchmarks for Open-Reasoner-7B:

  • MATH-500: 79.4 → 83.8 (+4.4)
  • AIME24: 16.6 → 26.6 (+10.0)
  • AIME25: 16.6 → 26.6 (+10.0)
  • GPQA-Diamond: 32.3 → 39.4 (+7.1)
  • AMC23: 50.0 → 60.0 (+10.0)
  • INEQ MATH: 46.0 → 60.0 (+14.0)

The same trend holds for Open-Reasoner-1.5B, indicating generalization across model scales. Notably, LRS outperforms both CoT and few-shot prompting on most benchmarks, while those prompt-based methods sometimes underperform standard zero-shot decoding on reasoning-tuned models.

Recovery Exceeds Degradation. A matched-pair analysis (N = 398, OR-7B) shows LRS recovers 17.3% of examples from wrong to correct while degrading only 7.8% from correct to wrong, yielding a net positive effect of +9.5%.

Latent Reward Signal Quality (RQ2). The learned reward model successfully distinguishes correct from incorrect reasoning traces. Final-token and trajectory-mean reward scores are consistently higher for correct traces across all benchmarks, confirming the reward model captures meaningful reasoning-quality signals in latent space without behavior annotations.

Implicit Cognitive Behavior Promotion (RQ3). Post-hoc GPT-4o annotation of matched base vs. LRS traces reveals:

  • Course Correction: +0.20
  • Constraint Grounding: +0.10
  • Solution Verification: +0.10
  • Structured Decomposition: +0.04
  • Strategic Planning: +0.04

These shifts suggest reward-guided latent optimization changes fragile reasoning steps in ways associated with more frequent helpful cognitive behaviors.

Selective Intervention is Essential. The ungated LRS BASIC improves some datasets but degrades others (e.g., GPQA-Diamond -1.5, AMC23 -5.0 on OR-7B), demonstrating that indiscriminate latent updates can harm healthy reasoning. Full LRS with gating fixes this.

Efficiency. Average wall-clock time increases by 1.35× (115.3s → 156.2s per problem). The gate skips 72.1% of tokens, steering only 27.9%. On average, 725 steering triggers occur per problem, with the average ||Δz|| per steered token being 1.24.

Critical Assessment

Novelty: The core conceptual contribution is framing cognitive behavior control as implicit latent-state optimization rather than explicit behavior selection. While representation-level steering is not new, eliminating the dependency on predefined behaviors, behavior-specific vectors, or fixed steering directions is a meaningful advance. The use of a learned reward model on SAE latents is a novel combination of ideas from mechanistic interpretability and inference-time optimization.

Rigor: The empirical evaluation is thorough: two model scales, six benchmarks, matched-pair trace analysis, reward-score diagnostics, post-hoc behavior annotation, ablations on gating and steering strength, and SAE interpretability analysis. The matched-pair recovery/degradation breakdown is particularly convincing evidence that gains are not random perturbations. However, the behavior annotation relies on GPT-4o as an external judge, which introduces some subjectivity, though the authors are transparent about this being diagnostic only.

Reproducibility: The authors release code and use publicly available pretrained SAE checkpoints. Training details are relegated to an appendix, but the main methodology is described in sufficient detail for replication. The reliance on specific SAE checkpoints (Venhoff et al., 2025a) means reproduction depends on the continued availability of those checkpoints.

Significance: The work situates itself at the intersection of three active threads: (1) inference-time compute scaling, (2) mechanistic interpretability via SAEs, and (3) cognitive behavior control for reasoning. By showing that reward-guided latent optimization can implicitly elicit useful behaviors without explicit supervision, it opens a direction that may generalize beyond math reasoning to other domains.

Red flags: The training-inference mismatch (reward model trained on sequence-level latents but queried on single-token states) is acknowledged as a limitation. The 1.35× slowdown, while modest by test-time scaling standards, is non-negligible for production use. The ethical considerations section is appropriately cautious about misaligned reward models amplifying undesirable patterns.

Strengths

  • Strong conceptual framing: The shift from explicit behavior control to implicit latent optimization is well-motivated and clearly articulated.
  • Comprehensive empirical validation: Multiple backbones, benchmarks, and diagnostic analyses support the main claims.
  • Selective gating is crucial and well-justified: The contrast between LRS BASIC and full LRS clearly demonstrates the necessity of selective intervention.
  • Mechanistic insight: The SAE dimension interpretability analysis and case studies provide intuition for why the method works.
  • Honest limitations: The authors openly discuss inference overhead, training-inference mismatch, and ethical risks.

Limitations

  • Inference overhead: Reward model evaluation and latent gradient updates during decoding introduce a 1.35× slowdown, and while gating reduces unnecessary interventions, LRS remains slower than standard decoding.
  • Training-inference mismatch: The reward model is trained on full latent reasoning sequences but queried on local token-level states for efficiency. Future work could explore prefix-level or memory-augmented reward estimation.
  • Limited domain scope: Evaluation is restricted to mathematical and scientific reasoning benchmarks. Generalization to open-ended generation, code, or safety-sensitive scenarios is not established.
  • Behavior annotation subjectivity: Post-hoc cognitive behavior labels are assigned by GPT-4o and used only diagnostically, but this still introduces potential bias in quantifying "implicit" behavior promotion.
  • Dependence on SAE quality: The method's effectiveness depends on the pretrained SAE exposing reasoning-relevant features, which may vary across models and training regimes.

Relevance

本文提出的隐式隐状态优化范式对当前推理时计算扩展和可解释性研究具有重要参考价值，尤其适合关注如何在不修改模型权重、不依赖人工设计提示的前提下自适应提升 LLM 推理质量的研究者和工程师。

Questions for Further Exploration

  1. How would LRS perform on non-mathematical reasoning tasks such as legal reasoning, medical diagnosis, or open-ended creative writing, where "correctness" is harder to define?
  2. Can the training-inference mismatch be resolved by training the reward model directly on single-token latent states, or by using a recurrent/memory-augmented architecture for the reward model?
  3. How sensitive is LRS to the choice of SAE checkpoint? Would jointly training the SAE and reward model end-to-end yield better or worse results?
  4. Can the implicit cognitive behaviors promoted by LRS be made more explicit and controllable, e.g., by conditioning the reward model on desired behavior types?
  5. What are the safety implications of latent steering when the reward model is trained on user-provided or potentially biased correctness signals?
