Hidden Thoughts Are Not Secret: Reasoning Trace Exposure in LLMs

Yu-An Lu, Ci-Yang Tsai, Yu-Lin Tsai, Raluca Ada Popa, Chia-Mu Yu

A lightweight prompting technique elicits hidden reasoning traces from LLMs via code-paradigm transfer, achieving 96.7% of oracle internal-trace distillation performance — proving interface-level trace hiding cannot prevent capability extraction. 仅通过提示词即可从隐藏推理的大模型中诱导出内部推理痕迹，蒸馏性能达到 oracle 的 96.7%，对 o1/o3/Gemini Thinking/Claude Extended Thinking 的安全部署具重要警示意义。

Editor's Note
This paper directly challenges the security model of hidden-reasoning APIs. If a simple prompt wrapper can recover nearly all internal reasoning utility, the "trade-secret" defense for reasoning chains collapses.

An Enigma of Artificial Reason

Mingzhong Sun, Teresa Yeo, Armando Solar-Lezama, Tan Zhi-Xuan

Frontier reasoning models achieve ≥94.7% accuracy on math problem solving but collapse to 47.9%–78.6% when evaluating correct-answer/flawed-reasoning solutions, revealing an "answer confirmation bias" that corrupts both behavior and internal representations. 发现前沿大型推理模型存在严重的"生成-评估鸿沟"，根源在于"答案确认偏见"，挑战了更强的推理生成能力自动意味着更强的评估能力这一假设。

eMoT: Evolving Memory-of-Thought

Xiang Li, Jiwei Wei, Ke Liu, Yitong Qin, Jinyu Guo, Malu Zhang, Peng Wang, Yang Yang

A unified neuro-symbolic framework treats multi-step reasoning as a dynamic, evolving memory repository. On Qwen-32B, it achieves 93.4% on GSM8K and 100% on Game of 24 — surpassing GPT-4-based structured reasoning baselines. 将多步推理建模为动态演化记忆库，在轻量级 Qwen-32B 上实现接近 GPT-4 的推理性能，证明结构化推理控制可弥补模型规模不足。
