Key Finding

A frozen 30B autoregressive context tower plus a trainable diffusion denoiser tower retains 98.7% of AR quality at 2.42× throughput.

Nemotron-TwoTower: Diffusion Language Modeling with Pretrained Autoregressive Context

Fitsum Reda, John Kamalu, Roger Waleffe, Mostofa Patwary, et al. · NVIDIA · 2026-06-25

diffusion-language-model masked-diffusion mixture-of-experts mamba decoding-acceleration nvidia

Key Metric

2.42×
wall-clock generation throughput vs. the AR baseline (BF16, 2×H100), while keeping 98.7% of aggregate benchmark quality.

Why This Matters

Turns masked diffusion into an open-source decoding-speedup adapter for large pretrained AR+MoE models, trained on only ~2.1T of the 25T-token backbone budget.

Critical Edge

The throughput win costs ~2× resident model-weight memory (a second 30B tower), is measured only on pre-RLHF base checkpoints, and code/math quality still sit below the AR baseline.

Methodology

Backbone. TwoTower instantiates on Nemotron-3-Nano-30B-A3B — an open 30B hybrid (52 layers: 23 Mamba-2, 6 self-attention, 23 MoE). Two copies of the pretrained net take complementary roles: a frozen AR context tower running causally over the prompt and committed tokens (producing per-layer KV caches and final Mamba states), and a trainable diffusion denoiser tower that iteratively refines each noisy block of S [MASK] tokens over T steps.

Dennoiser modifications. Attention is bidirectional within the block but causal w.r.t. past clean blocks, and is layer-aligned (denoiser layer i cross-attends to context layer i). Mamba-2 layers are seeded from the context tower's block-boundary SSM states — richer than broadcasting the final hidden state. A bidirectional Mamba variant was tried but gave only marginal gains at 2× SSM FLOPs, so causal Mamba was kept. Time conditioning uses adaLN-single (~1.5M extra params).

Objective & sampling. The sequence is factorized block-autoregressively; forward masking uses a linear schedule, the loss is mean NLL over masked positions. Sampling is block-wise AR outer loop + inner confidence-unmasking (commit predictions above threshold γ, keep uncertain tokens masked).

Training. Only the denoiser trains (context tower runs once, no-grad), following Nemotron-3-Nano's two-stage curriculum, BF16 AdamW, WSD schedule (peak 1e-4 → 1e-6). Implemented in Megatron-LM with Mamba chunk size matched to block size S.

Strengths & Limitations

✓ Strengths

  • Clean, well-motivated decoupling of context vs. denoising, with ablations showing it beats tying or continued joint AR training.
  • Practical framing: masked diffusion as an adaptation recipe for existing AR checkpoints, not a from-scratch paradigm.
  • Layer-aligned, multi-scale conditioning (per-layer cross-attention + context-seeded Mamba states) exploits the hybrid backbone.
  • Honest ablations, including a sampling-block sensitivity study that exposes a real fragility rather than hiding it.
  • Full open release (weights + Megatron-LM code + recipe) at credible 30B hybrid-MoE scale.

⚠ Limitations

  • ~2× resident model-weight memory from running two 30B towers; significant for memory-constrained serving.
  • Quality still below AR baseline, gap concentrated in code and math — a trade-off, not a strict Pareto win.
  • Brittle sampling block size: deploying larger than trained causes near-total collapse.
  • No aligned/chat evaluation: base-model results only; post-RLHF behavior unknown.
  • Throughput measured on a narrow setup (2×H100, base ckpt); production serving economics not characterized.

Questions for Further Exploration

  1. Does the diffusion path retain chat quality, reasoning, and instruction-following after instruction tuning / RLHF?
  2. Can the ~2× memory cost be cut (weight-sharing, low-rank adapters, offloading the frozen context tower) without sacrificing throughput or quality?
  3. Is the catastrophic sampling-block collapse (Table 4) fixable via multi-block-size curriculum or block-size-conditioned adaLN?
  4. How does it scale beyond 30B, and to denser or more attention-heavy backbones where the Mamba causal-bias story may not hold?
  5. Does the left-to-right commitment bias truly originate from the Mamba-dominated backbone (no intervention experiment confirms it)?
  6. How does TwoTower compare to pure-AR speculative decoding and other diffusion-LM baselines on matched compute, memory, and aligned checkpoints?
