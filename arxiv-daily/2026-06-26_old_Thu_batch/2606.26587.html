§Methodology

SharQ approximates a full-precision linear layer Y = XW with two complementary FP4 execution paths, reframing sparsification as routing, not dropping.

  1. Sparse backbone (online mask). An input-adaptive mask M(X) ∈ S_N:M selects top-k magnitudes per hardware block → X_sp = M(X) ⊙ X. No offline/learned mask.
  2. Quantize the backbone. X̃_sp = D_sp(Q_sp(X_sp)) via sparse-path operators.
  3. Dense residual vs quantized backbone. R = X − X̃_sp = (X−X_sp) + (X_sp−X̃_sp) — carries both mask-out values and sparse-path quantization error.
  4. Two-path reconstruction, shared weights. Ỹ = GEMM_sp(...) + GEMM_dn(...); both paths reuse one FP4 weight body with path-specific scale views; residual accumulates directly into sparse output.
  5. Fused prep kernel (Blackwell). RMSNorm/PTS + pair-topk 4:8 mask + sparse compression + residual construction in one operator, removing ~7.4 pp of prefill overhead.

Theory. ‖XW̄ − (X̃_sp + R̃)W̄‖_F ≤ ‖R − R̃‖_F · ‖W̄‖_2 — couples sparsification + quantization error into one residual-reconstruction problem.

Strengths

  • Training-free, calibration-free, format-agnostic — minimal deployment friction.
  • Principled residual jointly handles sparsification + quantization error, with theory.
  • Strong systems work: fused prep kernel, shared FP4 weights, accumulation-based residual.
  • Broad coverage: dense / MoE / VLM / video diffusion × three FP4 formats.
  • Honest reporting of overheads and failure modes; open-source code.

Limitations

  • Absolute accuracy still below FP16; gains are gap-recovery percentages.
  • Validated mostly on RTX 5090 at fixed batch 4; data-center / variable-load not shown.
  • Gains hinge on Blackwell-class FP4 + N:M sparsity hardware — limited portability.
  • ~11% prefill overhead narrows the win to decode-heavy workloads.
  • Limited head-to-head vs strongest non-FP4 baselines; per-task regressions exist.

Open Questions

  • Transfer to B200/B300 and 70B+ / larger MoE under realistic multi-batch serving?
  • Adaptive residual (skip / lower-precision per layer/token) to cut +11% prefill overhead?
  • Composition with weight-side methods (GPTQ, AWQ, SparseGPT) and rotation (QuaRot, SpinQuant)?
  • Worst-case behavior on adversarial / extreme-outlier activation distributions?
  • Learned-offline residual to push past 63% gap recovery — does it erode training-free value?
  • Interaction with attention accelerators (FlashAttention, SageAttention) beyond one video run?
