核心贡献

  • 揭示表征有效维度（effective dimensionality，以解释 99% 方差所需的主成分数度量）是决定模块化架构在持续学习中是否产生功能影响的关键控制变量，而非架构本身始终重要。
  • 在低维（rich）regime 下，模块化网络展现"梯度化几何"（graded geometry）：相似任务子空间高度对齐、中等差异任务部分正交化、差异大的任务强分离——单模块网络缺乏这种随相似度变化的子空间组织。
  • 在高维（lazy）regime 下，模块化与单模块架构的表现与几何差异极小，说明结构分离的有效性受表征空间约束程度的"门控"。
  • 指出模块化的关键并非完全隔离（本架构仍共享 readout），而是支持受约束的分工与相似度依赖的几何组织；提出"自适应几何"（adaptive geometry）作为持续学习的核心设计原则。

Key Contributions

  • Identify representational dimensionality (measured as the number of PCs explaining 99% of hidden-state variance) as the key organizing variable governing when structural separation becomes functionally relevant in continual learning.
  • Demonstrate a three-way interaction between network architecture (modular vs. single RNN), task similarity (same/near/far rule offset), and weight initialization scale γ (controlling lazy-vs-rich regime) in a sequential A1→B→A2 transfer-interference paradigm.
  • Reveal graded subspace geometry in modular networks under low-dimensional (rich, γ small) regimes: principal angles between task-specific subspaces scale with task dissimilarity (same < near < far), a pattern absent or weaker in the single-network baseline.
  • Show that in high-dimensional (lazy, γ large) regimes, both architectures maintain high effective dimensionality and display broadly similar subspace geometry, so architecture has minimal impact.
  • Argue that the modular network's shared readout (not full isolation) supports graded rather than fully discrete geometry, reframing continual learning as adaptive representational allocation rather than a binary share/isolate choice.

Methodology

Task paradigm: Sequential transfer-interference task from Holton et al. (2025), with three phases A1→B→A2. Each task maps 6 discrete plant cues to positions on a circular dial; summer and winter locations are related by a fixed angular offset (the "task rule"). Task B uses novel stimuli but the same formal structure. Three similarity conditions: Same (identical rule), Near (small angular shift), Far (large angular shift). Each stimulus encoded as a one-hot input vector; output is 4D (two cosine-sine pairs for summer/winter locations). Loss is MSE computed only on the probed season's output pair.

Architectures:

  • Single-network baseline: one recurrent population, h_t = tanh(W_ih x_t + W_hh h_{t-1}), shared readout y_t = W_out h_t.
  • Modular network: two recurrent modules M_A and M_B with task-routed inputs (binary masks m_A, m_B select task-relevant input dimensions), each evolving independently with no inter-module recurrent communication; pre-readout state is concatenation of both module states; shared readout maps to the 4D output. This isolates task-partitioned input processing while keeping output format comparable.

Regime control: After default PyTorch initialization, all trainable weights are rescaled by a global factor γ. Larger γ → lazy/high-dimensional regime; smaller γ → rich/low-dimensional regime. The authors treat γ as a practical probe of representational regime, acknowledging it may also affect optimization dynamics and conditioning.

Training: MSE loss on cosine-sine encoding; loss computed only on the currently probed feature's output pair. Each model initialized from a new random seed and trained on a single task sequence derived from the original experimental schedules.

Representational analysis: Hidden states extracted at the final time step for each stimulus and phase. Effective dimensionality = number of PCs to explain 99% variance. Principal angles computed between task-specific 2D subspaces (PCA within each of two stimulus groups split by stored feature ordering; report the largest principal angle). 3D PCA projections (jointly fit over A1, B, A2) for qualitative visualization. Ablations vary module size and input routing (results in code repository).

Results & Highlights

  • Dimensionality collapses with decreasing γ (Figure 3a,b): Both architectures maintain high effective dimensionality (many PCs for 99% variance) at large γ (lazy regime, γ=2), but dimensionality "decreases dramatically" as γ shrinks (rich regime, γ=0.001), confirming γ as a regime probe.
  • Architecture matters only in low-dimensional regime (Figure 3c): At large γ, both modular and single networks show similar principal-angle geometry across same/near/far conditions (minimal architectural effect). At small γ, the modular network exhibits clear similarity-dependent structure — same condition closely aligned (small principal angle), near intermediate, far strongly separated (large principal angle) — while the single network shows "weaker and less structured" dependence on task similarity.
  • Modular network preserves A2 performance across conditions (Figure 2a,b): The modular network maintains high accuracy across A1→B→A2 for all similarity conditions and γ values; the single network shows a stronger A2 performance drop in the Far condition at the lowest γ values, indicating stronger transfer-interference cost as tasks diverge.
  • Interference is gated by regime (Figure 2e,f): Interference (1 minus mixture weight on task-A component in A2 winter responses, via von Mises mixture fit) "remains low throughout" in the modular network across conditions, while the single network shows "substantially larger interference, especially in the near condition and at the lowest initialization scales."
  • Transfer profile stability (Figure 2c,d): The modular network shows "relatively stable low transfer across similarity conditions and weight scales," whereas the single network "becomes more sensitive to task dissimilarity, with the far condition showing the strongest degradation for small γ."
  • 3D PCA qualitative confirmation (Figure 4): At γ=2 (lazy), trajectories occupy a broad state space with weak structure and limited architectural differences. At γ=0.001 (rich), trajectories collapse into compact, low-dimensional geometry; the modular network shows same-task trajectories closely aligned, near-task intermediate reorganization, far-task stronger separation, while the single network's geometry is "less clearly adapted to task similarity."
  • Ablations confirm robustness: Varying module size and input routing left the qualitative patterns "unchanged" (full results in code repository).

Critical Assessment

  • Almost entirely qualitative/descriptive results: The paper reports trends ("decreases dramatically," "substantially larger," "relatively stable") and visual patterns in figures rather than precise numerical tables with means, standard deviations, significance tests, or seed counts. No specific accuracy/interference numeric values are tabulated, making quantitative comparison and reproduction verification difficult.
  • PCA-based dimensionality is a coarse proxy: The authors themselves acknowledge effective dimensionality is "measured using PCA-based summaries, which are informative but remain approximate proxies for the true intrinsic dimension." The 99%-variance threshold is arbitrary and sensitive to noise.
  • γ confounds dimensionality with optimization dynamics: The authors admit γ "likely affects not only dimensionality but also optimization dynamics, conditioning, and how easily the architectural biases can be expressed." This means the independent variable is not cleanly isolated — observed effects could stem from conditioning rather than dimensionality per se.
  • Restricted to a 2-task A-B-A sequence: The paradigm reproduces a specific human transfer-interference study but does not capture longer task sequences, varying similarity over time, or realistic continual learning scenarios with many tasks — limiting ecological validity.
  • Small-scale RNNs only: Experiments use small recurrent networks (hidden-state dimension not specified numerically in the read portion); generalization to Transformers, CNNs, or large-scale deep learning is untested.
  • Shared readout is a confound for the "graded geometry" claim: The authors attribute graded (rather than discrete) geometry partly to the shared readout coupling the two modules, but they do not ablate this design choice (e.g., compare to fully separate readouts) to confirm the causal mechanism.

Strengths

  • Clean, systematic three-factor experimental design (architecture × task similarity × γ) isolating when modularity matters.
  • The dimensionality-as-gating-variable insight is elegant and provides a unifying lens for conflicting results in prior modular continual learning literature.
  • Mechanistic explanation: graded geometry (principal angles scaling with task dissimilarity) explains both why modularity helps and why maximal isolation is suboptimal.
  • Well-motivated by cognitive neuroscience transfer-interference literature (Holton et al., 2025) and rich/lazy regime theory (Flesch et al., 2021).
  • Honest about limitations (PCA proxy, γ confound, 3D plots illustrative-only, restricted task structure).
  • Code and data publicly released.

Limitations

  • No quantitative tables with numeric means/std/significance; results are presented via figures and qualitative descriptions, limiting verifiability.
  • Experiments are on small-scale recurrent networks; scalability to Transformers and large-scale continual fine-tuning (e.g., LLMs) is untested.
  • The A-B-A two-task paradigm is a simplification; behavior over longer task sequences or time-varying similarity is unknown.
  • γ confounds dimensionality with optimization dynamics and conditioning, so the causal claim "dimensionality gates modularity" is not cleanly isolated.
  • PCA-based effective dimensionality (99% variance threshold) is an approximate, threshold-sensitive proxy for intrinsic dimension.
  • No comparison to regularization-based (EWC) or replay-based continual learning methods — only architectural separation vs. none.
  • The shared-readout design choice, hypothesized to produce graded (not discrete) geometry, is not ablated against fully separate readouts.

Questions for Further Exploration

  • How do these findings translate to Transformer architectures and large-scale continual fine-tuning of LLMs, where representation dimensionality is typically very high — does modularity become irrelevant unless dimensionality is explicitly constrained?
  • Can the effective dimensionality of intermediate representations be used as a real-time diagnostic metric to decide whether to invest in modular architecture during training?
  • Does the graded-geometry pattern (principal angles scaling with task dissimilarity) emerge in settings with more than two sequential tasks, or does the picture become more complex?
  • What are the theoretical conditions under which high-dimensional representations guarantee low interference — is there a dimensionality threshold relative to task complexity?
  • How does architectural modularity interact with replay-based or regularization-based (EWC) continual learning methods in the low-dimensional regime?
  • Does ablating the shared readout (using fully separate readouts) eliminate the graded geometry and produce fully discrete task separation, confirming the hypothesized mechanism?
  • Can dimensionality itself be regulated dynamically during learning (rather than set indirectly via γ) to activate modularity on demand as task structure changes?
