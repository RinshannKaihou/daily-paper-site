Memory Depth, Not Memory Access: Selective Parametric Consolidation for Long-Running Language Agents

arXiv: 2606.26806

Date: 2026-06-25

https://arxiv.org/abs/2606.26806

一句话总结： 本文将长期运行智能体的记忆区分为"可检索的记忆访问 (memory access)"与"持久塑造行为的记忆深度 (memory depth)"，提出 loop-drift 受控测试协议与 EVAF（惊奇度×效价门控的 LoRA 合并机制），在 GPT-2/TinyLlama 上实现"深度翻转"——RAG 擅长浅层事实回忆（0.956–0.973），而 EVAF 在目标持久性与卸载后恢复上更强（0.812–0.904），且每 200 事件仅需 2–3 次参数写入。
TL;DR: The paper separates memory access (retrieval) from memory depth (durable, goal-conditioned parametric tendencies that survive context unload). Using a synthetic loop-drift protocol and the EVAF surprise×valence-gated LoRA consolidation mechanism, it shows a "depth flip": RAG owns shallow factual recall while EVAF owns goal persistence and post-unload recovery with only 2–3 writes per 200 events, and factorizes selective consolidation into two separable controls — selection and actuation.

核心贡献

  • 概念区分 (memory access vs. memory depth)：明确区分"能检索到什么事实"(memory access) 与"哪些经验应在不被重新插入上下文的情况下持续塑造行为"(memory depth)，并指出两者互补、不可互相替代。
  • loop-drift 受控协议：设计了同时控制四要素（显式干扰、持久的检索索引、可观测的上下文卸载、独立的事实/目标探针）的压力测试协议，确保 EVAF 的优势不是"RAG 忘了"的伪迹。
  • EVAF 选择性合并机制：用惊奇度与效价的门控决定是否将事件写入小缓冲区，缓冲区满后以 replay + L2 anchor 更新 LoRA 适配器，实现稀疏、目标相关的参数写入。
  • 深度翻转的实证发现：在 GPT-2 与 TinyLlama 上，RAG 在短期事实上最强（0.956–0.973），而 EVAF 在目标持久性与卸载后恢复上最强（0.812–0.904），写入次数远少于 Naive-LoRA（200 次/流）。
  • 选择与驱动的可分离性：通过 matched-random-gate、fixed-inner（1/2/3 步）与 Mistral-7B 的反演实验，证明"选择哪些事件"与"写入多强"是两个独立的可控维度，且二者在线上呈非对称耦合（驱动反作用于后续选择）。

Key Contributions

  • A conceptual split: Formulates the memory-access vs. memory-depth distinction for long-running language agents, arguing retrieval answers "what can be fetched?" while parametric consolidation answers "what should keep shaping behavior after context unload?"
  • A controlled loop-drift protocol: A synthetic stress test (10 users × 200 events each) that keeps the retrieval index durable while unloading working context, isolating goal-conditioned persistence rather than retrieval recall.
  • EVAF consolidation mechanism: A surprise-times-valence gate admitting behavior-relevant events into a small buffer, with LoRA updates guarded by replay and an L2 anchor.
  • The "depth flip": Empirical evidence across GPT-2 and TinyLlama that RAG dominates shallow factual recall (0.956–0.973) while EVAF dominates goal persistence and post-unload recovery (0.812–0.904) at only 2–3 parametric writes per 200 events.
  • Selection–actuation factorization: Mechanism controls (matched random gate, fixed-inner steps, Mistral-7B matched-gate inversion) demonstrate that which events are selected and how strongly they are written are separable control factors with asymmetric online feedback.

Methodology

The paper introduces a loop-drift protocol: each run has 10 users with 200 events per user drawn from a controlled mixture of stable goal/preference reminders, distractors, transient opposite requests, conflicts, sibling-user contamination, and scheduled factual notes. Crucially, the retrieval index remains durable across a context unload — so an EVAF advantage on the goal layer is not a trivial "RAG forgot" artifact. It probes four memory layers: shallow episodic access, noisy episodic access, parametric tendency (goal persistence), and post-unload recovery.

EVAF maintains a small write buffer. For each event it computes a surprise score s_t (token negative log-likelihood) and a valence score v_t (embedding similarity to the user's durable goal/preferences). The admission gate is:

g_t = σ(k_s·(s_t − τ_s)) · σ(k_v·(v_t − τ_v)), with constants k_s=1, k_v=10, τ_v=0.5, τ_w=0.5.

When the buffer fills, a LoRA adapter is updated on the buffer plus replay from previously consolidated events, with an L2 anchor: Δθ_evaf ∝ −Σ_{t∈A}∇θℓ_t(θ) − λ(θ−θ_0).

Baselines include Frozen, Summary, RAG (durable embedding index, top-3 cosine), Naive-LoRA (writes every event, 200 writes/stream), EVAF, Routed EVAF+RAG. Mechanism controls add: matched random gate (same write count, random events), fixed-inner controllers (Fixed-1/2/3 inner-loop steps), a behavior-margin control, and the Mistral-7B four-seed fixed-inner audit. Results on GPT-2/TinyLlama are averaged over four seeds; the public Memora event streams serve as an external boundary diagnostic evaluated via a McNemar test.

Results & Highlights

  • Depth flip (Table 2, Figure 1): TinyLlama — EVAF 0.833 goal / 0.812 post-unload vs. RAG 0.396 / 0.394; GPT-2 — EVAF 0.904 / 0.900 vs. RAG 0.398 / 0.394, with only 2.4–2.6 writes and L2 drift ~21–29 (vs. Naive-LoRA's 200 writes and drift ~67–119).
  • Routed EVAF+RAG recovers shallow factual access while preserving/improving the goal layer (e.g., GPT-2: 0.935 short-fact, 0.927 goal, 0.925 post-unload), supporting complementarity.
  • Writing everything is not enough: Naive-LoRA writes every event yet does not cleanly solve the goal layer; at Mistral-7B, indiscriminate writing collapses to 0.333 goal persistence (below 0.500 chance).
  • Matched gate: On GPT-2 EVAF beats the random matched gate on all four seeds (goal/post 0.790/0.763 vs. 0.590/0.619), showing selection is not reducible to sparsity. On Mistral-7B under miscalibrated 5-step actuation the comparison inverts (EVAF 0.362/0.312 vs. Random 0.688/0.681) — a diagnostic of asymmetric selection–actuation coupling.
  • Actuation audit (Table 3): Write strength is a separable, model-dependent factor. Mistral-7B Fixed-2 recovers 0.796/0.775, Fixed-1 reaches 0.919/0.938 but contamination saturates at 1.000 across all seeds. No universal optimum: GPT-2 best at Fixed-1, TinyLlama pays high contamination for Fixed-1/Margin.
  • Memora boundary (Table 4): Directionally positive but not significant (91/222 → 95/222, McNemar p=0.57; high-mutation slice p=0.36), explicitly framing stale-memory invalidation as unsolved future work.

Authors & Credibility

Single author: Haoliang Han, Institute of Biomedical Strategy, China Pharmaceutical University. The work is positioned as a mechanistic probe with a deliberately narrow claim and a dedicated Scope and Limitations section; the author is unusually transparent that the public-benchmark (Memora) result is a boundary, not a win, and that the contribution is conceptual framing plus controlled mechanism controls rather than a SOTA benchmark score. The affiliation (biomedical-strategy institute at a pharmaceutical university) is atypical for a core LLM-agent-memory paper, which slightly raises questions about independent reproduction infrastructure, though the methodology itself is clearly and honestly reported.

Critical Assessment

  • Novelty (moderate): The conceptual split of memory access vs. memory depth is genuinely useful framing, and the selection/actuation factorization with asymmetric online coupling is a clean mechanistic contribution. However, EVAF itself is a recomposition of well-known ingredients — surprise/valence gating (related to novelty-based salience), LoRA, replay, and L2/EWC-style anchoring — so the algorithmic novelty is incremental; the value is in the controlled study and framing.
  • Rigor (strong on controls, weak on external validation): The mechanistic controls are commendable — matched-random-gate, fixed-inner audit, and the Mistral matched-gate inversion are well-designed ablations that genuinely isolate factors. Statistical reporting uses means ± std and a McNemar test for Memora. However, the central "depth flip" is partly engineered into the protocol by construction (RAG keeps the fact in its durable index, so it wins short-fact by design; EVAF's goal-conditioned gate rejects off-topic facts, so it "wins" the layer it is built for). The honest framing mitigates this, but readers should note the protocol defines the win condition.
  • Reproducibility (mixed): Gate constants, seed counts, and protocol parameters are reported, and the write/L2 accounting is carefully defined. But there is no code or data release link in the PDF, the loop-drift generator is synthetic and self-defined (not public), and absolute scores vary across tables in ways that require careful within-table-only comparison. Reproduction would require reimplementing the synthetic generator.
  • Significance (moderate, narrow): The claim is intentionally narrow and well-hedged. Validated only up to 7B (Mistral), with no frontier-model evidence. The mechanism claim holds, but its practical deployment impact is unproven.
  • Red flags:
  • Single author and atypical affiliation for the topic.
  • Heavy citation of a cluster of 2026 concurrent arXiv preprints (e.g., 2606.04536, 2606.05698, 2606.19172, 2606.22844, 2605.30260); these support positioning but cannot yet corroborate claims.
  • Synthetic, self-designed benchmark with the result partly built into its structure.
  • Heldout-paraphrase generalization is weak/absent (TinyLlama directional, GPT-2 not positive), so the durable-behavior claim is limited to held-in goal-conditioned behavior.
  • No code/data artifacts apparent.

Strengths

  • Clean, well-motivated conceptual distinction between retrieval-based access and parametric depth.
  • Strong, layered mechanism controls that genuinely factorize selection from actuation, including the elegant Mistral matched-gate inversion as a coupling diagnostic.
  • Honest, self-limiting claims: the paper repeatedly resists overclaiming (e.g., explicit "Memora is a boundary, not a victory benchmark").
  • Complementarity is demonstrated, not just asserted — Routed EVAF+RAG shows the division of labor concretely.
  • Detailed, conservative reading instructions for tables (within-run comparisons, L2 compared only within model family).

Limitations

  • Synthetic loop-drift protocol is self-designed; the only public benchmark (Memora) yields non-significant results.
  • No heldout-paraphrase generalization — behavior claims are limited to held-in goal-conditioned tendencies.
  • Only up to 7B scale; Mistral still shows high sibling contamination under strong actuation, and the Margin controller has high cross-seed variance.
  • High-actuation regimes trade goal movement for contamination (Fixed-1 contamination saturates at 1.000 on Mistral).
  • Forgetting/stale-memory invalidation is explicitly unsolved; negative-gradient/anti-training approaches were found unstable.
  • No apparent code or dataset release; absolute scores are not directly comparable across the main table and the audit table.

Relevance

这篇论文直接契合"agent memory / parametric consolidation"研究方向：它正是研究如何用 LoRA 形式的参数化合并为长期运行智能体建立持久的目标条件行为，区分于纯检索式记忆访问，是参数化记忆合并机制方面的一份精细机制研究，对设计具备持久偏好的智能体记忆系统有直接参考价值。

Questions for Further Exploration

  • Can the selection–actuation factorization be validated on frontier-scale models (e.g., 70B+), where does the Mistral fixed-inner trend extrapolate, and does contamination under strong actuation ever become controllable?
  • How would a validity-gated reactivation / reconsolidation scheme (the author's own proposed future direction) actually close the stale-memory gap exposed by Memora, and can it make that boundary result statistically significant?
  • Is the "depth flip" an artifact of the loop-drift protocol's construction (RAG index is durable by design), or does it replicate on existing public post-unload benchmarks if such probes are added to LongMemEval/LoCoMo?
  • What is the right way to jointly optimize selection and actuation online given their asymmetric coupling, rather than treating them as independently tuned hyperparameters?
  • Does the EVAF gate transfer across model families, or do k_s, k_v, τ_v, τ_w need per-model calibration, and can the warmup scheme be made fully adaptive?
