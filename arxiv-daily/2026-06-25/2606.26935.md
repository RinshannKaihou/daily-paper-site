> **TL;DR:** The paper asks whether CoT supervision in long-context LLM agents improves reasoning-based action *revision* or direct *promptвҶ’action* prediction. Across three agent benchmarks (ALFWorld, ScienceWorld, BFCL) and Qwen3-4B/8B/14B/32B + Llama-3.1-8B checkpoints, prompt-action accuracy rises in parallel with CoT-action accuracy and the CoT-vs-prompt gap stays flat (~0.1вҖ“0.2 acc-gap); GPT-5.4's CoT-win rate on ALFWorld holds near 68вҖ“76% across checkpoints. Later checkpoints increasingly preserve the prompt action under conflicting traces. Attention (вүҲ80% mass on prompt tokens) and gradient diagnostics (Table 4: 73вҖ“84% of action-loss gradient on prompt tokens) explain the prompt's structural advantage. A "reduced action supervision" intervention (mask action-token loss on k=0.3 of samples) improves OOD score, e.g. Qwen3-8B ScienceWorld OOD 0.40вҶ’0.44 and BFCL OOD 0.63вҶ’0.69 under GRPO (Table 2).

## Key Contributions

- **A diagnostic framework** that compares *prompt actions* (action decoded by prefilling the `<action>` tag, bypassing CoT) with *CoT actions* (action decoded after a step-by-step trace), giving a behavioral measure of how much the final action is already recoverable from the prompt alone.
- **Behavioral evidence (training dynamics + online eval + conflicting traces)** that CoT supervision in long-context agents improves direct promptвҶ’action prediction in parallel with CoT-conditioned prediction: the CoT-vs-prompt gap stays flat across checkpoints (Figures 1вҖ“3), and later checkpoints increasingly preserve the prompt action under a substituted conflicting trace (Figure 4) вҖ” i.e. the model becomes more, not less, prompt-anchored.
- **Mechanistic explanation via attention + gradient diagnostics**: in agent settings the prompt is much longer than the trace, prompt tokens receive вүҲ80% of action-time attention mass, and (Table 4) 73.2вҖ“84.6% of the action-only loss gradient lands on prompt tokens vs CoT tokens вҖ” a value-path gradient decomposition (вҲӮL_t/вҲӮv_i = Оұ_{t,i}В·Оҙ_t) shows this asymmetry mechanically biases optimization toward the prompt pathway.
- **A training intervention вҖ” reduced action supervision**: mask the loss on final action tokens for a random k% (default 0.3) of training examples, optimizing only the CoT span there while keeping advantage estimates unchanged in RL. This improves OOD generalization and widens the CoT-minus-prompt gap in ALFWorld/ScienceWorld.
- **A short-prompt contrast**: on MATH/MedQA/GPQA (short prompts, no interaction history) prompt/CoT consistency does *not* rise over training (Figure 9), isolating the long-prompt length asymmetry as the driver of the shortcut pattern.

## Methodology

**Environments.** Three agent benchmarks: ALFWorld (embodied household tasks), ScienceWorld (interactive scientific reasoning), and BFCL (Berkeley Function-Calling Leaderboard, tool use). Max interaction steps: 30 for ALFWorld/ScienceWorld, 15 for BFCL. OOD splits: ALFWorld вҖ” "Cool & Place" and "Pick Two & Place" unseen (following Zhang et al., 2025b); ScienceWorld вҖ” last task type of each topic held out; BFCL вҖ” long-context setting as OOD.

**Models & training.** Qwen3-4B/8B/14B/32B and Llama-3.1-8B-Instruct (plus Llama-3.2-3B-Instruct in appendix). SFT: 5 epochs, lr 1e-5, batch 16. RL: GRPO (Shao et al., 2024), 200 steps, lr 1e-6, batch 32, group size 8, KL coeff 0.01, format-violation reward penalty вҲ’0.1. Trajectories collected with GPT-5.4; successful ones used for SFT. Each setting run 3Г— with mean reported.

**Two decoding modes.** *CoT action*: model generates `<thinking>вҖҰ</thinking>` then `<action>вҖҰ</action>`. *Prompt action*: prefill the response with the `<action>` tag so the model emits the action directly with no reasoning. The gap between the two measures how much of the decision depends on reasoning vs is already in the prompt.

**Three-step diagnosis.** (1) Track prompt-action and CoT-action accuracy + their consistency over checkpoints (В§4.1). (2) Online evaluation on unseen tasks: trajectory success under prompt-action vs CoT-action execution, plus a local action-level comparison using GPT-5.4 as a judge (win rate of CoT over prompt in matched decision contexts) вҖ” manually checked GPT-5.4 agrees with humans on 100 examples 93% of the time (В§4.2). (3) Conflicting-trace test: replace original trace t1 with a random trace t2 supporting a different action, truncate t2 at ratios {0.1, 0.3, 0.5}, use prefilling so the model treats t2 as its own thinking, then append `<action>` and measure agreement with the original prompt action vs the t2-implied action (В§4.3).

**Mechanistic analysis (В§5, App. C).** Attention mass averaged across layers/heads on prompt vs CoT tokens during action generation. Plus a value-path gradient decomposition: at action position t, вҲӮL_t/вҲӮv_i = Оұ_{t,i}В·Оҙ_t, so the gradient share on prompt vs CoT tokens is weighted by their attention weights. They define an action-only loss L_act = вҲ’ОЈ M(t) log pОё(y_t | x_<t) and compute per-token gradient norms g_i = вҲӮL_act/вҲӮh_i, reporting R_prompt = ОЈ_{iвҲҲP} g_i / ОЈ_{iвҲҲPвҲӘC} g_i.

**Intervention (В§6).** For a random k% of training samples, mask the loss on action tokens and optimize only the CoT span: L_cot = вҲ’ОЈ M(t) log pОё(y_t | p, y_<t), where M(t)=1 on the CoT span. Keep supervision on the `<action>` tag so the model still learns when to switch from reasoning to action. Apply the same masking in GRPO while keeping advantage estimates unchanged. Default k=0.3; ablation over kвҲҲ{0.1,0.3,0.5,0.7} in Table 3.

## Results & Highlights

- **Parallel improvement, flat CoT gap (Figure 1, В§4.1).** Across Qwen3-4B/8B/14B and Llama-3.1-8B on ALFWorld/ScienceWorld/BFCL, prompt-action and CoT-action alignment-with-reference both rise over checkpoints while the gap between them stays roughly constant вҖ” the first signal that gains are increasingly prompt-recoverable.
- **Online OOD evaluation, ALFWorld (Figure 3).** (a) Trajectory success gap between CoT-action and prompt-action execution stays flat across checkpoints for Qwen3-4B/8B/14B/32B (acc_gap in In-Domain and Out-Domain вүҲ 0вҖ“0.2). (b) GPT-5.4 CoT-win rate over prompt actions in matched decision contexts holds at roughly **68вҖ“76%** across all checkpoints (e.g. Qwen3-32B In-Domain ~74%, Out-Domain ~76%; Qwen3-14B ~72%/70%), i.e. CoT helps but its advantage does not grow.
- **Conflicting-trace test (Figure 4, ALFWorld).** Later checkpoints more often preserve the original prompt-based action when given a substituted conflicting trace, at all truncation ratios {0.1,0.3,0.5} and in both in-domain and out-domain вҖ” replicated in BFCL (Figure 11) and ScienceWorld (Figure 12), and under RL (Figure 14) and for Llama (Figure 15). OOD settings are consistently *more* prompt-anchored than in-domain.
- **Attention dominance (Figure 5).** вүҲ**80%** of total attention mass during action generation falls on prompt tokens (not CoT tokens) across environments; even among the top-K% highest-attention positions, more than half come from the prompt вҖ” beyond what prompt length alone explains.
- **Gradient dominance (Table 4).** Share of action-only loss gradient on prompt vs response (CoT) tokens: Qwen3-4B ALFWorld **73.2 / 26.8**, ScienceWorld 76.5 / 23.5, BFCL 84.4 / 15.6; Qwen3-8B 72.7 / 27.3, 78.9 / 21.1, 84.6 / 15.4; Qwen3-14B 73.5 / 26.5, 79.2 / 20.8, 82.7 / 17.3 вҖ” prompt tokens consistently receive the dominant gradient share.
- **Reduced-action-supervision intervention, OOD (Figure 6).** Across model sizes {4B,8B,14B,32B} and environments, mean OOD score improves under the intervention vs vanilla CoT supervision, with the largest gains in ALFWorld and ScienceWorld (more modest/mixed in BFCL, whose structured prompts and constrained action space leave less room). The CoT-minus-prompt gap becomes larger in ALFWorld/ScienceWorld under the intervention (Figure 7).
- **GRPO results (Table 2).** Qwen3-4B ours vs GRPO: ALFWorld 0.83/0.82 (in), 0.52/0.44 (out), ScienceWorld 0.49/0.46 (in), 0.24/0.22 (out), BFCL 0.71/0.69 (in), 0.64/0.61 (out). Qwen3-8B ours vs GRPO: ALFWorld 0.95/0.95 (in), **0.89/0.86 (out)**, ScienceWorld 0.61/0.57 (in), **0.44/0.40 (out)**, BFCL 0.77/0.74 (in), **0.69/0.63 (out)** вҖ” the intervention helps most OOD.
- **Baseline comparison (Table 1, Qwen3-8B).** ALFWorld: SFT 0.55, DPO 0.59, FRODO 0.61, SFT+(ours) 0.63, DPO+(ours) 0.65; ScienceWorld: 0.25 / 0.29 / 0.30 / 0.27 / **0.35**; BFCL: 0.58 / 0.62 / 0.64 / 0.62 / **0.67** вҖ” the method underperforms alone with SFT on some tasks but performs best when combined with DPO.
- **Masking-ratio ablation (Table 3, Qwen3-8B).** k=0.1: ALFWorld 0.61, Sci 0.26, BFCL 0.61; k=0.3: 0.63 / 0.27 / 0.62; k=0.5: 0.61 / 0.24 / 0.59; k=0.7: 0.6 / 0.27 / 0.61 вҖ” k=0.3 is the sweet spot, supporting the default.
- **Short-prompt contrast (Figure 9, В§A.3).** On MATH/MedQA/GPQA (short prompts) prompt/CoT-action consistency does *not* rise over training for Qwen3-8B or Llama-3.1-8B, isolating long-prompt length asymmetry as the driver of the agent shortcut pattern.

## Critical Assessment

- **The behavioral proxy is not a direct measurement of internal computation.** The authors themselves state in Limitations that prompt actions are "only a behavioral proxy for how much of the final action can be predicted from the prompt rather than a direct measurement of internal computation" вҖ” so the strong claim that CoT training is *not* improving latent reasoning cannot be conclusively settled by this method, only shown to be consistent with the data.
- **GPT-5.4 as a judge introduces evaluation bias.** Local action-quality comparisons (Figure 3b, the ~68вҖ“76% win rates) rely on GPT-5.4; although the authors manually check 93% agreement with humans on 100 examples, that is a small sample and a single judge, and the flat-gap conclusion rests partly on this judge's stability across checkpoints.
- **The intervention's gains are environment-dependent and sometimes within-noise.** In BFCL the change is "more mixed," and Table 3's masking-ratio ablation (ALFWorld 0.61/0.63/0.61/0.6 across k=0.1/0.3/0.5/0.7) shows differences of 0.01вҖ“0.03 with no reported confidence intervals despite 3 seeds вҖ” making some "improvements" hard to distinguish from run-to-run variance.
- **SFT-only variant underperforms (Table 1).** The reduced-action-supervision objective with SFT alone "underperforms on some tasks" and only wins when combined with DPO, which complicates the claim that the intervention itself (rather than the DPO combination) drives the gains.
- **Single base model family dominates.** Most headline results are on Qwen3-4B/8B/14B/32B; Llama-3.1-8B-Instruct and Llama-3.2-3B-Instruct appear only in appendix figures, so generalization across architectures is thin.
- **No single-number success-rate table for the main diagnostic.** The flat-gap conclusion is supported by figures (Figures 1вҖ“4) rather than a consolidated table with confidence intervals, making it harder to assess effect sizes at a glance.

## Strengths

- Clean, well-motivated diagnostic: the prompt-action vs CoT-action decomposition is a simple, reusable probe for separating reasoning-mediated revision from prompt-based shortcuts.
- Multi-pronged evidence: offline training dynamics, online OOD evaluation, conflicting-trace tests, plus attention and gradient mechanistic analyses all converge on the same story.
- The value-path gradient decomposition (вҲӮL_t/вҲӮv_i = Оұ_{t,i}В·Оҙ_t) gives a genuine optimization-level reason вҖ” not just a correlation вҖ” for why long prompts dominate action supervision.
- The short-prompt contrast (MATH/MedQA/GPQA, Figure 9) is a strong control that isolates prompt-length asymmetry as the cause rather than CoT training per se.
- The reduced-action-supervision intervention is simple, cheap, and acts as a constructive check on the diagnosis (it predicts and exhibits widened CoT gap + better OOD).
- Honest about scope: the conclusion explicitly says the findings "do not settle whether latent reasoning ability improves in an absolute sense."

## Limitations

- Prompt-actions are a behavioral proxy, not a direct internal-computation measurement, so the strong "CoT training mainly helps the shortcut" claim is not fully settled.
- Reliance on GPT-5.4 as judge for local action-quality comparisons, with only 100-example human agreement validation.
- Main results concentrate on the Qwen3 family; Llama evidence is appendix-only.
- Intervention gains in BFCL are mixed and several ablation differences (Table 3) are small and reported without confidence intervals despite only 3 seeds.
- SFT-only intervention underperforms on some tasks; the headline wins require combining with DPO.
- Three agent benchmarks only вҖ” all text-based, long-context, single-agent; no multi-agent, code-execution, or multimodal agent settings.
- "OOD" splits are narrow (two ALFWorld task types, one task type per ScienceWorld topic, BFCL long-context subset), so generalization claims rest on a thin OOD slice.

## Questions for Further Exploration

1. The prompt-action probe bypasses CoT but still runs the same forward pass вҖ” could a *latent-state* probe (e.g. linear probe on hidden states at the action position, in the spirit of Paul et al. 2024 cited in the paper) confirm whether the action is already encoded *before* the trace is generated, strengthening the "shortcut" claim beyond a behavioral proxy?
2. The value-path gradient decomposition (Eq. 7) predicts prompt share scales with attention mass вҖ” does *length-normalizing* the prompt (or explicitly re-weighting attention toward CoT tokens during action generation) flatten the prompt/CoT gradient asymmetry and recover a widening CoT gap?
3. Reduced action supervision underperforms with SFT alone (Table 1) but wins with DPO вҖ” is the gain attributable to the intervention or to the DPO objective's preference signal, and would the same k=0.3 masking applied inside DPO (not just SFT) change the result?
4. The short-prompt control (Figure 9, MATH/MedQA/GPQA) shows no rising consistency вҖ” does the shortcut pattern reappear in *long-context but non-agent* settings (e.g. long-document QA), isolating "interaction history in the prompt" from "prompt length" as the driver?
5. Table 4's gradient shares (73вҖ“84% on prompt tokens) are aggregated across layers вҖ” at which transformer layers is the prompt dominance strongest, and does layer-specific gradient clipping on prompt tokens reproduce the reduced-action-supervision effect more cheaply?
6. The conflicting-trace test (Figure 4) shows later checkpoints preserve the prompt action more вҖ” is this because the trace's gradient leverage shrinks (consistent with В§5) or because the model's prior on the action sharpens independently? A per-checkpoint gradient-share measurement (Table 4 extended over training) would separate these.
